The billable items of Tencent Cloud LVB service include basic billing and value-added billing. Among them, the basic billing is for **traffic/bandwidth** fees, i.e., the postpaid fees for downstream traffic/bandwidth generated by connecting the user and the cache origin server. Value-added billing is for the postpaid fees of **LVB transcoding**, **LVB recording**, **LVB screencapture** and **intelligent porn detection**. These four items are value-added features for LVB and can be used as needed.
