1. Log in to the [Tencent Cloud Console](https://console.cloud.tencent.com/) and select **Products** > **Cloud Compute & Network** > **Virtual Private Cloud** to enter the VPC Console.
2. Click **Cloud Connect Network** in the left pane to enter the CCN management page.
3. 1. In the list, click the ID of the CCN to be associated to enter the details page and click **Associate instance** there. 
  ![](https://main.qcloudimg.com/raw/a76cafac390b4c1f35b196ba59f0d72a.png)
	2. In the pop-up box, select the network instance type (VPC/Direct Connect gateway), region and the specific instance to be associated.
	3. (Optional) If you need to associate another network instance, click **+ New**.
	4. Click **Submit**. 
	  ![](https://main.qcloudimg.com/raw/a76cafac390b4c1f35b196ba59f0d72a.png)
 
 Or:
 Add the instances that need to be associated when creating a CCN instance.
