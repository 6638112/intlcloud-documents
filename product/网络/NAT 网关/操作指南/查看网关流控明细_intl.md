If "Gateway Traffic Control Details" is enabled, you can view the traffic control details of the restricted IP.
1. Log in to [Tencent Cloud Console](https://console.cloud.tencent.com/), select **Products** -> **Virtual Private Cloud** to go to the VPC console, and then click **NAT Gateway** on the left pane.
2. In the list page, select the NAT gateway for which you want to view the details to go to its details page.
3. Click **Monitor** in the tab.
 ![](https://main.qcloudimg.com/raw/c3a7e616d7a8b01a769ea9d5a96ae84a.png)
4. Click **View Restricted IP** in the upper right of the gateway traffic control details table.
 ![](https://main.qcloudimg.com/raw/b77da9bc78fee9266764d383016026ea.png)
 Then you can view the gateway traffic control details.
 ![](https://main.qcloudimg.com/raw/c4385aeb3a96165f0f2b928d44b317cc.png)
