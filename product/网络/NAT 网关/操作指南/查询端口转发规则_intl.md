You can query the port forwarding rules by following the steps below.
1. Log in to [Tencent Cloud Console](https://console.cloud.tencent.com/), select **Products** -> **Virtual Private Cloud** to go to the VPC console, and then click **NAT Gateway** on the left pane.
2. In the NAT gateway list page, select the NAT gateway to be queried to go to its details page, and then select **Port Forwarding**.
 ![1](https://main.qcloudimg.com/raw/63cc05adc23859e8ce500862d121a483.png)
3. In the search box, select the resource attribute values or enter the keywords to query the relevant port forwarding rules.
> **Note:**
> Be sure to check whether an NAT gateway routing policy has been added to the route table associated with the subnet of the instance.

 ![2](https://main.qcloudimg.com/raw/9fc9003f505e40d0a88ea1604d54b734.png)
