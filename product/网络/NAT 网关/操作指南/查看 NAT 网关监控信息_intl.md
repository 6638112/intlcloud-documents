After creating an NAT gateway, you can view and export its monitoring data via the console.
1. Log in to [Tencent Cloud Console](https://console.cloud.tencent.com/), select **Products** -> **Virtual Private Cloud** to go to the VPC console, and then click **NAT Gateway** on the left pane.
2. In the NAT gateway list page, select the NAT gateway for which you want to view the monitoring data to go to its details page, and then click **Monitoring** tab to view its monitoring data. Click **Export Data** or **Export Image** to save the data locally.
![2](https://main.qcloudimg.com/raw/7761b0885f2c37209327605701e46d61.png)
 >**Note:**
 >Alternatively, you can click **Monitor** in the NAT gateway entry in the NAT gateway list page to view its monitoring data.
