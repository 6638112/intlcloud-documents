## API Description
This API (AddSrcBlackIP) is used to add IPs to blacklist of an Anti-DDoS Advance service pack
Protocol: `HTTPS`
Domain Name: `bgpip.api.qcloud.com`
`AddSrcBlackIP`

## Request Parameters
The following request parameter list only provides the API-specific request parameters. Common request parameters are required when the API is called. For more information, see [Common Request Parameters](https://cloud.tencent.com/document/product/1014/31224). The Action field for this API is `AddSrcBlackIP`.

| Parameter | Required | Type | Description |
|---------|---------|---------|---------|
| bgpId | Yes | String | ID of the Anti-DDoS service pack |
| blacklist | Yes | Array | Blacklist:<pre>"blacklist": [</br>"10.1.1.1",</br>"10.2.2.2"</br>"10.3.3.3"</br>]</pre> |

## Response Parameters
None.
