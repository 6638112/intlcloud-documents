## API Description
This API (BGPIPDeleteWadTransRules) is used to delete layer-7 forwarding rules of a protective IP.
Protocol: `HTTPS`
Domain Name: `bgpip.api.qcloud.com`
`BGPIPDeleteWadTransRules`

## Request Parameters
The following request parameter list only provides the API-specific request parameters. Common request parameters are required when the API is called. For more information, see [Common Request Parameters](https://cloud.tencent.com/document/product/1014/31224). The Action field for this API is `BGPIPDeleteWadTransRules`.

| Parameter | Required | Type | Description |
|---------|---------|---------|---------|
| ruleId | Yes | String | ID of the forwarding rule |
| bgpId | Yes | String | Resource ID of Anti-DDoS Advance service pack|

## Response Parameters
None.
