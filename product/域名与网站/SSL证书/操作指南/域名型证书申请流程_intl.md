## Applying for Domain Validated (DV) SSL Certificate

### 1. Application entry

Enter the SSL Certificates Service Console
![](https://mc.qcloudimg.com/static/img/f990389699184b164082dce1ebe809c6/1.png)

Click **Apply for Certificate**
![](https://mc.qcloudimg.com/static/img/4efe78b416cc29cacba1cbc2ba475bb6/2.png)

View the model of the DV certificate, and click **OK**
![](https://mc.qcloudimg.com/static/img/287bd4f6c633cb1a81e18a096f47d5ed/3.png)

### 2. Fill in the application

Enter the domain name, such as qcloud.com, cloud.tencent.com, or demo.test.qlcoud.com.
![](https://mc.qcloudimg.com/static/img/4961164252cd488c9695475e173c0b8c/4.png)

### 3.1 Manual DNS verification

The certificate supports manual DNS verification by default. See [here](https://cloud.tencent.com/doc/product/400/4142#2.-.E6.89.8B.E5.8A.A8dns.E9.AA.8C.E8.AF.81) for a detailed verification method.
![](https://mc.qcloudimg.com/static/img/2f90c6cdf51ec98ba0fd7a112a891e13/5.png)

### 3.2 Choose automatic DNS verification

If the domain name entered is added successfully to the [Tencent Cloud DNS Platform](https://console.cloud.tencent.com/cns/domains), it will support automatic DNS verification. See [here](https://cloud.tencent.com/doc/product/400/4142#1.-.E8.87.AA.E5.8A.A8dns.E9.AA.8C.E8.AF.81) for a detailed verification method.
![](https://mc.qcloudimg.com/static/img/8c10bfb9fa50a520e0b8b45f3b7a9f74/6.png)

### 4.1 Identity verification after submission

When the application is successfully submitted, a pop-up window will appear, notifying that you need to go to the **Certificate Details** page to obtain the CName record and add resolution:
![](https://mc.qcloudimg.com/static/img/7d99496ed47a163a3ee25c728187bf45/7.png)

The process of obtaining the CName record is shown in Tips. Please add resolution as soon as possible so that it can be approved by CAs:
![](https://mc.qcloudimg.com/static/img/1f0d7d113cd4ee14cda423a32e853fe4/8.png)

### 4.2 Failed to submit application

The pop-up window shown below indicates that the submitted domain name is not approved by CA's security verification. See [here](https://cloud.tencent.com/doc/product/400/5439) for detailed reasons.
![](https://mc.qcloudimg.com/static/img/25451d24cf3c717454830a44925642ec/1.png)
