| 接口名称 | 接口功能 |

| [DescribeClusterInstances](https://cloud.tencent.com/document/api/457/31863) | 查询集群节点信息 |
| [DescribeClusters](https://cloud.tencent.com/document/api/457/31862) | 查询集群列表 |
