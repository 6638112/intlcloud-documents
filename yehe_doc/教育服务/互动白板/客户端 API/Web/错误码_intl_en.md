
## TEduBoardErrorCode
Whiteboard error codes (critical) 


| Enumerated Value | Description |
| --- | --- |
| TEDU_BOARD_ERROR_INIT | Failed to initialize the whiteboard. |
| TEDU_BOARD_ERROR_AUTH | Authentication failed. To correct it, purchase the service first. |
| TEDU_BOARD_ERROR_LOAD | Failed to load files. |
| TEDU_BOARD_ERROR_TIM_INVALID | Tencent Cloud IMSDK is unavailable. |
| TEDU_BOARD_ERROR_HISTORYDATA | Failed to sync historical data. |
| TEDU_BOARD_ERROR_RUNTIME | The whiteboard encountered an error during runtime. |



## TEduBoardWarningCode
Whiteboard error codes (warning) 


| Enumerated Value | Description |
| --- | --- |
| TEDU_BOARD_WARNING_SYNC_DATA_PARSE_FAILED | Failed to parse the synchronization data from the other end. |
| TEDU_BOARD_WARNING_TIM_SEND_MSG_FAILED | Tencent IMSDK failed to send the message. |
| TEDU_BOARD_WARNING_H5PPT_ALREADY_EXISTS | The H5PPT to add already exists. |
| TEDU_BOARD_WARNING_BOARD_LOAD_FAILED | Failed to load whiteboard resources. Whiteboard resources are being reloaded. |
| TEDU_BOARD_WARNING_ILLEGAL_OPERATION | No operation is allowed until whiteboard historical data is successfully loaded. |
| TEDU_BOARD_WARNING_H5FILE_ALREADY_EXISTS | This warning occurs when the H5File to add already exists. |
| TEDU_BOARD_WARNING_VIDEO_ALREADY_EXISTS | This warning occurs when the image file to add already exists. |
| TEDU_BOARD_WARNING_IMAGESFILE_ALREADY_EXISTS | This warning occurs when the image file to add already exists. |
