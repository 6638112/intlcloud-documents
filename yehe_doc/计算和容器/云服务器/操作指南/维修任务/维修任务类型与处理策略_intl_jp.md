## タスクのステータス

| タスクのステータス | 意味                                                         |
| -------- | ------------------------------------------------------------ |
| 承認待ち  | ユーザーの承認を待機中であり、ユーザーはメンテナンス方式とメンテナンス時間を指定できます。48時間を超えてもユーザーが承認しない場合は、システムが自動的にステータスを処理中に変更し、メンテナンス操作を実行します。 |
| 予約済み   | ユーザーが操作を承認済みであり、メンテナンス時間を予約しました。タスク作成より48時間以内であれば承認したステータスを変更できます。 |
| 処理中   | メンテナンスタスクを実行中です。                                               |
| 完了   | メンテナンスタスク処理が完了しました。                                            |
| 回避済み   | インスタンスにメンテナンスタスクが発生した場合に、ユーザーがインスタンスの返却、破棄、設定調整などの操作を行うことで、今回のメンテナンスタスクフローが中断され回避されます。 |
| キャンセル済み  | 今回のメンテナンスタスクがシステムによってキャンセルされました。                                       |
