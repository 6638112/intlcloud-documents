## 작업 상태

| 작업 상태 | 의미                                                         |
| -------- | ------------------------------------------------------------ |
| 인증 대기 중   | 사용자 인증을 기다리는 동안 사용자는 점검 방법 및 점검 시간을 지정할 수 있습니다. 사용자가 48시간 이상 인증되지 않은 경우 시스템은 처리 중 상태로 자동 전환되고 점검 작업을 수행합니다. |
| 예약   | 사용자가 작업을 인증했으며 점검 시간을 예약합니다. 작업이 생성된 후 48시간 이내에 인증 상태를 수정할 수 있습니다. |
| 처리 중   | 점검 작업 진행 중                                               |
| 종료   | 점검 작업 처리 완료                                            |
| 방지   | 인스턴스에 유지 보수 작업이 발생하면 사용자는 인스턴스의 반환, 폐기 및 설정 변경과 같은 작업을 수행하여 이번 유지 보수 작업을 방지하는 프로세스를 중단합니다. |
| 취소   | 이번 유지 보수 작업은 시스템에 의해 취소되었습니다.                                       |
