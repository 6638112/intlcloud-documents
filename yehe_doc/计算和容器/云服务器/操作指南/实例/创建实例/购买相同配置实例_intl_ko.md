## 작업 시나리오
이미 구매했었던 CVM 인스턴스와 동일한 구성의 인스턴스를 다시 구매해야 한다면, CVM 콘솔이 제공하는 "동일한 구성 구매" 기능을 사용하여 동일한 구성의 CVM 인스턴스를 빠르게 구매할 수 있습니다. 이 기능을 사용함으로써 시간을 절약할 수 있을 뿐 아니라 특정 시나리오의 스케일 아웃 효율도 향상할 수 있습니다.

## 작업 순서

1. [CVM 콘솔](https://console.cloud.tencent.com/cvm/instance/index?rid=19)에 로그인합니다.
2. 인스턴스 관리 페이지의 상단에서 리전을 선택합니다.
3. 작업 대기 중인 인스턴스의 작업 열에서 [More] > [Purchase with same configurations]를 클릭합니다.
![](https://main.qcloudimg.com/raw/990ca94e95a27a66b46a7ee5d20e296a.png)
4. CVM 구매 수량을 설정한 뒤, 자동 선택된 구성 정보를 확인합니다.
실제 필요에 따라서 구매할 CVM의 매개변수 구성을 조정할 수 있습니다.

5. [구매하기]/[활성화]를 클릭하여 결제를 완료합니다.
