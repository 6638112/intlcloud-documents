### How do I expand a cloud disk?
If your CVM uses a cloud disk, you can expand the disk capacity. For more information about expansion operations, see [Expanding Cloud Disk Capacity](https://intl.cloud.tencent.com/document/product/362/5747).

### Can I reduce the capacity of a cloud disk?
No. If you want to reduce the capacity of a cloud disk you purchased, we recommend that you create a cloud disk of your desired capacity and attach it to the same instance that the original cloud disk is attached to. Then, copy the data stored in the original disk to the new disk and release the original disk.

### How do I expand a system disk?
You can expand a system disk through the CVM console. For more information, see [Expanding Cloud Disks](https://intl.cloud.tencent.com/document/product/213/32377).

### Can the system disks of all types of cloud disks be expanded?
Yes. The system disks of all types of cloud disks, including SSD, Premium Cloud Storage, and HDD, can be expanded.

### Can the system disks of any pay-as-you-go CVM instances be expanded?
Yes. The system disks of any pay-as-you-go CVM instances can be expanded.
