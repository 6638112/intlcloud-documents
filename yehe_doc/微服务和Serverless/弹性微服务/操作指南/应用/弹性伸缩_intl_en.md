## Overview

This document describes how to modify the number of application instances in the TEM console.

## Directions

1. Log in to the [TEM console](https://console.cloud.tencent.com/tem).
2. On the left sidebar, click **Application List** to enter the application list page and select your application deployment region.
3. Click **Scale** in the **Operation** column of the target application to modify the number of instances.
   - Manual adjustment: select the number of instances.
   - Auto-scaling: set the number of instances and auto-scaling rule.
4. Click **Submit** to modify the number of instances.
