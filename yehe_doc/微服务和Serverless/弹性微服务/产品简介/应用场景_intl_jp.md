Tencent Cloud Elastic Microserviceは主に以下のシーンに適用します。

## マイクロサービスアプリケーションユースケース

チームとビジネス規模の拡大に伴い、バックエンドサービスはモノリシックアーキテクチャからマイクロサービスアーキテクチャへと徐々に進化してきました。マイクロサービスアーキテクチャは、アジャイル開発、フレキシブルなデプロイ、高スケーラビリティを提供する一方で、サービスガバナンスと維持管理をより複雑化させ、技術的なハードルが高くなりました。

上述のマイクロサービスアーキテクチャによって生じる問題を改善するため、Tencent Cloud Elastic Microserviceは、アプリケーションホスティング、サービスの登録と検出、マイクロサービスガバナンス、コールチェーンおよび多次元的な監視などの機能を介して、すぐに使用できるマイクロサービスソリューションをお客様に提供します。企業ユーザーがマイクロサービスアプリケーションを迅速に構築し、維持管理の効率を大幅に向上させ、サービスガバナンスの複雑さと技術的ハードルを下げ、コアビジネスに集中できるように促し、お客様の成功をサポートします。

![](https://main.qcloudimg.com/raw/66707dd5f25fddebf46f17a7ea08a95f.svg)

## 自動スケーリング

eコマース、ビデオ、ゲーム、証券、eラーニングなどのアプリケーションでは、ビジネストラフィックが周期的な特性、急激な変化を示すケースで、突発的なトラフィックピークが出現しやすいことから、アクセス応答のタイムアウト、エラー率の増加などの問題が発生しやすくなります。Tencent Cloud Elastic Microserviceは、企業のお客様がトラフィックピークに手軽に対処できるよう、秒レベルの自動スケーリング機能を提供します。

![](https://main.qcloudimg.com/raw/4929c624253410282fb6e32d536b9d8e.svg)

## 継続的インテグレーションとデリバリー

Tencent Cloud Elastic Microserviceは、クライアントがマイクロサービスアプリケーションのインテグレーションとデリバリーを継続し、迅速なイテレーションを実現することに寄与します。コード開発フェーズからアプリケーションデリバリーフェーズまで、Tencent Cloud Elastic Microserviceは、IDEAプラグイン、複数のデプロイ形式、グレーリリースなど、複数のリリース機能を提供し、企業のお客様によるビジネスバリューの迅速な検証に寄与します。
