# 製品の価格設定
## 課金項目と単価
Tencent Cloud Elastic Microserviceの従量課金と単価は次のとおりです：

<table>
    <tr>
        <th>カテゴリ</th><th>課金項目</th><th>単価</th>
    </tr>
    <tr>
        <td>環境費用</td><td>環境</td><td>0.00024876 ドル/個/分</td>
    </tr>
    <tr>
        <td rowspan="2">アプリケーション費用</td><td>CPU</td><td> 0.00031344 ドル/コア/分</td>
    </tr>
    <tr>
        <td>メモリ</td><td> 0.0001195 ドル/GiB/分</td>
    </tr>
</table>

## 課金の例
Tencent Cloudマイクロサービス製品のTencent Cloudアカウントでのリソース使用量が次のようになっているとします：

- 2022年8月1日の10:00:10に環境を作成し、2022年8月1日の10:29:40にこの環境を破棄しました。
- 2022年8月1日の10:15:20に特定のアプリケーションをこの環境にデプロイし、5つの1C2GiBのインスタンスを作成し、2022年8月1日の10:24:59にインスタンスの実行を終了します。
- 他のTEMリソースはありません。

この場合、Tencent Cloudアカウントは、2022年8月1日の11:00:00〜12:00:00の間に請求書を受け取り、請求額は次のように計算されます：

<table>
    <tr>
        <th>カテゴリ</th><th>仕様</th><th>実行時間（1分未満の場合は、1分としてカウントされます）</th><th>金額</th>
    </tr>
    <tr>
        <td>環境費用</td><td>1つ</td><td>30分（2022年8月1日の10:29:40～2022年8月1日の10:00:10=29分30秒）</td><td>0.0074628 ドル（0.00024876*1*30）</td>
    </tr>
    <tr>
        <td>アプリケーション費用</td><td>5つの1C2GiB</td><td>10分（2022年8月1日の10:24:59～2022年8月1日の10:15:20=9分39秒 </td><td> 0.027622 ドル（5*（1*0.00031344+2*0.0001195）*10）</td>
    </tr>
</table>

この場合、このアカウントは0.0350848（0.0074628+0.027622）ドルを支払う必要があります。
