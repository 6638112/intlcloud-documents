# 1\. **背景**
本模块仅在您使用事件总线 EventBridge（简称“功能”）时适用。本模块包含在位于   [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据隐私和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致的范围，本模块应适用。

# 2\. **处理**
我们将因本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据**  （EventBusID、RuleID、规则、TargetID） | 我们仅为了按照您的特定配置向您提供本功能而处理此数据。 |
|**事件数据**   （您所提供的与规则错误或事件异常相关的业务数据）                                                              |我们仅为了向您提供本功能而处理此数据。请注意，这些数据集成并存储在我们的 TDMQ 功能中。                                                              |
# 3\. **服务区域**
见 DPSA 中的规定。

# 4\. **分包处理方**
见 DPSA 中的规定。

# 5\. **数据保留**
我们将存储因本功能处理的个人数据，具体如下所述：

| **个人信息**                                     | **保留政策**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据**|我们会一直保留此类数据，直至您将它们手动删除。此外，若您删除帐户或终止使用本功能，我们将在 15 天内删除此数据。 |
|**事件数据**|存储 15 天。 |

您可以根据 DPSA 请求删除此个人数据。

# 6\. **特殊条款**
您必须确保使用本功能的最终用户至少达到了个人可以同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
