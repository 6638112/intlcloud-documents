This document describes how to view, edit, delete, and send messages to an event bus.

## Prerequisites

You have [created an event bus](https://intl.cloud.tencent.com/document/product/1108/42285).

## Directions

### Viewing event bus

You can view the details of an event bus as follows:

1. Log in to the [EventBridge console](https://console.cloud.tencent.com/eb/).
2. In the drop-down list at the top of the page, select a region.
3. In the event bus list, click the target event bus.
4. The event bus page displays event bus information such as event bus name, ID, region, description, and configuration 

### Editing event bus

1. Log in to the [EventBridge console](https://console.cloud.tencent.com/eb/).
2. In the drop-down list at the top of the page, select a region.
3. In the event bus list, click the target event bus.
4. The event bus page displays the editable content such as event bus name and description 



### Deleting event bus

1. Log in to the [EventBridge console](https://console.cloud.tencent.com/eb/).
2. In the drop-down list at the top of the page, select a region.
3. In the event bus list, find the target event bus and click **Delete** in the **Operation** column on the right 
4. In the pop-up window, click **OK**.

     

### Sending message to event bus

1. Log in to the [EventBridge console](https://console.cloud.tencent.com/eb/).
2. In the drop-down list at the top of the page, select a region.
3. In the event list, select the event bus to which you want to send messages 
4. Click **OK**.
