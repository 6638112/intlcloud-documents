This document describes how to view, edit, and delete an event rule.


## Prerequisites

You have [created an event rule](https://intl.cloud.tencent.com/document/product/1108/42289).


## Directions

### Viewing event rule

You can view the details of an event bus as follows:

1. Log in to the EventBridge console and select **[Event Rule](https://console.cloud.tencent.com/eb/rule)** on the left sidebar.
2. In the drop-down lists at the top of the page, select a region and event bus.
3. In the event rule list, click the target event rule.
4. The event rule page displays the event rule information such as basic information, event match information, and event target 

### Editing event rule

1. Log in to the EventBridge console and select **[Event Rule](https://console.cloud.tencent.com/eb/rule)** on the left sidebar.
2. In the drop-down lists at the top of the page, select a region and event bus.
3. In the event rule list, click the target event rule.
4. On the event rule page, you can edit basic event information and event match information and delete/add event targets 

### Deleting event rule

1. Log in to the EventBridge console and select **[Event Rule](https://console.cloud.tencent.com/eb/rule)** on the left sidebar.
2. In the drop-down lists at the top of the page, select a region and event bus.
3. In the event rule list, find the target event rule and click **Delete** in the **Operation** column on the right 
4. In the pop-up window, click **OK**.
