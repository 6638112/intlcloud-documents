An event rule is used to filter and convert events as detailed below:


- The filter feature in an event rule is provided by the event pattern. For more information, please see [Event Pattern](https://intl.cloud.tencent.com/document/product/1108/42288).
- The trigger target feature in an event rule is provided by the event target.
- You can view, edit, and delete event rules. For more information, please see [Managing Event Rule](https://intl.cloud.tencent.com/document/product/1108/42290).
