A connector actively pulls events from event sources and pushes them to custom buses.



- EventBridge connectors support the following event sources:
  - TDMQ. For more information, please see [TDMQ Connector](https://intl.cloud.tencent.com/document/product/1108/42278).
  - API Gateway over HTTP. For more information, please see [API Gateway Connector](https://intl.cloud.tencent.com/document/product/1108/42279).
  - CKafka. For more information, please see [CKafka Connector](https://intl.cloud.tencent.com/document/product/1108/42280).
  - SaaS event delivery (provided by EIS). For more information, please see [SaaS Connector](https://intl.cloud.tencent.com/document/product/1108/42281).
- EventBridge provides the following connector management capabilities:
  - Creating connector
  - Viewing connector
  - Editing connector
  - Starting connector
  - Stopping connector
  - Deleting connector
