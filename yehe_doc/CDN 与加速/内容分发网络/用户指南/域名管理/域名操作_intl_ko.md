## 작업 시나리오
Tencent Cloud CDN 가속 서비스에 도메인을 액세스한 뒤, 액세스한 가속 도메인을 조회하거나 활성화/비활성화하려는 경우, <br/>[CDN 콘솔](https://console.cloud.tencent.com/cdn)에 로그인한 후 메뉴바에서 [도메인 관리]를 선택하여 관련 작업을 진행할 수 있습니다.

Tencent Cloud CDN 서비스는 사용자 정의 도메인 리스트 페이지를 지원합니다. 필요에 따라 여러 항목의 기본 설정 표시 및 정렬 변경이 가능하며, 도메인 가속 서비스의 일괄 활성화와 비활성화를 지원하므로 효율적으로 서비스를 관리할 수 있습니다.

## 리스트 작업 가이드
### 사용자 정의 리스트 표시
검색창 오른쪽의 [설정] 버튼을 클릭하여 리스트 설정 체크 박스를 엽니다.
![](https://main.qcloudimg.com/raw/790ae4b6c47b8b5ccd72e517701d34db.png)
특정 필드의 표시 여부를 선택할 수 있으며, 필드의 표시 순서를 변경할 수 있습니다.
![](https://main.qcloudimg.com/raw/1d123a3948ee2450204fa8cc78a11a36.png)

도메인 관리 리스트 페이지에서 아래와 같은 설정 필드를 사용자 정의로 설정할 수 있습니다.

+ 도메인: 필수, 표시 안 함으로 설정 불가.
+ 서비스 유형: 정적 가속, 다운로드 가속, 스트림 미디어 VOD 가속.
+ 상태: 도메인 서비스 상태로, 활성화, 비활성화, 배포 세 가지 상태가 있습니다.
+ CNAME: 도메인에 해당하는 CNAME.
+ 액세스 방식: 외부 원본 서버 액세스 또는 객체 스토리지 원본 서버(COS 원본) 액세스.
+ 서비스 리전: 중국 내, 중국 외, 글로벌.
+ 프로젝트: 도메인 프로젝트의 이름.
+ 메인 원본 서버 설정: 메인 원본 서버 주소.
+ 백업 원본 서버 설정: 백업 원본 서버 주소.
+ HTTPS 설정: 설정됨/미설정.
+ Origin-pull 프로토콜: HTTP 원본, HTTPS 원본 또는 프로토콜 따라가기
+ 호스트 헤더: 원본 도메인 설정.
+ 필터링 매개변수: 활성화 / 비활성화.

### 인벤토리 설정 내보내기

검색창 오른쪽의 [내보내기] 버튼을 클릭하면 도메인 리스트에 표시된 콘텐츠를 Excel 형식으로 내보낼 수 있습니다.

+ 내보내는 콘텐츠는 현재 도메인 리스트 페이지에 표시된 도메인 및 설정 프로젝트로, 풀네임이 아닌 도메인 및 그에 관한 모든 기본 설정 항목을 내보낼 수 있습니다.
+ 도메인은 매회 최대 3,000개까지 내보낼 수 있습니다.

![](https://main.qcloudimg.com/raw/8065bb602460bf3fcf9a8dbec9c3bd67.png)

## 도메인 작업 가이드
CDN 콘솔에서 CDN에 액세스한 가속 도메인의 가속 서비스에 대하여 활성화, 비활성화, 삭제와 더불어 서브 프로젝트 수정 등의 작업을 진행할 수 있습니다.
<span ID = "close"></span>

### 가속 서비스 비활성화

가속 서비스를 비활성화한 후에는 전체 네트워크 CDN 가속 노드에 있는 도메인 관련 설정의 운영이 정지됩니다. 이 상태에서 해당 도메인 액세스가 여전히 CDN 노드에 도달할 경우, 404를 리턴하며 서비스가 정상적으로 이루어지지 않습니다. 따라서 도메인을 비활성화하기 전에, 해당 도메인의 리졸브가 Tencent Cloud 이외의 다른 CDN이 할당한 CNAME 주소로 설정되었는지 확인해야 합니다.

>
> - 활성화 상태인 가속 도메인만 비활성화할 수 있으며, 배포 중인 도메인의 비활성화 기능은 현재 지원하지 않습니다.
> - 도메인을 비활성화해도 도메인 설정은 저장되며, 다음 번 활성화 시 바로 적용됩니다.
> - 도메인 가속 서비스가 완전히 비활성화된 후에는 어떠한 소모도 발생하지 않습니다.

도메인 오른쪽 작업바에서 [비활성화] 버튼을 클릭하여 직접 서비스를 비활성화할 수 있으며, [활성화] 상태인 도메인을 일괄 선택하여 상단의 [추가 관리]에서 일괄 비활성화를 진행할 수도 있습니다.

![](https://main.qcloudimg.com/raw/6536bf6c02870b792031a2c65645c31a.png)
<span ID = "open"></span>

### 가속 서비스 활성화

도메인 상태가 [비활성화]인 경우, 가속 서비스 활성화를 통해 도메인 설정을 전체 네트워크 가속 노드로 다시 전달할 수 있습니다.

도메인 오른쪽 작업창에서 [활성화] 버튼을 클릭하여 직접 서비스를 활성화할 수 있으며, [비활성화] 상태인 도메인을 일괄 선택하여 상단의 [추가 관리]에서 일괄 활성화를 진행할 수도 있습니다.

>
>활성화 상태인 도메인이 3개월 동안 어떠한 작업도 없거나 소모가 발생하지 않은 경우, 비활성 도메인으로 판단되어 Tencent Cloud CDN 시스템이 해당 가속 서비스를 자동으로 비활성화합니다.

![](https://main.qcloudimg.com/raw/fd6d029c82e516965ea6e093ef41c9f5.png)

### 가속 도메인 삭제

도메인 상태가 [비활성화]인 경우, 오른쪽의 추가 관리를 클릭하여 도메인을 삭제할 수 있습니다. 삭제한 후에는 도메인의 설정이 삭제되고 복구할 수 없으므로, 관련 통계 데이터를 더 이상 조회할 수 없으니 신중히 작업하시기 바랍니다.
![](https://main.qcloudimg.com/raw/bc58fc9c225efb79a0ec1454b6748e1c.png)
