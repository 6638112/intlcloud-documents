## 大量のリソース備蓄
### 中国本土のノード分布
Tencent Cloud CDNには、中国本土に1,100以上のアクセラレーションノードがあり、合計帯域幅は100 Tbpsを超えます。中国本土のアクセラレーションノードはすべてTencentのハイスペック、高セキュリティの自社構築データセンターであり、高品質のキャリアネットワークをご利用いただけます。ノードは中国本土のすべての省、自治区、および直轄市をカバーします。また、3つの主要なキャリアと50社以上お中小規模のキャリアのアクセスを推進することにより、4つの主要な中央ノードを構築して、中小規模のキャリアのアクセラレーション効果が大幅に向上させました。

## グローバルインテリジェントスケジューリング

ユーザーがリソースにアクセスする際に、キャリアネットワーク、クライアントの所在地、オリジンサーバーIDCキャリアのネットワーク帯域幅などのさまざまな要因が、応答時間とユーザーエクスペリエンスに影響を与える可能性があります。

Tencent Cloud CDNは、ネットワーク全体のリンクをリアルタイムで監視し、自社開発のGSLBスケジューリングシステムとインテリジェントルーティングテクノロジーを組み合わせて、ユーザーのアクセスリクエストを最適なエッジノードにスケジューリングして加速します。これにより、ユーザーが必要なリソースを迅速かつ安定的に取得できるようにします。

## クイック設定
Tencent Cloud CDNを使用してサービスを加速します。自社のサービスコンテンツを調整・変更する必要がなく、簡単かつ迅速にアクセスできます。

Tencent Cloudアカウントを登録して本人確認を完了すると、前払いなしでワンクリックでCDNサービスをアクティブ化できます。ビジネスドメイン名をCDNコンソールに追加するだけで、約5分待てば、ドメイン名関連設定がネットワーク全体のアクセラレーションノードに配信されます。上記の手順ではアクセラレーションサービスがまだ有効化されていませんので、現在のビジネスに影響しません。

アクセラレーションサービスを有効にする場合は、ドメイン名サービスプロバイダーで対応するCNAME解決設定を変更し、DNSが有効になると、アクセラレーションサービスも有効になります。

## 豊富な機能

CDNは、豊富で便利な操作コンソールを提供しています。必要に応じて、さまざまな設定の変更、データの表示などを行うことができます。

**ドメイン名管理**
+ ドメイン名の追加、削除、アクティブ化および非アクティブ化をサポートします。
+ アクセラレーションリージョンの切り替えをサポートします。アクセラレーション範囲として「中国本土」、「中国本土以外」、または「グローバル」を選択できます。
+ ドメイン名リスト表示ページをカスタマイズ設定可能で、さまざまな設定項目の表示、フィルター、およびクエリーをサポートします。

**ドメイン名の設定**
+ 外部オリジンサーバー（IPリスト、ドメイン名）を設定するか、COSをオリジンサーバーとして使用できます。ラウンドロビンback-to-origin、ホットバックアップオリジンサーバー、重みback-to-originをサポートします。
+ Refererブラックリスト/ホワイトリスト、IPブラックリスト/ホワイトリスト、タイムスタンプのリンク不正アクセス防止、IPアクセス頻度制限などのカスタムアクセスコントロールポリシーを設定できます。
+ ノードキャッシュ、ステータスコードキャッシュ、およびHTTPヘッダーキャッシュの有効期限をカスタマイズできます。
+ グローバルback-to-originリンクの最適化、Range back-to-originの最適化、および301/302 back-to-origin followの最適化設定をサポートします。
+ HTTPSアクセラレーション、HTTP 2.0アクセラレーション、およびリクエスト強制リダイレクトをサポートします。
+ 帯域幅上限の設定、HTTP Response Headerのカスタマイズ設定、インテリジェントな圧縮、SEO検索エンジン最適化などの高度な設定をサポートします。

**キャッシュ更新**
+ ネットワーク全体でのキャッシュ内容をセルフ更新可能です。ディレクトリ更新とURL更新をサポートします。

**リアルタイム監視**
+ CDNサービスは、1分間粒度の帯域幅、トラフィック、トラフィックヒット率、リクエスト数、およびアクセスリクエストによって生成されたステータスコードのリアルタイム監視をサポートします。プロジェクト、ドメイン名、省、キャリア、プロトコルフィルタリングをサポートし、包括的にサービス状態を表示します。
+ 1分間粒度のback-to-origin帯域幅、トラフィック、リクエスト数、失敗率、back-to-originリクエストによって生成されたステータスコードのリアルタイム監視をサポートします。プロジェクトとドメイン名でのフィルタリングをサポートし、ユーザーのオリジンサーバー状態監視を支援します。
+ 世界中のユーザー地域分布レポートのリアルタイム表示および中国本土のキャリア分布レポートのリアルタイム表示をサポートします。
+ 日次レポート、週次レポート、月次レポートなど複数の期間別の運用レポートを提供し、ビジネスの変動に関する最新情報を提供します。

**ログサービス**
+ アクセスリクエストによって生成されたすべてのログは1時間ごとにグループ化され、ダウンロードできます。
+ CDNアクセスログのリアルタイム収集とプッシュをサポートし、ログデータの高速な検索と分析を可能にします。

**豊富なAPIインターフェース**
CDNサービスは、上記のサポートされているすべての機能のAPIを提供し、お客様のメンテナンス担当者がお客様のビジネスを管理、監視、データ表示、分析をカスタマイズする必要がある場合は、直接[使用へ](https://intl.cloud.tencent.com/document/product/228/31719)に進んでください。
