## グローバルインテリジェントスケジューリング

ユーザーがリソースにアクセスする際に、キャリアネットワーク、クライアントの所在地、IDCオリジンサーバーのネットワーク帯域幅などのさまざまな要因が、応答時間とユーザーエクスペリエンスに影響を与える可能性があります。

Tencent Cloud CDNは、ネットワーク全体リンクをリアルタイムで監視し、自社開発のGSLBスケジューリングシステムとインテリジェントルーティング技術を組み合わせて、ユーザーのアクセスリクエストを最適なエッジアクセラレーションノードにスケジューリングしてサービスを提供し、ユーザーが必要なリソースを迅速かつ安定的に取得できるようにします。

## クイック設定
Tencent Cloud CDNを使用してサービスを高速化し、サービスコンテンツを調整・変更する必要がなく、アクセスプロセスがシンプルで迅速になります。　

Tencent Cloudアカウントを登録して本人確認を完了すると、前払いなしでワンクリックでCDNサービスをアクティブ化できます。ビジネスドメイン名をCDNコンソールに追加するだけで、約5分待てば、ドメイン名関連設定がネットワーク全体のアクセラレーションノードに配信されます。上記の手順ではアクセラレーションサービスがまだ有効化されていませんので、現在のビジネスに影響しません。

アクセラレーションサービスを有効にする場合は、ドメイン名サービスプロバイダーで対応するCNAME解決設定を変更する必要があります。DNSが有効になると、アクセラレーションサービスも有効になります。

## 豊富な機能

Tencent Cloud CDNは、豊富で便利な操作コンソールを提供しています。必要に応じて、さまざまな設定の変更、データの表示などを行うことができます。

**ドメイン名管理**
+ ドメイン名の追加、削除、アクティブ化および非アクティブ化をサポートします。
+ アクセラレーションリージョンの切り替えをサポートします。アクセラレーション範囲として「中国本土」、「中国本土以外」、または「グローバル」を選択できます。
+ ドメイン名リスト表示ページをカスタマイズ可能で、さまざまな設定項目の表示、フィルター、およびクエリーをサポートします。

**ドメイン名の設定**
+ 外部オリジンサーバー（IPリスト、ドメイン名）を設定するか、COSをオリジンサーバーとして使用できます。ラウンドロビンback-to-origin、ホットバックアップオリジンサーバー、重みback-to-originをサポートします。
+ Refererブラックリスト/ホワイトリスト、IPブラックリスト/ホワイトリスト、タイムスタンプのリンク不正アクセス防止、IPアクセス頻度制限などのカスタムアクセス制御ポリシーを設定できます。
+ ノードキャッシュ、ステータスコードキャッシュ、およびHTTPヘッダーキャッシュの有効期限をカスタマイズできます。
+ 国境を越えたback-to-originリンクの最適化、Range back-to-originの最適化、および301/302 back-to-originフォローの最適化設定をサポートします。
+ HTTPSアクセラレーション、HTTP 2.0アクセラレーション、およびリクエスト強制リダイレクトをサポートします。
+ ネットワーク帯域幅の制限の設定、HTTP応答ヘッダーのカスタマイズ設定、インテリジェントな圧縮、SEO検索エンジン最適化などの高度な設定をサポートします。

**キャッシュ更新**
+ キャッシュ更新、ディレクトリ更新、およびURL更新をサポートします。

**リアルタイム監視**
+ CDNサービスは、1分間粒度の帯域幅、トラフィック、トラフィックヒット率、リクエスト数、およびアクセスリクエストによって生成されたステータスコードのリアルタイム監視をサポートします。プロジェクト、ドメイン名、省、キャリア、プロトコルフィルタリングをサポートし、サービス状態の包括的なビューを提供します。
+ CDNサービスは、1分間粒度のback-to-origin帯域幅、トラフィック、リクエスト数、失敗率、back-to-originリクエストによって生成されたステータスコードのリアルタイム監視をサポートします。プロジェクトとドメイン名でフィルタリングをサポートし、ユーザーのオリジンサーバー状態監視を支援します。
+ CDNサービスは、世界中のユーザー地域分布レポートのリアルタイム表示および中国本土のキャリア分布レポートのリアルタイム表示をサポートします。
+ CDNサービスは、日次レポート、週次レポート、月次レポートなどの運用レポートを提供し、ビジネスの変動に関する最新情報を提供します。

**ログサービス**
+ アクセスリクエストによって生成されたすべてのログをダウンロードでき、1時間ごとにパッケージングされます。
+ CDNアクセスログのリアルタイム収集とプッシュをサポートし、ログデータの高速な検索と分析を可能にします。

**CDN API**
CDNサービスは、上記のサポートされているすべての機能のAPIを提供し、お客様のメンテナンス担当者がお客様のビジネスを管理、監視、データ表示、分析をカスタマイズする必要がある場合は、直接[使用へ](https://intl.cloud.tencent.com/document/product/228/31719)進んでください。
