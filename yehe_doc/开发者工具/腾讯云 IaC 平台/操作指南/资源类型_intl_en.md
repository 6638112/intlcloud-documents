The **Resource Types** page provides references to help you use Tencent Cloud resources. This document describes how to view resource information.


## Directions


1. Log in to the [TIC console](https://console.cloud.tencent.com/tic).
2. In the left sidebar, choose **Orchestration** -> **Resource Types** to go to the **Resource Types** page.
3. Find a resource type, and click **Details** to view the parameter information, use cases, and references, as shown in the following figure.
![](https://main.qcloudimg.com/raw/ac5427794d45a232da858892b252aa7e.png)
