本章节主要介绍消息队列CKafka专业版的计费项目，计费模式和实例价格。

## 计费项目

计费项目如下：


| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 峰值带宽  | 40MB吞吐指出和入的带宽峰值为40MB。考虑实例的副本个数，需要均分吞吐量。例如，客户要求40MB吞吐、3副本，则需要购买120MB/s的峰值带宽。 |
| 磁盘容量  | <li>不同实例规格对应不同的磁盘容量。</li><li> 消息队列CKafka专业版支持的磁盘类型为SSD和普通云磁盘。专业版计划支持高性能云硬盘（计划2021年4月支持）</li> |
| Partition | <li>不同实例规格对应不同的Partition数量，专业版的Partition可以单独额外购买。 </li><li>专业版Topic不再额外计费，最高可使用的Topic数量 = Partition数量/副本数</li><li>不支持缩减Partition数量。</li><li>实例级别的 Paritition 限制包含了副本数。例如：一个实例下有1个双副本、4分区的 Topic、 2个3副本、3分区的 Topic，则该实例的总 partition 个数为 （1 × 2 × 4）+（2 × 3 × 3）= 26个。</li> |

## 计费模式

消息队列 CKafka 以实例的形式售卖，采用**包年包月-预付费**的计费形式。

选购专业版实例时，请您做好对业务峰值带宽、Partition数和磁盘容量的估算，最终购买的专业版实例的总费用 = (基础套餐包费用 + Partition 包单价 × 额外 Partition 数量 / 100 + 磁盘容量单价 × 磁盘容量 / 100) × 月数

> ?包年包月暂不支持提前进行缩容，您可以先 [提交工单](https://console.cloud.tencent.com/workorder/category?level1_id=876&level2_id=951&source=0&data_title=消息服务 CKafKa&step=1) 申请缩容，等待到下一计费周期开始前，后台会为您执行缩容，执行后新的一个月/年会按照缩容后的规格计算。

## 专业版价格

CKafka 专业版提供了更为灵活的参数选配方式和更稳定的升降配能力，您可以根据具体的业务现状按需选购以及扩容。

您在选择峰值带宽大小后，会自动对应到一个基础套餐包的阶梯上，每个阶梯的基础套餐包含有一定数量的Topic 规格、Partition 和磁盘容量，最高可使用的Topic数量 = Partition数量/副本数。

| x = 40               | 1000                | 2000                    | 2400                    | 100                | 5000               |
| x = 120              | 1000                | 2000                    | 2800                    | 200                | 10000              |

峰值带宽基础包的价格按照不同阶梯进行计费，价格阶梯如下：

| --------- | -------------------- | ------------------------------------------- |
| 基础I型   | 40~100               | 310美元/月起步，每+20MB/s 增加 47美元/月    |
| 基础II型  | 120~300              | 522美元/月起步，每+20MB/s 增加 45美元/月    |
| 基础III型 | 320~600              | 948美元/月起步，每+20MB/s 增加 44美元/月    |

| 高性能型  | 1600~3200            | 3175美元/月起步，每+400MB/s 增加 780美元/月 |

例：一个180MB/s的实例，归属于基础II型，在120MB/s的基础上增加了60MB/s，则该套餐包的价格为 522 + 45 × ( 60 / 20 ) = 657 (美元/月)

**Partition 叠加包价格**

您在选择峰值带宽大小后，若所对应的基础套餐包内的 Partition 规格仍不满足需求，则可以额外单独购买 Partition 叠加包。Partition 叠加包按照100个分区为1个单位叠加，Partition 叠加包价格如下：

| 阶梯     | 单位Partition数（个） | Partition包单价（包年包月） |
| -------- | --------------------- | --------------------------- |
| 任何阶梯 | 100                   | 16美元/月                   |

| -------- | -------------- | ------------------ | -------------------- |
| 任何阶梯 | SSD 高性能云盘 | 100                | 16美元/月            |
| 任何阶梯 | 普通云硬盘     | 100                | 8美元/月             |
