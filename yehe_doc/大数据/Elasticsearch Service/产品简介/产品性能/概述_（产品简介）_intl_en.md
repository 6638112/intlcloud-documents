This section describes the results of stress testing on ES instances (v7.10.1) of different specifications through the benchmark rally script officially provided by Elasticsearch.

This section provides the stress test results of ES clusters with the following specifications:
- [4-Core 16 GB 3-Node Cluster Performance Test](https://intl.cloud.tencent.com/document/product/845/40290)
- [8-Core 32 GB 3-Node Cluster Performance Test](https://intl.cloud.tencent.com/document/product/845/40291)

It also provides the comparison of stress test results of 4-core 16 GB and 8-core 32 GB ES clusters. For more information, please see [Stress Test Result Comparison Between 4-Core 16 GB 3-Node Cluster and 8-Core 32 GB 3-Node Cluster](https://intl.cloud.tencent.com/document/product/845/40292).
