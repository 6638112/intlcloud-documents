## 1.	はじめに

本モジュールは、お客様がクラウド データ ウェアハウス（「**機能**」）を使用する場合に適応されます。このモジュールは、 https://intl.cloud.tencent.com/document/product/301/17345 にあるプライバシーポリシー（「**プライバシーポリシー**」）に組み込まれています。本モジュールで使用されてはいても定義されていない用語は、プライバシーポリシーで与えられた意味を持つものとします。プライバシーポリシーと本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## 2.	管理者

本モジュールに記載されている個人情報の管理者は、プライバシーポリシーに規定されているとおりです。

## 3.	可用性

本機能は世界中のユーザーが利用可能です。

## 4.	当社による個人情報の使用方法

当社は、以下の法的根拠に従い、以下の方法で個人情報を使用します。

| **個人情報**                                     | **用途**                                                      | **法的 根拠**                                             |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **ログ データの 実行**: ハードウェア、ソフトウェア、および システムの問題の詳細 | この情報を使用して、 問題を特定します。 このデータは、当社の Cloud Log Service (CLS) および TencentDB for MySQL 機能に保存され バックアップされますのでご注意ください。 | これらの情報は、お客様との契約を履行し、 本機能を提供するために必要な情報として処理されます。 |
| **データベース  アカウント データ**: データベースのユーザー名とパスワード   | この情報は、認証とデータベースへのアクセスの提供、 およびこの機能の提供に使用されます。 | これらの情報は、お客様との契約を履行し、 本機能を提供するために必要な情報として処理されます。 |



## 5.	当社による個人情報の保存および共有方法

プライバシーポリシーに規定されている通り。

## 6.	データ保持

当社は、以下の条件に従って個人情報を保持します。

| **個人 情報** | **保持ポリシー**                                         |
| ------------------------- | ------------------------------------------------------------ |
| 実行ログデータ          | そのようなデータは30日間保持されます。                             |
| データベースアカウントデータ     | お客様が本機能を使用する限り、 当社は当該データを保持します。このようなデータを手動で削除できます。本機能のご使用が終了すると、 7日後にこのデータが削除されます。 |
