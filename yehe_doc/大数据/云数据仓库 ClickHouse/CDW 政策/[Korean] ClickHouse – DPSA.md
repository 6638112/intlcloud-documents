## 1.	배경

이 모듈은 사용자가 클라우드 데이터 웨어하우스(**"기능"**)를 사용하는 경우에 적용됩니다. 이 모듈은 https://intl.cloud.tencent.com/document/product/301/17347 에 있는 데이터 개인 정보 보호 및 보안 계약(**“DPSA”**)에 통합되어 있습니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 의미를 갖습니다. DPSA와 이 모듈 간에 충돌이 발생할 경우, 이 모듈은 상충하는 범위까지만 적용됩니다.

## 2.	처리

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **이용**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **구성 데이터:** 클러스터  구성 정보(청구 모드, 지역, 가용성 영역,  클러스터 이름, 클러스터 설명, CVM의 CPU 수, 가상 사설 클라우드 (VPC) 저장소 데이터( 사용자가 결정한 스토리지 할당 관련 데이터 포함), 자동 갱신, 구매 시간 (주문 생성 시) 포함) | 당사는 사용자의  고유한 구성에 따라 사용자에 대한  기능 제공 목적으로만 이 데이터를 처리합니다.  이 데이터는 TencentDB for MySQL 기능에  저장된다는 점에 유의하십시오. |
| **데이터 모니터링:** CPU 및 메모리  사용률, 초당 열린 파일 수, 데이터 디스크 사용, 주요 지 표(초당 처리 쿼리 수, 처리 시간,  초당 연결 수,  초당 클러스터별 레코드 출력 수, 운영자의 총 계산 시간 포함) | 당사는 이 정보를 활용하여 작동 상태를 모니터링하고 성능 또는 소비  지표와 같은 기능의 다양한 이벤트에 대한  임계값 경보를 설정하도록 지원합니다.  이 데이터는 클라우드 모니터(CM) 기능에서  수행 및 저장되어  TencentDB for MySQL 기능에 저장된다는 점에 유의하십시오. |
| **비즈니스 데이터:** CBS 또는  이 기능에서 가져오거나 저장하는 비즈니스 및  구성 데이터의 이진 데이터 파일 | 당사는 사용자에게 기능을 제공함으로써 사용자 쿼리를 실행하는  목적으로만 이 데이터를 처리합니다.  이 데이터는 이러한 목적으로  CBS 기능에 통합되어 있다는 점에 유의하십시오. 계층화된 저장소 기능을  사용하도록 선택한 경우, 이 데이터는 COS 기능에도 저장됩니다. |

## 5.	데이터 보존

당사는 다음과 같이 기능과 관련하여 처리된 개인 데이터를 보관합니다.

| **개인 정보** | **보존 정책**                                         |
| ------------------------ | ------------------------------------------------------------ |
| 구성 데이터       | 당사는 귀하가 해당 기능을 사용하는 동안  이러한 데이터를 보존합니다. 당사는 귀하의 기능 사용이 종료된 시점으로부터  7일 후에 이 데이터를 삭제합니다. |
| 모니터링 데이터          | 당사는 귀하가 해당 기능을 사용하는 동안  이러한 데이터를 보존합니다. 당사는 귀하의 기능 사용이 종료된 시점으로부터  7일 후에 이 데이터를 삭제합니다. |
| 비즈니스 데이터            | 당사는 귀하가 해당 기능을 사용하는 동안  이러한 데이터를 보존합니다. 귀하는  기능에서 이러한 데이터를 수동으로 삭제할 수 있습니다. 이러한 데이터를 삭제하거나  기능 사용을 종료하도록 선택하면 데이터는 7일 후에 기능에서 삭제됩니다. |

DPSA에 따라 이러한 개인 데이터의 삭제를 요청할 수 있습니다.

## 6.	특수 조건

개인 데이터 처리에 동의할 수 있는 최소 연령을 만족하는 최종 사용자만 이 기능을 사용할 수 있습니다. 이는 최종 사용자의 해당 지역에 따라 다를 수 있습니다.
