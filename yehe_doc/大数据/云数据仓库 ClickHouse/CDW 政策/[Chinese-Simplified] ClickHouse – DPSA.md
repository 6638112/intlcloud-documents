## 1.	背景

本模块仅在您使用云数据仓库（简称“**功能**”）时适用。本模块已纳入 https://intl.cloud.tencent.com/document/product/301/17347 的数据隐私和安全协议（简称“**DPSA**”）中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。 

## 2.	处理

我们将因本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据：** 集群 配置信息（包括计费模式、区域、可用区、 集群名称、集群描述、CVM CPU 数量、私有网络  (VPC)、存储数据（包括由您 决定的存储分配数据）、自动续订、购买时间 （即将生成订单的时间）） | 我们仅为了 按照您的特定配置向您提供本功能 而处理此数据。 请注意， 此数据将存储在腾讯云数据库 MySQL 功能中。 |
| **监控数据：** CPU 及内存 利用率、每秒打开的文件数、数据磁盘使用率、关键指标 （包括每秒处理的查询数、处理时间、 每秒连接数、集群每秒 输出的记录数、操作员的计算时间总数） | 我们使用此信息帮助您监控操作状态， 为本功能中的各类事件设置阈值警报，例如性能或消耗 指标。 请注意， 此数据由我们的云监控 (CM) 功能执行和存储，同时也存储在 我们的腾讯云数据库 MySQL 功能中。 |
| **业务数据：** 您在  CBS 和/或本功能中导入/存储的业务数据和配置数据的 二进制数据文件 | 仅在向您提供功能时， 我们才会处理此数据供您执行查询。 请注意， 此数据与我们的 CBS 功能完美兼容。如果您选择 使用分层存储功能，则此数据也会存储在 COS 功能中。 |

## 5.	数据保留

我们将存储因本功能处理的个人数据，具体如下所述：

| **个人信息** | **保留政策**                                         |
| ------------------------ | ------------------------------------------------------------ |
| 配置数据       | 我们会在您 使用本功能期间保留此类数据。若您终止使用本功能， 我们将在 7 天后删除此数据。 |
| 监控数据          | 我们会在您使用 本功能期间保留此类数据。若您终止使用本功能， 我们将在 7 天后删除此数据。 |
| 业务数据            | 我们会在您使用 本功能期间保留此类数据。您可以手动删除 本功能上的此类数据。在您选择删除此类数据或终止使用 本功能后，我们将在 7 天后删除此数据。 |

您可以根据 DPSA 请求删除此类个人数据。

## 6.	特殊条款

您必须确保使用本功能的最终用户至少达到个人可同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
