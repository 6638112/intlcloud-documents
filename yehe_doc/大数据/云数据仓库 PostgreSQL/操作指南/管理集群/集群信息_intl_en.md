This document describes how to modify the basic information and billing mode of a CDWPG cluster in the console, as well as view its performance monitoring data, real-time queries, historical queries, and event monitoring data.

## Basic Information
On the cluster list page, enter the **Basic Configuration** tab of a cluster, where you can set the cluster name, network, and admin user/password.
>!Changing the network address will change the cluster access connection, so you need to change the call address simultaneously.

![](https://qcloudimg.tencent-cloud.cn/raw/29b4d6f87a1d9e1b9b478110218ee0bc.png)

## Billing Information
On the **Basic Configuration** tab, you can switch from pay-as-you-go to monthly subscription billing mode for the cluster and enable **Auto-Renewal** as needed.


## Performance Monitoring
On the **Performance Monitoring** tab, you can view various the metrics in node and cluster dimensions.


## Real-Time Query
On the **Real-Time Query** tab, you can view ongoing queries, including running, blocked, and waiting queries.
![](https://qcloudimg.tencent-cloud.cn/raw/69fa63c7c0ef014234590e87f3f5af15.png)
You can click **Terminate Query** to terminate a query in the list.

## Historical Query
On the **Historical Query** tab, you can view queries that have been completed or terminated.
![](https://qcloudimg.tencent-cloud.cn/raw/d9f5a3530014d3104afff686b548e8b8.png)

## Event Monitoring
On the **Event Monitoring** tab, you can query sensitive operation events in the cluster.
![](https://qcloudimg.tencent-cloud.cn/raw/908b9c227253ec4308af645de125fe6d.png)
