

| Parameter            | Feature                         | SQL Statement                 |
| --------------- | ---------------------------- | ------------------------ |
| CURRENT_CATALOG | Current database                   | SELECT CURRENT_CATALOG;  |
| CURRENT_ROLE    | Current user                     | SELECT    CURRENT_ROLE;  |
| CURRENT_SCHEMA  | Current database schema               | SELECT   CURRENT_SCHEMA; |
| CURRENT_USER    | Current user                     | SELECT   CURRENT_USER;   |
| LOCALTIMESTAMP  | Current time                     | SELECT   LOCALTIMESTAMP; |
| SESSION_USER    | Current system user                 | SELECT   SESSION_USER;   |
| USER            | Current user, same as `CURRENT_USER` | SELECT   USER;           |
