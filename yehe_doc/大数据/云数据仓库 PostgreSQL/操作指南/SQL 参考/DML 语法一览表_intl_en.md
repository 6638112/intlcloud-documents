Data Manipulation Language (DML) is used to perform operations on data in database tables, such as inserting, updating, querying, or deleting data.

## Inserting Data
Insert one or more rows of data into a table.

| Feature         | SQL Statement       |
| -------- | ------- |
| Insert data | INSERT  |

## Altering Data
Alter one or more rows of data in a table.

| Feature         | SQL Statement       |
| -------- | ------- |
| Alter data | UPDATE  |

## Querying Data
Query the data in a data table that meets the specified conditions.

| Feature         | SQL Statement       |
| -------- | ------- |
| Query data | SELECT  |

## Deleting Data
Delete one or more specified rows of data or empty a table.

| Feature         | SQL Statement       |
| -------- | -------- |
| Delete data | DELECT   |
| Clear data | TRUNCATE |

## Copying Data
Copy data between a file and a table.

| Feature         | SQL Statement       |
| -------- | ------- |
| Copy data | COPY    |
