Data Lake Compute provides a script file management feature to facilitate and accelerate repeated query tasks.
>? Up to 100 SQL scripts can be saved in the console.

## Creating a script
1. Log in to the [Data Lake Compute console](https://console.cloud.tencent.com/dlc), go to the **Query analysis** page, and click **Resource package**.
2. On the **Resource package** tab, hover over the **Script file** row and click ![](https://qcloudimg.tencent-cloud.cn/raw/fe05e578c45f627c65b9bd52f07ed2f7.png) > **Create script**.
![]()
3. Enter and save the script content.
![]()
	- Script name: It can contain up to 25 letters and underscores.
	- Script description: It can contain up to 2,048 characters.
	- SQL statement: It must be a standard SQL statement containing up to 1,000 characters within 2 MB in size.

## Running a script
1. Log in to the [Data Lake Compute console](https://console.cloud.tencent.com/dlc), go to the **Query analysis** page, and click **Resource package**.
2. On the **Resource package** tab, hover over the target script name and click ![](https://qcloudimg.tencent-cloud.cn/raw/219b0e200a9faea9ef58f527c13df187.png) > **Copy script to SQL**.
3. Select the compute engine and click **Run**.
![]()

## Viewing script information
1. Log in to the [Data Lake Compute console](https://console.cloud.tencent.com/dlc), go to the **Query analysis** page, and click **Resource package**.
2. On the **Resource package** tab, hover over the target script name and click ![](https://qcloudimg.tencent-cloud.cn/raw/bd16565db20713ef2ea0a452f6a16993.png) > **Learn more** to view the script details.
![]()
![]()

## Deleting a script
1. Log in to the [Data Lake Compute console](https://console.cloud.tencent.com/dlc), go to the **Query analysis** page, and click **Resource package**.
2. On the **Resource package** tab, hover over the target script name and click ![](https://qcloudimg.tencent-cloud.cn/raw/3c663d960f6bacbd4a2566d84f7688fe.png) > **Delete** to delete the script.
![]()

>! Note that a deleted script cannot be recovered. Proceed with caution.
