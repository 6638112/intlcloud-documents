登录腾讯云 [对象存储](https://console.cloud.tencent.com/cos5) 控制台，选择**存储桶列表**。
- 若没有存储桶可自行创建，选择**创建存储桶**。
![](https://main.qcloudimg.com/raw/fa538c7bbc01aad17564958354f9cc6f.png)

- 若已有对应存储桶，可选择对应桶，并查看对应存储桶的访问域名。
![](https://qcloudimg.tencent-cloud.cn/raw/1a6ce278db7cdd3d9be4a18b8f436cff.png)

- 查询密钥，单击左侧菜单栏**密钥管理**。
![](https://qcloudimg.tencent-cloud.cn/raw/52a4b66c89feea0f0af2e66d85a10315.png)

- 单击**云 API 密钥**。
![](https://qcloudimg.tencent-cloud.cn/raw/6934fbd79f696cf96d11280096c9ae1e.png)
