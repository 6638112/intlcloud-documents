Tencent Cloud EMR Console has a "task center" where you can view the execution status of all tasks in your clusters. 

## Task List
You can select the target cluster on the topbar to view the status, start time, end time, and operation information of all tasks in it.
![](https://main.qcloudimg.com/raw/a896a14beb35cbabd478553e133d03d3.png)

## Task Details
Click **Task Details** to view task parameters and completion status of execution steps.
 ![](https://main.qcloudimg.com/raw/39c33b808a5d10160196fb3d3f631171.png)

## Operation Details
In some operation steps in **Task Details**, you can click **Run Details** to view the task completion status on the corresponding nodes.
 ![](https://main.qcloudimg.com/raw/ff64a4147a79b5779b250d80d5f16a81.png)
