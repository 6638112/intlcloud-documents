## Overview
Client management provides the information of the client deployed in the component, including the configuration status, configuration group, node type, and node IP.
## Directions
1. Log in to the [EMR console](https://console.cloud.tencent.com/emr) and click the **ID/Name** of the target cluster in the cluster list to enter the cluster details page.
2. On the cluster details page, select **Cluster Service** and click **Operation** > **Client Management** in the top-right corner of the target component block to enter the **Client Management** page.
