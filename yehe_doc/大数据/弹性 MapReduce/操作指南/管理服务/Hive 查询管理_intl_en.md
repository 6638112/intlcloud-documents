## Feature overview
The execution status of a submitted Hive query can be quickly viewed through **Query Management**. The query list displays the execution status and other information of the query and helps you quickly associate it with the executed job.

## Directions
1. Log in to the [EMR console](https://console.cloud.tencent.com/emr) and click the **ID/Name** of the target cluster in the **Cluster list** to enter the cluster details page.
2. On the cluster details page, click **Cluster Service** and select the Hive service name or **Operation** > **Query Management** in the top-right corner of the Hive service panel to view the **Job Query**, **Query Statement**, **Query Plan**, **Execution Overview**, and **Profile**.
![](https://qcloudimg.tencent-cloud.cn/raw/8af36136fb35c79c44930161dda73372.png)
3. Click the application ID to enter the **Job Query** tab in YARN, and select the YARN job set associated with the Hive query.
>! The Hive query management feature is supported for clusters purchased after June 8, 2022 with Hive selected. To use this feature for older clusters with the Hive component deployed, submit a ticket for application.
