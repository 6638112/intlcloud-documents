## Feature Overview
You can quickly view the query statement, query status, user, database, query type, start time, end time, and other detailed metrics of Impala jobs.

## Directions

1. Log in to the [EMR console](https://console.cloud.tencent.com/emr) and click a cluster **ID/name** on the **Cluster List** page to enter the cluster details page.
2. On the cluster details page, click **Cluster Services** and select **Operation** > **Query Management** in the top-right corner of the Impala component block to query jobs.
