## Feature Overview
You can find the alarms for status changes in the cluster (e.g., a process was restarted) in the EMR Console.

## Directions
1. 	Log in to the [EMR Console](https://console.cloud.tencent.com/emr), select the target cluster in **Cluster List**, and click **Details** to enter the cluster details page.
2. 	On the cluster details page, select **Cluster Monitoring** > **Service Alarms** to view the cluster service alarms.
