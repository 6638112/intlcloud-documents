## 邮件发送相关问题
#### 邮件发送支持哪几种发信方式？
邮件发送服务支持两种发信方式：
-   通过邮件发送控制台发送邮件，支持发送普通和模板邮件。具体操作方法，请参见  [通过控制台发送邮件](https://intl.cloud.tencent.com/document/product/1070/38226)。
-   通过 API 接口发送邮件。具体操作方法，请参见API 文档。

#### 邮件中是否能够添加附件？
腾讯云 DMS 目前不支持添加附件，如果您有添加附件的需求，请[提交工单](https://console.cloud.tencent.com/workorder)。
