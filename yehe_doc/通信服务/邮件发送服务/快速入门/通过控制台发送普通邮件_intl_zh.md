## 通过控制台发送普通邮件
使用腾讯云 DMS 发送电子邮件的最简单方法是使用控制台。由于控制台要求您手动输入信息，因此您通常仅使用它来发送测试电子邮件。在正式开始使用之后，您很可能会使用 API 发送电子邮件，但控制台对管理和监控您的发送活动很有用。

在完成发件前的准备工作后，您可以按照以下步骤发送您的第一封测试邮件：
1. 登录[腾讯云 DMS 控制台](https://console.cloud.tencent.com/dms)。
2. 在左侧导航栏中单击 【邮件发送】，进入 【邮件发送】 页面。
3. 单击 【发送邮件】，进入邮件内容编辑页面，输入或上传任务名称、收件人地址、发信地址、发信人名称、回信地址以及邮件内容。
4. 确认内容无误后，单击 【确定】。

新的普通邮件发送任务将被创建在邮件任务列表中，发送完毕后，您可以查看任务总请求数和成功数，并前往相应的电子邮箱查收测试邮件。您也可以通过相似的步骤 **发送模板邮件**。
