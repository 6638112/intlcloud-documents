## 发件前的准备工作
在开始发件之前，您需要设置腾讯云 DMS。无论使用控制台还是 API 发送电子邮件，都需要：

-   **注册腾讯云：** 您需要创建一个腾讯云国际站账户，然后才能使用腾讯云 DMS 或其他腾讯云服务。有关信息，请参阅[腾讯云账号注册](https://intl.cloud.tencent.com/register)。
    
-   **验证您的电子邮件域名：**  使用腾讯云 DMS 发送电子邮件前，您需要验证“发件人”地址对应的域名以证明您是此域名的所有者。有关信息，请参阅  [设置发信域名](https://intl.cloud.tencent.com/document/product/1070/38223)。

- **上传邮件模板：** 在发送模板邮件前，您需要预先上传邮件模板，并在发送邮件是提供对应的邮件地址和自定义内容。有关邮件模板的格式和要求，请参阅 [管理邮件模板](https://intl.cloud.tencent.com/document/product/1070/38224)
