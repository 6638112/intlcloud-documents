## Differences Between the Standard Edition and the Lite Edition
>- SDK added the Lite Edition from SDK 5.0. The original edition is called the Standard Edition.
>- Both the Standard Edition and the Lite Edition support<a href="https://intl.cloud.tencent.com/document/product/1047/36169"> V2 API</a> and provide the same API capabilities.
>- The Lite Edition no longer supports legacy APIs, whereas the Standard Edition still supports them.
>- Compared with the Standard Edition, the Lite Edition has greatly reduced the SDK size and installation package increment.
>- The Lite Edition currently supports Android and iOS platforms and will gradually add support for Windows and Mac platforms in the future.
>- If you have not integrated legacy APIs, we recommend that you directly use<a href="https://intl.cloud.tencent.com/document/product/1047/36169"> V2 API</a> and choose the Lite Edition SDK.
>- If you have integrated legacy APIs, we recommend that you upgrade to<a href="https://intl.cloud.tencent.com/document/product/1047/36169"> V2 API</a> and gradually transition to the Lite Edition SDK.
