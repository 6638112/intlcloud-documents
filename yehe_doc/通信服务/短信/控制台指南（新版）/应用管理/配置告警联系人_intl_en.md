You can add an alarm contact and configure specific reminder events. When an event is triggered, the specified contact will receive relevant SMS messages and emails. Events include frequency limit reminder, over-limit messaging reminder, template and signature review notification, and package alarm notification.

> Up to 5 alarm contact can be added for each application. 

## Adding an Alarm Contact
1. Log in to the [SMS Console](https://console.cloud.tencent.com/sms).
2. In the application list, click the name of the target application to enter the application details page.
3. Select the **Notifications and Alarms** tab and click **Add Alarm Contact**.
4. Enter the name, mobile number, and email address of the contact and select the desired events.
5. Click **Confirm**.


## Editing an Alarm Contact
1. Log in to the [SMS Console](https://console.cloud.tencent.com/sms).
2. In the application list, click the name of the target application to enter the application details page.
3. Select the **Notifications and Alarms** tab and click **Edit** in the row of the contact.
4. You can modify the name, mobile number, and email address of the contact and the events.
5. Click **Confirm**.

## Deleting an Alarm Contact
>It should be noted that once deleted, an alarm contact **cannot be recovered**; therefore, please do so with caution.

1. Log in to the [SMS Console](https://console.cloud.tencent.com/sms).
2. In the application list, click the name of the target application to enter the application details page.
3. Select the **Notifications and Alarms** tab and click **Delete** in the row of the contact.
4. In the pop-up window, click **Confirm**.
