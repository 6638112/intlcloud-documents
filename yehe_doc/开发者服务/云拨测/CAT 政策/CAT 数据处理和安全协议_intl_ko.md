## 1. 배경
이 모듈은 Cloud Automated Testing(“기능”)을 사용하는 경우 적용됩니다. 이 모듈은 [(“DPSA”)](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 처리 및 보안 계약에 포함되어 있습니다. 본 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에서 정의된 것과 같은 의미를 갖습니다. DPSA와 본 모듈 간에 상충이 있는 경우, 해당 상충 범위까지 본 모듈이 적용됩니다.

## 2. 처리
당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

<table>
   <tr>
      <th>개인 정보</th>
      <th>사용</th>
   </tr>
   <tr>
      <td><b>작업 정보 데이터: </b>작업 로그 데이터(앱 ID, UIN, 하위 계정 UIN, 작업 ID, 작업 구성의 로그 데이터) 및 작업 결과</td>
      <td>당사는 귀하의 개별 구성에 따라 귀하 및 귀하의 최종 사용자에게 기능을 제공하는 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 당사의 Elasticsearch Service(ES) 기능에 저장되고 백업됩니다.</td>
</tr>
   <tr> 
</table>
