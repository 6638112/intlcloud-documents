## 1. CONSIDERAÇÕES INICIAIS
Este módulo se aplica ao uso do recurso Cloud Automated Testing (“Recurso”). Este Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em [(“DPSA”)](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.
