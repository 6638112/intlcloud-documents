This document describes how to quickly use CAT features.

## Step 1. Create a task
1. Log in to the [CAT console](https://console.cloud.tencent.com/cat/probe/tasklist).
2. On the left sidebar, click **Tasks**.
3. Click **Create task** at the top of the **Tasks** page.
4. On the **Create task** page, select the test type and location, enter the information such as test address and name, and click **Create**.
>?For more information on task configuration, see [Creating Tests](https://www.tencentcloud.com/document/product/1169/51989).


## Step 2. Analyze the test performance data in multiple dimensions
1. Go to the [**Test Statistics**](https://console.cloud.tencent.com/cat/analyze) page.
2. Filter by information such as test domain, type, and time range as needed to view the data of key metrics of test tasks. You can also view the details of a test task.


>?For more information, see [Test Statistics](https://www.tencentcloud.com/document/product/1169/51992).
