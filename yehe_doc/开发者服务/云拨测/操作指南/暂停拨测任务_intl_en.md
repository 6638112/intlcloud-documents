This document describes how to pause a test task, after which the Test Statistics page will not be able to display test data and billing will stop.

## Directions
1. Log in to the [CAT console](https://console.cloud.tencent.com/cat/probe/tasklist).
2. On the left sidebar, click **Tasks**.

### Pausing a test task
In the test task list, click **More** > **Pause** in the **Operation** column to pause a task.
![](https://staticintl.cloudcachetci.com/yehe/backend-news/bDCm478_29intl_%E5%A5%BD%E5%8E%8B%E7%9C%8B%E5%9B%BE.png)

### Batch pausing test tasks
Select multiple tasks and click **Pause** in the top-right corner to pause them.
![](https://staticintl.cloudcachetci.com/yehe/backend-news/rUMf740_30intl_%E5%A5%BD%E5%8E%8B%E7%9C%8B%E5%9B%BE.png)
