**Description:** Checks whether CAM sub-users are associated with user groups.
**Compliance check logic:** If a CAM user is associated with at least one user group, this rule is compliant.
**Rule ID:** `cam-user-group-bound`
**Risk level:** Low
**Supported resource types:** `QCS::CAM::User`
**Trigger type:** Periodic, 24 hours
**Keywords:** User, user group
**Input parameters:** None
