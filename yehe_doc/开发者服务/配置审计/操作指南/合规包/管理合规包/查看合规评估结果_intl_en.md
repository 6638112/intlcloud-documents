On the **Conformance pack list** page, you can view the evaluation result of each conformance pack. If all rules in a conformance pack are evaluated as compliant, the conformance pack will be evaluated as compliant. Otherwise, it will be non-compliant.
![](https://qcloudimg.tencent-cloud.cn/raw/602d0733ea0c61376473c612adda0771.png)
To view the evaluation result of each rule in a conformance pack, click the conformance pack name to go to the **Conformance pack details** page.
