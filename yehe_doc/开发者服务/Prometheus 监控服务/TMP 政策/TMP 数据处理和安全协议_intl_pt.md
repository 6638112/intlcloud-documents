## 1\. CONSIDERAÇÕES INICIAIS
Este Módulo se aplica se você usa o serviço de monitoramento Prometheus (TMP) (“Recurso”). Este Módulo é incorporado ao contrato de segurança e privacidade de dados localizado no   [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo se aplicará conforme a inconsistência.

## 2\. PROCESSAMENTO
Processaremos os seguintes dados em conexão com o Recurso:

<table>
   <tr>
      <th>Informações Pessoais</th>
      <th>Uso</th>
   </tr>
   <tr>
      <td><b>Dados de configuração:</b><br/>• Parâmetros de configuração (quando você usa a função de agente, você precisará inserir os dados de configuração, incluindo o IP do objeto capturado e o endereço necessário para a emissão de relatórios de dados)<br/>• Dados de configuração de alarme [você pode definir regras de alarme (por exemplo, lembretes de tráfego e lembretes de anormalidade) e definir quais circunstâncias acionarão o alarme]</td>
      <td>O objetivo de coletar os parâmetros de configuração é fornecer a você o Recurso capturando dados relatados pelo usuário. O objetivo de coletar os dados de configuração do alarme é fornecer a você o serviço de alerta.</td>
     </tr>
   <tr>

      <td>Nós apenas processamos os dados com o propósito de fornecer o Recurso para você e garantir que ele cumpra seus requisitos personalizados para processar e analisar seus dados relacionados com os negócios. </td>
    </tr>
   <tr>

## 5\. RETENÇÃO DE DADOS
Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

<table>
   <tr>
      <th>Informações Pessoais</th>
      <th>Política de Retenção</th>
   </tr>
   <tr>
      <td>Dados de configuração <br/>Dados de monitoramento </td>
      <td>Armazenado de acordo com o tempo de armazenamento selecionado por você, ou seja, 15, 30 ou 45 dias. Uma vez que o período de tempo atinja o tempo de armazenamento selecionado, esses dados serão excluídos automaticamente dentro de 7 dias. Se você solicitar a exclusão dos dados antes do término do tempo de armazenamento selecionado, esses dados serão excluídos imediatamente.</td>
</table>


Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.


## 6\. CONDIÇÕES ESPECIAIS
Você deve garantir que este Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
