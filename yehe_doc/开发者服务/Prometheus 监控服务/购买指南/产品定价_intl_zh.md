## 实例定价

### 计算公式
**总费用 = 日指标上报指标数（单位：百万条） x 单价+ 流量费用（目前免费）**

### 计费项说明

| 计费项     | 说明                                                   |
| :--------- | :----------------------------------------------------- |
| 日均上报量 | 根据您上报指标量按阶梯计费                             |
| 流量费用   | 公网流量费用，目前免费。                               |

## 计费示例

**举例1（上报指标数低于2亿）：**

假设用户当天上报的指标数量为6千万条指标数据，新建实例时选择的数据存储时间为15天。当天所需的费用为：60×0.19=11.4美元

注：6千万=60百万。

**举例2（上报指标数高于2亿）：**

假设用户每日上报的指标数量为9亿条指标数据，新建实例时选择的数据存储时间为30天。 当天所需的费用为：
前2亿条指标数据上报费用为：200×0.25=50美元
2亿-8亿条指标数据上报费用为：（800-200）×0.2=120美元
8亿-9亿条指标数据上报费用为：（900-800）×0.12=12美元
总费用：50+120+12=182美元

注：9亿=900百万。
