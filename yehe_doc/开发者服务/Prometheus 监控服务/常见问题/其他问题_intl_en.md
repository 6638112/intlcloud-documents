### How do I migrate from self-built Prometheus to TMP?[](id:question1)

In the configuration file of self-built Prometheus, add a remote write configuration pointing to TMP for migration. For more information, please see Migration from Self-Built Prometheus.

### Does TMP support customizing the reported data?[](id:question2)

Yes. TMP supports customizing the reported metric monitoring data in multiple languages and displaying such data in the integrated Grafana dashboards.
