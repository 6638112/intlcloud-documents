You can easily view the running status of the Prometheus agent to check whether it is working normally in real time.

## Prerequisites

- The Prometheus agent has been installed in the TKE cluster. For more information, please see Agent Management.
- Click a **Cluster ID** in the TKE cluster list to enter the **Integrate with TKE** page.

## Directions


Click **Agent Info** on the **Integrate with TKE** page to view the running status of the Prometheus agent in TKE as shown below:
