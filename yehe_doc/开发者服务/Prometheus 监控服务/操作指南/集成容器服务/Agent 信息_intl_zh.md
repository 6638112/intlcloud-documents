为了方便用户查看 Prometheus Agent 运行的状态，实时了解 Agent 是否工作正常。

## 前提条件

- 已经对某个容器服务 TKE 安装了 Prometheus Agent，详情请参见Agent 管理。
- 通过容器服务列表单击**集群 ID**进入到容器服务集成管理页面。

## 操作步骤
