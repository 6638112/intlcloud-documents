This document describes how to create a scrape task.

## Prerequisites

- You have created a TMP instance.
- You have created and installed an agent.

## Directions

1. Log in to the [TMP console](https://console.cloud.tencent.com/monitor/prometheus).
2. In the instance list, select the corresponding TMP instance and click **Agent Management** on the left.
3. Click an agent ID and click **Scrape Task**.
4. Enter the scrape task page and click **Create**.
5. On the scrape task creation page, enter the agent scrape task rule as prompted. For more information on the task configuration, please see the scrape configuration description.
   ![](https://qcloudimg.tencent-cloud.cn/raw/10b31eef61f57d10d70fce803f20b5a3.png)
6. After completing the configuration, click **Save**.
