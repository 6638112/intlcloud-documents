
By default, TMP instances in the current region are displayed in the TMP console. To help you quickly find the instances in the current region, Tencent Cloud provides the search feature and allows you to filter instances by resource attributes such as instance ID, name, status, AZ, IPv4 address, and tag.

## Directions

1. Log in to the [TMP console](https://console.cloud.tencent.com/monitor/prometheus).
2. Enter the conditions in the search box as needed and click <img src="https://main.qcloudimg.com/raw/3cca38f08eaa87087cdd1b81eaf08a0a.png" style="margin: 0;"></img>.
![](https://qcloudimg.tencent-cloud.cn/raw/c960453cecd7b58adc485de7126b38d0.png)
3. You can filter instances by different conditions. Currently, the following dimensions are supported:
<table>
<tr>
<th>Filter Condition</th>
<th>Description</th>
</tr>
<tr>
<td>Instance ID</td>
<td>You can directly enter multiple instance IDs for quick filtering. Each instance ID supports only exact match-based filtering.</td>
</tr>
<tr>
<td>Instance name</td>
<td>You can enter only one instance names for filtering. Fuzzy match-based filtering is supported.</td>
</tr>
<tr>
<td>Instance status</td>
<td>You can enter multiple status values for filtering. You can also configure the list header for quick filtering.</td>
</tr>
<tr>
<td>AZ</td>
<td>You can enter multiple AZs for filtering. After the region is switched, the corresponding AZs in the region will be displayed. You can also configure the list header for quick filtering.</td>
</tr>
<tr>
<td>IPv4 address</td>
<td>You can enter multiple IPv4 addresses for filtering. Each IPv4 address supports only exact match-based filtering.</td>
</tr>
<tr>
<td>Tag</td>
<td>You can enter multiple tags to filter instances. You can also directly click a tag value in the instance list for filtering.</td>
</tr>
</table>
