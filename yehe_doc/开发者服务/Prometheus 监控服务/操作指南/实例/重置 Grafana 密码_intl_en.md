If you forgot the Grafana Admin login password of an instance, you can reset it in the console.

## Directions

1. Log in to the [TMP console](https://console.cloud.tencent.com/monitor/prometheus).
2. In the instance list, select the TMP instance to be modified and click **More** > **Grafana** > **Change Password** on the right.
3. In the **Change Password** window that pops up, enter the new Grafana admin password and click **OK**.
