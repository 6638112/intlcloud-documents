## 1\. **CONSIDERAÇÕES**
Este Módulo se aplica se você usar CODING DevOps (“**Recurso**”). Este Módulo é incorporado ao contrato de segurança e privacidade de dados localizado no   [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo se aplicará conforme a inconsistência.

## 2\. **PROCESSAMENTO**
Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de negócios**   (código armazenado no repositório de código, informações do projeto, acesso da equipe e registro de operações) | Apenas processamos estes dados com a finalidade de fornecer o Recurso para você. |
|**Dados de administração**  (número de telefone, endereço de e-mail)                                                              |Se você permitir o acesso por determinado pessoal, o número de telefone ou e-mail relevante que você atribuir como parte do acesso a esse usuário será processado com o propósito de permitir tal acesso.                                                              |
## 3\. **REGIÃO DO SERVIÇO**
Conforme especificado no DPSA.

## 4\. **SUBPROCESSADORES**
Conforme especificado no DPSA.

## 5\. **RETENÇÃO DE DADOS**
Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações Pessoais**                                     | **Política de Retenção**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de negócios**| Retemos esses dados enquanto você usar o Recurso. Você pode excluir manualmente esses dados no Recurso. Esses dados serão excluídos automaticamente dois (2) anos após a data do último login ou na data que é seis (6) meses após a data em que os valores em atraso ficarem pagáveis em uma conta. |
|**Dados de administração**|Armazenado durante a duração do uso do Recurso, a menos que você opte por excluí-lo previamente. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. **CONDIÇÕES ESPECIAIS**
Você deve garantir que este Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
