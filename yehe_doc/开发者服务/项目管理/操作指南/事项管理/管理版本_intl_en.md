This document describes how to use Version Management in CODING Project Collaboration.

## Open Project

1. Log in to the CODING Console and click **Use Now** to go to CODING page.
2. Click <img src ="https://main.qcloudimg.com/raw/d94a8e60dd3a41d0af07d72ae0e9d70e.png" style ="margin:0"> in the upper-right corner to open the project list page and click a project icon to open the project.
3. Click **Project Collaboration** in the menu on the left.

### Action log[](#activity-log)

Changes or modifications to the **description** of issues are recorded in the action log, which clearly shows all the update details and version changes.
![](https://qcloudimg.tencent-cloud.cn/raw/9b5f3448c86ba84a1b0fbd45e4dd5ef6.png)

### Version backtracking[](#backdate)

In the action log, you can click **View the version** to view all earlier versions. Version backtracking allows you to restore an earlier version.
![](https://qcloudimg.tencent-cloud.cn/raw/6fae399a4758f9dbc554c49e1edf555e.png)
![](https://qcloudimg.tencent-cloud.cn/raw/2a518f60a0766262884196247b739779.png)
