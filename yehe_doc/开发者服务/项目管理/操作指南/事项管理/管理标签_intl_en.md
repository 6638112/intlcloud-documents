This document describes how to use Tag Management in CODING Project Collaboration.

## Open Project

1. Log in to the [CODING Console](https://console.cloud.tencent.com/coding) and click **Use Now** to go to CODING page.
2. Click <img src ="https://main.qcloudimg.com/raw/d94a8e60dd3a41d0af07d72ae0e9d70e.png" style ="margin:0"> in the upper-right corner to open the project list page and click a project icon to open the project.
3. Click **Project Collaboration** in the menu on the left.

## Add Tags[](#add)

On the issue details page, you can add **tags** to the issue to demonstrate key information. An issue can have more than one tag. No tags are added to an issue by default. You can create a tag by specifying the **title** and **tag color**.
![](https://qcloudimg.tencent-cloud.cn/raw/6260e38a5cd2cec085b70f0d05b2177f.png)

On the issue list page, tasks in the same category can be easily sorted and filtered by tags, allowing you to quickly integrate similar information into the project.
![](https://qcloudimg.tencent-cloud.cn/raw/4e07bc8166822f149af701984566a91b.png)

## Manage Tags[](#management)

Click **Project Settings** > **Projects and Members** > **Category Tag** in the lower-left corner on the project homepage to view statistics on associated tags, and creating, renaming or deleting tags.
>!Deleting a tag will remove the reference to this tag from all issues and merge requests. Proceed with caution.
>
![](https://qcloudimg.tencent-cloud.cn/raw/5c3a3cdc09f2c60be61a5233a32a32cd.png)
