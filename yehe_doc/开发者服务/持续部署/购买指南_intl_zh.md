## 关于私有化部署

由于 CODING DevOps 产品复杂度高，安装需要专业服务支持，在您完成产品采购后，我们的团队将帮助您聘请受到我们认证的服务供应商。
标准版私有化部署流程如下：
1. 需求调研：组织项目需求调研，参与实施信息收集、技术需求沟通，识别交付内容，确认项目实施小组，根据项目说明书的工作内容和交付计划，准备相关人力资源及外部支持专家。
2. 方案设计：根据调研中系统实际技术需求和各个子系统的业务需求，编写并向买方提交符合规范要求的《项目实施计划》，并由项目组评审，评审通过后由双方共同签署评审意见，并正式生效。
3. 部署实施：根据产品配置，进行软件在服务器端的安装和调试，包括初始化数据，原有系统的重要数据的转换导入，前后台软件的安装，配置参数调整等工作。
4. 产品演示：系统部署安装完成并确认可在系统正常运行后，开始相关产品演示。演示顺利完成后将开始软件在试点部门试用，将向用户提交《系统使用操作手册》，《软件功能清单》，这两种文档将详细描述软件的使用过程，软件所包含的全部系统功能模块。
5. 标准测试：按 CODING 官方标准测试用例完成标准功能测试，平台功能联调，确保部署完成的环境所有功能可用。

为提供更优质的服务体验，在标准版的基础上提供如下可选服务项:

试运行支持服务：额外提供 N 人天的试运行支持服务，帮助客户更快捷的实现核心业务系统的试运行。
