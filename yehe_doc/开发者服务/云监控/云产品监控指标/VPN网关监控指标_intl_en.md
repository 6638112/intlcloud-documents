Cloud Monitor provides the following monitoring metrics for VPN gateways:

| Metric | Description | Unit | Dimension |
| ----- | ------------ | ---- | ------ |
| Public network outbound bandwidth | Average outbound traffic per second of VPN gateway | Mbps | VPN gateway |
| Public network inbound bandwidth | Average inbound traffic per second of VPN gateway | Mbps | VPN gateway |
| Outbound packets | Average outbound packets per second of VPN gateway | Packets/second | VPN gateway |
| Inbound packets | Average inbound packets per second of VPN gateway | Packets/second | VPN gateway |
