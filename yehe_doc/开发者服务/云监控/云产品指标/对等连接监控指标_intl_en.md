Cloud Monitor provides the following monitoring metrics for peering connections:

| Metric | Description | Unit | Dimension |
| ----- | -------------- | ---- | ---- |
| Network inbound bandwidth | Average inbound traffic per second of the peering connection | Mbps | Peering connection |
| Network outbound bandwidth | Average outbound traffic per second of the peering connection | Mbps | Peering connection |
| Outbound packets | Average outbound packets per second of the peering connection | Packets/second | Peering connection |
| Inbound packets | Average inbound packets per second of the peering connection | Packets/second | Peering connection |
| Packet loss rate | Packet loss rate monitored by the peering connection in one minute | % | Peering connection |
