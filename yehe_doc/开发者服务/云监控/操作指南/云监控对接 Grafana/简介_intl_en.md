[Tencent Cloud Monitor](https://intl.cloud.tencent.com/product/cm) (CM) provides load and performance monitoring metrics for various Tencent Cloud services such as CVM and TencentDB. You can use the CM console and APIs to get relevant monitoring data. Tencent Cloud Monitor App is an application plugin that adapts to the open-source software Grafana. It calls CM APIs (version 3.0) to get and display monitoring data on custom dashboards. Supported Tencent Cloud services include:
