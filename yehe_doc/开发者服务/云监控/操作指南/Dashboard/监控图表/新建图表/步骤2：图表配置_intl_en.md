This document describes how to configure a chart when creating a monitoring chart.

## Directions

1. Log in to the [Cloud Monitor console](https://console.cloud.tencent.com/monitor).
2. In the left sidebar, click **Dashboard** > **Dashboard List** to go to the dashboard list page.
3. Find the dashboard for which you want to create charts, and click the dashboard name to go to the dashboard management page.
4. Click **<img src="https://main.qcloudimg.com/raw/827988040ba03fd73a5a95cc942eb5cd.png"  style="margin:0;" width="3%">** > **Create a Chart** to go to the chart editing page and configure the settings as follows:
   - Basic info: includes the chart name and chart notes.
   - Chart type: can be a line chart, a bar chart, a dashboard, a pie chart, or a numerical chart. More chart types will be available in the future. For different chart types, the chart elements, thresholds, and legend configurations are different. For more information, see [Use Cases of Different Chart Types](https://intl.cloud.tencent.com/document/product/248/38480).
- Data link: defines whether to display a link when you double-click the chart. After adding a data link, you can click a certain period in the chart to be redirected to the custom link.
  ![](https://main.qcloudimg.com/raw/42785905c5dc8d879927bf8a92a19d62.png)
- Chart link: defines the chart redirection link. After adding a chart link, you can click **<img src="https://main.qcloudimg.com/raw/56746ee22a2a29a5bef300c8720b1297.png"  style="margin:0;" width="3%">** in the chart to be redirected to the custom link, as shown in the figure below:
  ![](https://main.qcloudimg.com/raw/799023c3d76cc417ccda719d37c9584f.png)
5. After configuring the settings, click **<img src="https://main.qcloudimg.com/raw/2cf48d6910973ec3dc7074e05bac24db.png"  style="margin:0;" width="3%">**.
