This document describes how to delete a dashboard.

## Directions

1. Log in to the [Cloud Monitor console](https://console.cloud.tencent.com/monitor).
2. In the left sidebar, click **Dashboard** > **Dashboard List** to go to the dashboard list page.
3. Both individual deletion and batch deletion are supported, as described below:
 -  <b>Individual deletion</b>
Find the dashboard that you want to delete in the dashboard list, click **Delete** under the “Operation” column, and then confirm the deletion in the window that appears.
 ![](https://main.qcloudimg.com/raw/c832006705a99565bab6f5fe5d4959da.png)
 - <b>Batch deletion</b>
 i. Select all or multiple dashboards that you want to delete in the dashboard list, and click **Delete** above the list.
ii. Click **Delete** in the window that appears.
![](https://main.qcloudimg.com/raw/36346bb834eeb12d894de5ac2b168377.png)
