This document describes how to add a dashboard to Favorites.

## Directions

After adding dashboards to Favorites, you can quickly switch between the dashboards in Favorites by following the directions under “Switching Dashboards” in [Viewing Dashboards](https://intl.cloud.tencent.com/document/product/248/38469#.E5.88.87.E6.8D.A2-dashboard) and perform troubleshooting. You can quickly filter the dashboards in Favorites by following the directions in [Viewing Dashboards](https://intl.cloud.tencent.com/document/product/248/38469).

1. Log in to the [Cloud Monitor console](https://console.cloud.tencent.com/monitor).
2. In the left sidebar, click **Dashboard** > **Dashboard List** to go to the dashboard list page.
3. In the dashboard list, find the dashboard that you want to add to Favorites, and click the dashboard name to go to the dashboard management page.
4. Click **![](https://main.qcloudimg.com/raw/84c8d89448e297dbb3e1eedd795797ea.png)**, as shown in the figure below. If the "Added to Favorites successfully" message appears, the dashboard has been successfully added to Favorites.
![](https://main.qcloudimg.com/raw/37e10c11155bf986f464e0bf062c60f4.png)
