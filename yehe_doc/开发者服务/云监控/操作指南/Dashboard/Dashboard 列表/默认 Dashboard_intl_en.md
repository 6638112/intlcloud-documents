This document describes how to configure the default dashboard.


## Directions
1. Log in to the [Cloud Monitor console](https://console.cloud.tencent.com/monitor).
2. In the left sidebar, click **Dashboard** > **Dashboard List** to go to the dashboard list page.
3. In the dashboard list, hover over the name of the target dashboard and click **Set as "Default Dashboard"**.
![](https://main.qcloudimg.com/raw/7711e8b983320d77a56d08fedf81460d.png)
After specifying the default dashboard, you can click **Default Dashboard** in the menu bar on the left to access the dashboard instantly.
>? For custom dashboards, only one default dashboard can be specified. If you specify the default dashboard repeatedly, the previous default dashboard will be overwritten.
