## Feature Overview
Cloud Monitor (CM) provides a highly available managed Grafana service to reduce the costs of Grafana setup and OPS, which includes installation, upgrade, plugin installation, dashboard installation, and data source configuration features.

## Directions
### Step 1. Create Grafana[](id:step1)
1. Log in to the [TCMG console](https://console.cloud.tencent.com/monitor/grafana).
2. Click **Create** and configure the items as prompted. For detailed directions, see [Creating Grafana Instance](https://intl.cloud.tencent.com/document/product/1124/43953).
3. After completing the configuration, click **Buy Now**.

### Step 2. Configure Grafana[](id:step2)
1. In the [TCMG service list](https://console.cloud.tencent.com/monitor/grafana), find the target instance and click **Manage** in the **Operation** column.
2. On the Grafana management page, click **Settings**.
3. Click **Edit** in the top-left corner of the **Settings** page and modify the configuration file as prompted.
4. After completing the modification, click **OK**.

### Step 3. Log in to Grafana to view the monitoring data[](id:step3)
1. In the [TCMG service list](https://console.cloud.tencent.com/monitor/grafana), select the target instance and click the <img src="https://main.qcloudimg.com/raw/978c842f0c093a31df8d5240dd01016d.png" width="2%"> icon on the right of its instance ID.
2. In Grafana, enter your Grafana login account and password to access the operation section of the visual dashboard.
3. Click <img src="https://main.qcloudimg.com/raw/7e3fff6131aa085987552a9725e9ae54.png" width="2%"> to enter the monitoring dashboard list page.
4. On the monitoring dashboard list page, click the name of the target monitoring chart to view the monitoring data.
