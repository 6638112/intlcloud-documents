The **Resource Types** page provides references to help you use Tencent Cloud resources. This document describes how to query resource information.


### Directions


1. Log in to the [TIC console](https://console.cloud.tencent.com/tic).
2. In the left sidebar, choose **Orchestration** > **Resource Types** to go to the **Resource Types** page.
3. Click **Details** on the right side of any resource to view the parameter information, use cases, and references as shown in the following figure.
![](https://main.qcloudimg.com/raw/d092469450650eadf582940d76699207.png)
