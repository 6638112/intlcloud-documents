This document describes how to use allowlist management. This feature is used to allow specified users to report more information. Users in the allowlist will report all API request information, including API request and response.

## Directions

### Adding to allowlist
1. Log in to the [RUM console](https://console.cloud.tencent.com/rum).
2. On the left sidebar, click **Application Management** > **Allowlist Management** to enter the allowlist management page.
3. Click **Add to Allowlist** in the top-left corner and enter the user UIN and remarks in the pop-up window.
![](https://qcloudimg.tencent-cloud.cn/raw/eaa96cf9c3a261154f97fad30435f657.png)



### Removing from allowlist
Find the UIN of the target user in the allowlist and click **Delete** in the **Operation** column.
