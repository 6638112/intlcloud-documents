## 1\. CONSIDERAÇÕES INICIAIS
Este Módulo se aplica se você usar o Real User Monitoring (“Recurso”). Este Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.

   <tr>

## 6\. CONDIÇÕES ESPECIAIS
Você deve garantir que esse Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
