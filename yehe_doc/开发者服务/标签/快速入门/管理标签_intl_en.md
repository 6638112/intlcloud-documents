## Overview

This document describes how to add tags for and delete/modify the tags of specific resources.

## Directions

1. Log in to the [Tag console](https://console.cloud.tencent.com/tag) and go to **Resource Tag**.
2. Filter your resources. For detailed instructions, refer to [Querying Resources by Tag](https://intl.cloud.tencent.com/document/product/651/32582).
3. In the list below, select the target resources.
4. Click **Edit tag** to enter the editing page.
5. Manage tags as described below.
![](https://main.qcloudimg.com/raw/5d69a9230b5a96eba390a150ba8fa217.png)
	- Click **Add tag** and select a key and value to add a tag.
	- Click **Delete** to delete a tag.
	- Select a new key or value for an existing tag to modify it.
6. Click **Save**.
