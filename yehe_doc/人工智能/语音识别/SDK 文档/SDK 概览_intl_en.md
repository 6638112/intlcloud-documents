## SDK Description
Tencent Cloud ASR provides server, client, frontend, and mini program SDKs to help you integrate speech recognition features into your service in a convenient, fast, and flexible way.

ASR SDK supports the following features:
- [Recording file recognition](https://cloud.tencent.com/document/product/1093/37823)
- [Real-Time speech recognition](https://cloud.tencent.com/document/product/1093/48982)
- [Async audio stream recognition](https://cloud.tencent.com/document/product/1093/52061)
- [Ultrafast recording file recognition](https://cloud.tencent.com/document/product/1093/52097)
- [One-Sentence recognition](https://cloud.tencent.com/document/product/1093/35646)
