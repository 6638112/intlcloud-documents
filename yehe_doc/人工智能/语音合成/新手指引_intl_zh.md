腾讯云语音合成的计费模式为后付费。您需要全面了解语音合成的计费模式，有利于您选择最优的计费方案。详情请参见 [购买指南](https://intl.cloud.tencent.com/document/product/1154/47874)。

## 3. 新手入门
#### 3.1 登录注册
[登录](https://cloud.tencent.com/login?s_url=https%3A%2F%2Fcloud.tencent.com%2F) 腾讯云官网。如果没有腾讯云账号，请参考 [账号注册教程](https://intl.cloud.tencent.com/document/product/378/17985) 注册账号。

#### 3.2开通服务
注册账号后，进入 [语音合成控制台](https://console.cloud.tencent.com/tts)，开通语音合成服务。

#### 3.3 新建密钥
进入 [API 密钥管理页面](https://cloud.tencent.com/login?s_url=https%3A%2F%2Fconsole.cloud.tencent.com%2Fcam%2Fcapi)，单击【新建密钥】，生成  AppID、SecretId 与 SecretKey，用于 API 调用时生成签名。 

#### 3.4 进行接入
根据您的需求，使用 SDK 与 API 进行接入，推荐使用 SDK 方式接入。

#### 3.5 查看调用
接入后，您可以登录 [语音合成控制台](https://console.cloud.tencent.com/tts)，查看语音合成明细。
