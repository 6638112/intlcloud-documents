以下内容包含了腾讯云人脸核身接口调用的返回码及收费关系。

| 错误码                               | 错误描述                   | 是否收费 |
| :----------------------------------- | :------------------------- | -------- |
| FailedOperation.CompareFail          | 比对失败。                 | 是       |
| FailedOperation.CompareLowSimilarity | 比对相似度未达到通过标准。 | 是       |
