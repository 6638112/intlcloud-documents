## 1\. CONSIDERAÇÕES INICIAIS

Este Módulo se aplica se você usar o recurso de Machine Translation Feature (“**Recurso**”). Este Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em: (“**[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)**”). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência. 

## 2\. PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Texto enviado para tradução** (como inserido ou enviado por você ou pelos usuários finais da tradução) | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você.<br> O Recurso não prosseguirá com o processamento do texto inserido até que os serviços de tradução (de acordo com sua solicitação) estejam concluídos. |
| **Registro de solução de problemas** (contendo a entrada original e a saída traduzida, e seu ID da aplicação e número de identificação único) | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você. |

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma (salvo se de outra forma exigido pelas Leis de Proteção de Dados aplicáveis):

| **Informações pessoais**       | **Política de retenção**                                         |
| ------------------------------- | ------------------------------------------------------------ |
| Texto enviado para tradução | Excluído após 7 dias, a menos que você solicite a exclusão dos dados. Nesse caso, os dados serão excluídos imediatamente. |
| Registro de solução de problemas            | Excluído em 1 dia.                                        |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.
