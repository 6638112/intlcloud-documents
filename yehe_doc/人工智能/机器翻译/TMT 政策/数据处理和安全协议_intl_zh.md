## 1\. 背景

本模块仅在您使用 Machine Translation feature（简称“**功能**”）时适用。本模块已纳入位于 ("**[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)**") 的数据处理和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，本模块适用。

## 2\. 处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **提交用于翻译的文本**（由您或您的最终用户输入或提交以进行翻译） | 我们仅为了 向您提供本功能而处理此类数据。<br> 当翻译服务（根据您的请求）完成后，本功能不会进一步处理输入的文本。 |
| **故障排除日志**（包含原始输入和翻译输出，以及您的 APPID 和 UIN） | 我们仅为了 向您提供本功能而处理此类数据。 |

## 3\. 服务区域

见 DPSA 中的规定。

## 4\. 分包处理方

见 DPSA 中的规定。

## 5\. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述（除非适用的数据保护法另有规定）：

| **个人信息**       | **保留政策**                                         |
| ------------------------------- | ------------------------------------------------------------ |
| 提交用于翻译的 文本 | 7 天后删除，除非您请求 删除此类数据（这种情况下，此类数据将被立即删除）。 |
| 故障排除日志            | 1 天内删除。                                        |

您可以根据 DPSA 请求删除此类个人数据。
