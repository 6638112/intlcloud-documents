## 1.INTRODUÇÃO

Este módulo se aplica se você usar o recurso de Face Recognition (**Recurso**). Este Módulo é incorporado à política de privacidade localizada na [Política de privacidade](https://intl.cloud.tencent.com/document/product/301/17345). Os termos usados mas não definidos neste módulo devem ter o significado que lhes é atribuído na Política de privacidade. No caso de algum conflito entre a Política de privacidade e este módulo, o módulo deve ser aplicado na inconsistência.

 

 

## 2.CONTROLADORIA

O controlador das informações pessoais descritas neste Módulo é o especificado na Política de Privacidade.

 

 

## 3.DISPONIBILIDADE

Este recurso está disponível globalmente para os usuários.

 

 

## 4.COMO USAMOS AS INFORMAÇÕES PESSOAIS

Usaremos as informações das seguintes formas e de acordo com as seguintes bases legais:

 

| **Informações pessoais**                                     | **Uso**                                                      | **Base Jurídica**                                            |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Log de erro** (incluindo estados de upstream e downstream, entrada e saída do usuário, se a solicitação foi bem-sucedida ou não, informações de API da nuvem, incluindo AppID, UIN, SubUIN, Região, parâmetros de entradade API e código de retorno, informações de saída de algoritmos) | Usamos essas informações para garantir as funções do serviço conforme necessário e para analisar as causas dos erros oufalhas e para a solução desses problemas. | Processamos essas informações como necessário para a execução do nosso contrato com vocêpara fornecimento dos Serviços. |
| **Log de execução** (incluindo IP de solicitação do usuário, conta de usuário, solicitação de registro de data/hora, interface de solicitação, se foi faturado ou não, PersonID, tempo de acesso, resultados do reconhecimento) | Usamos essas informações para fins de faturamento.           | Processamos essas informações como necessário para que executemos nosso contrato com você de prestação de serviços. |
| **Características faciais:** conjunto de strings binárias geradas por transcodificação de algoritmo/modelo, sem conteúdo real e irreversibilidade.Elas são geradas a partir das imagens faciais coletadas dos usuários finais. | Usamos essas informações para garantir as funções do Serviço como requerido. | Processamos essas informações como necessário para que executemos nosso contrato com você de prestação de serviços. |

 

 

## 5.COMO ARMAZENAMOS E COMPARTILHAMOS INFORMAÇÕES PESSOAIS

Como especificado na Política de privacidade.

 

 

## 6.COMO COMPARTILHAMOS INFORMAÇÕES PESSOAIS

Como especificado na Política de privacidade, inclusive Tencent Cloud Computing (Pequim) Co., Ltd.

 

 

## 7.RETENÇÃO DE DADOS

Reteremos informações pessoais de acordo com o seguinte:

| **Informações pessoais** | **Política de retenção**                                      |
| ------------ | ------------------------------------------------- |
| Log de erro     | Automaticamente excluído após o espaço de armazenamento atingir 2 GB (eles geralmente são salvos por vários dias) |
| Log de execução     | Armazenado durante toda a vida útil do Serviço ou até que você envie uma solicitação de exclusão de dados     |
| Características faciais     | Armazenadas durante a vida útil do Serviço ou até que você envie uma solicitação de exclusão      |
