## 1.はじめに

本モジュールは、お客様がFace Recognition機能（「**本機能**」）を利用する場合に適用されます。本モジュールは、[プライバシーポリシー](https://intl.cloud.tencent.com/document/product/301/17345)内のプライバシーポリシーに組み込まれています。本モジュールで使用されているが定義されていない用語は、プライバシーポリシーでそれらに与えられた意味を持つものとします。プライバシーポリシーと本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

 

 

 

## 2.管理者

本モジュールに記載されている個人情報の管理者は、プライバシーポリシーに規定されているとおりです。

 

 

 

## 3.可用性

本機能は世界中のユーザーが利用可能です。

 

 

 

## 4.弊社による個人情報の使用方法

弊社は、以下の法的根拠に従って、以下の方法で個人情報を使用します。

 

| **個人情報**                                                 | **用途**                                                     | **法的根拠**                                                 |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **エラーログ**（アップストリームおよびダウンストリームの状況、ユーザーの入力と出力、リクエストが成功したかどうか、アプリID、UIN、サブ UIN、地域、API 入力パラメータおよび戻りコード、アルゴリズム出力情報を含むクラウド API 情報） | 弊社は、本サービスが必要に応じて確実に機能し、エラーや障害の原因を分析して、そのような問題のトラブルシューティングを行うために、これらの情報を使用します。 | 弊社は、お客様との契約を履行し、本機能を提供するために必要な情報として、この情報を処理します。 |
| **実行ログ**（ユーザーリクエスト IP、ユーザーアカウント、リクエストのタイムスタンプ、リクエストのインターフェイス、請求済みまたは未請求、PersonID、アクセス時間、認識結果を含む） | 弊社は、この情報を請求のために使用します。                   | 弊社は、お客様との契約を履行し、本サービスを提供するために必要な情報として、この情報を処理します。 |
| **顔の特徴：**アルゴリズム/モデルトランスコードによって生成された一連のバイナリ文字列で、実際の内容と不可逆性はありません。収集したエンドユーザーの顔画像から生成されます。 | 弊社は、本サービスが必要に応じて確実に機能するためにこの情報を使用します。 | 弊社は、お客様との契約を履行し、本サービスを提供するために必要な情報として、この情報を処理します。 |

## 7.データ保持

弊社は、以下の条件に従って個人情報を保持します。

 

 

| **個人情報** | **保持方針**                                           |
| ------------------ | ------------------------------------------------------------ |
| エラーログ         | 保存容量が 2GB に達した後に自動的に削除されます（通常は数日間保存されます）。 |
| 実行ログ           | サービスの有効期間中、またはお客様がデータ削除要求を提出するまで保存されます。 |
| 顔の特徴           | サービスの有効期間中、またはお客様がデータ削除要求を送信するまで保存されます。 |
