## 1.HISTÓRICO

Este módulo se aplica se você usar o recurso de Face Recognition (**Recurso**). Este módulo está incorporado ao contrato de segurança e privacidade de dados localizado no [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, ele deve ser aplicado conforme a inconsistência.


## 2.PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

 

| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Imagens faciais de seus usuários finais** Isso não é armazenado. | O Recurso extrairá dados de características faciais (na forma de strings binárias irreversíveis) e processará tais dados. O Recurso não armazenará nem processará ainda mais as imagens faciais depois que os dados das características faciais forem extraídos. |
| **PersonID**: identificador que você atribui a cada um de seus usuários finais. | Apenas processamos esses dados com a finalidade de lhe fornecer o Recurso. |

 

## 3.REGIÃO DOS SERVIÇOS

Conforme especificado no DPSA.



## 4.SUBPROCESSADORES

Conforme especificado no DPSA, inclusive Tencent Cloud Computing (Pequim) Co., Ltd.

 

 

 

## 5.RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:


**Imagens faciais de seus usuários finais**: não armazenamos isso.

**PersonID**: armazenado durante toda a vida útil do Serviço ou até você excluí-lo.

 

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

 


## 6.CONDIÇÕES ESPECIAIS

Você deve garantir que este Recurso seja usado apenas por usuários finais que têm pelo menos a idade mínima para que um indivíduo possa consentir com o processamento de seus dados pessoais ou com o consentimento dos pais obtido para pessoas com idade inferior à mínima exigida. Aqui pode haver diferença dependendo da jurisdição em que o usuário final está localizado.
