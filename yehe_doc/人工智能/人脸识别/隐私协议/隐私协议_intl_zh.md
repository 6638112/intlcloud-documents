## 1.简介

本模块仅在您使用人脸识别功能（简称“**功能**”）时适用。本模块包含在位于[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。


## 2.控制方

本模块中所述个人信息的控制方见隐私政策中的规定。



## 3.可用性

本功能可供全球范围内的用户使用。

 

## 4.我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

 

| **个人信息**                                                 | **用途**                                                     | **法律依据**                                                 |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **错误日志**（包括上游和下游状态、用户输入和输出，无论是否成功，云API 信息，包括 AppID、UIN、SubUIN、区域、API 输入参数和返回码、算法输出信息） | 我们将此信息用于确保服务按照要求工作，分析错误或故障原因并排除这些问题。 | 我们会处理这些信息，因为这对于我们履行与您签订的合同以便提供服务而言是必要的。 |
| **运行日志**（包括用户请求 IP、用户帐户、请求时间戳、请求接口、无论是否计费、PersonID、访问时间、识别结果） | 我们将此类信息用于计费目的。                                 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供服务而言是必要的。 |
| **面部特征：**通过算法/模型转码生成的二进制串集，无任何实际内容且具有不可逆性。这是通过收集的最终用户人脸图像生成的。 | 我们使用此信息是为了确保服务按照要求运行。                   | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供服务而言是必要的。 |

## 7.数据保留

我们将根据下列规定保留个人信息：

| **个人信息** | **保留政策**                                      |
| ------------ | ------------------------------------------------- |
| 错误日志     | 存储空间达到 2GB 后自动删除（它们一般会保存几天） |
| 运行日志     | 在服务有效期内存储，或直至您提交数据删除请求      |
| 面部特征     | 在服务有效期内存储，或直至您提交数据删除请求      |
