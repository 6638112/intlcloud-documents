To use the Tencent Cloud TI Platform, perform the following steps:

## Step 1. Sign up for a Tencent Cloud account

>?  If you already have a Tencent Cloud account, skip this step.

<div style="background-color:#0052D9; width: 340px; height: 35px; line-height:35px; text-align:center;"><a href="https://intl.cloud.tencent.com/register" target="_blank"  style="color: white; font-size:16px;">Click here for Tencent Cloud Registration</a></div>

## Step 2. Apply for access to Tencent Cloud TI Platform

Tencent Cloud TI Platform is in beta test and is open for application. Submit an application as follows: 

Click **Submit a Ticket** to go to the ticket system.

<div style="background-color:#0052D9; width: 150px; height: 35px; line-height:35px; text-align:center;"><a href="https://console.cloud.tencent.com/workorder/category" target="_blank"  style="color: white; font-size:16px;">Submit a Ticket</a></div>

1. Select the ticket type.

![](https://qcloudimg.tencent-cloud.cn/raw/f7b6ec5a17f9e591e1dc5e8502343893.png)

2. Select the issue type.

![](https://qcloudimg.tencent-cloud.cn/raw/bc0c9e4b63d82ed81acfcc850cf0fe5f.png)

3. Set information as required.

![](https://qcloudimg.tencent-cloud.cn/raw/ce2ef69bebfa482a8d9b2c5ae06bd01d.jpeg)

4. Submit the application. You can check the reply on this page.

![](https://qcloudimg.tencent-cloud.cn/raw/b9bd7697a8acd49045995323db9a7310.png)

**If you confirm from the reply that you have been granted access, you can access the console and search for Tencent Cloud TI Platform to go to its console.**
