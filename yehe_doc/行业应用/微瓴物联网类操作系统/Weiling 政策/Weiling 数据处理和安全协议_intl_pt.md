## 1.HISTÓRICO
Este Módulo se aplica se você usar o LinkBase (“**Recurso**”). Este módulo é incorporado ao contrato de segurança e privacidade de dados localizado no “**[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)**”. Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo deve ser aplicado conforme a inconsistência.
