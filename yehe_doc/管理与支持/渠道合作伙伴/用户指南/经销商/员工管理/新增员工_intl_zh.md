## 新增员工
经销商可以创建员工，并关联角色。关联角色后，新创建的员工即拥有该角色的权限。
创建员工成功后，经销商需要通过线下方式告知该员工账号名及密码。

### 操作步骤
1. [登入伙伴中心](https://www.tencentcloud.com/document/product/1085/42845)。
2. 在左侧的菜单中选**公司信息 > 员工管理**。
3. 单击**创建员工**。
>!如员工已开通子账号，但尚未录入员工信息，需补充维护员工信息并关联角色后才可继续使用该子账号（员工账号名必须和子账号名一致），该员工子账号已授权的权限需解除绑定，以免和经销商平台分权管理不一致。

![](https://staticintl.cloudcachetci.com/yehe/backend-news/sPDd712_create_user2.png) 

4. 填写员工信息，单击**提交**。
>! **员工账号名**必须唯一，且提交后无法修改，请谨慎填写。

![](https://staticintl.cloudcachetci.com/yehe/backend-news/HTe2968_create_user.png)

5. 系统提示，创建员工成功。
6. 为新增的员工分配角色。勾选角色列表中的角色名称，单击**确认**。
>!对于新增的员工，必须为其分配角色，一个员工最多可以被分配3个角色。
7. 创建员工和分配角色后，经销商通过线下方式告知员工登录信息。
| 登录信息     | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| 员工登录地址 | 复制员工登录地址进行登录，可带出主账号ID                     |
| 员工账号     | 提供员工账号名                                               |
| 员工初始密码 | 提供员工初始密码<br>如员工手机号码为 131XXXX4532<br>初始密码规则为手机号后四位与"@Tencent"组合，即初始密码为4532@Tencent |

### 员工相关操作
- 查询员工列表：可以按照特定查询条件查询员工列表。
- 编辑员工：在员工列表中单击“操作”列的“编辑”，可以编辑员工的详情。
