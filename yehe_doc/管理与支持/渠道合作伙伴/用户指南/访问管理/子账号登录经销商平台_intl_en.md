If you need to log in to the Partner Center console with a sub-account, you can select **CAM user login** on the login page. If you failed to log in, contact the root account admin.

## Directions

1. Go to the [Tencent Cloud website](https://www.tencentcloud.com/account/login).

2. Click **Log In** at the top-right corner to enter the login page.
   ![](https://qcloudimg.tencent-cloud.cn/raw/8d98f432de915726958f9e07c966d5f3.png)

3. On the login page, click **CAM user login**.
   ![](https://qcloudimg.tencent-cloud.cn/raw/2a807f0b4fb87b3d0e7a75e6b81c110c.png)

4. Enter the login information and click **Log In**.
   ![](https://qcloudimg.tencent-cloud.cn/raw/689757d64d136dbed49fe3d1b008ddc6.png)
   
   Login information description:
   | Field | Description |
   | -------------- | ------------------------------------------------------------ |
   | Root account ID | Enter the partnerâ€™s root account ID. </br>It is provided by the root account admin. |
   | Sub-user name | Enter the sub-account name. </br>It is provided by the root account admin. |
   | Password | Enter the initial password provided by the root account admin. </br>When you use a new sub-account to log in for the first time, modify the password as prompted. |

5. Click the profile photo at the top-right corner and click **Channel Management** to enter the Partner Center console.
 ![](https://qcloudimg.tencent-cloud.cn/raw/5bc5b4a31b9f920d260033267341294c.png)
