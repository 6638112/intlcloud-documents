[Consumption Details](https://console.cloud.tencent.com/expense/cost/info) shows each amortized cost as a separate entry, which is the same as the **L2-Consumption Details** file.
- You can filter the details by product, project, region, billing mode, and transaction type.
- You can search for details by instance ID or instance name.
- You can customize the fields to display in the list.
