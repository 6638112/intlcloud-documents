## Verifying Your Verification Mobile Number

1. Log in to the Tencent Cloud console and go to [Security Settings](https://console.cloud.tencent.com/developer/security). Find **Verification Mobile Number** and click the edit icon next to it.

   ![](https://qcloudimg.tencent-cloud.cn/raw/d2f5e314a7ab56576e258901f3bfadf1.png)

2. Verifying a verification mobile number is a sensitive action, for which you need to verify your identity.

   ![](https://qcloudimg.tencent-cloud.cn/raw/4bbc6ba90296f2d763440db6935e8fe7.png)

3. After identity verification, enter the verification mobile number you want to use and click **Send Verification Code**. Enter the code you receive and then click **OK**.

   ![](https://qcloudimg.tencent-cloud.cn/raw/f529c4b2d8d842f4068ec0c02d61323c.png)



## Modifying Your Verification Mobile Number

1. Log in to the Tencent Cloud console and go to [Security Settings](https://console.cloud.tencent.com/developer/security). Find **Verification Mobile Number** and click the edit icon next to it.

   ![](https://qcloudimg.tencent-cloud.cn/raw/1eaf64d0724edab5853d07662558e91b.png)

2. Verify your identity.

   ![](https://qcloudimg.tencent-cloud.cn/raw/a90aeab32fdfb6e8482551beb44a87b9.png)

3. After identity verification, enter the new verification mobile number you want to use and click **Send Verification Code**. Enter the code you receive and then click **OK**.

   ![](https://qcloudimg.tencent-cloud.cn/raw/9e410be3b8008c05a9c70b9faee504ad.png)

4. The modification is successful.

   ![](https://qcloudimg.tencent-cloud.cn/raw/8a28a25247bdcbac21a3efb2f8afc388.png)
