## 操作场景
本文将指导您在忘记腾讯云账号的情况下，通过账号 ID、手机等信息找回账号。 


## 操作步骤
如果您不慎遗忘了您的腾讯云账号，可以通过以下几种方式找回账号：

### 通过手机号找回
如果您的腾讯云账号已绑定了安全手机号，并且该手机号仍能正常使用，您可以通过使用手机接收验证码找回腾讯云账号。
前往 [查找账号](https://intl.cloud.tencent.com/services/forgotAccount) 页面，单击【手机号】，输入与腾讯云账号绑定的手机号，单击【发送】，验证完成后，系统核实该账号存在，则将账号信息发送至与腾讯云账号绑定的手机号上，请注意查收。
![](https://main.qcloudimg.com/raw/e66c1e0da893569b61b4646dd2d409fb.png)


### 通过账号 ID 找回
如果您还记得腾讯云账号ID，您可以通过账号 ID 找回腾讯云账号。
前往 [查找账号](https://intl.cloud.tencent.com/services/forgotAccount) 页面，单击【账号ID】，输入您腾讯云账号 ID，单击【下一步】。
![](https://main.qcloudimg.com/raw/a82073a0c9df9f9b2d4f9780f87f74ac.png)
验证完成后，确认接收信息的手机号，并输入完整手机号码，账号信息将发送至与腾讯云账号绑定的手机号上，请注意查收。


### 通过服务器 IP 找回
如果您曾用该腾讯云账号购买或管理云服务器并记得该服务器公网 IP，您可以通过服务器 IP找回腾讯云账号。
前往 [查找账号](https://intl.cloud.tencent.com/services/forgotAccount) 页面，单击【服务器IP】，输入您的服务器IP，单击【下一步】。
![](https://main.qcloudimg.com/raw/a3a6eacba46f6b8d5ccccc028fe998fb.png)
验证完成后，确认接收信息的手机号，并输入完整手机号码，账号信息将发送至与腾讯云账号绑定的手机号上，请注意查收。
