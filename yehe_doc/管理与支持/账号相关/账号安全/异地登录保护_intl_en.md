## Overview
Suspicious login refers to a login in a place other than the frequent login city. After suspicious login protection is enabled, the system will check whether the login IP address is in the frequent place and require identity verification if not.

## Directions
### Enabling suspicious login protection

1. Log in to the Tencent Cloud console and go to the [Security Settings](https://console.tencentcloud.com/developer/security) page.

2. In the **Suspicious Login Protection** section, set **Suspicious Login Verification** to ![](https://qcloudimg.tencent-cloud.cn/raw/e7d72436cc2974a594c9aae4699d6eac/%E5%BC%80%E5%90%AF.png).

   ![](https://qcloudimg.tencent-cloud.cn/raw/0814a91a0cbd5ce04566b7c285af73b5/suspicious%20login%20protection.png)

### Disabling suspicious login protection

1. Log in to the Tencent Cloud console and go to the [Security Settings](https://console.tencentcloud.com/developer/security) page.

2. In the **Suspicious Login Protection** section, set **Suspicious Login Verification** to ![](https://qcloudimg.tencent-cloud.cn/raw/aa85d1e0722868cce322b25d77ed928d/%E5%85%B3%E9%97%AD.png).

   ![](https://qcloudimg.tencent-cloud.cn/raw/cfad062a8a2c12bcdb1ed3a633032a26/suspicious%20login%20protection-%20close.png)
