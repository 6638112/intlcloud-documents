登录保护指在您进行登录操作的时候，腾讯云会给您增加一层保护。通常情况下，该保护的措施是在正确输入账号和密码的前提下，还需要额外输入一种能证明身份的凭证。

开启登录保护后，在登录腾讯云官方网站时需要验证身份。这样即使他人盗取您的密码，也无法登录您的账号，能够最大限度地保证您的账号安全。

## 登录保护类型
| 登录保护类型      | 作用                                                         |
| ----------------- | ------------------------------------------------------------ |
| 不开启            | 不进行二次验证，输入账号密码即可登录。                       |
| 启用 MFA 设备校验 | 输入账号密码并校验成功后后，需再输入正确的 MFA 密码才可完成登录。 |
| 开启邮箱校验      | 输入账号密码并校验成功后后，需再获取邮箱验证码并输入才可完成登录。 |
| 开启手机校验      | 输入账号密码并校验成功后后，需再获取手机验证码并输入才可完成登录。 |

## 开启登录保护

1. 登录腾讯云控制台，进入 [安全设置](https://console.tencentcloud.com/developer/security) 页面，单击登录保护右侧的图标按钮。
   ![](https://qcloudimg.tencent-cloud.cn/raw/d376297806543a247e5d63ec576ba21c.png)

2. 根据页面提示，完成身份验证。

3. 通过身份验证后，选择合适的保护类型即成功开启登录保护。

   ![](https://qcloudimg.tencent-cloud.cn/raw/187a483c62aba15145c7cc4437be2375.png)
