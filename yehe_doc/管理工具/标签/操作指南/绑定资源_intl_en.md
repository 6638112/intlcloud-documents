## Overview

This document describes how to tag resources in the console so that you can manage your resources centrally and by category.

## Prerequisites
You have created a tag. For detailed instructions, please see [Creating Tags](https://intl.cloud.tencent.com/document/product/651/41684).

## Directions

1. Log in to the [Tag console](https://console.cloud.tencent.com/tag).
2. Click **Tag List** on the left sidebar.
3. Find the key-value pair you created and click **Bind Resources** in the **Operation** column.
![](https://main.qcloudimg.com/raw/b25635adadff0b765549f8d33e689bfe.png)
4. On the **Bind Resources** page, select a Tencent Cloud service and resource type and choose the resources to tag.
![](https://main.qcloudimg.com/raw/c854c2779ea7eb2aba4bda2baf3b5402.png)
5. Click **OK** to tag the resources.
