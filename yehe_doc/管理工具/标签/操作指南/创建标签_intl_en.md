## Overview

If there are resources of multiple Tencent Cloud services, e.g., CVM instances, CBS instances, and COS buckets under your account, you can create tags and bind them to your resources, so that you can manage the resources by category.



## Directions

1. Log in to the [Tag console](https://console.cloud.tencent.com/tag).
2. Click **Tag List** on the left sidebar.
3. Click **Create Tag** and, in the dialog box that pops up, create a tag or add a tag value for an existing tag key.
	<dx-tabs>
	::: Create Tag
	Click **Add tag key** and enter a tag key and tag value.
	![](https://main.qcloudimg.com/raw/9e8232b5249cb7e96e720c0770d39a90.png)
	:::
	::: Add Tag value
	Select an existing tag key and enter a new tag value.
	![](https://main.qcloudimg.com/raw/1ad02aed3636afd201e92ed95b39c33a.png)
	:::
	</dx-tabs>

 - Tag key (required): A tag key is used to mark a tag. It must contain 1-127 characters. Letters, digits, Chinese characters, and special characters (=._:/@()[]（）【】) are allowed.
 - Tag value (required): A tag value is the content of a tag. It must contain 1-255 characters. Letters, digits, Chinese characters, and special characters (+-=._:/@()[]（)【】) are allowed.
4. Click **OK** to complete the creation.
