## Overview
If you no longer need to categorize, search for, or aggregate cloud resources by tag, you can delete tags in the console.
>!
>- Double-check before you delete tags in batches.
>- The operation cannot be undone.


## Prerequisites
- You have created a tag. For detailed instructions, please see [Creating Tags](https://intl.cloud.tencent.com/document/product/651/41684).
- If a tag has been bound to resources, you need to [unbind](https://intl.cloud.tencent.com/document/product/651/41686) it first. Note that **you cannot delete a tag until 5 minutes after unbinding it**.

## Directions
You can delete one tag at a time or multiple tags in batches using the methods below.
<dx-tabs>
::: Delete Single Tag
1. Log in to the [Tag console](https://console.cloud.tencent.com/tag).
2. Click **Tag List** on the left sidebar.
3. Find the tag you want to delete and click **Delete** in the **Operation** column.
![](https://main.qcloudimg.com/raw/9ea2314c7c11dd682c2437cbe4121731.png)
:::
::: Delete Multiple Tags
1. Log in to the [Tag console](https://console.cloud.tencent.com/tag).
2. Click **Tag List** on the left sidebar.
3. Select the tags you want to delete and click **Delete** in the top left.
![](https://main.qcloudimg.com/raw/2cf4489302511a569a1db05599483f29.png)
:::
</dx-tabs>
