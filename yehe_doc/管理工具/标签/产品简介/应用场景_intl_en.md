Below are the main applications for Tencent Cloud Tag.

## Managing Tencent Cloud resources
You can use tags to mark and categorize your existing Tencent Cloud resources. For detailed instructions, please see [Creating Tags and Binding Resources](https://intl.cloud.tencent.com/document/product/651/41575). You can query resources by region, resource type, or tag via the console or using tag APIs. The results will be displayed in a list. For details, please see [Viewing Tagged Resources](https://intl.cloud.tencent.com/document/product/651/41688).


## Managing Access to Resources
By using the tagging service together with CAM, you can grant CAM users different access to cloud resources.
