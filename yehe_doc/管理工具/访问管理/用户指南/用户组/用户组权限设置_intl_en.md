## Overview
This document describes how to associate/disassociate a policy with/from a user group. Sub-accounts in the group can manage the resources under the root account within the scope of the granted permissions.
## Prerequisites
Log in to the CAM Console and enter [User Groups](https://console.cloud.tencent.com/cam/groups).
## Directions
### Associating a policy with a user group
1. On the user group management page, click the name of a user group to enter the user group details page.
2. Go to **Permissions**.
3. Click **Associate Policies**.
4. Select one or more policies to be associated in the pop-up window and click **OK** to associate the policies with the user group.

### Disassociating a policy from a user group
1. On the user group management page, click the name of a user group to enter the user group details page.
2. Go to **Permissions**.
3. Locate the policy and click **Disassociate** on the right side.
3. Click **OK** to disassociate the policy from the user group.
