## Introduction
This document describes how to add/delete one or multiple users to/from a user group.
## Prerequisites
Log in to the CAM Console and enter the [User Group](https://console.cloud.tencent.com/cam/groups) page.
## Directions
### Adding a user to a user group
#### Adding a user to one user group
1. On the user group page, locate the user group to which you want to add the user.
2. Click **Add User** in the "Operation" column on the right.
3. In the pop-up window, check the user to be added.
4. Click **OK** to add the user to the user group.

#### Add a user to multiple user groups
1. On the user group page, select the user groups to which you want to add the user.
2. Click **Add User** in the top-left corner.
3. In the pop-up window, check the user to be added.
4. Click **OK** to add the user to the user groups.

### Deleting a user from user group
#### Deleting one user from a user group
1. On the user group page, locate the user group from which you want to delete the user.
2. Click the user group name to enter the user group details page.
3. On the user group details page, click **User** tab.
4. Locate the user to be deleted and click **Remove from Group** in the "Operation" column on the right.
5. Click **OK** to delete the user from the user group.

#### Deleting multiple users from a user group
1. On the user group page, locate the user group from which you want to delete multiple users.
2. Click the user group name to enter the user group details page.
3. On the user group details page, click **User** tab.
4. Select the users to be deleted.
5. Click **Remove User** > **OK** to delete the selected users from the user group.
