Cloud Monitor provides the following monitoring metrics for VPN tunnels:

| Metric | Description | Unit | Dimension |
| ----- | ------------ | ---- | ----- |
| Network outbound bandwidth | Average outbound traffic per second of VPN tunnel | Mbps | VPN tunnel |
| Network inbound bandwidth | Average inbound traffic per second of VPN tunnel | Mbps | VPN tunnel |
| Outbound packets | Average outbound packets per second of VPN tunnel | Packets/second | VPN tunnel |
| Inbound packets | Average inbound packets per second of VPN tunnel | Packets/second | VPN tunnel |
| Packet loss rate | Packet loss rate monitored by VPN in one minute | % | VPN tunnel |
| Latency | Average latency monitored by VPN in one minute | ms | VPN tunnel |
| Tunnel connectivity | Real-time connection status of VPN tunnel | 0/1 | VPN tunnel |
