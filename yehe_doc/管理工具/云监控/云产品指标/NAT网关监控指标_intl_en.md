Cloud Monitor provides the following monitoring metrics for NAT gateways:

| Metric | Description | Unit | Dimension |
| ----- | ------------ | ---- | ----- |
| Public network outbound bandwidth | Average outbound traffic per second of NAT gateway | Mbps | NAT gateway |
| Public network inbound bandwidth | Average inbound traffic per second of NAT gateway | Mbps | NAT gateway |
| Outbound packets | Average outbound packets per second of NAT gateway | Packets/second | NAT gateway |
| Inbound packets | Average inbound packets per second of NAT gateway | Packets/second | NAT gateway |
| Connections | Number of real-time concurrent connections of NAT gateway | Connections/second | NAT gateway |
