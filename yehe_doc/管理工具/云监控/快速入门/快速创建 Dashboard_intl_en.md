This document describes how to quickly create a dashboard.

## Feature Overview

Dashboards provide users with a custom monitoring dashboard service, which allows them to customize basic metrics and monitoring metrics across products and instances in order to display the resources of a business or cluster in one place.

## Preparations

1. Log in to the [Cloud Monitor console](https://console.cloud.tencent.com/monitor).
2. In the left sidebar, click **Dashboard** > **Dashboard List** to go to the dashboard list page.

## Step 1: Create a Dashboard

1. On the dashboard list page, click **Create** in the upper-left corner to go to the management page for creating a dashboard.
2. Click the dashboard area **<img src="https://main.qcloudimg.com/raw/4faa1888e40f4ae2f153ce82163988a7.png"  style="margin:0;" width="3%">**, enter a dashboard name, and click **OK** to finish creating the dashboard.
   ![](https://main.qcloudimg.com/raw/b337d36b8fd7554165c86e5534047597.png)

## Step 2: Create a Monitoring Chart

1. Click the dashboard that you created in Step 1 to go to the dashboard management page.
2. Click **Create Chart** to go to the chart editing page.
3. Edit the metric information below the chart, specify the chart name on the right, and then click **<img src="https://main.qcloudimg.com/raw/4faa1888e40f4ae2f153ce82163988a7.png"  style="margin:0;" width="3%">** > **OK**.
![](https://main.qcloudimg.com/raw/e10740c568b8636fa35ac3c87f69f60b.png)

## Step 3: View the Monitoring Chart

After creating the monitoring chart, you can view it on the dashboard management page.
 ![](https://main.qcloudimg.com/raw/d413687013c978d3f31855b8119aad37.png)
