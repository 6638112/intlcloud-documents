This document describes how to delete a monitoring chart.

## Directions
1. Log in to the [Cloud Monitor console](https://console.cloud.tencent.com/monitor).
2. In the left sidebar, click **Dashboard** > **Dashboard List** to go to the dashboard list page.
3. In the dashboard list, find the dashboard from which you want to delete a chart, and click the dashboard name to go to the dashboard management page.
4. Find the chart that you want to delete, and click **<img src="https://main.qcloudimg.com/raw/3bbe823981cbb9f1cebbbf55f85f83fc.png"  style="margin:0;" width="3%">** > **Delete**.
5. Click **Delete Now** in the window that appears.
