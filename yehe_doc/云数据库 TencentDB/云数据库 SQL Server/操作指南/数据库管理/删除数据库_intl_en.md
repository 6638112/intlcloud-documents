## Operation Scenarios
TencentDB for SQL Server supports deleting databases on the **Manage Database** tab in the console.
>Batch database deletion is not supported currently.

## Directions
1. Log in to the [TencentDB for SQL Server Console](https://console.cloud.tencent.com/sqlserver). In the instance list, click an instance name or **Manage** in the "Operation" column to enter the instance management page.
2. Select the **Manage Database** tab, select the database to be deleted in the list, and click **Delete** in the "Operation" column.
>Before deleting the database, please make sure that there are no active connections using the database. If any active connection is detected during the deletion process, the deletion operation will be suspended.
>
![](https://main.qcloudimg.com/raw/d107d30fef31744a3d01a445fa13a267.png)
3. In the pop-up dialog box, click **Delete**.
