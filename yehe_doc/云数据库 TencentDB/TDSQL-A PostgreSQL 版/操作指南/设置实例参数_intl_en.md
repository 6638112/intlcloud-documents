This document describes how to view and modify certain parameters and query parameter modification records in the TDSQL-A for PostgreSQL console.

>!To ensure the instance stability, the console allows you to modify certain parameters only as displayed on the parameter configuration page in the console.

## Batch Modifying Parameters
1. Log in to the [TDSQL-A for PostgreSQL](https://console.cloud.tencent.com/tdsqla/tdapg) console and click an instance ID in the instance list to enter the instance management page.
2. On the instance management page, select the **Parameter Configuration** tab and click **Batch Modify Parameters**.
![](https://main.qcloudimg.com/raw/9c967affb9297711308122af080cb323.png)
3. Enter the target parameter value as prompted in the **Acceptable Values** column and click **Confirm** to save the change. You can click **Cancel** to cancel the operation.
>? CN and DN nodes can be modified separately.
>
![](https://main.qcloudimg.com/raw/dfd50e8f9d8888f7b479f84d8a84e9a6.png)


## Viewing Parameter Modification Record
1. Log in to the [TDSQL-A for PostgreSQL](https://console.cloud.tencent.com/tdsqla/tdapg) console and click an instance ID in the instance list to enter the instance management page.
2. On the instance management page, select **Parameter Configuration** > **Recent Modification Records** to view recent parameter modifications.
 ![](https://main.qcloudimg.com/raw/ff3401183f63b2b73ae6a5a1475aab48.png)
