### Notes
TencentDB for MongoDB provides two usernames `rwuser` and `mongouser` by default to support the MONGODB-CR and SCRAM-SHA-1 authentication methods, respectively. The connecting URIs for the two authentication methods are formed differently. For more information, please see [Connection Sample](https://intl.intl.cloud.tencent.com/document/product/240/7092).

In PHP, [two sets of drivers](https://docs.mongodb.com/ecosystem/drivers/php/) can be used to connect to and operate on MongoDB:
- mongodb ([PHP official document](http://php.net/manual/en/set.mongodb.php)): it is officially recommended by MongoDB, but PHP v5.4 or above is required.
- mongo ([PHP official document](http://php.net/manual/en/book.mongo.php)): it is older than mongodb. Use v1.6 if necessary.

Both drivers are used below to demonstrate the process of connecting to TencentDB for MongoDB and perform read/write operations.

### Using the mongodb driver
For more information on how to install mongodb, please see the [installation steps on the official website](http://php.net/manual/zh/mongodb.installation.php).
**The mongodb driver supports both MONGODB-CR and SCRAM-SHA-1 authentication**. For more information, please see [Connection Sample](https://intl.cloud.tencent.com/document/product/240/7092).

Sample code:
```
<?php
// Form the connection URI
$uri = 'mongodb://mongouser:thepasswordA1@10.66.187.127:27017/admin';
$manager = new MongoDB\Driver\Manager($uri);

// Prepare to write data
$document1 = [
    'username' => 'lily',
    'age'      => 34,
    'email'    => 'lily@qq.com'
];

// The driver preprocesses the data. You can see that the `_id` of MongoDB is generated by the driver.
$bulk = new MongoDB\Driver\BulkWrite;
$_id1 = $bulk->insert($document1);

$result = $manager->executeBulkWrite('tsdb.table1', $bulk);

// Or, include the following code if necessary to ensure data is written to most nodes
// $writeConcern = new MongoDB\Driver\WriteConcern(MongoDB\Driver\WriteConcern::MAJORITY, 1000);
// $result = $manager->executeBulkWrite('testdb.testcollection', $bulk, $writeConcern);

// Query
$filter = ['_id' => $_id1];
$query = new MongoDB\Driver\Query($filter);
$rows = $manager->executeQuery('tsdb.table1', $query); // You can also choose to read from the slave database first. For more information, please see the documentation
foreach($rows as $r){
   print_r($r);
}


```
Output:
```
stdClass Object
(
    [_id] => MongoDB\BSON\ObjectID Object
        (
            [oid] => 582c001618c90a16363abc31
        )

    [username] => lily
    [age] => 34
    [email] => lily@qq.com
)
```


### Using the mongo driver
**The mongo driver only supports MONGODB-CR authentication** and thus connects to the database only through `rwuser`. For more information, please see [Connection Sample](https://intl.intl.cloud.tencent.com/document/product/240/7092).

Sample code:

```
<?php
// It is recommended to use URI for connection. Either of the following two URIs is OK
$uri = "mongodb://rwuser:thepasswordA1@10.66.187.127:27017/admin?authMechanism=MONGODB-CR";
$uri = "mongodb://rwuser:thepasswordA1@10.66.187.127:27017/?authMechanism=MONGODB-CR&authSource=admin";
$connection = new MongoClient($uri);

/*
// Or, you can use the following code
$connection = new MongoClient("mongodb://10.66.116.103:27017/admin",
    array(
        "username" => "rwuser",
        "password" => "password",
        "authMechanism" => "MONGODB-CR"
    )
);
*/
$db = $connection->tsdb;
$collection = $db->table1;

$q = array(
    'id' => 1,
    'test1' => 'xxx',
    'ss' => 'xxxxxxxx',
);
$collection->save($q);
$one = $collection->findOne();
var_dump($one);
```

### Using the PHPLIB library (encapsulated based on the mongodb driver)
It is recommended to use the mongodb driver together with [PHPLIB](http://php.net/manual/zh/mongodb.tutorial.library.php). For more information, please see the [relevant documentation](http://mongodb.github.io/mongo-php-library/tutorial/crud/).
For more information on how to install PHPLIB, please see the [installation steps on the official website](http://mongodb.github.io/mongo-php-library/getting-started/). Note that PHPLIB depends on the mongodb driver.

Sample code:
```
<?php
require_once __DIR__ . "/vendor/autoload.php";

// Initialize
$mongoClient = new MongoDB\Client('mongodb://mongouser:thepasswordA1@10.66.187.127:27017/admin');

// Use the `users` collection in the `demo` database
$collection = $mongoClient->demo->users;

// Write a data entry
$insertOneResult = $collection->insertOne(['name' => 'gomez']);

printf("Inserted %d document(s)\n", $insertOneResult->getInsertedCount());
var_dump($insertOneResult->getInsertedId());

// Query data
$document = $collection->findOne(['name' => 'gomez']);

var_dump($document);

```
