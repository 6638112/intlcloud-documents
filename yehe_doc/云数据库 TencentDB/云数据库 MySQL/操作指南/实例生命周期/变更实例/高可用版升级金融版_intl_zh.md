## 操作场景
本文为您介绍通过 MySQL 控制台升级 [高可用版](https://intl.cloud.tencent.com/document/product/236/17136) 实例至 [金融版](https://intl.cloud.tencent.com/document/product/236/17136) 实例的操作。
>
>- 仅支持 MySQL 5.6、5.7 高可用版实例升级至金融版实例。
>- 升级过程不会影响实例的正常使用。

## 操作步骤
1. 登录 [MySQL 控制台](https://console.cloud.tencent.com/cdb)，在实例列表中，单击实例名或“操作”列的【管理】，进入详情页。
2. 在详情页的“配置信息”处，单击【升级为金融版】。
![](https://main.qcloudimg.com/raw/3317539eac412f04156b1a15b8a5d100.png)
3. 在弹出的页面，确认无误后，单击【升级】。
![](https://main.qcloudimg.com/raw/5b5a5a4790e9c274a695c21c0d6facb5.png)
4. 确认价格后，返回实例列表，实例状态为“配置变更中”，待状态变为“运行中”，控制台功能即可正常使用。
