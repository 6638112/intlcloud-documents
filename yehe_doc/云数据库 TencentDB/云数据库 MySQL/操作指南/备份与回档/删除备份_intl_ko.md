
데이터베이스 백업을 삭제해 백업 용량 비용을 절약할 수 있습니다.
>?자동 백업은 직접 삭제할 수 없으며, 수동 백업만 백업 리스트에서 직접 삭제할 수 있습니다.


## 작업 순서
1. [MySQL 콘솔](https://console.cloud.tencent.com/cdb)에 로그인한 뒤, 인스턴스 리스트에서 인스턴스 이름을 클릭하여 인스턴스 관리 페이지로 이동합니다.
2. 인스턴스 관리 페이지에서 [Backup and Restore] 페이지를 선택하고, Backup List의 'Operation' 열에서 [삭제]를 클릭합니다.
![](https://main.qcloudimg.com/raw/32ddb8d2ad17032351d44d149c8b5033.png)
3. 팝업된 대화 상자에서 삭제할 백업 파일을 확인하고 [OK]를 클릭합니다.
>!삭제한 후에는 백업 파일을 복구할 수 없습니다.
