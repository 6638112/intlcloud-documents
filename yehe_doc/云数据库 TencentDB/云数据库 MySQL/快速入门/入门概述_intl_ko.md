본 문서는 사용자의 빠른 이해를 돕기 위해 TencentDB for MySQL의 전체적인 사용 방법에 대해 소개합니다. 인스턴스의 생성부터 기본적인 사용 방법을 알기 위해서는 아래의 작업을 완료해야 합니다.

### 1. MySQL 인스턴스 생성
TencentDB for MySQL 콘솔을 통해 여러 과금 방식 및 다양한 구성의 MySQL 인스턴스를 효율적으로 생성할 수 있습니다.

### 2. MySQL 인스턴스 초기화
MySQL 인스턴스를 생성한 후에는 MySQL 인스턴스의 초기화 작업을 통해 손쉽게 인스턴스를 실행할 수 있습니다. 자세한 내용은 [MySQL 인스턴스 초기화](https://intl.cloud.tencent.com/document/product/236/3128)를 참조 바랍니다.

### 3. MySQL 인스턴스 연결
MySQL 인스턴스를 초기화한 후, 다양한 방법을 통해 MySQL 인스턴스에 연결할 수 있으며, 연결한 다음에는 데이터베이스에 대한 갖가지 관리 작업을 수행할 수 있습니다. 자세한 내용은 [MySQL 인스턴스 연결](https://intl.cloud.tencent.com/document/product/236/3130)을 참조 바랍니다.
