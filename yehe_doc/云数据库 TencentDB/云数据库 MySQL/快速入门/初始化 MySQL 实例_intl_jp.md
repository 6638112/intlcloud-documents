TencentDB for MySQLインスタンスを作成した後、MySQLインスタンスを初期化して、インスタンスを簡単に有効にする必要もあります。

### 操作手順
1. [MySQLコンソール](https://console.cloud.tencent.com/cdb)にログインし、対応するリージョンを選択した後、インスタンスリストでステータスが「未初期化」のインスタンスを選択し、「操作」列の【初期化】をクリックします。
![](https://main.qcloudimg.com/raw/945a4e69bef68eb706a520d4cbac13cf.png)
2. 表示された初期化ダイアログで、初期化関連パラメータを設定し、【OK】をクリックします。
 - **サポートされる文字セット**：LATIN1、GBK、UTF8、UTF8MB4文字セットがサポートされます。デフォルト値はUTF8です。インスタンスを初期化した後、コンソールのインスタンス詳細ページで文字セットを変更することもできます。詳細については、[文字セット](https://intl.cloud.tencent.com/document/product/236/7259)をご参照ください。
 - **テーブル名の大文字と小文字の区別**：テーブル名の大文字と小文字が区別されるか。デフォルトでは大文字と小文字が区別されます。
 - **カスタムポート*：データベースのアクセスポート。デフォルトは3306です。
 - **rootアカウントのパスワードの設定**：新規作成されたMySQLデータベースのユーザー名はデフォルトでrootです。ここでは、rootアカウントのパスワードを設定します。
 - **パスワードの確認**：パスワードを再入力します。
3. インスタンスリストに戻り、インスタンスのステータスが「実行中」に変わると、インスタンスは正常に使用できます。


## 後続の操作
Windows CVMインスタンスまたはLinux CVMインスタンスを使用して、パブリックネットワークとプライベートネットワークの2つの異なる方法でTencentDB for MySQLにアクセスできます。詳細については、[MySQLインスタンスへの接続](https://intl.cloud.tencent.com/document/product/236/37788)をご参照ください。
