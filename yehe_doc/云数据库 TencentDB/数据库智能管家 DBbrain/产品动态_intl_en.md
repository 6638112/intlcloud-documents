<table>
<tr>
<th width=20%>Update</th>

<tbody><tr>
<td>Audit log analysis (formerly SQL insight) is supported</td>
<td>The audit log analysis feature can carry out in-depth SQL analysis for a database instance. It can statistically analyze, sample, and aggregate all SQL statements and their execution information based on the audit logs generated by the database within a specified time period. <br>It analyzes the SQL performance based on the execution plan result, comprehensive resource consumption, sizes of scan set and result set, and index usage rationality of the aggregated SQL statements. Then, it provides optimization advice for poorly performing SQL statements by taking into account the index conditions and database/table design.</td>
<td>August 2019</td>
<td><a href="https://intl.cloud.tencent.com/document/product/1035/36041" target="_blank">Audit Log Analysis</a></td>
</tr>
<tr>
<td>Exception diagnosis is supported</td>
<td>The exception diagnosis feature has real-time and historical view modes to provide you with real-time performance monitoring, health checks, and failure diagnosis and optimization, so that you can intuitively know the real-time operation status of database instances, locate newly appeared performance exceptions in real time, and optimize the system based on the optimization advice.</td>
<td>August 2019</td>
<td><a href="https://intl.cloud.tencent.com/document/product/1035/36036" target="_blank">Exception Diagnosis</a></td>
</tr>
<tr>
<td>Slow SQL analysis is supported</td>
<td>The slow SQL analysis feature calculates, samples, and aggregates records and execution information (source information, number of executions, execution duration, etc.) of slow SQL statements on the instance. <br>It analyzes the performance of slow SQL statements based on the execution plan, comprehensive resource consumption, sizes of scan and result sets, and index usage rationality of the aggregated SQL statements and provides optimization advice.</td>
<td>August 2019</td>
<td><a href="https://intl.cloud.tencent.com/document/product/1035/36038" target="_blank">Slow SQL Analysis</a></td>
</tr>
<tr>
<td>Disk space analysis is supported</td>
<td>The space analysis feature allows you to view the utilization of instance space, including data and log spaces, daily increase in space utilization, estimated number of available days, and spaces used by tablespaces under the instance.</td>
<td>August 2019</td>
<td><a href="https://intl.cloud.tencent.com/document/product/1035/36039" target="_blank">Space Analysis</a></td>
