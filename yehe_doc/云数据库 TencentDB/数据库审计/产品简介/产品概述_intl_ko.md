Tencent Cloud가 자체 연구 개발한 데이터베이스 감사는 데이터베이스 보안을 효율적이고 전면적으로 실시간 모니터링할 수 있는 전문 감사 제품입니다. 데이터베이스 감사를 통해 실시간으로 Tencent CDB의 이벤트를 기록하고 데이터베이스 작업을 디테일하게 감사하여 컴플라이언스 관리를 돕습니다. 데이터베이스의 리스크에 대해 알람을 발송하고 SQL 인젝션, 작업 오류 등을 기록하여 고객의 CDB에 완벽한 보안 진단과 관리 기능을 제공하고 데이터 자산 보안 등급 향상에 기여합니다.

데이터베이스 감사 기능은 다음과 같은 리스크 대응에 도움이 됩니다.

- **감사 리스크**
감사 로그가 완벽하지 않으면 보안 사고 발생 시 원인 분석이 어렵습니다.  
국가 보호 등급 기준 (3급)에 미치지 못합니다. 
업계 정보 보안 컴플라이언스 파일 기준에 미치지 못합니다. 

- **관리 리스크**
엔지니어의 조작 미숙, 불법ㆍ월권 작업 행위는 업무 시스템의 안전한 운영을 위협합니다. 
3rd party 개발 유지보수 인력의 조작 미숙, 악의적인 조작 및 수정
슈퍼 관리자의 권한이 과도할 경우, 감사 모니터링을 할 수 없습니다. 

- **기술적 문제점**
데이터베이스 시스템 SQL 인젝션 공격을 통해 악의적으로 DB 테이블 정보를 탈취합니다. 
슬로우 로그가 아닌 다량의 데이터 베이스 요청이 갑자기 발생하면 신속한 원인 분석이 어렵습니다.
