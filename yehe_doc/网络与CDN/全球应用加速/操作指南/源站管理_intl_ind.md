## Menambahkan server asal
Login ke [Konsol Global Application Acceleration](https://console.cloud.tencent.com/gaap). Pada halaman "Origin Server Management" (Manajemen Server Asal), klik [Add] (Tambahkan) untuk menambahkan informasi tentang semua server yang memerlukan akselerasi akses. Anda dapat memasukkan IP station sumber atau nama domain, dan memisahkan beberapa server asal dengan tombol Enter.
![](https://main.qcloudimg.com/raw/d3a80af94e4dded6dfdca9c0361c6954.png)

## Menghapus server asal 
Masuk ke [Konsol Global Application Acceleration](https://console.cloud.tencent.com/gaap). Pada halaman "Origin Server Management" (Manajemen Server Asal), pilih server asal yang akan dihapus dan klik [Delete] (Hapus).
![](https://main.qcloudimg.com/raw/60f772f8cc3cf43c9be4fc1545962cf5.png)
>! Tautan server asal harus dilepaskan dari pendengar terlebih dahulu agar dapat dihapus.
