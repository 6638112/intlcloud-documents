## Menambahkan Server Asal
Login ke [konsol GAAP](https://console.cloud.tencent.com/gaap). Di halaman "Origin Server Management" (Manajemen Server Asal), klik **Add** (Tambahkan) untuk menambahkan informasi tentang semua server yang memerlukan akselerasi akses. Anda dapat memasukkan IP server asal atau nama domain, dan memisahkan beberapa server asal dengan tombol Enter. IP server asal mendukung alamat IPv4 dan IPv6.
 ![](https://main.qcloudimg.com/raw/d3a80af94e4dded6dfdca9c0361c6954.png)

## Menghapus Server Asal
Login ke konsol GAAP. Di halaman **Origin Server Management** (Manajemen Server Asal), pilih server asal yang akan dihapus, lalu klik **Delete** (Hapus).
>!Jika server asal yang akan dihapus terikat ke koneksi yang ada, lepaskan ikatan sebelum server asal dihapus.
> 
![](https://main.qcloudimg.com/raw/60f772f8cc3cf43c9be4fc1545962cf5.png)

## Memodifikasi Nama
1. Login ke konsol GAAP. Di halaman **Origin Server Management** (Manajemen Server Asal), klik ikon edit di sebelah kanan nama server asal untuk memodifikasinya.
![](https://main.qcloudimg.com/raw/4c910b3ba40999f31e2772a0219b4992.png)
2. Di kotak dialog pop-up, masukkan nama baru dan klik **OK** (Oke).
 ![](https://main.qcloudimg.com/raw/df44e0e6d652ed0c1e584ec37f43dcfe.png)

## Melihat Status Kesehatan
1. Login ke [konsol GAAP](https://console.cloud.tencent.com/gaap). Di halaman **Origin Server Management** (Manajemen Server Asal), klik ikon di sebelah kanan server asal yang sehat.
>!Fitur ini tidak tersedia jika server asal tidak terikat ke pendengar.
>
![](https://qcloudimg.tencent-cloud.cn/raw/5d987f08bf034cb6cd1de5d7aad891d5.jpg)
2. Di jendela pop-up di sudut kanan atas, Anda dapat melihat status kesehatan server asal selama periode dan perincian waktu yang berbeda. `1` berarti server asal normal dan `0` berarti server tidak normal.
<img src="https://qcloudimg.tencent-cloud.cn/raw/6317c3ccdb5cd1442a1cf4e0105c9896.jpg" width="70%">

 

## Mengedit Tag
1. Login ke [konsol GAAP](https://console.cloud.tencent.com/gaap). Di halaman **Origin Server Management** (Manajemen Server Asal), klik **Edit Tag** di sebelah kanan server asal.
 ![](https://qcloudimg.tencent-cloud.cn/raw/9f10b4b55e4601c3ad6bf602bee99c83.jpg)
2. Pilih tag untuk mengategorikan dan mengelola server asal dalam banyak dimensi. Klik **OK** (Oke).
 ![](https://qcloudimg.tencent-cloud.cn/raw/74527ba60e680b59f6cd0a7ccc23feac.jpg)
