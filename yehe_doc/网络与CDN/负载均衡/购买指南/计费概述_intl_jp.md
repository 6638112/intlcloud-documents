ここではCloud Load Balancer（CLB）の課金説明についてご紹介します。


## 料金の構成
Cloud Load Balancer（CLB）の料金は、CLBインスタンス料金、パブリックネットワーク料金、クロスリージョンバインディング料金、ロードバランサキャパシティユニット（LCU）料金の4つの部分から成ります。
<dx-alert infotype="explain" title="">
- LCU料金は「LCUタイプ」のCLBインスタンスにのみ発生します。
- 現在、Tencent Cloudアカウントには標準アカウントタイプと従来型アカウントタイプがあり、2020年6月17日0時以降に登録したアカウントはすべて標準アカウントタイプとなります。この時点より前に登録したユーザーは、<a href="https://console.cloud.tencent.com/cvm/eip">パブリックIPコンソール</a>のインスタンスリストページ上部で、アカウントタイプをご確認ください。
- ハードウェア専有インスタンスの使用をご希望の場合は、担当のビジネスマネージャーまでご連絡ください。ハードウェア専有インスタンスは3か月から購入できます。

## 標準アカウントタイプ課金説明
+ プライベートネットワークCLBにはパブリックネットワーク料金はかかりませんが、インスタンス料金は発生します（詳細な説明は[CLBインスタンスのアップグレードに伴う価格調整のお知らせ](https://intl.cloud.tencent.com/zh/document/product/214/41565)をご参照ください）。
+ パブリックネットワークCLBにはCLBインスタンス料金とパブリックネットワーク料金がかかります。
+ パブリックネットワークCLBで<a href="https://intl.cloud.tencent.com/zh/document/product/214/38441"> クロスリージョンバインディング2.0</a>機能を有効にすると、CLB上ではなく、Cloud Connect Network（CCN）上でクロスドメイン料金が計算されます。設定していない場合、料金は発生しません。
+ LCUタイプのインスタンスではLCU料金が発生します。共有タイプでは発生しません。

## 従来型アカウントタイプ課金説明
+ プライベートネットワークCLBではインスタンス料金は発生しますが、パブリックネットワーク料金は発生しません（詳細な説明は[CLBインスタンスのアップグレードに伴う価格調整のお知らせ](https://intl.cloud.tencent.com/zh/document/product/214/41565)をご参照ください）。
+ パブリックネットワークCLBはインスタンス料金のみ発生します。パブリックネットワークはCVMでご購入ください。料金については[パブリックネットワーク料金](https://intl.cloud.tencent.com/zh/document/product/213/39743)をご参照ください。
+ 従来型アカウントタイプではクロスリージョンバインディング機能をサポートしていないため、クロスリージョンバインディング料金は発生しません。
+ LCUタイプのインスタンスではLCU料金が発生します。共有タイプでは発生しません。
