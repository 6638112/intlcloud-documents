## 按量计费[](id:pay-as-you-go)

### 计费示例
<dx-accordion>
::: 共享型实例
#### 示例1（公网网络费计费模式为按流量计费）：
假设您在广州地域的 6月1日 10:00:00 - 6月2日 09:59:59 时间内，使用了按量计费且公网网络费的计费模式为按流量计费的负载均衡，且从负载均衡到公网的总流量为2GB，则：
- 实例费：实例费单价 × 使用时长 = 0.686美元/天 × 1天 = 0.686美元。
- 公网网络费：每单位流量费 × 总流量 = 0.12美元/GB × 2GB = 0.24美元。
总费用 = 实例费 + 公网网络费 = 0.686美元 +0.24美元 = 0.926美元。

#### 示例2（公网网络费计费模式为共享带宽包）：
假设您在广州地域的 6月1日 10:00:00 - 6月2日 09:59:59  时间内，使用了按量计费且公网网络费的计费模式为共享带宽包的负载均衡，则：
- 实例费：实例费单价 × 使用时长 = 0.686美元/天 × 1天 = 0.686美元。
- 公网网络费：由于 <a href="https://intl.cloud.tencent.com/document/product/684/15254">共享带宽包费用</a> 是按月结算，因此在6月1日 10:00:00 - 6月2日 09:59:59 时间内仅结算实例费。
总费用 = 实例费 = 0.686美元。
:::
::: 性能容量型实例
#### 示例 1（公网网络费计费模式为按流量计费）：
假设您在广州地域的6月1日 10:00:00 - 6月2日 09:59:59 时间内，使用了按量计费且公网网络费的计费模式为按流量计费的负载均衡，且从负载均衡到公网的总流量为2GB，且每小时平均消耗的性能容量单位 LCU 数为 2 个，则：
- 实例费：每天实例费 × 使用时长 =0.686美元/天 × 1天 = 0.686美元。
- 公网网络费：每单位流量费 × 总流量 = 0.12美元/GB × 2GB = 0.24美元。
- 性能容量 LCU 费 = 2个 LCU × 0.0072美元/个/小时 × 24小时 = 0.3456美元。
总费用 = 实例费 + 公网网络费 = 0.686美元 + 0.24美元 + 0.3456美元 = 1.2716美元。

#### 示例 2（公网网络费计费模式为共享带宽包）：
假设您在广州地域的 6月1日 10:00:00 - 6月2日 09:59:59 时间内，使用了按量计费且公网网络费的计费模式为共享带宽包的负载均衡，且每小时平均消耗的性能容量单位 LCU 数为 2 个，则：
- 实例费：每天实例费 × 使用时长 = 0.686美元/天 × 1天 = 0.686美元。
- 公网网络费：由于 <a href="https://intl.cloud.tencent.com/document/product/684/15254">共享带宽包费用</a> 是按月结算，因此在  6月1日 10:00:00 - 6月2日 09:59:59  时间内仅结算实例费。
- 性能容量 LCU 费 = 2个 LCU × 0.0072美元/个/小时 × 24小时 = 0.3456美元。
总费用 = 实例费 + 性能容量单位 LCU 费 = 1.0316美元。

:::
