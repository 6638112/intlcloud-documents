ここではCLB製品のメリットについてご説明します。

## ハイパフォーマンス
CLBの単一のクラスター（単一のCLBインスタンスではありません）は億単位の同時接続数をサポートし、1秒間に100万パケット単位のデータ量を処理します。1日あたりのアクセス数が1000万を超えるECサイト、ソーシャルプラットフォーム、ゲームビジネスなどにも簡単に対応できます。

## 高い可用性
CLBはクラスター化デプロイを採用し、可用性は99.95%に達します。使用可能なCLBインスタンスが1台しかないという極端な状況であっても、1000万単位の同時接続数を維持することが可能です。またクラスターシステムによって、障害が発生したインスタンスを速やかに除外し、正常なインスタンスをフィルタリングすることで、バックエンドサーバーによる業務の正常な実行が保証されます。

## 高い柔軟性
CLBクラスターは業務負荷に応じて水平スケーリングを行い、アプリケーションシステムの外部に対するサービス機能を柔軟に拡張すると同時に、自動スケーリング（Auto Scaling）の動的なスケーリンググループによってCVMインスタンスを自動的に作成およびリリースすることができます。動的モニタリングや秒レベルの料金計算システムと組み合わせることで、リソースへの介入や予測を手動で行う必要がなくなり、コンピューティングリソースの適正な割り当てとリソースの浪費防止が実現します。

## 安全かつ安定的
CLBは分散型のAnti-DDoSシステムによってほとんどのネットワーク攻撃（DDoS、CC、Web侵入など）を防御することができ、トラフィック攻撃に対し秒レベルでのクレンジングを行って、IPブロック、帯域幅占有などの発生を強力に防止します。CLBが備えるsynproxy攻撃防御メカニズムは、Anti-DDoSシステムが有効になる前にバックエンドCVMが攻撃されてクラッシュすることを防ぎ、データをより安全かつ安定的に保護します。
CLBは各テナントのトラフィックを厳重に隔離し、能動的なAnti-DDoS機能を提供します。パブリックネットワークCLBはデフォルトで[Anti-DDoS Basic](https://intl.cloud.tencent.com/document/product/1029/41715)をサポートしています。CLBはその他にも[Anti-DDoS Pro](https://intl.cloud.tencent.com/document/product/1029)、[Anti-DDoS Advanced](https://intl.cloud.tencent.com/document/product/297)、[Web Application Firewall](https://intl.cloud.tencent.com/document/product/627)などのさまざまなセキュリティ製品をサポートし、業務の安全性を保障します。

>? 
> - ピーク時のトラフィックに対応するためにより高い要件が必要な場合は、[Anti-DDoS Pro](https://intl.cloud.tencent.com/product/ddos-bgp)を別途ご購入いただけます。Tencent Cloudのご提供するAnti-DDoS機能は30 Gbps以上ですが、最大防御能力は各リージョンの実際のネットワーク状態に応じて動的に調整されます。
> - アプリケーション層に対するセキュリティ保護が必要な場合は、[Tencent Cloud Web Application Firewall（WAF）](https://intl.cloud.tencent.com/product/waf)を別途ご購入いただけます。WAFはアプリケーション層のWebセキュリティ保護を提供し、Web脆弱性攻撃、悪意あるクローラーやCC攻撃などの行為を食い止め、サイトやWebアプリケーションのセキュリティを保護します。 
> 

## 低コスト
Tencent Cloud CLBを使用すると、追加のロードバランシングハードウェアの購入や、煩雑すぎる運用保守作業に費用を投じる必要がなくなり、99%ものハードウェア代や人件費を節約することができます。CLBはさまざまな課金モデルをサポートしており、ニーズに応じて選択が可能です。
