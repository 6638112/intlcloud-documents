ここではCLB製品のメリットについてご説明します。

## 高性能
CLBの単一のクラスター（単一のCLBインスタンスではありません）は億単位の同時接続数をサポートし、1秒間に100万パケット単位のデータ量を処理します。1日あたりのアクセス数が1000万を超えるECサイト、ソーシャルプラットフォーム、ゲームビジネスなどにも簡単に対応できます。

## 高い可用性
CLBはクラスター化デプロイを採用し、可用性は99.95%に達します。使用可能なCLB物理サーバーが1台しかないという極端な状況であっても、1000万単位の同時接続数を維持することが可能です。またクラスターシステムによって、障害が発生したインスタンスを速やかに除外し、正常なインスタンスをフィルタリングすることで、バックエンドサーバーによる業務の正常な実行が保証されます。

## 高い柔軟性
CLBクラスターは、業務負荷に合わせてスケールアウトを行い、アプリケーションの外部へのサービス機能を弾力的に拡張します。同時に、Auto Scalingの動的スケーリンググループ利用して、CVM インスタンスを自動的に作成およびリリースします。動的監視システムと秒レベルの課金システムとの組み合わせにより、手動でリソースを管理・予測する必要はなく、合理的にコンピューティングリソースの分配を実現し、余計なリソース利用を防ぎます。

## セキュリティと安定性
CLBは分散型のAnti-DDoSシステムによってほとんどのネットワーク攻撃（DDoS、Web侵入など）を防御することができ、トラフィック攻撃に対し秒レベルでのクレンジングを行って、IPブロック、帯域幅占有などの発生を強力に防止します。CLBが備えるsynproxy攻撃防御メカニズムは、Anti-DDoSシステムが有効になる前にバックエンドCVMが攻撃されてクラッシュすることを防ぎ、データをより安全かつ安定的に保護します。
CLBは各テナントのトラフィックを厳重に隔離し、能動的なAnti-DDoS機能を提供します。パブリックネットワークCLBはデフォルトで[Anti-DDoS Basic](https://www.tencentcloud.com/document/product/1029/41715)をサポートしています。CLBはその他にも[Anti-DDoS Pro](https://www.tencentcloud.com/document/product/1029)、[Anti-DDoS Advanced](https://www.tencentcloud.com/document/product/297)、[Web Application Firewall](https://www.tencentcloud.com/document/product/627)などのさまざまなセキュリティ製品をサポートし、業務の安全性を保障します。
>? 
> - ピーク時のトラフィックに対応するためにより高い要件が必要な場合は、[Anti-DDoS Pro](https://www.tencentcloud.com/product/ddos-bgp)を別途ご購入いただけます。Tencent Cloudのご提供するAnti-DDoS機能は30 Gbps以上ですが、最大防御能力は各リージョンの実際のネットワーク状態に応じて動的に調整されます。
> - アプリケーション層に対するセキュリティ保護が必要な場合は、[Tencent Cloud Web Application Firewall（WAF）](https://www.tencentcloud.com/product/waf)を別途ご購入いただけます。WAFはアプリケーション層のWebセキュリティ保護を提供し、Web脆弱性攻撃、悪意あるクローラーやCC攻撃などの行為を食い止め、サイトやWebアプリケーションのセキュリティを保護します。 
> 

## 低コスト
Tencent Cloud CLBを使用すると、追加のロードバランシングハードウェアの購入や、煩雑すぎる運用保守作業に費用を投じる必要がなくなり、99%ものハードウェア代や人件費を節約することができます。CLBはさまざまな課金モデルをサポートしており、ニーズに応じて選択が可能です。
