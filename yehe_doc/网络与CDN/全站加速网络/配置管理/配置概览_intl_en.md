## Basic Configuration
| Configuration Name  | Document Description     |
| -------- | ----------- |
| [Getting Started](https://intl.cloud.tencent.com/document/product/570/8651) | It describes how to activate the service and quickly connect domain names to the service.  |
| [Domain Name Connection Configuration](https://intl.cloud.tencent.com/document/product/570/10361)| It describes how to connect a domain name to ECDN for acceleration. |
| [CNAME Record Configuration](https://intl.cloud.tencent.com/document/product/570/11134) | It describes how to configure a CNAME record. |
| [Domain Name Status Switch](https://intl.cloud.tencent.com/document/product/570/10362) | It describes how to enable, disable, and delete domain name acceleration service. |
| [Project Configuration](https://intl.cloud.tencent.com/document/product/570/11137)  | It describes how to modify a domain name's project and acceleration region. |
| [Origin Server Configuration](https://intl.cloud.tencent.com/document/product/570/11137#.E4.BF.AE.E6.94.B9.E6.BA.90.E7.AB.99.E9.85.8D.E7.BD.AE)  | It describes how to change the origin server type to origin IP or origin domain.   |


## Advanced Configuration
| Configuration Name  | Document Description     |
