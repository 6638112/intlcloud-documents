You can query the monitoring information (outbound and inbound network bandwidth) of connections in the Tencent Cloud Console in the following steps:
1. Log in to the [Direct Connect](https://console.cloud.tencent.com/vpc/dc) Console.
2. Click **Connection** on the left sidebar to enter the management page.
3. Find the target connection in the list and click the **Monitor** icon.
![](https://main.qcloudimg.com/raw/7df55fe02205a3f47420dea1cfc37897.png)

You can also use an API to view the monitoring information. For more information, please see [Direct Connect - Physical Tunnel Monitoring API](https://intl.cloud.tencent.com/document/product/248/10994).
