下面将为您介绍在官网购买 NAT 网关的具体操作。
1. 登录[ NAT 网关控制台](https://console.cloud.tencent.com/vpc/nat?fromNav)。
3. 选择地区和私有网络，单击**新建**。
4. 在“新建 NAT 网关”弹窗中依次输入或确定参数：网关名称、网关类型、所属网络、弹性 IP ，单击**创建**即可。
