O [Elastic Network Interface](https://intl.cloud.tencent.com/document/product/576) (ENI) fica vinculado a um CVM em um VPC e pode ser migrado livremente entre os CVMs. Os ENIs ajudam a configurar redes de gerenciamento e criar soluções de rede altamente confiáveis.
É possível vincular vários ENIs na mesma zona de disponibilidade a um CVM, com base nas especificações do CVM, a fim de garantir uma rede altamente disponível. Também é possível vincular vários endereços IP privados a um ENI para implantar vários endereços IP para um único CVM.
