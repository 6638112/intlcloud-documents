### Tencent Cloud SCDN을 사용하려면 어떤 전제 조건이 있습니까?

SCDN은 CDN, ECDN 두 기본 제품을 기반으로 제공되는 보안 서비스로, Tencent Cloud CDN 또는 ECDN 서비스를 이미 연결한 도메인은 SCDN을 원클릭 실행할 수 있습니다.

### SCDN은 어떤 유형의 공격에 대응할 수 있습니까?

SCDN은 Web 응용 레이어 공격, DDoS 공격, CC 공격, BOT/크롤러 유형의 공격을 방어할 수 있으며, 비즈니스 니즈에 따라 복합 액세스 제어 규칙을 사용자 정의할 수 있습니다. 자세한 내용은 [제품 장점](https://intl.cloud.tencent.com/document/product/1032/42543)을 참고하십시오.

### SCDN은 CDN, ECDN의 콘텐츠 배포, 전송 가속 기능을 갖추고 있습니까?

네, 갖추고 있습니다. SCDN은 CDN, ECDN 기반의 보안 부가 서비스로 기본적인 CDN, ECDN 기능은 SCDN 활성화 후에도 계속 적용됩니다. 

### SCDN은 어떻게 과금됩니까?

SCDN 요금은 CDN/ECDN 서비스 요금, SCDN 패키지 요금, DDoS/CC 탄력적 방어 요금, 도메인 확장 패키지 요금 등으로 구성됩니다. 자세한 내용은 [과금 안내](https://intl.cloud.tencent.com/document/product/1032/42540)를 참고하십시오.

### SCDN은 어떤 비즈니스 시나리오에 적합합니까?

SCDN은 정부와 기업, 금융, 게임, 스트림 미디어, 이커머스 등 공격, 도용을 당하기 쉬우며 높은 가용성과 품질을 요구하는 비즈니스 시나리오에 적합합니다. 자세한 내용은 [응용 시나리오](https://intl.cloud.tencent.com/document/product/1032/42544)를 참고하십시오.

### SCDN 사용 시, 도메인 캐시 규칙, 원본 서버 규칙, HTTPS 등 항목을 설정하는 방법은 무엇입니까?

SCDN 활성화 후, 기존 CDN, ECDN의 도메인 설정은 여전히 유효하며 CDN, ECDN 콘솔 도메인 관리 페이지에서 관련 설정을 변경할 수 있습니다.
