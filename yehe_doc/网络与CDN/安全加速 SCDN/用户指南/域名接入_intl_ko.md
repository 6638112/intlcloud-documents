SCDN 패키지 활성화 후, [SCDN 콘솔](https://console.intl.cloud.tencent.com/cdn/scdn)에 로그인하여 SCDN 도메인을 추가할 수 있습니다. 도메인 추가 후에는 관련 도메인 보안 설정이 전체 네트워크 SCDN 노드로 전달되며, 현재 네트워크 비즈니스에는 영향을 미치지 않습니다. 

## 도메인 추가

보호 설정 페이지에서 [도메인 추가]를 클릭합니다.

![영어](https://tva1.sinaimg.cn/large/008i3skNgy1gvbiq5g7lwj62540cc76a02.jpg)

도메인 추가 페이지는 두 부분으로 구성되어 있습니다.

- 기본 보호 설정
- 사용자 정의 보안 정책

### 기본 보호 설정

SCDN을 연결할 도메인을 선택하고 Web 공격 방어, CC 공격 방어, DDoS 방어 관련 설정을 진행합니다.

![영어](https://tva1.sinaimg.cn/large/008i3skNgy1gvblkqghl5j60au08pwem02.jpg)

- 도메인 보호

  - SCDN을 연결할 1개 또는 여러 개 도메인을 선택합니다(와일드카드 서브도메인 미지원).
  - 도메인은 CDN을 연결한 상태여야 하며, CDN 서비스가 '실행 중' 상태여야 합니다.
  - 한번에 최대 10개의 도메인을 추가할 수 있으며, 총 도메인 수는 현재 SCDN 패키지의 최대 도메인 수를 초과할 수 없습니다. 

- Web 공격 방어

  ![영어](https://tva1.sinaimg.cn/large/008i3skNgy1gvbllxn8knj60l5050dfw02.jpg)

  - Web 공격 방어 활성화를 선택할 수 있습니다. 
  - 방어 모드: 차단을 선택하면 노드에서 Web 공격 요청을 감지할 경우, 해당 요청을 차단하고 차단 페이지 설정에 따라 기본 차단 페이지 및 403 상태 코드를 반환하거나 요청을 사용자 정의 차단 페이지로 리디렉션합니다. 
  - 사용자 정의 페이지 주소: 차단 페이지를 ‘사용자 정의 페이지’로 설정할 경우, 노드는 모든 차단된 Web 공격 요청을 지정된 사용자 정의 차단 페이지로 리디렉션합니다. 
  - 리디렉션 상태 코드: 상기 리디렉션 상태 반환 코드를 301 또는 302로 지정할 수 있습니다. 

- CC 방어 / DDoS 방어

  - 도메인 CC 공격 방어, DDoS 방어는 기본적으로 활성화되어 있습니다.
  - 구매한 패키지가 탄력적 방어를 지원하는 경우, 탄력적 방어는 기본적으로 활성화되고, 패키지가 탄력적 방어를 지원하지 않는 경우, 기본적으로 비활성화됩니다. 자세한 내용은 [과금 안내](https://intl.cloud.tencent.com/document/product/1032/42540)를 참고하십시오. 

### 사용자 정의 보호 정책

SCDN은 IP, URI, Referer, User-Agent, Params 등 필드의 복잡한 액세스 규칙 설정을 지원합니다. 비즈니스 시나리오에 맞춰 사용자 정의 보호 정책을 설정하고 다중 조건을 조합하여 요청을 필터링할 수 있습니다. ![사용자 정의 규칙 추가](https://tva1.sinaimg.cn/large/008i3skNgy1gvblp56xkij60wf06k74l02.jpg)![규칙 설정](https://tva1.sinaimg.cn/large/008i3skNgy1gvblq4rg28j60l90cqt9602.jpg)

- 보호 규칙
  - 5개의 사용자 정의 규칙을 생성할 수 있으며 모든 규칙에 5개의 매칭 조건을 설정할 수 있습니다. 규칙, 매칭 조건은 **‘또는’**의 관계로, 실행 순서는 리스트 순서에 따라 위에서 아래로 진행되며 **어떤 규칙의 1개 조건이라도 만족하면 바로 차단합니다**.
  - 매칭 필드: Params, URL(path 부분. 예: `/test/1.jpg`. `?` 이후의 매개변수 미포함), IP(클라이언트 IP), Referer, User-Agent 지원.
  - 매칭 조건: 포함, 미포함, 같음, 같지 않음, 최대 길이, 길이, 최소 길이 지원.
  - 매칭 값은 **‘1개’**의 매칭 항목만 입력할 수 있고, 현재 정규 매칭은 지원하지 않습니다. 입력하지 않으면 기본값은 공란입니다. 

### 설정 제출

도메인 설정 완료 후 [제출]을 클릭하면 도메인이 추가됩니다. 팝업 창에서 [보호 설정 반환]을 클릭하면 SCDN 도메인 리스트 페이지가 반환되며, 도메인 설정을 조회하거나 도메인 보안 설정을 변경할 수 있습니다. [계속 추가]를 클릭하면 SCDN 도메인을 계속 추가할 수 있으며, 시스템 백그라운드에서 추가된 도메인에 관련 설정을 배포하는데 약 5분 정도 소요됩니다.
