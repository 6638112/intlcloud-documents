## SCDN 연결

Tencent Cloud SCDN을 연결할 도메인은 우선 Tencent Cloud CDN/ECDN을 연결해야 하며, SCDN 서비스 리전을 활성화해야 합니다. 또한, 동시에 CDN/ECDN 서비스를 활성화해야 합니다. CDN 서비스 사용 및 도메인 연결 제한 관련 내용은 [CDN 사용 제한](https://intl.cloud.tencent.com/document/product/228/32981)을 참고하십시오. 

## SCDN 서비스

| 유형         | 제한 설명                                                     |
| ------------ | ------------------------------------------------------------ |
| SCDN 도메인 | SCDN 패키지에는 기본 방어 도메인 20개가 포함되어 있습니다. 도메인 확장 패키지를 구매하면 최대 수량을 120개까지 늘릴 수 있습니다. SCDN 도메인은 해당 CDN 서비스 리전의 상태를 ‘실행 중’으로 설정해야 하며, 해당 리전의 CDN 서비스를 비활성화하면 SCDN 서비스도 함께 비활성화됩니다. 현재 와일드카드 서브도메인 연결은 지원하지 않습니다. |
| Statistics     | 최근 1년 동안의 전체 공격 데이터 쿼리 기능을 기본 제공하며, 일부 별도 표기된 데이터 항목은 제외됩니다. |
| 이벤트 로그     | 최근 7일 간의 공격 이벤트 로그 쿼리를 지원하며, 생성된 로그 다운로드 작업은 7일 동안 보관됩니다. 작업 1회 당 최대 1000개의 로그를 다운로드 할 수 있으며, 매일 100개의 다운로드 작업을 생성할 수 있습니다. |
