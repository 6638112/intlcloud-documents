## 豊富な機能

Tencent CloudのSecure Content Delivery Network(SCDN)は、ユーザーに高速かつ安定したコンテンツ配信アクセラレーションサービスを提供するという前提のもと、ワンクリックで一体型セキュリティソリューションを有効にし、Web攻撃防御、Anti-DDoS、CC攻撃防御といった複数のセキュリティ機能を統合し、アクセラレーションネットワークのセキュリティ保護機能を全面的に向上させます。

### 分散型DDoS攻撃防御

SCDN DDoS攻撃防御は、先進的な特徴点識別アルゴリズムをベースとして、トラフィックを正確にクリーニングし、SYN Flood、TCP Flood、ICMP Floodといったさまざまな大規模トラフィック攻撃を防御して、サービスの正常かつ安定的な運用を保証します。

SCDNセキュアノードネットワークは分散型アーキテクチャをベースとして構築されており、グローバルなCloud Load Balanceをインテリジェントにスケジューリングします。攻撃トラフィックを受けてクリーニングしきい値に達すると、トラフィックはSCDN高防御クラスターにスケジューリングされ、大規模トラフィックのDDoS攻撃に対応し、突然攻撃を受けたときもスムーズで安定したビジネスアクセスを保証します。

Tencent Cloud SCDNは、ネットワーク全体で最大20TbpsのDDoS攻撃に対抗でき、セキュアノード1個につき、最大1TbpsのDDoS攻撃に対抗できます。

### CC攻撃防御

SCDN CC攻撃防御は、カーネルレベルのブロック、機械学習アルゴリズムを介しヒューマンマシン認識といった方法と組み合わせ、CC攻撃をインテリジェントに識別し、かつブロックします。

SCDNセキュアノード1個につき、最大60万QPSのCC攻撃に対抗でき、ユーザーが受ける異常な高頻度アクセスの影響による瞬間的なクラッシュを防止します。分散型アーキテクチャでは、SCDNはインテリジェントなスケジューリングを通じてノード間の負荷を分散すると同時に、大規模な攻撃に対抗することができます。

### Webアプリケーション層攻撃防御

SCDN Web攻撃防御は、Tencent CloudのWeb Application Firewall機能をベースとして構築されています。Tencentの大規模なWeb攻撃サンプルライブラリとAI +ルールによって、一般的なWeb攻撃をインテリジェントに識別します。また、SQLインジェクション、不正アクセス、クロスサイトスクリプティング(XSS)、クロスサイトリクエストフォージェリ(CSRF)、Webshel​​によるトロイの木馬のアップロードなどの脅威を正確かつ効果的に防御し、バイパス防止、誤検知の低さ、見逃しの低さといった特性を備えています。

突発的なネットワークセキュリティイベントを自主的に監視して対応し、24時間以内にリスクの高いWeb脆弱性を配布します。0day脆弱性保護の仮想パッチによって、保護されたサイトを突発的なWeb脆弱性から遠ざけます。

> ! SCDN Web攻撃防御が新登場。さらに期間限定で無料。

### クローラー対抗（BOT保護）

SCDN BOT保護は、AI +ポリシーエンジンとルールライブラリをベースとしており、既知および公開されているマシンの動作とクローラー（Webクローラー、検索エンジン、コンテンツアグリゲーターなど、1000種類を超えるパブリッククローラー動作を含む）を直接識別、監視、ブロックします。また、サイトユーザーのデータ漏洩、コンテンツ侵害、競争力のある価格の比較、在庫調査、ブラックハットSEO、事業戦略の漏洩など、BOT動作によって引き起こされるビジネスリスクに関する問題点を企業が回避できるように支援します。

公開ポリシーによって識別されていないBOT攻撃については、UA、referrer、レート、パスおよび特殊なパラメータとのマッチング設定をサポートしており、動作の観察、特徴の抽出、業務の調整といった方法によって常に動的な対策を講じています。

> ! SCDN BOT保護機能はベータ版テスト中です。

### カスタム保護ルール

SCDNは、CDNでサポートされる基本IP、referer、UAブラックリスト/ホワイトリストなど基本的なアクセス制御機能に加えて、ユーザー定義の複雑なアクセス制御ルールをサポートしています。すなわち、クライアントIP、URI、Referer、User-Agent、Paramsなどのフィールドを指定し、「と等しい」、「を含む」、「を含まない」などのルールによるマッチングを行います。ビジネスシナリオに応じて、リクエストの複数の条件を組み合わせたフィルタリングを行うことができます。

##大量のリソース備蓄

Tencent Cloud SCDNは、Tencent Cloud CDN/ECDNノードネットワークに依存して構築されています。コンテンツの配信と送信をアクセラレーションすると同時に、Web攻撃、DDoS攻撃、CC攻撃といったサイバー攻撃を全方位的に防御し、ビジネスを安全に保護します。Tencent Cloud CDN ノードネットワークには、中国本土に2000以上、中国本土以外には800以上のアクセラレーションノードがあります。ネットワーク全体の予約済み帯域幅の合計は150Tbps以上です。すべてのSCDNノードは、ハイスペックでセキュリティパフォーマンスの高いコンピュータルームであり、キャリアのハイクオリティなネットワークを利用することができます。

## インテリジェントなスケジューリング

Tencent Cloud SCDNは、ネットワークリンク全体をリアルタイムに監視することによって、自社開発のGSLBスケジューリングシステムとインテリジェントルーティングテクノロジーを組み合わせ、最も優れたSCDNノードにアクセストラフィックをスケジューリングしてサービスを提供し、通常のビジネスリクエストに迅速かつ確実に対応します。大量のトラフィック攻撃が発生した場合、SCDNは、ノード間のCLBをインテリジェントにスケジューリングして、業務の高可用性を確保します。

## ワンクリックアクセス

Tencent Cloud CDN/ECDNをベースとしてTencent Cloud SCDNを使用すると、既存のCDN設定とドメイン名の解決を調整することなく、ドメイン名のSCDNサービスを開始できるので、接続の手順がシンプルかつ手軽になります。

セキュリティサービスが必要なドメイン名をTencent Cloud CDN/ECDNに接続すれば、ユーザーはSCDNコンソールでドメイン名セキュリティサービスをオンにし、ビジネスニーズに応じてさまざまなセキュリティ設定を構成するだけです。約5分間待機すると、ドメイン名に関するセキュリティコンフィグレーションは、ネットワーク全体のSCDNノードに配布されます。全プロセスにおいてアクセラレーションサービスは中断されることなく、既存のネットワーク業務に影響を与えることもありません。
