## シナリオ概要

Tencent Cloud SCDNは、複数のセキュリティ機能を組み合わせて、さまざまな業界のさまざまなネットワークの脅威に対応し、さまざまな業界のコンテンツ配信とWeb Application Securityに対して全面的な保証を提供することができます。

| アプリケーション業界     | 業界のセキュリティリスク                                                 | SCDNセキュリティ機能                       | Tencent Cloud SCDNソリューションの優位性                                         |
| ------------ | ------------------------------------------------------------ | ----------------------------------- | ------------------------------------------------------------ |
| 政府・企業・メディア   | 政府・企業・メディア系のWebサイトやAPPは、政府機関が政務情報を公開し、企業・事業体・メディアが情報を伝播するという、一般の人々とコミュニケーションを図るための重要な窓口です。政府・企業・メディア系のWebサイトまたは情報システムが攻撃を受けると、サービスが利用できなくなったり、コンテンツが改ざんされたりして、政府・企業・事業体のイメージに影響を与え、情報の正常な伝播に支障をきたし、PR面でのトラブルが起こるリスクもあります。 | Web攻撃防御、DDoS/CC攻撃防御、HTTPS | 政府・企業系のWebサイトやAPPの高可用性を保証し、Web攻撃に対抗し、Webサイトのコンテンツが攻撃によって改ざんされ、PRリスクを引き起こすことを防ぎます。 |
| 金融         | 金融分野には、銀行・証券・ファンド・株式などを含みますがこれらに限られないさまざまな分野があり、ハッカー攻撃の主な標的分野の一つとなっています。金融事業には、可用性に対する非常に高い要件があります。攻撃によってシステムやネットワークのリソースが消費されて業務が中断されると、通常のユーザーは取引や注文ができなくなり、重大な経済的損失が発生します。これと同時に、金融事業のオリジンサーバーには大量の機密データが保存されており、オリジンサーバーが侵害されて情報が漏洩すると、より大きな影響と損失につながります。 | Web攻撃防御、DDoS/CC攻撃防御、HTTPS | 金融アプリケーションの安定性と高可用性を確保し、Web攻撃に対抗し、オリジンサーバーのコンテンツとデータ漏洩のリスクを軽減します。 |
| ゲーム、ストリーミングメディア | ゲームやビデオ業界は非常に競争が激しく、常にDDoS攻撃の発生率が高い分野です。2017年以降、テーブルゲーム系のゲームやオンラインビデオなどが爆発的な成長段階に入り、大量の資本が流入しユーザーが殺到しました。そのため同業者間で過当競争が起こり、ネットワークトラフィック攻撃が続発し、すさまじいDDoS攻撃が行われるようになり、テラビット級のトラフィック攻撃も頻発しています。関連業界では、高いサービス可用性と低いラグを確保することがプレイヤーとユーザーを維持するための前提条件になっています。 | DDoS攻撃防御、Web攻撃防御、HTTPS    | ゲームコンテンツ、リソースパックのダウンロードおよびストリーミングメディア送信の安定性を保証します。過当競争が頻発する業界において、大量のトラフィック攻撃によるサービスの利用不能やユーザーの流失を回避する上で役立ちます。 |
| eコマース         | eコマース業界は、業務の繁忙期、特に年の中頃や年末など、複数のイベントサイクルがあるときに、同業者間の過当競争が生じやすくなります。また、ハッカーのランサムウェアなどに起因してDDoS・CC・Web攻撃を受け、サービスの利用不能といった事故が発生し、事業者の信用が損われてマーチャントやユーザーが流失し、莫大な経済的損失につながることもあります。 | DDoS/CC攻撃防御、Web攻撃防御、HTTPS | DDoS・CC・Web攻撃を防御することによって、特にイベントピーク期の攻撃によって生じるサービスの利用不能を防止します。 |
