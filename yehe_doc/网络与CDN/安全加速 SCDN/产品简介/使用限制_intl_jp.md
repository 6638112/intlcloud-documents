## SCDNへの接続

Tencent Cloud SCDNのドメイン名に接続するには、まずTencent Cloud CDN/ECDNに接続する必要があります。また、SCDNのサービス提供リージョンをアクティブ化するには、CDN/ECDNサービスを同時にアクティブする必要があります。CDNサービスの使用とドメイン名の接続に関する制限については、[CDN使用制限](https://intl.cloud.tencent.com/document/product/228/32981)をご参照ください。

## SCDNサービス

| タイプ         | 制限の説明                                                     |
| ------------ | ------------------------------------------------------------ |
| SCDNドメイン名 | SCDNパッケージには、20個の基本的な保護ドメイン名が含まれています。上限を引き上げる場合は、ドメイン拡張パッケージを購入する必要があり、上限は120個まで引き上げられます。CDNの対応するサービス提供リージョンにおけるSCDNドメイン名のステータスを「アクティブ」にする必要があります。対応するリージョンのCDNサービスをオフにすると、SCDNサービスもオフになります。汎用ドメイン名の接続は現在、サポートしていません。 |
| データ統計     | デフォルトでは特段の説明のある一部データ項目を除いて、過去1年間の全体的な攻撃データ照会結果が提供されます。 |
| イベントログ     | 過去7日間の攻撃イベントログの照会をサポートしています。作成に成功したログダウンロードタスクは7日間保存されます。タスク1件につき、最大1000件のログのダウンロードがサポートされています。1日あたり100件のダウンロードタスクの作成が可能です。 |
