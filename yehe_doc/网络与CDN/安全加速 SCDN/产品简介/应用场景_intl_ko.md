## 시나리오 개요

Tencent Cloud SCDN은 다양한 보안 기능을 결합하고, 여러 산업의 각종 네트워크 위협으로부터 콘텐츠 전송, 웹 페이지를 완벽하게 보호합니다. 

| 업계     | 보안 리스크                                                 | SCDN 보안 기능                       | Tencent Cloud SCDN 솔루션 장점                                        |
| ------------ | ------------------------------------------------------------ | ----------------------------------- | ------------------------------------------------------------ |
| 정부, 기업, 미디어   | 정부와 기업 미디어의 웹 사이트, APP은 정부 기관이 업무 정보를 공개하고, 기업이나 미디어가 정보 전파하여 대중과 인터랙션하는 중요한 소통 창구입니다. 정부와 기업 미디어의 웹 사이트나 정보 시스템이 공격을 받아 서비스가 마비되거나 콘텐츠가 조작되면 정부와 기업 이미지가 타격을 입고 정보의 정상적인 전파을 방해하며 PR 리스크가 커집니다. | Web 공격 방어 DDoS/CC 공격 방어 HTTPS | 정부와 기업의 웹 사이트, APP 애플리케이션의 가용성을 보장하고 Web 공격을 방어하며 웹 사이트 콘텐츠 조작으로 인한 PR 리스크를 방지합니다. |
| 금융         | 은행, 증권, 펀드, 주식 외에도 다양한 유형을 포함하는 금융 분야는 해커들의 주요 공격 타깃입니다. 금융 비즈니스는 높은 가용성이 요구되며, 공격으로 시스템과 네트워크 리소스가 소모되어, 비즈니스가 중단되면 거래가 불가능해지고 막대한 경제적 손실이 초래됩니다. 또한 금융 비즈니스 원본 서버 스토리지는 대규모의 민감한 데이터를 포함하고 있어, 정보가 유출되면 더욱 큰 손실이 발생합니다. | Web 공격 방어 DDoS/CC 공격 방어 HTTPS | 금융 애플리케이션의 안정성과 가용성을 보장하고, Web 공격을 방어하며, 원본 서버 콘텐츠 및 데이터 유출 리스크를 낮춥니다. |
| 게임, 스트림 미디어 | 게임, 비디오 산업은 경쟁이 치열하여 오랫동안 DDoS 공격이 자주 발생하는 분야였습니다. 2017년 이후 보드 유형 게임, 온라인 비디오 등이 폭발적으로 성장하면서, 많은 자본과 사용자가 유입되고, 동종업계의 악의적인 경쟁 행태가 많이 나타났습니다. 네트워크 트래픽 공격, DDoS 공격, T 레벨 트래픽 공격이 끊이지 않았고, 이로 인해 비즈니스 가용성을 보장하고 랙 발생을 줄이는 것이 유저 이탈 방지의 전제 조건이 되었습니다. | DDoS 공격 방어 Web 공격 방어 HTTPS    | 게임 콘텐츠, 리소스 패키지 다운로드 및 스트림 미디어의 안정적인 전송을 보장합니다. 악의적인 경쟁 행태가 빈번히 발생하는 상황에서 대규모 트래픽 공격으로 인한 서비스 마비 및 사용자 이탈을 방지합니다. |
| 이커머스         | 이커머스 산업은 비즈니스 성수기 특히 한 해 중반, 연말 등 이벤트가 많은 기간에 동종업계의 악의적인 경쟁 행위나 해커의 DDoS, CC 공격 및 Web 공격이 빈번히 발생했습니다. 이로 인해 서비스 마비, 기업 이미지 타격, 업체 및 고객 이탈 등 문제가 발생하여 막대한 경제적 손실이 발생했습니다. | DDoS/CC 공격 방어 Web 공격 방어 HTTPS | DDoS, CC 공격, Web 공격을 방어하고 이벤트가 많은 시기에 공격으로 인한 서비스 마비를 방지합니다. |
