## CDN/ECDN 서비스 요금

SCDN 활성화 전에 먼저 CDN/ECDN에서 도메인을 연결하고 해당 서비스 리전의 CDN/ECDN 서비스를 활성화해야 합니다. CDN, ECDN 서비스 요금은 제품 가격별로 개별 부과되며 본인의 비즈니스 상황에 맞춰 적합한 과금 방식을 선택할 수 있습니다. CDN, ECDN 과금 관련 자세한 내용은 CDN, ECDN ‘정가’를 참고하십시오. 

- [Tencent Cloud CDN 정가](https://buy.intl.cloud.tencent.com/pricing/cdn)
- [Tencent Cloud ECDN 정가](https://intl.cloud.tencent.com/product/ecdn)

## 탄력적 방어

SCDN 표준 패키지를 구매한 경우, DDoS/CC/BOT 패키지 방어 한도 초과 시 탄력적 방어가 활성화됩니다. 

### DDoS/CC 탄력적 방어

SCDN은 최고 1Tbps(DDoS 공격), 1500,000 QPS(CC 공격)의 방어 기능을 제공하며, 일별 공격 방어 피크값 해당 차등 구간 단가에 따라 일 단위 후불 결산 됩니다. 차등 단가는 다음과 같습니다. 

| DDoS 일 피크 구간           | CC 공격 일 피크 구간                    | 가격(USD/일) |
| -------------------------- | --------------------------------- | --------------- |
| 20 Gbps - 30 Gbps(포함)    | 60,000  QPS - 100,000 QPS(포함)   | 225.71          |
| 30 Gbps - 40 Gbps (포함)   | 100,000  QPS - 130,000 QPS(포함)  | 397.14          |
| 40 Gbps - 50 Gbps(포함)    | 130,000  QPS - 160,000 QPS(포함)  | 568.57          |
| 50 Gbps - 60 Gbps (포함)   | 160,000  QPS - 200,000 QPS(포함)  | 740.00          |
| 60 Gbps - 70 Gbps(포함)    | 200,000  QPS - 230,000 QPS(포함)  | 1168.57         |
| 70 Gbps - 80 Gbps (포함)   | 230,000  QPS - 260,000 QPS(포함)  | 1454.29         |

> 2. DDoS/CC 공격이 동시에 SCDN 패키지 보호 한도를 초과하고 일별 피크가 서로 다른 구간에 속할 경우 더 높은 구간의 가격을 기준으로 합니다. 
> 3. WAF는 현재 탄력적 방어를 지원하지 않으며 패키지 QPS 보호 한도를 넘어선 경우 더 이상 보호되지 않습니다. 

### BOT 탄력적 방어

BOT 탄력적 방어는 차등 구간 가격에 따라 후불 일 결산됩니다. 

| BOT 행동 분석   | 가격(USD/Qps/일) |
| -------------- | ------------------- |
| 0 ~ 10000 QPS  | 0.0286              |
| 10000 Qps 초과 | 0.0143              |

> !
>
> 1. SCDN 패키지(기본/표준)를 구매하면 BOT 탄력적 방어를 활성화할 수 있으며 **BOT 방어 기본 패키지**는 필수가 아닙니다.
> 2. BOT 탄력적 방어는 기본적으로 비활성화되어 있으며, 기본 패키지 구매 후 자체적으로 활성화할 수 있습니다.



## 도메인 확장 패키지

SCDN 패키지는 기본적으로 20개의 SCDN 도메인 연결을 지원하며 **도메인 확장 패키지**를 구매하여 최대 수량을 늘릴 수 있습니다. 모든 도메인 확장 패키지에는 SCDN 도메인 10개가 포함되어 있습니다. 도메인 확장 패키지는 선불 정액 과금제이며, SCDN 패키지 구매 후 구매 가능합니다. 유효 기간은 패키지와 동일하게 유지됩니다.

단일 SCDN 패키지는 최대 50개의 도메인 확장 패키지를 구매할 수 있으며(최대 520개 SCDN 도메인 연결 지원), 최대 36개월까지 누적 구매 가능하고, 자동 연장 설정은 패키지와 동일하게 유지됩니다. 도메인 확장 패키지 가격은 다음과 같습니다. 

| 부가 서비스 항목                  | 가격(USD/개/월) |
| --------------------------- | ------------------ |
| 도메인 확장 패키지(도메인 10개 추가) | 142.86             |

> ! 패키지 활성화 시 도메인 확장 패키지를 추가 구매하는 것 외에도 서비스 활성화 후 실제 비즈니스 상황에 따라 도메인 확장 패키지를 추가 또는 삭제할 수 있습니다. 도메인 확장 패키지는 환불되지 않습니다.
