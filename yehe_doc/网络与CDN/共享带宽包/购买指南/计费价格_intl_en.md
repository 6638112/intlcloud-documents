This document describes the BWP pricing of different bandwidth types.

## Bandwidth Type
Tencent Cloud offers the following bandwidth packages according to the bandwidth type.
<table>
<tr><th width="30%">Bandwidth Type</th><th>Description</th></tr>
<tr><td><a href="#bgp">BGP Bandwidth Package</a></td><td>A bandwidth package using BGP IP lines, which can be manually created.</td></tr>
<tr><td><a href="#cn2">Dedicated BGP Bandwidth Package</a></td><td>A bandwidth package using dedicated BGP IP lines, which can be manually created.</td></tr>
<tr><td><a href="#anycast">AIA BGP Bandwidth Package</a></td><td>A bandwidth package using AIA BGP IP lines, which can only be automatically created by the system.</td></tr>
</table>
