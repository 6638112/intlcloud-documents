本文介绍标准账户类型的用户如何删除 IP 带宽包。

## 前提条件
删除 IP 带宽包前，请确保包内的所有资源已全部移除，否则无法删除。详情请参见 [移除资源](https://intl.cloud.tencent.com/document/product/684/34600)。

## 操作步骤

1. 登录 [私有网络控制台](https://console.cloud.tencent.com/vpc/vpc?rid=1)。
2. 单击左侧导航栏的【共享带宽包】。
3. 在“共享带宽包”页面，选择所在地域，找到目标共享带宽包实例，单击右侧"操作"列的【删除】。
![](https://qcloudimg.tencent-cloud.cn/raw/866321e5c0ea516cff38b553de3e65aa.png)
4. 在弹出的对话框中，单击【确认】。
![](https://qcloudimg.tencent-cloud.cn/raw/c42428374bd73fc2590839bb0e799343.png)
