

연결할 가속 도메인이 다음과 같은 상황인 경우 도메인 소유권을 인증해야 합니다.

- 처음으로 연결하는 도메인
- 다른 사용자에 의해 연결된 도메인
- 와일드카드 서브도메인

## 작업 단계

1. **인증 방법**을 클릭하여 DNS 인증에 필요한 리졸브 레코드 정보를 가져옵니다. 인증 완료 전까지는 이 페이지를 열어둡니다.

2. 리졸브 업체가 Tencent Cloud인 경우, [도메인 서비스 콘솔](https://console.cloud.tencent.com/cns)에서 해당 도메인을 찾아 **리졸브**를 클릭하고, 레코드 유형이 TXT인 DNS 레코드를 추가한 후, 호스트 레코드를 `_cdnauth`로 입력합니다.

3. TXT 리졸브가 적용될 때까지 기다렸다가 **인증** 버튼을 클릭합니다.
