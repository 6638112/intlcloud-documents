You need to create logsets and log topics to store and view the flow logs.
1. Log in to the [CLS Console](https://console.cloud.tencent.com/cls) and click **Logset** on the left sidebar to enter the "Logset Management" page. 

2. Select a desired region on the top bar and click **Create Logset**.

   ![](https://main.qcloudimg.com/raw/af5a948f72821b33e4cc54f0e2d9f962.png)

3. After creating a logset, click the logset ID to enter its details page. Now you can click **Add Log Topic** to create a log topic, and then click the log topic ID to enter its details page.

4. Select the **Index Configuration** tab and click **Edit** in the top-right corner.

   ![](https://main.qcloudimg.com/raw/80858b11c7dd046d76950f46f591e48c.png)

5. Enable the index and click **Save**. Now you can view and search for flow logs.

   ![](https://main.qcloudimg.com/raw/37946eff0820e0c86e3d78e936a9a288.png)

>
>- You do not need to install agents or be concerned about the server group status. 
>- If you do not need to publish flow logs to services such as [COS](https://intl.cloud.tencent.com/document/product/436/6222), you do not need to be concerned about log publishing tasks.
