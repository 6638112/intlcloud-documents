You can configure monitoring alarms for your connection by following these steps:
1. Log in to the [Cloud Monitor](https://console.cloud.tencent.com/monitor/overview) console.
2. In the left sidebar, click **Alarm Configuration** > **Alarm Policy** to go to the management page.
3. Click **Add**.
4. Edit the alarm policy on the page that appears:
 - Enter a name for the alarm policy.
 - Select “Connection” as policy type.
 - Project: select a project as needed.
 - Alarm object: select the associated dedicated tunnel.
 - Available alarm triggers: outbound bandwidth, inbound bandwidth, bandwidth utilization.
6. Set the alarm channel and click **OK** to complete the alarm policy configuration.

After the connection alarm is configured, you will receive system alarms based on how the alarm recipient groups are set. For more monitoring operations, refer to [Cloud Monitor Overview](https://intl.cloud.tencent.com/document/product/248/32799).
