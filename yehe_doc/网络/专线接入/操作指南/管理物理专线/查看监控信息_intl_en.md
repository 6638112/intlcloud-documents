You can query the monitoring information of connections (outbound bandwidth and inbound bandwidth) in the Tencent Cloud console by following these steps:
1. Log in to the [Direct Connect](https://console.cloud.tencent.com/vpc/dc) console.
2. In the left sidebar, click **Connections** to go to the management page.
3. Locate the connection you want to view in the list, and click the **Monitoring** icon.

You can also query monitoring information by using Tencent Cloud APIs. For more information, refer to the [API Documentation](https://intl.cloud.tencent.com/document/product/216/18404).
