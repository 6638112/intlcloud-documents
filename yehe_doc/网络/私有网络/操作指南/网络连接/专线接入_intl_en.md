Direct Connect provides a fast and secure approach to connect Tencent Cloud with on-premises IDCs. You can access Tencent Cloud computing resources in multiple regions in one go using a connection to implement flexible and reliable hybrid cloud deployment.

For the common operations of Direct Connection, please see:
+ [Quick Start](https://intl.cloud.tencent.com/document/product/216/7557)
+ [Managing Connections](https://intl.cloud.tencent.com/document/product/216/38523)
+ [Managing Direct Connect Gateways](https://intl.cloud.tencent.com/document/product/216/19255)
+ [Dedicated Tunnels](https://intl.cloud.tencent.com/document/product/216/19249)
