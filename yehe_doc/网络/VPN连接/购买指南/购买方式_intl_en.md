You can purchase VPN connections on the Tencent Cloud official website. The purchase procedure is described below.
1. Log in to the [Virtual Private Cloud Console](https://console.cloud.tencent.com/vpc/vpc?rid=1).
2. In the left sidebar, choose **VPN Connection** -> **VPN Gateway** to go to the management page.
3. Select a region and VPC, and then click **+ New**.
4. In the pop-up window, enter the gateway name, select the network, the bandwidth cap, and the billing method, and then click **Create**.
 - **Gateway Name**: enter a VPN gateway name, which cannot exceed 60 characters.
 - **Network**: select the VPC to which the VPN gateway belongs.
 - **Bandwidth Cap**: select the maximum bandwidth for the VPN gateway as needed.
 - **Billing Method**: select the billing mode for the VPN gateway as needed.
    - **Bill-by-traffic**: this mode is suitable for scenarios where the bandwidth fluctuates greatly.
