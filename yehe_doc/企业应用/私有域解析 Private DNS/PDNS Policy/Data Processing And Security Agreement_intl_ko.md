## 1.배경

Tencent Cloud Private DNS 서비스("**기능**")를 사용하는 경우 이 모듈이 적용됩니다. 이 모듈은 [**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 개인 정보 보호 및 보안 계약에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 의미를 갖습니다. DPSA와 이 모듈 간에 충돌이 발생할 경우, 이 모듈은 상충하는 범위까지만 적용됩니다.

## **2.**처리

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.


| **개인 정보**                                                | **이용**                                               |
| ------------------------------------------------------------ | ------------------------------------------------------ |
| **도메인 이름 확인 레코드**에는 다음이 포함됩니다. DNS 도메인 이름, UIN, APPID, 호스트, 유형, 값, 가중치, 만료 시간, TTL. | 도메인 이름 확인을 완료하기 위해 이 정보를 사용합니다. |


## 3.서비스 지역

 

DPSA에 명시된 바와 같습니다.


## 4.하위 프로세서

 

DPSA에 명시된 바와 같습니다.


## 5.데이터 보유

 

당사는 다음과 같이 기능과 관련하여 처리된 개인 데이터를 저장합니다.


| **개인** **정보**           | **보유 정책**                                                |
| --------------------------- | ------------------------------------------------------------ |
| DNS 도메인 이름 확인 레코드 | 데이터는 계정이 존재하는 동안(사이버 보안 사고로 인해 이러한 기록이 위험에 처해 있다고 판단되지 않는 한) 보관됩니다. |

 

DPSA에 따라 이러한 개인 데이터의 삭제를 요청할 수 있습니다.



## 6.특수 조건

 

개인 데이터 처리에 동의할 수 있는 최소 연령을 만족하는 개인인 최종 사용자만 이 기능을 사용할 수 있습니다. 이는 최종 사용자의 해당 지역에 따라 다를 수 있습니다.
