## 1.Histórico


Este módulo se aplica se você usa o serviço Tencent Cloud Private DNS **Recurso**. Este módulo está incorporado ao contrato de segurança e privacidade de dados localizado no [**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, ele deve ser aplicado conforme a inconsistência.


## 2.PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Registro de resolução de nome de domínio**, incluindo: Nome de domínio DNS, UIN, APPID, host, tipo, valor, peso, tempo de validade e TTL. | Usamos essas informações para completar a resolução do nome de domínio. |


## 3.região do serviço

 

Conforme especificado no DPSA.



## 4.SUB-Processadores

 

Conforme especificado no DPSA.



## 5.retenção de dados

 

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:




| **Informações** **pessoais**                 | **Política de retenção**                                     |
| -------------------------------------------- | ------------------------------------------------------------ |
| Registro de resolução de nome de domínio DNS | Os dados são mantidos enquanto uma conta existir (a menos que seja determinado que tais registros estejam em risco como resultado de um incidente de segurança cibernética). |

 

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.


## 6.condições especiais

 

Você deve garantir que este recurso seja usado somente por usuários finais com a idade exigida para que possam consentir com o processamento de seus dados pessoais. Aqui pode haver diferença dependendo da jurisdição em que o usuário final está localizado.
