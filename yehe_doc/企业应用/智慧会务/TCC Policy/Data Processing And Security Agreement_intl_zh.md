## 1.背景

本模块仅在您使用腾讯云智慧会务功能（简称“**功能**”）时适用。本模块包含在位于“[**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347)”的数据隐私和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。

 

## 2.处理

我们将处理与本功能相关的以下数据（前提是，您已指示我们作为本功能的一部分集成处理这些数据所需的相关功能）： 

| **个人信息**                                                 | **用途**                                     |
| ------------------------------------------------------------ | -------------------------------------------- |
| **配置数据**内容管理：会务信息、时间表信息、发言者信息（图片、姓名和介绍）、展示信息、产品信息、商业机会信息（姓名、电子邮件、联系号码、产品名称、备注）访问管理：角色/访问权（听众、参展者）模板管理：页面配置、主题，模板调查问卷：标题、问题小程序/网页配置：appID、appSecret、PC 和 H5 设置 | 这使您可以设计、配置和管理本功能。           |
| **用户管理数据**您在管理控制台中指定的数据字段（其中可能包括电话号码、姓名、公司名称、电子邮件地址、地址、国家/地区、兴趣、年龄、行业、职业和性别） | 这可以让您管理本功能的 用户。                |
| **仪表板数据**页面视图、最前访问排名、访客数据（区域分布/分布、兴趣、行业、性别、年龄、职业）、直播视频音量、直播视频观看持续时间，或您指定的任何其他数据字段（包括您在上文“用户管理数据”中指定的任何数据集） | 这有助于概览与会务相关的用户交互和统计信息。 |
| **OCR 数据**名片数据（如在扫描的名片上指定）                 | 这可以让您上传本功能的用户数据。             |
| **会议数据**统计：公司名称、管理员姓名、指定的公司名称、指定的管理员姓名、计划的会议时间、会议状态 企业信息管理：用户、角色、企业标准名称、代表姓名、备注和操作 | 这可以让您管理本功能的会议安排。             |
| **推送数据**用户电话号码、电子邮件地址                       | 这可以让您在本功能中向用户推送消息。 |
| **交互活动数据**基本设置、奖品设置、参与者设置（姓名、电话号码和性别从上文“用户管理数据”同步或由您直接收集） | 这可以让您在本功能中管理交互活动。   |
| **票证管理数据**票证类型、价格、票证总量限制、审批要求、默认票证设置、显示设置、订票信息，包括订单编号、订单时间、姓名、电话号码、票证类型、单价、数量、总金额 | 这可以让您在本功能中管理票证。       |
| **LVB 数据** LVB 管理：LVB 类型、LVB 名称和介绍、播放时间、串流名称、pushStreamURL、回放与否、评论设置审核设置（如适用）评论回顾：LVB 名称、发布时间、用户名、评论和操作（通过/删除） | 这使您可以在本功能中管理现场直播。   |
| **参展者管理数据**角色设置数据、子帐户信息（参展者帐户名、密码、帐户拥有者、电话号码、角色和展示模板 | 这使您可以在本功能中管理参展者帐户。 |

## 5.数据保留

按照客户指示保留数据（且在任何情况下，都会在服务终止后 30 天内删除），但仅为了供您实时查看而处理的仪表板数据除外。

您可以根据 DPSA 请求删除这些个人数据。


## 6.特殊条款


您必须确保使用本功能的最终用户至少达到了个人可以同意处理自己个人数据的最低年龄，或者未达到最低年龄的个人使用此功能时获得了家长同意。根据最终用户所处的司法管辖区，这一要求可能有所不同。
