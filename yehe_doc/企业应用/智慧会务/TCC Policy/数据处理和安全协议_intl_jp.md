## 1.背景

本モジュールは、お客様が Tencent Cloud Conference 機能（「**本機能**」）を利用する場合に適用されます。本モジュールは、「[**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347)」のデータプライバシーとセキュリティ契約に組み込まれています。本モジュールで使用されているが定義されていない用語は、DPSA でそれらに与えられた意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。



## 2.処理

弊社は、本機能に関連して以下のデータを処理します（当該データの処理に必要な関連機能を本機能の一部として統合するようにお客様から指示された範囲内で）。

 

| **個人情報**                                                 | **用途**                                           |
| ------------------------------------------------------------ | -------------------------------------------------- |
| **構成データ**コンテンツ管理：会議情報、スケジュール情報、講演者情報（画像、氏名、紹介）、展示会情報、製品情報、ビジネス機会情報（氏名、メールアドレス、連絡先番号、製品名、メモ）アクセス管理:役割/アクセス権（参加者、出展者）テンプレート管理：ページ構成、テーマ、テンプレートアンケート：タイトル、質問ミニプログラム/Web ページの構成：アプリID、appSecret、PC と H5 の設定 | これにより、本機能の設計や構成、管理を処理します。 |
| **ユーザー管理データ**管理コンソール内でお客様が指定したデータフィールド（電話番号、氏名、会社名、メールアドレス、住所、国/地域、趣味、年齢、業種、職業、性別など） | これにより、本機能のユーザーを管理できます。       |
| **ダッシュボードデータ**ページビュー、トップ訪問ランキング、訪問者データ（地域分布/分布、趣味、業種、性別、年齢、職業）、ライブビデオ再生量、ライブビデオ視聴時間、またはお客様によって指定されたその他のデータフィールド（上記のユーザー管理データに指定された一連のデータを含む） | これにより、会議に関連するユーザーの対話と統計情報の概要を把握できます。 |
| **OCR データ**名刺データ（スキャンした名刺に明記の通り）     | これにより、本機能のためのユーザーデータをアップロードできます。 |
| **会議データ**統計：会社名、管理者名、指定会社名、任命された管理者名、予定会議時間、会議状況企業情報管理：ユーザー、役割、企業標準名、代表者名、発言、操作 | これにより、本機能の会議スケジュールを管理できます。         |
| **プッシュデータ**ユーザーの電話番号、メールアドレス         | これにより、本機能内のユーザーにメッセージをプッシュ通知できます。 |
| **インタラクティブイベントデータ**基本設定、賞品設定、参加者設定（名前、電話番号、性別は上記のユーザー管理データから同期されるか、お客様が直接収集します） | これにより、本機能内でインタラクティブなイベントを管理できます。 |
| **チケット管理データ**チケットの種類、価格、総チケット量の制限、承認要件、デフォルトのチケット設定、表示設定、チケット注文情報（注文番号、注文時刻、名前、電話番号、チケットタイプ、単一価格、数量、合計金額を含む） | これにより、本機能内でチケットを管理できます。               |
| **LVB データ**LVB 管理：LVB タイプ、LVB 名と導入、放送時間、ストリーム名、pushStreamURL、リプレイの有無、コメント設定、モデレーション設定（該当する場合）コメントレビュー：LVB 名、発行時間、ユーザー名、コメント、操作（パス/削除） | これにより、本機能内で内でのライブ放送管理できます。         |
| **出展者管理データ**役割設定データ、サブアカウント情報（出展者のアカウント名、パスワード、アカウント所有者、電話番号、役割と展示テンプレート） | これにより、本機能内で出展者のアカウントを管理できます。     |



## 3.サービス地域

DPSA で規定されている通り。


## 4.復処理者

DPSA で規定されている通り。Tencent Cloud Computing（北京）Co., Ltd.、First App Holdings Limited、Aceville Pte Ltd. などが含まれます。


## 5.データ保持

データは、お客様の指示に従って保持されます（いかなる場合も、サービスの終了から 30日以内に削除されます）。ただし、お客様がリアルタイムで閲覧するためにのみ処理されるダッシュボードデータは除きます。

お客様は DPSA に従って、当該個人データの削除を要請することができます。



## 6.特別な条件

お客様は、本機能を、個人データの処理に同意できる最低年齢に達しているエンドユーザー、または最低年齢に達していない個人に対して親の同意が得られているエンドユーザーのみが使用することを保証する必要があります。これは、エンドユーザーが所在する法域によって異なる場合があります。
