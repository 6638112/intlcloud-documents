## 1\. Considerações

Este Módulo se aplica se você usar o recurso Tencent Ecard (“**Recurso**”). Esse Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em: [(“**DPSA**”)](https://intl.cloud.tencent.com/document/product/301/17347 ). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.
