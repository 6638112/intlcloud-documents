
## 概述
本文档将指导您如何开通移动解析 HTTPDNS。

## 操作步骤
### 开通移动解析 HTTPDNS 服务
1. 登录 [移动解析 HTTPDNS 管理控制台](https://console.cloud.tencent.com/httpdns)，进入 “概览” 管理页面。
2. 在 “概览” 管理页面中，单击**开通服务**。如下图所示：
![](https://qcloudimg.tencent-cloud.cn/raw/808b250b9e86751ad274419d98bc1401.png)
3. 在弹出的 “开通服务” 窗口中，勾选**同意移动解析服务协议**，并单击**开通服务**，即可完成开通移动解析 HTTPDNS 操作。如下图所示：
![](https://qcloudimg.tencent-cloud.cn/raw/365f59bcdcd7bbd1d1d7ac3dc7e71a4e.png)

>? 完成开通操作后，系统将为您颁发授权 ID 和密钥信息，您可以在 [移动解析 HTTPDNS 管理控制台](https://console.cloud.tencent.com/httpdns) 进行查看。

### 已拥有授权 ID
>?如您已通过线下方式申请授权 ID 和密钥，请按照以下步骤进行验证，验证通过后请将授权 ID 与当前账号进行绑定。授权 ID 跟当前账号绑定后，才可使用服务地址 `119.29.29.99/98` 提供的服务。具体可查看 [计费说明](https://intl.cloud.tencent.com/document/product/1130/44459)。
>
1. 登录 [移动解析 HTTPDNS 管理控制台](https://console.cloud.tencent.com/httpdns)，进入 “概览” 管理页面。
2. 在 “概览” 管理页面中，单击**开通服务**。如下图所示：
![](https://qcloudimg.tencent-cloud.cn/raw/0559dd69ad23dd1f600e306d10513ad3.png)
3. 在弹出的 “开通服务” 窗口中，单击**点此绑定**。如下图所示：
![](https://qcloudimg.tencent-cloud.cn/raw/61dffef1f1e208cd37cbfa792eb5ee25.png)
3. 在弹出的 “绑定 ID 开通服务” 窗口中，填写企业 ID 和密钥信息。如下图所示：
![](https://qcloudimg.tencent-cloud.cn/raw/1fb1eb094a8d53220e42ec3bea3ae732.png)
4. 单击**验证**，即可完成操作。
