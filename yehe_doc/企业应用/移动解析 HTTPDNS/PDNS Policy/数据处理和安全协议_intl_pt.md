## 1\. CONSIDERAÇÕES INICIAIS

Este Módulo se aplica se você utiliza HTTPDNS (“**Recurso**”). Este módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado no [DPSA](https://intl.cloud.tencent.com/document/product/301/17347 ). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. Este Módulo se aplicará conforme a inconsistência caso ele entre em conflito com o DPSA. 

## 2\. PROCESSAMENTO

Processaremos os seguintes dados em conexão com o recurso:

| **Informações Pessoais**                | **Uso**                                                      |
| --------------------------------------- | ------------------------------------------------------------ |
| **Nome do domínio DNS, nome do aplicativo, tipo de aplicativo** | Nós processamos esses dados apenas com o propósito de fornecer o Recurso para você. Observe que esses dados são armazenados com backup no nosso recurso TencentDB for MySQL. |



## 3\. REGIÃO DO SERVIÇO

Conforme especificado no DPSA.

## 4\. SUBPROCESSADORES

Conforme especificado no DPSA.

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o recurso da seguinte forma:

| **Informações Pessoais**            | **Política de Retenção**                                         |
| ----------------------------------- | ------------------------------------------------------------ |
| Nome do domínio DNS, nome do aplicativo, tipo de aplicativo | Retemos esses dados até que você exclua manualmente tais dados \[a não ser em circunstâncias especiais, como sua conta estar sujeita a incidente de segurança cibernética, e nesse caso, analisaremos com você e (se considerado necessário) excluiremos os dados dentro de 5 dias após a análise\]. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. CONDIÇÕES ESPECIAIS

Você deve garantir que este recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
