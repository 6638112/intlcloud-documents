## 1.  简介

本模块仅在您使用腾讯移动推送（简称“**功能**”）时适用。本模块包含在位于[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。

## 2.  控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3.  可用性

本功能可在全球范围内使用。

## 4.  我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| <b>个人信息</b>                                                 | <b>用途</b>                                                     | <b>法律依据</b>                                                 |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| TPNS 管理信息/设置：应用程序名称、Android 安装包名称/iOS BundleID、提供商通道和 AppId/  appkey/AppSecret/SecretKey | 确保提供推送通知服务。                                       | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能下的服务而言是必要的。 |
| 技术信息：与您通过本功能发送的推送消息相关的服务使用数据服务与标记日志信息。 | 解决和找到问题，帮助我们更好地了解本功能的运行情况，并优化和改进本功能的质量。 | 使用这些信息来改进和本功能的运行符合我们的合法权益。         |

见隐私政策中的规定。

## 6.  我们如何分享个人信息

## 7.  数据保留

我们将根据下列规定保留个人信息：

| <b>个人信息</b>  | <b>保留政策</b>                                                  |
| ------------ | ------------------------------------------------------------ |
| 管理信息     | 这些数据保存至：(i) 您选择通过 TPNS 管理门户删除此类信息之时，当您作出此选择后，这些信息将会被删除；或 (ii) 我们不再向您提供本功能之时，这些信息将在我们的服务到期或终止之日后三十 (30) 天内被删除。 |
| 技术信息     | 自创建之日起 90 天                                           |
