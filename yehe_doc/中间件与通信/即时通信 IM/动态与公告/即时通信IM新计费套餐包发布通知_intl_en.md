The original billing mode of Chat was disused on December 26, 2022.
- If you created your first Chat application in a selected IDC before December 26, 2022, all Chat applications of the IDC use the original billing mode.
- If you created your first Chat application in a selected IDC after December 26, 2022, all Chat applications of the IDC use the new billing mode.

>?
>- If you are using the original billing mode and want to migrate to the new billing mode, [contact sales](https://www.tencentcloud.com/contact-us) or [submit a ticket](https://console.tencentcloud.com/workorder). Once migrated, the billing mode cannot be restored.
>- The new billing mode applies only to the Singapore, Frankfurt, Seoul, Mumbai, and Silicon Valley IDCs. Other IDCs do not support the new billing mode.


In the original billing mode, Chat offers the Free, Pro, and Ultimate editions of billing plans.
In the new billing mode, Chat offers the Developer, Standard, and Premium editions of billing plans.
>!The Ultimate edition supports all features of the Premium edition, and users of the Developer edition can experience all features of the Premium edition free of charge.
