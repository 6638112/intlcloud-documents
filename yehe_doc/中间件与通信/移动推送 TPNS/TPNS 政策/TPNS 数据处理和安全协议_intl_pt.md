# 1.  Histórico

Este módulo se aplica se você usar o Tencent Push Notiﬁcation Service ("**Recurso**”). Este Módulo é incorporado ao Contrato de segurança e processamento de Dados localizado em [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este módulo, este módulo deverá ser aplicado conforme a inconsistência.

# 2.  PROCESSAMENTO

Processaremos os seguintes dados pessoais conectados ao recurso:
| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| Notificações por push                                        | Processamos a  notificação por push para entregá-la de acordo com suas preferências. Não nos  responsabilizamos pelo conteúdo contido nas notificações por push, que são  definidas exclusivamente por você. |
| Token ID  (criado pelo recurso para cada dispositivo)        | Ele é criado  para garantir que as notificações por push possam ser entregues aos  dispositivos selecionados por você. |
| Conta/Login                                                  | Se você optar  por agrupar um Token ID com detalhes específicos de conta/login (por exemplo,  um ID do WeChat) processaremos os detalhes de conta/login associados de  acordo com suas instruções. |
| Informações  técnicas (para dispositivos que recebem notificações por push: Versão do  SISTEMA OPERACIONAL, versão do aplicativo, localização, uso de aplicativos,  linguagem do sistema, versão de TPNS, modelo de dispositivo móvel, marca) | Se você optar por  enviar notificações por push para um grupo de usuários específico, então  usaremos essas informações para fornecer as notificações por push a esse  grupo. |

| **Informações pessoais** | **Uso**                                                      |
| ------------------------ | ------------------------------------------------------------ |
| IDFA                     | Se você optar por  agrupar o ID do token e o IDFA, processaremos o IDFA associado de acordo com  suas instruções. |
