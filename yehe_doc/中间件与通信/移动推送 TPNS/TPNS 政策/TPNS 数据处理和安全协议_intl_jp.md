## 1.  背景

本モジュールは、お客様が Tencent Push Notiﬁcation Service（以下「**本機能**」）を利用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)のデータ処理とセキュリティ契約に組み込まれています。本モジュールで使用されているが定義されていない用語は、DPSA でそれらに与えられた意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## 2.  処理

弊社は本機能に関連して、次の個人データを処理します。

| <b>個人情報</b>                                                 | <b>用途</b>                                                    |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| プッシュ通知                                 | お客様のご希望に合わせてプッシュ通知を配信するために、プッシュ通知を処理します。プッシュ通知に含まれる内容については、お客様が設定した内容となり、弊社は責任を負いません。 |
| トークン ID（各デバイスに本機能が作成）                      | これは、プッシュ通知が選択したデバイスに確実に配信されるために作成されます。 |
| アカウント / ログイン                                        | お客様がトークン ID と特定のアカウント / ログイン情報（例：WeChat ID）をバンドルすることを選択した場合、弊社はお客様の指示に従って関連するアカウント / ログイン情報を処理します。 |
| 技術情報（プッシュ通知を受け取るデバイスの情報。OS のバージョン、アプリのバージョン、位置情報、アプリの使用状況、システム言語、TPNS のバージョン、モバイル機器のモデル、ブランド） | お客様が特定のユーザーグループにプッシュ通知を送信することを選択した場合、弊社は、この情報を使用してプッシュ通知を当該グループに配信します。 |
| IDFA         | お客様がトークン ID と IDFA をバンドルすることを選択した場合、弊社は、お客様の指示に従って関連する IDFA を処理します。 |
