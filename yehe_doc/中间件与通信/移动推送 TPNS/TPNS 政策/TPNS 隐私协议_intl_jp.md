## 1.  はじめに

本モジュールは、お客様が Tencent Push Notiﬁcation Service（以下「**本機能**」）を利用する場合に適用されます。本モジュールは、[プライバシーポリシー](https://intl.cloud.tencent.com/document/product/301/17345)内のプライバシーポリシーに組み込まれています。本モジュールで使用されているが定義されていない用語は、プライバシーポリシーでそれらに与えられた意味を持つものとします。プライバシーポリシーと本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## 2.  管理者

本モジュールに記載されている個人情報の管理者は、プライバシーポリシーに規定されているとおりです。

## 3.  可用性

本機能は世界中で利用可能です。

## 4.  弊社による個人情報の使用方法

弊社は、以下の法的根拠に従って、以下の方法で個人情報を使用します。

| <b>個人情報</b>                                       | <b>用途</b>                                            | <b>法的根拠</b>                                   |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| TPNS 管理情報 / 設定：アプリケーション名、アンドロイドパッケージ名、iOS バンドル ID、ベンダーチャネルとアプリケーション ID/appkey/AppSecret/ SecretKey | Push Notiﬁcation Serviceを確実に行うため。 | 弊社は、お客様との契約を履行して本機能に基づくサービスを提供するために必要なものとして、この情報を処理します。 |
| 技術情報：お客様が、本機能を通じて配信するプッシュメッセージに関連するサービス利用データ、サービスおよびトークンのログ情報。 | 問題を解決して特定し、本機能の稼働状況をよりよく理解し、本機能の品質を最適化および向上させるため。 | 本機能の運用を改善するためにこの情報を使用することは、弊社の正当な利益につながります。 |

プライバシーポリシーに規定されている通り。

## 7.  データ保持

弊社は、以下の条件に従って個人情報を保持します。

| <b>個人情報</b> | <b>保持方針</b>                              |
| ------------ | ------------------------------------------------------------ |
| 管理情報 | このデータは以下の期間まで保持されます。(i) お客様が TPNS 管理ポータルから当該情報を削除することを選択し、それに伴い情報が削除される、または (ii) 弊社が本機能をお客様に提供しなくなり、弊社のサービスが終了した日から 30 日以内に削除されます。 |
| 技術情報   | 作成日から 90日  |
