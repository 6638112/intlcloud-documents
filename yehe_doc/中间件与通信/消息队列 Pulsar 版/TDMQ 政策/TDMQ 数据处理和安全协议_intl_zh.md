## 1. 背景

本模块在您使用腾讯云消息队列 (TDMQ)（简称“**功能**”）时适用。本模块已纳入 [**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347) 的数据隐私和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2. 处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                 |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据：** 每秒执行事务数 (TPS) 配额、存储配额、主题配额、消息持续时间、群集名称、命名空间策略、主题名称、主题类型、订阅名称、角色密钥、角色登录认证    | 我们仅为了按照您的特定配置向您提供本功能而处理这些数据。   |
| **消息数据：** 消息属性（例如消息长度、消息 ID）、消息密钥、消息内容   | 仅在向您提供本功能时，我们才会处理此数据。 |

## 3. 服务区域

见 DPSA 中的规定。

## 4. 分包处理方

见 DPSA 中的规定。

## 5. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述：

| **个人信息** | **保留政策**                     |
| ------------------------ | ------------------------------------------------------------ |
| 配置数据       | 我们会一直保留此类数据，直到您将它们手动删除。此外，若您删除帐户或终止使用本功能，我们将在 7 天内删除此数据。 |
| 消息数据             | 我们会一直保留此类数据，直到您接收和使用消息。 |

您可以根据 DPSA 请求删除此类个人数据。

## 6. 特殊条款

您必须确保使用本功能的最终用户至少达到个人可同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
