## 1. 배경

이 모듈은 Tencent 분산 메시지 큐(TDMQ)를 사용하는 경우 적용됩니다("**기능**"). 이 모듈은 [**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347) 에 있는 데이터 개인정보 보호 및 보안 계약에 통합됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 것과 같은 의미를 갖습니다. DPSA와 이 모듈 간에 상충이 있는 경우, 이 모듈은 상충되지 않는 범위 내에서 적용됩니다.

## 2. 처리

당사는 시스템과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **이용**                                 |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **구성 데이터:** 초당 트랜잭션 (TPS) 할당량, 저장소 할당량, 토픽 할당량, 메시지  지속 시간, 클러스터 이름, 네임 스페이스 정책, 토픽 이름, 토픽 유형,  구독 이름, 역할 토큰, 역할 인증 | 당사는 사용자의  고유한 구성에 따라 사용자에 대한 기능 제공 목적으로만 이 데이터 를 처리합니다. |
| **메시지 데이터:** 메시지  속성(예: 메시지 길이, 메시지 ID), 메시지 키, 메시지  내용 | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터 를 처리합니다. |

## 3. 서비스 지역

DPSA에 명시된 바와 같습니다.

## 4. 하위 프로세서

DPSA에 명시된 바와 같습니다.

## 5. 데이터 보존

당사는 시스템과 관련하여 처리된 개인 데이터를 다음과 같이 저장합니다.

| **개인 정보** | **보존  정책**                    |
| ------------------------ | ------------------------------------------------------------ |
| 구성 데이터       | 당사는 귀하가 이러한 데이터를 수동으로 삭제할 때까지  이러한 데이터를 보존합니다. 귀하가 계정을 삭제하거나  기능의 사용을 종료하는 경우 당사는 7일 후에 이러한 데이터를 삭제합니다. |
| 메시지 데이터             | 당사는 귀하가 메시지를 수신 하고 사용할 때까지 이러한 데이터를 보관합니다. |

귀하는 DPSA에 따라 이러한 개인 데이터의 삭제를 요청할 수 있습니다.

## 6. 특수 조건

개인 데이터 처리에 동의할 수 있는 최소 연령을 충족하는 최종 사용자만 이 시스템을 사용할 수 있습니다. 이는 최종 사용자의 해당 지역에 따라 다를 수 있습니다.
