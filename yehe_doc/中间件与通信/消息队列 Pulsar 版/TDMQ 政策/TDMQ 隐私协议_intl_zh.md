## 1. 简介

本模块在您使用腾讯云消息队列 (TDMQ)（简称“**功能**”）时适用。本模块已纳入 [**隐私政策**](https://intl.cloud.tencent.com/document/product/301/17345) 的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3. 可用性

本功能可供全球范围内的用户使用。

## 4. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **日志数据：** 时间戳、日志信息（包括连接信息，例如 IP 地址、消息长度、消息生成时间）  | 我们为向您提供本功能和排除故障而使用此类信息。     <br/> 请注意，此数据存储在我们的日志服务 (CLS) 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。   |
| **监控数据：** 生成率、使用率、生成吞吐量（MQ 客户端产生的消息的字节数）、使用吞吐量（MQ 客户端使用的消息的字节数）、消息存储使用情况、已解包的消息数     | 我们为向您提供本功能和排除故障而使用此类信息。     <br/>请注意，此数据存储在我们的云监控 (CM) 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。   |



## 5. 我们如何存储和分享个人信息

见隐私政策中的规定。

## 6. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息** | **保留政策**                     |
| ------------------------ | ---------------------------------------- |
| 日志数据                 | 存储 15 天。                      |
| 监控数据             | 存储 30 天。 |
