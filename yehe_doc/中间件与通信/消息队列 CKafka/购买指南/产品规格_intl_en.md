CKafka instances are divided into Standard Edition and Pro Edition according to their specifications. The differences between the two editions are as follows:


| Items                              | Pro Edition                                                       | Standard Edition                                                     |
| --------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| Version | <li>Compatible with open-source versions 0.10, 1.1, 2.4, and 2.8. </li><li>You can select one of the versions upon purchase.</li> | <li>Compatible with open-source versions 0.10 and 1.1, with v1.1 installed by default. </li><li>Customized versions are not supported.</li> |
| Instance type                          | Dedicated instance             | Shared physical node resource                                             |
| Stability SLA                        | 99.995%                                                      | 99.95%                                                       |
| Bandwidth specification range                     | Up to 20,000 MB/s. For more information, see [Billing Overview](https://intl.cloud.tencent.com/document/product/597/11745).         | Up to 150 MB/s. For more information, see [Billing Overview](https://intl.cloud.tencent.com/document/product/597/11745).          |
| Public network bandwidth                          | The public network bandwidth can be increased for a fee. For the specific prices, see [Billing Overview](https://intl.cloud.tencent.com/document/product/597/11745). | 3 Mbps public network bandwidth is provided for free by default.                                    |
| <nobr>Topic/Partition specification</nobr> | <li>The topic/partition capacity is much larger than that on the Standard Edition under the same bandwidth.</li><li>Additional partition packages can be purchased within a certain range to increase the upper limit.</li> | Each model has a fixed upper limit. |
| Scalability | Its scalability is high, and the upper limits of bandwidth, topics/partitions, and disks can be increased separately. | Its scalability is low, and its disk can be expanded separately. |
| Broker repair and upgrade time                   | Quick upgrade           | Subject to shared cluster resources, it takes a long time to upgrade.                         |
| High availability | Custom multi-AZ deployment in the same region is supported to improve the disaster recovery capabilities. | Multi-AZ deployment in the same region is not supported. |
| Advanced features                          | <li>Timed rebalance. You can customize the rebalance execution time during upgrade to avoid business peak hours. For more information, see [Upgrading Instance](https://intl.cloud.tencent.com/document/product/597/40650).</li><li>Advanced monitoring. This includes network stability analysis and request delay analysis. For more information, see [Querying Advanced Monitoring (Pro Edition)](https://intl.cloud.tencent.com/document/product/597/40038).</li><li>The message retention policy can be adjusted dynamically based on disk utilization. For more information, see [Automatic Disk Capacity Expansion](https://intl.cloud.tencent.com/document/product/597/40211).</li><li>The disk capacity can be adjusted dynamically based on disk utilization. For more information, see [Automatic Disk Capacity Expansion](https://intl.cloud.tencent.com/document/product/597/40211).</li><li>Advanced Ops. This includes displaying top 10 topics by disk/traffic usage. For more information, see [Viewing Advanced Ops Features (Pro Edition)](https://intl.cloud.tencent.com/document/product/597/41378).</li><li>Topic traffic throttling rules can be configured. For more information, see [Setting Topic Traffic Throttling Rule](https://intl.cloud.tencent.com/document/product/597/47588).</li> | The message retention policy can be adjusted dynamically based on disk utilization. For more information, see [Automatic Disk Capacity Expansion](https://intl.cloud.tencent.com/document/product/597/40211). |
