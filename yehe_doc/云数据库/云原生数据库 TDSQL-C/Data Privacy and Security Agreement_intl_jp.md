## 1. 背景

 本モジュールは、お客様が Cloud Native Database TDSQL-C（「**本機能**」）を使用する場合に適用されます。本モジュールは、https://intl.cloud.tencent.com/document/product/301/17347にあるデータプライバシーとセキュリティ契約（「**DPSA**」）に組み込まれています。本モジュールで使用されているが定義されていない用語は、DPSA でそれらに与えられた意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。 

## 2. 処理

 弊社は本機能に関連して、次のデータを処理します。

| **個人情報**                                                 | **用途**                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **構成データ**：パラメータ名とパラメータ値                   | 弊社は、お客様の特定の設定に基づいて本機能を提供する目的でのみ、このデータを処理します。 |
| **データベースデータ**：データベースファイル（個人データをデータベースに保存する場合）とシステムファイル | 弊社は、このデータをお客様に本機能を提供する目的でのみ処理します。なお、弊社は、データベースに保存されている個人データがある場合でも、それにアクセスすることはできず、管理もできません。 |
| **データの監視**：メトリックの監視                           | 弊社は、お客様が操作状況を監視できるようにするためにのみ、このデータを処理します。なお、本機能は、お客様が有効にしているクラウド監視機能によって実行されます。この情報は、弊社のクラウド監視機能により保存およびバックアップされます。 |

## 3. サービス地域

 DPSA で規定されている通り。

 DPSA で規定されている通り。Aceville Pte Ltd. を含みます。

## 6. 特別な条件

 お客様は、本機能を、個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが使用することを保証する必要があります。これは、エンドユーザーが所在する法域によって異なる場合があります。
