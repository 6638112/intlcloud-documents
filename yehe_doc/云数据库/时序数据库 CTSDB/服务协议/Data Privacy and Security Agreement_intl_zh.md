 

 

## 1.  背景

本模块仅在您使用腾讯云时序数据库（简称**“功能”**）时适用。本模块包含在位于 https://intl.cloud.tencent.com/document/product/301/17347 的数据隐私和安全协议（简称“**DPSA**”）中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。 

## 2.  处理 

我们将因本功能处理以下数据： 

| **个人信息**                                                 | **用途**                       |
| ------------------------------------------------------------ | ------------------------------ |
| **数据库数据**（如果您在数据库中存储个人数据）：数据库名称和密码（存储在您的数据库中并由您管理） | 我们仅为了归档而处理此类数据。 |

## 3.  服务区域

 见 DPSA 中的规定。

## 4.  分包处理方

 见 DPSA 中的规定。

## 5.  数据保留

我们将存储因本功能处理的个人数据，具体如下所述：

| **个人信息** | **保留政策**                                                 |
| ------------ | ------------------------------------------------------------ |
| 数据库数据   | 在您使用本功能期间存储（或存储至本功能被暂停或到期），除非您提前删除。如果您不续订本功能，数据将在本功能的最后一个服务期间结束之日后四 (4) 天内自动删除。 |

您可以根据 DPSA 请求删除这些个人数据。 

## 6.  特殊条款

 您必须确保使用本功能的最终用户至少达到了个人可以同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
