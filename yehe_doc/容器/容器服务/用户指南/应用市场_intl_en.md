
The Tencent Kubernetes Engine (TKE) application market provides a variety of products and services including Helm Chart, Tencent Container Registry (TCR), and software services that are classified by cluster type or application scenario. This document describes how to create applications quickly by using the application market in the TKE console.



## Viewing an Application

1. Log in to the [TKE Console](https://console.cloud.tencent.com/tke2).
2. Click **Marketplace** in the left sidebar to go to the **Marketplace** management page, as shown in the following figure.
![](https://main.qcloudimg.com/raw/383f2bb6305e3c15c7e61bd107d48784.png)
You can perform the following operations on the **Marketplace** management page:
 - Filter applications: filter applications by cluster type or application scenario, or by entering keywords.
    - Cluster type: cluster, elastic cluster, and edge cluster
    - Application scenario: database, big data, tools, log analysis, monitoring, CI/CD, storage, networks, and blogs
 - View applications: click the application package to view to go to the application details page.


## Creating an Application

1. On the **Application Management** page, select the desired application package to go to the application details page.
2. On the **Application Details** page, click **Create Application** in the **Basic Info** module.
3. In the displayed **Create Application** window, configure the application, as shown in the following figure.
   ![](https://main.qcloudimg.com/raw/926b52aab01a4849209a609c7e5b0876.png)
4. Click **Create** to create the application.
