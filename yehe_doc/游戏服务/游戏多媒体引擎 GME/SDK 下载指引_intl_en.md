This document describes how to download the SDKs for Tencent Cloud GME.

## Updates

To make it easier for developers to integrate GME, we recommend developers to check [Release Notes](https://intl.cloud.tencent.com/document/product/607/35323) first before downloading any SDKs.

## Download and Use

If you have any problems with your downloaded SDKs or DEMO, see [Download and Use](https://intl.cloud.tencent.com/document/product/607/30256) or [submit a ticket](https://console.cloud.tencent.com/workorder/category) for assistance.
