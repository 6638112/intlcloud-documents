
## 1.HISTÓRICO

Este módulo se aplica se você usa o Game Multimedia Engine (“Recurso”). Este Módulo é incorporado ao Contrato de segurança e processamento de Dados localizado em [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, ele deve ser aplicado conforme a inconsistência.

## 3.REGIÃO DO SERVIÇO
Conforme especificado no DPSA.


## 4.SUB-PROCESSADORES
Conforme especificado no DPSA, incluindo Tencent Cloud Computing (Pequim) Limited.



## 5.RETENÇÃO DE DADOS
Armazenaremos dados pessoais processados em conexão com o Recurso de acordo com suas preferências selecionadas. No caso de mensagens de voz (modo offline), essas mensagens são armazenadas por um período de 90 dias a partir da data de coleta. Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.



## 6.CONDIÇÕES ESPECIAIS
Você deve garantir que este recurso seja usado somente por usuários finais com a idade exigida para que possam consentir com o processamento de seus dados pessoais. Aqui pode haver diferença dependendo da jurisdição em que o usuário final está localizado.
