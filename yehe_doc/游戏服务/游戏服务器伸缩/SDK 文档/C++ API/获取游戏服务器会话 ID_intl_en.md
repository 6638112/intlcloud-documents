### API Description
This API is used to get the ID of a game server session.		

### Parameter Description
No parameters.



### Returned Value Description
Game server session ID in [StringOutcome](https://intl.cloud.tencent.com/document/product/1055/36700#jtlx) type.

### Use Cases
```
TencentCloud::Gse::StringOutcome sessionIdOutcome = 
	TencentCloud::Gse::Server::GetGameServerSessionId();
```
