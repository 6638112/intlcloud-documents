# 1. **배경**
이 모듈은 Game Server Elastic-scaling("기능”)을 사용하는 경우에 적용됩니다. 이 모듈은  데이터 개인 정보 보호 및 보안 계약 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347)에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 의미를 갖습니다. DPSA와 이 모듈 간에 충돌이 발생할 경우, 이 모듈은 상충하는 범위까지만 적용됩니다.

# 2. **처리**
당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **이용**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **배경 프로그램 데이터** (룸 관계, 게임 룸의 플레이어 수, 플레이어 입장 및 종료 시간, 고객 입력 플레이어 고유 ID) | 당사는 귀하와 최종 사용자에게 기능을 제공하기 위해서만 이 데이터를 처리합니다. |

# 3. **서비스 지역**
DPSA에 명시된 바와 같습니다.

# 4. **하위 프로세서**
DPSA에 명시된 바와 같이, Aceville Pte Ltd 등.

# 5. **데이터 보유**
당사는 다음과 같이 기능과 관련하여 처리된 개인 데이터를 저장합니다.

- **배경 프로그램 데이터**: 기능을 종료한 후 30일까지 저장됩니다.
귀하는 DPSA에 따라 이러한 개인 데이터의 삭제를 요청할 수 있습니다(기본적으로 이러한 데이터는 위에 명시된 대로 보관됩니다).


# 6. **특수 조건**
개인 데이터 처리에 동의할 수 있는 최소 연령을 만족하는 개인이나 부모의 동의를 얻은 최소 연령 미만의 개인인 최종 사용자만 이 기능을 사용할 수 있습니다. 이는 최종 사용자의 해당 지역에 따라 다를 수 있습니다.
