
## 操作シナリオ

このドキュメントでは、主に「ビギナー向け事例」を通して、GSEのメインプロセスを体験し、ゲームサーバーホスティングサービスを始める方法を説明しています。

## 操作手順

### 手順1：デモパッケージのアップロード
 1. [GSEコンソール](https://console.cloud.tencent.com/gse)にログインし、左側メニューの**ビギナー向け事例**をクリックします。
 2. 左上のサービスエリアを選択し、**デモパッケージのワンクリックアップロード**をクリックして、デモパッケージをアップロードし、作成完了と表示された後、**次のステップ**をクリックします。
    ![](https://main.qcloudimg.com/raw/aba9f81e1e145dd9233c962a71f9938e.png)
    - GSE 为您提供的示范包已经集成了 gRPC 框架，游戏进程和 GSE 通过 gRPC 通信。
    - 如果是您自行创建，请您参考 [创建生成包](https://intl.cloud.tencent.com/document/product/1055/36674) 操作指南。

### 手順2：サーバーフリートの作成
**サーバーフリートのワンクリック作成**をクリックすると、順に「新規作成（進行中）」、「ダウンロード中（進行中）」が表示されます。「サーバーフリートの作成完了」が表示された後、**次のステップ**をクリックします。
 - サーバーフリートの作成には次の6つの手順が必要となります：新規作成（完了）、ダウンロード中（完了）、検証中（完了）、発行中（完了）、アクティベート中（完了）、アクティブ（完了）。
 - 現在の状態：作成中、サーバーフリートの作成が完了しました。
   ![](https://main.qcloudimg.com/raw/8854d79ca8ce4f1edbb4f93529766bdb.png)
   ![](https://main.qcloudimg.com/raw/013f88008a3833369c8ab2ca1ae8a0a9.png)
   ![](https://main.qcloudimg.com/raw/d395303b0afa601cd1e7bd29713816e2.png)
   - サーバーフリートを作成し、デモパッケージをサーバーフリート上にデプロイしました。
   - サーバーフリートは、自動スケーリング機能を持つサーバー群で、デモパッケージは容易にグローバル展開できます。
   - ご自身で作成される場合は、[サーバーフリート作成](https://intl.cloud.tencent.com/document/product/1055/36675)操作ガイドをご参照ください。

### 手順3：ゲームサーバーセッションの作成とプレイヤーセッション
- **ゲームサーバーセッションの作成**をクリックし、ゲームサーバーセッションの作成を表示させます。
  ![](https://main.qcloudimg.com/raw/00e1ad7d8190f60cf021f5a1ec82d437.png)

    - 该操作调用“创建游戏服务器会话”云 API，GSE 创建一个游戏服务器会话，为其分配一个服务进程。
    - 如果是您自行创建，请您参考 [创建游戏服务器会话](https://intl.cloud.tencent.com/zh/document/product/1055/37139) API 文档。 
- **プレイヤーセッションの作成**をクリックし、プレイヤーセッションの作成を表示させます。
  ![](https://main.qcloudimg.com/raw/3ac88d02e60888dafce042625927cba4.png)
    - 该操作调用“加入游戏服务器会话”云 API，GSE 创建一个玩家会话，将玩家加入到游戏服务器会话中。
    - 如果是您自行创建，请您参考 [加入游戏服务器会话](https://intl.cloud.tencent.com/zh/document/product/1055/39130) API 文档。

### 手順4：クライアントのゲームサーバー接続
   **クライアントページへジャンプ**をクリックし、クライアントのゲームサーバー接続ページへ進みます。**接続**をクリックすると、サーバー：接続完了が表示されます。
![](https://main.qcloudimg.com/raw/c4975e69e97cfb875b72af739c290888.png)

>?




>- プレイヤーセッションの作成完了後、プレイヤー（クライアント）は1分以内にサーバーへ接続する必要があります。そうでない場合は無効となります。
>- このデモパッケージはチャットサービスです。複数のプレイヤーがこのサーバーへ接続すると、チャットを行うことが可能となります。
>
 以上の4つの手順により、GSEの接続プロセス全体をシミュレーションしました。より詳しい情報については、[開発ガイド](https://intl.cloud.tencent.com/zh/document/product/1055/36683)ドキュメントをご参照ください。
