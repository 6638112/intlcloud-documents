
## 概要

このドキュメントでは、主に「デモ」を使用して、GSEのメインプロセスを体験し、ゲームサーバーホスティングサービスを始める方法を説明します。

## 操作手順

### 手順1：デモパッケージのアップロード
 1. [GSEコンソール](https://console.cloud.tencent.com/gse/asset)にログインし、左側メニューの【デモ】をクリックします。
 2. 左上隅のサービスリージョンを選択し、【デモパッケージのワンクリックアップロード】をクリックして、デモパッケージをアップロードし、作成完了と表示された後、【次のステップ】をクリックします。
    ![](https://main.qcloudimg.com/raw/aba9f81e1e145dd9233c962a71f9938e.png)
 >?
    - GSEが提供するデモパッケージには、ゲームプロセスがGSEと通信するためのgRPCフレームワークがすでに統合されています。
    - 自分で作成する場合は、[アセットパッケージの作成](https://intl.cloud.tencent.com/document/product/1055/36674)をご参照ください。

### 2：サーバーフリートの作成
【サーバーフリートのワンクリック作成】をクリックすると、順に「新規作成（進行中）」、「ダウンロード中（進行中）」が表示されます。「サーバーフリートの作成完了」が表示された後、【次のステップ】をクリックします。
 - サーバーフリートの作成には次の6つの手順が必要となります：新規作成（完了）、ダウンロード中（完了）、検証中（完了）、発行中（完了）、アクティベート中（完了）、アクティブ（完了）。
 - 現在の状態：作成中、サーバーフリートの作成が完了しました。
![](https://main.qcloudimg.com/raw/8854d79ca8ce4f1edbb4f93529766bdb.png)
![](https://main.qcloudimg.com/raw/013f88008a3833369c8ab2ca1ae8a0a9.png)
![](https://main.qcloudimg.com/raw/d395303b0afa601cd1e7bd29713816e2.png)
 >?
   - サーバーフリートを作成し、デモパッケージをサーバーフリートにデプロイされます。
   - サーバーフリートは、自動スケーリング機能を持つサーバー群で、デモパッケージは容易にグローバル展開できます。
   - 自分で作成する場合は、 [サーバーフリートの作成](https://intl.cloud.tencent.com/document/product/1055/36675)をご参照ください。

### 手順3：ゲームサーバーセッションとプレーヤーセッションの作成
- 【ゲームサーバーセッションの作成】をクリックすると、ゲームサーバーセッションが正常に作成されたことを示すメッセージが表示されます。 
![](https://main.qcloudimg.com/raw/00e1ad7d8190f60cf021f5a1ec82d437.png)
 >?
    - この操作は「 CreateGameServerSession TencentCloud API 」を呼び出し、GSEはゲームサーバーセッションを作成し、それにサービスプロセスを割り当てます。
    - 自分で作成する場合は、APIドキュメント[CreateGameServerSession](https://intl.cloud.tencent.com/zh/document/product/1055/37139)をご参照ください。 
- 【プレイヤーセッションの作成】をクリックすると、プレーヤーセッションが正常に作成されたことを示すメッセージが表示されます。
![](https://main.qcloudimg.com/raw/3ac88d02e60888dafce042625927cba4.png)
>?
    - この操作は「JoinGameServerSession TencentCloud API」を呼び出し、GSEはプレーヤーセッションを作成し、プレーヤーをゲームサーバーセッションに追加します。
    - 自分で作成する場合は、APIドキュメント[JoinGameServerSession](https://intl.cloud.tencent.com/zh/document/product/1055/39130)をご参照ください。

### 手順4：クライアントをゲームサーバーに接続
   【クライアントページへジャンプ】をクリックし、クライアントのゲームサーバー接続ページへ進みます。【接続】をクリックすると、サーバー：接続完了が表示されます。 

>?
>
>- プレイヤーセッションの作成完了後、プレイヤー（クライアント）は1分以内にサーバーへ接続する必要があります。そうでない場合は無効となります。
>- このデモパッケージはチャットサービスです。複数のプレイヤーがこのサーバーへ接続すると、チャットを行うことが可能となります。

 上記の4つの手順はGSE統合プロセス全体をシミュレートします。より詳細な説明をご覧になる場合は[開発ガイド](https://intl.cloud.tencent.com/zh/document/product/1055/36683)をご参照ください。
