This document describes how to view the total CVM usage period and network traffic and other information of server fleets in different regions through the statistics feature.
## Prerequisites

You have [created a server fleet](https://intl.cloud.tencent.com/document/product/1055/36675).

## Directions

### Statistics

1. Log in to the [GSE Console](https://console.cloud.tencent.com/gse/asset) and click **Statistics** on the left sidebar.
2. In the top-left corner of the page, select the **region** where the target server fleet resides to enter the statistics page of the region and view **Usage Statistics** and **Period History** information.
![](https://main.qcloudimg.com/raw/45861b47b04d408e8ba18c27cbce9488.png)
 - **Usage Statistics**: select a **query time period** and **server fleet name** to get the CVM usage period and network traffic information of the fleet in the specified time period.
![](https://main.qcloudimg.com/raw/94ca7fefd9398c84a9b54ad0c80e04c0.png)
>?You can get a detailed data sheet by clicking "Download".
 - **Period History**: select a **query time period** and **server fleet name** to get the fleet list details, including fleet ID, instance ID, instance IP, start time, end time, run duration, and operating traffic.
![](https://main.qcloudimg.com/raw/8b6a20c97439d069739dde098e0854cf.png)
