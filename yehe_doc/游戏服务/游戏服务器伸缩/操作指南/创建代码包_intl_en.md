## Overview
This document describes how to create an asset package.

## Prerequisites
You have completed the [GSE Application Form](https://intl.cloud.tencent.com/apply/p/k0b6pvbhs6) and your application has been approved.

## Directions
1. Log in to the [GSE Console](https://console.cloud.tencent.com/gse/asset) and click **Asset Package** on the left sidebar.
2. Select the service region in the top-left corner and click **Create**.
You can create an asset package by uploading one on the page or through the command line.
 - **Page Upload**: enter the basic package information, the package name, the package version, and the OS, select the submission method, upload the code, and click **OK**.
![](https://main.qcloudimg.com/raw/65cf10dd55e5fb30b5dd05e320c1b13b.jpg)
 - **CLI Upload**: create an asset package with [TCCLI](https://cloud.tencent.com/document/product/440/39027).
![](https://main.qcloudimg.com/raw/58b995e5e3032e8c54dfb7ea72435ff6.jpg)
3. After the asset package is successfully created, you can edit or delete it, [create a server fleet](https://intl.cloud.tencent.com/document/product/1055/36675) for it, or perform other operations on it on the asset package page.
