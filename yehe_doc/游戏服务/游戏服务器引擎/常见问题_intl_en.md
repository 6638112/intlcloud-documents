### What are the purposes of the game server hosting service?
It is a dedicated game server hosting service that supports stateful game server deployment, auto scaling, and nearby scheduling.

### What is a game server queue? What does it do?
A game server queue is actually a queue of server fleets with priority configured. It can contain server fleets around the world to implement nearby scheduling and disaster recovery.


### In which regions is GSE available?
Currently, it is available in the Shanghai and North America regions. If you need to use it in other regions, please [submit a ticket](https://console.cloud.tencent.com/workorder/category) for application.

### Does GSE support custom frameworks?
Yes.
