### API Description
This API is used to get the ServerSDK version information.


### Parameter Description
No parameters.


### Returned Value Description
Version number in [StringOutcome](https://intl.cloud.tencent.com/document/product/1055/36700#jtlx) type.


### Use Cases
```
TencentCloud::Gse::StringOutcome SdkVersionOutcome = 
   TencentCloud::Gse::Server::GetSdkVersion();
```
