

クライアント APIはTencentCloud APIの形式で提供されます。具体的には次の3つの呼び出し方法があります。

### 1. SDKを介した呼び出し
Tencent Cloud Developer Tool Kit（SDK）3.0を使用してクライアントのTencentCloud APIを呼び出します。サポート言語には、PHP、Python、Java、Go、.NET、Node.js、C ++が含まれます。 
 >? GSEはSDK 3.0バージョンをサポートしており、具体的な操作については、 SDK の概要をご参照ください。

### 2. オンラインデバッグ
API Explorerツールを使用してクライアントTencentCloud APIを呼び出します。このツールは、オンラインコール、署名検証、SDKコードの生成およびクイックAPI検索などの機能を提供します。

 >?[API 3.0 Explorer](https://console.cloud.tencent.com/api/explorer?Product=gse&Version=2019-11-12&Action=DeleteScalingPolicy&SignVersion=) 内で「Game Server Engine」製品を選択し、「コンソール関連インターフェース」または「サービス管理関連インターフェース」内のTencentCloud APIを選択すれば、オンラインデバッグを実行できます。

### 3. 直接パッケージ
ドメイン名とインターフェース名のHTTPリクエスト方法を介してクライアントのTencentCloud APIを呼び出します。

 >?GSE のTencentCloud API はすでに3.0バージョンにアップグレードされています。具体的な操作については、 TencentCloud API 呼び出し方法をご参照ください。
