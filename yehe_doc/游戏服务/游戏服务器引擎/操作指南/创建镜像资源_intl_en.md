


This document describes how to use image resources to create a server fleet.

## Prerequisites

- You already have [Cloud Virtual Machine](https://intl.cloud.tencent.com/product/cvm) instances.

## Directions

1. Log in to the [CVM console](https://console.cloud.tencent.com/cvm/instance/index?rid=1) and click **Instances** on the left sidebar to enter the instance management page.  
2. Select the instance to create a custom image. For detailed directions, see [Creating Custom Images](https://intl.cloud.tencent.com/document/product/213/4942).
3. After the image is successfully created, upload the image resource to your Tencent Cloud CVM, and save the launch process in the `/local/game/` path for GSE.
