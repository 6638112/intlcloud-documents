
## 1\. 简介

本模块仅在您使用 Game Multimedia Engine（简称“**功能**”）时适用。本模块已纳入位于  [（“**隐私政策**”）](https://intl.cloud.tencent.com/document/product/301/17345)部分的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，本模块适用。

## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. 可用性

本功能可供全球用户使用。

## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **控制台配置数据**（选择功能内特定服务的开/关；功能内特定服务的计费模式） | 我们使用此信息来维护您的配置设置，确保发现和补救程序错误。<br>请注意，此类数据备份在云数据库 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合约向您提供本服务而言是必要的。|
| **语音聊天室配置数据**（房间语音质量类型、房间 ID） | 我们为向您提供本功能而使用这些信息。<br>请注意，此数据存储和备份在我们的日志服务 (CLS) 功能和云数据库 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合约向您提供本服务而言是必要的。 |
| **日志数据**（用户访问时间；最终用户设备信息（型号、操作系统、CPU 信息、内存信息）、网络状况（Wifi、4G）、网络丢包率、连接质量统计、IP 地址） | 我们使用此信息来确保消息传递，并确保发现和补救程序错误。 | 我们会处理此类信息，因为这对于我们履行与您签订的合约向您提供本服务而言是必要的。 |



## 5\. 我们如何存储和共享个人信息

见隐私政策中的规定。

## 6\. 我们如何共享个人信息

按照《隐私政策》的规定，包括 Tencent Cloud Europe B.V.、Tencent Cloud LLC、Tencent Korea Yuhan Hoesa、Aceville Pte. Ltd 以及 Aceville Pte. Ltd。

## 7\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息**               | **保留政策**                                         |
| -------------------------------------- | ------------------------------------------------------------ |
| **控制台配置数据** | 在持有帐户的期间保存，然后此后 7 天删除。 |
| **语音聊天室配置数据**      | 存储 15 天。                                          |
| **日志数据**                           | 最多存储 15 天。                                    |
