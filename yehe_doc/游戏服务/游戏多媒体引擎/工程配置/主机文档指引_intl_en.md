This document lists the documentation for the GME SDK for Nintendo Switch.

## Nintendo Switch
To use the GME SDK on the Nintendo Switch platform, you need to configure your project according to the project configuration document first.

[Project Configuration](https://intl.cloud.tencent.com/document/product/607/36215)


If you use the Unity engine for development, please configure your project as instructed in the following document:

[Project Configuration (for Unity)](https://intl.cloud.tencent.com/document/product/607/36214)


After the configuration is completed, please integrate the SDK as instructed in the API documentation.

[API Documentation](https://intl.cloud.tencent.com/document/product/607/36216)
