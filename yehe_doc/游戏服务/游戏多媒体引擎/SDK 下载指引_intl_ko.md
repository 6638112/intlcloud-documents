본문에서는 Tencent Cloud Game Multimedia Engine(GME)용 SDK를 다운로드하는 방법을 설명합니다.

## 변경 히스토리

Tencent Cloud GME 제품을 편리하게 액세스하기 위해 SDK를 다운로드하기 전에 먼저 [Release Notes](https://intl.cloud.tencent.com/zh/document/product/607/35323)를 확인하십시오.

## 신규 사용자 시작하기

SDK를 처음 사용하는 경우 [User Tutorial](https://intl.cloud.tencent.com/document/product/607/39696)을 참고하십시오.

## 다운로드 및 사용

### Demo 사용

다운로드한 SDK 또는 Demo를 사용할 때 발생하는 문제는 [Demo](https://intl.cloud.tencent.com/document/product/607/39521)를 참고하거나 지원을 위해 [온라인 문의](https://console.cloud.tencent.com/workorder/category) 하시기 바랍니다.

> ?다운로드한 Demo를 컴파일하고 실행하려면 관련 문자열을 신청한 SDKAppid 및 Key로 바꿔야 합니다. 예를 들어 Unity Demo를 사용하려면 코드 파일 **UserConfig.cs**를 수정해야 합니다. 서비스 신청에 대한 자세한 내용은 [Voice Service Activation Guide](https://intl.cloud.tencent.com/zh/document/product/607/10782)를 참고하십시오.

### Demo 디버깅

디버깅을 수행할 때 다음 문서를 참고할 수 있습니다.

- 방 입장 실패 문제는 [Room Entering Failed](https://intl.cloud.tencent.com/document/product/607/39523)를 참고하십시오.
- 소리가 나오지 않는 문제는 [Sound and Audio](https://intl.cloud.tencent.com/document/product/607/39524)를 참고하십시오.
- 서비스 호출 오류는 [Error Codes](https://intl.cloud.tencent.com/document/product/607/33223)를 참고하십시오.
- 문제가 지속되면 로그를 얻은 다음 [티켓을 제출](https://console.cloud.tencent.com/workorder/category)하여 GME 개발자에게 문의하시기 바랍니다.



### Demo 내보내기

Demo를 실행 파일로 내보낼 때 발생하는 문제는 [Program Export](https://intl.cloud.tencent.com/document/product/607/39522)를 참고하십시오.
