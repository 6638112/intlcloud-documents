
### GME(Game Multimedia Engine)에는 어떤 기능이 있습니까?

GME는 실시간 음성 채팅, 음성 메시지, 음성 텍스트 변환 및 음성 분석 필터 기능을 지원합니다.

### GME의 게임 애플리케이션 시나리오는 무엇입니까?

e스포츠, 클랜전 게임, 캐주얼 게임, 마피아 게임 등 입니다.

### 어떤 게임이 GME를 사용합니까?
자세한 사항은 [GME 공식 소개 페이지 - 고객 성공 사례](https://www.tencentcloud.com/products/gme)를 참고하십시오.

### GME는 중국 본토 이외의 국가 및 지역을 지원합니까?

지원합니다. Tencent의 10G 클라우드 기반 데이터 센터와 20개의 BGP 라인을 기반으로 하는 GME는 도메인 이름 가속화를 제공하고 안정적이고 매우 매끄러운 게임 경험을 제공합니다. 또한 타사 데이터 센터에서의 배포를 지원하고 중동, 남미 및 호주와 같은 기존 클라우드 공급업체가 접근하기 어려운 지역을 포괄하여 전세계적으로 서비스를 사용할 수 있도록 지원합니다.

### GME 음성 채팅은 휴대폰 시스템을 지원합니까?
지원합니다.

### GME는 WeChat 미니 게임을 지원합니까?
GME는 현재 WeChat 미니 게임을 지원하지 않고 미니 프로그램만 지원합니다.

### GME는 어떤 게임 엔진과 플랫폼을 지원합니까?

GME는 Unity, Unreal, Cocos2d 엔진과 Windows, Mac, iOS, Android 및 H5 플랫폼을 지원합니다.

### GME는 미니프로그램을 지원합니까?

GME는 미니프로그램 사용을 지원하지만 미니프로그램에서는 풀 스트림(듣기)만 지원하고 푸시 스트림(말하기)은 지원하지 않습니다.

### GME는 미니프로그램 DEMO의 다운로드 주소를 제공합니까?

미니프로그램은 당분간 Demo 다운로드 주소를 제공하지 않습니다.

### Android의 GME와 iOS의 GME는 서로 통신할 수 있습니까?

예. 동일한 SDKAppId를 사용하여 동일한 방에 입장하여 상호 연결할 수 있습니다.

### GME는 휴대폰에서 블루투스 전환을 지원합니까?

지원하지 않습니다. 사운드 재생의 전환은 장치 OS에 따라 다릅니다. GME는 블루투스 장치의 재생을 지원합니다.

### GME 룸은 마이크 시퀀스 기반 노래방을 지원합니까?

GME의 HD 음질은 노래방의 요구 사항을 충족할 수 있지만 마이크 시퀀스는 자체 제품의 App 레이어에서(예: 전달 프로토콜을 통해) 구현되어야 합니다.

### Windows 클라이언트에서 반주를 재생하기 위해 다른 플레이어(예: QQ Player)가 필요한 경우 어떻게 해야 합니까?

자세한 내용은 Windows 플랫폼 반주 기능 API 사용 설명서를 참고하십시오. 타사 플레이어 기능이 포함된 반주는 고급 API를 사용하므로 지원을 위해 [티켓 제출](https://intl.cloud.tencent.com/contact-sales)하고 `tmg_adv_win.h` 헤더 파일을 제공해야 합니다.


### 미성년자를 위한 GME 음성 인식 액세스를 활성화하는 방법은 무엇입니까?
현재 미성년자 음성 인식은 **얼로우리스트**를 통해 평가판으로 개방되어 있습니다.
- 생성된 애플리케이션에 대해 미성년자 음성 인식 서비스를 활성화하고 싶다면 직접 [티켓을 제출하여 신청](https://console.cloud.tencent.com/workorder/category?level1_id=438&level2_id=445&source=0&data_title=%E6%B8%B8%E6%88%8F%E5%A4%9A%E5%AA%92%E4%BD%93%E5%BC%95%E6%93%8EGME&step=1)하십시오.
- 애플리케이션을 생성하지 않으셨다면 먼저 콘솔에 로그인 한 후, [애플리케이션 생성](https://console.cloud.tencent.com/gamegme/create)을 완료하고 [티켓을 제출하여 신청](https://console.cloud.tencent.com/workorder/category?level1_id=438&level2_id=445&source=0&data_title=%E6%B8%B8%E6%88%8F%E5%A4%9A%E5%AA%92%E4%BD%93%E5%BC%95%E6%93%8EGME&step=1)을 통해 평가판 미성년자 음성인식 서비스를 이용해보십시오.



### GME 음성 메시지 및 텍스트로 변환 서비스는 실시간 음성 채팅과 함께 사용할 수 있나요?
예, SDK는 내부적으로 호환 처리를 수행합니다.


### GME 노래방 녹화는 어떤 파일 형식을 지원합니까?
MP3 및 OGG 형식이 지원됩니다.
