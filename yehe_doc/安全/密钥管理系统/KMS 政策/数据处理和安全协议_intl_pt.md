## 1\. CONSIDERAÇÕES INICIAIS

Este Módulo se aplica se você usar o Key Management Service (“**Recurso**”). Este Módulo está incorporado ao Contrato de Privacidade e Segurança de Dados localizado no (“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)”). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.

## 2\. **PROCESSAMENTO**

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                       | **Uso**                                                      |
| ---------------------------------------------- | ------------------------------------------------------------ |
| **Dados de criptografia**: sua CMK (chaves gerenciadas pelo cliente) após a criptografia | Nós processamos esses dados apenas com o propósito de fornecer o Recurso para você, de acordo com sua configuração específica. Observe que esses dados são armazenados com backup no nosso recurso TencentDB for MySQL. Observe também que não temos acesso ao suas CMKs antes da criptografia. |

## 3\. REGIÃO DO SERVIÇO

Conforme especificado no DPSA.

## 4\. SUBPROCESSADORES

Como especificado no DPSA, incluindo Aceville Pte. Ltd.

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações Pessoais**   | **Política de Retenção**                                         |
| -------------------------- | ------------------------------------------------------------ |
| Dados de criptografia            | Armazenado até que você solicite a exclusão, após o qual tais dados serão excluídos dentro de 5 dias úteis, a menos que exigido de outra forma pelas leis de proteção de dados aplicáveis. Quando você não solicitar a exclusão, esses dados serão excluídos após 1 mês após o término do seu uso deste Recurso, a menos que exigido de outra forma pelas leis de proteção de dados aplicáveis. |
| Dados de propriedade de criptografia | Armazenado até que você solicite a exclusão, após o qual tais dados serão excluídos dentro de 5 dias úteis, a menos que exigido de outra forma pelas leis de proteção de dados aplicáveis. Quando você não solicitar a exclusão, esses dados serão excluídos após 1 mês após o término do seu uso deste Recurso, a menos que exigido de outra forma pelas leis de proteção de dados aplicáveis. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. CONDIÇÕES ESPECIAIS

Você deve garantir que esse Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
