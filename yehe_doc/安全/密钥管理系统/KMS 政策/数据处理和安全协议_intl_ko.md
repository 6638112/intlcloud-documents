## 1\. 배경

이 모듈은 Key Management Service (“**기능**”)를 사용하는 경우 적용됩니다. 이 모듈은 ("[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)")에 있는 데이터 개인 정보 보호 및 보안 계약에 포함됩니다. 본 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에서 정의된 것과 같은 의미를 갖습니다. DPSA와 본 모듈 간에 상충이 있는 경우, 해당 상충 범위까지 본 모듈이 적용됩니다.

## 2\. **처리**

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                       | **사용**                                                      |
| ---------------------------------------------- | ------------------------------------------------------------ |
| **암호화 데이터**: 암호화 후 귀하의 CMK | 당사는 귀하의 특정 구성에 따라 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다. 이 데이터는 TencentDB for MySQL 기능에 저장되고 백업됩니다. 당사에서는 암호화 전에 귀하의 CMK에 대해 액세스할 수 없다는 점을 참고해 주십시오. |

## 3\. 서비스 지역

DPSA에 명시된 바와 같습니다.

## 4\. 하위 프로세서

DPSA에 명시된 대로 Aceville Pte. Ltd.를 포함합니다.

## 5\. 데이터 보존

당사는 기능과 관련하여 처리된 개인 데이터를 다음과 같이 저장합니다.

| **개인 정보**   | **보존 방침**                                         |
| -------------------------- | ------------------------------------------------------------ |
| 암호화 데이터            | 귀하가 삭제를 요청할 때까지 저장되며, 해당 데이터 보호법에서 달리 요구하지 않는 한 해당 데이터는 영업일 기준 5일 이내에 삭제됩니다. 귀하가 삭제를 요청하지 않는 경우, 해당 데이터 보호법에서 달리 요구하지 않는 한 해당 데이터는 이 기능 사용이 종료된 후 1개월 후에 삭제됩니다. |
| 암호화 속성 데이터 | 귀하가 삭제를 요청할 때까지 저장되며, 해당 데이터 보호법에서 달리 요구하지 않는 한 해당 데이터는 영업일 기준 5일 이내에 삭제됩니다. 귀하가 삭제를 요청하지 않는 경우, 해당 데이터 보호법에서 달리 요구하지 않는 한 해당 데이터는 이 기능 사용이 종료된 후 1개월 후에 삭제됩니다. |

귀하는 DPSA에 따라 해당 개인 데이터의 삭제를 요청할 수 있습니다.

## 6\. 특수 조건

본 기능을 사용하려는 경우, 개인 데이터 처리에 동의할 수 있는 최소 연령 이상인 최종 사용자여야 합니다. 이는 최종 사용자가 거주하는 관할권에 따라 다를 수 있습니다.
