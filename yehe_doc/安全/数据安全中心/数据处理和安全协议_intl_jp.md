

## 1\. 背景

本モジュールは、お客様がT-Secデータセキュリティセンター（「**本機能**」）を使用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)のデータプライバシーとセキュリティ契約に組み込まれています。本モジュールで使用されていても定義されていない用語は、DPSAで定義された意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## 2\. **処理**

当社は本機能に関連して、次のデータを処理します:

| 個人情報                                     | 用途                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **構成データ:** 顧客のAPPID、UIN、資産識別権限スイッチ、タスクスキャン時間、タスクスキャン頻度、タスクサンプル数、および保存済みタスクサンプルデータの数。 | 弊社は、お客様の特定の設定に基づいて本機能を提供する目的でのみ、このデータを処理します。 |
| **ログデータ：**<br/>1.顧客オペレーションログ：バックエンドロジックの動作時間、インターフェース呼び出しの一意の識別、バックエンドコードの位置決め、動作内容（製品事業データベースとの相互作用、主要事業ノードマーク：インターフェース消費時間統計、購入請求、仕様、期間および更新情報）。<br/>2.通常スケジューリングログ：バックグラウンドロジック操作時間、バックグラウンドコード位置決め、操作内容（製品事業DBとの相互作用、事業主要ノードマーク、システム初期化の開始および終了時刻、タスク開始および終了時刻、スキャンタスクDB内で対応するテーブル総数に対応するスキャンタスク）、SQL情報、SQLパラメータ、内部主要ノードプロンプト情報（ユーザータスク開始時刻、タスク完了時刻、対応テーブルの総数）、顧客APPID、UIN | 当社は、このデータをお客様に機能を提供し、サービスの促進およびトラブルシューティングを実施する目的で使用します。 |
| **使用データ：** お客様が購入したデータ資産の総数、および承認済みの資産数。 | 当社はこのデータを処理することで、使用済みデータ資産の監視を可能にします。 |

## 5\. データ保持

当社は、本機能に関連して処理された個人データを以下のように保管します。

| **個人情報** | **保持ポリシー**                                         |
| ------------------------ | ------------------------------------------------------------ |
| 構成データ       | お客様が本機能を使用する限り、当社はこのデータを保持します。本機能のご使用が終了すると、7日後にこのデータが削除されます。 |
| ログデータ                 | ログデータは、最初に記録されてから60日後に自動的に削除されます。 |
| 利用状況データ               | お客様が本機能を使用する限り、当社はこのデータを保持します。本機能のご使用が終了すると、7日後にこのデータが削除されます。 |

お客様は DPSA に従って、この個人データの削除を要請できます。

## 6\. 特別な条件

お客様は、本機能を、個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが使用することを保証する必要があります。これは、エンドユーザーが所在する法域によって異なる場合があります。
