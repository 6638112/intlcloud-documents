

## 1\. 背景

本模块仅在您使用 T-Sec 数据安全中心（简称“**功能**”）时适用。本模块已纳入 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据隐私和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. **处理**

我们将针对本功能处理以下数据：

| 个人信息                                     | 用途                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据：** 客户的 APPID、UIN、资产识别权限开关、任务扫描时间、任务扫描频率、任务样本量、保存的任务样本数据量。 | 我们仅为了按照您的特定配置向您提供本功能而处理这些数据。 |
| **日志数据：**<br/>1\. 客户操作日志：后台逻辑操作时间、接口调用唯一标识、后台代码定位、操作内容（与产品业务数据库的交互；关键业务节点标记：接口耗时统计、付费购买、规格、持续时间和续订信息）。<br/>2\. 定期调度日志：后台逻辑操作时间、后台代码定位、操作内容（与产品业务数据库的交互；关键业务节点标记：系统初始化的开始和结束时间、任务的开始和结束时间、与扫描任务数据库中的对应表总数相对应的扫描任务）、SQL 信息、SQL 参数、内部关键节点提示信息（用户任务开始时间、任务完成时间、对应表总数）、客户 APPID、UIN | 我们为向您提供本功能、促进服务及排除故障而处理这些数据。 |
| **使用数据：**客户购买的数据资产总数和授权资产数量。 | 我们为能够监控已使用的数据资产而处理这些数据。 |

## 5\. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述：

| **个人信息** | **保留政策**                                         |
| ------------------------ | ------------------------------------------------------------ |
| 配置数据       | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 7 天后删除此数据。 |
| 日志数据                 | 日志数据在首次记录 60 天后会自动删除。 |
| 使用数据               | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 7 天后删除此数据。 |

您可以根据 DPSA 请求删除此类个人数据。

## 6\. 特殊条款

您必须确保使用本功能的最终用户至少达到个人可同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
