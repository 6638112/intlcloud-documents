

## 1\. CONSIDERAÇÕES

Este Módulo se aplica se você usar o Centro de Segurança de Dados T-Sec (“**Recurso**”). Este módulo está incorporado ao contrato de segurança e privacidade de dados localizado no [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo se aplicará conforme a inconsistência.

## 2\. **PROCESSAMENTO**

Processaremos os seguintes dados em conexão com o Recurso:

| Informações Pessoais                                     | Uso                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de configuração**: ID do aplicativo do cliente, UIN, switch de autoridade de identificação de ativos, tempo de varredura de tarefas, frequência de varredura de tarefas, número de amostras de tarefas, número de dados de amostra de tarefa salvos. | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você de acordo com a sua configuração específica. |
| **Dados de registro:**<br/>1\. Registro de operação do cliente: tempo de operação lógica back-end, identificação exclusiva de chamada de interface, posicionamento de código back-end, conteúdo de operação (interação com banco de dados de negócios de produtos; marca de nó de negócios-chave: interface estatísticas demoradas; informações de compra, especificação, duração e renovação de faturamento).<br/>2\. Registro de agendamento regular: tempo de operação da lógica de fundo, posicionamento do código de fundo, conteúdo de operação (interação com o DB do negócio de produtos; marca de nó de negócios-chave: inicialização do sistema início e fim do tempo, início e fim da tarefa, tarefa de varredura correspondente ao número total de tabelas correspondentes na tarefa de varredura DB), informações SQL, parâmetros SQL, informações internas de solicitação de nó-chave (tempo de início da tarefa do usuário, tempo de conclusão da tarefa, número total de tabelas correspondentes), ID do aplicativo do cliente, UIN | Usamos esses dados para fins de fornecer o Recurso a você e para que possamos realizar análises estatísticas para solução de problemas. |
| **Dados de uso:** o número total de ativos de dados adquiridos por um cliente e o número de ativos autorizados. | Processamos esses dados para nos permitir monitorar os ativos de dados que foram usados. |

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações Pessoais** | **Política de Retenção**                                         |
| ------------------------ | ------------------------------------------------------------ |
| Dados de configuração       | Retemos esses dados enquanto você usar o Recurso. Quando seu uso do Recurso é encerrado, excluiremos esses dados após 7 dias. |
| Dados de registro                 | Os dados de registro serão excluídos automaticamente após 60 dias de registro. |
| Dados de uso               | Retemos esses dados enquanto você usar o Recurso. Quando seu uso do Recurso é encerrado, excluiremos esses dados após 7 dias. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. CONDIÇÕES ESPECIAIS

Você deve garantir que este Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
