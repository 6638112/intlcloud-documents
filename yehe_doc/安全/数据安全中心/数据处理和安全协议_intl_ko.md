

## 1\. 배경

이 모듈은 T-Sec 데이터 보안 센터("**기능**")를 사용하는 경우 적용됩니다. 이 모듈은 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 개인 정보 보호 및 보안 계약에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 것과 같은 의미를 갖습니다. DPSA와 이 모듈 간에 상충이 있는 경우, 이 모듈은 상충되지 않는 범위 내에서 적용됩니다.

## 2\. **처리**

당사는 시스템과 관련하여 다음과 같은 데이터를 처리합니다.

| 개인 정보                                     | 이용                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **구성 데이터:** 고객의 APPID, UIN, 자산 식별 기관 스위치, 작업 스캔 시간, 작업 스캔 빈도, 작업 샘플 수, 저장된 작업 샘플 데이터 수. | 당사는 사용자의 고유한 구성에 따라 사용자에 대한 기능 제공 목적으로만 이 데이터를 처리합니다. |
| **로그 데이터:**<br/>1\. 고객 작업 로그: 백 엔드 로직 작동 시간, 인터페이스 호출의 고유 식별, 백 엔드 코드 위치 지정, 작업 내용(제품 비즈니스 데이터베이스와의 상호 작용, 주요 비즈니스 노드 마크: 시간이 많이 걸리는 통계 인터페이스, 청구 구매, 사양, 기간 및 갱신 정보).<br/>2\. 일반 일정 로그: 백그라운드 로직 작동 시간, 백그라운드 코드 위치 지정, 작업 내용(제품 비즈니스 DB와의 상호 작용; 주요 비즈니스 노드 마크: 시스템 초기화 시작 및 종료 시간, 작업 시작 및 종료 시간, 스캔 작업 DB의 총 해당 테이블 수에 해당하는 작업 스캔), SQL 정보, SQL 매개 변수, 내부 주요 노드 프롬프트 정보(사용자 작업 시작 시간, 작업 완료 시간, 해당 테이블의 총 수), 고객 APPID, UIN | 당사는 귀하에 대한 기능 제공 및 서비스 원활화, 그리고 문제 해결을 목적으로 이 데이터를 사용할 수 있습니다. |
| **사용 데이터:** 고객이 구매한 총 데이터 자산 수와 승인된 자산 수. | 당사는 사용된 데이터 자산을 모니터링하기 위해 이 데이터를 처리합니다. |

## 4\. 하위 프로세서

DPSA에 명시된 바와 같습니다.

## 5\. 데이터 보존

당사는 시스템과 관련하여 처리된 개인 데이터를 다음과 같이 저장합니다.

| **개인 정보** | **보존 정책**                                         |
| ------------------------ | ------------------------------------------------------------ |
| 구성 데이터       | 당사는 귀하가 해당 기능을 사용하는 동안 그러한 데이터를 보존합니다. 당사는 귀하의 기능 사용이 종료된 시점으로부터 7일 후에 이 데이터를 삭제합니다. |
| 로그 데이터                 | 로그 데이터는 처음 기록된 시점으로부터 60일 후에 자동으로 삭제됩니다. |
| 사용 데이터               | 당사는 귀하가 해당 기능을 사용하는 동안 그러한 데이터를 보존합니다. 당사는 귀하의 기능 사용이 종료된 시점으로부터 7일 후에 이 데이터를 삭제합니다. |

귀하는 DPSA에 따라 이러한 개인 데이터의 삭제를 요청할 수 있습니다.

## 6\. 특수 조건

개인 데이터 처리에 동의할 수 있는 최소 연령을 충족하는 최종 사용자만 이 시스템을 사용할 수 있습니다. 이는 최종 사용자의 해당 지역에 따라 다를 수 있습니다.
