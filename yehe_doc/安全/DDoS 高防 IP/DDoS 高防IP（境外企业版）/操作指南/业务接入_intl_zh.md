1.	登录 [DDoS 高防 IP（境外企业版）](https://console.cloud.tencent.com/ddos/ddos-basic) 控制台 ，在左侧导航中，单击 **DDoS 高防 IP** > **业务接入** > **IP 接入**。
>?DDoS 高防 IP（境外企业版）仅支持 IP 接入。

![](https://qcloudimg.tencent-cloud.cn/raw/10b364ac6ab8e031a9b684773851501f.png)
2. 在 IP 接入页面，单击**开始接入**，进入“绑定资源”弹窗。
![](https://qcloudimg.tencent-cloud.cn/raw/20f3a1e328cf13bb3afd9c614ed6ba36.png)
3. 在“绑定资源”弹窗中，选择关联 Anycast 高防 IP、地区和可用实例后，单击**确定**，即可完成绑定资源。
>?
>- 可用实例支持中国香港、新加坡、首尔、孟买、曼谷、东京、硅谷、弗吉尼亚、法兰克福、莫斯科地区筛选。
>- 当前支持绑定实例类型：云主机（CVM）和负载均衡（CLB）。

![](https://qcloudimg.tencent-cloud.cn/raw/399d71b2b1b2be3c2dd8fe0b56a1d524.png)
