1. Log in to the [Anti-DDoS Advanced (Global Enterprise Edition) console](https://console.cloud.tencent.com/ddos/ddos-basic) and select **Anti-DDoS Advanced** > **Application Accessing** > **IP Access**.
>? Anti-DDoS Advanced (Global Enterprise Edition) only supports IP access.

2. Click **Start Access**.
3. Select an Anycast IP, Anti-DDoS Advanced instance and a region, and click **OK** to bind the resource.
>?
>- You can select any region that is available to your Anti-DDoS Advanced instance.
>- Only the following resource types can be bound: CVM and CLB.
