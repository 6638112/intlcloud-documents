
弹性防护峰值指大禹高防服务可提供抵御攻击流量的能力范围。若攻击流量超过最高防护峰值，则被攻击 IP 将触发封堵。

## 前提条件
您需要成功 [购买 DDoS 高防 IP ](https://intl.cloud.tencent.com/document/product/297/15483)。

## 操作步骤
1. 登录  [DDoS 高防 IP（新版）管理控制台](https://console.cloud.tencent.com/ddos/antiddos-advanced/package)，在左侧导航栏中，单击【实例列表】。
2. 在目标 DDoS 高防 IP 实例所在行的防护规格中，单击弹性峰值后面的编辑图标。
![](https://main.qcloudimg.com/raw/3c5bdc1cfe4dea7116f77e23908c25f8.png)
3. 在设置弹性防护弹框中，根据实际防护需求选择弹性防护峰值。
![](https://main.qcloudimg.com/raw/0a68b2234b956845a0f9494069541b26.png)
4. 选择完成后，单击【确定】即可。
