 Anti-DDoS allows you to limit traffic rate for business IPs and ports.

## Prerequisites
You have purchased an [Anti-DDoS Advanced instance](https://intl.cloud.tencent.com/document/product/297/37241) and set the object to protect.

## Directions
1. Log in to the [Anti-DDoS Advanced Console](https://console.cloud.tencent.com/ddos/antiddos-advanced/package) and select **Anti-DDoS Advanced (New)** > Configurations** on the left sidebar.
2. Select an Anti-DDoS Advanced instance ID in the list on the left, such as "xxx.xx.xx.xx bgpip-000003n2".
![](https://main.qcloudimg.com/raw/dcb8de6c5b81f6f522b25962f4ad3c11.png)
3. Click **Set** in the **IP/Port Speed Limit** section on the right.
![](https://main.qcloudimg.com/raw/ef1dedadf78a150fea126532b79f7f68.png)
4. Click **Create**.
5. In the pop-up window, select a protocol for the port you set, set a rate limit, and then click **OK**.
![](https://main.qcloudimg.com/raw/5d95d9a43dcf820567019c7cce62aca5.png)
6. Now the new rule is added to the list. You can click **Configuration** on the right of the rule to modify it.
![](https://main.qcloudimg.com/raw/3c0af9d892ef890d41c61c2add90b15f.png)
