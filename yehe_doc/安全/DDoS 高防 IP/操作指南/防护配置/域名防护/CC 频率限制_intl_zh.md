

## 前提条件
您需要成功 [购买 DDoS 高防 IP](https://intl.cloud.tencent.com/document/product/297/15483) ，并设置防护对象。

## 操作步骤
1. 登录 [DDoS 高防 IP（新版）管理控制台](https://console.cloud.tencent.com/ddos/antiddos-advanced/package) ，在左侧导航中，选择【防护配置】。
2. 在左边的列表选中高防 IP 的 ID 下面的域名，如"212.64.xx.xx bgpip-000002je" > "http:80" > "www.xxx.com"。
![](https://main.qcloudimg.com/raw/6a4d39de7a74962647c56b0608bb1bf1.png)
3. 右侧卡片中单击“CC 频率限制”卡片中的【设置】，进入频率限制规则列表。
![](https://main.qcloudimg.com/raw/84c6f634c7417bd73dc78df30c26f8b0.png)
4. 单击【新建】，创建频率限制规则，填写相关字段，填写完成后，单击【确定】。
![](https://main.qcloudimg.com/raw/98ed17178231f248c2747e147860bb0e.png)
5. 新建完成后，频率限制列表将增加一条频率限制规则，可以在右侧操作列，单击【配置】，修改频率限制规则。
![](https://main.qcloudimg.com/raw/65c453e0d0d8ec07a795e777f1410fce.png)
