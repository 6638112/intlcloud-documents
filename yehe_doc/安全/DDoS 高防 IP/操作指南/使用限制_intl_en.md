## Scenario
Anti-DDoS Advanced is well-suited for protecting IP addresses/domain names that provide website (layer-7) and non-website (layer-4) businesses in and outside Tencent Cloud.


## Capability
By default, one Anti-DDoS Advanced instance supports a total of 60 forwarding rules for layer-4 access and layer-7 access. An Anti-DDoS Advanced instance supports 500 forwarding rules at most. For non-website (layer-4) protocols, each rule supports 20 source IP addresses or domain names. For website (layer-7) protocols, each rule supports 16 source IP addresses or domain names.

## Blacklist/Whitelist
- For DDoS protection, up to 100 IP addresses can be added to the blacklist and the whitelist in total.
- Adding URLs to the allowlist is not supported.

## Available Regions
Anti-DDoS Advanced is available both in and outside the Chinese mainland. Regions outside the Chinese mainland include Hong Kong (China), Taiwan (China), Singapore, Seoul, Tokyo, Virginia, Silicon Valley, and Frankfurt.
