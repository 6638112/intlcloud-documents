 Anti-DDoS allows you to limit traffic rate for business IPs and ports.

## Prerequisites
You have purchased an Anti-DDoS Advanced (Global Enterprise Edition) instance and set your target to protect.

## Directions
1. Log in to the [Anti-DDoS Advanced (Global Enterprise Edition) Console](https://console.cloud.tencent.com/ddos/ddos-basic) and select **Anti-DDoS Advanced** -> **Configurations** on the left sidebar.
2. Select an Anti-DDoS Advanced instance ID in the list on the left, such as "xxx.xx.xx.xx bgpip-000003n2".
![](https://main.qcloudimg.com/raw/d6586694f3fc6b1cbc6099b6b1498c38.png)
3. Click **Set** in the **IP/Port Speed Limit** section on the right.
![](https://main.qcloudimg.com/raw/daec94b7558b4a9ca2213de692e6211a.png)
4. Click **Create**.
5. In the pop-up window, select a protocol for the port you set, set a rate limit, and then click **OK**.
![](https://main.qcloudimg.com/raw/919e3dc04e38281d04ad58feafff6d1e.png)
6. Now the new rate limiting rule is added to the list, and you can click **Configuration** to modify it.
![](https://main.qcloudimg.com/raw/431338c9232bea891d3880c1bb5cf9e9.png)
