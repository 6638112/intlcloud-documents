 DDoS 边界防护为已接入防护的网站业务提供频率控制防护策略，支持限制源 IP 的访问频率。您可以自定义频率控制规则，检测到单一源 IP 在短期内异常频繁地访问某个页面时，将设置人机校验或丢弃策略。


## 前提条件
您需要成功[ 购买 DDoS 边界防护](https://intl.cloud.tencent.com/document/product/297/42172)，并设置防护对象。


## 操作步骤
1. 登录 [DDoS 边界防护管理控制台](https://console.cloud.tencent.com/ddos/antiddos-edge/policy/ddos) ，在左侧导航中，单击【防护策略】，并选择【CC 防护】。
2. 在左边的列表选中高防实例的 ID，如“edge-xxxxxxx”。
![](https://main.qcloudimg.com/raw/63a3421ccfb92d88e9a053a5a4053389.png)
3. 在右侧卡片中单击“CC 频率控制”卡片中的【设置】，进入 CC 频率控制页面。
![](https://main.qcloudimg.com/raw/9eae775ebe603f38d725ab5a37c6299b.png)
4. 在 CC 频率控制页面中，单击【新建】。
5. 在新建 CC 频率限制弹窗中，填写相关字段，单击【确定】，创建 CC频率控制规则。
![](https://main.qcloudimg.com/raw/f1128c978153cfdf59863a5922931b1b.png)
6. 新建完成后 CC 频率控制列表中，将新增一条 CC 频率控制规则，可以在右侧操作列，单击【配置】，修改 CC 频率控制规则。
![](https://main.qcloudimg.com/raw/8152349ee35d418ccb9b20fbc7f52cdd.png)
