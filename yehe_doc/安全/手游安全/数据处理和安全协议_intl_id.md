## 2.**PEMROSESAN**

Kami akan memproses data berikut sehubungan dengan fitur:

| **Informasi Pribadi**                                        | **Tujuan Penggunaan**                                        |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| Data Pribadi Pengguna Akhir (akun game, file host, layar dalam game; sistem versi, jenis jaringan, merek, arsitektur CPU, ROM, pengenal unik, layar resolusi, waktu boot sistem, kode negara dan bahasa, memori yang tersisa, sisa ruang penyimpanan built-in/eksternal; baik root atau simulator yang berjalan, direktori cache aplikasi, nama paket aplikasi, nama proses, versi aplikasi, waktu startup) | Kami menggunakan informasi ini untuk memberikan Fitur sesuai dengan instruksi Anda. |
