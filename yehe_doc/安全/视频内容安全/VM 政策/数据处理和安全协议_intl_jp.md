## 1\. 背景

本モジュールは、お客様がTian Yu’s video moderation（「**機能**」）を使用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)のデータプライバシーとセキュリティ契約に組み込まれています。本モジュールにおける用語のうち本書内に定義のないものは、DPSA内で定義された意味を有するものとします。DPSAと本モジュールとの間に矛盾がある場合には、その不一致の範囲で本モジュールを優先するものとします。

## 2\. 処理

当社は、本機能に関連して以下のデータを処理します。

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **お客様がキャンセルした機能のタスクの説明に関する情報：** アクション、バージョン、リージョン、タスクID。 | 当社は、お客様の指示およびタスクに基づいて本機能を提供する目的でのみ、このデータを処理します。 <br/>この目的のため、このデータは当社のTencentDB for MongoDB（MongoDB）機能 と統合されますのでご注意ください。 |
| **機能のタスク詳細の説明に関する情報：** アクション、バージョン、リージョン、タスクID、すべてのセグメントを表示。 | 当社は、お客様の指示およびタスクに基づいて本機能を提供する目的でのみ、このデータを処理します。 <br/>この目的のため、このデータは当社のTencentDB for MongoDB（MongoDB）機能 と統合されますのでご注意ください。 |
| **お客様によるビデオモデレーションタスクの作成に関する情報：** アクション、バージョン、リージョン、優先順位、コールバックURL、シード、業種、種類、チャネル、カスタムアプリID、カスタムUIN、カスタムサブアカウントUIN、タスク、データID、名前、入力、種類、URL、バケット情報、バケット、リージョン、オブジェクト。 | 当社は、お客様の指示およびタスクに基づいて本機能を提供する目的でのみ、このデータを処理します。 <br/>この目的のため、このデータは当社のTencentDB for MongoDB（MongoDB）機能 と統合されますのでご注意ください。 |
| **機能のタスクの説明に関する情報：** アクション、バージョン、リージョン、制限、ページトークン、開始時間、終了時間、フィルタ、業種、種類、提案、タスクステータス。 | 当社は、お客様の指示およびタスクに基づいて本機能を提供する目的でのみ、このデータを処理します。 <br/>この目的のため、このデータは当社のTencentDB for MongoDB（MongoDB）機能 と統合されますのでご注意ください。 |

## 5\. データ保持

当社は、本機能に関連して処理された個人データを以下のとおり保管します。

| **個人情報**                                     | **保持ポリシー**                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| タスクのキャンセルに関する情報、タスク詳細の説明に関する情報、ビデオモデレーションタスクの作成に関する情報、タスクの説明に関する情報。 | 当社は本データを6か月間保管し、適用されるデータ保護法によって求められている場合を除き、その期間が過ぎたら自動的に削除します。 |

DPSAに従って、また適用法により当社がかかる削除を行うことが禁止されていない限り、当該個人データの削除を要求することができます。
