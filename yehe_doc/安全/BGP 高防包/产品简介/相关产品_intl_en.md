Anti-DDoS Pro can be activated for the following products:
- [Tencent Cloud Virtual Machine (CVM)](https://intl.cloud.tencent.com/document/product/213) is a scalable cloud computing service that frees you from estimation of resource usage and upfront investment. With Tencent Cloud CVM, you can start CVMs and deploy applications immediately.

- [Cloud Load Balancer (CLB)](https://intl.cloud.tencent.com/document/product/214) is a service that distributes traffic to multiple CVM instances securely and quickly so as to eliminate single points of failure for a higher availability.

- [Web Application Firewall (WAF)](https://intl.cloud.tencent.com/document/product/627) is an AI-based, one-stop web service protection solution.
- [NAT Gateway](https://intl.cloud.tencent.com/document/product/1015) is a service that supports IP address translation and provides the SNAT and DNAT capabilities. It provides secure and high-performance Internet access for resources in VPCs.
- [VPN connection](https://intl.cloud.tencent.com/document/product/1037) is a transfer service based on network tunneling technology that connects local IDCs and resources on Tencent Cloud. It helps build a secure and reliable encrypted tunnel over the Internet quickly.

- [Global Application Acceleration Platform (GAAP)](https://intl.cloud.tencent.com/document/product/608) is a PAAS product that allows optimum access delay for businesses across the globe. Via high-speed connections, cluster forwarding and intelligent routing among global nodes, it allows users in different regions to access the closest nodes and forwards traffic to the origin server, reducing access lag and latency.

- An [Elastic Network Interface (ENI)](https://intl.cloud.tencent.com/document/product/576) is used to bind CVM within a VPC instance, and can be freely migrated among CVMs. ENIs can help configure and manage, as well as develop highly reliable network solutions.
