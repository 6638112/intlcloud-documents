


## Prerequisites
You need to [purchase an Anti-DDoS Pro instance](https://intl.cloud.tencent.com/document/product/1029/36115) and set the protected object first.

## Directions
1. Log in to the [new Anti-DDoS Pro Console](https://console.cloud.tencent.com/ddos/antiddos-native/package) and click **Protection Configuration** on the left sidebar.
2. Select an Anti-DDoS Pro instance ID in the list on the left, such as "bgp-0000008o", and select **IP Port Protection** at the top on the right.
![](https://main.qcloudimg.com/raw/d569c6f266845e5d16c47ed6f744ed9a.png)
3. Click **Set** in the "Protocol Blocking" block on the right to enter the protocol blocking list.
4. Click **Create** to create a protocol blocking rule.
![](https://main.qcloudimg.com/raw/b41ed3445a08df63b20111d67c11de1b.png)
5. After the creation is completed, a protocol blocking rule will be added to the protocol blocking list. You can modify the rule by clicking **Configure** in the "Operation" column on the right.
![](https://main.qcloudimg.com/raw/4d27b76b40f27942e8fcfb1267cf206b.png)
