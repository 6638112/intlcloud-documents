

## Prerequisites
You need to [purchase an Anti-DDoS Pro instance](https://intl.cloud.tencent.com/document/product/1029/36115) and set the protected object first.

## Directions
1. Log in to the [Anti-DDoS Pro Console](https://console.cloud.tencent.com/ddos/antiddos-native/package) and select **Protection Configuration** on the left sidebar.
2. Select an Anti-DDoS Pro instance ID in the list on the left, such as "bgp-000000co", and click **Domain Name Protection** at the top on the right.
![](https://main.qcloudimg.com/raw/29f77e71117c9c4217176f19373cf44e.png)
3. Click **Set** in the "CC Protection" block on the right to enter the CC protection level rule list.
4. Click **Create** to create a CC protection domain name rule and set the protection level.
![](https://main.qcloudimg.com/raw/3eb6c187a8d9eb7dcb8ce8cc1e2ebf3e.png)
5. After the creation is completed, a CC protection domain name rule will be added to the CC protection domain name list. You can modify the CC protection domain name level by clicking **Configure** in the "Operation" column on the right.
![](https://main.qcloudimg.com/raw/5021b84f5f0de3dad64c77b17652855a.png)
