## Scenarios
This document describes how to view instances, and purchase and renew instances and extra packs in the WAF console.

## Directions
### Creating and viewing instances
1. Log in to the [WAF console](https://console.cloud.tencent.com/guanjia/tea-overview) and select **Asset Center** > **Instance Management** on the left sidebar.
2. On the page displayed, click **Create instance** to switch to the purchase page where you can purchase instances as needed. For more details, see [Purchase Guide](https://intl.cloud.tencent.com/document/product/627/47799).
![](https://qcloudimg.tencent-cloud.cn/raw/d49ce4e840451802a72d53790261d07a.png)
3. On the instant management page, select an instance you want to view and click its instance ID.
![](https://qcloudimg.tencent-cloud.cn/raw/a0591e6ab5bc60410ddb2b8515a26f7a.png)
4. The details about the instance and the plan you purchase will be displayed. If you do not want to renew the plan, toggle off the switch ![](https://main.qcloudimg.com/raw/f8730ea596ac7d86131e5726ce67ce77.png).
![](https://qcloudimg.tencent-cloud.cn/raw/c6811632975cf806f1d9e978a0480f8b.png)


### Searching existing instances
Searching WAF instances by region and instance type or keywords is supported.
1. Log in to the [WAF console](https://console.cloud.tencent.com/guanjia/tea-overview) and select **Asset Center** > **Instance Management** on the left sidebar.
2. On the page displayed, you can search your WAF instances by region and instance type or keywords.
 - To filter WAF instances by instance type, you can click the drop-down list in the top left corner and select an instance type.
 ![](https://qcloudimg.tencent-cloud.cn/raw/8c606c3905725075d1b6c8632305a56a.png)
 - To filter WAF instances by region, you can click the drop-down list in the top left corner and select a region.
![](https://qcloudimg.tencent-cloud.cn/raw/cd17355124714cdffdb2c72927fae35e.png)
 - To search WAF instances by keywords, you can enter keywords in an instance ID, instance name or billing method.
![](https://qcloudimg.tencent-cloud.cn/raw/9d6632ed5a02fa342fe5553f4b8af219.png)


### Managing domain names
After you find the instance you want to view, you need to configure domain names for the instance.
1. On the [Instance management page](https://console.cloud.tencent.com/guanjia/tea-instance-new), click **Manage domain name** to switch to the domain name list page.
![](https://qcloudimg.tencent-cloud.cn/raw/9bb245de55915baf00520b221e3d2f25.png)
2. On the [domain name list](https://console.cloud.tencent.com/guanjia/tea-domain) page, you can create, edit and delete a domain name. For more details, see [Domain Name List](https://intl.cloud.tencent.com/document/product/627/47531).

### Renewing instances and purchasing extra packs
After you find the instance you want to view, you can check its plan expiration date and extra pack usage on the [domain name list](https://console.cloud.tencent.com/guanjia/tea-instance-new) page.
![](https://qcloudimg.tencent-cloud.cn/raw/389858e42f8d93ddfc8a540c04a97072.png)
1. If the plan is about to expire, you can click **Renew** to renew it as needed. On the pop-up renewal page, select a validity period before clicking **Renew now**.
![](https://qcloudimg.tencent-cloud.cn/raw/68b6659a129fd13b6d813ba8927e3235.png)
2. If your domain packs or QPS quota is about to run out, you can click **More** to display all billable items, and purchase a domain pack, an extra QPS pack, or the bot and application security service as needed.
>? If the instance or domain pack already expires, please renew it first before you purchase another one.

 - Purchase domain pack: Up to 500 packs can be purchased. After you specify the number of domain packs to purchase, click **Buy now**.
 ![](https://qcloudimg.tencent-cloud.cn/raw/5c288bf8456f32b7b639a8f9024dd842.png)
 - Purchase extra QPS pack: Up to 500 packs can be purchased. After you specify the number of extra QPS packs to purchase, click **Buy now**.
 ![](https://qcloudimg.tencent-cloud.cn/raw/031e1085e76b8459ff4c72e0b14b51f9.png)
 - Bot and application security: Select the protection services as needed, and then click **Buy now**.
![](https://qcloudimg.tencent-cloud.cn/raw/694f1bb9c86b0e65b5bccbcf40300cc0.png)
