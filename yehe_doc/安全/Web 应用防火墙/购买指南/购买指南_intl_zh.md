当前 Web 应用防火墙提供两种产品形态，SaaS型WAF和负载均衡型WAF，两种WAF计费项目基本相同，且计费模式与价格相同.您可以根据不同接入方式选择购买不同类型的 Web 应用防火墙。

Web 应用防火墙支持包年包月（预付费）计费方式+弹性后付费方式，暂不支持按量付费的计费方式。

### 计费方式

Web 应用防火墙采取套餐+扩展包的方式进行计费：

- 您可以按包年或包月的预付费模式进行购买。
- 扩展包：用户选购基础套餐后，可根据需要购买**扩展域名包**、**QPS扩展包**和**安全日志服务扩展包**。
