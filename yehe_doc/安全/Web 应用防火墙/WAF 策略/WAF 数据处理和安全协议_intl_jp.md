## **1.背景**

本モジュールは、お客様がWeb Application Firewall（「**本機能**」）を利用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)のデータ処理とセキュリティ契約に組み込まれています。本モジュールで使用されているが定義されていない用語は、DPSA でそれらに与えられた意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## **2.処理**

弊社は本機能に関連して、次のデータを処理します。


| **個人情報**                                                 | **用途**                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **構成データ** （お客様の保護ドメイン名。セキュリティ防衛ルール。自己構成 IP） | 弊社は、このデータをお客様に本機能を提供する目的でのみ処理します。 |

## **3.サービス地域**

DPSA で規定されている通り。

## **4.復処理者**

DPSA で規定されている通り。Tencent Cloud Computing（北京）Co., Ltd. が含まれます。

## **5.データ保持**

弊社は、本機能に関連して処理された個人データを以下のように保管します。

**構成データ**：お客様が当該データを手動で削除するまで、またはサービス期間終了までに当該データを手動で削除しない場合は、サービス期間終了後 7日間まで。

お客様は DPSA に従って、当該個人データの削除を要請することができます。
