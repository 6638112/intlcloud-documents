## **1. INTRODUÇÃO**

Este módulo se aplica se você usar o Firewall de aplicativos da Web ("**Recurso**”). Este Módulo é incorporado à política de privacidade localizada na [Política de privacidade](https://intl.cloud.tencent.com/document/product/301/17345). Os termos usados mas não definidos neste módulo devem ter o significado que lhes é atribuído na Política de privacidade. No caso de algum conflito entre a Política de privacidade e este módulo, o módulo deve ser aplicado na inconsistência.

## **2. CONTROLADORIA**
O controlador das informações pessoais descritas neste Módulo é o especificado na Política de Privacidade.

## **3. DISPONIBILIDADE**

Este recurso está disponível globalmente para os usuários. 

## **4. COMO USAMOS AS INFORMAÇÕES PESSOAIS**

Usaremos as informações das seguintes formas e de acordo com as seguintes bases legais:

| **Informações pessoais**                                     | **Uso**                                                      | **Base Jurídica**                                            |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados do Log** (Registro de acesso: IP de origem, URL de acesso, Consulta, Remetente, Cookie, Usuário-Agente, X-Forwarder-For, código de resposta WAF, código de resposta do site-fonte, corpo) | Usamos estas informações para garantir que o Serviço funcione conforme necessário e para garantir que os bugs sejam detectados e corrigidos. | Processamos estas informações como necessário para executarmos nosso contrato com você de fornecimento do Serviços. |
| **Dados do Log** (Log de ataque: nome de domínio, tipo de ataque, número de ataques agregados, fonte IP do ataque, ID de regra de visitas, nome da regra de visitas, método de solicitação, nível de risco, tempo de ataque, fonte de correspondência, ação de execução, URL de solicitação, conteúdo do ataque) | Usamos essas informações para garantir que o Serviço funcione como requerido e que os bugs sejam detectados e corrigidos. | Processamos essas informações como necessário para que executemos nosso contrato com você de prestação de serviços. |

## **5. COMO ARMAZENAMOS E COMPARTILHAMOS INFORMAÇÕES PESSOAIS**

Como especificado na Política de privacidade.

## **6. COMO COMPARTILHAMOS INFORMAÇÕES PESSOAIS**
Como especificado na Política de privacidade, inclusive a Tencent Cloud Computing (Pequim) Limited.

## **7. RETENÇÃO DE DADOS**
Reteremos informações pessoais de acordo com o seguinte:

| **Informações pessoais** | **Política de retenção**                            |
| ------------------------ | --------------------------------------------------- |
| Dados do log             | Até 6 meses (dependendo da configuração escolhida). |
