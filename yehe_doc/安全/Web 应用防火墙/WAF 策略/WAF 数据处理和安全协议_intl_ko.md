
## **1.배경**

이 모듈은 Web Application Firewall("**기능**”)을 사용하는 경우 적용됩니다. 이 모듈은 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 처리 및 보안 계약에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 의미를 갖습니다. DPSA와 이 모듈 간에 충돌이 발생할 경우, 이 모듈은 상충하는 범위까지만 적용됩니다.

## **2.처리**

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.


| **개인 정보**                                                | **이용**                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **구성 데이터**(고객 보호 도메인 이름, 보안 방어 규칙, 자체 구성 IP) | 당사는 사용자에게 기능을 제공하기 위해서만 이 데이터를 처리합니다. |


## **3.서비스 지역**

DPSA에 명시된 바와 같습니다.

## **4.하위 프로세서**

DPSA에 명시된 대로, Tencent Cloud Computing (Beijing) Limited 등.

## **5.데이터 보유**

당사는 다음과 같이 기능과 관련하여 처리된 개인 데이터를 저장합니다.

**구성 데이터**: 사용자가 데이터를 수동으로 삭제할 때까지, 또는 서비스 기간이 종료된 후 이러한 데이터를 수동으로 삭제하지 않는 경우, 서비스 기간 종료 후 7일까지.

DPSA에 따라 이러한 개인 데이터의 삭제를 요청할 수 있습니다.
