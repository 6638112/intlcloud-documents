## **1. HISTÓRICO**

Este módulo se aplica se você usar o Web Application Firewall ("**Recurso**”). Este Módulo é incorporado ao Contrato de segurança e processamento de Dados localizado em [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, ele deve ser aplicado conforme a inconsistência.

## **2. PROCESSAMENTO**

Processaremos os seguintes dados em conexão com o recurso:
| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de configuração** (nome de domínio de proteção ao cliente; regras de defesa de segurança; IP auto-configurado) | Apenas processamos esses dados com a finalidade de lhe fornecer o Recurso. |

## **3. REGIÃO DOS SERVIÇOS**

Conforme especificado no DPSA.
## **4. SUBPROCESSADORES**

Conforme especificado no DPSA, incluindo Tencent Cloud Computing (Pequim) Limited.

## **5. RETENÇÃO DE DADOS**

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

**Dados de configuração**: até excluir manualmente tais dados ou, se você não excluir manualmente tais dados até o final do prazo de serviço, até 7 dias após o término do prazo de serviço.

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.
