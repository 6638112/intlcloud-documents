## 1\. CONSIDERAÇÕES

Este Módulo se aplica se você usar o Vulnerability Scan Service (“**Recurso**”). Este módulo está incorporado ao contrato de segurança e privacidade de dados localizado no [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo se aplicará conforme a inconsistência.

## 2\. PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Informações básicas do site:** endereço de ativo, tipo, tempo adicional (se mais recursos forem necessários após a varredura da vulnerabilidade), status de autenticação, status de login simulado, hora da última varredura | Apenas processamos esses dados com a finalidade de fornecer este Recurso a você. Mediante sua solicitação, também processaremos esses dados para solução de problemas.<br/>Observe que esses dados são armazenados em nossos recursos TencentDB for Redis, TencentDB for MySQL e TencentDB for MongoDB para este fim. |
| **Informações básicas do host:** servidor IP, tempo adicional (se mais recursos forem necessários após a varredura da vulnerabilidade), status de autenticação, hora da última varredura | Apenas processamos esses dados com a finalidade de fornecer este Recurso a você. Mediante sua solicitação, também processaremos esses dados para solução de problemas.<br/>Observe que esses dados são armazenados em nossos recursos TencentDB for Redis, TencentDB for MySQL e TencentDB for MongoDB para este fim. |
| **Informações básicas da API:** nome da API, nome do domínio da API, método de autenticação, status de autenticação, tempo adicional (se mais recursos forem necessários após a varredura da vulnerabilidade), resultado da última varredura, hora da última varredura | Apenas processamos esses dados com a finalidade de fornecer este Recurso a você. Mediante sua solicitação, também processaremos esses dados para solução de problemas.<br/>Observe que esses dados são armazenados em nossos recursos TencentDB for Redis, TencentDB for MySQL e TencentDB for MongoDB para este fim. |
| **Dados de configuração da tarefa:** nome da tarefa, tipo, ativo monitorado, plano de varredura, período de varredura, velocidade de varredura, última varredura, status | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você de acordo com a sua configuração específica. <br/>Observe que esses dados são armazenados em nossos recursos TencentDB for Redis, TencentDB for MySQL e TencentDB for MongoDB para este fim. |

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações Pessoais**  | **Política de Retenção**                                         |
| ------------------------- | ------------------------------------------------------------ |
| Informações básicas do site | Retemos esses dados enquanto você usar o Recurso. Você pode excluir manualmente esses dados no Recurso. Uma vez que você optar por excluir tais dados ou encerrar o uso do Recurso, excluiremos esses dados após 7 dias. |
| Informações básicas do host    |  Retemos esses dados enquanto você usar o Recurso. Você pode excluir manualmente esses dados no Recurso. Uma vez que você optar por excluir tais dados ou encerrar o uso do Recurso, excluiremos esses dados após 7 dias.                                                            |
| Informações básicas da API     |    Retemos esses dados enquanto você usar o Recurso. Você pode excluir manualmente esses dados no Recurso. Uma vez que você optar por excluir tais dados ou encerrar o uso do Recurso, excluiremos esses dados após 7 dias.                                                          |
| Dados de configuração de tarefas   |   Retemos esses dados enquanto você usar o Recurso. Você pode excluir manualmente esses dados no Recurso. Uma vez que você optar por excluir tais dados ou encerrar o uso do Recurso, excluiremos esses dados após 7 dias.                                                           |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. CONDIÇÕES ESPECIAIS

Você deve garantir que este Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
