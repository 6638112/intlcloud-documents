## 1\. 背景

このモジュールは、脆弱性スキャンサービス（「**機能**」を使用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)のデータプライバシーとセキュリティ契約に組み込まれています。本モジュールで使用されていても定義されていない用語は、DPSAで定義された意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## 2\. 処理

当社は本機能に関連して、次のデータを処理します:

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **基本的なウェブサイト情報：** アセットアドレス、種類、追加時間（脆弱性スキャン後にさらにリソースが必要な場合）、認証ステータス、模擬ログインステータス、最終スキャン時間 | 私たち このデータを、この機能を提供する目的でのみ処理 します。ご希望により、トラブルシューティングのためにこのデータも処理します。<br/>このデータは、この目的のために、TencentDB for Redis、TencentDB for MySQL、TencentDB for MongoDBの機能に保存されます。 |
| **基本的なホスト情報：** サーバーIP、追加時間（脆弱性スキャン後さらにリソースが必要な場合）、認証ステータス、最終スキャン時間 | 私たち このデータを、この機能を提供する目的でのみ処理 します。ご希望により、トラブルシューティングのためにこのデータも処理します。<br/>このデータは、この目的のために、TencentDB for Redis、TencentDB for MySQL、TencentDB for MongoDBの機能に保存されます。 |
| **基本的なAPI情報：** API名、APIドメイン名、認証方法、認証ステータス、追加時間（脆弱性スキャン後にさらにリソースが必要な場合）、最終スキャン結果、最終スキャン時間 | 私たち このデータを、この機能を提供する目的でのみ処理 します。ご希望により、トラブルシューティングのためにこのデータも処理します。<br/>このデータは、この目的のために、TencentDB for Redis、TencentDB for MySQL、TencentDB for MongoDBの機能に保存されます。 |
| **タスク設定データ：** タスク名、種類、監視対象アセット、スキャン計画、スキャン期間、スキャン速度、最終スキャン、ステータス | 弊社は、お客様の特定の設定に基づいて本機能を提供する目的でのみ、このデータを処理します。 <br/>このデータは、この目的のために、TencentDB for Redis、TencentDB for MySQL、TencentDB for MongoDBの機能に保存されます。 |

## 5\. データ保持

当社は、本機能に関連して処理された個人データを以下のように保管します。

| **個人情報**  | **保持ポリシー**                                         |
| ------------------------- | ------------------------------------------------------------ |
| ウェブサイトの基本情報 | お客様が本機能を使用する限り、当社はこのデータを保持します。この機能では、 このようなデータを手動で削除できます。このようなデータを削除する、もしくは本機能の使用を終了することを選択すると、このデータは7日後に削除されます。 |
| 基本ホスト情報    |  お客様が本機能を使用する限り、当社は当該データを保持します。この機能では、 このようなデータを手動で削除できます。このようなデータを削除する、もしくは本機能の使用を終了することを選択すると、このデータは7日後に削除されます。                                                            |
| 基本API情報     |    お客様が本機能を使用する限り、当社は当該データを保持します。この機能では、 このようなデータを手動で削除できます。このようなデータを削除する、もしくは本機能の使用を終了することを選択すると、このデータは7日後に削除されます。                                                          |
| タスク設定データ   |   お客様が本機能を使用する限り、当社は当該データを保持します。この機能では、 このようなデータを手動で削除できます。このようなデータを削除する、もしくは本機能の使用を終了することを選択すると、このデータは7日後に削除されます。                                                           |

お客様は DPSA に従って、当該個人データの削除を要請することができます。

## 6\. 特別な条件

お客様は、本機能を、個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが使用することを保証する必要があります。これは、エンドユーザーが所在する法域によって異なる場合があります。
