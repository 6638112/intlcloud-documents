## 1\. 背景

本模块仅在您使用漏洞扫描服务（以下称为“**本功能**”）时适用。本模块包含在位于 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据隐私和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. 处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **网站基本信息：** 资产地址、类型、附加时间（漏洞扫描后如需更多资源）、身份验证状态、模拟登录状态、上次扫描时间 | 我们仅为了向您提供本功能而处理这些数据。如您要求，我们还将为排除故障而处理这些数据。<br/>请注意，这些数据存储在我们的腾讯云数据库 Redis、腾讯云数据库 MySQL 和腾讯云数据库 MongoDB 功能中。 |
| **主机基本信息：** 服务器 IP、附加时间（漏洞扫描后如需更多资源）、身份验证状态、上次扫描时间 | 我们仅为了向您提供本功能而处理这些数据。如您要求，我们还将为排除故障而处理这些数据。<br/>请注意，这些数据存储在我们的腾讯云数据库 Redis、腾讯云数据库 MySQL 和腾讯云数据库 MongoDB 功能中。 |
| **API 基本信息：** API 名称、API 域名、身份验证方法、身份验证状态、附加时间（漏洞扫描后如需更多资源）、上次扫描结果、上次扫描时间 | 我们仅为了向您提供本功能而处理这些数据。如您要求，我们还将为排除故障而处理这些数据。<br/>请注意，这些数据存储在我们的腾讯云数据库 Redis、腾讯云数据库 MySQL 和腾讯云数据库 MongoDB 功能中。 |
| **任务配置数据：** 任务名称、类型、受监控资产、扫描计划、扫描周期、扫描速度、上次扫描、状态 | 我们仅为了按照您的特定配置向您提供本功能而处理这些数据。 <br/>请注意，这些数据存储在我们的腾讯云数据库 Redis、腾讯云数据库 MySQL 和腾讯云数据库 MongoDB 功能中。 |

## 5\. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述：

| **个人信息**  | **保留政策**                                         |
| ------------------------- | ------------------------------------------------------------ |
| 网站基本信息 | 我们会在您使用本功能期间保留此类数据。您可以手动删除本功能上的此类数据。一旦您选择删除此类数据或终止使用本功能，我们将在 7 天后删除这些数据。 |
| 主机基本信息    |  我们会在您使用本功能期间保留此类数据。您可以手动删除本功能上的此类数据。一旦您选择删除此类数据或终止使用本功能，我们将在 7 天后删除这些数据。                                                            |
| API 基本信息     |    我们会在您使用本功能期间保留此类数据。您可以手动删除本功能上的此类数据。一旦您选择删除此类数据或终止使用本功能，我们将在 7 天后删除这些数据。                                                          |
| 任务配置数据   |   我们会在您使用本功能期间保留此类数据。您可以手动删除本功能上的此类数据。一旦您选择删除此类数据或终止使用本功能，我们将在 7 天后删除这些数据。                                                           |

您可以根据 DPSA 请求删除此类个人数据。

## 6\. 特殊条款

您必须确保使用本功能的最终用户至少达到个人可同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
