## 1\. 배경

이 모듈은 귀하가 Vulnerability Scan Service(“**기능**”)를 사용하는 경우에만 해당됩니다. 이 모듈은 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 개인 정보 보호 및 보안 계약에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 것과 같은 의미를 갖습니다. DPSA와 이 모듈 간에 상충이 있는 경우, 이 모듈은 상충되지 않는 범위 내에서 적용됩니다.

## 2\. 처리

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **이용**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **기본 웹 사이트 정보:** 자산 주소, 유형, 추가 시간(취약점 스캔 후 더 많은 리소스가 필요한 경우), 인증 상태, 모의 로그인 상태, 마지막 스캔 시간 | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다. 또한 귀하의 요청 시에 문제 해결을 위해 이 데이터를 처리합니다.<br/>이 데이터는 이러한 목적으로 TencentDB for Redis, TencentDB for MySQL, TencentDB for MongoDB에 저장됩니다. |
| **기본 호스트 정보:** 서버 IP, 추가 시간(취약점 스캔 후 더 많은 리소스가 필요한 경우), 인증 상태, 마지막 스캔 시간 | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다. 또한 귀하의 요청 시에 문제 해결을 위해 이 데이터를 처리합니다.<br/>이 데이터는 이러한 목적으로 TencentDB for Redis, TencentDB for MySQL, TencentDB for MongoDB에 저장됩니다. |
| **기본 API 정보:** API 이름, API 도메인 이름, 인증 방법, 인증 상태, 추가 시간(취약점 스캔 후 더 많은 리소스가 필요한 경우), 마지막 스캔 결과, 마지막 스캔 시간 | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다. 또한 귀하의 요청 시에 문제 해결을 위해 이 데이터를 처리합니다.<br/>이 데이터는 이러한 목적으로 TencentDB for Redis, TencentDB for MySQL, TencentDB for MongoDB에 저장됩니다. |
| **작업 구성 데이터:** 작업 이름, 유형, 모니터링하는 자산, 스캔 계획, 스캔 기간, 스캔 속도, 마지막 스캔, 상태 | 당사는 사용자의 고유한 구성에 따라 사용자에 대한 기능 제공 목적으로만 이 데이터를 처리합니다. <br/>이 데이터는 이러한 목적으로 TencentDB for Redis, TencentDB for MySQL, TencentDB for MongoDB에 저장됩니다. |
