## 1\. LATAR BELAKANG

Modul ini berlaku jika Anda menggunakan Layanan Pemindaian Kerentanan ("**Fitur**"). Modul ini dimasukkan ke dalam Perjanjian Privasi dan Keamanan Data yang ada di [DPSA](https://intl.cloud.tencent.com/document/product/301/17347). Istilah-istilah yang digunakan, tetapi tidak didefinisikan dalam Modul ini memiliki arti yang sama dengan yang ditentukan dalam DPSA. Jika terdapat perbedaan antara DPSA dan Modul ini, Modul inilah yang akan berlaku meskipun terjadi inkonsistensi.

## 2\. PEMROSESAN

Kami akan memproses data berikut sehubungan dengan Fitur:

| **Informasi Pribadi**                                     | **Tujuan Penggunaan**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Informasi Situs Web Dasar:** Alamat aset, jenis, waktu tambahan (jika lebih banyak sumber daya diperlukan setelah pemindaian kerentanan), status autentikasi, status login tiruan, waktu pemindaian terakhir | Kami hanya memproses data ini dengan tujuan menyediakan Fitur ini kepada Anda. Atas permintaan Anda, kami juga akan memproses data ini untuk pemecahan masalah.<br/>Perhatikan bahwa data ini disimpan di fitur TencentDB for Redis, TencentDB for MySQL, dan TencentDB for MongoDB untuk tujuan ini. |
| **Informasi Host Dasar:** IP Server, waktu tambahan (jika lebih banyak sumber daya diperlukan setelah pemindaian kerentanan), status autentikasi, waktu pemindaian terakhir | Kami hanya memproses data ini dengan tujuan menyediakan Fitur ini kepada Anda. Atas permintaan Anda, kami juga akan memproses data ini untuk pemecahan masalah.<br/>Perhatikan bahwa data ini disimpan di fitur TencentDB for Redis, TencentDB for MySQL, dan TencentDB for MongoDB untuk tujuan ini. |
| **Informasi API Dasar:** Nama API, nama domain API, metode autentikasi, status autentikasi, waktu tambahan (jika lebih banyak sumber daya diperlukan setelah pemindaian kerentanan), hasil pemindaian terakhir, waktu pemindaian terakhir | Kami hanya memproses data ini dengan tujuan menyediakan Fitur ini kepada Anda. Atas permintaan Anda, kami juga akan memproses data ini untuk pemecahan masalah.<br/>Perhatikan bahwa data ini disimpan di fitur TencentDB for Redis, TencentDB for MySQL, dan TencentDB for MongoDB untuk tujuan ini. |
| **Data Konfigurasi Tugas:** nama tugas, jenis, aset yang dipantau, rencana pemindaian, periode pemindaian, kecepatan pemindaian, pemindaian terakhir, status | Kami hanya memproses data ini dengan tujuan menyediakan Fitur kepada Anda sesuai konfigurasi spesifik Anda. <br/>Perhatikan bahwa data ini disimpan di fitur TencentDB for Redis, TencentDB for MySQL, dan TencentDB for MongoDB untuk tujuan ini. |
