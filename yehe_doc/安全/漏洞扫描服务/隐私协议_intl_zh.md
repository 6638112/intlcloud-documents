## 1\. 简介

本模块仅在您使用漏洞扫描服务（以下称为“**本功能**”）时适用。本模块包含在位于[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. 可用性

本功能可供全球范围内的用户使用，但为确保最佳性能，主要供与所选服务区域相同的国家/地区内的用户使用。

## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **漏洞扫描引擎运行数据：** 引擎运行时间、引擎运行次数、引擎运行状态、引擎任务日志数据 | 我们使用此类信息来确保引擎运行符合要求，从而向您提供本功能。 <br/>请注意，这些数据存储在我们的腾讯云数据库 Redis (Redis)、腾讯云数据库 MySQL (MySQL) 和腾讯云数据库 MongoDB (MongoDB) 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |
| **资产映射引擎运行数据：** 引擎运行图 | 我们使用此类信息来向您提供本功能，包括生成安全扫描、风险和相关报告，提供风险警告和漏洞检测通知，以及提供修复建议。<br/> 请注意，这些数据存储在我们的 Redis、MySQL 和 MongoDB 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |
| **用户配额数据：** 到期日期和时间、域名数量、IP 数量、API 数量。 | 我们使用此类信息来计算您的用户配额和费用，从而进行计费。<br/>请注意，这些数据存储在我们的 Redis、MySQL 和 MongoDB 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |
| **任务日志数据：** 任务名称、监控类型、扫描资产扫描计划、扫描速度、扫描时间、扫描结果 | 我们为向您提供本功能和排除故障而使用此类信息。 <br/>请注意，这些数据存储在我们的腾讯云数据库 Redis、腾讯云数据库 MySQL 和腾讯云数据库 MongoDB 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |
| **漏洞匹配数据：** 漏洞名称、威胁级别、受影响的资产、漏洞类型、上次发现时间 | 我们为向您提供本功能和排除故障而使用此类信息。 <br/>请注意，这些数据存储在我们的腾讯云数据库 Redis、腾讯云数据库 MySQL 和腾讯云数据库 MongoDB 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |
| **端口匹配数据：** 端口、漏洞响应建议、协议、端口服务 | 我们为向您提供本功能和排除故障而使用此类信息。 <br/>请注意，这些数据存储在我们的腾讯云数据库 Redis、腾讯云数据库 MySQL 和腾讯云数据库 MongoDB 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |

## 6\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息**                     | **保留政策**                                         |
| -------------------------------------------- | ------------------------------------------------------------ |
| 漏洞扫描引擎运行数据 | 存储 7 天。                                           |
| 资产映射引擎运行数据          |  存储 7 天。                                                            |
| 用户配额数据                              | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 7 天后删除此数据。 |
| 任务日志数据                                |    我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 7 天后删除此数据。                                                          |
| 漏洞匹配数据                  |  我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 7 天后删除此数据。                                                            |
| 端口匹配数据                           |   我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 7 天后删除此数据。                                                           |
