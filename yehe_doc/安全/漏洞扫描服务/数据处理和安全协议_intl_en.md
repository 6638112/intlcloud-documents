Terms used but not defined in this Module shall have the meaning given to them in the DPSA. ## 2. PROCESSING

We will process the following data in connection with the Feature:

| **Personal Information**                                     | **Use**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Basic Website Information:** Asset address, type, additional time (if more resources are required after the vulnerability scan), authentication status, mock login status, last scan time | We only process this data for the purposes of providing this Feature to you. Upon your request, we will also process this data for troubleshooting.<br/>Please note that this data is stored in our TencentDB for Redis, TencentDB for MySQL and TencentDB for MongoDB features for this purpose. |
| **Basic Host Information:** Server IP, additional time (if more resources are required after the vulnerability scan), authentication status, last scan time | We only process this data for the purposes of providing this Feature to you. Upon your request, we will also process this data for troubleshooting.<br/>Please note that this data is stored in our TencentDB for Redis, TencentDB for MySQL and TencentDB for MongoDB features for this purpose. |
| **Basic API information:** API name, API domain name, authentication method, authentication status, additional time (if more resources are required after the vulnerability scan), result of last scan, time of last scan | We only process this data for the purposes of providing this Feature to you. Upon your request, we will also process this data for troubleshooting.<br/>Please note that this data is stored in our TencentDB for Redis, TencentDB for MySQL and TencentDB for MongoDB features for this purpose. |
| **Task Configuration Data:** task name, type, monitored asset, scanning plan, scanning period, scanning speed, last scan, status | We only process this data for the purposes of providing the Feature to you in accordance to your specific configuration. <br/>Please note that this data is stored in our TencentDB for Redis, TencentDB for MySQL and TencentDB for MongoDB features for this purpose. |

| Basic Host Information    |  We retain such data for as long as you use the Feature. You may manually delete such data on the Feature. Once you choose to delete such data or terminate your use of the Feature, we will delete this data after 7 days.                                                            |
| Basic API information     |    We retain such data for as long as you use the Feature. You may manually delete such data on the Feature.
