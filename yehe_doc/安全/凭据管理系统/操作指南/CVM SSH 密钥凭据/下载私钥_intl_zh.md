本文档将指导您如何在凭据管理系统控制台中下载私钥内容。

## 前提条件
- 已创建 [CVM SSH 密钥的凭据](https://intl.cloud.tencent.com/document/product/1078/42683)。 

## 操作步骤
1. 登录 [凭据管理系统](https://console.cloud.tencent.com/ssm) 控制台，在左侧导航栏中，单击 **CVM SSH 密钥管理**，进入 CVM SSH 密钥页面。
   ![](https://qcloudimg.tencent-cloud.cn/raw/065475881ba4b315348c9264cf6ca161.png)
2. 在 CVM SSH 密钥页面中，单击左上角的“区域下拉框”，切换区域。
   ![](https://qcloudimg.tencent-cloud.cn/raw/9489836502937f07a2ff1bb3ca1bf33a.png)
3. 在 CVM SSH 密钥页面中，单击搜索框，可通过“标签和凭据名称”等关键字对凭据进行查找。
![](https://qcloudimg.tencent-cloud.cn/raw/32072ac7a79d64de36fc31aa014a4271.png)
4. 在筛选结果列表中，找到需要查看的凭据，单击**凭据名称**，进入凭据详情页面。
![](https://qcloudimg.tencent-cloud.cn/raw/674cd635eb0b3b2d16f74a4dfb6af5d4.png)
5. 在凭据详情页面中，SSH 的密钥信息展示了密钥的名称、ID 及公钥内容，其中单击私钥内容右侧的**下载**，即可完成下载。
![](https://qcloudimg.tencent-cloud.cn/raw/363d33d882f6718b197115289869cb9f.png)
