This document describes how to use tags in the SSM console.

### Usage limits
There are limits on using a tag (tag key and tag value). For more information, see [Usage Limits](https://intl.cloud.tencent.com/document/product/651/13354).


### Directions
1. Log in to the [SSM Console](https://console.cloud.tencent.com/ssm) and click **Database Credential** on the left sidebar.
   ![](https://main.qcloudimg.com/raw/14b18d660859b5ffa1c2f0536e8c3dfa.png)
2. Click the drop-down button in the top left corner of the credential list to modify the region.
   ![](https://main.qcloudimg.com/raw/752da25ebcf954e244cbb055c69004d9.png)
  - **Editing a tag**
    1. Select a credential to edit its tag and click **Edit Tag** on the right.
    ![](https://main.qcloudimg.com/raw/c9494d613a5054430cec8cb69fa1825b.png)
    2. In the pop-up window, add or delete tags as needed.
>?For information about how to use tags, see [Examples of Management via Tags](https://intl.cloud.tencent.com/document/product/1078/41807).
 - **Editing tags in batch**
    1. Select a credential to edit its tag and click **Edit Tag** at the top of the credential list.
    ![](https://main.qcloudimg.com/raw/e8dcffeb3a687e70cacae787d9998ff5.png)
    2. In the pop-up window, add or delete tags as needed.
    >?For information about how to use tags, see [Examples of Management via Tags](https://intl.cloud.tencent.com/document/product/1078/41807).
