### Overview

This document describes how to edit tags of resources.

### Use Limits

There are some limits on the usage of tags (tag key and tag value). For more information, please see [Use Limits](https://intl.cloud.tencent.com/document/product/651/13354).

### Prerequisites

1. You have logged in to the [SSM Console](https://console.cloud.tencent.com/ssm).
2. You have selected the region where the secret to be edited is located.

### Directions

#### Editing tags of a secret

1. Find the secret whose tags need to be edited and click **Edit Tag** in the **Operation** column on the right.![](https://main.qcloudimg.com/raw/0620fe335e899503548b6de832ec0f02.png)

2. In the "1 resource selected" area in the pop-up window, **Add** or **Delete** tags as needed.

#### Editing tags of multiple secrets

1. Select the secrets whose tags need to be edited and click **Edit Tag** at the top of the page. ![](https://main.qcloudimg.com/raw/8360c93a46b2b0087be839c2958e743a.png)
2. In the "n resources selected" area in the pop-up window, **Add** or **Delete** tags as needed.

For information about how to use tags, please see [Example of Managing Secrets with Tags](https://intl.cloud.tencent.com/document/product/1078/38607).
