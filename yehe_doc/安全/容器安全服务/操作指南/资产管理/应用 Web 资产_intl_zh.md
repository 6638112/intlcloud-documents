本文档为您介绍应用 Web 资产功能，以及如何查看Web 服务、运行应用和数据库应用数量。

## 查看Web 服务
###  筛选 Web 服务
1. 登录 [容器安全服务控制台](https://console.cloud.tencent.com/tcss)，在左侧导航中，单击**资产管理**，进入资产管理页面。
2. 在资产管理页面，单击“Web 服务总数”进入Web 服务列表页面，可查看全部进程资产列表。
<img src="https://qcloudimg.tencent-cloud.cn/raw/d3e94e1469dd4f2f61422e40f4858cdb.png" style="zoom:67%;" />
3. 在 Web 服务列表页面，可按服务类型对 Web 服务资产进行筛选，或搜索框通过“容器名称、主机名、启动用户”等关键字对 Web 服务进行查找。
 - 单击左上角的服务类型下拉框，按服务类型对 Web 服务资产进行筛选。 
<img src="https://qcloudimg.tencent-cloud.cn/raw/4922c3eea0c3247bc9ae597d30ebee3a.png" style="zoom:67%;" />
 - 单击搜索框，可通过“容器名称、主机名、启动用户”等关键字对 Web 服务进行查找。
<img src="https://qcloudimg.tencent-cloud.cn/raw/c25721604f45542be9e3a1892814c6cb.png" style="zoom:67%;" />

### 查看 Web 服务列表
1. 登录 [容器安全服务控制台](https://console.cloud.tencent.com/tcss)，在左侧导航中，单击**资产管理**，进入资产管理页面。
2. 在资产管理页面，单击“Web 服务总数”进入Web 服务列表页面，可查看全部进程资产列表。
<img src="https://qcloudimg.tencent-cloud.cn/raw/2203b3a0d11fda7f09ed659c3bf6202c.png" style="zoom:67%;" />
3. 在 Web 服务列表页面，主机 IP：单击“主机 IP”，右侧弹出抽屉展示主机详情，包括主机基本信息、Docker 信息、相关镜像数和相关容器数。
>? 在抽屉中，可单击“数字”查看主机相关镜像数和相关容器数详情。
><img src="https://qcloudimg.tencent-cloud.cn/raw/c0e4c46ec1905394cf39668f1bc329d3.png" style="zoom: 67%;" />

![](https://qcloudimg.tencent-cloud.cn/raw/6f3475393c90ef80b99e10717881939a.png)
4. 在 Web 服务列表页面，单击**查看详情**，对话框展示 Web 应用服务详情，包括基本信息和关联进程列表。
![](https://qcloudimg.tencent-cloud.cn/raw/fb87b39e3cef372bedd123631ef523e0.png)

### 自定义列表管理
1. 登录 [容器安全服务控制台](https://console.cloud.tencent.com/tcss)，在左侧导航中，单击**资产管理**，进入资产管理页面。
2. 在资产管理页面，单击“Web 服务总数”进入Web 服务列表页面，可查看全部进程资产列表。
<img src="https://qcloudimg.tencent-cloud.cn/raw/7ed390f6a38c259d91c0c192bf932aca.png" style="zoom:67%;" />
3. 在 Web 服务列表页面，单击![](https://main.qcloudimg.com/raw/d42b27540eef9bf90a9e30f96b500bf3.png)图标，弹出自定义列表管理弹窗，在弹窗中可以自定义设定列表管理。
4. 在自定义列表管理弹窗，选择所需的类型后，单击**确定**，即可完成设置自定义列表管理。
<img src="https://qcloudimg.tencent-cloud.cn/raw/cdc0f385fb27e7d4d662b6d8ab74becc.png" style="zoom:67%;" />
