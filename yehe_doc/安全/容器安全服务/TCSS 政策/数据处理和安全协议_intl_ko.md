## 1\. 배경

이 모듈은 Tencent Container Security Service("**기능**")를 사용하는 경우 적용됩니다. 이 모듈은 ([“DPSA”](https://intl.cloud.tencent.com/document/product/301/17347))에 있는 데이터 처리 및 보안 계약에 포함되어 있습니다. 본 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에서 정의된 것과 같은 의미를 갖습니다. DPSA와 본 모듈 간에 상충이 있는 경우, 해당 상충 범위까지 본 모듈이 적용됩니다.

## 2\. 처리

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **사용**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **자산 기본 데이터:** 서버, 컨테이너, 이미지, 클러스터의 기본 정보(자산 핑거프린트 정보(리소스 모니터링, 계정, 포트, 소프트웨어 애플리케이션, 프로세스, 데이터베이스, 웹 애플리케이션, 웹 서비스, 웹 프레임워크, 웹 사이트, Jar 패키지 시작 서비스, 예정된 태스크, 환경 변수, 커널 모듈), 이미지 리포지토리 기본 정보(이미지 리포지토리 주소, 리포지토리 유형, 이미지 이름, 이미지 ID, 이미지 버전, 이미지 크기, 보안 위험(보안 취약점, 트로이 목마 바이러스, 민감한 정보 포함), 구성 이력) | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 TencentDB for MongoDB(MongoDB) 및 TencentDB for MySQL(MySQL) 기능에 저장 및 백업되고 Cloud Object Storage(COS) 기능에 임시로 저장 및 백업되며,  귀하가 구독한 관련 호스팅 서비스(이 기능에 대한 개인정보 보호정책 모듈에 정의됨)와 통합됩니다. Tencent Kubernetes Engine(TKE) 클러스터를 구매한 경우, 귀하의 권한 부여에 따라 해당 기능을 귀하에게 제공하기 위한 목적(보안 검사 수행 포함)으로 이미지 리포지토리의 기본 정보도 공유됩니다. |
| **콘솔 구성 데이터:**<br/><li>이미지, 클러스터, 취약점, 기준선, 파일,  로그 탐지 구성: 일반 탐지, 취약성/기준선 무시, 파일 신뢰 확인 또는 파일 격리, 프로세스 가로채기, 이미지 가로채기, 컨테이너 네트워크 액세스 가로채기)<li>컨테이너 이스케이프, 리버스 쉘, 파일 검사, 비정상 프로세스, 파일 변조, 고위험 시스템 드롭 콜 및 기타 침입 방지 기능에 대한 화이트리스트 구성: 화이트리스트 조건, 유효 미러 범위 기타 구성 정보: 자동 서버 업그레이드 보호 설정, 자동 갱신 설정, 알람 설정 | 당사는 사용자의 환경 설정에 따라 사용자에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 TencentDB for MongoDB(MongoDB) 및 TencentDB for MySQL(MySQL) 기능에 저장 및 백업되고 귀하가 구독한 관련 호스팅 서비스(이 기능에 대한 개인정보 보호정책 모듈에 정의됨)와 통합됩니다. |

## 5\. 데이터 보존

당사는 기능과 관련하여 처리된 개인 데이터를 다음과 같이 저장합니다.

| **개인 정보**   | **보존 방침**                                         |
| -------------------------- | ------------------------------------------------------------ |
| 자산 기본 데이터           | 이러한 데이터는 최소 180일간 보관됩니다. 당사는 귀하가 관련 호스팅 서비스 이용을 종료할 때까지 해당 데이터를 보유하며, 이 경우 당사는 해당 데이터를 COS 기능에 임시로 보관하고 (i) 이후 30일 또는 (ii) 최소 데이터 보존 기준(해당하는 경우)을 충족하는 데 필요한 최소 일수인 180일 이후에 동일한 데이터를 삭제합니다. |
| 콘솔 구성 데이터 | 귀하가 삭제를 요청할 때까지 저장되며, 요청 시 이러한 데이터는 영업일 기준 30일 이내에 삭제됩니다. 귀하가 삭제를 요청하지 않는 경우, 해당 데이터 보호법에서 달리 요구하지 않는 한 해당 데이터는 이 기능 사용이 종료되고 1개월 후에 삭제됩니다. |

귀하는 DPSA에 따라 해당 개인 데이터의 삭제를 요청할 수 있습니다.

## 6\. 특수 조건

본 기능을 사용하려는 경우, 개인 데이터 처리에 동의할 수 있는 최소 연령 이상인 최종 사용자여야 합니다. 이는 최종 사용자가 거주하는 관할권에 따라 다를 수 있습니다.

본 기능은 민감한 정보를 처리하기 위한 것이 아닙니다. 귀하는 귀하 또는 귀하의 최종 사용자가 본 기능을 통해 민감한 정보를 전송하거나 처리하지 않도록 해야 합니다.
