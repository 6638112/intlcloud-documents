## 1\. 背景

本模块仅在您使用 Captcha 功能（下称“**功能**”）时适用。本模块已纳入位于 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据处理和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，本模块适用。

## 2\. **处理**

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Captcha 配置数据：** Captcha APPID、Captcha 名称、UIN、 APPID、域限制、加密密钥、Captcha 类型、场景类型、智能验证、智能引擎、配色方案颜色、语言、恶意拦截等级、流量阈值、顶部全屏、创建时间、更新时间 | 我们仅为了按照您的特定配置向您提供本功能而处理这些数据。<br/>请注意，此类数据通过我们的云数据库 Redis（下称“Redis”）功能实现共享，并且存储和备份在我们的云数据库 MySQL（下称“MySQL”）功能中。 |
| **Captcha 统计数据（最终用户）：** Captcha APPID、时间、请求信息（票证操作、验证、吞吐量和拦截）、票证信息（票证计数、吞吐量和拦截） | 我们仅因向您提供本功能的使用统计情况之目的处理这些数据。<br/>请注意，这些数据将存储和备份在我们的 MySQL 功能中。 |
| **人工验证数据（最终用户）：** 浏览器语言、页面 URL、Web Worker 状态（如有）、验证尝试、浏览器功能、用户代理、屏幕分辨率、页面和屏幕尺寸、页面推荐、操作系统、CPU、画布值、字体、属性哈希、浏览器调整大小、WebGL 信息、GPU、时区、插件、颜色深度、Web 驱动程序、字符集、设备和光标位置、加载时间、时间戳、登录 Cookie、JavaScript 设置、电池状态、IP 地址、存储和数据库状态、DoNotTrack 状态、AdBlock 状态、域 | 我们为了向您提供本功能而处理这些数据。<br/>请注意，这些数据将存储和备份在我们的 MySQL 功能中。 |
| **Cookie 数据（最终用户）：** TDC_itoken                    | 我们向您提供本功能而使用 Cookie（确定最终用户的请求是否来自同一设备）。 |

## 5\. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述（除非适用的数据保护法另有规定）：

| **个人信息**   | **保留政策**                                         |
| -------------------------- | ------------------------------------------------------------ |
| Captcha 配置数据 | 我们会一直保留此类数据，直到您将它们手动删除。此外，若您删除帐户，我们将在 7 天后删除此类数据。 |
| Captcha 统计数据   | 我们会将此类数据保留 1 年，除非您手动删除（这种情况下，此类数据将在 7 天内删除）。 |
| 人工验证数据    | 此类数据将存储 30 天，除非请求删除（这种情况下，此类数据将在 7 天内删除）。 |
| Cookie 数据                | 该 Cookie 在最终用户清除其浏览历史记录时过期。 |

您可以根据 DPSA 请求删除此类个人数据。
