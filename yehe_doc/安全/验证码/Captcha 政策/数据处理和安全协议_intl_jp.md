## 1\. 背景

本モジュールは、Captcha の機能（以下「**本機能**」）を使用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347) に掲載されている Data Processing and Security Agreement （データ処理およびセキュリティに関する契約）（以下「DPSA」）に組み込まれます。本モジュールにおける用語のうち本書内に定義のないものは、DPSA 内で定義された意味を有するものとします。DPSA と本モジュールとの間に矛盾がある場合には、その不一致の範囲で本モジュールを優先するものとします。

## 2\. **処理**

当社は、本機能に関連して以下のデータを処理します。

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Captcha 構成データ：** Captcha APPID、Captcha 名、UIN、APPID、ドメイン制限、暗号化キー、Captcha タイプ、シーンタイプ、スマートベリファイ、スマートエンジン、配色、言語、悪のインターセプトグレード、トラフィックしきい値、トップフルスクリーン、作成時間、更新時間 | 当社は、このデータを、お客様の特定の設定に従い、本機能をお客様に提供する目的においてのみ処理します。  <br/>このデータは、この目的において、当社の TencentDB for Redis（Redis）機能と共有され、当社の TencentDB for MySQL（MySQL）機能で保存およびバックアップされます。 |
| **（エンドユーザーの）Captcha 統計データ：** Captcha APPID、時刻、リクエスト情報（チケットアクション、検証、スループット、インターセプト）、チケット情報（チケットカウント、スループット、インターセプト） | このデータは、お客様による本機能の使用に関する統計情報を提供するためだけに処理されます。<br/>このデータは、当社の MySQL 機能で保存およびバックアップされます。 |
| **（エンドユーザーの）人間検証データ：** ブラウザの言語、ページの URL、Web ワーカーのステータス（存在する場合）、検証の試行、ブラウザ機能、ユーザーエージェント、画面解像度、ページと画面のサイズ、ページリファラー、OS、CPU、キャンバス値、フォント、プロパティハッシュ、ブラウザのサイズ変更、WebGL 情報、GPU、タイムゾーン、プラグイン、色深度、Web ドライバ、文字セット、デバイスとカーソルの位置、ロード時間、タイムスタンプ、ログインクッキー、javascript 設定、バッテリーの状態、IP アドレス、ストレージと DB のステータス、DoNotTrack ステータス、AdBlock ステータス、ドメイン | 弊社は、お客様に本機能を提供する目的で左記データを処理します。<br/>このデータは、当社の MySQL 機能で保存およびバックアップされます。 |
| **（エンドユーザーの）クッキーデータ：** TDC_itoken                     | 当社は、本機能をお客様に提供する目的（エンドユーザーのリクエストが同一のデバイスから発信されたかどうかを識別するため）において、このクッキーを使用します。 |

## 5\. データ保持

当社は、本機能に関連して処理された個人データを、以下のとおり保管します（適用されるデータ保護法によって別途要求される場合を除きます）。

| **個人情報**   | **保持ポリシー**                                         |
| -------------------------- | ------------------------------------------------------------ |
| Captcha 構成データ | 左記データは、手動で削除するまで保持されます。また、お客様がアカウントを削除されると、その 7 日後に当社は当該データを削除します。 |
| Captcha 統計データ   | 当社は当該データを 1 年間保持します。ただし、お客様が当該データを手動で削除された場合を除きます（この場合、当該データは 7 日以内に削除されます)。 |
| 人間検証データ    | データは 30 日間保持されます。ただし、お客様が当該データを手動で削除された場合を除きます（この場合、当該データは 7 日以内に削除されます)。 |
| クッキーデータ                | このクッキーは、エンドユーザーが閲覧履歴を消去すると期限切れになります。 |

お客様は、DPSA に従って、当該個人データの削除を要請することができます。

## 6\. 特記条件

お客様は、本機能を、個人が自己の個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが使用するようにするか、または最低年齢に達していない個人については保護者の同意を得るようにする必要があります。これは、エンドユーザーが所在する司法管轄区によって異なる場合があります。
