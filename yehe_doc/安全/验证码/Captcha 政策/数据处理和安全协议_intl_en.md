## 2. **PROCESSING**

We will process the following data in connection with the Feature:

| **Personal Information**                                     | **Use**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Captcha Configuration Data:** Captcha APPID, Captcha name, UIN, APPID, domain limit, encrypt key, Captcha type, scene type, smart verify, smart engine, scheme color, language, evil intercept grade, traffic threshold, top full screen, create time, update time | We only process this data for the purposes of providing the Feature to you, in accordance to your specific configuration.<br/>Please note that this data is shared with our TencentDB for Redis (Redis) feature for this purpose, and stored and backed up in our TencentDB for MySQL (MySQL) feature. |
| **Captcha Statistical** **Data (of end user)**: Captcha APPID, time, request information (ticket action, verify, throughput, and intercept), ticket information (ticket count, throughput, and intercept) | We only process this data to provide you with statistics of your use of the Feature.<br/>Please note that this data is stored and backed up in our MySQL feature. |
| **Human Verification Data (of end user):** browser language, page URL, status of web worker (if any), verification attempts, browser features, user-agent, screen resolution, page and screen dimensions, page referrer, OS, CPU, canvas value, font, property hash, browser resize, WebGL info, GPU, time zone, plugins, color depth, web driver, character set, device and cursor position, load time, time stamp, login cookie, javascript settings, battery status, IP address, storage and DB status, DoNotTrack status, AdBlock status, domain | We process this data for the purposes of providing the Feature to you.<br/>Please note that this data is stored and backed up in our MySQL feature. |
| **Cookie Data (of end user):** TDC_itoken                    | We use this cookie for the purposes of providing the Feature to you (to identify whether the end user's request originates from the same device). |

## 5. DATA RETENTION

We will store personal data processed in connection with the Feature as follows (unless otherwise required by applicable Data Protection Laws):

| **Personal Information**   | **Retention Policy**                                         |
| -------------------------- | ------------------------------------------------------------ |
| Captcha Configuration Data | We retain such data until you manually delete such data. Otherwise, when you delete your account, we will delete such data after 7 days. |
| Captcha Statistical Data   | We retain such data for 1 year, unless you manually delete such data (in which case such data will be deleted within 7 days).] |
| Human Verification Data    | Stored for 30 days, unless you request for deletion of such data (in which case such data will be deleted within 7 days). |
| Cookie Data                | This cookie expires when the end user clears their browsing history. |

You can request deletion of such personal data in accordance with the DPSA.
