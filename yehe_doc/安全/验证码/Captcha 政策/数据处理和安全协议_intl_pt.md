## 1\. CONSIDERAÇÕES INICIAIS

Esse Módulo se aplica se você usar o recurso Captcha (“**Recurso**”). Esse Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em:  (“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)”). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.

## 2\. **PROCESSAMENTO**

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de configuração do Captcha:** ID de aplicação do Captcha, nome do Captcha, número de identificação único, ID da aplicação, limite do domínio, chave criptografada, tipo de Captcha, tipo de cena, verificação inteligente, mecanismo inteligente, esquema de cores, idioma, prevenção de evilgrade, limite de tráfego, tela cheia, data de criação e horário de atualização | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você conforme sua configuração específica.<br/>Observe que esses dados são compartilhados com nosso recurso TencentDB for Redis (Redis) para esse fim, bem como são armazenados com backup em nosso recurso TencentDB for MySQL (MySQL). |
| **Dados estatísticos** **do Captcha (usuário final):** ID de aplicação do Captcha, horário, informações da requisição (ação de entrada, verificação, acesso e interceptação), informações sobre entradas (contagem de entradas, acesso e interceptação) | Apenas processamos esses dados para fornecer a você estatísticas de seu uso do Recurso.<br/>Observe que esses dados são armazenados com backup em nosso recurso MySQL. |
| **Dados de verificação humana (usuário final):** linguagem do navegador, URL da página, status do funcionário da Web (se houver), tentativas de verificação, recursos do navegador, representante do usuário, resolução de tela, dimensões da página e da tela, referência de página, sistema operacional, CPU, valores de proporção de tela, fonte, hash de propriedade, redimensionamento do navegador, informações do WebGL, GPU, fuso horário, plugins, profundidade de cor, driver de rede, conjunto de caracteres, posição do dispositivo e cursor, tempo de carregamento, registro de data e hora, cookie de login, configurações JavaScript, status da bateria, endereço de IP, armazenamento e status do DB, status do DoNotTrack, status do AdBlock e domínio | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você.<br/>Observe que esses dados são armazenados com backup em nosso recurso MySQL. |
| **Dados de Cookie (usuário final):** TDC_itoken                    | Usamos este cookie com o propósito de fornecer o Recurso para você (para identificar se a requisição do usuário final se origina do mesmo dispositivo). |

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma (salvo se de outra forma exigido pelas Leis de Proteção de Dados aplicáveis):

| **Informações Pessoais**   | **Política de Retenção**                                         |
| -------------------------- | ------------------------------------------------------------ |
| Dados de configuração do Captcha | Retemos esses dados até que você os exclua manualmente. Caso contrário, ao excluir a conta ou encerrar o uso do Recurso, excluiremos esses dados após 7 dias. |
| Dados estatísticos do Captcha   | Retemos esses dados por 1 ano, a menos que você os exclua manualmente (nesse caso, os dados serão excluídos dentro de 7 dias). |
| Dados de verificação humana    | Armazenados por 30 dias, a menos que você solicite a exclusão dos dados (nesse caso, os dados serão excluídos em 7 dias). |
| Dados de cookies                | Este cookie expira quando o usuário final limpa seu histórico de navegação. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.
