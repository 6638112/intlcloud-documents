


## Prerequisites
You need to [purchase an Anti-DDoS Pro instance](https://intl.cloud.tencent.com/document/product/1029/36115) and set the protected object first.

## Directions
1. Log in to the [new Anti-DDoS Pro Console](https://console.cloud.tencent.com/ddos/antiddos-native/package) and select **Protection Configuration** on the left sidebar.
2. Select an Anti-DDoS Pro instance ID in the list on the left, such as "bgp-000000co", and select **Domain Name Protection** at the top on the right.
![](https://main.qcloudimg.com/raw/9eae775ebe603f38d725ab5a37c6299b.png)
3. Click **Set** in the "Frequency Control" block on the right to enter the frequency control rule list.
![](https://main.qcloudimg.com/raw/1432f0f510e3d495c606ce30855315ad.png)
4. Click **Create** to create a frequency control rule and fill in relevant fields.
![](https://main.qcloudimg.com/raw/f1128c978153cfdf59863a5922931b1b.png)
5. After the creation is completed, a frequency control rule will be added to the frequency control list. You can modify the rule by clicking **Configure** in the "Operation" column on the right.
