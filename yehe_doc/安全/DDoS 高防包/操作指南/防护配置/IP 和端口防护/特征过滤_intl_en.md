
## Prerequisites
You need to [purchase an Anti-DDoS Pro instance](https://intl.cloud.tencent.com/document/product/1029/36115) and set the protected object first.

## Directions
1. Log in to the [new Anti-DDoS Pro Console](https://console.cloud.tencent.com/ddos/antiddos-native/package) and select **Protection Configuration** on the left sidebar.
2. Select an Anti-DDoS Pro instance ID in the list on the left, such as "bgp-0000008o", and select **IP Port Protection** at the top on the right.
![](https://main.qcloudimg.com/raw/1905685ff9969234a19c876911f3fdc4.png)
3. Click **Set** in the "Attribute Filtering" block on the right to enter the attribute filtering list.
![](https://main.qcloudimg.com/raw/ff909b4e2d970a91d3d38dfe4665c552.png)
4. Click **Create** to create an attribute filtering rule.
![](https://main.qcloudimg.com/raw/fdca75e1e5ae72f857ec8038f6324be3.png)
5. After the creation is completed, an attribute filtering rule will be added to the attribute filtering list. You can modify the rule by clicking **Configure** in the "Operation" column on the right.
![](https://main.qcloudimg.com/raw/4ee84e26721d2a96217377a73856e6eb.png)
