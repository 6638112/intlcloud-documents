

## Prerequisites
You need to [purchase an Anti-DDoS Pro instance](https://intl.cloud.tencent.com/document/product/1029/36115) and set the protected object first.

## Directions
1. Log in to the [Anti-DDoS Pro Console](https://console.cloud.tencent.com/ddos/antiddos-native/package) and select **Protection Configuration** on the left sidebar.
2. Select an Anti-DDoS Pro instance ID in the list on the left, such as "bgp-0000008o", and click **IP Port Protection** at the top on the right.
![](https://main.qcloudimg.com/raw/07b8daad2b3a04be6bd6d6dc68ea9d7e.png)
3. Click **Set** in the "IP Blacklist/Whitelist" block on the right to enter the IP blacklist/whitelist.
![](https://main.qcloudimg.com/raw/980fe2e07d4404255a30101a206de02c.png)
4. Click **Create** to create an IP blacklist/whitelist rule and select the list type.
![](https://main.qcloudimg.com/raw/e16a265499ee7762b497d1aac0370ab4.png)
5. After the creation is completed, an IP blacklist/whitelist rule will be added to the IP blacklist/whitelist. You can click **Delete** in the "Operation" column on the right to delete the rule.
![](https://main.qcloudimg.com/raw/a7f5b24ff9d6ebcdac1282297e85c432.png)
