


## Prerequisites
You need to [purchase an Anti-DDoS Pro instance](https://intl.cloud.tencent.com/document/product/1029/36115) and set the protected object first.

## Directions
1. Log in to the [new Anti-DDoS Pro Console](https://console.cloud.tencent.com/ddos/antiddos-native/package) and click **Protection Configuration** on the left sidebar.
2. Select an Anti-DDoS Pro instance ID in the list on the left, such as "bgp-0000008o", and click **IP Port Protection** at the top on the right.
![](https://main.qcloudimg.com/raw/6f220eac7761d162804a1d1a2fb20967.png)
3. Click **Set** in the "Watermark Protection" block on the right to enter the watermark protection list.
4. Click **Create** to create a watermark protection rule.
![](https://main.qcloudimg.com/raw/8dc9704b4d48778c13dd746a48102010.png)
5. After the creation is completed, a watermark protection rule will be added to the watermark protection list. You can view and configure keys by clicking **Configure Key** in the "Operation" column on the right.
![](https://main.qcloudimg.com/raw/ff45a681317a11dde09ed6f7c5ec6be9.png)
6. On the key configuration page, you can view or copy keys.
![](https://main.qcloudimg.com/raw/03ef828f435bd639211bbf46224a8453.png)
7. On the key configuration page, you can add or delete keys. You can delete a key only if there are two keys. There can be up to two watermark keys.
![](https://main.qcloudimg.com/raw/b6b1d64d9812ac7b945f1e8a3bb2af76.png)
