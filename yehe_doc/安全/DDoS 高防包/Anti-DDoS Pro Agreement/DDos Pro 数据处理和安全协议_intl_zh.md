## **1. 背景**

本模块仅在您使用 DDoS 高防包服务（简称“功能）时适用。本模块包含在位于[（简称“**DPSA**”](https://intl.cloud.tencent.com/document/product/301/17347)）的数据处理和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。

## **2. 处理**

我们将因本功能处理以下数据：


| 个人信息     | 用途                                                     |
| ------------ | -------------------------------------------------------- |
| 客户配置设置 | 客户能够配置本功能，并且这些设置可按照客户指示进行调整。 |
| 客户访问流量 | 我们处理此信息是为了识别恶意流量并进行相应过滤。         |
