## **1. 简介**

本模块仅在您使用 DDoS 高防包服务（简称**“功能”）**时适用。本模块包含在位于（“[**隐私政策**](https://intl.cloud.tencent.com/document/product/301/17345)”）的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。

## **2. 控制方**

本模块中所述个人信息的控制方见隐私政策中的规定。

## **3. 可用性**

本功能可供日本、韩国、新加坡、泰国、印度、美国、俄罗斯、德国和加拿大的用户使用。

## **4. 我们如何使用个人信息**

我们将按照以下法律依据和方式使用信息：

| 个人信息 | 用途                                                         | 法律依据                                                     |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 操作日志 | 我们保存与本功能相关的所采取行动的日志，以确保本功能按照您的指示运行。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供服务而言是必要的。 |
| 流量统计 | 我们保存流量信息日志，以便确定攻击形态并提供本功能的用户报告。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供服务而言是必要的。 |

## **5. 我们如何存储和分享个人信息**

见隐私政策中的规定。

## **6. 我们如何分享个人信息**

见隐私政策中的规定，包括 Tencent Cloud Computing (Beijing) Limited 和 Radware Limited。

## **7. 数据保留**

我们将根据下列规定保留个人信息：

| 个人信息           | 保留政策               |
| ------------------ | ---------------------- |
| 操作日志           | 自收集之日起 3 个月    |
| 流量统计           | 自收集之日起 9 个月    |
| APPID、UIN、用户名 | 在持有帐户的期间保存。 |
