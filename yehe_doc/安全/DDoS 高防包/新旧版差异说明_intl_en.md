To provide better service, Anti-DDoS plans to simplify its product configuration and launch a whole-new Anti-DDoS Pro product. Differences between the new and old versions are compared as follows:

| Configuration Item | New Anti-DDoS Pro | Old Anti-DDoS Pro | Update |
| ------------ | ------------ | -------- | ------------------------------------------------------------ |
| Instance type | Anti-DDoS Pro instances | Anti-DDoS Pro available in Single-IP and multi-IP instances | Only Anti-DDoS Pro instances are provided. |
| Elastic protection | Unavailable | Available | The protection mode "Base protection + Elastic protection" is adjusted to an all-out protection mode. |
| Billing method | Number of protected IPs + Bandwidth | Base bandwidth + Elastic bandwidth | The billing method is now based on the number of protected IPs and protection bandwidth.|
| Protection policy | Independent configuration | Unified configuration | Protection policies can be configured on its own configuration page. |
