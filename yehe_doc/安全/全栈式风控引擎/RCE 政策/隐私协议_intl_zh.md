## 1\. **简介**

本模块仅在您使用风控引擎（下称“**功能**”）时适用。本模块已纳入  的隐私政策（“[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345 )”）中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. **控制方**

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. **可用性**

本功能可供全球范围内的用户使用，但为确保最佳性能，主要供与所选服务区域相同的国家/地区内的用户使用。

## 4\. **我们如何使用个人信息**

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 联系方式：姓名、电话号码、电子邮件、邮寄地址。<br/>交易数据：电子邮件、购买的项目、支付价格、订单状态、退款信息、部分信用卡信息（即信用卡号前 6 位数字、有效期、持卡人姓名、卡号 MD5）。<br/>帐户信息：网站或 App 上的帐户和偏好信息。 | 我们为向您提供本功能而使用这些信息。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |
| 日志和 API 调用信息：用户 IP、通话音量、APPID、UIN、持续时间、时间戳。 | 我们为向您提供本功能而使用这些信息。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |
| SDK 数据：<br/>1\. 浏览器、设备和连接数据：<br/> 访问客户网站的个人计算机或移动设备的信息，包括浏览器、设备型号和操作系统，以及唯一设备标识符。<br/>2\. 行为数据：<br/> 商业网站或 APP 中的活动信息，例如访问时间和频率、浏览的页面、屏幕滑动操作的分布位置和停留时间。 | 我们为向您提供本功能而使用这些信息。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |



## 5\. **我们如何存储和分享个人信息**

见隐私政策中的规定。

## 6\. **数据保留**

我们将根据下列规定保留个人信息：

| **个人信息**                                     | **保留政策**                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 联系方式：姓名、电话号码、电子邮件、邮寄地址。<br/>交易数据：电子邮件、购买的项目、支付价格、订单状态、退款信息。<br/>帐户信息：网站或 App 上的帐户和偏好信息。 | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 5 天内删除此数据。 |
| 日志和 API 调用信息：用户 IP、通话音量、APPID、UIN、持续时间、时间戳。 | 这些数据在首次记录 180 天后会自动删除。 |
| SDK 数据：<br/>1\. 浏览器、设备和连接数据： <br/>访问客户网站的个人计算机或移动设备的信息，包括浏览器、设备型号和操作系统，以及唯一设备标识符。<br/>2\. 行为数据： <br/>商业网站或 APP 中的活动信息，例如访问时间和频率、浏览的页面和停留时间。 | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 5 天内删除此数据。 |
