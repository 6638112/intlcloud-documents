## 1\. **背景**

リスクコントロールエンジン（「**本機能**」）を使用する場合は、このモジュールが適用されます。本モジュールは、「[DPSA](https://intl.cloud.tencent.com/document/product/301/17347 )」のデータプライバシーとセキュリティ契約に組み込まれています。本モジュールで使用されていても定義されていない用語は、DPSAで定義された意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## 2\. **処理**

当社は本機能に関連して、次のデータを処理します:

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **ログデータ：**<br/>API 呼び出し情報：<br/>ユーザー IP、通話量、APPID、UIN、持続時間、タイムスタンプ | 当社は、このデータをお客様に機能を提供し、トラブルシューティングのための統計分析を行う目的で使用します。 |
| **SDK データ：**<br/>1\. ブラウザ、デバイス、接続データ：<br/> お客様のウェブサイトにアクセスするパーソナルコンピュータまたはモバイル機器の情報（ブラウザ、デバイスモデル、オペレーティングシステム、一意のデバイス識別子など）。<br/>2\. 行動データ：<br/> アクセス時間と頻度、閲覧したページ、画面上のスワイプの分布、滞在時間など、ビジネスウェブサイトまたはAPP上のアクティビティに関する情報。 | 当社は、リスク結果の包括的な評価を可能にし、機能の機能をお客様に提供するために、このデータを処理します。 |

## 5\. **データ保持**

当社は、本機能に関連して処理された個人データを以下のように保管します。

| **個人情報** | **保持ポリシー**                                         |
| ------------------------ | ------------------------------------------------------------ |
| ログデータ                 | ログデータは、最初に記録されてから180日後に自動的に削除されます。 |
| SDK データ                 | お客様が本機能を使用する限り、当社はこのデータを保持します。お客様が本機能の使用を終了すると、当社はこのデータを5日以内に削除します。 |

お客様は DPSA に従って、この個人データの削除を要請できます。

## 6\. **特別な条件**

お客様は、本機能を、個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが使用することを保証する必要があります。これは、エンドユーザーが所在する法域によって異なる場合があります。
