## 1\. **LATAR BELAKANG**

Modul ini berlaku jika Anda menggunakan Mesin Kontrol Risiko (“**Fitur**”). Modul ini dimasukkan ke dalam Perjanjian Privasi dan Keamanan Data yang berlokasi di (“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347 )”). Istilah-istilah yang digunakan, tetapi tidak didefinisikan dalam Modul ini memiliki arti yang sama dengan yang ditentukan dalam DPSA. Jika terdapat perbedaan antara DPSA dan Modul ini, Modul inilah yang akan berlaku apabila terjadi inkonsistensi.

## 2\. **PEMROSESAN**

Kami akan memproses data berikut sehubungan dengan Fitur:

| **Informasi Pribadi**                                     | **Penggunaan**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Data log:**<br/>Informasi panggilan API:<br/>IP pengguna, volume panggilan, APPID, UIN, durasi waktu, dan stempel waktu | Kami menggunakan data ini untuk tujuan menyediakan Fitur kepada Anda dan memungkinkan kami untuk melakukan analisis statistik untuk pemecahan masalah. |
| **Data SDK:**<br/>1\. Data browser, perangkat, dan koneksi:<br/> Informasi komputer pribadi atau perangkat seluler yang mengakses situs web pelanggan, termasuk browser, model perangkat dan sistem operasi, serta pengenal perangkat unik.<br/>2\. Data perilaku:<br/> Informasi tentang aktivitas di situs web bisnis atau APP, seperti waktu dan frekuensi akses, halaman yang dilihat, distribusi gesekan di layar, serta waktu kunjungan. | Kami memproses data ini untuk memungkinkan penilaian komprehensif terhadap hasil risiko dan untuk menyediakan fungsionalitas Fitur kepada Anda. |

## 5\. **RETENSI DATA**

Kami akan menyimpan data pribadi yang diproses sehubungan dengan Fitur sebagai berikut:

| **Informasi Pribadi** | **Kebijakan Retensi**                                         |
| ------------------------ | ------------------------------------------------------------ |
| Data Log                 | Data log akan dihapus secara otomatis setelah 180 hari sejak pertama kali direkam. |
| Data SDK                 | Kami menyimpan data tersebut selama Anda menggunakan Fitur. Apabila penggunaan Fitur Anda berakhir, kami akan menghapus data ini setelah 5 hari. |

Anda dapat meminta penghapusan data pribadi tersebut sesuai dengan DPSA.

## 6\. **KETENTUAN KHUSUS**

Anda harus memastikan bahwa Fitur ini hanya digunakan oleh pengguna akhir yang setidaknya berusia minimum yang memungkinkan individu untuk menyetujui pemrosesan data pribadi. Persyaratan ini mungkin berbeda-beda, tergantung pada wilayah yurisdiksi tempat pengguna akhir berada.
