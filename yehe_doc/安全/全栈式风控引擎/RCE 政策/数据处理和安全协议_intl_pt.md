## 1\. **CONSIDERAÇÕES**

Este Módulo se aplica se você usar o Risk Control Engine (“**Recurso**”). Este Módulo está incorporado ao Contrato de privacidade e segurança de dados localizado no (“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347 )”). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo se aplicará conforme a inconsistência.

## 2\. **PROCESSAMENTO**

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de registro:**<br/>Informações de requisições da API:<br/>IP do usuário, volume de requisições, ID do aplicativo, número de identificação do usuário, duração do tempo e relógio | Usamos esses dados para fins de fornecer o Recurso a você e para que possamos realizar análises estatísticas para solução de problemas. |
| **Dados do SDK:**<br/>1\. Dados do navegador, do dispositivo e da conexão:<br/> Informações do computador pessoal ou dispositivo móvel que acessou o site de um cliente, incluindo navegador, modelo do dispositivo, sistema operacional e identificador exclusivo de dispositivos.<br/>2\. Dados comportamentais:<br/> Informações sobre atividades no site ou aplicativo de negócios, como tempo e frequência de acesso, páginas visualizadas, distribuição de deslizadas nas telas e tempo de permanência. | Processamos esses dados para permitir uma avaliação abrangente dos resultados de risco e fornecer a funcionalidade do Recurso para você. |


## 3\. **REGIÃO DO SERVIÇO**

Conforme especificado no DPSA.

## 4\. **SUBPROCESSADORES**

Conforme especificado no DPSA.

## 5\. **RETENÇÃO DE DADOS**

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações Pessoais** | **Política de Retenção**                                         |
| ------------------------ | ------------------------------------------------------------ |
| Dados de registro                 | Os dados de registro serão excluídos automaticamente após 180 dias de registro. |
| Dados do SDK                 | Retemos esses dados enquanto você usar o Recurso. Quando a sua utilização do Recurso for encerrada, excluiremos esses dados dentro de 5 dias. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. **CONDIÇÕES ESPECIAIS**

Você deve garantir que este Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
