## 1\. **背景**

本模块仅在您使用风控引擎（下称“**功能**”）时适用。本模块已纳入  的数据隐私和安全协议（“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347 )”）中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. **处理**

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **日志数据：**<br/>API 调用信息：<br/>用户 IP、通话音量、APPID、UIN、持续时间和时间戳 | 我们使用这些数据是为了向您提供本功能，并使我们能够通过统计分析来排除故障。 |
| **SDK 数据：**<br/>1\. 浏览器、设备和连接数据：<br/> 访问客户网站的个人计算机或移动设备的信息，包括浏览器、设备型号和操作系统，以及唯一设备标识符。<br/>2\. 行为数据：<br/> 商业网站或 APP 中的活动信息，例如访问时间和频率、浏览的页面、屏幕滑动操作的分布位置和停留时间。 | 我们处理这些数据是为了全面评估风险结果并向您提供本功能。 |

## 5\. **数据保留**

我们将存储因本功能而处理的个人数据，具体如下所述：

| **个人信息** | **保留政策**                                         |
| ------------------------ | ------------------------------------------------------------ |
| 日志数据                 | 日志数据在首次记录 180 天后会自动删除。 |
| SDK 数据                 | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 5 天内删除此数据。 |

您可以根据 DPSA 请求删除此类个人数据。

## 6\. **特殊条款**

您必须确保使用本功能的最终用户至少达到个人可同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
