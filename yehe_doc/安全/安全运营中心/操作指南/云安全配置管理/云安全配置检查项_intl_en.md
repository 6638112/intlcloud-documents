1. Log on to the [Security Operation Center Console](https://console.cloud.tencent.com/ssav2/config). In the left navigation pane, click **Cloud Security Configuration Management** > **Cloud Security Configuration Check Items** to go to the Cloud Security Configuration Check Items page.
2. In the Cloud Security Configuration Check Items page, you can search or filter to retrieve the required check items.

- **Search**
  In the upper-right corner of the "Cloud Security Configuration Check Items" list, you can search check items by entering the keywords of configuration requirements.
- **Filter**
  To filter the check items, click on "Check Type", "Check Object", "Risk Level", or "Related Specification" in the headers of the "Cloud Security Configuration Check Items" list.
  You can view the details of each check item and search and filter the check items.

3. After the required check item is retrieved, click **Configuration Requirements**. In the "Configuration Details" pop-up window that appears, you can view the basic information and related information of the check item.
4. In the "Configuration Details" pop-up window, move the pointer over the Referenced Policy link in the "Basic information" section until a list of configuration check policies appears. Then click on the name of the policy link. The system redirects you to a list of configuration check policies, where you can view the policy and perform other actions. To view the online Help Documentation for the current configuration item, you can click the Help Documentation link in the "Related information" section.
