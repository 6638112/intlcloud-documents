## 1\. 背景

本モジュールは、お客様がTencentCloud EdgeOne（「**機能**」）を使用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)のデータプライバシーとセキュリティ契約に組み込まれています。本モジュールで使用されていても定義されていない用語は、DPSAで定義された意味を持つものとします。DPSAと本モジュールとの間に齟齬が生じた場合、その齟齬の範囲内で本モジュールが適用されるものとします。

## 2\. 処理

弊社は本機能に関連して、次のデータを処理します：

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **システム構成データ:** ドメイン名、DNS設定レコード (ホストレコード、レコードタイプ、レコード値、アクセラレーションタイプ、TTL)、オリジンサイト設定 (オリジンサイトアドレス、ポート)、CDN設定レコード (アクセスルール、キャッシュルール、MD5認証パスワード、キー、小規模機能のCDN設定など)、SSL証明書とキー、セキュリティ設定 (保護ポリシーID、TCPパケットルールなどのカスタム保護ルール、 HTTP パケットルール) | 弊社は、お客様固有の設定に基づいて本機能をご提供する目的でのみ左記データを処理します。SSL証明書と鍵は、機能の一部としてセキュリティサービスを提供するために当社のSSL証明書サービス機能と統合され、ドメイン名とDNS構成レコードは、機能の一部としてDNSサービスを提供するために当社のDNSPod機能と統合されることに注意してください。また、このデータはクラウドオブジェクトストレージ (COS) 機能に保存され、TencentDB for MySQL機能に保存およびバックアップされることにご留意ください。 |
| **CDN キャッシュデータ:** エンドユーザーのオリジンサーバーのHTTP応答 (キャッシュされたコピー) | 当社は、お客様に機能を提供する目的でのみこのデータを処理します。このデータは、コンテンツ配信ネットワーク (CDN) 機能に保存されます。 |

## 5\. データ保持

弊社は、本機能に関連して処理された個人データを以下のように保管します：

| **個人情報**  | **保持ポリシー**                                         |
| ------------------------- | ------------------------------------------------------------ |
| システム構成データ  | お客様が本機能を使用する限り、当社はこのデータを保持します。本機能のご使用が終了すると、30日後にこのデータが削除されます。 |
| CDN キャッシュ データ            | お客様が本機能を使用する限り、当社はこのデータを保持します。本機能のご使用が終了すると、30日後にこのデータが削除されます。 |

お客様はDPSAに従って、この個人データの削除を要請できます。

## 6\. 特別な条件

お客様は、個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが本機能を使用することを保証する必要があります。これは、エンドユーザーが所在する管轄区域によって異なる場合があります。
