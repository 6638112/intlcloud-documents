## 1\. 简介
本模块仅在您使用边缘安全加速平台（简称“功能”）时适用。本模块已纳入[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，则本模块适用。
## 2\. 控制方
本模块中所述个人信息的控制方见隐私政策中的规定。
## 3\. 可用性
本功能可供全球范围内的用户使用。
## 4\. 我们如何使用个人信息
我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **边缘节点访问日志数据：** 访问时间戳、客户端 IP 地址、节点服务器公共 IP 地址、主机、端口、URL、访问的 HTTP 标头（最终用户的 HTTP 请求，如用户代理、引用站点、Cookie）、POST 数据正文（前 10KB）、是否存在恶意攻击、触发的安全规则 | 我们出于为您提供本功能的目的而使用此信息，包括：<li>处理关于不同维度的业务流量和顶级域名访问状态（如 URL、UA）的用户查询；<li>分析流量风险并识别恶意特征；以及<li>排除故障。<br/>请注意，这些数据将与我们的云数据仓库 ClickHouse 功能集成，以便存储和预聚合，并且还将存储在我们的云服务器 (CVM)、对象存储 (COS) 和云数据仓库 ClickHouse 功能中。| 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |
| **流量数据：** 每个主机在单个服务器 IP 的访问流量数据（传入流量、传出流量、请求数量） | 我们出于计算用户访问次数以便计费的目的而使用此信息。 <br/>请注意，这些数据将与我们的云数据仓库 ClickHouse 功能集成，以便存储和预聚合，并且还将在存储和备份在我们腾讯云数据库 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |
| **系统配置数据：** 节点配置数据（IP、ISP、地址）、服务器集群配置数据、API 配置数据、数据库地址和密钥 | 我们出于为您提供本功能的目的而使用此信息，包括确保内容分发网络 (CDN) 的正常运行。 <br/>请注意，这些数据存储和备份在我们的腾讯云数据库 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |
| **API 访问日志数据：** 访问时间戳、客户端 IP 地址、API 服务器内网 IP 地址、URL、请求正文（包括您的配置数据）、响应正文（包括您的配置数据） | 我们出于故障排除的目的而使用此信息。 <br/>请注意，这些数据将与我们的云审计 CloudAudit 功能集成，以提供 CAM 身份验证和审计日志，并存储在我们的日志服务功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |

## 5\. 我们如何存储和分享个人信息
见隐私政策中的规定。
此外，应用 ID 和 UIN 将存储在我们的腾讯云数据库 MySQL 功能中。
## 6\. 数据保留
我们将根据下列规定保留个人信息：

| **个人信息**          | **保留政策** |
| -------------------------           | -------------------- |
| 边缘节点访问日志数据        | 存储 30 天。  |
| 流量数据        | 存储 30 天，然后对这些数据进行匿名化处理。 |
| 系统配置数据            | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 30 天后删除此数据。 |
| API 访问日志数据         | 存储 180 天。 |
