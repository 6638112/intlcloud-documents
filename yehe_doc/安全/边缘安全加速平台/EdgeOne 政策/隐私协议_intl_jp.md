## 1\. はじめに
本モジュールは、お客様がTencentCloud EdgeOne（「機能」）を使用する場合に適用されます。本モジュールは、[プライバシーポリシー](https://intl.cloud.tencent.com/document/product/301/17345)内のプライバシーポリシーに組み込まれています。本モジュールで使用されてはいても定義されていない用語は、プライバシーポリシーで与えられた意味を持つものとします。プライバシーポリシーと本モジュールとの間に齟齬が生じた場合、その齟齬の範囲内で本モジュールが適用されるものとします。
## 2\. 管理者
本モジュールに記載されている個人情報の管理者は、プライバシーポリシーに規定されているとおりです。
## 3\. 可用性
本機能は世界中のユーザーが利用可能です。
## 4\. 弊社による個人情報の使用方法
弊社は、以下の法的根拠に従って、以下の方法で個人情報を使用します。

| **個人情報**                                     | **用途**                                                      | **法的根拠**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |

| **システム構成データ:** ノード構成データ (IP、ISP、アドレス)、サーバークラスター構成データ、API 構成データ、DB アドレスおよびキー | 弊社は、コンテンツ配信ネットワーク (CDN) の適切な機能の確保を含む、お客様に機能を提供する目的でこの情報を使用します。 <br/>このデータは、当社の TencentDB for MySQL 機能に保存されバックアップされますのでご注意ください。 | 左記情報は、お客様との契約を履行し、本機能を提供するために必要な情報として処理されます。 |
| **APIアクセスログデータ:** アクセスタイムスタンプ、クライアントIPアドレス、API サーバー イントラネットIPアドレス、URL、レスポンスボディ(構成データを含む)、レスポンスボディ(構成データを含む) | この情報は、トラブルシューティングの目的に使用します。 <br/>このデータは、CAM 認証および監査ログを提供するために当社のクラウド監査機能と統合され、当社のクラウド ログ サービス機能に保存されることに注意してください。 | 左記情報は、お客様との契約を履行し、本機能を提供するために必要な情報として処理されます。 |

## 5\. 当社による個人情報の共有および保存方法
プライバシーポリシーに規定されている通りです。
さらに、APPIDおよびUINはTencentDB for MySQL機能に保存されます。
## 6\. データ保持
弊社は、以下の条件に従って個人情報を保持します：

| **個人情報**          | **保持ポリシー** |
| -------------------------           | -------------------- |
| エッジノードアクセスログデータ        | 30日間保存されます。  |
| トラフィックデータ        | 30日間保存され、その後、これらのデータは匿名化されます。 |
| システム構成データ             | お客様が本機能を使用する限り、当社はこのデータを保持します。本機能のご使用が終了すると、30日後にこのデータが削除されます。 |
| APIアクセスログデータ         | 180日間保存されます。 |
