## 1\. 背景

本模块仅在您使用边缘安全加速平台（简称“**功能**”）时适用。本模块已纳入 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据隐私和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. 处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **系统配置数据：** 域名、DNS 配置记录（主机记录、记录类型、记录值、加速类型、TTL）、源站配置（源站地址、端口）、CDN 配置记录（如访问规则、缓存规则、MD5 认证密码、密钥、小功能的 CDN 配置）、SSL 证书和密钥、安全配置（保护策略 ID、TCP 数据包规则和 HTTP 数据包规则等自定义保护规则） | 我们仅为了按照您的特定配置向您提供本功能而处理这些数据。请注意，SSL 证书和密钥将与我们的 SSL 证书服务功能集成，作为该功能的一部分提供安全服务；域名和 DNS 配置记录将与我们的 DNSPod 功能集成，作为该功能的一部分提供 DNS 服务。请注意，此数据还将存储在我们的对象存储 (COS) 功能中，并存储和备份在我们的腾讯云数据库 MySQL 功能中。 |
| **CDN 缓存数据：** 最终用户源服务器的 HTTP 响应（缓存副本） | 我们仅为了向您提供本功能而处理这些数据。请注意，这些数据存储在我们的内容分发网络 (CDN) 功能中。 |

## 5\. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述：

| **个人信息**  | **保留政策**                                         |
| ------------------------- | ------------------------------------------------------------ |
| 系统配置数据 | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 30 天后删除此数据。 |
| CDN 缓存数据            | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将在 30 天后删除此数据。 |

您可以根据 DPSA 请求删除此类个人数据。

## 6\. 特殊条款

您必须确保使用本功能的最终用户至少达到个人可同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
