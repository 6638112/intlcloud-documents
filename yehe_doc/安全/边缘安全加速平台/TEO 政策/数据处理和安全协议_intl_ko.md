## 1\. 배경

이 모듈은 TencentCloud EdgeOne(“**기능**”)을 사용하는 경우 적용됩니다. 이 모듈은 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 개인 정보 보호 및 보안 계약에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 것과 같은 의미를 갖습니다. DPSA와 이 모듈 간에 상충이 있는 경우, 이 모듈은 상충되지 않는 범위 내에서 적용됩니다.

## 2\. 처리

당사는 시스템과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **이용**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **시스템 구성 데이터:** 도메인 이름, DNS 구성 레코드(호스트 레코드, 레코드 유형, 레코드 값, 가속 유형, TTL), 원본 사이트 구성(원본 사이트 주소, 포트), CDN 구성 레코드(예: 액세스 규칙, 캐시 규칙, MD5 인증 암호, 키, 작은 기능의 CDN 구성), SSL 인증서 및 키, 보안 구성(보호 정책 ID, TCP 패킷 규칙과 같은 사용자 지정 보호 규칙, HTTP 패킷 규칙) | 당사는 사용자의 고유한 구성에 따라 사용자에 대한 기능 제공 목적으로만 이 데이터를 처리합니다. SSL 인증서 및 키는 기능의 일부로 보안 서비스를 제공하기 위해 SSL 인증서 서비스 기능과 통합되며, 도메인 이름 및 DNS 구성 레코드는 기능의 일부로 DNS 서비스를 제공하기 위해 DNSPod 기능과 통합됩니다. 이 데이터는 Cloud Object Storage(COS) 기능에도 저장되고, TencentDB for MySQL 기능에 저장 및 백업됩니다. |
| **CDN 캐시 데이터:** 최종 사용자 원본 서버의 HTTP 응답(캐시 된 복사본) | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다. 이 데이터는 콘텐츠 전송 네트워크(CDN) 기능에 저장됩니다. |

## 3\. 서비스 지역

DPSA에 명시된 바와 같습니다.

## 4\. 하위 프로세서

DPSA에 명시된 바와 같습니다.

## 5\. 데이터 보존

당사는 시스템과 관련하여 처리된 개인 데이터를 다음과 같이 저장합니다.

| **개인 정보**  | **보존 정책**                                         |
| ------------------------- | ------------------------------------------------------------ |
| 시스템 구성 데이터 | 당사는 귀하가 해당 기능을 사용하는 동안 그러한 데이터를 보존합니다. 당사는 귀하의 기능 사용이 종료된 시점으로부터 30일 후에 이 데이터를 삭제합니다. |
| CDN 캐시 데이터            | 당사는 귀하가 해당 기능을 사용하는 동안 그러한 데이터를 보존합니다. 당사는 귀하의 기능 사용이 종료된 시점으로부터 30일 후에 이 데이터를 삭제합니다. |

귀하는 DPSA에 따라 이러한 개인 데이터의 삭제를 요청할 수 있습니다.

## 6\. 특수 조건

개인 데이터 처리에 동의할 수 있는 최소 연령을 충족하는 최종 사용자만 이 시스템을 사용할 수 있습니다. 이는 최종 사용자의 해당 지역에 따라 다를 수 있습니다.
