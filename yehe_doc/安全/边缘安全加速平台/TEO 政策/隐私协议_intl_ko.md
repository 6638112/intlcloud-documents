## 1\. 소개
이 모듈은 TencentCloud EdgeOne(“기능”)을 사용하는 경우 적용됩니다. 이 모듈은 [개인정보 처리방침](https://intl.cloud.tencent.com/document/product/301/17345)에 있는 개인정보 처리방침에 통합됩니다. 이 모듈에서 사용되지만 정의되지 않은 용어는 개인정보 처리방침에 정의된 것과 같은 의미를 갖습니다. 개인정보 처리방침과 이 모듈 간에 상충이 있는 경우, 이 모듈은 상충하지 않는 범위 내에서 적용됩니다.
## 2\. 컨트롤러
이 모듈에 언급된 개인정보 관리는 개인정보 처리방침에 명시되어 있습니다.
## 3\. 가용성
이 기능은 전 세계적으로 사용할 수 있습니다.
## 4\. 개인 정보를 이용하는 방식
당사는 다음과 같은 방법으로, 다음과 같은 법적 근거에 따라 정보를 이용합니다.

| **개인 정보**                                     | **이용**                                                      | **법적 근거**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Edge Node 액세스 로그 데이터:** 액세스 타임스탬프, 클라이언트 IP 주소, 노드 서버 공용 IP 주소, 호스트, 포트, URL, 액세스한 HTTP 헤더(사용자 에이전트, 참조자, 쿠키와 같은 최종 사용자의 HTTP 요청), POST 데이터 본문(처음 10KB), 악성 공격이 있는지 여부, 트리거 된 보안 규칙 | 당사는 다음을 포함하여 귀하에게 기능을 제공하기 위한 목적으로 이 정보를 사용합니다.<li>비즈니스 트래픽 및 다양한 차원(URL, UA 등)의 상위 액세스 상태에 대한 사용자 쿼리<li>트래픽 위험 분석 및 악성 특성 식별<li>문제 해결<br/>이 데이터는 저장 및 사전 집계 목적으로 ClickHouse 기능과 통합되며 Cloud Virtual Machine(CVM), Cloud Object Storage(COS), ClickHouse 기능에도 저장됩니다. | 당사는 기능 제공을 통해 귀하와 맺은 계약을 이행하는 데 필요하므로 이 정보를 처리합니다. |
| **트래픽 데이터:** 단일 서버 IP에서 각 호스트의 트래픽 데이터에 액세스(들어오는 트래픽, 나가는 트래픽, 요청 수) | 당사는 이 정보를 비용 청구 목적으로 사용자 방문 횟수를 계산하는 데사용합니다. <br/>이 데이터는 저장 및 사전 집계 목적으로 ClickHouse 기능과 통합되며 TencentDB for MySQL 기능에도 저장 및 백업됩니다. | 당사는 기능 제공을 통해 귀하와 맺은 계약을 이행하는 데 필요하므로 이 정보를 처리합니다. |
| **시스템 구성 데이터:** 노드 구성 데이터(IP, ISP, 주소), 서버 클러스터 구성 데이터, API 구성 데이터, DB 주소 및 키 | 당사는 콘텐츠 전송 네트워크(CDN)의 올바른 작동을 보장하는 것을 포함하여 귀하에게 기능을 제공하기 위한 목적으로 이 정보를 사용합니다. <br/>이 데이터는 TencentDB for MySQL 기능에 저장 및 백업됩니다. | 당사는 기능 제공을 통해 귀하와 맺은 계약을 이행하는 데 필요하므로 이 정보를 처리합니다. |
| **API 액세스 로그 데이터:** 액세스 타임스탬프, 클라이언트 IP 주소, API 서버 인트라넷 IP 주소, URL, 요청 본문(구성 데이터 포함), 응답 본문(구성 데이터 포함) | 당사는 문제 해결 목적으로 이 정보를 사용합니다. <br/>이 데이터는 CAM 인증 및 감사 로그를 제공하기 위해 Cloud Audit 기능과 통합되며 Cloud Log Service 기능에 저장됩니다. | 당사는 기능 제공을 통해 귀하와 맺은 계약을 이행하는 데 필요하므로 이 정보를 처리합니다. |

## 5\. 당사에서 개인 정보를 저장하고 공유하는 방식
개인정보 처리방침에 명시된 바와 같습니다. 
또한, APPID와 UIN은 TencentDB for MySQL 기능에 저장됩니다.
## 6\. 데이터 보존
당사는 다음에 따라 개인 정보를 보존합니다.

| **개인 정보**          | **보존 정책** |
| -------------------------           | -------------------- |
| Edge Node Cache Data: 액세스 로그 데이터        | 30일 동안 저장됩니다.  |
| 트래픽 데이터        | 30일 동안 저장되며, 그 후 해당 데이터는 익명으로 처리됩니다. |
| 시스템 구성 데이터            | 당사는 귀하가 해당 기능을 사용하는 동안 그러한 데이터를 보존합니다. 당사는 귀하의 기능 사용이 종료된 시점으로부터 30일 후에 이 데이터를 삭제합니다. |
| API 액세스 로그 데이터         | 180일 동안 저장됩니다. |
