1. Log in to the [Anti-DDoS Advanced (Global Enterprise Edition) Console](https://console.cloud.tencent.com/ddos/ddos-basic) and select **Anti-DDoS Advanced** -> **Application Accessing**.
2. Click **IP Access** on the application accessing page.
![](https://main.qcloudimg.com/raw/499cbf6c5aee1040aca5fc99b05aec42.png)
3. Click **Add Rule** on the IP accessing page.
![](https://main.qcloudimg.com/raw/4793023eccd1a6023f93d16b0e90b1c0.png)
4. In the pop-up **Bound Resource** window, select an Anycast IP and an Anti-DDoS Advanced instance, then click **OK** to bind the resource.
>?You can select any region that is available to your Anti-DDoS Advanced instance.
>
