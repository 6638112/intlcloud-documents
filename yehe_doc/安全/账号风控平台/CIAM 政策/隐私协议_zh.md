## 1\. 简介

如果您使用客户身份和访问管理 (CIAM)（下称“**功能**”），则本模块适用。本模块已纳入位于（“[**隐私政策**](https://intl.cloud.tencent.com/document/product/301/17345)”）的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，本模块适用。


## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。


## 3\. 可用性

本功能可供全球范围内的用户使用。


## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **管理员操作日志：** 地区、API 版本、时间戳、应用 ID、UIN、子账号 UIN、客户端 IP、请求源、CAM 上下文（客户端 IP 地址、q 令牌）、日志服务 (CLS) 账号区域、请求 ID    | 我们使用这些信息来确保本功能按要求运行，包括用于： <li>安全审查；和<br/><li>故障排除。  <br/> 请注意，这些数据集成并存储在我们的日志服务 (CLS) 功能中。 | 为履行合同要求以向您提供本功能，我们必须要处理此信息。 |
| **腾讯云账号 API 调用日志：** 时间戳、请求 IP   | 我们使用这些信息来确保本功能按要求运行，包括用于进行故障排除。   <br/>请注意，这些数据集成并存储在我们的云审计 CloudAudit 功能中。 | 为履行合同要求以向您提供本功能，我们必须要处理此信息。 |
| **统计数据：** 控制台统计信息（用户目录中的用户总数、应用程序总数、最近 7 天登录总数、最近 7 天成功注册总数、SMS 使用总数、邮箱使用总数、用户增长趋势（昨天、7 天、30 天）、用户分布图、最近登录的用户和最近注册成功的用户）；资源使用统计（每月活跃用户总数、使用状态、本月最大用户限制和服务开通时间）         | 我们使用这些信息为您提供有关管理和每月消费的统计数据。   <br/>请注意，这些数据集成、存储和备份在我们的云数据库 MongoDB（下称“MongoDB”）功能中。 | 为履行合同要求以向您提供本功能，我们必须要处理此信息。 |
| **用户操作日志：** 操作时间、操作结果、客户端 IP   | 我们使用这些信息来帮助您管理本功能的运行。   <br/>请注意，这些数据集成、存储并备份在我们的 MongoDB 功能中。 | 为履行合同要求以向您提供本功能，我们必须要处理此信息。 |


## 5\. 我们如何存储和共享个人信息

见隐私政策中的规定。


## 6\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息**               | **保留政策**                                         |
| -------------------------------------- | ------------------------------------------------------------ |
| 管理员操作日志               | 存储 180 天。                                         |
| 腾讯云账号 API 调用日志 | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将删除此数据。 |
| 统计数据                       | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将删除此数据。 |
| 用户操作日志                     | 我们会在您使用本功能期间保留此类数据。若您终止使用本功能，我们将删除此数据。 |
