## 1\. INTRODUÇÃO

Este módulo se aplica se você usar o Customer Identity and Access Management (Gerenciamento de Identidade e Acesso do Usuário, CIAM) (“**Recurso**”). Este Módulo é incorporado à política de privacidade localizada em (“[**Política de Privacidade**](https://intl.cloud.tencent.com/document/product/301/17345)"). Os termos usados, mas não definidos nesse Módulo, devem ter o significado dado a eles na Política de Privacidade. No caso de qualquer conflito entre a Política de Privacidade e esse Módulo, esse Módulo será aplicado conforme a inconsistência.


## 2\. CONTROLADORIA

O controlador das informações pessoais descritas nesse Módulo é o especificado na Política de Privacidade.


## 3\. DISPONIBILIDADE

Esse Recurso está disponível para os usuários em todo o mundo.


## 4\. COMO USAMOS AS INFORMAÇÕES PESSOAIS

Usaremos as informações das seguintes formas e de acordo com as seguintes bases jurídicas:

| **Informações Pessoais**                                    | **Uso**                                                      | **Base Jurídica**                                             |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Registro de ação do administrador:** Região, versão do API, registro de data e hora, ID do aplicativo, número de identificação do usuário (User Identification Number, UIN), UIN da subconta, IP do usuário, origem da solicitação, contexto do gerenciamento de contas do usuário (Customer Account Management, CAM) (endereço de IP do usuário, q-token), área de conta da Cloud Log Service (CLS), ID da requisição | Usamos essas informações para garantir que o recurso funcione conforme necessário, inclusive para: <li>auditorias de segurança; e<br/><li>solução de problemas. <br/> Observe que esses dados estão integrados e armazenados em nosso recurso Cloud Log Service (CLS). | Processamos essas informações porque são necessárias para a execução de nosso contrato com você e para o fornecimento do Recurso. |
| **Registro de requisição da API da conta Tencent Cloud:** Registro de data e hora, IP da requisição | Usamos essas informações para garantir que o recurso funcione conforme necessário, inclusive para solução de problemas.  <br/>Observe que esses dados estão integrados e armazenados em nosso recurso Cloud Audit. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você e para o fornecimento do Recurso. |
| **Dados estatísticos:** Estatísticas de painéis de controle \[número total de usuários no diretório de usuários, número total de aplicações, número total de logins nos últimos 7 dias, número total de registros bem-sucedidos nos últimos 7 dias, número total de uso de SMS, número total de uso de caixa de correio eletrônico, tendência de crescimento do usuário (ontem, 7 dias, 30 dias), mapa de distribuição de usuários, usuários conectados recentemente e usuários registrados com sucesso recentemente\]; estatísticas de uso de recursos (quantidade total mensal de usuário ativo, status de uso, limite máximo de usuário e tempo de abertura do serviço deste mês) | Usamos essas informações para fornecer estatísticas sobre gestão e consumo mensal.  <br/> Observe que esses dados estão integrados, armazenados e têm uma cópia de segurança em nosso recurso TencentDB for MongoDB (MongoDB). | Processamos essas informações porque são necessárias para a execução de nosso contrato com você e para o fornecimento do Recurso. |
| **Registro de operação do usuário:** Tempo de operação, resultado da operação, IP do cliente | Usamos essas informações para facilitar a gestão do funcionamento do recurso.  <br/> Observe que esses dados estão integrados, armazenados e têm uma cópia de segurança em nosso recurso MongoDB. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você e para o fornecimento do Recurso. |


## 5\. COMO ARMAZENAMOS E COMPARTILHAMOS INFORMAÇÕES PESSOAIS

Conforme especificado na Política de Privacidade.


## 6\. RETENÇÃO DE DADOS

Reteremos informações pessoais de acordo com o seguinte:

| **Informações Pessoais**               | **Política de Retenção**                                         |
| -------------------------------------- | ------------------------------------------------------------ |
| Registro de ação do administrador               | Armazenados por 180 dias.                                         |
| Registro da requisição de API da conta Tencent Cloud | Retemos esses dados enquanto você usar o  Recurso. Quando você encerrar o uso do recurso, excluiremos esses dados. |
| Dados estatísticos                       | Retemos esses dados enquanto você usar o Recurso. Quando você encerrar o uso do recurso, excluiremos esses dados. |
| Registro de operações do usuário                     | Retemos esses dados enquanto você usar o Recurso. Quando você encerrar o uso do recurso, excluiremos esses dados. |
