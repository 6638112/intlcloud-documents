## Ikhtisar
Jika Anda lupa kata sandi akun database Anda atau perlu mengubahnya saat menggunakan TencentDB for SQL Server, Anda dapat mengatur ulang kata sandi di konsol.

>?
>- Untuk TencentDB for SQL Server, fitur pengaturan ulang kata sandi telah terhubung ke [CAM](https://intl.cloud.tencent.com/document/product/236/14469); dengan demikian, sebaiknya Anda melakukan kontrol yang lebih ketat atas izin ke API pengaturan ulang kata sandi atau sumber daya sensitif instans TencentDB for SQL Server dengan memberikan izin tersebut hanya kepada personel yang sesuai.
>- Demi keamanan data, sebaiknya atur ulang kata sandi secara berkala minimal tiga bulan sekali.


## Petunjuk
1. Login ke [konsol TencentDB for SQL Server](https://console.cloud.tencent.com/sqlserver). Dalam daftar instans, klik ID instans atau **Manage** (Kelola) di kolom **Operation** (Operasi) untuk masuk ke halaman manajemen instans.
2. Pada halaman manajemen instans, pilih tab **Account Management** (Manajemen Akun), pilih akun target, dan klik **Reset Password** (Atur Ulang Kata Sandi) di kolom **Operation** (Operasi).
![](https://main.qcloudimg.com/raw/77a9ce4d8ec4ea21ebea829d6d3d3f18.png)
3. Di jendela pop-up, masukkan dan konfirmasikan kata sandi baru dan klik **OK** (OKE).
