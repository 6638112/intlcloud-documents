TencentDB for SQL Server mendukung penetapan instans ke berbagai proyek manajemen.
Login ke [Konsol TencentDB for SQL Server](https://console.cloud.tencent.com/sqlserver). Dalam daftar instans, klik ID instans untuk masuk ke halaman detail, dan klik **Switch to another project** (Beralih ke proyek lain) di "Proyek" untuk menetapkan instans ke proyek.

- Instans baca saja adalah instans terkait dari instans master dan harus berada dalam proyek yang sama dengan instans master.
- Menetapkan dan memindahkan instans database di seluruh proyek tidak akan memengaruhi layanan yang disediakan oleh instans.
- Anda perlu menentukan proyek yang berada di instans baru saat membelinya. Proyek defaultnya adalah **Proyek Default**.
- Instans yang ditetapkan dapat dipindahkan ke proyek lain melalui fitur **Beralih ke proyek lain**.
