## 操作场景
云数据库 SQL Server 提供克隆数据库功能，支持一键克隆已有的数据库到本实例下，克隆时需指定新库名称，新克隆数据库的数据库账号权限等相关信息和克隆前的数据库保持一致。

## 前提条件
源数据库的状态须为**运行中**。

## 操作步骤
1. 登录 [SQL Server 控制台](https://console.cloud.tencent.com/sqlserver)。
2. 在上方选择地域，找到需要克隆数据库的实例，单击实例 ID或**操作**列的**管理**，进入实例管理页。
![](https://qcloudimg.tencent-cloud.cn/raw/5ab42e35de0b77910062814cae0ab4b7.png)
3. 在实例管理页，选择**数据库管理**，在下方找到需要克隆的数据库，在其**操作**列单击**克隆**。
![](https://qcloudimg.tencent-cloud.cn/raw/5ab42e35de0b77910062814cae0ab4b7.png)
4. 在弹出的对话框，设置新库的名称，单击**确定**。
![](https://qcloudimg.tencent-cloud.cn/raw/b9fc28833c5d884ed1a0d4733df640ff.png)
