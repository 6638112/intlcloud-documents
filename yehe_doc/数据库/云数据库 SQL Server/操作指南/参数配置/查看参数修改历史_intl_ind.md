
Anda dapat melihat log modifikasi parameter di konsol TencentDB for SQL Server.

## Petunjuk
1. Login ke [konsol TencentDB for SQL Server](https://console.cloud.tencent.com/sqlserver). Dalam daftar instans, klik ID instans untuk mengakses halaman manajemen instans.
2. Lihat log modifikasi parameter di **Parameter Configuration** (konfigurasi Parameter) > **Modification Log** (Log Modifikasi).
