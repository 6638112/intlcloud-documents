## Mode Penagihan
TencentDB for SQL Server menawarkan mode penagihan berikut:

| Mode Penagihan | Mode Pembayaran | Kasus Penggunaan | 
|---------|---------|---------| 
| Bayar sesuai pemakaian | Pascabayar. Anda dapat mengajukan permohonan berdasarkan sumber daya untuk penggunaan sesuai permintaan dan akan dikenakan biaya berdasarkan penggunaan aktual pada saat pelunasan pembayaran. | Sangat cocok untuk bisnis yang mungkin sangat berfluktuasi dan seketika. Dalam mode ini, instans dapat segera dirilis setelah digunakan untuk mengurangi biaya. |
