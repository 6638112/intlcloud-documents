  
TencentDB for SQL Server will change the time zone in the maintenance information on February 24, 2023. Specifically, the maintenance time will be changed from the local time to Beijing time.

## Change time
Friday, February 24, 2023, Beijing time.

## Change description
The maintenance time of an instance will be changed from the **local time** to **Beijing time**.

## Impact description
After the change, the effective time of [configuration adjustment](https://intl.cloud.tencent.com/document/product/238/44352) and the switch time during [cross-AZ migration](https://intl.cloud.tencent.com/document/product/238/42695) will be in Beijing time.
