This document describes TencentDB for SQL Server instance specification types.
>?Currently, TencentDB for SQL Server only supports dedicated instance.


| Specification Type | Description | 
|---------|---------|
| Dedicated | <li>The purchased instance has exclusive access to the CPU, memory, and disk resources and features long-term stability. <li>The instance deployed with local disks are not be affected by the activities of other instances on the physical machine, while the instance deployed with cloud disks is considered a dedicated cloud database. <li>A dedicated instance with the highest configurations can monopolize a physical machine and all of its resources. |
