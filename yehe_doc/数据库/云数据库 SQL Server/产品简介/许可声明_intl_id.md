## Disertakan Lisensi
Tencent Cloud menawarkan instans TencentDB dengan **license included** (disertakan lisensi). Harga model **license included** (disertakan lisensi) mencakup lisensi perangkat lunak SQL Server, sumber daya perangkat keras yang mendasari, dan fungsi manajemen TencentDB, jadi tidak perlu membeli lisensi Microsoft SQL Server secara terpisah.

Anda hanya perlu membayar untuk spesifikasi instans dan durasi penggunaan, dan tidak perlu khawatir tentang perangkat keras dan biaya lisensi saat menggunakan database "license included" (disertakan lisensi). Anda dapat memangkas biaya TI dengan hanya membayar biaya variabel yang rendah daripada biaya tetap yang tinggi.


>!- Membeli TencentDB for SQL Server tidak berarti Anda membeli kontrak lisensi jangka panjang Microsoft SQL Server; oleh karena itu, Anda tidak boleh menggunakan lisensi TencentDB di luar layanan ini. Untuk informasi selengkapnya, lihat [Persyaratan Produk Lisensi Volume Microsoft dan Ketentuan Layanan Online](http://www.microsoftvolumelicensing.com/ProductResults.aspx?doc=Product%20Terms,OST&fid=20).
>- Harga dasar TencentDB for SQL Server **does not include** (tidak menyertakan) layanan Microsoft atau Tencent Cloud lainnya.
