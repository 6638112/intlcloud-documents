Ini memiliki penggunaan yang luar biasa, stabilitas tinggi, keandalan, dan keamanan, perlindungan keamanan data, dan failover dalam hitungan detik, yang memungkinkan Anda untuk lebih berfokus pada pengembangan aplikasi itu sendiri.

Berkat dukungan komprehensifnya untuk aplikasi berbasis Windows .NET Framework, SQL Server ini banyak digunakan di berbagai bidang seperti layanan pemerintah, keuangan, perawatan kesehatan, ritel, pendidikan, dan gaming.

### Arsitektur Deployment
TencentDB for SQL Server mendukung tiga arsitektur deployment:

#### Edisi Dasar
Edisi Dasar memisahkan komputasi dan penyimpanan berdasarkan disk cloud premium dan di-deploy pada satu node.

#### Edisi Ketersediaan Tinggi Server Ganda
Edisi Ketersediaan Tinggi Server Ganda terdiri dari satu database SQL Server dan satu database SQL Server cermin yang di-deploy di seluruh rak/AZ.

#### Edisi Kluster
Edisi Kluster mengadopsi arsitektur Always On, termasuk satu utama dan satu replika yang di-deploy di seluruh rak/AZ secara default.
