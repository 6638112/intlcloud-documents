### Can I change a TencentDB for SQL Server instance from Dual-Server High Availability Edition to Basic Edition?
No. To adjust the architecture, purchase a new Basic Edition instance and migrate the data as instructed in [Cold Backup Migration](https://intl.cloud.tencent.com/document/product/238/39005).

### Can I upgrade a TencentDB for SQL Server instance from Dual-Server High Availability Edition to Cluster Edition?
Yes. For detailed directions, see [Adjusting Instance Architecture](https://intl.cloud.tencent.com/document/product/238/44353).

### Does TencentDB for SQL Server support version upgrade?
TencentDB for SQL Server Basic, Dual-Server High Availability, and Cluster Editions support version upgrade. For more information, see [Adjusting Instance Version](https://intl.cloud.tencent.com/document/product/238/44354).

### Does TencentDB for SQL Server support version downgrade?
TencentDB for SQL Server doesn't support version downgrade; for example, you cannot downgrade a Basic Edition instance from v2019 to v2017.
