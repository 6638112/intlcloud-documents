### Bagaimana penentuan harga TencentDB for SQL Server?
TencentDB for SQL Server hadir dengan lisensi Microsoft asli dan dapat ditagih dengan cara bayar sesuai pemakaian (pascabayar).
Harga terdiri dari dua bagian: spesifikasi instans (kapasitas memori) dan ruang penyimpanan (kapasitas disk). Untuk detail harga di berbagai wilayah, silakan lihat [Harga Produk](https://intl.cloud.tencent.com/document/product/238/8294).

### Bagaimana cara membeli instans TencentDB for SQL Server?
Masuk ke [Konsol TencentDB for SQL Server](https://console.cloud.tencent.com/sqlserver) untuk membuat instans atau membeli instans langsung di [halaman pembelian](https://buy.cloud.tencent.com/sqlserver).


### Apakah saya harus membeli lisensi SQL Server?
Tencent Cloud dapat menyediakan instans TencentDB yang sudah "termasuk lisensi". Harga model "termasuk lisensi" mencakup lisensi perangkat lunak SQL Server, sumber daya perangkat keras yang mendasarinya, dan fitur manajemen TencentDB, jadi Anda tidak perlu membeli lisensi Microsoft SQL Server secara terpisah.
Dengan database "termasuk lisensi", Anda hanya perlu membayar untuk spesifikasi instans dan durasi penggunaan serta tidak perlu khawatir tentang perangkat keras dan biaya lisensi. Oleh karena itu, Anda hanya membayar biaya variabel yang rendah alih-alih biaya tetap yang tinggi, sehingga membantu Anda mengurangi biaya TI.
>
- Membeli TencentDB for SQL Server tidak sama dengan membeli kontrak lisensi jangka panjang Microsoft SQL Server; oleh karena itu, Anda tidak boleh menggunakan lisensi TencentDB di luar layanan ini. Untuk informasi selengkapnya, silakan lihat [Ketentuan Produk Lisensi Volume dan Ketentuan Layanan Online Microsoft](http://www.microsoftvolumelicensing.com/ProductResults.aspx?doc=Product%20Terms,OST&fid=20).
- Harga dasar TencentDB for SQL Server tidak termasuk layanan Microsoft atau Tencent Cloud lainnya.

Harap baca [deskripsi pengembalian dana](https://intl.cloud.tencent.com/document/product/238/31431) dan kebijakan sebelum mengembalikan instans.
