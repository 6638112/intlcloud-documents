
## 1\. 背景

本模块仅在您使用数据库审计（简称“**功能**”）时适用。本模块包含在位于 https://intl.cloud.tencent.com/document/product/301/17347 的数据隐私和安全协议（简称“**DPSA**”）中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致的范围，本模块应适用。

## 2\. 处理

我们将因本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据**  （您设置的审计策略、审计策略的名称、资源 ID、资源 名称、地区、资源配置详情、您的 Elasticsearch 服务 数据库密码） | 我们 仅为了按照您的特定配置 向您提供本功能而处理此数据。请注意，此数据与我们的腾讯云数据库 MySQL、 腾讯云数据库 MongoDB、TDSQL-C 功能集成，并存储在我们的 Elasticsearch  服务功能中。 |
| **审计日志数据**  （包括 SQL 语句执行时间、线程 ID、行明细、处理次数明细 、失败 SQL 语句错误代码、审计规则号、 客户端 IP、实例名称、实例 ID、用户名（无论是您创建的还是 随机）、数据库名称、审计策略名称、SQL 语句和 SQL 语句 类型） | 我们 仅为了向您提供本功能而处理此数据，以便 记录数据库操作。请注意 ，此数据是从目标数据库功能接收的，并且 （实例 ID 除外）存储在我们的 Elasticsearch 服务功能中。 |


## 3\. 服务区域

见 DPSA 中的规定。

## 4\. 分包处理方

见 DPSA 中的规定。

## 5\. 数据保留

我们将存储因本功能处理的个人数据，具体如下所述：

| **个人 信息**                                    | **保留政策**                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 配置数据  审计日志数据（不存储 的实例 ID 除外） | 我们会在您选择的数据保留期限内 保留此数据。如果您不选择 数据保留期限，我们将在存储 30 天 后自动删除此数据。如果您在您的数据保留期限结束前 请求删除此数据，此数据将在提出删除请求后的 3 天内 删除。如果本功能被停止，此数据将保留 3 天， 然后删除。如果您在服务到期后不续订服务，此数据将在到期后 3 天内 自动删除。 |

您可以根据 DPSA 请求删除此个人数据。

## 6\. 特殊条款

您必须确保使用本功能的最终用户至少达到了个人可以同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
