
## 1\. 背景

本モジュールは、お客様がデータベース監査（「**本機能**」）を使用する場合に適用されます。本モジュールは、 https://intl.cloud.tencent.com/document/product/301/17347 にあるデータプライバシーとセキュリティ契約（「**DPSA**」）に組み込まれています。本モジュールで使用されているが定義されていない用語は、DPSAで定義された意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

## 2\. 処理

当社は本機能に関連して、次のデータを処理します。

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **構成データ**  (監査ポリシーセット、監査ポリシーの名前、リソース ID、リソース 名、リージョン、リソース構成の詳細、Elasticsearch サービス データベースのパスワード) | 当社は、お客様の特定の設定に基づいて 本機能を提供する目的でのみ、このデータを処理 いたします。このデータは、当社のMySQL用テンセントDB、 モンゴDB用テンセントDB、TDSQL-C機能と統合されており、Elasticsearchサービス 機能に保管されていることをご了承ください。 |
| **監査ログデータ**  (SQL ステートメントの実行時間、スレッド ID、行の 詳細、処理時間の詳細、失敗した SQL ステートメントのエラーコード、監査規則番号、 クライアント IP、インスタンス名、インスタンス ID、ユーザー名 (ユーザーまたは ランダムのどちらによって作成されたか)、データベース名、監査ポリシー名、SQL ステートメント、および SQL ステートメント タイプを含む) | 当社は、 お客様に本機能を提供し 、データベースの運用状況を記録する目的でのみ、このデータを処理します。このデータは、 ターゲットデータベース機能から受信され、 (インスタンスID以外)、 弾性検索サービス機能に保管されることについてご了承ください。 |


## 3\. サービス地域

DPSA で規定されている通り。

## 4\. 復処理者

DPSA で規定されている通り。

## 5\. データ保持

当社は、本機能に関連して処理された個人データを以下のように保管します。

| **個人 情報**                                    | **保持方針**                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 構成データ 監査ログデータ(インスタンス ID 以外、 保管されません) | お客様が選択したデータ 保持期間の制限に従って、このようなデータを保持します。データ保持期限を選択しない場合、30 日の保管後にデータが自動的に削除されます。データ保持期限が終了する前に そのようなデータの削除を要求した場合、削除要求が行われてから 3  日以内に削除されます。本機能が終了した場合、当該データは 3日間保持された後に 削除されます。お客様がサービス期限終了時にサービスを更新しない場合、当該データはサービス期限 終了の 3日後に自動的に削除されます。 |

お客様は DPSA に従って、当該個人データの削除を要請することができます。

## 6．特別な条件

お客様は、本機能を、個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが使用することを保証する必要があります。これは、エンドユーザーが所在する法域によって異なる場合があります。
