Tencent Cloud는 사용자가 보다 쉽게 계정을 조회할 수 있도록 2022년 6월 15일부터 데이터베이스 감사 리소스 ID를 데이터베이스 인스턴스 리소스 ID로 단계적으로 변경합니다.


## 변경 일자
2022년 6월 15일(수) ~ 2022년 6월 30일(목).

## 관련 설명
이번 변경은 청구서 내역 중 리소스 ID 표시에만 영향을 미치며 과금에는 영향을 주지 않습니다.

## FAQ
#### 2022년 6월 청구서의 통합 과금 항목에 두 개의 다른 리소스 ID가 표시됩니까?
정상적인 현상입니다. 감사의 기존 리소스 ID는 dbAudit#로 시작하며, 이제 데이터베이스 리소스 ID로 변경됩니다. dbAudit#으로 시작하는 리소스 ID 발생 비용은 변경 이전의 비용입니다.

#### 리소스 ID가 변경되면 다른 기능이 영향을 받습니까?
아무런 영향을 받지 않습니다.
