
This document describes how to compare the monitoring data of multiple TencentDB for MongoDB instances in the same chart in the console for easier troubleshooting.

## Directions
1. Log in to the [TencentDB for MongoDB console](https://console.cloud.tencent.com/mongodb), click the icon in the **Monitoring/Status** column in the instance list, and enter the instance monitoring page.
![](https://main.qcloudimg.com/raw/d54d5be086cbc7d44cf067a1a9a7c187.png)
2. Click **Compare Monitoring Data of Instances** on the right to enter the [dashboard page](https://console.cloud.tencent.com/monitor/dashboard2/) in the Cloud Monitor console.
3. On the **Dashboard List** page, [create a dashboard](https://intl.cloud.tencent.com/document/product/248/38468) and [a chart](https://intl.cloud.tencent.com/document/product/248/38477) to view and compare monitoring data of multiple instances.
