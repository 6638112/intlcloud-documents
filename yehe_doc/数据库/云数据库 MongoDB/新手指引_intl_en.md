## 2. TencentDB for MongoDB Billing Modes
TencentDB for MongoDB adopts the **pay-as-you-go** billing mode. Please see [Billing Overview](https://intl.cloud.tencent.com/document/product/240/3550) for details.

## 3. Getting Started
#### 3.1 Creating a TencentDB for MongoDB instance
Before using TencentDB for MongoDB, you need to sign up for a Tencent Cloud account and purchase a TencentDB for MongoDB instance. For more information, please see [Creating TencentDB for MongoDB Instance](https://intl.cloud.tencent.com/document/product/240/3551).

#### 3.2 Connecting to a TencentDB for MongoDB instance
You can access a TencentDB for MongoDB instance from a CVM instance by different means. For more information, please see [Connecting to TencentDB for MongoDB Instance](https://intl.cloud.tencent.com/document/product/240/7092).
