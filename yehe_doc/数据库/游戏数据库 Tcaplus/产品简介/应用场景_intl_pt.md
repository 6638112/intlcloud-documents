### Jogos para dispositivos móveis

Como os jogos para dispositivos móveis são caracterizados por tempo de jogo fragmentado, interações frequentes dos jogadores, suporte para os modos todos os servidores/todas as regiões e vários servidores/várias regiões, taxa de iteração rápida e várias atividades operacionais, há vários requisitos de dados e, portanto, baixa a latência é muito importante para o armazenamento de dados. O TcaplusDB pode dar suporte e otimizar jogos para dispositivos móveis de acordo com o aproveitamento de tecnologias como operação em lote, mesclagem automática de servidor, expansão/redução de capacidade com serviço ininterrupto e troca de dados frios e quentes. Além disso, o TcaplusDB também provê suporte direcionado e otimização com relação a reversões de dados frequentes, alta disponibilidade, atualizações de dados e outras funcionalidades relacionadas a dados.

### Jogos para computador

Os jogadores de jogos para computador costumam ficar logados por longos períodos e os jogos para computador têm um longo ciclo de vida. A maioria dos jogos para computador é executada no modo vários servidores/várias regiões; portanto, há grandes quantidades de registros de dados e, consequentemente, altos requisitos para baixa latência. O TcaplusDB pode prover suporte e otimizar jogos para computador usando tecnologias como recuperação automática de desastres, particionamento de dados, empacotamento automático de registros e cache combinado com armazenamento em disco SSD.

### Jogos para navegador

Os jogos para navegador apresentam inicialização/mesclagem de servidores constantes e geralmente são executados de forma ininterrupta. Como os navegadores têm capacidade limitada de armazenar dados em cache, altos requisitos são impostos ao sistema de armazenamento de dados de back-end. O TcaplusDB pode prover suporte e otimizar jogos para navegador por meio de mesclagem automática de servidores, expansão/redução de capacidade com serviço ininterrupto e cache combinado com armazenamento em disco de alta velocidade.

### Aplicativos de redes sociais

Em aplicativos de redes sociais, os dados podem ser criados livremente pelos usuários e os comentários são usados com frequência. Os conteúdos são agregados por tópico, e campos como Texto, Link e Hora possuem comprimento fixo. A atividade de dados é determinada pela hora, e mais dados são lidos e menos dados são gravados. O TcaplusDB pode prover suporte e otimizar aplicativos de redes sociais usando tecnologias como armazenamento de lista, suporte a tipos de dados heterogêneos, troca de dados quentes e frios e separação de leitura/gravação.
