O TcaplusDB pode ser acessado e lido de várias maneiras dentre ferramenta de cliente, kits de ferramentas de SDK em várias linguagens de programação e API RESTful.

## Acesso do TcaplusDB por meio da ferramenta de cliente
Você pode usar [tcaplus_client](https://intl.cloud.tencent.com/document/product/1016/37719) para Linux para acessar o TcaplusDB.

## Acesso do TcaplusDB por meio da API SDK para C++
Você pode usar o SDK para [C++](https://intl.cloud.tencent.com/document/product/1016/30289) para acessar o TcaplusDB.

## Acesso do TcaplusDB por meio da API RESTful
Você pode acessar o TcaplusDB por meio da API RESTful. Para obter mais informações, consulte [Descrição](https://intl.cloud.tencent.com/document/product/1016/37721).

- Você pode acessar o TcaplusDB por meio da API RESTful para Python conforme as instruções [aqui](https://intl.cloud.tencent.com/document/product/1016/37725).

- Você pode acessar o TcaplusDB por meio da API RESTful para PHP conforme as instruções [aqui](https://intl.cloud.tencent.com/document/product/1016/37724).

- Você pode acessar o TcaplusDB por meio da API RESTful para Go conforme as instruções [aqui](https://intl.cloud.tencent.com/document/product/1016/37722).

- Você pode acessar o TcaplusDB por meio da API RESTful para Java conforme as instruções [aqui](https://intl.cloud.tencent.com/document/product/1016/37723).
