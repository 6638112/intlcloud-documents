## 1\. CONSIDERAÇÕES

Este Módulo se aplica se você usar o Centro de Gerenciamento de Banco de Dados (“**Recurso**”). Este Módulo está incorporado ao Contrato de Privacidade e Segurança de Dados localizado em https://intl.cloud.tencent.com/document/product/301/17347 (“**DPSA**”). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo deve ser aplicado conforme a inconsistência.

## 2\. PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados Básicos**  (ID de recurso, como endereço IP e porta, nome do recurso, nome de usuário e  senha na instância do banco de dados, modelo SQL, ID do app, SQL executado) | Apenas  processamos esses dados com a finalidade de fornecer o Recurso a você  de acordo com a sua configuração específica.  Note que esses dados estão armazenados em nosso recurso MySQL.  Note que certos dados (endereço IP, porta, senha) são recebidos  a partir de recursos de banco de dados de destino, como MySQL, MariaDB, TDSQL-C, Redis e  TDSQL para recursos MySQL. |

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações  pessoais** | **Política de retenção**                                         |
| ------------------------- | ------------------------------------------------------------ |
| Dados básicos                | Retemos tais dados de acordo com o seu  limite de tempo de retenção de dados escolhido ou até você excluir tais dados manualmente. Caso contrário, quando  seu uso do Recurso e/ou conta é encerrado ou não renovado, nós vamos  excluir esses dados após 7 dias. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. CONDIÇÕES ESPECIAIS

Você deve garantir que este recurso seja usado somente por usuários finais com a idade exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
