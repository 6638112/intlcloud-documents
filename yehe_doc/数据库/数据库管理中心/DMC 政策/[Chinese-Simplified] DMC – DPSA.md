## 1\. 背景

本模块仅在您使用数据库管理中心（简称“**功能**”）时适用。本模块包含在位于 https://intl.cloud.tencent.com/document/product/301/17347 的数据隐私和安全协议（简称“**DPSA**”）中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致的范围，本模块应适用。

## 2\. 处理

我们将因本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **基本数据**  （资源 ID \[例如 IP 地址和端口\]、资源名称、数据库实例中的用户名及密码、SQL 模板、应用程序 ID、已执行的 SQL） | 我们仅为了按照您的特定配置向您提供本功能而处理此数据。请注意，这些数据存储在我们的 MySQL 功能中。请注意，某些数据（IP 地址、端口、密码）接收自目标数据库功能，例如 MySQL、MariaDB、云原生数据库 TDSQL-C、Redis 和 TDSQL MySQL 版功能。 |

 

## 3\. 服务区域

见 DPSA 中的规定。

## 4\. 分包处理方

见 DPSA 中的规定。

## 5\. 数据保留

我们将存储因本功能处理的个人数据，具体如下所述：

| **个人信息** | **保留政策**                                         |
| ------------------------- | ------------------------------------------------------------ |
| 基本数据                | 我们会在您选择的数据保留期限内保留此数据，或直到您手动删除此等数据。此外，若您终止使用功能和/或帐户或不再进行续订，我们将在 7 天内删除此数据。 |

您可以根据 DPSA 请求删除此个人数据。

## 6\. 特殊条款

您必须确保使用本功能的最终用户至少达到了个人可以同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
