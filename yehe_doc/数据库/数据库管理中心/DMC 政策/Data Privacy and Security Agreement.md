## 2. PROCESSING

We will process the following data in connection with the Feature:

| **Personal Information**                                     | **Use**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Basic Data**  (resource ID such as IP address and port, resource name, username and  password in the database instance, SQL template, APP ID, executed SQL) | We  only process this data for the purposes of providing the Feature to you in  accordance to your specific configuration.  Please note that this data stored in our MySQL feature.  Please note that certain data (IP address, port, password) is received  from target database features such as MySQL, MariaDB, TDSQL-C, Redis, and  TDSQL for MySQL features. |
