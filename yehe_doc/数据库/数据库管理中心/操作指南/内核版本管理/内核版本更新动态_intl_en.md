This document describes the TencentDB for PostgreSQL kernel version updates. To upgrade the kernel, [submit a ticket](https://console.cloud.tencent.com/workorder/category).

#### Version changes
- Integrated all the features of open-source PostgreSQL 13.3.
- Added the admin role `pg_tencentdb_superuser`.
- Integrated Tencent Cloud's historical optimization feature packages for PostgreSQL.
