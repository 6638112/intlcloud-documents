TDSQL-C for MySQL allows you to assign a cluster to different projects for management.

This document describes how to modify the project of a cluster in the console.

## Directions
1. Log in to the [TDSQL-C for MySQL console](https://console.cloud.tencent.com/cynosdb).
2. Click the **ID** of the target cluster in the cluster list or **Manage** in the **Operation** column to enter the cluster details page.
3. On the cluster details page, select the **Basic Info** tab, find **Project**, and click **Switch to Another Project**.
![](https://qcloudimg.tencent-cloud.cn/raw/46c9fdf68b0fd3f9433eebc6cbf86fe5.png)
4. In the pop-up window, select a project and click **OK**.
![](https://qcloudimg.tencent-cloud.cn/raw/0afef3fb20f21f93186434658b62dcd4.png)
