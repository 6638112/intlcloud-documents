您可以在控制台授权管理已有的数据库账号，可以对数据库账号授予全局特权或者对象级特权，同时您也可以取消其授权。

## 操作步骤
1. 登录 [TDSQL-C MySQL 版控制台](https://console.cloud.tencent.com/cynosdb)，在集群列表，找到需要的集群，单击集群 ID，进入集群管理页面。
2. 在集群管理页面，选择**账号管理**页，单击**操作**列的**修改权限**。
![](https://staticintl.cloudcachetci.com/yehe/backend-news/VDQH725_1.png)
3. 在弹出的对话框，选中或者取消需要授予的权限，单击**确定**。
   - 全局特权：拥有实例下所有数据库的所有权限。
   - 对象级特权：拥有实例下特定数据库的权限。
   ![](https://staticintl.cloudcachetci.com/yehe/backend-news/r5fR784_2.png)

>?
>- 如果您需要批量授权或回收库表的权限，可单击**批量授权/回收表权限**，在弹窗下，选择**授权**或**回收**，然后选择需授权的多个库表以及所需权限，单击**确定**。
![](https://staticintl.cloudcachetci.com/yehe/backend-news/yP7a902_3.png)
>- 如果您想要预览账号权限的变更，您可以单击**预览**查看。
>![](https://staticintl.cloudcachetci.com/yehe/backend-news/A3Xj843_4.png)
