This document describes how to reset the password of the database admin account in the console.

## Directions
1. Log in to the [TDSQL-C for MySQL console](https://console.cloud.tencent.com/cynosdb), find the target cluster in the cluster list, and click the cluster ID to enter the cluster management page.
2. On the cluster management page, select the **Account Management** tab and click **Reset Password** in the **Operation** column.
![](https://staticintl.cloudcachetci.com/yehe/backend-news/6dbw695_16.png)
3. In the pop-up window, enter the new password and click **OK**.
![](https://staticintl.cloudcachetci.com/yehe/backend-news/BwGZ633_17.png)
>?Password must contain 8-64 characters in at least three of the following four types: uppercase letters, lowercase letters, digits, and symbols (`~!@#$%^&*_-+=|\(){}[]:;'<>,.?/`).
