This document describes how to modify the remarks of a database account in the console to allocate and manage the account resources.

## Directions
1. Log in to the [TDSQL-C for MySQL console](https://console.cloud.tencent.com/cynosdb).
2. Select the region, find the target cluster in the cluster list, and click the cluster ID or **Manage** in the **Operation** column to enter the cluster management page.
3. Click **Account management**, find the target account, and click ![](https://staticintl.cloudcachetci.com/yehe/backend-news/ksnw677_6.png) after the remarks.
![](https://staticintl.cloudcachetci.com/yehe/backend-news/TZfZ660_7.png)
4. In the pop-up window to modify the remarks, enter the remarks (up to 255 characters), and click**OK**.
![](https://staticintl.cloudcachetci.com/yehe/backend-news/hrjX728_8.png)
