To disable an existing database account, you can delete it in the console.
>!
>- A database account cannot be restored once deleted.
>!To prevent an unintentional deletion from disrupting business, you need to make sure that the database account to be deleted is no longer in use by any applications.

1. Log in to the [TDSQL-C for MySQL console](https://console.cloud.tencent.com/cynosdb), find the target cluster in the cluster list, and click the cluster ID to enter the cluster management page.
2. On the cluster management page, find the target account in the account list on the **Account Management** tab, click **More** > **Delete Account** in the **Operation** column.
