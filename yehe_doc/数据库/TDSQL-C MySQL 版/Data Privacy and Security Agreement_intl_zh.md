## 1. 背景
本模块仅在您使用云原生数据库 TDSQL-C（简称“**功能**”）时适用。本模块包含在位于 https://intl.cloud.tencent.com/document/product/301/17347 的数据隐私和安全协议（简称“**DPSA**”）中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。


## 2. 处理

我们将因本功能处理以下数据：

| **个人信息**                                                 | **用途**                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据**：参数名称和参数值                               | 我们仅为了按照您的特定配置向您提供本功能而处理此数据。       |
| **数据库数据**：数据库文件（如果您在数据库中存储个人数据）和系统文件 | 我们仅为了向您提供本功能而处理此数据。请注意，我们无法访问存储在数据库中的任何个人数据，亦无法控制这些数据。 |
| **监控数据**：监控指标                                       | 我们仅为了允许您监控操作状态而处理此数据。请注意，此功能通过我们的“云监控”功能执行，您可以为此服务启用“云监控”功能。这些信息将通过我们的“云监控”功能进行存储和备份。 |


## 3. 服务区域

 见 DPSA 中的规定。


## 4. 分包处理方

见 DPSA 中的规定，包括 Aceville Pte Ltd。

## 6. 特殊条款

您必须确保使用本功能的最终用户至少达到了个人可以同意处理自己个人数据的最低年龄。根据最终用户所处的司法管辖区，这一要求可能有所不同。
