This document describes the TencentDB for PostgreSQL kernel version updates. To upgrade the kernel, [submit a ticket](https://console.cloud.tencent.com/workorder/category).

## PostgreSQL 14
### v14.2_r1.1 
- Supported the cos_fdw extension.

### v14.2_r1.0
- Integrated open-source PostgreSQL 14.2 and introduced TencentDB for PostgreSQL's kernel change capabilities.
- Supported the pg_cron extension.
- Supported the TopN extension.
- Supported TDE.
