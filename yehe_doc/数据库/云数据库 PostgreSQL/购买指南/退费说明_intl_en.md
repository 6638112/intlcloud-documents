- Pay-as-you-go instances can be returned, but there will be no refunds.

## Return Process
Go to the [ticket](https://console.cloud.tencent.com/workorder/category) page, select **Problem Category** > **TencentDB for PostgreSQL** > **Instance Return**, enter the instance name, IP, port, reason for return, problem description, mobile number, and time available for SMS receipt, confirm that everything is correct, and click **Submit Ticket**.
