
## 1\. 简介

本模块仅在您使用云数据库 Tendis（简称“**功能**”）时适用。本模块包含在位于 https://intl.cloud.tencent.com/document/product/301/17345 的隐私政策（简称“**隐私政策**”）中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致的范围，本模块应适用。

## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. 可用性

本功能可供全球范围内的用户使用，但为确保最佳性能，主要供与所选服务区域相同的国家/地区内的用户使用。

## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                             |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **慢 SQL 日志数据：** 根据慢 SQL 规则执行的慢 SQL 语句的详细信息（包括 IP、命令、节点、详情、获取、时间戳、实例 ID、代理地址）。 | 我们使用此信息进行故障排除和性能分析。 请注意，这些数据将在腾讯云时序数据库功能中存储和备份。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |
| **数据库管理帐户密码**                     | 我们使用这些信息来方便您使用您的数据库管理帐户。 请注意，这些数据将在腾讯云数据库 MySQL 功能中存储和备份。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同以便提供本功能而言是必要的。 |

## 7\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息**             | **保留政策**                                         |
| ------------------------------------- | ------------------------------------------------------------ |
| 慢 SQL 日志数据                     | 只要您使用本功能，我们就会保留此类数据。若您终止使用本功能，我们将在 7 天内删除此数据。 |
| 数据库管理帐户密码 | 只要您使用本功能，我们就会保留此类数据。若您终止使用本功能，我们将在 7 天内删除此数据。 |
