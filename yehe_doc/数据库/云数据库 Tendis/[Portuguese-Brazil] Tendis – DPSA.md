

## 1.HISTÓRICO

Este módulo se aplica se você usa o TencentDB for Tendis (“**Recurso**”). Este Módulo está incorporado ao Contrato de Privacidade e Segurança de Dados localizado em https://intl.cloud.tencent.com/document/product/301/17347 (“**DPSA**”). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo deve ser aplicado conforme a inconsistência.

## 2.PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de configuração** (mensagem de aviso de instância, tarefas na fila e procedimentos Tendis) | Apenas  processamos esses dados com a finalidade de fornecer o Recurso a você  de acordo com a sua configuração específica.  Note que esses dados estão armazenados e tem uma cópia de segurança em nosso recurso TencentDB  for MySQL. |
| **Dados de negócios**  (se você armazenar dados pessoais no banco de dados) | Apenas processamos esses dados  com a finalidade de lhe fornecer o Recurso.  Observe que não temos  acesso aos dados pessoais armazenados no banco de dados (exceto se você  compartilhar seus detalhes da conta conosco para a solução de problemas), caso existam, nem controle sobre  os dados.  Observe que esses dados são armazenados em nosso recurso de armazenamento em blocos de nuvem  e apoiados em nosso recurso de armazenamento de objetos na nuvem através  dos recursos de gerenciamento de backup próprios do recurso. |

## 5.RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações  pessoais** | **Política de retenção**                                         |
| ------------------------- | ------------------------------------------------------------ |
| Dados de configuração        | Nós retemos esses dados enquanto você usar  o Recurso. Quando seu uso do Recurso é encerrado, nós vamos excluir esses  dados após 7 dias. |
| Dados de negócios             | Nós retemos esses dados enquanto você usar  o Recurso. Você pode excluir manualmente esses dados no Recurso. Uma vez  que você optar por excluir esses dados, ele será excluído imediatamente do Recurso.  Quando seu uso do Recurso é encerrado, nós vamos excluir esses dados após 7  dias. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6.CONDIÇÕES ESPECIAIS

Você deve garantir que este recurso seja usado somente por usuários finais com a idade exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.
