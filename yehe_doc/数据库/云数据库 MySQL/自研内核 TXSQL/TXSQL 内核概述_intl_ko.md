TXSQL은 Tencent CDB 팀에서 관리되는 MySQL 커널 분지이고 네이티브 MySQL 버전을 100% 호환됩니다. TXSQL은 엔터프라이즈 수준의 TDE, 감사, 스레드 풀(Thread Pool), 암호화 함수, 백업 복구 등 MySQL 엔터프라이즈 버전과 유사한 여러 기능을 제공합니다.

TXSQL은 InnoDB 스토리지 엔진, 검색 최적화, 복제 성능 등의 방면에서 대량의 최적화를 진행하였습니다. 또한 TencentDB for MySQL의 용이성과 유지보수성을 향상시켰습니다. 사용자에게 MySQL의 모든 기능을 제공하는 동시에 엔터프라이즈 수준의 재해 복구, 복구, 모니터링, 성능 최적화, 읽기/쓰기 분리, TDE, 감사 등의 고급 기능도 제공합니다.	 

TXSQL 커널에 대한 더 많은 정보:
- TencentDB for MySQL 커널 버전 업데이트 동향에 대해 [커널 버전 업데이트 동향](https://intl.cloud.tencent.com/document/product/236/35989)를 참조 바랍니다.
- TencentDB for MySQ은 인스턴트 알고리즘을 통해 데이터 복사를 피할 수 있도록 지원하므로 큰 테이블의 빠른 열 추가 기능을 실현합니다. 자세한 내용은 [빠른 열 추가 기능](https://intl.cloud.tencent.com/document/product/236/35990)을 참조하십시오. 
- TencentDB for MySQL은 자동 또는 수동으로 커널 마이너 버전 업그레이드를 지원합니다. 자세한 내용은 [커널 마이너 버전 업그레이드](https://intl.cloud.tencent.com/document/product/236/36816)를 참조하십시오.
- TencentDB for MySQL은 CVM를 통해 MySQL 인스턴스에 로그인하고 커널 마이너 버전 조회를 지원합니다. 자세한 내용은 [커널 마이너 버전 조회](https://intl.cloud.tencent.com/document/product/236/35995)를 참조하십시오.
