Saat Anda menghentikan instans TencentDB for MySQL, semua data (termasuk data cadangan) yang tersimpan di dalamnya akan dimusnahkan. Tencent Cloud tidak akan menyimpan data atau memulihkan instans Anda secara aktif.

Untuk informasi selengkapnya, lihat [Menghentikan Instans](https://intl.cloud.tencent.com/document/product/236/31895).
