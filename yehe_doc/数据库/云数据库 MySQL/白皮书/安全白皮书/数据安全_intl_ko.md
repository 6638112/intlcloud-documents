
## 민감한 데이터 감지
민감한 데이터 감지 기능은 식별 규칙을 통해 인스턴스 민감한 데이터를 자동으로 감지하고 감지된 민감한 데이터에 대해 자동화 분류 및 등급 보호를 구현할 수 있습니다. 최대 20개의 내장된 민감한 데이터 식별 규칙을 지원하며, 컴플라이언스 충족을 기반으로 데이터 세트의 모든 필드에서 중요 속성을 식별하여 대용량 텍스트에 숨겨진 중요 정보가 적절하게 처리될 수 있도록 보장합니다. 동시에 이 제품은 사용자에 맞춤 설정된 데이터 보호 요구 사항을 충족하기 위해 사용자 정의 민감한 데이터 유형의 자동 감지도 지원합니다.

## 데이터 마스킹
데이터 마스킹 기능에는 여러 고급 마스킹 알고리즘이 내장되어 있어 마스킹 작업을 스마트하게 실행 및 관리하고 다양한 비즈니스 시나리오에 대한 데이터 마스킹을 실현하고 엔터프라이즈 핵심 데이터를 기밀로 유지하는 효과를 얻을 수 있습니다. 데이터 마스킹은 보안성을 보장하지만 처리된 데이터는 여전히 원시 데이터의 분포 특성과 데이터 형식을 유지하므로 통계 분석, 테스트, R&D, 교육 등 용도에는 영향을 미치지 않습니다.
- 컴플라이언스 측면에서 데이터 마스킹은 익명화 기술을 통해 마스킹 후 데이터를 복원할 수 없도록 할 수 있습니다. 기업 고객의 컴플라이언스 요구 사항을 충족하기 위해 관련 법률 및 규정을 엄격하게 준수하여 기업 개인 정보를 사용합니다.
- 시스템 성능 측면에서 데이터 마스킹은 작업 시작 시 사용자의 최신 실시간 백업 데이터를 풀링하므로 원본 데이터베이스를 중지할 필요가 없으며 데이터베이스 성능에 추가적인 영향 및 비용을 더 주지 않습니다. 마스킹 작업은 사용자가 감지하지 못하는 상황에서 온라인 상에서 동적으로 실시간 완료될 수 있습니다.
