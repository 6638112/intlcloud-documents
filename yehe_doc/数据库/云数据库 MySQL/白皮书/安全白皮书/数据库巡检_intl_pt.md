
A [inspeção de banco de dados](https://intl.cloud.tencent.com/document/product/1035/37178) é usada para realizar verificações de integridade automática e regularmente em todas as instâncias. Você também pode configurar inspeções personalizadas com base em suas próprias necessidades, para ajudar a solucionar possíveis problemas de instância e fornecer soluções. Um relatório de inspeção de banco de dados contém as seguintes seções: visão geral, informações básicas, integridade, status da instância, diagnóstico de exceção, análise de SQL lento, contas de alto risco, análise de tabelas grandes e curvas de desempenho.
