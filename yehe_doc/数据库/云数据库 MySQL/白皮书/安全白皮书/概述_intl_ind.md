
TencentDB for MySQL memungkinkan Anda untuk dengan mudah men-deploy dan menggunakan instans database MySQL yang dapat diskalakan, database relasional sumber terbuka paling populer di dunia, di cloud dalam hitungan menit. TencentDB for MySQL memiliki efisiensi biaya tinggi dan skalabilitas perangkat keras elastis tanpa waktu henti. Sebagai solusi database lengkap dengan beragam fitur seperti pencadangan, pengembalian, pemantauan, penskalaan cepat, dan transfer data, TencentDB for MySQL menyederhanakan OPS TI Anda dan memungkinkan Anda untuk lebih berfokus pada pengembangan bisnis.

TencentDB for MySQL menyediakan beragam fitur penguatan keamanan untuk memastikan keandalan dan keamanan data Anda. Untuk lebih mengamankan instans TencentDB for MySQL Anda, disarankan untuk menggunakan fitur keamanan berikut berdasarkan kebutuhan bisnis Anda:
- Jaringan: [grup keamanan](https://intl.cloud.tencent.com/document/product/236/14470) dan [VPC](https://intl.cloud.tencent.com/document/product/215/535)
- Penyimpanan: [enkripsi data](https://intl.cloud.tencent.com/document/product/236/38491) dan [pencadangan otomatis](https://intl.cloud.tencent.com/document/product/236/37796)
- Izin: [kontrol akses](https://intl.cloud.tencent.com/document/product/236/35296)
