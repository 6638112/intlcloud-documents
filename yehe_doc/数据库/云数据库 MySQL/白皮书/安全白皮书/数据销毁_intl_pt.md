Ao encerrar sua instância do TencentDB for MySQL, todos os dados (incluindo os dados de backup) armazenados nela serão destruídos. A Tencent Cloud não reterá os dados nem recuperará ativamente sua instância.

Para mais informações, consulte [Encerramento de instâncias](https://intl.cloud.tencent.com/document/product/236/31895).
