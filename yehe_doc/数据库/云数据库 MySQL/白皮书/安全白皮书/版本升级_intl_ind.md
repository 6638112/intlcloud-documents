TencentDB for MySQL akan memberikan layanan database versi terbaru. Ketika bug parah atau risiko keamanan terjadi di sistem, instans TencentDB for MySQL Anda akan ditingkatkan selama termin waktu pemeliharaan, dan pemberitahuan pemutakhiran akan dikirimkan kepada Anda sebelumnya. Proses peningkatan versi dapat menyebabkan waktu pemutusan koneksi pada tingkat kedua; oleh karena itu, pastikan bisnis Anda memiliki mekanisme koneksi ulang.
