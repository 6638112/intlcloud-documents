
O TencentDB for MySQL permite que você implante e use facilmente instâncias escaláveis do banco de dados MySQL na nuvem em questão de minutos. O MySQL é o banco de dados relacional de código aberto mais conhecido do mundo. Ele possui alta eficiência de custo e escalabilidade de hardware elástica sem tempo de inatividade. Sendo uma solução de banco de dados completa com várias funcionalidades, como backup, reversão, monitoramento, dimensionamento rápido e transferência de dados, ele simplifica suas operações de TI e permite que você se concentre mais no desenvolvimento de negócios.

O TencentDB for MySQL fornece funcionalidades diversificadas de reforço de segurança para garantir a confiabilidade e a segurança de seus dados. Para tornar suas instâncias do TencentDB for MySQL mais seguras, recomendamos usar as seguintes funcionalidades de segurança com base em suas necessidades de operações:
- Rede: [grupo de segurança](https://intl.cloud.tencent.com/document/product/236/14470) e [VPC](https://intl.cloud.tencent.com/document/product/215/535)
- Armazenamento: [criptografia de dados](https://intl.cloud.tencent.com/document/product/236/38491) e [backup automático](https://intl.cloud.tencent.com/document/product/236/37796)
- Permissão: [controle de acesso](https://intl.cloud.tencent.com/document/product/236/35296)
