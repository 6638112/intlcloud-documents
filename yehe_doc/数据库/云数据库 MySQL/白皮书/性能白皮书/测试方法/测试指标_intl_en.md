This document describes the metrics of TencentDB for MySQL performance test.

## Test Metrics
- **Transactions per second (TPS)**
The number of successfully committed transactions executed by the database per second.
- **Queries per second (QPS)**
The number of SQL statements executed by the database per second, including `INSERT`, `SELECT`, `UPDATE`, `DETELE`, and `COMMIT`.
