
## Método de faturamento para instâncias com upgrade
Após o upgrade, as instâncias com pagamento conforme o uso serão faturadas com base nas suas novas especificações a partir do próximo ciclo de faturamento.

## Método de faturamento para instâncias com downgrade

#### Pagamento conforme o uso
Se você optar por fazer o downgrade de uma instância de banco de dados com pagamento conforme o uso, ela será faturada no preço de pagamento conforme o uso da 1ª camada da nova configuração.

## Documentação
O TencentDB for MySQL permite o ajuste rápido da especificação da instância e operações de dimensionamento flexíveis. Para mais informações, consulte [Ajuste da especificação da instância de banco de dados](https://intl.cloud.tencent.com/zh/document/product/236/19707).
