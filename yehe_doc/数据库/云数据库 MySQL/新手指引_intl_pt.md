## 3. Introdução
#### 3.1 Aquisição de uma instância do TencentDB for MySQL
Antes de usar o TencentDB for MySQL, você precisa criar uma conta da Tencent Cloud e adquirir uma instância do TencentDB for MySQL. Para mais informações, consulte [Criação de instância do MySQL](https://intl.cloud.tencent.com/document/product/236/37785).

#### 3.2 Inicialização da instância do TencentDB for MySQL
Após a aquisição de uma instância do TencentDB for MySQL, será necessário inicializá-la. Para mais informações, consulte [Inicialização de instância do MySQL](https://intl.cloud.tencent.com/document/product/236/3128).

#### 3.3 Conexão à instância do TencentDB for MySQL
Após a inicialização, você pode acessar a instância do TencentDB for MySQL pelas redes privada e pública a partir de uma instância da CVM do Windows ou do Linux. Para mais informações, consulte [Conexão à instância do MySQL](https://intl.cloud.tencent.com/document/product/236/37788).
