## 2. Mode Penagihan TencentDB for MySQL
TencentDB for MySQL mengadopsi mode penagihan **pay-as-you-go** (bayar sesuai pemakaian). Harap lihat [Harga Produk](https://buy.cloud.tencent.com/price/cdb/overview) untuk detailnya.

-----

## 3. Memulai
#### 3.1 Membeli instans TencentDB for MySQL
Sebelum menggunakan TencentDB for MySQL, Anda harus mendaftar akun Tencent Cloud dan membeli instans TencentDB for MySQL. Untuk informasi lebih lanjut, harap lihat [Membuat Instans MySQL](https://intl.cloud.tencent.com/document/product/236/37785).

#### 3.2 Menginisialisasi instans TencentDB for MySQL
Setelah instans TencentDB for MySQL dibeli, Anda perlu menginisialisasinya. Untuk informasi selengkapnya, harap lihat [Menginisialisasi Instans MySQL](https://intl.cloud.tencent.com/document/product/236/3128).

#### 3.3 Menghubungkan ke instans TencentDB for MySQL
Setelah inisialisasi, Anda dapat mengakses instans TencentDB for MySQL melalui jaringan pribadi dan publik dari instans CVM Windows atau Linux. Untuk informasi selengkapnya, harap lihat [Menghubungkan ke Instans MySQL](https://intl.cloud.tencent.com/document/product/236/37788).
