## 2. TencentDB for MySQL Billing Modes
TencentDB for MySQL supports **pay-as-you-go** and **monthly subscription** billing modes. For more information, see [Billing Overview](https://intl.cloud.tencent.com/document/product/236/18335).

-----

## 3. Getting Started
#### 3.1 Purchasing a TencentDB for MySQL instance
Before using TencentDB for MySQL, you need to sign up for a Tencent Cloud account and purchase a TencentDB for MySQL instance. For more information, see [Creating MySQL Instance](https://intl.cloud.tencent.com/document/product/236/37785).

#### 3.2 Connecting to the TencentDB for MySQL instance
After purchasing a TencentDB for MySQL instance, you can access it over both private and public networks from a Windows or Linux CVM instance. For more information, see [Connecting to MySQL Instance](https://intl.cloud.tencent.com/document/product/236/37788).
