본문은 세션 수준 연결 풀 기능을 설정하는 방법을 소개합니다.

## 배경 및 조건
- [데이터베이스 프록시 버전 1.1.2 이상](https://intl.cloud.tencent.com/document/product/236/45627).
- 프라이머리 인스턴스의 데이터베이스 버전은 MySQL 5.7이며, 데이터베이스 커널의 마이너 버전은 20211031 이상입니다. 이를 충족하기 위해 먼저 [데이터베이스 커널의 마이너 버전 업그레이드](https://intl.cloud.tencent.com/document/product/236/36816)할 수 있습니다.

## 활성화되지 않은 데이터베이스 프록시
1. [MySQL 콘솔](https://console.cloud.tencent.com/cdb)에 로그인합니다.
2. 위에서 리전을 선택하고 인스턴스 ID 또는 **작업** 열에서 **관리**를 클릭하여 인스턴스 관리 페이지로 이동합니다.
3. 인스턴스 관리 페이지의 **데이터베이스 프록시** 페이지에서 **지금 활성화**를 클릭합니다.
4. 팝업 창에서 연결 풀 기능을 활성화할 수 있습니다.
![](https://qcloudimg.tencent-cloud.cn/raw/ae2436546dfc7c8f0f8b6c6d6f154482.png)

## 활성화된 데이터베이스 프록시
1. [MySQL 콘솔](https://console.cloud.tencent.com/cdb)에 로그인합니다.
2. 위에서 리전을 선택하고 인스턴스 ID 또는 **작업** 열에서 **관리**를 클릭하여 인스턴스 관리 페이지로 이동합니다.
3. 인스턴스 관리 페이지의 **데이터베이스 프록시** 페이지에서 개요 아래의 연결 풀 **활성화**를 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/9524cadfc5fe0d67a8a050f44ea5fada.png)
4. 팝업 창에서 관련 설정을 선택하고 **확인**을 클릭합니다.
   - **연결 풀 상태**: 연결 풀을 활성화하도록 설정합니다.
   - **연결 풀 유형**: 세션 수준 연결 풀을 지원합니다.
   - **연결 보관 임계값**: 유휴 연결이 프록시의 연결 풀에 보관되는 시간 임계값을 설정합니다. 범위는 1초 - 300초입니다.
>!연결 풀 설정 변경 완료 시 몇 초간 연결이 끊길 수 있으니 서비스 재연결 메커니즘을 확보하시기 바랍니다.
>
![](https://qcloudimg.tencent-cloud.cn/raw/f354aa701758d2823c67468392e37617.png)
5. 연결 풀이 성공적으로 활성화되면 데이터베이스 프록시 페이지의 **개요** > **기본 정보**에서 세부 사항을 확인하고 설정을 편집할 수 있습니다.

## 연결 풀 기능 비활성화
1. [MySQL 콘솔](https://console.cloud.tencent.com/cdb)에 로그인합니다.
2. 위에서 리전을 선택하고 인스턴스 ID 또는 **작업** 열에서 **관리**를 클릭하여 인스턴스 관리 페이지로 이동합니다.
3. 인스턴스 관리 페이지의 **데이터베이스 프록시** 페이지에서 개요 아래의 연결 풀 **세부 사항 / 편집**을 클릭합니다.
4. 팝업 창에서 관련 설정을 선택하고 **확인**을 클릭합니다.
