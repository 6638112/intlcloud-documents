
본문에서는 TencentDB for MySQL 콘솔을 통한 데이터베이스 프록시 노드 모니터링 방법에 대해 소개합니다.

##  전제 조건
- [데이터베이스 프록시 활성화]가 완료 되어 있어야 합니다.  (https://intl.cloud.tencent.com/document/product/236/42052)

- 작업 순서
1. [MySQL 콘솔](https://console.cloud.tencent.com/cdb)에 로그인한 뒤, 인스턴스 리스트에서 활성화된 프록시의 마스터 인스턴스를 선택하고 인스턴스 ID 혹은 '작업'열의 [관리]를 클릭하여 인스턴스 관리 페이지에 접속합니다.
2. 인스턴스 관리 페이지에서 [데이터베이스 프록시]를 선택하여 [성능 모니터링] 페이지로 접속한 후, 노드 명칭을 클릭하면 각 프록시 노드 모니터링을 확인할 수 있습니다. 
>? 데이터 분할 정도가 5초인 모니터링, 시간 간격이 4시간 이내인 모니터링은 데이터 분할 정도가 5초로 기본 전환 되어있습니다.
