This document describes how to disable database proxy in the TencentDB for MySQL console.
>?Before disabling the database proxy, make sure that your client does not access the database through the proxy address, as the generated data will be deleted and cannot be recovered after the database proxy is disabled.

## Prerequisites
You have enabled the database proxy. For more information, see [Enabling Database Proxy](https://intl.cloud.tencent.com/document/product/236/42052).

## Directions
1. Log in to the [TencentDB for MySQL console](https://console.cloud.tencent.com/cdb). In the instance list, select the desired instance, click an instance ID or **Manage** in the **Operation** column to enter the instance management page.
2, On the instance management page, select **Database Proxy** > **Overview** > **Basic Info** > **Disable Database Proxy**.
![](https://main.qcloudimg.com/raw/e5f0cab009c4d0376a4a294b67be4ba0.png)
3. In the pop-up dialog box, confirm that everything is correct and click **OK**.
