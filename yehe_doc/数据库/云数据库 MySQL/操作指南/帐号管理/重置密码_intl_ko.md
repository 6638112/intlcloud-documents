## 작업 시나리오
TencentDB for MySQL을 사용하는 동안 데이터베이스 계정 비밀번호를 잊어버렸거나 변경해야 하는 경우 콘솔에서 재설정할 수 있습니다.
>?
>- TencentDB for MySQL의 경우 비밀번호 재설정 기능이 [CAM](https://intl.cloud.tencent.com/document/product/236/14469)에 연결되었습니다. 따라서 적절한 담당자에게만 권한을 부여하여 TencentDB for MySQL의 비밀번호 재설정 API 또는 민감한 리소스에 대한 권한을 더 엄격하게 제어하는 것이 좋습니다.
>- 데이터 보안을 위해 최소 3개월에 한 번 이상 정기적으로 비밀번호를 재설정하는 것이 좋습니다.


## 작업 단계
1. [TencentDB for MySQL 콘솔](https://console.cloud.tencent.com/cdb/) 로그인 후, 인스턴스 리스트에서 인스턴스 ID 또는 **작업**열의 **관리**를 클릭하여 인스턴스 관리 페이지로 이동합니다.
2. **데이터베이스 관리** > **계정 관리**를 선택하고 비밀번호를 재설정할 계정을 찾은 다음 **비밀번호 재설정**을 클릭하거나 또는 **더 보기** > **비밀번호 재설정**을 선택합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/b162dc614a92c25db597c45581c8fd50.png)
3. 팝업 창에서 새 비밀번호를 입력하고 확인한 다음 **확인**을 클릭합니다.
>?데이터베이스 비밀번호는 영어, 숫자 및 부호 \_+-&=!@#$%^\*() 중 두 가지 이상의 문자 유형으로 8 – 64자를 포함해야 합니다.
>
