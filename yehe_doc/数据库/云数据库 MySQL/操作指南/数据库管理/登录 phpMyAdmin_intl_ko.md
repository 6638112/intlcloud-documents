
## 작업 시나리오
본 문서는 TencentDB for MySQL 콘솔에서 phpMyAdmin(PMA) 콘솔에 접속하는 방법을 소개합니다.

## 작업 순서

1. [MySQL 콘솔](https://console.cloud.tencent.com/cdb)에 로그인한 뒤 Instance List에서 '작동 중' 상태의 인스턴스를 선택한 다음 '작업' 열에서 [로그인]을 클릭합니다.
![](https://main.qcloudimg.com/raw/980f72998460c77e2ea70741e7e4fb58.png)
2. 데이터 관리 콘솔의 로그인 페이지에서 계정은 root로 입력하고, 비밀번호는 초기화 시에 설정한 root 계정의 비밀번호를 입력한 다음 [로그인]을 클릭합니다.
>?비밀번호를 잊어버렸을 경우 [비밀번호 재설정](https://intl.cloud.tencent.com/document/product/236/31901)을 통해 변경할 수 있습니다.
>
![](https://main.qcloudimg.com/raw/e2410310dc33c3b78b1e76deb3e687bf.png)
3. 데이터 관리(구버전) 페이지에서 Instance Status, Basic Info 및 Table 등을 조회할 수 있으며 [Go to PMA]를 클릭합니다.
![](https://main.qcloudimg.com/raw/81e2c2fb6042bef1b2d7b3a662203914.png)
4. phpMyAdmin을 통해 MySQL 데이터베이스에 연결한 후, 해당 페이지에서 MySQL 데이터베이스의 다양한 모드와 객체를 확인할 수 있으며 테이블 생성, 데이터 삽입 및 조회 등의 작업을 수행할 수 있습니다.
![](https://main.qcloudimg.com/raw/ed0dbceaaa7ec9e86f9e16286f16d218.png)
