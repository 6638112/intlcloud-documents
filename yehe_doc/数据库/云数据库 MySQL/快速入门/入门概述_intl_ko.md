
본 문서는 사용자의 빠른 이해를 돕기 위해 TencentDB for MySQL의 전체적인 사용 방법에 대해 소개합니다. 인스턴스의 생성부터 기본적인 사용 방법을 알기 위해서는 아래의 작업을 완료해야 합니다.

### 1. MySQL 인스턴스 생성
TencentDB for MySQL 콘솔을 통해 여러 과금 방식 및 다양한 구성의 MySQL 인스턴스를 효율적으로 생성할 수 있습니다. 자세한 방법은 [MySQL 인스턴스 생성](https://intl.cloud.tencent.com/document/product/236/37785)을 참고하십시오.

### 2. MySQL 인스턴스 연결
MySQL 인스턴스를 생성한 후, 다양한 방법을 통해 MySQL 인스턴스에 연결할 수 있으며, 연결한 다음에는 데이터베이스에 대한 갖가지 관리 작업을 수행할 수 있습니다. 자세한 내용은 [MySQL 인스턴스 연결](https://intl.cloud.tencent.com/document/product/236/37788)을 참고 바랍니다.

MySQL 인스턴스 연결 과정에서 인스턴스 연결 불가 관련 문제가 발생하는 경우 [인스턴스 연결 불가 문제 해결](https://intl.cloud.tencent.com/document/product/236/40333)을 참고하여 처리하십시오.
