
Dokumen ini menjelaskan cara memulai TencentDB for MySQL dari pembuatan instans hingga penggunaan dasar. Untuk menggunakan instans, Anda harus menyelesaikan operasi berikut.

### 1. Buat instans TencentDB for MySQL
Di konsol TencentDB for MySQL, Anda dapat membuat instans MySQL tempat mode penagihan dan konfigurasi dapat disesuaikan. Untuk informasi selengkapnya, lihat [Membuat Instans MySQL](https://intl.cloud.tencent.com/document/product/236/37785).

### 2. Hubungkan ke instans TencentDB for MySQL
Setelah membuat instans TencentDB for MySQL, Anda dapat menghubungkannya dengan banyak cara dan kemudian melakukan berbagai operasi manajemen database. Untuk informasi selengkapnya, lihat [Menghubungkan ke Instans MySQL](https://intl.cloud.tencent.com/document/product/236/37788).

Jika Anda mengalami kesalahan koneksi, lihat [Kegagalan Koneksi Instans](https://intl.cloud.tencent.com/document/product/236/40333).
