
Este documento descreve como começar a usar o TencentDB for MySQL desde a criação da instância até o uso básico. Para usar uma instância, você precisa concluir as seguintes operações.

### 1. Criar uma instância do TencentDB for MySQL
No console do TencentDB for MySQL, você pode criar uma instância do MySQL em que o modo de faturamento e as configurações sejam personalizáveis. Para mais informações, consulte [Criação de instância do MySQL](https://intl.cloud.tencent.com/document/product/236/37785).

### 2. Conectar à instância do TencentDB for MySQL
Após criar a instância do TencentDB for MySQL, você pode se conectar a ela de várias maneiras e, em seguida, realizar várias operações de gerenciamento de banco de dados. Para mais informações, consulte [Conexão à instância do MySQL](https://intl.cloud.tencent.com/document/product/236/37788).

Se você encontrar erros de conexão, consulte [Falha de conexão da instância](https://intl.cloud.tencent.com/document/product/236/40333).
