
## Visão geral
O TencentDB for MySQL é um serviço de armazenamento de dados distribuído de alto desempenho desenvolvido pela Tencent Cloud com base no banco de dados open-source MySQL. Ele permite que você configure, manipule e dimensione bancos de dados relacionais na nuvem com mais facilidade.
O TencentDB for MySQL contempla as seguintes funcionalidades:
- Fornece serviço de armazenamento de dados em nuvem para aplicações de internet.
- É totalmente compatível com os protocolos do MySQL e adequado para cenários de aplicações orientadas a tabelas. O TencentDB for MySQL pode ser utilizado onde quer que o MySQL seja usado.
- Oferece serviço de cluster do MySQL que apresenta alto desempenho, confiabilidade de dados de até 99,9996%, facilidade de uso e conveniência.
- Integra várias funcionalidades de banco de dados, como backup, dimensionamento e migração. Além disso, está equipado com a ferramenta de banco de dados de nova geração DMC para gerenciamento ainda mais conveniente de bancos de dados.



## Conceitos
[Instância](https://intl.cloud.tencent.com/document/product/236/7268): um recurso do TencentDB for MySQL na Tencent Cloud.

[Tipo de instância](https://intl.cloud.tencent.com/document/product/236/7268): uma combinação de contagem de nós, capacidade de leitura/gravação e região de implantação de uma instância do TencentDB for MySQL.

[Instância somente leitura](https://intl.cloud.tencent.com/document/product/236/7270): uma instância do TencentDB for MySQL que só pode ser lida.

[Grupo do RO](https://intl.cloud.tencent.com/document/product/236/11361): uma ferramenta lógica usada para gerenciar uma ou mais instâncias somente leitura. Ele pode atender aos requisitos de balanceamento de carga em cenários de separação de leitura/gravação e aprimorar significativamente a capacidade de carga de leitura de um banco de dados.

[Instância de recuperação de desastres](https://intl.cloud.tencent.com/document/product/236/7272): uma instância do TencentDB for MySQL que permite a recuperação de desastres entre AZs e regiões diferentes.

[VPC](https://intl.cloud.tencent.com/document/product/215/535): um espaço de rede virtual personalizado que é "logicamente" isolado de outros recursos.

[Grupo de segurança](https://intl.cloud.tencent.com/document/product/236/14470): controle de acesso de segurança para instâncias do TencentDB for MySQL especificando as regras de IP, protocolo e porta para acesso à instância.

[Região e AZ](https://intl.cloud.tencent.com/document/product/236/8458): localização física de uma instância do TencentDB for MySQL e outros recursos.

[Console da Tencent Cloud](https://console.cloud.tencent.com/cdb): UIs baseadas na web.
