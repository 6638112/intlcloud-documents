## Jogos
Normalmente, os cenários de jogos requerem dimensionamento elástico e reversão rápida.
- O TencentDB for MySQL apresenta dimensionamento elástico de recursos de computação, permitindo aumentar a produtividade e implantar servidores de jogos em várias regiões em questão de minutos.
- Ele fornece a reversão e operação em lote, que permite restaurar para qualquer momento, facilitando a reversão de jogos.

## Aplicativos Web/móveis
O TencentDB for MySQL pode servir como meio de armazenamento de dados do servidor em aplicativos Web e móveis. Para cenários em que há mais leituras do que gravações, é possível criar instâncias somente leitura para bancos de dados que são lidos com frequência, o que melhora muito o desempenho de leitura.
