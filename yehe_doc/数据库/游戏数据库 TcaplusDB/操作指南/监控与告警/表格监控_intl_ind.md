Untuk membantu Anda melihat dan terus mengikuti info terbaru terkait informasi TcaplusDB yang dijalankan, TcaplusDB menyediakan berbagai metrik pemantauan performa. Ini mendukung pemantauan tabel dan menawarkan tampilan pemantauan independen untuk setiap tabel untuk memudahkan kueri.
Anda dapat masuk ke [Konsol TcaplusDB](https://console.cloud.tencent.com/tcaplusdb/table), klik ID tabel untuk masuk ke halaman manajemen tabel, dan masuk ke tab **Table Monitoring** (Pemantauan Tabel) untuk melihat tampilan pemantauan.

>
>- Anda juga bisa mendapatkan metrik pemantauan instans dengan memanggil API data pemantauan TcaplusDB di Cloud Monitor.
>- Saat ini, Anda dapat melihat data pemantauan TcaplusDB selama 60 hari terakhir.

## Metrik Pemantauan Tabel
Tencent Cloud Monitor menyediakan metrik pemantauan berikut untuk tabel TcaplusDB:

| Nama Metrik | Parameter | Deskripsi | Unit |
| ----------------- | ---------------- | ------------------------------------------------------------ | -------- |
| Tingkat kesalahan rata-rata | Rata-rata Tingkat Kesalahan | Persentase rata-rata kesalahan operasi tabel | % |
| Tingkat kesalahan umum | Tingkat Kesalahan Umum | Persentase kesalahan operasi tabel umum | % |
| Unit kapasitas pembacaan aktual | Unit Kapasitas Pembacaan Aktual | Jumlah unit kapasitas pembacaan aktual dari tabel | Unit/dtk |
| Latensi baca rata-rata | Latensi Pembacaan Rata-Rata | Latensi rata-rata dalam pembacaan data | kami |
| Tingkat kesalahan sistem | Tingkat Kesalahan Sistem | Persentase kesalahan sistem | % |
| Kapasitas penyimpanan | Kapasitas Penyimpanan | Kapasitas penyimpanan yang digunakan oleh tabel | KB |
| Latensi penulisan rata-rata | Latensi Penulisan Rata-Rata | Latensi rata-rata dalam penulisan data | kami |
| Unit kapasitas penulisan aktual | Unit Kapasitas Penulisan Aktual | Jumlah unit kapasitas tulis aktual dari tabel | Unit/dtk |
