TcaplusDB dapat diakses dan dibaca melalui berbagai cara seperti alat klien, toolkit SDK dalam berbagai bahasa pemrograman, dan RESTful API.

## Mengakses TcaplusDB Melalui Alat Klien
Anda dapat menggunakan [tcaplus_client](https://intl.cloud.tencent.com/document/product/1016/37719) untuk Linux untuk mengakses TcaplusDB.

## Mengakses TcaplusDB Melalui SDK untuk C++ API
Anda dapat menggunakan SDK untuk [C++](https://intl.cloud.tencent.com/document/product/1016/30289) untuk mengakses TcaplusDB.

## Mengakses TcaplusDB Melalui RESTful API
Anda dapat mengakses TcaplusDB melalui RESTful API. Untuk informasi selengkapnya, silakan lihat [Deskripsi](https://intl.cloud.tencent.com/document/product/1016/37721).

- Anda dapat mengakses TcaplusDB melalui RESTful API untuk Python seperti yang diinstruksikan [di sini](https://intl.cloud.tencent.com/document/product/1016/37725).

- Anda dapat mengakses TcaplusDB melalui RESTful API untuk PHP seperti yang diinstruksikan [di sini](https://intl.cloud.tencent.com/document/product/1016/37724).

- Anda dapat mengakses TcaplusDB melalui RESTful API untuk Go seperti yang diinstruksikan [di sini](https://intl.cloud.tencent.com/document/product/1016/37722).

- Anda dapat mengakses TcaplusDB melalui RESTful API untuk Java seperti yang diinstruksikan [di sini](https://intl.cloud.tencent.com/document/product/1016/37723).
