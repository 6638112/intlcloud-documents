## Ikhtisar
Perintah ini digunakan untuk mengkueri jumlah total catatan dalam sebuah tabel.

## Sintaksis
```
jumlah [tableName];
```

## Kesalahan
Untuk informasi selengkapnya, silakan lihat [Kode Kesalahan](https://intl.cloud.tencent.com/document/product/1016/38791).

## Sampel
Mengkueri jumlah total catatan dalam tabel:
```
tcaplus> count game_player;
--------------------------------------------------------------------------------
| TableName                                 Count                              |
--------------------------------------------------------------------------------
| table_list                                16                                 |
--------------------------------------------------------------------------------
```
