## Visão geral
Este comando é usado para consultar a quantidade total de registros em uma tabela.

## Sintaxe
```
count [tableName];
```

## Erros
Para obter mais informações, consulte [Códigos de erro](https://intl.cloud.tencent.com/document/product/1016/38791).

## Exemplo
Consulte a quantidade total de registros em uma tabela:
```
tcaplus> count game_player;
--------------------------------------------------------------------------------
| TableName                                 Count                              |
--------------------------------------------------------------------------------
| table_list                                16                                 |
--------------------------------------------------------------------------------
```
