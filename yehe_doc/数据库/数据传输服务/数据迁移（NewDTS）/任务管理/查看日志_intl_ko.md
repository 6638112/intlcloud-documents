
## 작업 시나리오
마이그레이션 작업 중에 마이그레이션 로그를 보고 작업 진행 상황을 알 수 있습니다.

## 작업 단계
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/migration)에 로그인하면 다음 두 가지 방법으로 작업을 조회할 수 있습니다.
 - 방법1: 데이터 마이그레이션 페이지에서 대상 마이그레이션 작업을 선택하고 작업 이름을 클릭합니다.
 - 방법2: 데이터 마이그레이션 페이지에서 대상 마이그레이션 작업을 선택하고 **작업** 열에서 **더보기** > **조회**를 클릭합니다.
2. 마이그레이션 로그를 조회합니다.
   탭을 전환하여 작업 로그를 조회합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/90fba5ec01e9c9312a270218f4f48d66.png)

## 작업 상태 설명

| **상태**   | **설명**                                                     |
| ---------- | ------------------------------------------------------------ |
| 설정됨     | 마이그레이션 작업 설정 완료.                                         |
| 인증 중     | 마이그레이션 작업 확인 중.                                         |
| 인증 통과   | 마이그레이션 작업 확인 완료.                                           |
| 인증 실패 | 마이그레이션 작업 확인 실패.                                         |
| 준비 중   | 확인 완료, 데이터 마이그레이션 시작 준비.                                 |
| 작업 실행 중   | 마이그레이션 작업 실행 중.                                           |
| 준비 완료   | 작업 실행은 기본적으로 완료되었으며 완료 상태로 들어갈 준비가 되었습니다.                         |
| 작업 성공   | 마이그레이션 작업 완료.                                           |
| 재시도 가능한 오류 발생 | 마이그레이션 중에 예외로 인한 마이그레이션 작업 중단. 콘솔에서 작업을 재시도하고 재개할 수 있습니다. |
| 작업 실패   | 마이그레이션 작업 실패.                                               |
| 종료 중     | 마이그레이션 작업 실행 중 수동 종료 중.     |
| 완료 중     | 증분 마이그레이션 중에 완료를 수동으로 클릭하여 작업 완료 상태 전환 중.         |
| 작업 오류   | 마이그레이션하는 동안 예외로 인해 작업이 중단되어 계속할 수 없습니다.     |
