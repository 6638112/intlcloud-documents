
## Overview
In data migration, you can map the databases/tables to be migrated to the target database. 

## Directions
1. Log in to the [DTS console](https://console.cloud.tencent.com/dts/migration), select **Data Migration** on the left sidebar, and create a migration task.
2. In the **Set migration options and select migration objects** step, in **Selected Object** on the right, hover over an object to be modified and click the displayed **Edit** icon.
![](https://qcloudimg.tencent-cloud.cn/raw/546d29fba4f35f6c0a282f05adc2741b.png)
3. In the pop-up window, rename the object and click **OK**.
  - Database name mapping: in the pop-up window, set the name of the database in the target database. 
![](https://qcloudimg.tencent-cloud.cn/raw/cb5e3d3d15dc89a613a10d2bcfc630e4.png)
 - Table name mapping: in the pop-up window, set the name of the table in the target database.
![](https://qcloudimg.tencent-cloud.cn/raw/de1b559c4cd611662acfff7aef3529bb.png)
