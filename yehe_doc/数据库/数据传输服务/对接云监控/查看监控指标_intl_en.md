
## Overview
You can view the metric monitoring data in real time to understand the performance metrics of the system. 

> ?Currently, in data migration and data sync tasks, real-time monitoring metric data can be obtained only during incremental migration and sync. 

## Viewing Monitoring Metric
1. Log in to the [DTS console](https://console.cloud.tencent.com/dts/migration) and select **Data Migration**, **Data Sync**, or **Data Subscription** on the left sidebar
2. Select the target task and click the task name.
![](https://qcloudimg.tencent-cloud.cn/raw/e5466252130b4d259ac6b9aeddd4726f.png)
3. Click the **Monitoring Data** tab to view the metric data.

(1) View data migration metric monitoring data.
![](https://qcloudimg.tencent-cloud.cn/raw/6f41b990058782cfc241f581f9a8a940.png)
(2) View data sync metric monitoring data.
![](https://qcloudimg.tencent-cloud.cn/raw/6639dc9892b805f229e75d3bdf82f00e.png)
(3) View data subscription metric monitoring data.
