
## 작업 시나리오
메트릭 모니터링 데이터를 실시간으로 확인하여 시스템의 성능 메트릭을 이해할 수 있습니다. 

> ?현재 데이터 마이그레이션 및 데이터 동기화 작업에서 실시간 모니터링 메트릭 데이터는 증분 마이그레이션 및 동기화 중에만 얻을 수 있습니다. 

## 모니터링 메트릭 조회
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/migration)에 로그인하고 왼쪽 사이드바에서 데이터 마이그레이션, 데이터 동기화 또는 데이터 구독을 선택합니다.
2. 대상 작업을 선택하고 작업 이름을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/e5466252130b4d259ac6b9aeddd4726f.png)
3. 측정 데이터를 보려면 **모니터링 데이터** 탭을 클릭합니다.

(1) 데이터 마이그레이션 메트릭 모니터링 데이터를 조회합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/6f41b990058782cfc241f581f9a8a940.png)
(2) 데이터 동기화 메트릭 모니터링 데이터를 조회합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/6639dc9892b805f229e75d3bdf82f00e.png)
(3) 데이터 구독 메트릭 모니터링 데이터를 조회합니다.
