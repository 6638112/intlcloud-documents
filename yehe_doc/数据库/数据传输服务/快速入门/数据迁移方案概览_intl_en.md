
The data migration feature of Data Transmission Service (DTS) supports data migration among various types of databases on Tencent Cloud, including MySQL, Redis, MongoDB, PostgreSQL, MariaDB, and TDSQL-C databases, etc. It is suitable for scenarios such as migrating data to cloud and data migration between TencentDB instances.
