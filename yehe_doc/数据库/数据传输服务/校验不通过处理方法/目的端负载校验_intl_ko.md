
## 확인 사항
- 확인 요건: DTS 마이그레이션은 대상 데이터베이스의 로드를 증가시킵니다. 마이그레이션 중 대상 데이터베이스에서 실행 중인 비즈니스가 있는 경우 확인 경고가 트리거됩니다. 작업을 차단하지는 않지만 비즈니스에 영향을 미칩니다. 경고 무시 여부를 평가 및 결정해야 합니다.
- 비즈니스에 미치는 영향: MongoDB DTS는 데이터 마이그레이션을 위해 논리적 동기화를 사용하므로 대상 데이터베이스의 CPU 로드에 특정 압력이 가해집니다. 대상 데이터베이스에서 실행 중인 비즈니스가 있는 경우 마이그레이션 작업을 신중하게 평가하고 시작해야 합니다.

## 수정 방법
대상 데이터베이스에서 실행 중인 모든 비즈니스를 중지하고 확인 작업을 다시 실행합니다.
