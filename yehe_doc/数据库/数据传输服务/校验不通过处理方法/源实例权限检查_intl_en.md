
## Check Details
Check whether you have the operation permissions of the database. 

## Fix
If you don't have the operation permissions, you need to get authorized according to the permission requirements in the corresponding scenario ([data migration](https://cloud.tencent.com/document/product/571/58688) or [data sync](https://cloud.tencent.com/document/product/571/56516)) and run the verification task again.
