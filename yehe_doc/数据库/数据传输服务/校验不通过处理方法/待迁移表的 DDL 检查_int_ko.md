## TDSQL for MySQL/MariaDB 세부 정보 확인

- 대상 데이터베이스가 TDSQL for MySQL 또는 TencentDB for MariaDB인 경우 마이그레이션할 테이블의 이름이나 열 이름에 숫자, 언더바 또는 영어 알파벳 이외의 문자가 포함되어 있으면 확인에 실패합니다.
- 참고: TDSQL for MySQL 또는 TencentDB for MariaDB 데이터베이스/테이블/열 이름에는 숫자, 언더바 또는 영어 알파벳만 포함될 수 있습니다.

## 문제 해결

프롬프트에 따라 잘못된 문자가 포함된 테이블을 제거하고 확인 작업을 다시 실행합니다.
