## 확인 사항
- 확인 요건: MongoDB 마이그레이션 작업에서 원본 데이터베이스가 샤딩된 데이터베이스인 경우 mongos, config server 및 mongod 노드 정보를 입력해야 합니다.
- 확인 설명: mongos, config server 및 mongod 노드의 정보는 무질서할 수 없습니다. 그렇지 않으면 데이터 마이그레이션도 무질서하게 됩니다. 예를 들어, mongos 노드 정보는 mongod 노드 상자에 입력하면 안 됩니다. 각 샤드에 대해 하나의 mongod 노드 정보만 입력하면 됩니다.

## 수정 방법
- DTS 작업 구성 항목에 올바른 노드 정보를 입력합니다.
- 각 샤드에 대해 하나의 mongod만 입력하십시오.
