## TDSQL for MySQL 세부 정보 확인

- 소스 데이터베이스가 TDSQL for MySQL인 경우 레벨2로 파티션된 테이블이 있는 경우 마이그레이션 작업에서 알람을 전송합니다. 알람을 무시하고 작업을 계속할 수 있습니다. 이 경우 레벨2로 파티션된 테이블은 마이그레이션을 위해 건너뜁니다. 데이터 동기화 작업에 이러한 테이블이 있는 경우 확인에 실패하므로 이러한 테이블을 제거해야 합니다. 자세한 내용은 [서브 파티션](https://cloud.tencent.com/document/product/557/58907)을 참고하십시오.
- 레벨2 서브 파티션 테이블의 기본 로직은 서브 테이블을 기반으로 구현되므로 대상 데이터베이스는 레벨2 서브 파티션 테이블을 마이그레이션하거나 동기화할 때 예외를 보고합니다.

## 문제 해결

마이그레이션하거나 동기화할 테이블 목록에서 레벨2로 파티션된 테이블을 제거합니다.
