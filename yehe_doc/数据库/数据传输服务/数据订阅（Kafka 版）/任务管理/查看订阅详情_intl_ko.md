데이터 구독 작업 생성 완료 후 세부 정보를 볼 수 있습니다.

## 전제 조건
[데이터 구독 작업 생성](https://intl.cloud.tencent.com/document/product/571/42664)을 완료해야 합니다.

## 작업 단계
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/dss)에 로그인한 뒤 왼쪽 사이드바에서 **데이터 구독**을 선택해 데이터 구독 페이지로 이동합니다.
   - 방법1: 지정된 구독 작업을 선택하고 해당 이름을 클릭합니다.
   - 방법2: 지정된 구독 작업을 선택하고 **작업** 열에서 **구독 세부 정보 보기**를 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/6ff1018df298395b849e22aa4c3b61b0.png)
2. 다른 탭 간에 전환하여 구독 세부 정보, 구독 객체, 소비 관리 및 모니터링 데이터의 특정 정보를 조회합니다.
