
## 작업 시나리오
구독 작업 중에 해당 로그를 보고 작업 진행 상황을 알 수 있습니다.

## 작업 단계
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/migration)에 로그인하고 왼쪽 사이드바에서 **데이터 구독**을 선택하여 데이터 구독 페이지로 이동합니다.
   - 방법1: 대상 구독 작업을 선택하고 해당 이름을 클릭합니다.
   - 방법2: 대상 구독 작업을 선택하고 **작업** 열에서 **구독 세부 정보 보기**를 클릭합니다.
     ![](https://qcloudimg.tencent-cloud.cn/raw/1bf25aa341f6f7db0630c90a31e36669.png)
2. 다른 탭 간에 전환하여 작업 로그를 봅니다.
<img src="https://qcloudimg.tencent-cloud.cn/raw/865915e3ac57221c583c30ca249e81f9.png" style="zoom:67%;" />

## 작업 상태 설명

| **상태**   | **설명**                         |
| ---------- | -------------------------------- |
| 시작되지 않음     | 구매가 완료되었지만 구독 작업이 구성되지 않았습니다.   |
| 확인 중     | 구독 작업 확인 중.             |
| 확인 통과   | 구독 작업 확인 성공.           |
| 확인 실패 | 구독 작업 확인 실패.             |
| 실행 중     | 구독 작업 실행 준비.               |
| 작업 실행 중   | 구독 작업 실행 중.                 |
| 중지 중     | 구독 객체 재설정 시 구독 작업 중지. |
| 작업 오류   | 오류로 인한 구독 작업 중단.     |
