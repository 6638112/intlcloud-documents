## 작업 시나리오

데이터 구독 Kafka 에디션은 암호 변경, 소비자 그룹 삭제 및 소비 offset 수정을 포함한 소비자 그룹 관리를 지원합니다.

또한 소비 관리에서 Client ID(데이터를 소비하는 클라이언트의 ID), 파티션 ID, 소비 오프셋, 소비 지연 등의 정보를 볼 수 있습니다.

## 전제 조건

[소비자 그룹](https://intl.cloud.tencent.com/document/product/571/39534)이 생성되어 있어야 합니다.

## 주의 사항

다른 Kafka 클라이언트가 동일한 소비자 그룹의 데이터를 사용하는 경우 소비 오프셋을 수정할 수 없습니다.

## 소비자 그룹 비밀번호 변경
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/dss)에 로그인한 뒤 왼쪽 사이드바에서 **데이터 구독**을 선택해 데이터 구독 페이지로 이동합니다.
2. 데이터 구독 리스트에서 원하는 데이터 구독을 선택한 후 구독 이름 또는 **작업**열의 **구독 상세 보기**를 클릭해 구독 관리 페이지로 이동합니다.
3. 구독 관리 페이지에서 **소비 관리** 탭을 선택하고 **작업** 열에서 **비밀번호 변경**을 클릭합니다.
4. 팝업된 대화 상자에서 비밀번호를 변경한 뒤, **변경**을 클릭합니다.

## 보안 그룹 삭제
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/dss)에 로그인한 뒤 왼쪽 사이드바에서 **데이터 구독**을 선택해 데이터 구독 페이지로 이동합니다.
2. 데이터 구독 리스트에서 원하는 데이터 구독을 선택한 후 구독 이름 또는 **작업**열의 **구독 상세 보기**를 클릭해 구독 관리 페이지로 이동합니다.
3. 구독 관리 페이지에서 **소비 관리** 탭을 선택하고 **작업**열에서 **삭제**를 클릭합니다.
4. 팝업 창에서 오류가 없는지 확인한 다음 **확인**을 클릭합니다.
>!소비자 그룹을 삭제하면 해당 소비자 그룹의 소비자 목록은 삭제되지만 데이터 구독 데이터는 삭제되지 않습니다. 신중히 작업하시기 바랍니다.

## 소비 offset 수정

1. [DTS 콘솔](https://console.cloud.tencent.com/dts/dss)에 로그인한 뒤 왼쪽 사이드바에서 **데이터 구독**을 선택해 데이터 구독 페이지로 이동합니다.
2. 데이터 구독 리스트에서 원하는 데이터 구독을 선택한 후 구독 이름 또는 **작업**열의 **구독 상세 보기**를 클릭해 구독 관리 페이지로 이동합니다.
3. 구독 관리 페이지에서 **소비 관리** 탭을 선택하고 대상 소비자 그룹을 선택한 다음 **offset 수정**을 클릭합니다.
4. 팝업 창에서 offset 객체를 선택하고 **다음**을 클릭합니다.	
5. 데이터 소비 재설정 방법을 설정하고 **완료**를 클릭합니다.
 - 최신 오프셋에서: 소비 오프셋을 Kafka 메시지의 최신 오프셋으로 재설정합니다.
 - 시작 오프셋에서: Kafka 메시지가 저장되기 시작하는 가장 빠른 오프셋으로 소비 오프셋을 재설정합니다.
 - 지정된 시점부터: 시간을 입력할 수 있으며, DTS는 입력된 시간에 해당하는 소비 오프셋을 찾아 소비를 시작합니다.
   구성된 시점이 Kafka 메시지 시간 이후인 경우 소비 오프셋은 Kafka 메시지의 최대 오프셋이 됩니다. 그렇지 않으면 소비 오프셋은 Kafka 메시지의 최소 오프셋이 됩니다.
