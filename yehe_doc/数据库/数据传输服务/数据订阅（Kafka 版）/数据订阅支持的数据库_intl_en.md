Currently, you can subscribe to only TencentDB instances but not databases in other deployment modes.

| **Source Database Type**           | **Source Database Version**                  | **Subscribable Data Types** |
| ------------------------------------- | ---------------------------------------- | ---------------------------------------- |
| TencentDB for MySQL | TencentDB for MySQL 5.5.x, 5.6.x, 5.7.x, and 8.0.x | <li>Data update<li>Structure update<li>Full instance |
| TencentDB for MariaDB           | TencentDB for MariaDB 10.0.10 and 10.1.9 | <li>Data update<li>Structure update<li>Full instance |
| TencentDB for MariaDB (Percona) | TencentDB for MariaDB (Percona) 5.7.17 | <li>Data update<li>Structure update<li>Full instance |
| TDSQL for MySQL | TDSQL for MySQL 8.0.18 and Percona 5.7.17 | <li>Data update<li>Structure update<li>Full instance |
| TDSQL-C for MySQL | TDSQL-C for MySQL 5.7.x and 8.0.x | <li>Data update<li>Structure update<li>Full instance |
| TDSQL for PostgreSQL | TDSQL for PostgreSQL | Data update |
