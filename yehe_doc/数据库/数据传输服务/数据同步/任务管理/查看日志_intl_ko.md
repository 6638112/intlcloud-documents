
## 작업 시나리오
동기화 작업 중에 동기화 작업 로그를 보고 작업 진행 상황을 알 수 있습니다.

## 작업 단계
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/migration)에 로그인하면 다음 두 가지 방법으로 작업을 볼 수 있습니다.
 - 방법1: 데이터 동기화 페이지에서 대상 동기화 작업을 선택하고 작업 이름을 클릭합니다.
 - 방법 2: 데이터 동기화 페이지에서 대상 동기화 **작업**을 선택하고 작업 열에서 **더보기** > **조회**를 클릭합니다.
2. 동기화 로그를 조회합니다.
탭을 전환하여 작업 로그를 조회합니다.

## 작업 상태 설명

| **상태**   | **설명**                                                     |
| ---------- | ------------------------------------------------------------ |
| 초기화되지 않음   | 구매가 완료되었지만 동기화 작업이 구성되지 않았습니다.                               |
| 초기화됨   | 동기화 작업 구성 완료.                                       |
| 확인 중     | 동기화 작업 확인 중.                                     |
| 인증 통과   | 동기화 작업 확인 완료.                                           |
| 확인 실패 | 동기화 작업 확인 실패.                                         |
| 준비 중   | 동기화 작업 시작 준비.                                           |
| 작업 실행 중   | 동기화 작업 실행 중.                                             |
| 재시도 가능한 오류 발생 | 동기화 중에 예외로 인해 동기화 작업이 중단되었습니다. 콘솔에서 작업을 다시 시도하고 재개할 수 있습니다. |
| 종료 중     | 동기화 작업 실행 중 수동 중지 중.     |
| 종료됨     | 작업 중지.                                               |
| 작업 실패   | 동기화 작업 실패.                                               |
| 삭제 중     | 작업 삭제 중. 완료, 실패 또는 불필요한 작업을 수동 삭제할 수 있습니다. |
| 삭제됨     | 작업 삭제 완료. 삭제되면 더 이상 존재하지 않거나 리소스를 차지하지 않습니다.         |
