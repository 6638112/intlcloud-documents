
## Overview
Data sync is a continuous task that will not stop on its own. You need to manually stop it at an appropriate time. 

## Prerequisites
The source-target database data gap is 0 MB, and the source-target database time lag is 0s. 

## Directions
1. Log in to the [DTS console](https://console.cloud.tencent.com/dts/migration), select **Data Sync** on the left sidebar, select the target sync task, and click **More** > **Stop** in the **Operation** column.
>?The task must meet the following conditions before it can be stopped: the source-target database data gap is 0 MB, and the source-target database time lag is 0s.
>
![](https://qcloudimg.tencent-cloud.cn/raw/218d83698ce70a8379e37f527004bc0b.png)
2. In the pop-up window, confirm that everything is correct and click **OK**.<br>
