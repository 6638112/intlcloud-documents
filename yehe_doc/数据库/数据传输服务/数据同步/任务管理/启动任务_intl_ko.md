## 작업 시나리오
동기화 작업이 시작되기 전에 작업 확인 중에 예외가 발생하면 수정한 후 작업을 다시 확인합니다. 확인 성공 후 작업을 시작해야 합니다.

## 전제 조건
작업이 데이터 확인이 완료되어야 합니다.

## 작업 단계
1. [DTS 콘솔](https://console.cloud.tencent.com/dts/migration)에 로그인하고 왼쪽 사이드바에서 **데이터 동기화**를 선택하고 대상 동기화 작업을 선택한 후 **작업** 열에서 **더보기** > **시작**을 클릭합니다.
2. 팝업 창에서 오류가 없는지 확인한 다음 **확인**을 클릭합니다.<br>
