본문에서는 데이터베이스 감사 서비스가 활성화된 후 로그 저장 기간을 수정하는 방법에 대해 설명합니다.

## 작업 단계
1. [TDSQL for MySQL 콘솔](https://console.cloud.tencent.com/tdsqld/instance-tdmysql) 로그인 후, 왼쪽 사이드바에서 **데이터베이스 감사**를 선택하고, 상단의 리전 선택 후, **감사 로그** 탭을 클릭합니다.
2. **감사 로그** 탭의 오른쪽 상단에서 **서비스 구성**을 클릭합니다.
   ![](https://qcloudimg.tencent-cloud.cn/raw/5f93d2a4440298529b363a1291f98668.png)
3. 팝업 창에서 로그 저장 기간을 수정하고 **제출**을 클릭합니다.
