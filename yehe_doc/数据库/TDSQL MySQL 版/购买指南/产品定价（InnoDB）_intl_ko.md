
## 과금 공식
**총 요금 = 인스턴스 요금 + 백업 공간 요금(현재 무료) + 트래픽 요금(현재 무료) + 감사 요금(현재 무료)**
**인스턴스 요금 = 노드 가격 x 노드 수 x 샤드 수 = (노드 메모리 크기 x 메모리 가격 + 노드 디스크 크기 x 디스크 가격) x 노드 수 x 샤드 수**

>?노드 수는 인스턴스 사양에 지정된 원본 노드와 복제본 노드의 합계입니다. 예를 들어 1-원본-1-복제본 사양은 2노드, 1-원본-2-복제본 사양은 3노드를 의미합니다.

## 과금 예시
[정액 과금제 예시]: 사용자가 1개월 동안 광저우 리전에서 2개의 샤드가 포함된 TDSQL for MySQL 정액 과금제 인스턴스 1개를 구매했다고 가정합니다. 각 샤드에는 2개의 노드(기본 1개 및 보조 1개)가 있으며 각 노드에는 2GB 메모리와 500GB 디스크 용량이 있습니다.
요금은 다음과 같이 계산됩니다.
인스턴스 요금 = (노드 메모리 크기 x 메모리 가격 + 노드 디스크 크기 x 디스크 가격) x 노드 수 x 샤드 수 = (2GB x 9.43 USD/GB/월 + 500GB x 0.06 USD/GB/월) x 2 노드 x 2 샤드 x 1개월 = 195.44 CNY
	
[종량제 예시]: 사용자가 베이징 리전에서 400시간 동안 2개의 샤드가 있는 TDSQL for MySQL 종량제 인스턴스 1개를 구매했다고 가정합니다. 각 샤드에는 2개의 노드(기본 1개 및 보조 1개)가 있으며 각 노드에는 2GB 메모리와 500GB 디스크 용량이 있습니다.
요금은 다음과 같이 계산됩니다.

 - 1단계 요금: (2GB x 0.026194 USD/GB/시간 + 500GB x 0.000250 USD/GB/시간) x 2개 노드 x 2개 샤드 x 96시간 = 68.116992 USD
 - 2단계 요금: (2GB x 0.013097 USD/GB/시간 + 500GB x 0.000250 USD/GB/시간) x 2개 노드 x 2개 샤드 x 264시간 = 159.660864 USD
 - 3단계 요금: (2GB x 0.019646 USD/GB/시간 + 500GB x 0.000250 USD/GB/시간) x 2개 노드 x 2개 샤드 x 40시간 = 26.28672 USD
인스턴스 요금 = 1단계 요금 + 2단계 요금 + 3단계 요금 = 254.064576 USD
