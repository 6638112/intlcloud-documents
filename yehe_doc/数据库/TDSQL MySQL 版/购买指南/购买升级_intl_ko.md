## 인스턴스 구매
[TDSQL for MySQL 구매 페이지](https://console.cloud.tencent.com/tdsqld/tdmysql-buy)에서 인스턴스를 구매할 수 있습니다. 각 인스턴스 노드(예: 프라이머리 1과 세컨더리 1은 노드 2개)에 대해 한 번에 최대 8개의 샤드를 구입할 수 있습니다.
구매한 인스턴스가 정상적으로 실행되려면 초기화가 필요합니다.


## 인스턴스 업그레이드
인스턴스 업그레이드는 현재 TDSQL for MySQL 인스턴스를 더 높은 사양으로 업그레이드하는 것을 의미합니다. 분산 데이터베이스는 여러 개의 샤드로 구성되기 때문에 인스턴스 업그레이드는 ‘샤드 추가와 샤드 확장’으로 나눌 수 있습니다. 서비스는 업그레이드 중에 중지되지 않지만 일부 샤드는 몇 초 동안 읽기 전용이 될 수 있습니다. 따라서 사용량이 적은 시간에 인스턴스를 업그레이드하는 것이 좋습니다.
>?최적의 분산 성능을 보장하려면 최대 8개의 샤드를 사용하는 것이 좋습니다. 구성을 업그레이드할 때 먼저 샤드를 확장하고 추가 확장이 불가능한 경우에만 더 많은 샤드를 추가하는 것을 고려하십시오.

### 샤드 추가
1. [TDSQL for MySQL 콘솔](https://console.cloud.tencent.com/tdsqld)에 로그인합니다. 인스턴스 목록에서 인스턴스 ID 또는 **작업** 열의 **관리**를 클릭하여 인스턴스 관리 페이지에 액세스합니다.
2. 인스턴스 관리 페이지에서 **샤드 관리** 탭을 선택하고 **작업** 열에서 **샤드 추가**를 클릭합니다.
![](https://main.qcloudimg.com/raw/d9a2e9b9261b3252a0db00472b1c3c92.png)
3. 팝업 창에서 신규 샤드의 사양을 선택합니다.

### 확장 샤드
이 작업은 새 샤드를 추가하지 않고 단일 샤드의 사양을 더 높은 수준으로 업그레이드합니다.
인스턴스 관리 페이지로 이동하여 **샤드 관리** 탭을 선택하고 **작업** 열에서 **샤드 설정 조정**을 클릭하여 조정합니다.

**업그레이드 비용 = (목표 사양 가격 - 원래 사양 가격) x 남은 유효 기간**
