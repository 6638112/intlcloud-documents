
TDSQL for MySQL 인스턴스를 생성한 후에는 이를 사용하기 전에 초기화해야 합니다.

## 작업 단계
1. [TDSQL for MySQL 콘솔](https://console.cloud.tencent.com/dcdb)에 로그인하여 리전을 선택하고 인스턴스 목록에서 ‘초기화되지 않은’ 인스턴스를 선택하고 ‘작업’ 열에서 [더보기]>[초기화]를 선택합니다.
2. 팝업 창에서 필요에 따라 구성을 선택하고 [확인]을 클릭합니다.
 - 지원되는 문자 집합: MySQL에서 지원하는 문자 집합을 선택합니다.
 - 대소문자 구분 테이블 이름: 데이터베이스의 테이블 이름이 대소문자 구분 여부를 설정합니다.
 - 강력한 동기화 활성화: 기본 노드가 실패할 경우 보조 노드에서 데이터 일관성을 보장하기 위해 강력한 동기화를 활성화합니다. 이 옵션이 작동하려면 최소 2개의 노드가 필요합니다.
![](https://main.qcloudimg.com/raw/2deee0a9b8a0bbc564b3ef1a290f060c.png)
3. 인스턴스를 찾으려면 인스턴스 목록으로 돌아가십시오. 상태가 ‘실행 중’으로 변경되면 인스턴스를 연결할 수 있습니다.

## 후속 작업
초기화 후에는 Windows 또는 Linux CVM 인스턴스에서 사설 및 공용 네트워크를 통해 TDSQL for MySQL 인스턴스에 액세스할 수 있습니다. 자세한 내용은 [인스턴스 연결](https://intl.cloud.tencent.com/document/product/1042/33338)을 참고하십시오.
