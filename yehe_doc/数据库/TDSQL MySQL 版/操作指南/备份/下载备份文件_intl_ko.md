
## 작업 시나리오
TDSQL for MySQL 콘솔에서 TencentDB 콜드 백업 데이터 및 binlog를 다운로드할 수 있습니다.

## 작업 단계
1. [TDSQL for MySQL 콘솔](https://console.cloud.tencent.com/dcdb)에 로그인하고 **작업** 열에서 인스턴스 ID 또는 **관리**를 클릭하여 인스턴스 관리 페이지로 들어갑니다.
2. 인스턴스 관리 페이지에서 **백업과 복원**을 클릭합니다.
3. **백업과 복원** > **콜드 백업 목록** 또는 **Binlog 목록**을 클릭하고 샤드 ID 및 시간을 선택한 다음 **작업** 열에서 **다운로드**를 클릭합니다.
4. VPC 주소는 팝업되는 다운로드 대화 상자에 제공됩니다. **복사**를 클릭하여 주소를 얻을 수 있습니다.
>?
>- 주소는 15분 동안 유효합니다. 만료된 후 페이지를 새로고침한 후 VPC 네트워크의 VPC 주소에 액세스합니다.
>- 다운로드 주소를 복사하여 데이터베이스가 상주하는 VPC 아래 CVM(Linux 시스템)에 로그인한 후 wget 명령어를 실행하여 다운로드할 것을 권장합니다.
