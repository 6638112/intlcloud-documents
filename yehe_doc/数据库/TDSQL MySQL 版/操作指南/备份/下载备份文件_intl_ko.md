
## 작업 시나리오
TDSQL for MySQL 콘솔에서 TencentDB 콜드 백업 데이터 및 binlog를 다운로드할 수 있습니다.

## 작업 단계
1. [TDSQL for MySQL 콘솔](https://console.cloud.tencent.com/dcdb)에 로그인하고 **작업** 열에서 인스턴스 ID 또는 **관리**를 클릭하여 인스턴스 관리 페이지로 들어갑니다.
2. **샤드 관리**를 선택하고 샤드 ID를 클릭하여 샤드 관리 페이지로 들어갑니다.
3. 백업 및 복원 탭으로 이동하여 **콜드 백업 목록** 또는 **Binlog 목록**을 선택하고 **작업** 열에서 **다운로드**를 클릭합니다.
3. VPC 주소는 팝업되는 다운로드 대화 상자에 제공됩니다. **복사**를 클릭하여 주소를 얻을 수 있습니다.
>?
>- 주소는 15분 동안 유효합니다. 만료된 후 페이지를 새로고침한 후 VPC 네트워크의 VPC 주소에 액세스합니다.
>- 다운로드 주소를 복사하고 TencentDB 인스턴스와 동일한 VPC의 (Linux) CVM 인스턴스에 로그인한 다음 다운로드를 위해 wget 명령을 실행하는 것이 좋습니다.
