
腾讯云为您提供了两种方式购买云数据 Redis：控制台购买和 API 购买。

## 控制台购买
用户可通过 [Redis 购买页](https://buy.intl.cloud.tencent.com/redis) 进行购买。用户可以购按量计费类型的云数据库 Redis，下面介绍购买这种类型的具体操作。

### 按量计费
1. 登录 [Redis 购买页](https://buy.intl.cloud.tencent.com/redis)。
2. 计费模式选择**按量计费**。
3. 根据实际情况和需求选择地域、可用区、版本、内存规格等，并根据需要选择网络类型。
4. 按量计费数据库创建时会冻结一个小时的数据库费用，请确保您的账户余额充足再进行购买。
5. 订单支付后立即开通服务，返回实例列表，即可看到。

>?
>- 开通按量计费云数据库 Redis 后，请确保您的账户余额充足。账户余额不足可能导致欠费甚至实例被回收，请参见 [到期提醒](https://intl.cloud.tencent.com/document/product/236/5159)。
>- 更多购买说明与注意事项请参见 [创建实例](https://intl.cloud.tencent.com/document/product/239/37712)。

## API 购买
通过 API 购买云数据库 Redis 的用户，可参考 API 文档 [创建实例](https://intl.cloud.tencent.com/document/product/239/32069)。
