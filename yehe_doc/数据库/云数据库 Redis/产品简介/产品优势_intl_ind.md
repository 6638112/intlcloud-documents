### Pemulihan Bencana Otomatis
TencentDB for Redis mengadopsi arsitektur hot backup primer/replika. Jika terjadi kegagalan server utama, akses dapat dialihkan ke replika dalam hitungan detik. Proses peralihan tidak memerlukan operasi apa pun dari Anda, sehingga menghemat biaya tenaga dan waktu untuk mengembangkan arsitektur sistem primer/replika.
