

### Gaming
Dalam industri game, data non-peran, seperti data papan peringkat, dapat disimpan di TencentDB for Redis untuk akses cepat, dan jenis data Set yang Diurutkan asli dapat dengan mudah membantu Anda mengurutkan data pemain.
![](https://main.qcloudimg.com/raw/e76810228c64a3d085d999f4537162ee.svg)

### Internet dan Aplikasi
Dalam bisnis internet dan aplikasi, data pengguna dasar dapat di-cache ke TencentDB for Redis untuk meningkatkan kinerja baca/tulis. Selanjutnya, gambar dan sumber daya statis dapat di-cache di sana juga untuk mempercepat pemuatan aplikasi.
![](https://main.qcloudimg.com/raw/5ec0c42091bced4415a7d54b2b6d112d.svg)

### Tampilan Ecommerce
Dalam tampilan ecommerce, data seperti gambar produk dan rekomendasi dapat disimpan di TencentDB for Redis untuk akses cepat. Selain itu, kinerja QPS yang tinggi (hingga puluhan juta) Redis dapat mempertahankan permintaan serentak yang tinggi, menjadikannya ideal untuk kampanye skala besar (misalnya, penjualan kilat).
