
DBbrain 除了大 Key、热点 Key 分析外，还提供 Redis 命令字分析，帮您更全面掌握当前数据库情况。

## 查看命令字分析
1. 登录 [DBbrain 控制台](https://console.cloud.tencent.com/dbbrain)，在左侧导航选择**诊断优化**，在上方选择对应数据库，然后选择**延迟分析** > **命令字分析**页。
2. 在命令字分析页，可查看实时和历史情况。
 - 实时情况
 - 历史情况
3. 在左上角可过滤命令字，来查看命令字分析。
