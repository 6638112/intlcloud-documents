 

## 1. 배경 

이 모듈은 Cloud Native Database TDSQL-C("**기능**")를 사용하는 경우 적용됩니다. 이 모듈은 https://intl.cloud.tencent.com/document/product/301/17347의 데이터 개인 정보 보호 및 보안 계약("**DPSA**")에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에 정의된 의미를 갖습니다. DPSA와 이 모듈 간에 충돌이 발생할 경우, 이 모듈은 상충하는 범위까지만 적용됩니다.

## 2. 처리

 당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다. 

| **개인** **정보**                                            | **이용**                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **구성 데이터**: 매개 변수 이름 및 매개 변수 값              | 당사는 사용자의 개별 구성에 따라 사용자에게 기능을 제공하기 위해서만 이 데이터를 처리합니다. |
| **데이터베이스 데이터**: 데이터베이스 파일(데이터베이스에 개인 데이터를 저장하는 경우) 및 시스템 파일 | 당사는 사용자에게 기능을 제공하기 위해서만 이 데이터를 처리합니다. 당사는 데이터베이스에 저장된 개인 데이터에 액세스할 수 없으며, 데이터를 제어할 수 없습니다. |
| **모니터링 데이터**: 모니터링 메트릭                         | 당사는 사용자가 운영 상태를 모니터링할 수 있게 하는 목적으로만 이 데이터를 처리합니다. 이 기능은 사용자가 이러한 서비스에  대해 사용할 수 있는 클라우드 모니터 기능으로 수행됩니다. 이 정보는 클라우드 모니터 기능으로 저장 및 백업됩니다. |

## 6. 특수 조건

 개인 데이터 처리에 동의할 수 있는 최소 연령을 만족하는 개인인 최종 사용자만 이 기능을 사용할 수 있습니다. 이는 최종 사용자의 해당 지역에 따라 다를 수 있습니다.
