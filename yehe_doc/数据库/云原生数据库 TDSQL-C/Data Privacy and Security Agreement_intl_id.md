 

## 1.LATAR BELAKANG

 Modul ini berlaku jika Anda menggunakan Cloud Native Database TDSQL-C ("**Fitur**”). Modul ini dimasukkan ke dalam Perjanjian Privasi dan Keamanan Data yang berlokasi di https://intl.cloud.tencent.com/document/product/301/17347 ("**DPSA**”). Istilah yang digunakan tetapi tidak didefinisikan dalam Modul ini akan memiliki arti yang diberikan kepada mereka di dalam DPSA. Apabila terjadi konflik antara DPSA dan Modul ini, Modul ini akan berlaku sejauh inkonsistensi yang ada.

 

## 2.PEMROSESAN

 Kami akan memproses data berikut sehubungan dengan Fitur:

| **Informasi Pribadi**                                        | **Tujuan Penggunaan**                                        |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Data Konfigurasi**: nama parameter dan  nilai parameter    | Kami hanya memproses  data ini untuk tujuan menyediakan Fitur kepada Anda dan sesuai konfigurasi  spesifik Anda. |
| **Data Database**: file database (jika  Anda menyimpan data pribadi dalam database) dan file sistem | Kami hanya memproses  data ini untuk tujuan menyediakan Fitur kepada Anda. Harap dicatat bahwa kami  tidak memiliki akses ke data pribadi, jika ada, yang disimpan dalam database  atau kontrol atas data. |
| **Data Pemantauan**: metrik pemantauan                       | Kami hanya memproses  data ini untuk memungkinkan Anda memantau status operasi. Harap dicatat bahwa  fungsi ini dilakukan oleh fitur Cloud Monitor kami yang dapat Anda aktifkan  untuk layanan tersebut. Informasi ini akan disimpan dan dicadangkan oleh  fitur Cloud Monitor kami. |
