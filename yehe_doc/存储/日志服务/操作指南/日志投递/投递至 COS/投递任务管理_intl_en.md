## Overview
CLS provides shipping task management, which allows you to query shipping tasks at point in time based on the time, logset, log topic, and shipping task name.
## Directions
## Querying Shipping Tasks
1. Log in to the [CLS Console](https://console.cloud.tencent.com/cls).
2. Select **Shipping Task** in the left sidebar to go to the Shipping Task Management page.
3. Select the region, logset, log topic, and shipping task name to filter and view the details of shipping tasks.

### Modifying Shipping Configuration
1. Log in to the [CLS Console](https://console.cloud.tencent.com/cls).
2. Select **Shipping Task** in the left sidebar to go to the Shipping Task Management page.
3. Select the region, logset, log topic, and shipping task name to locate the shipping task. Click **Modify Shipping Config** in the operation column.


### Disabling Shipping Tasks
1. Log in to the [CLS Console](https://console.cloud.tencent.com/cls).
2. Select **Shipping Task** in the left sidebar to go to the Shipping Task Management page.
3. Select the region, logset, log topic, and shipping task name to locate the shipping task. Click **Disable Shipping** in the operation column and click **Disable** when prompted for confirmation.
