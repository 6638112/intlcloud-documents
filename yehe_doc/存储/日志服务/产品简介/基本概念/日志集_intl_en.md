A logset is a project management unit of CLS and is used to distinguish logs of different projects. A logset corresponds to a project or application. Youâ€™re advised to use logsets to manage the logs of different projects and products. For example, if your company provide both e-commerce service and payment service, you can create two logsets.

A logset can contain multiple [log topics](https://intl.cloud.tencent.com/document/product/614/32849) and each log topic has the following basic attributes:
- Logset Name: the name of a logset.
- Logset ID: the unique ID of a logset.
- Region: the [region](https://intl.cloud.tencent.com/document/product/614/18940) to which a logset belongs.
- Retention period: the retention period of data in the current logset, which can be 1-366 days. To set a longer period, please [submit a ticket](https://console.cloud.tencent.com/workorder/category).
