Cloud Log Service (CLS) supports pay-as-you-go (postpaid), and you can choose CLS services based on your business needs.

>? If you know the resource usage and need to know the cost estimate and how to choose billing modes, use the [Price Calculator](https://buy.intl.cloud.tencent.com/price/cls/calculator) to conduct estimation and export the cost estimate list.
>

## Billing Mode

| Billing Mode                                                    | Description                                                      |
| ---------------------- | ------------------------------------------------------------ |
| **Pay-as-you-go (postpaid)** | This is the default billing mode of CLS supported in all regions as described in [Available Regions](https://intl.cloud.tencent.com/document/product/614/18940). Users are charged based on their actual storage usage, requests, traffic, and other billable items on a daily basis. |




## Billing Details

| Category     | Description                                                         |
| -------- | ------------------------------------------------------------ |
| Billing cycle | Daily. The fees generated for the current day will be settled at 00:00 on the next day.</br> |
| Price | [See CLS [Product Pricing](https://intl.cloud.tencent.com/document/product/614/37510). |
