## 소개
COS 콘솔을 통해 객체 속성 정보 조회(객체 크기, 객체 주소) 및 객체 관련 설정(객체의 액세스 권한, 스토리지 유형 변경 설정 등)을 할 수 있습니다.

## 작업 순서
1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하여 왼쪽 메뉴에서 [버킷 리스트]를 클릭해 버킷 리스트 페이지로 이동합니다.
2. 객체 정보를 보려는 버킷을 찾아 버킷 이름을 클릭하여 버킷의 파일 리스트 페이지로 이동합니다.
3. 버킷 세부 페이지에 있는 “파일 리스트” 선택 항목 페이지에서 속성 정보 조회가 필요한 객체를 찾고 오른쪽 작업 열에서 [자세히]를 클릭합니다.
   ![](https://main.qcloudimg.com/raw/9a4305b1423c5ce771082490981b9827.png)
4. [자세히]를 클릭하면 객체 크기, 객체 주소, 획득한 서명 링크 등의 관련 정보를 조회하고 객체 관련 설정을 할 수 있습니다.
   ![](https://main.qcloudimg.com/raw/d56eb8baa6221ac57b8b9d323f96d2b2.png)
