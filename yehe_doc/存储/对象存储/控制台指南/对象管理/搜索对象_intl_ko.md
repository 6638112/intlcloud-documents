## 소개
COS 콘솔을 통해 이미 업로드 완료한 객체, 폴더 및 폴더에 속한 객체를 검색할 수 있습니다. COS는 버킷에 속한 객체(혹은 폴더) 검색을 지원하며 버킷의 다중 폴더와 해당 폴더에 있는 객체 검색을 지원합니다.

>?객체 검색 시 입력하는 객체 접두사는 영어 대소문자를 구분해서 써야 합니다.

## 버킷의 현재 객체 검색

1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 왼쪽 메뉴에서 [버킷 리스트]를 클릭하여 버킷 리스트 페이지로 이동합니다.
2. 객체가 속한 버킷을 클릭하여 버킷 관리 페이지로 이동합니다.
![](https://main.qcloudimg.com/raw/81403794e014551e53a57a9935f55f83.png)
3. [파일 리스트] 선택 항목 페이지에서 오른쪽 상단의 검색 상자에 객체 이름 접두사를 입력한 후 검색 버튼을 클릭하면 현재 버킷에 **동일한 이름 접두사**를 가진 객체나 폴더가 보여집니다. 구체적으로 객체 검색이 필요한 경우 완전한 객체 키를 입력할 수 있습니다. 예시: `exampleobject.txt`
![](https://main.qcloudimg.com/raw/9eadf9e5e0244cea5d94b52ca004309e.png)

## 다중 폴더의 객체 검색

1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 왼쪽 메뉴에서 [버킷 리스트]를 클릭하여 버킷 리스트 페이지로 이동합니다.
2. 객체가 속한 버킷을 클릭하여 버킷 관리 페이지로 이동합니다.
![](https://main.qcloudimg.com/raw/81403794e014551e53a57a9935f55f83.png)
3. [파일 리스트]선택 항목 페이지에서 오른쪽 상단의 검색 상자에 객체가 속한 완전한 경로(혹은 폴더 이름)와 객체 접두사(완전한 입력 불필요)를 입력한 후 검색 버튼을 클릭하면 동일 폴더에 **동일한 객체 접두사**를 가진 검색 결과가 보여집니다.
![](https://main.qcloudimg.com/raw/b49019d31fb82b6d9d0e3316bb5a9cb2.png)
