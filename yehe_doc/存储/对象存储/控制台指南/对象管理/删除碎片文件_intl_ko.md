## 소개
지정 버킷을 삭제할 때 “**삭제 실패, 버킷에 있는 유효한 데이터를 먼저 삭제하십시오**”라는 알림이 뜬다면 **파일 조각** 관리 페이지로 이동하여 업로드 미완료 파일 조각을 조회해 삭제할 수 있습니다. 버킷에 있는 업로드 미완료 파일이나 업로드 완료 파일을 조회한 후 전부 삭제하면 버킷 삭제가 가능합니다.

>
>- 객체 업로드 과정에서 업로드 중지나 업로드 취소 파일은 모두 [파일 조각] 관리 페이지에서 보여집니다. 업로드 완료 후 [파일 리스트]에서 업로드 완료 파일을 찾을 수 있습니다.
>- 파일 조각은 정상 객체와 같이 버킷 공간을 차지하기 때문에 버킷 용량 비용이 청구될 수 있습니다.

## 파일 조각 수동 삭제

1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 왼쪽 메뉴에서 [버킷 리스트]를 클릭하여 버킷 리스트 페이지로 이동합니다.
2. 삭제가 필요한 파일 조각의 버킷을 클릭하여 버킷에 있는 상세 페이지로 이동합니다.

3. **파일 리스트**페이지에서 [파일 조각]을 클릭해 업로드가 완성되지 않은 조각 파일을 조회합니다.

4. 파일 조각 오른쪽에 있는 [삭제]를 클릭하여 업로드 미완료 파일을 삭제하거나 상단의 [조각 제거]를 클릭하여 모든 업로드 미완료 조각 파일을 한번에 삭제합니다.
![](https://main.qcloudimg.com/raw/48ba3386157a800bb6896d655e52a975.png)
5. “조각 제거”혹은 “삭제” 작업 후 보여지는 리스트에는 아무 것도 없습니다.


## 라이프사이클 기한 설정으로 파일 조각 제거하기

1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 왼쪽 메뉴에서 [버킷 리스트]를 클릭하여 버킷 리스트 페이지로 이동합니다.
2. 삭제가 필요한 파일 조각의 버킷을 클릭하여 버킷에 있는 상세 페이지로 이동합니다.
![](https://main.qcloudimg.com/raw/a5c567ac2e21364e10e0873fcb658a1a.png)
3. 왼쪽 메뉴에서 [기본 설정] > [라이프사이클]을 클릭해 하단에서 [라이프사이클] 설정 페이지를 찾습니다.
![](https://main.qcloudimg.com/raw/f9ba822a0f34eae84ede38d302f22092.png)
4. [규칙 추가]를 클릭해 아래 그림에 보이는 정보대로 설정합니다. 전체 버킷 범위 내에서 조각이 생성 7일 후 삭제되도록 규칙을 설정합니다.
![](https://main.qcloudimg.com/raw/9bb0397b0a87e5dadbdf40183329de5e.png)
5. [확인]을 클릭하면 콘솔에서 설정 완료한 라이프사이클 규칙을 볼 수 있습니다.
![](https://main.qcloudimg.com/raw/48ba0e8a81fa2a8eda8bcff276503fbe.png)
