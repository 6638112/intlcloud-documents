## 소개
COS 콘솔을 통해 버킷에 업로드된 단일 파일이나 여러 파일을 삭제할 수 있습니다.

>!데이터 삭제 후 복구가 불가능하니 신중하게 작업해주십시오.


## 단일 객체 삭제
1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 왼쪽 메뉴에서 [버킷 리스트]를 클릭하여 버킷 리스트 페이지로 이동합니다.
2. 해당 버킷에서 버킷 이름을 클릭하여 버킷의 파일 리스트 페이지로 이동합니다.
![](https://main.qcloudimg.com/raw/156823c7ad23708feb85f5d682c61d50.png)
3. 삭제하려는 객체를 찾아 [더 많은 조작] > [삭제]를 클릭하면 파일 삭제 대화 상자가 팝업됩니다.
![](https://main.qcloudimg.com/raw/952657b354b3f554780c57b9ba1a23ad.png)
4. [확인]을 클릭하면 객체가 삭제됩니다.
![](https://main.qcloudimg.com/raw/3e579dc3f90f83c6ececbb76dae519f7.png)


## 여러 객체 삭제
1. COS 콘솔에서 삭제하려는 객체를 선택하고 [더 많은 조작]의 드롭다운 리스트에서 [삭제]를 클릭하면 파일 삭제 대화 상자가 팝업됩니다.
![](https://main.qcloudimg.com/raw/0f780f2130fd234c3ae5268d6b9bf091.png)
2. [확인]을 클릭하면 객체가 일괄 삭제됩니다.
![](https://main.qcloudimg.com/raw/66e599c3801dd5e51c9aab65f7274fcc.png)
