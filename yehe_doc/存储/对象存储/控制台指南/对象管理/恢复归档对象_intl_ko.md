## 소개

CAS 혹은 DEEP ARCHIVE 유형의 객체를 COS 콘솔을 통해 복구(동결 해제)시킬 수 있습니다. 복구 작업은 새로운 표준 스토리지 유형의 객체 사본을 생성하며 설정한 유효 기간 동안 객체 사본으로 읽기나 다운로드 등의 정상적인 작업을 진행할 수 있습니다. 기간 만료 후에 객체 사본은 자동으로 삭제됩니다. 객체의 스토리지 유형에 대한 자세한 내용은 [스토리지 유형](https://intl.cloud.tencent.com/document/product/436/30925)을 참조하십시오.

>! CAS 유형 데이터를 복구(동결 해제)하면 객체 사본으로 인해 발생한 비용은 표준 스토리지 요금에 따라 과금됩니다. DEEP ARCHIVE 유형 데이터를 복구(동결 해제)하면 객체 사본으로 인해 발생한 요청 비용은 표에 보이는 단가대로 과금되며 트래픽 비용과 표준 스토리지 비용은 같습니다. 상세 가격 관련 정보는 [제품 가격](https://intl.cloud.tencent.com/pricing/cos) 문서를 참고하십시오.
>

## 작업 순서

1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인합니다.
2. 왼쪽 메뉴에서 [버킷 리스트]를 클릭하여 버킷 리스트 페이지로 이동합니다.
3. 객체가 속한 버킷을 찾아 해당 버킷 이름을 클릭하여 버킷 관리 페이지로 이동합니다.
4. 왼쪽 메뉴에서 [파일 리스트]를 선택하여 파일 리스트 페이지로 이동합니다.
5. 복구할 객체를 찾아 오른쪽 작업 열에 있는 [복구]를 클릭합니다.
![](https://main.qcloudimg.com/raw/2d34eaee99c4bcb469025f857ed983f1.png)
여러 보관 유형의 객체에 대해 일괄 복구를 진행하는 경우, 여러 객체를 선택한 후 상단에 있는 [더 많은 작업]>[복구]를 클릭하십시오.
![](https://main.qcloudimg.com/raw/eaa698982cd42fe1918aace11854b693.png)
4. 팝업 창에서 복구 모드와 복제본 유효 일수를 설정합니다.
![](https://main.qcloudimg.com/raw/3d3163c09213423aa959b179237ab137.png)
설정 관련 정보는 다음과 같습니다.
 - **복구 모드**: 고속 모드, 표준 모드, 일괄 모드가 있습니다.
    - 고속 모드: 필요 시간이 가장 짧습니다. 통상적으로 CAS 유형 파일 복구는 1 - 5분정도 소요됩니다. 특수 상황에서 급하게 아카이브 데이터가 필요한 경우 쾌속 모드를 사용해 소요 시간을 단축하고 효율을 높일 수 있습니다. **쾌속 모드는 DEEP ARCHIVE 유형 파일 복구를 지원하지 않습니다.**
    - 표준 모드: 표준 모드를 사용하면 CAS 유형 파일은 일반적으로 3- 5시간 내에 복구 완료되며 DEEP ARCHIVE 유형 파일은 일반적으로 12시간 내에 복구 완료됩니다.
    - 일괄 모드: 비용이 가장 저렴합니다. 긴급하지 않은 경우 일괄 모드를 이용해 낮은 비용으로 대량의 아카이브 데이터를 얻을 수 있습니다. CAS 유형 파일은 일반적으로 5 - 12시간 내에, DEEP ARCHIVE 유형 파일은 48시간 내에 복구가 완료됩니다.
      >? 데이터 복구 요청에는 QPS 제한이 있으며, 100회/초로 제한됩니다.
      >
 - **복제본 유효 기간**: 복제본 유효 기간을 설정한 후 기간이 만료되면 자동으로 삭제됩니다. 최소 설정 범위는 1일이며 최장 365일입니다. 객체 복구 완료 후 다시 [복구]를 클릭하여 팝업창에서 복제본 유효 기간을 수정할 수 있습니다.
7. [확인]을 클릭하면 객체 복구 과정을 시작합니다.
진행 과정 중, [상세 내용]을 클릭하면 객체 상세 페이지로 이동하여 복구 완료 여부를 확인할 수 있습니다.
![](https://main.qcloudimg.com/raw/4686bc71f95d7178a8690f3eb49721e6.png)
8. 객체 복구 완료를 확인한 후 다시 [복구]를 클릭하면 복제본의 유효 기간을 수정할 수 있습니다.
![](https://main.qcloudimg.com/raw/e794f58d2593df1ecb675fc7ffa5019c.png)
이 때, 객체의 상세 페이지로 이동하여 객체 액세스, 다운로드 등의 작업을 할 수 있습니다.
![](https://main.qcloudimg.com/raw/c464c9fa3e83213f7cf8692f32fa72ab.png)
