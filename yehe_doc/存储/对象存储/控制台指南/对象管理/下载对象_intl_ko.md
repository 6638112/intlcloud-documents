## 소개
COS 콘솔로 버킷 객체 다운로드가 가능하며 콘솔로 단일 객체를 다운로드하거나 COSBrowser 툴로 객체를 일괄 다운로드할 수 있습니다.

## 전제 조건
객체 다운로드 전에 버킷에 객체가 있는지 확인하십시오. 객체를 업로드하지 않았다면 [객체 업로드](https://intl.cloud.tencent.com/document/product/436/13321) 문서를 참고하여 작업을 진행해 주십시오.

## 단일 객체 다운로드
1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 [버킷 리스트]를 클릭하여 버킷에 있는 리스트 페이지로 이동합니다.
2. 객체가 속한 버킷을 찾아 그 버킷 이름을 클릭하여 버킷에 있는 관리 페이지로 이동합니다.
![](https://main.qcloudimg.com/raw/46307132ac1ef1e8422667abd896f878.png)
3. “파일 리스트” 탭에서 다운로드가 필요한 객체를 찾습니다. 단일 객체 다운로드만 필요할 경우 여러가지 방법을 선택할 수 있습니다.
 1. 선택한 객체 오른쪽에서 바로 [다운로드]를 클릭하여 다운로드합니다. 단일 객체를 선택한 경우 [더 많은 조작] 드롭다운 리스트의 [다운로드]를 클릭합니다.
![](https://main.qcloudimg.com/raw/4ee4cb3f5ff0a5634ef8ed49d26c549b.png)
 2. 객체 오른쪽의 [자세히]를 클릭하여 파일의 상세 페이지로 이동합니다. 파일 상세 페이지에서 [객체 다운로드]를 클릭하여 다운로드하거나 [임시 링크 복사]를 클릭하여 링크를 브라우저 주소란에 붙여넣습니다. Enter 키를 누르면 그 객체가 다운로드됩니다.
![](https://main.qcloudimg.com/raw/934b43a085b8cae825ddad88a0177caf.png)

>
>- 객체가 속한 버킷 속성이 개인 읽기/쓰기로 설정되어 있다면 복사한 주소는 자동으로 서명이 계산되어 접미사가 추가됩니다. 서명 생성 방법에 대한 상세 내용은 [서명 요청](https://intl.cloud.tencent.com/document/product/436/7778) 문서를 참조하십시오.
>- 서명을 가진 임시 링크는 **객체 상세 조회**후 1시간 이내로 유효하며 [유효기간 갱신]을 클릭해 서명 유효 기간을 갱신합니다.

## 객체 및 폴더 일괄 다운로드
>COS 콘솔은 단일 객체 다운로드만 지원합니다. 여러 객체나 폴더의 일괄 다운로드가 필요할 경우 [COSBrowser 클라이언트](https://intl.cloud.tencent.com/document/product/436/11366) 사용 설치를 권장합니다. 다음은 콘솔에서 **COSBrowser 클라이언트**를 결합하여 객체나 폴더를 일괄 다운로드하는 방법입니다.

1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 [버킷 리스트]를 클릭하여 버킷에 있는 리스트 페이지로 이동합니다.
2. 객체가 속한 버킷을 찾아 그 버킷 이름을 클릭하여 버킷에 있는 관리 페이지로 이동합니다.

3. 여러 객체를 선택한 경우 [더 많은 조작] 드롭다운 리스트의 [다운로드]를 클릭합니다.
![](https://main.qcloudimg.com/raw/26499cc4f21d5a76580627fe9fb0db56.png)
4. 팝업 알림에 따라 설치하거나 COSBrowser 클라이언트를 열어 로그인합니다.
5. COSBrowser를 연 후 파일 저장 위치를 선택하면 선택된 파일은 자동으로 다운로드 큐에 포함되어 다운로드됩니다. [다운로드 리스트]를 클릭하여 조회하십시오.
![](https://main.qcloudimg.com/raw/140c741db0854e17d24d2b69e89bd268.png)
