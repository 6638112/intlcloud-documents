## 소개

객체 태그 기능은 객체에 키 값 쌍 형식의 식별자를 추가하여 실현하며 사용자가 버킷의 객체를 분할 관리하는 데 도움을 줍니다. 객체 태그는 태그 키(`tagKey`)와 태그 값(`tagValue`), `=`으로 구성됩니다. 예시로 `group = IT`와 같은 형식입니다. 사용자는 지정 객체에 태그 설정, 조회, 삭제 작업을 할 수 있습니다.

> 객체 태그는 과금 항목에 속합니다. 자세한 가격은 [제품 가격](https://intl.cloud.tencent.com/document/product/436/6239) 문서를 참조하십시오.

객체 태그를 사용할 때 다음 제한 사항을 주의하십시오.

- 사용자는 객체 하나당 최대 10개 태그를 추가할 수 있으며 태그는 중복될 수 없습니다.
- 태그 키는 `qcs:`, `project`, `항목` 등을 첫 글자로 사용할 수 없습니다. 이 문자들은 시스템에 예약된 태그 키입니다.
- 태그 키와 태그 값은 UTF-8 포맷이 나타내는 문자, 빈칸, 숫자, 특수 문자 `+ - = ._ : / @ `를 지원합니다. 길이 범위는 문자 1-127자이며 영어는 대소문자를 구분해야 합니다.

제한 내용 설명에 대한 자세한 내용은 [객체 태그 개요](https://intl.cloud.tencent.com/document/product/436/35665)를 참조하십시오.

## 객체 업로드 시 태그 추가

1. 다음 그림과 같이 [객체 업로드](https://intl.cloud.tencent.com/document/product/436/13321) 시 추가할 수 있습니다.
   ![](https://main.qcloudimg.com/raw/f9ed9a4b46cf8299c9f5cc2e14415765.png)
2. 업로드가 완료되면 객체 태그를 추가할 수 있습니다. 객체 상세 페이지의 [객체 태그 관리] 설정에서 추가된 태그를 조회할 수 있습니다.
   ![](https://main.qcloudimg.com/raw/b49fa4a5d1df2b40fe7caf2f5b453f9f.png)
3. 태그를 수정하거나 삭제하려면 [객체 태그 관리] 설정 페이지에서 태그 오른쪽의 [편집] 혹은 [삭제]를 클릭합니다.


## 업로드한 객체에 태그 추가하기

객체를 업로드할 당시 태그를 추가하지 못한 경우 아래 순서에 따라 객체에 태그를 추가할 수 있습니다.

1. [객체 정보 조회](https://intl.cloud.tencent.com/document/product/436/13326)를 참고하여 태그 추가가 필요한 객체의 상세 페이지로 이동합니다.
2. 객체 상세 페이지의 [객체 태그 관리] 설정 항목에서 [태그 추가]를 클릭하고 객체에 태그를 클릭합니다.
   ![업로드된 객체에 태그 추가](https://main.qcloudimg.com/raw/d4bda3a72248e8ad7c33178e8047075b.png)
3. 태그를 수정하거나 삭제하려면 [객체 태그 관리] 설정 페이지에서 태그 오른쪽의 [편집] 혹은 [삭제]를 클릭합니다.
