## 소개

Tencent Cloud COS는 객체 잠금 기능을 제공하며, 이 기능은 설정한 기간 내에 객체를 수정하거나 삭제할 수 없으며 즉시 액세스할 수 있습니다. 본 문서는 콘솔에서 객체 잠금 기능을 활성화하는 방법에 대해 소개합니다.


## 작업 순서

1. [COS 콘솔](https://console.cloud.tencent.com/cos5)에 로그인하고 왼쪽 메뉴바의 [버킷 리스트]로 이동해 객체 잠금을 설정할 버킷을 클릭합니다.
2. 왼쪽의 [보안 관리]>[객체 잠금]을 클릭하고 [객체 잠금] 설정 페이지를 찾은 후 [편집]을 클릭해 현재 상태를 '활성화'로 변경합니다.
3. 설정 창에 보관 기간을 입력하고 설정 완료 후 [저장]을 클릭합니다. 다시 팝업창이 뜨면 [확인]을 클릭해 설정을 완료합니다.
 - 보관 기간: 자연수를 입력합니다. 보관 주기는 연장만 가능하고 단축할 수 없으니 합리적으로 설정하시기 바랍니다.
![](https://main.qcloudimg.com/raw/ce97b907b05577d7803ba6e5a1822957.png)
4. 설정 완료 후 객체 잠금의 만료 날짜를 확인할 수 있습니다. 왼쪽의 [파일 리스트]를 클릭해 확인할 파일을 선택하고 파일 오른쪽의 [상세 내용]을 클릭하면 기본 정보에서 객체 잠금의 만료 날짜(베이징 시간)를 확인할 수 있습니다.
