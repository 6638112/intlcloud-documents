## 課金の例

>? 次の例に記載した料金価格は参考用です。実際の価格についてはCOS[製品価格](https://buy.cloud.tencent.com/price/cos)をご参照ください。
>


### 事例：標準ストレージ容量料金 + オブジェクトタグ料金 + リクエスト料金

ユーザーのAさんが2020年11月1日に10GBの標準ストレージタイプのデータを広州リージョンのCOSバケットにアップロードし、当日に10万個のオブジェクトにタグを一括追加し、かつ10万回のリクエスト回数が発生し、それ以外の時間には他の操作を何も行わなかったとします。ストレージ容量料金、リクエスト料金は月次決済、オブジェクトタグ料金は日次決済となります。この場合、2020年11月2日にオブジェクトタグ料金が、2020年12月1日に標準ストレージ容量料金と標準ストレージリクエスト料金がそれぞれ決済され、従量課金方式に基づく分析は次のようになります。

- 従量課金：
 - 標準ストレージ容量料金 = 0.024米ドル/GB/月 x 10GB = 0.24米ドル。
 - オブジェクトタグ料金 = 0.01米ドル/1万個あたりのタグ x オブジェクト10万個 = 0.1米ドル
 - 標準ストレージリクエスト料金 = 0.002米ドル/万回 x 10万回 = 0.02米ドル

上記の分析を総合すると、11月中のAさんの料金総額は0.24 + 0.1 + 0.02 = 0.36米ドルとなります。
