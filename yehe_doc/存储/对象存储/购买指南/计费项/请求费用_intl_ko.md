- 사용자 요청 횟수: 사용자가 API, SDK 또는 콘솔 등을 이용해 데이터를 업로드, 다운로드, 조회, 삭제하는 등의 작업은 Tencent Cloud COS로 요청을 발송하여 수행됩니다.

### 사례: 표준 스토리지 용량 비용 + 표준 스토리지 요청 비용

사용자 A가 2020년 11월 1일 표준 스토리지 유형의 10GB 데이터를 광저우 리전의 COS 버킷에 업로드하고, 당일 GET Bucket API를 호출하여 객체 리스트를 총 5만 번 반복 조회했으며, 다른 시간에는 아무 작업도 하지 않았다고 가정합니다. 매월 1일에 전월 발생한 스토리지 용량 비용과 요청 비용을 결산 및 공제하므로 2020년 12월 1일 스토리지 용량 비용과 표준 스토리지 요청 비용이 결산됩니다. 종량제 방식에 따른 비용 분석은 다음과 같습니다.

 -  종량제:
  -  표준 스토리지 용량 비용 = 0.024USD/GB/월 x 10GB = 0.24USD
  -  표준 스토리지 요청 비용 = 0.002USD/1만 회 x 5만 회 = 0.01USD


위 금액을 모두 합산하면 전체 11월 사용자 A의 총 요금은 0.24 + 0.01 = 0.25USD입니다.
