### 사례1: 스탠다드 스토리지 사용량 요금 + 스탠다드 스토리지 요청 요금

사용자 A가 2020년 11월 1일 스탠다드 스토리지 유형의 10GB 데이터를 광저우 리전의 COS 버킷에 업로드하고 100번의 요청이 발생하였으며 다른 시간에는 아무 작업도 하지 않았다고 가정합니다. 매월 1일 전월 발생한 스토리지 용량 비용과 요청 비용을 결산 및 차감하므로, 2020년 12월 1일에 스탠다드 스토리지 용량 비용과 스탠다드 스토리지 요청 비용이 결산되며, 종량제 방식에 따른 비용 분석은 다음과 같습니다.

- 종량제:
 - 스탠다드 스토리지 용량 비용 = 0.024USD/GB/월 x 10GB = 0.24USD
 - 스탠다드 스토리지 요청 비용 = 0.002USD/1만 회 x 1만 회 = 0.02USD

위 금액을 모두 합산하면 전체 11월 사용자 A의 총 요금은 0.24 + 0.02 = 0.26USD입니다.


### 사례2: 스탠다드IA 스토리지 사용량 요금 + 스탠다드IA 스토리지 요청 요금
