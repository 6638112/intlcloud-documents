## 과금 유의 사항

1. 아카이브 유형의 데이터는 데이터 읽기 및 다운로드가 불가능합니다. 해당 유형의 데이터를 읽거나 다운로드하려면 먼저 데이터를 표준 스토리지 유형으로 복구(또는 동결 해제)한 후 읽어와야 합니다. 따라서 트래픽이 표준 스토리지에 집계됩니다.
2. 3rd party CDN을 사용해 Tencent Cloud COS로 Origin-pull할 경우, 공인 네트워크 다운스트림 트래픽이 발생합니다.
3. Tencent Cloud CDN 가속 활성화 이후, **Tencent Cloud CDN 가속 도메인**을 통해 클라이언트에서 COS 데이터를 조회하거나 다운로드할 때 발생하는 트래픽은 CDN Origin-pull 트래픽에 속합니다.
4. 사용자가 **객체 링크**로 객체를 직접 다운로드하거나 **정적 웹 사이트 원본 서버**로 객체를 조회할 때 발생하는 트래픽은 공인 네트워크 다운스트림 트래픽에 속합니다. 
5. 인터페이스 복사 또는 리전 간 복제 기능으로 데이터를 한 리전 버킷에서 다른 리전의 버킷으로 전송할 때 발생하는 트래픽은 리전 간 복제 트래픽입니다. 리전 간 복제 트래픽은 원본 버킷이 속한 리전의 단가를 기준으로 요금이 부과됩니다.
6. INTELLIGENT TIERING 유형의 데이터로 인해 발생하는 트래픽 요금은 전환한 스토리지 유형의 요금과 동일합니다.

## 과금 예시

>?
> - 다음 예시의 가격은 참고용입니다. 실제 가격은 COS [제품 가격](https://buy.cloud.tencent.com/price/cos)을 참조하십시오.
> - 요청 요금은 **1만 회**가 최소 단위입니다. 따라서 요청의 성공 여부에 관계없이 월간 누적 요청 횟수가 1만 번 미만이면 1만 번으로 계산합니다.
> 

### 예시: 표준 스토리지 용량 요금 + 표준 스토리지 요청 요금 + 공인 네트워크 다운스트림 트래픽 요금

사용자 A가 2020년 11월 1일에 표준 스토리지 유형의 10GB 데이터를 광저우 리전의 COS 버킷에 업로드하였고 100번의 요청이 발생하였으며, 둘째 날과 셋째 날 공용 네트워크를 사용해 CDN을 활성화하지 않은 상태에서 해당 데이터를 로컬에 다운로드하고, 다른 시간에는 아무런 작업도 하지 않았습니다. 스토리지 용량 요금과 요청 요금은 월 결산되며, 트래픽 요금은 일 결산됩니다. 이에 따라 2020년 11월 3일과 4일에 트래픽 요금이 결산되고, 2020년 12월 1일에 표준 스토리지 용량 요금과 표준 스토리지 요청 요금이 결산됩니다. 요금을 분석하면 다음과 같습니다.

 - 표준 스토리지 용량 요금 = 0.024USD/GB/월 x 10GB = 0.24USD
 - 표준 스토리지 요청 요금 = 0.002USD/1만 회 x 1만 회 = 0.02USD
 - 공인 네트워크 다운스트림 트래픽 요금 = 0.1USD/GB x 10GB x 2 = 2USD


위 요금을 종합해보면, 전체 11월 사용자 A의 총 결제 금액은 0.24 + 0.02 + 2 = 2.26USD입니다.
