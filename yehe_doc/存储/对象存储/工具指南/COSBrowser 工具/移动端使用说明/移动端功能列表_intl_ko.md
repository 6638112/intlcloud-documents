COSBrowser App은 Tencent Cloud Cloud Object Storage(COS)에서 공식 출시한 모바일 애플리케이션으로，COS의 리소스를 편리하고 빠르게 조회, 관리할 수 있고 언제 어디서나 클라우드 리소스를 확인할 수 있습니다. COSBrowser App에서 지원하는 기능은 다음과 같습니다. 


## 데이터 모니터링

| 작업 이름                                                       | 작업 설명                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 사용량 개요 | 최근 데이터 사용 상황을 표시할 수 있습니다.                                |               
| 버킷 모니터링| 버킷을 기준으로 최근 데이터 사용 상황을 조회할 수 있습니다.                                    |
| 미니 컴포넌트 | 미니 컴포넌트 모드 데이터 개요를 지원하며, App을 열지 않고도 데이터 사용 상황을 조회할 수 있습니다.   |

## 버킷 관리 및 작업

| 작업 이름                                                       | 작업 설명                                                     |
|---------------------------------------- | ---------------------------------- |
| 버킷 리스트 조회 | 버킷을 생성할 수 있습니다.                                          |
| 액세스 경로 추가 | 버킷 리스트 액세스 권한이 없는 서브 계정으로 로그인한 경우, 액세스 경로를 추가하여 액세스할 수 있습니다.                                             |
| 버킷 생성 | 버킷을 생성할 수 있습니다.                                      |
| 버킷 검색| 버킷 리스트 페이지에서 키워드로 퍼지 검색을 할 수 있습니다.                                  |
| 버킷 기본 정보 조회 | 모바일 기기로 버킷 이름, 리전, 생성 시간 및 AZ 활성화 여부 등 기본 정보를 조회할 수 있습니다.                                |
| 버킷 권한 관리 | 버킷의 공용 권한과 사용자 권한을 수정할 수 있습니다.                                   |
| 글로벌 가속 활성화 | 모바일 버킷에 있는 파일을 일괄 업로드, 다운로드, 삭제, 복사, 이동할 수 있습니다.           |
| 버킷 전송 설정| 해당 기능을 통해 업로드 및 다운로드에 사용하는 도메인을 설정할 수 있습니다.  


## 파일 작업 및 관리

| 작업 이름                                                       | 작업 설명                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 폴더 생성 | 버킷 내에 폴더를 생성할 수 있습니다.                                          |
| 폴더 삭제 |현재 디렉터리 및 서브 디렉터리의 모든 파일을 삭제할 수 있습니다.                                          |
| 파일 업로드 | 모바일 COSBrowser를 사용하여 로컬 파일, 원격 파일, 기타 App의 파일 및 파일 관리자를 COS에 업로드 할 수 있으며 파일 스토리지 유형, 액세스 권한, 암호화 방식, 객체 태그, 메타데이터 등 정보를 설정할 수 있습니다.                                          |
| 파일 백업 | COSBrowser는 자동 백업 기능을 제공하며 백업을 활성화한 후, 이미지 라이브러리 파일을 특정 버킷에 자동 백업합니다.                                   |
| 파일 다운로드 | 버킷 파일을 App로 다운로드하거나 로컬 이미지 라이브러리에 저장할 수 있습니다.            |
| 일괄 작업 | 버킷의 파일을 일괄 업로드, 다운로드, 삭제, 복사, 이동할 수 있습니다.                        |
| 파일 미리보기 |  버킷의 이미지, 비디오, 오디오, 문서 등 다양한 형식의 파일을 미리보기 할 수 있습니다.                               |
| 파일 온라인 압축 해제| zip, tar, gz 세 종류의 압축 형식 파일을 온라인 압축 해제할 수 있습니다.                             |
| 파일 공유 | COSBrowser는 폴더 및 파일 공유 기능을 제공하여 단시간 내에 버킷 내의 데이터를 수집하거나 다른 사용자에게 공유할 수 있습니다.                             |
| 파일 이름 변경  | 버킷의 파일 이름을 변경할 수 있습니다.             |
| 파일 검색 |  퍼지 검색을 지원하며, 폴더, 이미지, 비디오, 문서, 오디오 등 유형별로 검색할 수 있습니다.   |
| 객체 정렬 또는 필터링 | 버킷 파일을 파일 이름, 크기, 수정 시간을 기준으로 정렬하고 스토리지 유형별로 필터링 할 수 있습니다.                 |
| 파일 권한 관리 | 파일 단위 액세스 권한 설정이 가능하며 해당 권한은 버킷의 액세스 권한보다 우선순위가 높습니다.                                           |
