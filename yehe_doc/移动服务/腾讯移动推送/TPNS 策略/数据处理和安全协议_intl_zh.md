# 1.  背景

本模块仅在您使用腾讯移动推送（简称“**功能**”）时适用。本模块包含在位于 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据处理和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，本模块应在该不一致的范围内适用。

# 2.  处理

我们将因本功能处理以下个人数据： 

| <b>个人信息</b>                                                 | <b>用途</b>                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 推送通知                                                     | 我们处理推送通知是为了根据您的偏好提供该通知。我们不负责推送通知中包含的内容，这些内容完全由您设置。 |
| 标记 ID（通过本功能为各设备创建）                            | 这是为了确保可向您选择的设备提供推送通知而   创建。          |
| 帐户/登录                                                    | 如果您选择将标记 ID 与特定帐户/登录详细信息（例如，微信号）绑定，我们将按照您的指示处理关联的帐户/登录详细信息。 |
| 技术信息（对于接收推送通知的设备而言，包括：操作系统版本、应用程序版本、位置、应用程序使用、系统语言、TPNS 版本、移动设备型号、品牌） | 如果您选择向特定用户群发送推送通知，我们将使用此信息向该群组提供推送通知。 |
| IDFA                                                         | 如果您选择将标记 ID 与 IDFA 绑定，我们将按照您的指示处理关联的 IDFA。 |
