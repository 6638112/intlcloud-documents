## 작업 시나리오
하나의 CVM 인스턴스에는 하나 혹은 여러 개의 보안 그룹을 바인딩할 수 있습니다. CVM 인스턴스에 여러 개의 보안 그룹을 바인딩할 경우, 보안 그룹은 우선 순위(1, 2 등의 숫자)에 따라 차례대로 실행됩니다. 다음 조작을 통해 보안 그룹의 우선 순위를 조정할 수 있습니다.

## 전제 조건
CVM 인스턴스에 두 개 이상의 보안 그룹이 추가되어 있어야 합니다.

## 작업 순서
1. [CVM 콘솔](https://console.cloud.tencent.com/cvm/index)에 로그인합니다.
2. 인스턴스 관리 페이지에서 CVM 인스턴스 ID를 클릭해 상세 페이지에 진입합니다.
3. [Security Group] 항목 탭을 선택해 보안 그룹 관리 페이지에 진입합니다.
4. 오른쪽의 "Bound to security group" 모듈에서 [sort]를 클릭하고, 보안 그룹 오른쪽의 <img src="https://main.qcloudimg.com/raw/15a91d68b0396cb89bfd75a578b42130.png" style="margin:-3px 0px;width:20px"> 선택 후 위에서 아래로 드래그해 보안 그룹의 우선 순위를 조정합니다. 위쪽에 위치할수록 보안 그룹의 우선 순위가 높습니다.
5. 조정을 완료하면 [Save]를 클릭합니다.
