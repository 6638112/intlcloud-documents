
## ユースケース
Tencent Cloud CVMは、ストレージハードウェアメディアの調整をサポートしています。ストレージハードウェアメディアを調整することにより、異なる業務のストレージ要件に柔軟に対応できます。
Tencent Cloudは、[CBS](https://intl.cloud.tencent.com/document/product/213/4953) および[ローカルディスク](https://intl.cloud.tencent.com/document/product/213/5798)の2種類のストレージを提供します。現在、ローカルディスクをCBSに調整できます。 本ドキュメントでは、ディスクメディアの調整方法と関連する注意事項を説明します。
ローカルディスクCVMには次の制限があります：
- ホストリソースの影響を受けるため、自ら設定を調整することができません。
- スナップショットや作成の加速などの機能はサポートされていません。
- データの信頼性が低い。
- 長時間にわたるホスト障害の影響を受けます。

ローカルディスクCVMの制限を解除したい場合は、アカウントの既存のローカルディスクCVMをCBS CVMに変更して対応可能です。

## 前提条件
- **CVMの状態**
 この操作は、CVMが【終了】状態の時のみ実行できます。まずは、CVMをシャットダウンしてください。
- **CVMの制限**
 - ビッドCVMは、ローカルディスクをCBSに調整できません。
 - 専用型CVMは、ローカルディスクをCBSに調整できません。
 - ビッグデータモデルD1、D2と高 I/O モデルI3、I4などのCVMは、ローカルディスクをCBSに調整できません。
 - ベアメタルインスタンスは、ローカルディスクをCBSに調整できません。
- **CVMの設定**
 - CVMのシステムディスクまたはデータディスクには、少なくとも1つの「**通常ローカルディスク**」または「**SSDローカルディスク**」がある場合のみ、ローカルディスクをCBSに調整できます。
 - CVMが配置されているアベイラビリティーゾーンで使用できるCBSタイプはあり、尚且つ現在のインスタンスのローカルディスクサイズはCBSのサポート範囲内である場合のみ、ローカルディスクをCBSに調整できます。
 - CVMのシステムディスクとデータディスクが両方ローカルディスクの場合、ローカルディスクをCBSに調整すると、CVMの**すべて**のローカルディスクがCBSに調整されます。部分的な調整はサポートされていません。 各ディスクは、調整対象のCBSタイプを個別に設定できます。
 つまり、完全なローカルディスクのCVMでディスクメディア調整を実行する場合、システムディスクのみCBSに調整することまたはデータディスクのみCBSに調整することはサポートされていません。完全に調整する必要があります。
 - ディスクメディアを調整しても、ディスクのサイズは変わりません。 調整が完了したら、[CBSのスケールアウト](https://intl.cloud.tencent.com/document/product/362/5747) を使用してディスクのサイズを調整できます。
 - ローカルディスクをCBSに調整する操作では、CVMのライフサイクル、インスタンスID、プライベート/パブリックIP、ディスクデバイス名及びマウントポイントは変更されません。

<span id="LocalDiskNotice"></span>
## 注意事項

- ローカルディスクをCBSに調整する操作では、データコピー方式を利用してローカルディスク上のデータをCBSシステムにコピーします。これは、データ量とデータ転送速度によって制限されます。 この操作は完了するまでに時間がかかる場合があり、しばらくお待ちください。
- ローカルディスクをCBSに調整することのみをサポートします。 ローカルディスクがCBSに調整されている場合、CBSをローカルディスクに戻すことはできません。
- **調整操作が完了したら、スタートアップしてCVMにログインし、データの整合性を確認することをお勧めします**。

## 操作手順
1. [CVMコンソール](https://console.cloud.tencent.com/cvm)にログインし、CVMのインスタンスリストに入ります。
>? CVMが「シャットダウン済み」状態の場合、[手順3](#step3)を直接実行します。
2. （オプション）調整対象のCVM行で、【その他】>【インスタンス状態】>【シャットダウン】を選択して、シャットダウン操作を実行します。
<span id="step3"></span>
3. 調整対象のCVM行で、【その他】>【リソース調整】>【ディスク媒体の調整】をクリックします。
4. 【ディスクメディアの調整】ダイアログボックスで、システムディスク/データディスクに調整したいターゲットCBSタイプを選択し、同意の説明をチェックし、【すぐに変換する】をクリックします。
5. 情報を確認し、支払いが必要な注文を完了して、操作が完了するまでお待ちください。
