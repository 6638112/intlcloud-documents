## CVM 인스턴스 구매 계정 제한

- Tencent Cloud 계정이 필요합니다. 회원가입 가이드는 [Tencent Cloud 회원가입](https://intl.cloud.tencent.com/document/product/378/17985)을 참조 바랍니다.
- 실명 인증을 진행해야 합니다. 실명 인증 가이드는 [실명 인증 가이드](https://intl.cloud.tencent.com/document/product/378/3629)를 참조 바랍니다.
- 종량제 CVM을 생성할 때 시스템이 CVM의 1시간의 요금을 동결하므로, 주문서를 제출하기 전에 계정의 잔액이 충분한지 확인 바랍니다.

## CVM 인스턴스의 사용 제한

- 가상화 소프트웨어 설치와 가상화의 재진행(VMware 또는 Hyper-V를 사용하는 경우)을 한시적으로 지원하지 않습니다.
- 사운드 카드 애플리케이션을 한시적으로 지원하지 않으므로, 외장 하드웨어 디바이스(마운트된 ISO 파일, U 디스크, 외장 하드디스크, USB 보안 키 등)를 직접 로딩해야 합니다.
- 공용망 게이트웨이는 현재 Linux 시스템만 지원합니다.

## CVM 인스턴스의 구매 제한

- 각 가용존에서 한 사용자가 구매 가능한 종량제 CVM 인스턴스의 **총 수량**은 30대입니다.
- 자세한 내용은 [CVM 인스턴스 구매 제한](https://intl.cloud.tencent.com/document/product/213/2664)을 참조 바랍니다.


## 이미지 관련 제한

- 공용 이미지는 현재 사용 제한이 없습니다.
- 사용자 정의 이미지: 각 리전마다 최대 10개의 사용자 정의 이미지를 지원합니다.
- 공유 이미지: 사용자 정의 이미지는 각각 최대 50명의 Tencent cloud 사용자와 공유할 수 있으며, 동일한 리전 내의 계정에만 공유 가능합니다.
- 자세한 내용은 [이미지 유형 제한](https://intl.cloud.tencent.com/document/product/213/4941)을 참조 바랍니다.

> 2019년 9월 18일 0시 이전에 구매한 CVM은 이 제한을 받지 않으며, 해당 CVM에 바인딩 가능한 공인 IP 수량은 서버가 지원하는 [개인 IP 수량](https://intl.cloud.tencent.com/document/product/576/18527)과 같습니다.

## ENI 관련 제한

## 보안 그룹 관련 제한

- 보안 그룹은 리전마다 다르며, CVM은 같은 리전 내의 보안 그룹에만 바인딩할 수 있습니다.
- 보안 그룹은 [네트워크 시나리오](https://intl.cloud.tencent.com/document/product/213/5227)의 모든 CVM 인스턴스에 사용하기 적합합니다.
- 각 사용자는 각 리전에서 항목별 최대 50개의 보안 그룹을 설정할 수 있습니다.
- 하나의 보안 그룹에 인바운드 혹은 아웃바운드 액세스 정책을 각각 최대 100건씩 설정할 수 있습니다.
- 하나의 CVM에 여러 보안 그룹을 추가할 수 있으며, 하나의 보안 그룹을 여러 CVM에 동시 연결할 수 있습니다.
- **기본 네트워크**의 CVM에 바인딩된 보안 그룹은 Tencent Cloud의 관계형 데이터베이스(TencentDB), 탄력적 캐시(Redis 및 Memcached)의 데이터 패킷을 **필터링할 수 없습니다.** 이러한 인스턴스의 트래픽은 iptables를 사용해 필터링할 수 있습니다.
