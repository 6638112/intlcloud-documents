## 작업 시나리오

본 문서는 공인 IP 주소를 변경하는 방법에 대해 안내합니다.

## 주의 사항
- 단일 리전의 단일 계정일 경우, 변경 횟수는 하루에 3회를 초과할 수 없습니다.
- 단일 인스턴스는 **공인 IP를 1회만 변경**할 수 있습니다.
- **변경 후 기존 공인 IP는 릴리스 됩니다.**

## 전제 조건

[CVM 콘솔](https://console.cloud.tencent.com/cvm/index)에 로그인한 상태

## 작업 순서

1. 아래 이미지와 같이 인스턴스 관리 페이지에서 IP를 변경할 CVM 행을 선택하고 [More]>[IP/ENI]>[Change Public IP]를 클릭합니다.
![](https://main.qcloudimg.com/raw/4225bd84b4cf4d5346ddc77a1d7fd0df.png)
2. 팝업된 “IP 변경” 알림 상자에서 [OK]를 클릭하면 변경이 완료됩니다.
