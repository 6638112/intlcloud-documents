## ユースケース


VNC ログインはTencent Cloudがユーザーのために提供している、Webブラウザーのリモートコネクティングにより CVM に接続する方法です。リモートログインクライアントをインストールしていない、あるいはクライアントのリモートログインが利用できない場合は、ユーザーは VNC でログインしてCVMに接続し、CVMの状態を確認する上、CVM アカウントを通して基本的な CVM 管理操作を行うことが可能です。

## ローカルOSに適する

Windows、Linux と Mac OS 

## 使用制限

- VNC によってログインされた CVM は、コピーアンドペースト機能、中国語入力方法、およびファイルのアップロードとダウンロードをサポートしていません。
- VNC で CVM にログインする場合、メジャーのブラウザーを使用する必要があります、例えば、Chrome、Firefox、IE 10以降のバージョンなど。
- VNC ログインは専用端末です。すなわち同時にVNCでログインできるユーザーは1人だけです。

## 前提条件
インスタンスにログインするための管理者アカウント及びパスワード(またはキー)は取得しました。
- システムのデフォルトパスワードを利用して、インスタンスにログインする場合、[サイト内メッセージ](https://console.cloud.tencent.com/message)にアクセスして取得してください。
- パスワードを忘れた場合、[インスタンスパスワードをリセット](http://intl.cloud.tencent.com/document/product/213/16566)をしてください。

## 操作手順

1. [CVMコンソール](https://console.cloud.tencent.com/cvm/index)にログインします。
2. インスタンスの管理画面で、ログインしたいLinux CVMを選択して、【ログイン】をクリックします。下記画像に示すように：
![](https://main.qcloudimg.com/raw/a4cc736f2dc7f13bf39756b8e39532d4.png)
3. ポップアップした【Linuxインスタンスのログイン】ウインドウで、【その他の方式(VNC）】を選択して、【今すぐログイン】をクリックします。下図に示すように：
![](https://main.qcloudimg.com/raw/1bd4877abc15d06adb8c54fc7ed1318e.png)
4. ポップアップしたダイアログボックスで、ユーザー名とパスワードを入力してログインできます。下記画像に示すように：
![](https://main.qcloudimg.com/raw/03a8492f66e8342221858709b6068669.png)
