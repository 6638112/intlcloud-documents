
## 작업 시나리오
Tencent Cloud CVM은 스토리지 하드웨어 미디어에 대한 조정을 지원합니다. 사용자는 스토리지 하드웨어 미디어에 대한 조정을 통해 서로 다른 비즈니스의 스토리지에 대한 요구사항을 효율적으로 대응할 수 있습니다.
Tencent Cloud는 [CBS](https://intl.cloud.tencent.com/document/product/213/4953)와 [로컬 디스크](https://intl.cloud.tencent.com/document/product/213/5798) 두 가지 블록 스토리지 유형을 제공하며 현재 로컬 디스크를 CBS로 조정하는 것을 지원합니다. 사용자는 본 문서를 통해 디스크 미디어를 조정하는 작업 방법 및 관련 주의 사항을 파악할 수 있습니다.
로컬 디스크 CVM은 다음 제한이 있습니다.
- 호스트 리소스의 영향을 받아 자주적으로 설정을 조정할 수 없습니다.
- 스냅샷, 생성 가속 등 기능을 지원하지 않습니다.
- 데이터 신뢰성이 낮습니다.
- 호스트 장애의 영향을 받아 시간이 깁니다.

로컬 디스크 CVM에 존재하는 제한을 해제해야 할 경우, 사용자는 본인 계정 아래에 존재하는 로컬 디스크 CVM을 CBS CVM으로 변경할 수 있습니다.

## 전제 조건
- **CVM 상태**
 본 작업은 CVM이 [종료] 상태일 때에만 수행할 수 있으니 우선 CVM을 종료하십시오.
- **CVM 제한**
 - 입찰 CVM은 로컬 디스크를 CBS로 조정하는 것을 지원하지 않습니다.
 - 전용 CVM은 로컬 디스크를 CBS로 조정하는 것을 지원하지 않습니다.
 - 빅 데이터 모델 D1, D2와 하이 I/O 모델 I3, I4 등 CVM 유형은 로컬 디스크를 CBS로 조정하는 것을 지원하지 않습니다.
 - 베어 메탈 인스턴스는 로컬 디스크를 CBS로 조정하는 것을 지원하지 않습니다.
- **CVM 설정**
 - CVM의 시스템 디스크 또는 데이터 디스크에 적어도 한 블록의 “**일반 로컬 디스크**” 또는 “**SSD 로컬 디스크**”가 있을 때만 로컬 디스크를 CBS로 조정하는 것을 지원합니다.
 - CVM이 소재한 가용존에 사용 가능한 CBS 유형이 있고 현재 인스턴스의 로컬 디스크 크기가 CBS의 지원 범위 내에 있을 때만 로컬 디스크를 CBS로 조정하는 것을 지원합니다.
 - 만일 CVM의 시스템 디스크와 데이터 디스크가 모두 로컬 디스크이면 로컬 디스크를 CBS로 조정할 때 CVM의 **모든**로컬 디스크를 전부 CBS로 조정하고 부분 조정을 지원하지 않으며 각 블록의 디스크마다 조정이 필요한 목표 CBS 유형을 단독 설정할 수 있습니다.
 즉 1대의 풀 로컬 디스크의 CVM에 대해 디스크 미디어를 조정할 때 시스템 디스크만 CBS로 조정하거나 데이터 디스크만 CBS로 조정하는 것을 지원하지 않으며 반드시 전부 조정해야 합니다.
 - 디스크 미디어의 조정 작업은 디스크의 용량을 변경하지 않습니다. 조정 완료 후 사용자는 [CBS 확장](https://intl.cloud.tencent.com/document/product/362/5747)을 통해 디스크 용량을 조정할 수 있습니다.
 - 로컬 디스크를 CBS로 조정하는 작업은 CVM의 라이프사이클, 인스턴스 ID, 내부/외부 네트워크 IP, 디스크 디바이스 이름 및 마운트 포인트를 변경하지 않습니다.

<span id="LocalDiskNotice"></span>
## 주의사항

- 로컬 디스크를 CBS로 조정하는 작업은 데이터 카피의 방식으로 로컬 디스크 상의 데이터를 CBS 시스템으로 카피하며 데이터량의 크기 및 데이터 전송 속도의 제약을 받습니다. 이 작업은 긴 시간이 소요되므로 완료될 때까지 기다려주십시오.
- 로컬 디스크를 CBS로 조정하는 것만 지원합니다. 로컬 디스크를 이미 CBS로 조정하였으면 해당 CBS를 다시 로컬 디스크로 조정하는 것은 지원하지 않습니다.
- **조정 작업이 완료된 후 컴퓨터를 시작하고 CVM에 로그인하여 데이터의 완전성을 확인할 것을 권장합니다**.

## 작업 순서
1. [CVM 콘솔](https://console.cloud.tencent.com/cvm)에 로그인하여 CVM 인스턴스 리스트에 진입합니다.
>? CVM이 “종료” 상태이면 바로 [3단계](#step3)를 실행합니다.
2. (선택 가능) 조정이 필요한 CVM 줄에서 [More]>[Instance Status]>[Shutdown]를 선택하여 종료 작업을 진행합니다.
<span id="step3"></span>
3. 조정이 필요한 CVM 줄에서 [More]>[Resource Adjustment]>[Change Disk Media Type]을 클릭합니다.
4. [Change Disk Media Type] 대화창에서 시스템 디스크/데이터 디스크의 조정이 필요한 목표 CBS 유형을 선택하여 설명 동의를 체크하고 [즉시 전환]을 클릭합니다.
5. 정보를 확인하고 결제에 필요한 주문서를 완성한 후 작업 완료를 기다립니다.
