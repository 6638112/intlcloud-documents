## Overview

This document describes to create CVM instances from RI.

## Prerequisites
Your account already has RIs in the same region/availability zone as the CVM instance you want to create.

## Directions

1. Log in to the [CVM Console](https://console.cloud.tencent.com/cvm/instance/index?rid=1).
2. Click **Reserved Instance** on the left sidebar to enter the management page.
3. Select the target region, locate the reserved instance from which you want to create CVM instances, and click **Create Instance** to enter the CVM purchase page.

![image-20200909173349604](https://main.qcloudimg.com/raw/b334d2d61be1b407d8d796f86a98955e.png)

4. Complete instance configurations as prompted by the page.
The **Region**, **Availability Zone** and **Instance** fields are automatically filled. Complete the other configurations of the instance as needed. For more information, see [Creating Instances via CVM Purchase Page](https://intl.cloud.tencent.com/document/product/213/4855).
