##  Windows 回復モードとは

**Windows システムリカバリモード( Recovery )とは**、ユーザーがシステムを修復、バックアップ、または復元できる状態です。 Windows が自動修復機能を利用し、特定のシステムの問題を検出し、継続的な使用がシステムの損傷を引き起こると判断し、Windowsの起動を阻止して、システムリカバリモードに入ります。
システム回復オプションには、「スタートアップ修復」、「システムの復元」、「Windowsメモリ診断」などのいくつかのツールが含まれています。 これらのツールを使用して、問題の修復、データのバックアップ、およびシステムの復元などの操作を行うことができます。

ユーザーがCloud Virtual Machineにリモートでログインできなく、かつコンソールからCVMにログインするときに次の図が表示される場合は、 Windows CVMはリカバリモードに入ったことを意味します。
![](//mc.qcloudimg.com/static/img/e278c336a415066dcb8fc58333395ac3/image.png)

## リカバリーモードに入る原因
リカバリーモードに入る一般的な原因：

- **Windowsの実行中またはシャットダウン中に強制的に電源が切断されました。**これには、Windowsの実行中またはシャットダウン中にコンソールでCVMをシャットダウンすることが含まれます。不適切なシャットダウンにより、システムが重要なデータが失われる可能性があるため、Windowsはリカバリモードに入りました。
- **Windowsの更新中に電源が切断されます。**更新の一時停止により、重要なデータが失われた可能性があるため、Windowsはリカバリモードに入りました。
- **システムがトロイの木馬またはウイルスによって損壊されます。**
- **Windowsコアサービス のBUG 。**Windows がリスクを自動検出し、そしてリカバリーモードに入りました。
- **重要なデータが失われたか、システムが損壊されました。**ユーザーが誤操作でシステムファイルを破損したため、Windowsがリカバリモードに入りました。

## 予防措置
Tencent Cloudは下記の予防措置をユーザーに勧めます。

 - CVMをシャットダウンする時、コンソールに移動してWindows のシャットダウンプロセスを観察します。Tencent Cloudのソフトシャットダウンはタイムアウトメカニズムを利用し 、ソフトシャットダウンが実行された後、予定の時間経てもシステムはシャットダウンしなかった場合、障害を返します。シャットダウンプロセスが遅いまたWindowsUpdateの場合は、強制的にシャットダウンしないでください、シャットダウンが完了するまで待ちます。詳細については、 [シャットダウン失敗のユースケース](https://intl.cloud.tencent.com/document/product/213/4933/doc/product/213/2917#2.-.E5.85.B3.E6.9C.BA.E5.A4.B1.E8.B4.A5.E7.9A.84.E5.87.A0.E7.A7.8D.E5.9C.BA.E6.99.AF2) をご参照ください。
 - システムにトロイの木馬またはウイルスなど異常なプログラムとプロセスがあるかを確認します。
 - システム管理とウイルス対策ソフトウェアが正常に実行されるかを確認します。
 - タイムリーに Windows の更新プログラムを更新します。特に重要な更新プログラムとセキュリティ更新プログラム。
 - 定期的にシステムイベントログをチェックし、コアサービスに異常があるかどうかを確認します。

## ソリューション
 Windows がリカバリーモードには入ったら、起動を続行し、自動修復してみます。軽微な問題はWindows が自動修復できます。下記操作を実行します：

 1.  [CVMコンソール](https://console.cloud.tencent.com/cvm)からCVMにログインします。
 2. リカバリーモードのインターフェースが表示されたら、【Next】をクリックします。
	![](//mc.qcloudimg.com/static/img/94a1cf0f55d2c449a9d026bbbad5e4cd/image.png)
 3. システム回復オプションが表示されたら、【Next】をクリックし、デフォルトのソリューションを使用します。
 ![](//mc.qcloudimg.com/static/img/d178865f822d2146eb3bb58f1b851294/image.png)
 4. 【Restart】をクリックし、すぐに【F8】キーを押します。
 ![](//mc.qcloudimg.com/static/img/ab2fdd697015fcb7e53b287052086b65/image.png)
 5. 「Windowsを通常起動」を選択します。
 ![](https://main.qcloudimg.com/raw/c3c62a6d77a2fe41d0ad02899faa06ed.png)
 6. 起動できない場合、コンソールでシステムを再インストールしてください、詳細については、[コンソールを利用してシステムを再インストールする](https://intl.cloud.tencent.com/doc/product/213/4933#.E4.BD.BF.E7.94.A8.E6.8E.A7.E5.88.B6.E5.8F.B0.E9.87.8D.E8.A3.85.E7.B3.BB.E7.BB.9F3)をご参照ください。
