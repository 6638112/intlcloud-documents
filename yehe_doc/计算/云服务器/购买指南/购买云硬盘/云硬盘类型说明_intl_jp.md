Cloud Block Storage（CBS）は、高可用性、高信頼性、低コスト、カスタマイズ可能なネットワークブロックデバイスであり、CVMの独立かつ拡張可能なディスクとして利用できます。ブロックレベルのデータストレージを提供し、3レプリカの分散式メカニズムを用いてデータ信頼性を確保します。CBSは、**SSD CBS**、**高性能CBS**、と**普通 CBS**3種類のクラウドディスクを提供します。ディスク種類、性能、機能、価格はそれぞれ異なりますので、デプロイするアプリケーションの要求に応じて選択できます。

- **SSD CBS**：SSD CBSは、NVMe SSDストレージメディアに基づいて、3レプリカ分散式メカニズムを使用して、低遅延、高ランダムIOPS、高スループットI/O能力、及びデータ安全性が99.9999999%に達する高性能なストレージを提供します。SSD CBSは、I/O性能要求の高い業務に適します。
- **高性能CBS**：高性能CBSは、Tencent Cloudがリリースしたハイブリッドストレージ型です。CacheメカニズムによりSSDに近い高性能ストレージ能力を提供し、3レプリカ分散式を採用してデータの信頼性を確保します。高性能CBSは、高いデータ信頼性や、標準性能が要求される中小アプリケーションや、MySQL、SQL Server、PostgreSQLなどの中小規模のリレーショナルデータベースに適します。
- **普通 CBS**：普通 CBSは、Tencent Cloudが提供する前世代クラウドディスクタイプであり、I/O負荷が低く、データがあまりアクセスしない業務に適します。磁気記憶をストレージメディアとされ、3レプリカの分散式メカニズムを採用して高信頼性データストレージを実現します。

**SSD CBS**、**高性能CBS**と**普通 CBS**の性能指標は：

|               | SSD CBS                                                    | 高性能CBS                                              | 普通 CBS                                                   |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| ランダムIOPS   | 最大ランダムIOPS = 1800 + ストレージ容量（GB）× 30<br/>最大ランダムIOPSは26000を超えない | 最大ランダムIOPS = 1800 + ストレージ容量（GB）× 8<br>最大ランダムIOPSは6000を超えない | 10GB - 16000GBの仕様オプションと数百のランダムIOPS性能を提供する |
| スループット（MB/s） | 最大スループット = 120 + ストレージ容量（GB）× 0.2<br/>最大スループットは260 MB/sを超えない | 最大スループット = 100 + 存储容量（GB）× 0.15<br>最大スループットは150MB/sを超えない | 50 MB/sのI/Oスループット性能 をサポートする                           |
| 遅延           | <3ms                                                 | <4ms                                                 | -                                                         |

**SSD CBS**と**高性能CBS**の主な違いは、I/Oへの要求に現れています。

**SSD CBSは集中型I/Oの場面に適する**、例えば：
- 高性能、高デ信頼性：高負荷、コア業務システムに適用します。3つのデータ複製、完全なデータバックアップ、スナップショット、及び数秒以内にデータのリカバリ機能を提供します。
- 中大規模データベース：MySQL、Oracle、SQL Server、MongoDBなど、数百万行のテーブルを含む中大規模のリレーショナルデータベースをサポートします。
- 大規模 NoSQL：HBaseやCassandraといったNoSQLを用いる業務のストレージ性能要件を満たします。
- コア業務システム：高いデータ信頼性が要求される集中型I/Oのコア業務システムです。
- ビッグデータ分析：TBやPBレベルのデータに分散式処理機能を提供し、データ分析、データマイニング、ビジネスインテリジェンスなどの領域に適します。

**高性能CBS以下のような場面に応用される**：
- 中小規模データベースやWebサーバに適用し、長期安定したI/O性能を提供します。
- 企業のオフィス業務やログサービスなど、ストレージ容量とパフォーマンスとのバランスを要求する業務に適します。
- コア業務テストと開発結合テスト環境のI/O要求を満たします。
  

この3種類のCBSの価格はリージョンによって異なります。通常、高性能なSSD CBSが最も高価です。これら3種類のCBSの料金に関する詳細な情報は、[料金一覧](https://intl.cloud.tencent.com/document/product/362/2413) をご参照ください。
