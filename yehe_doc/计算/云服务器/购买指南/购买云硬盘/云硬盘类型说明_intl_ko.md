Cloud Block Storage(CBS)는 높은 가용성, 높은 신뢰성, 낮은 원가, 맞춤화된 네트워크 블록 디바이스로 Cloud Virtual Machine(CVM)의 독립형 확장 디스크로 사용됩니다. 또한 데이터 블록 레벨의 데이터 스토리지를 제공하고 3중 백업 분산형 메커니즘을 사용하여 CVM에 데이터 신뢰성을 보장합니다. 아래와 같이 **SSD CBS**, **고성능 CBS** 및 **일반 CBS** 등 3가지 CBS 유형을 제공하며, 디스크의 유형, 성능, 특징 및 가격이 다양하므로 설치한 애플리케이션 조건에 맞춰 선택할 수 있습니다.

- **SSD CBS**: SSD CBS는 풀 NVMe SSD 스토리지 미디어를 기반으로 3중 백업 분산형 메커니즘을 사용하여 저지연, 고랜덤 IOPS, 높은 처리량의 I/O 능력 및 99.9999999%에 달하는 데이터 보안성을 가진 고성능 스토리지를 제공합니다. SSD CBS는 I/O 성능에 대한 요구가 높은 시나리오에 적합합니다.
- **프리미엄 CBS**: 프리미엄 CBS는 Tencent Cloud가 선보인 하이브리드 스토리지 유형으로 Cache 메커니즘을 통해 솔리드 스테이트 스토리지에 가까운 고성능 스토리지를 제공하는 동시에, 3중 백업 분산형 메커니즘을 사용하여 데이터의 신뢰성을 보장합니다. 프리미엄 CBS는 높은 데이터 신뢰성 및 일반적인 중도 성능을 요구하는 중소형 애플리케이션과 MySQL, SQL Server, PostgreSQL 등 관계형 중소 데이터베이스 애플리케이션의 시나리오에 적합합니다.
- **일반 CBS**: 일반 CBS는 Tencent Cloud가 제공하는 이전 세대 CBS 유형으로, 데이터 액세스가 자주 발생하지 않는 I/O 저부하 비즈니스 시나리오에 적합합니다. 마그네틱 매체를 스토리지 미디어로 사용하였고 3중 백업 분산형 메커니즘으로 높은 신뢰성의 데이터 스토리지를 구현합니다.

**SSD CBS**, **프리미엄 CBS** 및 **일반 CBS**의 성능 지표는 각각 다음과 같습니다.

|               | SSD CBS                                                    | 프리미엄 CBS                                                 | 일반 CBS                                                   |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 랜덤 IOPS       | 최대 랜덤 IOPS = 1800 + 스토리지 용량(GB) × 30<br/> 및 최대 랜덤 IOPS 26000 미만 | 최대 랜덤 IOPS = 1800 + 스토리지 용량(GB) × 8<br> 및 최대 랜덤 IOPS 6000 미만| 10GB - 16000GB의 사양 선택 및 수백 대의 랜덤 IOPS 성능 제공 |
| 처리량(MB/s) | 최대 처리량 = 120 + 스토리지 용량(GB) × 0.2<br/> 및 최대 처리량 260MB/s 미만| 최대 처리량 = 100 + 스토리지 용량(GB) × 0.15<br> 및 최대 처리량 150MB/s 미만| 50MB/s의 I/O 처리 성능 지원                            |
| 딜레이 시간           | <3ms                                                 | <4ms                                                 | -                                                         |

**SSD CBS**와**프리미엄 CBS**는 주로 I/O의 요구를 기준으로 구분합니다.

**SSD CBS는 집약형 I/O 시나리오에 더 적합합니다**. 예시:
- 높은 성능, 높은 데이터 신뢰성: 고부하, 핵심 서비스 시스템에 적합합니다. 3부 데이터 이중화를 제공하며, 완벽한 데이터 백업, 스냅샷, 몇 초 내 데이터 복구 능력을 갖췄습니다.
- 중대형 데이터베이스: 백만 행의 테이블을 가진 MySQL, Oracle, SQL Server, MongoDB 등 관계형 중대 데이터베이스 애플리케이션에 사용할 수 있습니다.
- 대형 NoSQL: HBase, Cassandra 등 NoSQL 비즈니스의 스토리지 성능 요구를 충족할 수 있습니다.
- 핵심 서비스 시스템: 데이터 신뢰성에 대한 요구가 높은 I/O 집약형 등의 핵심 비즈니스 시스템.
- 빅 데이터 분석: TB, PB급 데이터에 대한 분산형 처리 능력을 제공하고 데이터 분석, 마이닝, BI 등 분야에 적합합니다.

**프리미엄 CBS는 주로 다음과 같은 데이터 시나리오에 응용됩니다**:
- 중소형 데이터베이스, Web 서버 등에 적합하고, 장기간의 안정적인 I/O 성능 출력을 제공합니다.
- 오피스 비즈니스, CLS(Cloud Log Service) 등 스토리지 용량과 성능에 대해 평균적인 요구 사항이 있는 기업 시나리오에 적합합니다.
- 핵심 서비스 테스트, 연동 테스트 환경 개발 시의 I/O 수요를 만족시킬 수 있습니다.
  

3가지 CBS의 요금은 지역에 따라 차이가 있으며, 전체적으로 프리미엄 SSD CBS의 가격이 가장 높습니다. 3가지 CBS의 자세한 요금은 CBS [가격 리스트](https://intl.cloud.tencent.com/document/product/362/2413)를 참조하시기 바랍니다.
