# 공용 네트워크 과금 방식

## 과금 개요

Tencent Cloud가 제공하는 네트워크 유형의 통신사는 모두 BGP 멀티 라인에 연동되어 있어 라인의 품질을 보장합니다.

현재는 공용 네트워크의 총 데이터 전송량(GB)에 따라 결산하는 트래픽 과금 기준의 네트워크 과금 방식만 제공됩니다.

### 과금 유형

- 현재 공용 네트워크는 모두 아웃바운드 트래픽에 따라 과금됩니다. 그중 아웃바운드 대역폭이란 CVM에서 공용 네트워크로 나가는 대역폭을 의미합니다. 예시: 사용자가 클라이언트를 사용하여 CVM 인스턴스 내부 리소스를 다운로드하는 경우.
- 급격한 트래픽 증가로 인해 높은 비용이 발생하지 않도록 대역폭 최댓값을 지정하여 제한할 수 있습니다. 최댓값을 초과하면 패킷이 손실되어 요금이 청구되지 않도록 기본 설정되어 있습니다.

### 공용 네트워크 가격

| 리전                                       | 가격(단위: 달러/GB) |
| ------------------------------------------ | --------------------- |
| 중국대륙(홍콩, 마카오, 대만 제외), 서울, 중국홍콩 | 0.12                  |
| 싱가포르                                     | 0.081                 |
| 프랑크푸르트, 토론토, 실리콘밸리                     | 0.077                 |
| 모스크바, 도쿄                               | 0.13                  |
| 버지니아주                                   | 0.075                 |
| 방콕, 뭄바이                                 | 0.1                   |
