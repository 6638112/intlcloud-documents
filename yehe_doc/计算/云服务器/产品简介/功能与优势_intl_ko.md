## 전면성
Tencent Cloud CVM은 광범위한 서비스를 제공합니다.
- **다중 리전 다중 가용존**: 중국대륙 리전에는 화남, 화동, 화북, 서남 등 총 4개의 리전을 커버하며, 중국 외 노드에는 동남아, 아시아 태평양, 북미, 미국 서부 및 유럽 등 총 5개 리전을 커버합니다. 딜레이 시간 단축을 위해 사용자와 가까운 리전에 애플리케이션을 배포합니다.
- **다중 모델 구성**:
 - **표준형**(중소형 Web 애플리케이션, 중소형 데이터베이스에 적합)
 - **메모리형**(대량의 메모리 작업, 쿼리 및 컴퓨팅이 필요한 애플리케이션에 적합)
 - **고IO형**(낮은 딜레이 시간, I/O 집약형 애플리케이션에 적합)
 - **컴퓨팅형**(대형 게임 서버, 광고 서비스 엔진, 고성능 컴퓨팅 및 기타 컴퓨팅 집약형 애플리케이션에 적합)
 - **빅 데이터형**(Hadoop 분산형 컴퓨팅, 대용량 로그 처리, 분산형 파일 시스템과 대형 데이터 웨어하우스 등 처리 집약형 워크로드 애플리케이션에 적합)
 - **이종형**(딥러닝, 과학 컴퓨팅, 비디오 코딩/디코딩 및 그래픽 워크스테이션 등 고성능 애플리케이션에 적합)
 - **배치형**(렌더링, 유전자 분석, 약학 등 단시간에 초대형 컴퓨팅 노드의 사용이 빈번한 컴퓨팅 집약형 애플리케이션에 적합)

## 유연성
업계 최고의 엘라스틱 클라우드 서버 관리 플랫폼 구축에 주력하며 다음과 같은 성능을 제공합니다.
- **하드웨어 구성: **CBS의 CVM에 기반하여 하드웨어 구성을 **즉시 업/다운그레이드**합니다.
- **디스크 변경: **CBS의 CVM에 기반하여 디스크를 **즉시 확장**합니다.
- **네트워크 대역폭: **CVM에서 대역폭을 **즉시 업/다운그레이드**합니다.
- **운영 체제: **중국대륙 지역의 CVM은 Windows와 Linux 시스템을 수시로 변경할 수 있으며, 다른 지역은 이러한 기능을 지원하지 않습니다.
- **EIP: **각종 네트워크 환경에서의 호스트 바인딩을 지원합니다.
- **이미지 종류**: 공용 이미지(다양한 Linux 및 Windows 운영 체제 유형), 사용자 정의 이미지(사용자가 이미지 생성 기능을 통해 생성한 이미지)를 지원하며, 리전 간의 이미지 변경 및 복제를 지원합니다.
- **사용자 정의 네트워크 아키텍처**: Virtual Private Cloud(VPC)는 사용자에게 독립적인 네트워크 용량, 사용자 정의 IP 대역 구분, IP 주소 및 사용자 정의 라우팅 정책 등을 제공합니다. 포트 수준의 인/아웃 액세스 제어를 제공하여 전면적인 네트워크 로직 격리를 구현합니다.

## 신뢰성

업계에서 가장 신뢰도 높은 클라우드 서버(CVM) 구축에 주력합니다.
- **CVM 신뢰성: **호스트 서비스 가용성은 99.95% 달하고, 데이터 신뢰성은 99.9999999%에 달합니다. 다운 시 마이그레이션 감지 불가, 데이터 스냅샷, 자동 알람 등의 기능을 지원하여 서버의 보안을 보호합니다.
- **CBS 정책: **3중 백업 전용 스토리지 정책을 제공하여 단일 장애점을 제거하고, 데이터 신뢰성을 보장하여 데이터 손실 우려 없이 안심하고 데이터를 클라우드에 저장할 수 있습니다.
- **안정적인 네트워크 아키텍처: **성숙한 네트워크 가상화 기술 및 ENI 바인딩 기술로 네트워크의 고가용성을 보장합니다. T3+ 이상의 데이터센터에서 실행되며 실행 환경의 신뢰성을 보장하여 네트워크 가용성을 확보합니다.

## 빠른 속도

사용자 작업 또는 CVM 성능에 상관없이 빠르고 간편한 서비스 제공에 주력합니다.
- **간편하고 빠른 조작: **한 개, 수백 개 심지어 수천 개의 서버 인스턴스를 단 몇 분 만에 손쉽게 얻을 수 있으며, 원클릭으로 서비스를 빠른 구매, 설정, 관리 및 확장할 수 있습니다.
- **빠른 공용 네트워크의 품질**: 20라인이 넘는 BGP 공용 네트워크로 대다수의 네트워크 사업자를 커버함으로, 고객이 어느 ISP를 사용하든 동일하게 빠른 속도의 대역폭과 초 수준의 장애 전환을 이용할 수 있습니다.
- **빠른 내부 네트워크의 품질**: Tencent Cloud의 같은 리전 내 데이터 센터는 상호 연결되어 있으며, 하위 레이어에는 모두 10기가 또는 1기가 네트워크로 내부 네트워크 통신의 품질을 보장합니다.

## 보안

Tencent Cloud는 CVM의 보안을 위해 다양한 솔루션을 제공하며, 데이터 보안성을 위한 백업 및 롤백 메커니즘 또한 제공합니다.
- **다양한 CVM 원격 로그인 방식: **키 로그인, 비밀번호 로그인, VNC 로그인 등 다양한 로그인 방식을 제공합니다.
- **다양한 보안 서비스: **DDoS 보호, DNS 하이재킹 점검, 침입 감지, 취약점 스캐닝, 트로이 목마 검사, 로그인 보호 등의 보안 서비스를 제공하여 서버를 안전하게 보호합니다.
- **무료 클라우드 모니터링: **무료 클라우드 모니터링과 함께 다양한 실시간 알람을 지원합니다.
- **사용자 정의 액세스 제어: **보안 그룹, 네트워크 ACL 사용자 정의 호스트 및 네트워크 액세스 정책을 통해 효율적이고 자유롭게 다른 인스턴스에 다른 방화벽을 설정할 수 있습니다.

Tencent Cloud 보안 서비스는 다음과 같은 특징이 있습니다.
- **전방위의 보안**
CVM에 보안 진단(취약점 스캐닝, 트로이 목마 점검, 웹 사이트 백도어 검사, 포트 보안 검사 등) 및 보안 방어(Anti-DDoS, 침입 감지, 액세스 제어로 데이터 보안 및 사용자 프라이버시) 등 일체화된 보안 서비스를 제공합니다.
- **실시간 정기 분석 알람**
연중무휴 24시간 보안 서비스, 실시간 무료 알림 서비스로 취약점을 발견하는 즉시 알려드립니다.
- **간편한 무료 보안 서비스**
클라우드 서비스를 위해 값비싼 보안 디바이스를 구매할 필요 없이, 클라우드 서비스를 구매해 클라우드 보안 서비스를 무료로 이용할 수 있습니다. 원클릭 활성화 및 무중단 배포로 편리하고 간편합니다.
- **전문팀으로 신뢰성 보장**
클라우드 서비스는 다년간의 보안 서비스 경력을 가진 Tencent 보안 팀이 만들었습니다. 이와 같이 신뢰할 수 있는 클라우드 보안 서비스를 사용자에게 제공합니다.

## 손쉬운 사용

공식 인증된 다양한 애플리케이션 소프트웨어와 유지보수 툴로 더욱 편리하게 유지보수 하며, 관리 툴에 대해 더는 고민하지 않도록 도와드립니다.
- Tencent Cloud CVM은 Web 기반의 사용자 인터페이스(UI), 즉 콘솔을 제공하여 실제 기계처럼 CVM에 대한 인스턴스 시작, 구성 변경, 시스템 재설치 등 작업을 실행할 수 있습니다. Tencent Cloud 계정에 이미 가입했다면, 바로 [CVM 콘솔에 로그인](https://console.cloud.tencent.com/cvm)하여 CVM 작업을 시작할 수 있습니다.
- Tencent Cloud CVM에서는 API를 사용하여 사용자의 내부 모니터링, 운영 시스템과 CVM을 서로 결합하여 비즈니스 수요에 맞고 완전 자동화된 비즈니스 유지보수를 구현할 수 있습니다. 이러한 요청은 HTTP 또는 HTTPS 요청에 속하며, CVM API 작업에 관한 더 많은 정보는 [API 문서](https://intl.cloud.tencent.com/document/api/213/15688)를 참조 바랍니다.

- 필요에 따라 구매하고 합리적으로 소비할 수 있습니다. 하드웨어 리소스를 사전에 구매하거나 준비할 필요가 없어, 인프라 구축 비용을 효과적으로 절감할 수 있습니다.
