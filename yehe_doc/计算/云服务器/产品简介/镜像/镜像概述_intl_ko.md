## 미러 이미지란 무엇입니까?
Tencent Cloud 미러 이미지는 CVM에서 인스턴스 실행에 필요한 모든 정보를 제공합니다. 필요한 미러 이미지를 지정하면, 해당 미러 이미지에서 임의 수량의 인스턴스를 실행하거나, 필요에 따라 임의로 다수의 미러 이미지에서 인스턴스를 실행할 수 있습니다. 쉽게 말해, 미러 이미지는 CVM의 "설치 디스켓"이라고 볼 수 있습니다.

## 미러 이미지 종류
Tencent Cloud는 다음과 같은 미러 이미지를 제공합니다.

- **공용 이미지:** 모든 사용자가 사용할 수 있으며, 대부분의 주요 운영 체제에서 사용할 수 있습니다.
- **마켓플레이스 미러 이미지:** 모든 사용자가 사용할 수 있으며, 운영 체제 외에도 일부 특정 애플리케이션을 통합하였습니다.
- **사용자 정의 이미지:** 실행자와 공유 객체만 사용할 수 있습니다. 기존에 실행 중인 인스턴스에서 생성하거나 외부에서 가져올 수 있습니다.
- **공유 미러 이미지:** 다른 사용자가 공유한 미러 이미지로, 인스턴스 생성용으로만 사용할 수 있습니다.

더 많은 미러 이미지 유형은 [미러 이미지 유형](/doc/product/213/4941)을 참고 바랍니다.

## 미러 이미지의 응용
 - **특정 소프트웨어 환경 배포**
공유/사용자 정의 이미지, 마켓플레이스 미러 이미지를 사용하여 소프트웨어 환경을 빠르게 구축함으로써 환경 자체 설정, 소프트웨어 설치 등의 번거로움과 시간 소모를 줄일 수 있습니다. 사이트 구축, 애플리케이션 개발, 시각화 관리 등 다양한 고객의 커스텀 니즈를 만족시키며, CVM이 'Out of the box'로 인해 더 효율적이고 편리합니다.

 - **소프트웨어 환경 일괄 배포**
환경이 미리 배포되어 있는 CVM 인스턴스에 미러 이미지를 생성하고, CVM 인스턴스를 일괄 생성 시 해당 미러 이미지를 운영 체제로 사용할 수 있습니다. CVM 인스턴스 구축이 완료되면 소프트웨어 환경이 기존 CVM 인스턴스와 일치한 환경을 갖게 되어, 이로써 소프트웨어 환경 일괄 배포가 완료됩니다.

 - **서버 실행 환경 백업**
한 대의 CVM 인스턴스에 미러 이미지 백업 실행 환경을 만듭니다. 해당 CVM 인스턴스를 사용할 때, 소프트웨어 환경 오류로 인해 정상적으로 실행할 수 없을 경우 미러 이미지 복구를 사용할 수 있습니다.

## 미러 이미지 라이프사이클

다음은 사용자 정의 이미지의 라이프사이클을 정리한 이미지입니다. 신규 사용자 정의 이미지를 실행 혹은 불러온 후, 사용자는 이를 신규 인스턴스(기존의 공유 미러 이미지 또는 마켓플레이스 미러 이미지에서 인스턴스를 실행해도 됨) 실행에 사용할 수 있습니다. 사용자 정의 이미지를 동일 계정의 다른 리전에 복사하여, 해당 리전의 독립 미러 이미지로 사용할 수 있습니다. 또한, 사용자 정의 이미지를 다른 사용자와 공유할 수도 있습니다.
