사용자는 다음과 같은 특징에 따라 이미지를 선택할 수 있습니다.
- 위치([리전 및 가용존](https://intl.cloud.tencent.com/document/product/213/6091)을 참조하십시오)
- 운영 체제 유형
- 아키텍처(32비트 또는 64비트)

Tencent Cloud에서는 다양한 출처에 따라 공용 이미지, 사용자 정의 이미지, 공유 이미지 등의 유형을 제공합니다.

## 공용 이미지
<b>공용 이미지</b> Tencent Cloud에서 공식적으로 제공하는 이미지로서 기본 운영 체제와 더불어 Tencent Cloud가 제공하는 초기화 모듈을 포함하므로 모든 사용자가 사용할 수 있습니다.

공용 이미지 특징:
 - <b>운영 체제 유형:</b>자유롭게 선택(예: Linux 유형 시스템 또는 Windows 유형 시스템 기반)하며, 정기적으로 업데이트합니다.
 - <b>소프트웨어 지원:</b>Tencent Cloud에서 제공하는 소프트웨어 패키지(예: API 등)를 통합하고 여러 버전의 Java, MySQL, SQL Server, Python, Ruby, Tomcat 등 소프트웨어 및 완전한 권한을 지원합니다.
 - <b>보안:</b>제공된 운영 체제는 합법적이며, 모두 공식 정품 운영 체제를 사용합니다. Tencent Cloud의 전문 보안 및 유지보수팀의 엄격한 테스트를 거쳐 제작되며, Tencent Cloud 보안 모듈을 선택할 수 있습니다.
 - <b>제한:</b>사용 제한이 없습니다.
 - <b>요금:</b> 중국대륙 지역 외의 기타 리전은 Windows 유형 이미지 사용 시 License 요금이 부과되며, 나머지는 모두 무료입니다.
 - <b>기술 지원:</b>
    - Tencent Linux는 Tencent Cloud에서 지원 및 유지보수하며, 다음과 같은 기술 지원을 제공합니다.
    - 3rd party 이미지 관련 문제는 오픈소스 커뮤니티나 운영 체제 제조사에 문의하여 기술 지원을 받으십시오. Tencent Cloud는 문제 조사를 위한 기술적 협력을 제공할 것입니다.



## 사용자 정의 이미지
**사용자 정의 이미지**는 사용자가 이미지 생성 기능을 통해 생성하거나 이미지 가져오기 기능을 통해 가져온 이미지입니다. 생성자와 공유자만 사용할 수 있습니다.

사용자 정의 이미지 특징:
 - <b>응용 시나리오:</b>애플리케이션이 배포된 CVM 인스턴스의 이미지 생성을 통해 동일한 구성의 인스턴스를 더 많이, 더 빠르게 생성할 수 있습니다.
 - <b>기능 지원:</b>사용자가 자유롭게 생성, 복제, 공유 및 폐기할 수 있습니다.
 - <b>제한:</b>리전마다 최대 10개의 사용자 정의 이미지를 지원합니다.
 - <b>요금:</b>구체적인 가격은 인스턴스를 생성할 때 표시되는 가격을 기준으로 합니다. 사용자 정의 이미지의 리전 간 복제는 현재 무료입니다.

자세한 작업 방법 및 제한은 [사용자 정의 이미지 생성](https://intl.cloud.tencent.com/document/product/213/4942), [사용자 정의 이미지 복사](https://intl.cloud.tencent.com/document/product/213/4943), [사용자 정의 이미지 공유](https://intl.cloud.tencent.com/document/product/213/4944), [사용자 정의 의미지 공유 취소](https://intl.cloud.tencent.com/document/product/213/7148), [사용자 정의 이미지 가져오기](https://intl.cloud.tencent.com/document/product/213/4945)를 참조하십시오.

## 공유 이미지
<b>공유 이미지</b>는 다른 Tencent Cloud 사용자가 이미지 공유 기능을 통해 사용자 정의 이미지를 현재 사용자에게 공유한 이미지입니다.
공유된 이미지는 공유받은 사용자의 원래 이미지와 동일한 리전 내에 표시됩니다.

공유 이미지 특징:
 - <b>응용 시나리오:</b>다른 사용자가 CVM을 빠르게 생성하도록 도와줍니다.
 - <b>기능 지원:</b>이지미 공유는 CVM 생성에만 사용할 수 있으며 이름 수정, 복제 및 공유 등 다른 작업은 실행할 수 없습니다.
 - <b>보안:</b>공유된 이미지는 Tencent Cloud의 심사를 거치지 않으므로 보안 리스크가 발생할 수 있습니다. 신뢰하지 않는 이미지는 사용하지 마십시오.
 - <b>제한:</b>각 사용자 정의 이미지는 최대 50명의 Tencent Cloud 사용자와 공유할 수 있으며, 동일한 리전 내의 계정에만 공유할 수 있습니다.

자세한 작업 방법 및 제한은 [사용자 정의 이미지 공유](https://intl.cloud.tencent.com/document/product/213/4944), [사용자 정의 이미지 공유 취소](https://intl.cloud.tencent.com/document/product/213/7148)를 참조하십시오.
