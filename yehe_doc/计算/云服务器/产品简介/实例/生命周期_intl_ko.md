Tencent Cloud CVM 인스턴스의 라이프사이클이란, 인스턴스의 실행부터 릴리스까지 거치는 상태를 말합니다. Tencent Cloud 인스턴스가 실행되어 폐기되기까지 합리적인 관리를 통해 인스턴스의 응용 프로그램이 더욱 효율적으로 실행되도록 보장합니다.

## 인스턴스 시작
 - 인스턴스 작업을 시작한 후에는 인스턴스는 생성 중 상태로 진입합니다. 생성 중인 인스턴스는 지정된 [인스턴스 사양](https://intl.cloud.tencent.com/document/product/213/11518)에 따라 하드웨어 사양을 구성하며, 시스템이 시작 시 지정된 이미지를 사용하여 인스턴스를 시작합니다.
 - 인스턴스 생성 완료 후에는 실행 중 상태로 진입합니다. 실행 중 상태의 인스턴스가 정상적으로 연결 및 액세스 서비스를 시작합니다.

인스턴스 시작 정보에 대한 자세한 내용은 [인스턴스 생성](https://intl.cloud.tencent.com/document/product/213/4855), [Windows 인스턴스 로그인](https://intl.cloud.tencent.com/document/product/213/5435)과 [Linux 인스턴스 로그인](https://intl.cloud.tencent.com/document/product/213/5436)을 참조 바랍니다.

## 인스턴스 재시작
사용자가 Tencent Cloud 콘솔, Tencent Cloud API 중에서 임의로 선택하여 인스턴스를 재시작할 것을 권장하며, 인스턴스에서 운영 체제 재시작 명령어를 실행하지 않습니다.
 - 인스턴스 재시작을 실행한 후에는 재시작 중 상태로 진입합니다.
 - 인스턴스 재시작은 컴퓨터 재시작과 같다고 볼 수 있으며, 재시작 후 인스턴스는 공유 IP 주소, 개인 IP 주소 및 디스크의 모든 데이터를 유지합니다.
 - 인스턴스 재시작은 일반적으로 몇십 초에서 몇 분 정도가 소요되며, 구체적인 소요 시간은 인스턴스 구성에 따라 결정됩니다.

인스턴스 재시작 정보에 대한 자세한 내용은 [인스턴스 재시작](https://intl.cloud.tencent.com/document/product/213/4928)을 참조 바랍니다.

## 인스턴스 종료
사용자는 콘솔 또는 API 등 방식을 사용하여 인스턴스를 종료할 수 있습니다.
 - 인스턴스 종료는 컴퓨터 종료와 같다고 볼 수 있습니다.
 - 인스턴스 종료 후에는 외부에 서비스를 제공하지 않지만, 과금은 중단되지 않습니다.
 - 종료된 인스턴스는 계속해서 콘솔에 표시됩니다.
 - 종료는 하드웨어 구성 변경, 비밀번호 재설정 등과 같이 일부 구성 작업의 전제 조건입니다.
 - 종료 작업은 CVM의 공유 IP 주소, 개인 IP 주소 및 디스크의 모든 데이터를 변경하지 않습니다.
 
인스턴스 종료 정보에 대한 자세한 내용은 [인스턴스 종료](https://intl.cloud.tencent.com/document/product/213/4929)를 참조 바랍니다.

## 인스턴스 폐기 및 반환
사용자가 CVM 인스턴스를 사용하지 않을 경우 해당 인스턴스를 폐기 및 반환할 수 있습니다. Tencent Cloud 콘솔 또는 Tencent Cloud API 를 통해 구현할 수 있습니다.

- 수동 폐기: 연체되지 않은 종량제 유형의 인스턴스는 수동으로 폐기할 수 있습니다. 종량제 유형의 인스턴스는 휴지통에 최대 2시간 동안 보관된 다음 완전히 반환됩니다.
- 만료/연체 시의 자동 폐기 방식: 종량제 유형 인스턴스의 잔액이 0보다 작은 상태가 지속된 지 26시간이 지나면 자동 반환됩니다(초반 2시간 동안은 계속 과금되며, 이후 24시간 동안은 종료 및 과금 중지됩니다. 연체된 종량제 인스턴스는 휴지통에 보관되지 않으니 인스턴스 리스트에서 조회할 수 있습니다). 정해진 시간 내에 연장하면 계속 사용할 수 있습니다.

인스턴스를 폐기할 때 인스턴스의 시스템 디스크와 디스크 구매 시 지정한 데이터 디스크는 반환되지만 마운트한 CBS는 영향을 받지 않습니다.
인스턴스 폐기 정보에 대한 자세한 내용은 [인스턴스 폐기/반환](https://intl.cloud.tencent.com/document/product/213/4930)을 참조 바랍니다.
