参照用の公式Microsoftドキュメント：[Get started with Remote Desktop on Mac](https://docs.microsoft.com/en-us/windows-server/remote/remote-desktop-services/clients/remote-desktop-mac)。この公式ドキュメントは、Microsoft Remote Desktop for Mac を取得する2つの方法を提供します。 実際の状況に応じて、次の3つの方法でMac OSのリモートログインクライアントを取得できます：
 1. テスト版のリモートログインクライアント
Microsoftは、Mac用のテスト版クライアントを提供します。[Microsoft Remote Desktop for Mac](https://rink.hockeyapp.net/apps/5e0c144289a51fca2d3bfa39ce7f2b06/)にアクセスしてダウンロードできます。このテスト版のクライアントはMicrosoftによってメンテナンスされます。最初にこのバージョンのクライアントを使用することをお勧めします（Microsoftは2017年に公式サイトで提供されているダウンロードリンクを削除し、子会社のHockeyAppの画面からBeta版をリリースしました）。
> **注意事項：**
> 中国大陸のユーザーは、ネットワークの問題によりアクセスできないか、アクセスが遅くなる場合があります。
 2. Mac App Store からMicrosoftのリモートデスクトップクライアントをダウンロードします。
> **注意事項：**
> このクライアントは中国のユーザーには公開されていません。ダウンロードするには他の地域のAppleIDアカウントが必要です。
 3.  検索エンジンを利用して、自分で検索してダウンロードすることもできます。
