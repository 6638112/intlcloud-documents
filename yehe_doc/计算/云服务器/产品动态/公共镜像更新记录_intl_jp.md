## CoreOS
CoreOSコミュニティは、2020年5月26日以降はCoreOS Container Linuxがサポートまたは更新されないことを発表しました。そのため、Tencent Cloudは以下のように説明します。
- 2020年10月30日以降、Tencent Cloudが提供するCoreOS Container Linuxを使用してCVMを作成することはできなくなります。
- 2020年5月26日より、Tencent CloudはCoreOS Container Linuxへの技術サポートを停止しますが、CoreOS Container LinuxがすでにインストールされているCVMインスタンスは影響を受けずに使用を継続することができます。このOSのライフサイクルが終了したため、公式より安全に関する更新パッチが提供されていませんので、このイメージを使用しないことをお勧めします。
- Fedora CoreOSコミュニティは、CoreOS Container Linuxの代わりにFedora CoreOSを推奨し、Tencent Cloudも2020年9月にFedora CoreOSパブリックイメージをリリースする予定です。
