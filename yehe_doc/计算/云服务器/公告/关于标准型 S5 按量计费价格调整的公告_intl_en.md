
The pay-as-you-go pricing for Tencent Cloud Series 3 **Standard S5** CVMs is updated as follows:

## Updates

Since January 6, 2020, you can **purchase** a pay-as-you-go Standard S5 CVM **at a discount of 48%**.

### Description

- The pay-as-you-go Standard S5 CVM adopted the tiered pricing before, as detailed in the table below.
<table>
	<tr><th>Tier</th><th>Usage Duration</th><th>Discount</th></tr>
	<tr><td>Tier 1</td><td>≤ 96 hours</td><td>No discount</td></tr>
	<tr><td>Tier 2</td><td>> 96 hours and ≤ 360 hours</td><td>50% off the list price</td></tr>
	<tr><td>Tier 3</td><td>> hours</td><td>66% off the list price</td></tr>
</table>
For more information about the earlier tiered pricing, see<a href="https://intl.cloud.tencent.com/document/product/213/2176">Price of CVM Instance</a>.
- Since January 6, 2020, you can purchase a pay-as-you-go Standard S5 CVM **at a discount of 48%**, regardless of the usage duration.

For more information about the billing options, see [Instance Billing Modes](https://intl.cloud.tencent.com/document/product/213/2180). If you want to query the latest price of a pay-as-you-go CVM, use the [Price Calculator](https://buy.cloud.tencent.com/calculator/cvm).
