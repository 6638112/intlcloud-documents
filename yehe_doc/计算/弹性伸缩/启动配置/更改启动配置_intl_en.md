Launch configurations are not editable, but you can change the launch configuration associated with a scaling group. After an application upgrade or data update, you can change the launch configuration by performing the following steps:

1. Create a new launch configuration by referring to [Creating Launch Configuration](https://intl.cloud.tencent.com/document/product/377/3579).
2. Go to the **[Scaling Groups](https://console.cloud.tencent.com/autoscaling/group)** page and select the ID of the scaling group to configure to go the details page of the scaling group.
3. Click **Edit** in the top right corner of the **Basic Information** module to update launch configuration items. See the figure below:
![](https://main.qcloudimg.com/raw/dcf76968439d728860e23c70d1fc63f6.png)
4. After editing, click **Save**.
