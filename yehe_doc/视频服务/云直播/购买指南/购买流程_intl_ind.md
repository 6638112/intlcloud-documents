Cloud Streaming Services (CSS) adalah layanan dengan pembayaran sesuai pemakaian dan siap digunakan dengan melakukan beberapa langkah saja. Untuk mengaktifkan layanan CSS, Anda hanya perlu mendaftar untuk membuat akun Tencent Cloud.
 
## Membuat Akun Tencent Cloud

Jika Anda sudah memiliki akun Tencent Cloud, lewati langkah ini.
Jika Anda belum memiliki akun Tencent Cloud, silakan ikuti langkah-langkah di [Sign up for Tencent Cloud (Membuat Akun Tencent Cloud)](https://intl.cloud.tencent.com/document/product/378/17985) untuk membuat akun Tencent Cloud.
Anda dapat mendaftarkan alamat email sebagai akun Tencent Cloud Anda.

## Memverifikasi Identitas

Jika Anda sudah menyelesaikan verifikasi identitas, lewati langkah ini.
Jika Anda belum menyelesaikannya, silakan ikuti [Identity Verification Guide (Panduan Verifikasi Identitas)](https://intl.cloud.tencent.com/document/product/378/3629) untuk memverifikasi identitas Anda.
Setelah identitas pribadi atau organisasi Anda terverifikasi, Anda dapat mengaktifkan layanan CSS.

## Mengaktifkan CSS

Buka halaman [konsol CSS](https://console.cloud.tencent.com/live), pilih [Agree to Tencent Cloud Service Agreement (Setuju dengan Perjanjian Layanan Tencent Cloud)](https://intl.cloud.tencent.com/document/product/301/9248), lalu klik **Apply for Activation** (Kirim Permintaan Aktivasi) untuk mengaktifkan layanan CSS.
