릴레이용 로컬 모드는 CSS의 부가 기능입니다. 로컬 모드를 활성화하면 릴레이 작업의 MP4 파일이 릴레이되기 전에 로컬 노드에 캐시되어 더 매끄럽고 안정적인 재생을 보장합니다.

## 주의 사항
로컬 모드를 사용하면 부가 기능 요금 외에도 [릴레이 요금](https://www.tencentcloud.com/document/product/267/41059)이 발생합니다.

## 가격

| 과금 유형     | 가격(USD/과금 단위) |
| ------------ | --------------------- |
| 과금 단위 단가 | 0.01515               |

**과금 대상 시간과 실제 시간의 비율:**

| 부가 기능                   | 비율<br>과금 대상 시간(과금 단위) : 실제 시간(분) |
| -------------------------- | ---------------------------------------------------------- |
| 로컬 릴레이 모드           | 0.02 : 1                                                   |

## 과금 안내
- 과금 항목: 부가 기능 기본 요금.
- 과금 방식: 종량제 후불.
- 과금 주기: 일별 과금으로, 매일 발생한 부가 서비스 요금은 다음 날 계정에서 차감됩니다. 실제 요금 공제 및 청구서 생성 시간은 청구서를 참고하십시오.

## 계산 공식
- 과금 대상 시간 = 실제 시간 × 비율.
- 부가 서비스 요금 = 부가 서비스 기본 요금 × 과금 대상 시간.

## 과금 예시

2022년 11월 23일에 릴레이 작업에 대해 로컬 모드를 활성화하였고, 작업 시간은 로컬 파일 60분을 포함하여 100분이었다고 가정할 때, 11월 24일에 지불해야 하는 요금은 다음과 같습니다.
- 릴레이 요금: `0.00032(USD/분) x 100(분) = 0.032(USD)`
- 부가 기능 요금: `0.01515(USD/과금 단위) x 0.02(과금 단위/분) × 60(분) =0.01818(USD)`
