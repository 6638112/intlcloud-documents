릴레이 서비스는 콘텐츠 풀링 및 푸시 기능을 제공합니다. 라이브 방송 푸시 스트리밍 없이 빠르게 기존의 비디오/라이브 방송을 빠르게 풀링하여 타깃 주소로 푸시할 수 있습니다. 릴레이 서비스 요금에는 [릴레이 작업 시간 요금](#time), [타사 릴레이 요금](#third_part) 및 [부가 기능 요금](#value)의 세 가지가 포함됩니다.

## 참고 사항
- 릴레이 기능은 **2021년 7월 1일 00:00부터 정식으로 과금되며**, 2021년 7년 1일 00:00 이후 릴레이 작업으로 인한 시간 소모 발생 시 요금이 발생합니다(작업 생성 시간 구분하지 않음).
- 기존 소스에서 데이터를 가져오면 재생/다운로드 요금이 발생합니다. Tencent Cloud CSS, VOD, COS에서 가져오면 해당 제품의 과금 규정이 적용됩니다.

## 릴레이 작업 시간
### 과금 가격
작업 시간(분)에 따라 과금됩니다.

| 과금 유형         | 가격(USD/분) |
| ---------------- | ----------------- |
| 릴레이 작업 시간 | 0.00032           |

###  과금 설명
- 과금 항목: 릴레이 작업 시간.
- 과금 방식: 종량제 후불.
- 과금 주기: 일 단위로 과금됩니다. 매일 오전 10시에 릴레이 일 결산 청구서가 발행되며(자세한 사항은 청구서 시간을 기준으로 함) 전일 발생한 요금이 차감됩니다.
- 과금 규정: 현재 실행되고 있는 릴레이 작업이 일시 중지되거나 만료된 작업의 경우 시간이 소모되지 않으며, 작업을 복구하여 실행한 이후부터 계속해서 작업 시간이 계산됩니다.

###  과금 예시
사용자가 2022년 10월 01일에 릴레이 기능을 총 100분 동안 사용한 경우, 2022년 10월 02일 다음과 같이 청구서가 생성됩니다.
`0.00032(USD/분) x 100(분) = 0.032(USD)`


>! 비정상적인 소스 콘텐츠로 인해 풀링이 실패할 경우 지정된 종료 시간까지 릴레이 작업이 중지되지 않고 **해당 작업 기간이 청구됩니다**.

## 타사 릴레이 대역폭
>! 
>- 릴레이 기능은 현재 작업을 생성하는 Tencent Cloud 계정의 CSS 푸시 스트림 주소로만 푸시할 수 있으며, 해당 시나리오에서는 타사 릴레이 요금이 발생하지 않습니다.
>- Tencent Cloud의 CSS URL이 아닌 타사 주소로 릴레이하는 경우 타사 릴레이 요금이 발생합니다.

### 과금 가격
타사 릴레이 요금은 각 청구 기간에 릴레이에 사용되는 최고 대역폭(Mbps)을 기준으로 합니다. 가격은 스트림이 릴레이되는 리전에 따라 다릅니다. 결제 기간에 여러 리전으로 릴레이하는 경우 각 리전의 피크 대역폭 사용량에 따라 요금이 별도로 부과됩니다.

| 리전             | 가격(USD/Mbps/월) |
| :--------------- | :------------------- |
| 중국 본토(대륙) | 12.67                |
| 중국홍콩         | 12.67                |
| 싱가포르           | 8.04                 |
| 프랑크푸르트         | 7.1                  |
| 서울             | 16.56                |
| 인도             | 23.66                |
| 태국             | 13.01                |
| 미국 실리콘밸리         | 7.1                  |
| 미국 버지니아     | 7.1                  |
| 자카르타           | 17.4                 |
| 일본             | 13.01                |
| 상파울루           | 23.66                |
| 기타             | 12.67                |

### 과금 안내
- 과금 방식: 후불 월 결산.
- 과금 주기: 월 단위로 과금되며, 매월 청구서는 다음 달 1일과 3일 사이에 생성됩니다.
- 과금 규정: 기본적으로 타사 릴레이 요금은 매월 평균 일일 피크 대역폭 사용량(모든 타사 릴레이 작업에 대해)을 기준으로 종량제 모드로 청구됩니다. 귀하의 계정에서 LVB 서비스에 대해 다른 청구 모드가 사용되는 경우 해당 모드는 타사 릴레이에 적용됩니다.

### 과금 예시
사용자는 2022년 10월에 5일 동안 타사 릴레이 기능을 사용했습니다. 릴레이 리전은 중국 본토(대륙)이며, 일일 최대 대역폭은 각각 10Mbps, 80Mbps, 70Mbps, 75Mbps, 60Mbps입니다.
2022년 10월의 최대 일수는 31일이며, 중국 대륙(중국 내)에서 사용자의 LVB에 대한 과금 방식은 일 결산 대역폭입니다.
10월에 생성된 타사 릴레이 요금은 다음과 같습니다.
`(10Mbps + 80Mbps + 70Mbps + 75Mbps + 60Mbps)/31일 x 12.67(USD/Mbps/월) = 120.569USD`.

## 부가 기능 과금
릴레이용 로컬 모드는 CSS의 부가 기능입니다. 로컬 모드를 활성화하면 릴레이 작업의 MP4 파일이 릴레이되기 전에 로컬 노드에 캐시되어 더 매끄럽고 안정적인 재생을 보장합니다.

### 주의 사항
로컬 모드를 사용하면 부가 서비스 요금 외에도 [릴레이 요금](https://intl.cloud.tencent.com/document/product/267/41059)이 발생합니다.

### 과금 가격

| 과금 유형     | 가격(USD/과금 단위) |
| ------------ | --------------------- |
| 과금 단위 단가 | 0.01515               |

**과금 대상 시간과 실제 시간의 비율:**

| 부가 기능                   | 비율<br>과금 대상 시간(과금 단위) : 실제 시간(분) |
| -------------------------- | ---------------------------------------------------------- |
| 로컬 릴레이 모드           | 0.02 : 1                                                   |

### 과금 안내
- 과금 항목: 부가 기능 기본 요금.
- 과금 방식: 종량제 후불.
- 과금 주기: 일별 과금으로, 매일 발생한 부가 서비스 요금은 다음 날 계정에서 차감됩니다. 실제 요금 공제 및 청구서 생성 시간은 청구서를 참고하십시오.


### 계산 공식
- 과금 대상 시간 = 실제 시간 × 비율.
- 부가 서비스 요금 = 부가 서비스 기본 요금 × 과금 대상 시간.

### 과금 예시
2022년 11월 23일에 릴레이 작업에 대해 로컬 모드를 활성화하였고, 작업 시간은 로컬 파일 60분을 포함하여 100분이었다고 가정할 때, 11월 24일에 지불해야 하는 요금은 다음과 같습니다.
- 릴레이 요금: `0.00032(USD/분) x 100(분) = 0.032(USD)`
- 부가 기능 요금: `0.01515(USD/과금 단위) x 0.02(과금 단위/분) × 60(분) =0.01818(USD)`
