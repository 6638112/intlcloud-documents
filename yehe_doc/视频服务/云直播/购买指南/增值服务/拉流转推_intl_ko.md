풀 스트림 푸시 서비스는 콘텐츠 풀링 및 푸시 기능을 제공합니다. 라이브 방송 푸시 스트리밍 없이 빠르게 기존의 비디오/라이브 방송을 빠르게 풀링하여 타깃 주소로 푸할 수 있습니다. 풀 스트림 푸시 서비스 요금에는 [풀 스트림 푸시 작업 시간 요금](#time)과 [3rd party 푸시 요금](#third_part) 두 가지가 포함됩니다.



## 주의 사항
- 풀 스트림 푸시 기능은 **2021년 7월 1일 00:00부터 정식으로 과금되며**, 2021년 7년 1일 00:00 이후 풀 스트림 푸시 작업으로 인한 시간 소모 발생 시 비용이 발생합니다(작업 생성 시간 구분하지 않음).
- 풀 스트림 푸시 서비스 중 출처 주소에서의 풀 스트림은 재생에 해당하며, 출처 주소에서 해당 재생에 대한 트래픽/대역폭 소모가 발생합니다. 출처 주소는 Tencent Cloud의 CSS, VOD, COS 등의 주소를 의미하며, 해당 제품의 재생/다운로드 비용이 발생합니다.

## 풀 스트림 푸시 작업 시간

### 과금 가격
작업 시간(분)에 따라 과금됩니다.

| 과금 유형         | 가격(USD/분) |
| ---------------- | ----------------- |
| 풀 스트림 푸시 작업 시간 | 0.00032           |



###  과금 설명
- 과금 항목: 풀 스트림 푸시 작업 시간
- 과금 방식: 후불 
- 과금 주기: 일 단위로 과금됩니다. 매일 오전 10시 일 결산 풀 스트림 푸시 청구서가 발행되며(자세한 사항은 청구서 시간을 기준으로 함) 전날 발생한 비용이 차감됩니다.
- 과금 규정: 현재 실행되고 있는 풀 스트림 푸시 작업이 일시 중지되거나 만료된 작업의 경우 시간이 소모되지 않으며, 작업을 복구하여 실행한 이후부터 계속해서 작업 시간이 계산됩니다.

>! 출처 콘텐츠 오류로 콘텐츠를 정상적으로 풀링할 수 없는 경우 작업이 만료될 때까지 중단되지 않으며, **작업이 종료될 때까지 발생하는 시간은 과금 사용량으로 집계**되므로 유의하시기 바랍니다.

## 3rd party 푸시
>! 
>- 풀 스트림 푸시 기능은 현재 작업을 생성하는 Tencent Cloud 계정의 CSS 푸시 스트림 주소로만 푸시할 수 있으며, 해당 시나리오에서는 3rd party 푸시 비용이 발생하지 않습니다.
>- 현재 계정이 아닌 CSS 푸시 스트림 주소가 모두 3rd party 주소인 경우 3rd party 푸시 비용이 발생합니다.

### 과금 가격

| 과금 유형       | 가격(USD/Mbps/월) | 설명                       |
| :------------- | :------------------- | :------------------------- |
| 3rd party 푸시 과금 | 12.67                  | 3rd party 푸시 대역폭 소모에 따라 과금 |


###  과금 설명
- 과금 방식: 후불 월 결산
- 과금 주기: 월 단위로 과금되며, 익월 1일 - 3일에 전월 푸시 요금 청구서가 발행됩니다.
- 과금 규정: 3rd party에 푸시된 모든 풀 스트림 푸시 작업의 동시 접속 대역폭 값이 과금되며, 기본적으로 일 피크 월 평균(과금 주기 동안 발생한 일 피크값의 평균 값) 대역폭으로 후불 과금됩니다. 계정의 LVB가 기타 월별 대역폭 과금 방식인 경우 3rd party 푸시 대역폭 과금 방식 또한 LVB 과금 방식을 따릅니다.
