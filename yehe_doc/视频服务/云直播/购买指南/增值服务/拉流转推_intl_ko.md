릴레이 서비스는 콘텐츠 풀링 및 푸시 기능을 제공합니다. 라이브 방송 푸시 스트리밍 없이 빠르게 기존의 비디오/라이브 방송을 빠르게 풀링하여 타깃 주소로 푸시할 수 있습니다. 릴레이 서비스 요금에는 [릴레이 작업 시간 요금](#time)과 [타사 릴레이 요금](#third_part) 두 가지가 포함됩니다.

## 주의 사항
- 릴레이 기능은 **2021년 7월 1일 00:00부터 정식으로 과금되며**, 2021년 7년 1일 00:00 이후 릴레이 작업으로 인한 시간 소모 발생 시 요금이 발생합니다(작업 생성 시간 구분하지 않음).
- 기존 소스에서 데이터를 가져오면 재생/다운로드 요금이 발생합니다. Tencent Cloud CSS, VOD, COS에서 가져오면 해당 제품의 과금 규정이 적용됩니다.

## 릴레이 작업 시간
### 과금 가격
작업 시간(분)에 따라 과금됩니다.

| 과금 유형         | 가격(USD/분) |
| ---------------- | ----------------- |
| 릴레이 작업 시간 | 0.00032           |

###  과금 설명
- 과금 항목: 릴레이 작업 시간.
- 과금 방식: 종량제 후불.
- 과금 주기: 일 단위로 과금됩니다. 매일 오전 10시에 릴레이 일 결산 청구서가 발행되며(자세한 사항은 청구서 시간을 기준으로 함) 전일 발생한 요금이 차감됩니다.
- 과금 규정: 현재 실행되고 있는 릴레이 작업이 일시 중지되거나 만료된 작업의 경우 시간이 소모되지 않으며, 작업을 복구하여 실행한 이후부터 계속해서 작업 시간이 계산됩니다.

>! 비정상적인 소스 콘텐츠로 인해 풀링이 실패할 경우 지정된 종료 시간까지 릴레이 작업이 중지되지 않고 **해당 작업 기간이 청구됩니다**.

## 타사 릴레이
>! 
>- 릴레이 기능은 현재 작업을 생성하는 Tencent Cloud 계정의 CSS 푸시 스트림 주소로만 푸시할 수 있으며, 해당 시나리오에서는 타사 릴레이 요금이 발생하지 않습니다.
>- 현재 계정이 아닌 CSS 푸시 스트림 주소가 모두 타사 주소인 경우 타사 릴레이 요금이 발생합니다.

### 과금 가격
타사 릴레이 과금은 과금 기간 중 릴레이 서비스가 위치한 리전에서 발생하는 최대 대역폭(단위:Mbps)을 기준으로 합니다. 릴레이 서비스가 위치한 리전은 릴레이 아웃바운드 리전입니다. 릴레이 서비스가 동일한 청구 기간에 여러 리전에서 발생하는 경우 해당 리전의 피크 대역폭을 기준으로 청구됩니다.

### 과금 안내
- 과금 방식: 후불 월 결산.
- 과금 주기: 월 단위로 과금되며, 익월 1일 - 3일에 전월 릴레이 요금 청구서가 발행됩니다.
- 과금 규정: 모든 타사 릴레이 작업의 동시 접속 대역폭 값이 과금되며, 기본적으로 일 피크의 월 평균(과금 주기 동안 발생한 일 피크값의 평균 값) 대역폭으로 후불 과금됩니다. 계정의 LVB가 기타 월별 대역폭 과금 방식인 경우 타사 릴레이 대역폭 과금 방식 또한 LVB 과금 방식을 따릅니다.
