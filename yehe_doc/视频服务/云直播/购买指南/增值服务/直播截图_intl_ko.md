CSS는 라이브 스트림의 화면을 캡처할 수 있으며, 화면 캡처 이미지는 COS에 저장됩니다. 화면 캡처 기능을 사용할 경우 비용이 발생합니다. **당월 누적 화면 캡처 수를 정산 기준으로 합니다**.

### 주의사항

- 화면 캡처 기능은 기본적으로 꺼져 있고 콘솔 또는 클라우드 API를 통해 해당 기능을 활성화할 수 있습니다. 
- CSS 화면 캡처 이미지는 COS에 저장되며, 기능을 켠 후에는 COS 저장 비용이 발생합니다. 자세한 사항은 [COS 제품 과금 가격](https://intl.cloud.tencent.com/pricing/cos)을 참조하세요.
- 화면 캡처 기능을 사용할 경우 비용이 발생하며, **매월 1000건까지 무료로 제공되고 1000건을 넘으면 초과분에 대한 비용을 청구합니다**. 



### 과금 가격

| 화면 캡처 수 | 요금(USD/천건)   |비고                     |
| -------- | ----------------- | ------------------------ |
| ≤ 1000    | 0 USD          |매월 1000건까지 무료       |
| > 1000    | 0.0176 USD    | 1000건 이하인 경우 1000건으로 계산 |

### 과금 설명

- 과금 항목: 라이브 방송 화면 캡처 수
- 과금 방식: 후불
- 과금 주기: 일별 과금, 매월 1일~5일 지난 달 청구서를 발행하며 비용을 차감함. 자세한 사항은 요금 청구서를 기준으로 함.
- 과금 규정: 1개월 내 화면 캡처 총 수 x 단가



### 과금 예시

2021년 1월 1일부터 2021년 2월 1일까지 라이브 방송 화면 캡처 서비스를 이용하였고, 당월 화면 캡처 총 수가 168,000건인 경우, 2021년 2월 2일 결제해야 하는 라이브 방송 화면 캡처 청구 비용은 다음과 같습니다.
1월 CSS 화면 캡처 비용 = 0.0176(USD/천건) × (168 - 1)천건 = 2.9392 USD
