> ! Tencent LVB는 2021년 2월 1일부터 해당 연체 서비스 정지에 대한 안내를 실시합니다.

## 연체 설명
- LVB의 일 결산 후불 과금 항목은 사용 후 익일 오전에 일간 청구서가 발행되며, 월 결산 후불 과금 항목은 사용 후 익월 1일에 월간 청구서가 발행됩니다. 청구서 내역 조회는 [청구서 조회](https://intl.cloud.tencent.com/document/product/267/36278)를 참조하십시오.
- 고객의 Tencent Cloud 계정 잔액이 부족하여 계정 연체가 확인될 경우 계정 연체 알림을 받게 됩니다. 연체 후 24시간의 충전 시간이 부여되며, 24시간 이내에 계정에 충전하는 경우 LVB 서비스가 정지되지 않고, 24시간 이내에 계정에 충전하지 않는 경우 LVB 서비스가 정지됩니다.
- 계정에 연체가 발생해도 LVB와 클라우드 스튜디오에서 진행 중인 라이브 방송 또는 스튜디오 PGM 전송을 즉시 중지하지 않고 계속 실행하여 정상적으로 과금합니다. 클라우드 스튜디오 서비스를 정지하려면 수동으로 라이브 방송 또는 PGM(메인 모니터링)을 종료하십시오.



## 서비스 정지 설명
- Tencent LVB 서비스는 연체 시간 경과시에도 충전하지 않거나 콘텐츠 규정 위반 또는 기타 Tencent Cloud 서비스 이용약관에 위배되는 계정에 대해서는 서비스 정지 기간 **60일**이 초과하면 자동으로 LVB 리소스를 회수하고 해당 계정의 LVB 관련 데이터와 기록을 정기적으로 삭제할 권한을 가집니다.
- 서비스 정지 후에는 LVB 콘솔과 API를 사용할 수 없으며, 라이브 방송 녹화로 생성된 VOD 파일에도 액세스할 수 없습니다. COS에 저장된 라이브 방송 화면 캡처와 음란물 감지 등의 이미지는 COS의 [서비스 정지 설명](https://intl.cloud.tencent.com/document/product/436/10044)을 따르므로 유의해 주십시오.
- 콘텐츠 규정 위반 또는 기타 Tencent Cloud 서비스 이용약관에 위배되어 LVB 서비스가 정지된 Tencent Cloud 계정에 대한 복구는 지원하지 않습니다.



## 회수 설명
- 서비스 정지 상태로 **60일**이 초과한 계정은 Tencent LVB가 자동으로 LVB 리소스를 회수하고 해당 계정의 LVB 관련 데이터와 기록(설정, 서비스 로그, 터미널 권한 등)을 정기적으로 삭제합니다. 삭제된 후에는 데이터를 복구할 수 없습니다.
- LVB 서비스를 더이상 이용하지 않는 경우, 라이브 방송 서비스에 대한 정지 처리를 하고 LVB 계정 데이터를 삭제해야 합니다. 비즈니스 매니저 또는 [Submit Ticket](https://console.cloud.tencent.com/workorder/category)을 통해 서비스 정지를 신청할 수 있습니다.
