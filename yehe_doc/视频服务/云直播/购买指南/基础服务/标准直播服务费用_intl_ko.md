## 주의 사항

- 과금 방식: 일 결산 후불 방식
- 과금 주기: 일별 과금, 매일 오전 10시 전날 발생한 트래픽 요금 차감, 자세한 사항은 요금 청구서를 기준으로 함
- 트래픽/대역폭 단위 환산법은 1000임. 예: 1TB=1000GB
- **신규 사용자가 LVB를 활성화할 경우 기본적으로 트래픽 일별 과금 방식이 적용됩니다**.
- LVB은 베이징 시간 00:00에서 23:59까지를 기준으로 자연일로 계산됩니다.

## 트래픽 대역폭
라이브 방송 트래픽/대역폭 비용은 Tencent Cloud LVB의 기본 요금제로, LVB 서비스를 통해 라이브 방송 콘텐츠 시청 시 발생하는 다운스트림 트래픽/대역폭 비용입니다. [트래픽 과금](#flow) 과 [대역폭 과금](#bandwidth) 두 가지 방식을 제공하며, 모두 후불 일별 과금 방식 입니다. 사용자는 필요 시 자신에게 적합한 과금 방식을 선택할 수 있습니다.

## 트래픽 과금 방식
#### 과금 가격
LVB의 트래픽 과금 방식은 하루 발생하는 트래픽 구간별로 과금하며, 자세한 일일 트래픽 구간별 단가는 다음 표를 참조하십시오.

| 트래픽 구간            | 요금(USD/GB/Day)|
| ----------------- | ---------------- |
| 0 - 500GB             | 0.0459            |
| 500GB（포함）- 2TB  | 0.0441            |
| 2TB（포함）- 50TB    | 0.0406            |
| 50TB（포함）- 100TB | 0.0335            |
| ≥ 100TB               | 0.0282            |

#### 과금 설명
- 과금 항목: LVB 시청으로 발생하는 중국 내(대륙) 전체 네트워크 다운스트림 트래픽
- 과금 규정: 하루 발생 구간별 과금, 자연일 하루 내 발생하는 트래픽 누적치 × 해당 구간 단가
- 계산 공식: 
  - 소모 트래픽 = 비트레이트 / 8 × 전체 인원의 총 시청 시간 
    
    >?전체 인원의 총 시청 시간 = 평균 온라인 인원수 × 인 평균 시청 시간, 예: 1명이 60분과 60명이 1분을 시청한 총 시간은 같습니다.
  - 트래픽 비용: 소모 트래픽 × 당일 누적 다운스트림 트래픽에 해당하는 구간의 GB 당 단가

#### 과금 예시

- 라이브 방송 부호율이 1Mbps(해당 부호율은 오디오 부호율과 비디오 부호율의 합으로, 트랜스 코딩을 활성화하고 특정 부호율을 설정하는 경우 해당 설정값은 비디오 부호율임. 반드시 오디오 부호율을 추가하여 트래픽을 계산해야 함.)이고 라이브 방송 시간이 2시간, 이 중에 라이브 방송을 1시간동안 본 시청 인원수는 100명이고 2시간동안 본 시청 인원수는 50명인 경우 소모 트래픽은 대략적으로 다음과 같습니다.
  1(Mbps)/8 × 7200(s) × 50(명)+1(Mbps)/8 × 3600(s) × 100(명) = 90000(MB) = 90GB
- 2021년 1월 1일 라이브 방송 서비스를 이용하였으며 90GB 다운스트림 트래픽이 발생한 경우, 2021년 1월 2일 일 결산해야 하는 라이브 방송 트래픽 청구 비용은 다음과 같습니다.
  0.0459(USB/GB) × 90(GB) = 4.131 USD

### 대역폭 과금 방식

#### 과금 가격

LVB의 대역폭 과금 방식은 하루 발생하는 대역폭 구간별로 과금하며, 당일 피크 대역폭을 과금 기준으로 합니다. 자세한 일일 대역폭 구간별 단가는 다음 표를 참조하십시오.

| 대역폭 구간                | 요금(USD/Mbps/Day) |
| -------------------- | ------------------ |
| 0 - 500Mbps              | 0.1129               |
| 500Mbps（포함）- 5Gbps | 0.1094               |
| 5Gbps（포함）- 20Gbps  | 0.1041                |
| ≥ 20Gbps                 | 0.1024                |

#### 과금 설명

- 과금 항목: LVB 시청으로 발생하는 중국 내(대륙) 전체 네트워크 다운스트림 대역폭
- 과금 규정: 하루 발생 구간별 과금, 자연일 하루 내 발생하는 대역폭 누적치 × 해당 구간 단가

#### 과금 예시

- 라이브 방송 부호율이 500Kbps(해당 부호율은 오디오 부호율과 비디오 부호율의 합으로, 트랜스 코딩을 활성화하고 특정 부호율을 설정하는 경우 해당 설정값은 비디오 부호율임. 반드시 오디오 부호율을 추가하여 대역폭을 계산해야 함.)이고 라이브 방송 시간이 1시간, 시청 인원 수가 100명인 경우 소모 대역폭은 대략적으로 다음과 같습니다.
  500(Kbps) × 100 = 50000(Kbps) = 50Mbps
- 2021년 1월 1일 라이브 방송 서비스를 이용하였으며 50Mbps 다운스트림 대역폭이 발생한 경우, 2021년 1월 2일 결제해야 하는 라이브 방송 대역폭 청구 비용은 다음과 같습니다.
  하루 대역폭 라이브 방송 비용 = 0.1129(USD/Mbps/Day) × 50(Mbps) = 5.645 USD

>? 라이브 방송 업무량이 많아서 Tencent Cloud 청구 금액이 1만 달러를 초과하였거나 초과가 예상되어 일별 과금 방식으로 수요를 만족하지 못하는 경우, 비즈니스 매니저에게 문의해 주십시오. 업무 협의를 통해 과금 방식 및 요금을 결정할 수 있습니다. 자세한 문의사항은 [티켓 제출](https://console.cloud.tencent.com/workorder/category)을 통해 문의해 주세요.
