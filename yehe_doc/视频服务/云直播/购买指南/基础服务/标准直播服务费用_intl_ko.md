일일 트래픽 라이브 방송 요금 = 업스트림 트래픽 요금 + 다운스트림 트래픽 요금 = 0.0417(USD/GB/일)×(10(GB)+ 90(GB)) = 4.17USD

## 트래픽 대역폭

라이브 방송 트래픽/대역폭 요금은 Tencent Cloud LVB의 기본 과금 항목으로, LVB 서비스를 통해 라이브 방송 콘텐츠를 시청할 때 생성되는 다운스트림/대역폭 요금입니다. 제공되는 일별 후불 과금 방식은 [트래픽 과금](#flow)와 [대역폭 과금](#bandwidth) 2가지가 있으며, 사용자는 필요에 따라 자기에게 적합한 과금 방식을 선택할 수 있습니다. 

<span id="flow"></span>

### 트래픽 과금

#### 과금 가격

LVB의 트래픽 과금은 일일 발생량을 구간별로 과금하며, 자세한 트래픽 구간별 일일 단가는 다음 표와 같습니다. 

| 트래픽 구간          | 요금(USD/GB/일)|
| ----------------- | ---------------- |
| 0 - 500GB         | 0.0417           |
| 500GB(포함) - 2TB  | 0.0403           |
| 2TB(포함)- 50TB   | 0.0371           |
| 50TB(포함)- 100TB | 0.031           |
| ≥ 100TB           | 0.0258           |

###  과금 설명

- 과금 항목: LVB 시청으로 발생하는 중국 내(대륙) 전체 네트워크 다운스트림 트래픽.

- 과금 규정: 일일 발생 구간별 과금, 하루 동안 발생하는 트래픽 누적치×해당 구간 단가

- 계산 공식: 

  - 소모된 트래픽 = 비트 레이트 / 8 × 모든 사람이 본 총시간. 

    >? 모든 사람이 본 총시간 = 평균 온라인 인원수 × 1인당 본 시간, 예를 들면 한 사람이 60분동안 본 것은 60명이 모두 1분씩 본 것과 같습니다.

  - 트래픽 비용 = 소모 트래픽 × 당일 누적 다운스트림 트래픽에 해당하는 구간의 GB당 단가

#### 과금 예시

- 라이브 방송 비트 레이트는 1Mbps(해당 비트 레이트는 오디오 비트 레이트와 비디오 비트 레이트의 합으로, 트랜스 코딩을 활성화하고 특정 비트 레이트를 설정하는 경우 해당 설정값은 비디오 비트 레이트이므로, 반드시 오디오 비트 레이트를 추가하여 트래픽을 계산해야 합니다.)이고 라이브 방송 시간이 2시간, 이 중에 라이브 방송을 1시간 동안 본 시청 인원수는 100명이고 2시간 동안 본 시청 인원수는 50명인 경우 소모 트래픽은 대략적으로 다음과 같습니다.
  1（Mbps）/8 × 7200（s） × 50（명）+1（Mbps）/8 × 3600（s） × 100（명） = 90000（MB） = 90GB。
- 2021년 7월 13일에 라이브 방송 서비스를 이용하였으며 90GB 다운스트림 트래픽이 발생한 경우, 2021년 7월 14일에 결산해야 하는 라이브 방송 트래픽 청구 비용은 다음과 같습니다.
  0.0417(USD/GB) × 90(GB) = 3.753USD
- 다운스트림 재생 요금만 기본적으로 청구되고, 업스트림/다운스트림 사용이 불균형한 비즈니스 시나리오(업스트림 푸시 스트리밍:다운스트림 재생>1:10)에 대해 푸시 스트리밍의 일 피크 대역폭이 100Mbps를 초과하면 실제 푸시 스트리밍 사용량에 따라 별도로 푸시 스트리밍 요금을 청구합니다. 업스트림 푸시 스트리밍과 다운스트림 재생의 과금 방식 및 정가는 동일하며, 사용량이 일정 구간에 도달하면 개별적으로 과금됩니다.
  <span id="bandwidth"></span>

### 대역폭 과금

#### 과금 가격

LVB의 대역폭 과금 방식은 하루 발생하는 대역폭에 대해 구간별로 과금하며, 당일 피크 대역폭을 과금 기준으로 합니다. 자세한 일일 피크 대역폭 구간별 단가는 다음 표와 같습니다. 

| 대역폭 구간             | 요금(USD/Mbps/일) |
| -------------------- | ------------------ |
| 0 - 500Mbps          | 0.1041             |
| 500Mbps(포함) - 5Gbps | 0.1008             |
| 5Gbps(포함) - 20Gbps  | 0.0959             |
| ≥ 20Gbps             | 0.0943             |

###  과금 설명

- 과금 항목: LVB 시청으로 발생하는 중국 내(대륙) 전체 네트워크 다운스트림 대역폭

#### 과금 예시

- 라이브 방송 비트 레이트가 500Kbps이고(해당 비트 레이트는 오디오 비트 레이트와 비디오 비트 레이트의 합으로, 트랜스 코딩을 활성화하고 특정 비트 레이트를 설정하는 경우 해당 설정값은 비디오 비트 레이트이므로 반드시 오디오 비트 레이트를 추가하여 대역폭을 계산해야 합니다.) 라이브 방송 시간이 1시간, 시청 인원 수가 100명인 경우 소모한 대역폭은 대략적으로 다음과 같습니다.
  500(Kbps) × 100 = 50000(Kbps) = 50Mbps
- 2021년 7월 13일에 라이브 방송 서비스를 이용하고 50Mbps의 다운스트림 대역폭이 발생한 경우, 2021년 7월 14일에 결제해야 하는 라이브 방송 대역폭 청구 비용은 다음과 같습니다.
  일일 대역폭 라이브 방송 비용 = 0.1041(USD/Mbps/일)× 50(Mbps)= 5.205 USD
- 다운스트림 재생 요금만 기본적으로 청구되고 업스트림/다운스트림 사용이 불균형한 비즈니스 시나리오(업스트림 푸시 스트리밍:다운스트림 재생>1:10)에 대해 푸시 스트리밍의 일 피크 대역폭이 100Mbps를 초과하면 실제 푸시 스트리밍 사용량에 따라 별도로 푸시 스트리밍 요금을 청구합니다. 업스트림 푸시 스트리밍과 다운스트림 재생의 과금 방식 및 정가는 동일하며, 사용량이 일정 구간에 도달하면 개별적으로 과금됩니다.

>? 라이브 방송 업무량이 많아 Tencent Cloud 청구 금액이 1만 달러를 초과하였거나 초과가 예상되어 일별 과금 방식으로 수요를 만족하지 못하는 경우, 비즈니스 매니저에게 문의 바랍니다. 업무 협의를 통해 과금 방식 및 요금을 결정할 수 있습니다. 자세한 문의 사항은 [티켓 제출](https://console.cloud.tencent.com/workorder/category)을 통해 문의 바랍니다.
