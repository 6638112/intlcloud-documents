## 주의 사항

- 과금 방식: **일별 후불 과금**,
- 과금 주기: 일별 과금, 매일 오전 10시 전날 발생한 트래픽 요금을 차감하며, 자세한 사항은 요금 청구서를 기준으로 합니다.
- 트래픽/대역폭 환산 단위는 1000입니다. 예시: 1TB = 1000GB
- CSS는 중국 표준시 00:00-23:59을 1일로 계산합니다.
- LEB는 초저지연 링크를 적용하여 트래픽/대역폭 요금이 LVB보다 높습니다.
- LEB의 원활한 재생을 위해 라이브 방송 스트림의 B-프레임을 제거해야 합니다. 푸시 스트림의 라이브 방송 스트림에 B-프레임이 포함되어 있는 경우 시스템이 트랜스 코딩을 통해 B-프레임을 제거하도록 기본 설정되어 있어, 라이브 방송 트랜스 코딩 비용이 발생할 수 있습니다.
- Web 브라우저 풀 스트림은 표준 webRTC 프로토콜만 지원하며 AAC 오디오 포맷은 지원하지 않습니다. 따라서 푸시 스트림의 AAC 오디오 포맷을 OPUS 오디오 포맷으로 변환하여 재생해야 하는 경우, 오디오 트랜스 코딩 비용이 발생할 수 있습니다.

## 트래픽 대역폭

라이브 방송 트래픽/대역폭 비용은 Tencent Cloud LEB의 기본 과금 항목으로, LEB 서비스를 통해 라이브 방송 콘텐츠 시청 시 발생하는 다운스트림 트래픽/대역폭 비용입니다.
[트래픽 과금](#flow)과 [대역폭 과금](#bandwidth) 두 가지 일별 후불 과금 방식을 제공합니다. 사용자는 필요에 따라 자신에게 적합한 과금 방식을 선택할 수 있으며, **신규 사용자가 LEB를 개통할 경우, 기본적으로 트래픽 과금 방식이 적용됩니다**.

### 트래픽 과금 방식

**과금 가격**

LEB의 트래픽 과금 방식은 하루 발생하는 트래픽에 대해 구간별로 과금하며, 자세한 일일 트래픽 구간별 단가는 다음 표를 참조하십시오.

| 트래픽 구간          | 요금(USD/GB/일)|
| ----------------- | ---------------- |
| 0 - 500GB         | 0.0835           |
| 500GB(포함) - 2TB  | 0.0797           |
| 2TB(포함)- 50TB   | 0.0733           |
| 50TB(포함)- 100TB | 0.0620           |
| ≥ 100TB           | 0.0516           |

**과금 설명**

- 과금 항목: LEB 시청으로 발생하는 중국 내(대륙) 전체 네트워크 다운스트림 트래픽.
- 과금 규정: 일일 발생 구간별 과금, 하루 동안 발생하는 트래픽 누적치 × 해당 구간 단가.

**과금 예시**

- 라이브 방송 비트 레이트가 500Kbps이고(해당 비트 레이트는 오디오 비트 레이트와 비디오 비트 레이트를 더한 값입니다. 만약 트랜스 코딩을 활성화하고 특정 비트 레이트를 설정하는 경우, 해당 설정값은 비디오 비트 레이트가 되며, 오디오 비트 레이트를 더해 트래픽을 계산해야 합니다) 라이브 방송 시간이 1시간, 시청 인원수가 100명인 경우, 소모 트래픽은 대략적으로 다음과 같습니다. 500/8 x 3600 x 100 = 22500000KB = 22.5GB.
- 2020년 10월 1일 LEB 서비스를 이용하여 22.5GB의 다운스트림 트래픽이 발생한 경우, 2020년 10월 2일 결제해야 하는 라이브 방송 트래픽 청구 비용은 다음과 같습니다.
  일일 라이브 방송 트래픽 비용: 0.0891(USD/GB) × 22.5(GB)= 2.0048USD.

| 대역폭 구간             | 요금(USD/Mbps/일) |
| -------------------- | ------------------ |
| 0 - 500Mbps          | 0.2065             |
| 500Mbps(포함) - 5Gbps | 0.2000             |
| 5Gbps(포함) - 20Gbps  | 0.1903             |
| ≥ 20Gbps             | 0.1871             |

**과금 설명**

- 과금 항목: LEB 시청으로 발생하는 중국 내(대륙) 전체 네트워크 다운스트림 대역폭.
