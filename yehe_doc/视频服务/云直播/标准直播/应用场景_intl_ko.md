## 게임 라이브 방송 

Tencent Cloud LVB는 게임 플랫폼에 업스트리밍 가속, 고품질 트랜스 코딩, 가속 배포 등 서비스를 제공하고, MLVB SDK, 라이브 방송 서비스 호환 등을 통해 게임 라이브 방송 플랫폼에서 신속하게 라이브 방송 기능을 구현하여 저지연성, 낮은 랙 발생률 등을 갖춘 게임 플랫폼 전체 네트워크 라이브 방송 구축을 돕습니다.

## 쇼 라이브 방송

Tencent Cloud LVB는 라이브 쇼 스트리밍을 위한 라이브 방송 바로 재생, 컬러 필터, 수동 노출 등 기능을 제공하며, 퀵 스타트, 활성화 직후 바로 사용을 가능케 하여 라이브 방송의 인기를 높이고 분위기를 고조시킵니다.

## 온라인 교육

Tencent Cloud LVB는 온라인 교육 고객에게 원스톱 서비스를 제공합니다. 유연한 콘솔, API 방식으로 교육용 비디오를 업로드 및 풀링합니다. 플레이어 비밀번호, 동적 URL 링크 도용 방지의 2대 보안 기능으로 유저가 정성껏 제작한 교육용 콘텐츠를 지켜줍니다.플레이어 인터페이스를 전면 개방하여 사용자 정의 시간 제한 재생, 드래그 금지 등 각종 교육 기능을 편리하게 이용할 수 있습니다.

## 방송계 뉴 미디어

Tencent Cloud LVB는 뉴미디어 업계에 최적화된 라이브 방송 서비스를 제공하고, 뉴스, 미디어, 예능 등의 전문 라이브 방송 서비스 플랫폼을 빠르게 구현하여, 기존 미디어 보다 쉽게 인터넷 미디어로 전환할 수 있게 합니다.

## 전자상거래 라이브 방송

Tencent Cloud LVB는 전자상거래 플랫폼의 발전을 도와, 업체가 상품 정보를 더욱 복합적으로 전달할 수 있게 하고, 사용자의 효과적인 의사 결정을 도와 마케팅 비용 절감과 거래량을 증가를 달성했습니다.
