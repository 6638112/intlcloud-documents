## Cloud Streaming Services

### ゲームのライブブロードキャスト

Tencent Cloud Cloud Streaming Services(CSS)は、ゲームのライブブロードキャストプラットフォームに上りアクセラレーション、高品質なトランスコーディングとアクセラレーション配信などのサービスを提供し、ゲームプラットフォームでの低遅延、低ラグのネットワーク全体ライブブロードキャストの実現を支援します。
![](https://main.qcloudimg.com/raw/73204ceafdcdee21267363baf424af5f.jpg)

### ショーのライブブロードキャスト

Tencent Cloud CSSはショーのライブブロードキャストにライブブロードキャストのインスタントブロードキャスティング、カラーフィルター、マニュアル露出などの機能を提供することにより、早く上手になります、スタートして即利用でき、ライブブロードキャストの人気を高め、ライブブロードキャストの雰囲気アップにつながります。
![](https://main.qcloudimg.com/raw/a973df9e4e4e86f1bf016cecad4c07b7.jpg)


### オンライン教育

Tencent Cloud CSSはオンライン教育ユーザーに便利なワンストップサービスを提供します。柔軟なコンソール、API方式で教育ビデオをアップロード・ダウンロードし､さらにプレーヤーパスワード、ダイナミックURLリンク不正アクセス防止という2つのセキュリティ機能があり、ユーザーが作成した大切な教育コンテンツを保護します。全体的にオープンなプレイヤーインターフェースを提供することにより、ユーザーに対し、再生時間制限のカスタマイズ、ドラッグ禁止などの教育機能が便利になります。
![](https://main.qcloudimg.com/raw/ad2f3e46e0a049a09475b9ccf651e1a3.jpg)

### ラジオ・テレビの新メディア

Tencent Cloud CSSはラジオ・テレビの新メディア業界に完備なビデオライブブロードキャストサービスを提供します。ニュース、メディアとバラエティクラスなどのプロフェッショナルなライブブロードキャストサービスのプラットフォームに迅速にアクセスでき、従来のメディアからインターネットメディアへの転換を容易に実現します。
![](https://main.qcloudimg.com/raw/a7c40c03f0b8facebfc80d8571e6a343.jpg)

### eコマースのライブブロードキャスト

Tencent Cloud標準ライブブロードキャストはe、コマースプラットフォームの発展をサポートします。販売会社がより全面的に商品情報を発信し、ユーザーにおける有効な意思決定を促進して、マーケティングコストを削減し、取引量を増やします。
![](https://main.qcloudimg.com/raw/591d72bf55e2b7451020d7f7edfb4141.jpg)

## ライブイベントブロードキャスト

### スポーツ試合

ライブイベントブロードキャストは、スポーツ試合に超低遅延のライブブロードキャスト機能を提供し、試合結果の情報はライブブロードキャストを通じてユーザーにすばやく届き、視聴者はスポーツ試合状況をリアルタイムで楽しむことができます。
![](https://main.qcloudimg.com/raw/fa67e4cc7c2915dd0524d4917e5fb7a5.jpg)

### eコマースのライブブロードキャスト

eコマースライブブロードキャストでは、商品オークションやキャンペーンなどのトランザクションフィードバックには、ライブブロードキャストの高いリアルタイム性が求められています。ライブイベントブロードキャストの超低遅延機能により、キャスターと視聴者はトランザクションフィードバックをタイムリーに取得できるため、視聴しながら購入するエクスペリエンスが向上します。
![](https://main.qcloudimg.com/raw/591d72bf55e2b7451020d7f7edfb4141.jpg)

### オンライン授業

ライブイベントブロードキャストの超低遅延機能により、教師と生徒は、ライブブロードキャストによるオンライン授業を行うことができます。ライブイベントブロードキャストにより、教師と生徒はリアルタイムでライブブロードキャスト画面を同期し、対面のように自然な形でオンライン授業を進めることができます
![](https://main.qcloudimg.com/raw/c35da4634957c2732ff9962204e94282.png)

### オンライン質問回答

従来のオンラインでの質問への回答では、遅延が発生するため、視聴者側とキャスター側の両方で同時に表示させるには、フレームを補足する必要がある場合があります。ライブイベントブロードキャストの超低遅延はこの問題を完璧に解決し、両方がリアルタイムで回答画面を見ることができるようにし、実現の難易度を軽減し、エクスペリエンスをより快適にします。
![](https://main.qcloudimg.com/raw/933597c189a62420c8c70687cd92347e.jpg)

### ライブショーでのインタラクション

ライブイベントブロードキャストはライブショーに適しており、視聴者からのプレゼント贈呈など画面のリアルタイム性が求められているライブブロードキャストのインタラクティブシーンでの視聴者インタラクティブエクスペリエンスを大幅に向上させます。
![](https://main.qcloudimg.com/raw/879b411565f49be1059f684c84e23bc7.jpg)
