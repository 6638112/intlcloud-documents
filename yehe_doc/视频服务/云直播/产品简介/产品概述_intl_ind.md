Cloud Streaming Services (CSS) memberikan layanan streaming langsung berbasis cloud yang cepat, stabil, dan profesional kepada pengguna. CSS menawarkan Live Video Broadcasting (LVB) untuk pemutaran ulang real-time berskala besar dan Live Event Broadcasting (LEB) untuk streaming langsung dengan latensi ultra-rendah sehingga cocok untuk berbagai skenario streaming. CSS juga dapat digunakan bersama SDK Mobile Live Video Broadcasting (SDK MLVB) untuk menciptakan solusi streaming langsung audiovisual yang lengkap.
