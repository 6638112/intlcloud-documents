## CSS

### 게임 라이브 방송

Tencent Cloud CSS는 게임 라이브 방송 플랫폼에 업스트림 가속, 고품질 트랜스 코딩, 가속 배포 등 서비스를 제공하여, 게임 플랫폼에서 네트워크 전체에 딜레이와 랙이 적은 라이브 방송을 구현할 수 있도록 합니다.
![](https://main.qcloudimg.com/raw/73204ceafdcdee21267363baf424af5f.jpg)

### 쇼 라이브 방송

Tencent Cloud CSS는 쇼 라이브 방송를 위한 라이브 방송 바로 재생, 컬러 필터, 수동 노출 등 기능을 제공하며, 빠르게 시작하고 켜는 즉시 사용할 수 있게 하여 라이브 방송의 인기를 높이고 분위기를 고조시킵니다.
![](https://main.qcloudimg.com/raw/a973df9e4e4e86f1bf016cecad4c07b7.jpg)


### 온라인 교육

Tencent Cloud CSS는 온라인 교육 고객에게 원스톱 서비스를 제공합니다. 유연한 콘솔, API 방식을 통해 교육용 비디오를 업로드할 수 있으며, 플레이어 비밀번호, 동적 URL 링크 도용 방지의 두 가지 보안 기능으로 사용자가 열심히 제작한 교육용 콘텐츠를 철저히 보호합니다. 플레이어 인터페이스를 전면 개방하여 사용자 정의 시간 제한 재생, 드래그 금지 등 각종 교육 기능을 편리하게 이용할 수 있습니다.
![](https://main.qcloudimg.com/raw/ad2f3e46e0a049a09475b9ccf651e1a3.jpg)

### 방송계 뉴 미디어

Tencent Cloud CSS는 뉴미디어 방송 업계에 최적화된 라이브 방송 서비스를 제공하고, 뉴스, 미디어, 예능 등의 전문 라이브 방송 서비스 플랫폼에 빠르게 액세스하여, 전통 미디어를 보다 쉽게 인터넷 미디어로 전환할 수 있게 합니다.
![](https://main.qcloudimg.com/raw/a7c40c03f0b8facebfc80d8571e6a343.jpg)

### 전자상거래 라이브 방송

Tencent Cloud CSS는 전자상거래 플랫폼의 발전을 도와, 업체가 상품 정보를 더욱 복합적으로 전달할 수 있게 하고, 사용자의 효과적인 의사 결정을 도와 마케팅 비용 절감과 거래량을 증가를 달성했습니다.
![](https://main.qcloudimg.com/raw/591d72bf55e2b7451020d7f7edfb4141.jpg)

## LEB

### 스포츠 경기

LEB는 스포츠 경기에 대해 초저 딜레이의 라이브 방송 기능을 제공하여, 라이브 방송을 통해 경기 결과를 빠르게 사용자에게 전달하여 시청자에게 경기 상황을 실시간으로 알려줍니다.
![](https://main.qcloudimg.com/raw/fa67e4cc7c2915dd0524d4917e5fb7a5.jpg)

### 전자상거래 라이브 방송

전자상거래 라이브 방송에서 상품 할인 판매, 프로모션 등의 거래 피드백은 실시간성이 매우 중요하며, LEB의 초저 딜레이 기능으로 호스트와 시청자가 거래 피드백을 실시간으로 주고 받으며, 보면서 구매하는 향상된 경험을 제공합니다.
![](https://main.qcloudimg.com/raw/591d72bf55e2b7451020d7f7edfb4141.jpg)

### 온라인 수업

LEB의 초저 딜레이 기능으로 교사와 학생에게 라이브 방송을 통한 온라인 수업을 제공합니다. LEB는 교사와 학생 양쪽의 라이브 방송 화면에 실시간으로 상대방을 동기화하여 대면 수업처럼 자연스러운 온라인 수업을 진행합니다.
![](https://main.qcloudimg.com/raw/c35da4634957c2732ff9962204e94282.png)

### 온라인 퀴즈

전통적인 온라인 퀴즈는 딜레이가 존재하여 종종 시청자 단에서 프레임 보정을 해야 시청자와 진행자 양쪽을 동시에 표시할 수 있었습니다. LEB의 초저 딜레이 기능으로 이 문제를 완벽히 해결하고, 양쪽 모두가 실시간으로 퀴즈 화면을 볼 수 있게 하였으며, 구현 난이도를 낮추고 원활한 체험을 보장했습니다.
![](https://main.qcloudimg.com/raw/933597c189a62420c8c70687cd92347e.jpg)

### 쇼 인터랙션

LEB를 쇼 라이브 방송 시나리오에 적용하여, 시청자 선물 증정 등 화면의 실시간 특성이 중요한 라이브 방송 인터랙션 시나리오에서의 시청자 인터랙션 체험을 극대화했습니다.
![](https://main.qcloudimg.com/raw/879b411565f49be1059f684c84e23bc7.jpg)
