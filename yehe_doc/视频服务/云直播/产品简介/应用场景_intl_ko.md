
## 쇼 라이브 방송

Tencent Cloud CSS는 MLVB SDK와 IM 방 관리와 결합하여 쇼 라이브 방송을 위한 바로 재생, 컬러 필터, 호스트 시청자 라이브 마이크 연결 등의 기능을 제공합니다. 관련 기능은 빠르고 편리하게 사용할 수 있고 켜는 즉시 사용이 가능하여 라이브 방송의 인기를 높이고 분위기를 고조시킵니다.

## 비디오 모니터링
대규모의 비디오 트래픽 업스트림은 클라우드 녹화, 트랜스 코딩, 타임 시프트 등 기능과 결합하여 실시간 교통 상황 모니터링, 캠퍼스 안전, 상점 모니터링 등 니즈를 만족시킵니다.

## 온라인 교육
Tencent Cloud CSS 서비스는 효율적인 콘솔, API 방식으로 교육 비디오를 업로드 풀링하며, 특히 플레이어 암호화, 동적 URL 링크 도용 방지라는 두 보안 매커니즘을 통해 교육 콘텐츠 유출을 방지합니다. 초저지연 LEB로 실시간 인터랙션이 필요한 라이브 방송의 마이크 연결이 가능합니다.

## 상업용 라이브 방송
Tencent Cloud는 기업의 상업용 라이브 방송, 보면서 사는 라이브 커머스 등의 시나리오를 기반으로 상업용 라이브 방송 솔루션을 출시했습니다. SaaS 솔루션은 시나리오별 페이지 템플릿과 모듈 및 라이브 방송 페이지의 빠른 구성 기능을 제공하여 기업의 빠르고 자유로운 라이브 방송을 돕습니다.
