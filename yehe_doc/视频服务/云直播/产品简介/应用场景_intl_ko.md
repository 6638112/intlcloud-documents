## CSS

### 게임 라이브 방송

Tencent Cloud CSS는 게임 라이브 방송 플랫폼에 업스트림 가속, 고품질 트랜스 코딩, 가속 배포 등 서비스를 제공하여, 게임 플랫폼에서 네트워크 전체에 딜레이와 랙이 적은 라이브 방송을 구현할 수 있도록 합니다.

<img src="https://main.qcloudimg.com/raw/b4e6ca9c4c81a9caa570639578b856b8.png" width="700px">

### 쇼 라이브 방송

Tencent Cloud CSS는 쇼 라이브 방송를 위한 라이브 방송 바로 재생, 컬러 필터, 수동 노출 등 기능을 제공하며, 빠르게 시작하고 켜는 즉시 사용할 수 있게 하여 라이브 방송의 인기를 높이고 분위기를 고조시킵니다.
<img width="700px" src="https://main.qcloudimg.com/raw/9633bb89a49ef3f2615c737babb4e8cb.png" >


### 온라인 교육

Tencent Cloud CSS는 온라인 교육 고객에게 원스톱 서비스를 제공합니다. 유연한 콘솔, API 방식을 통해 교육용 비디오를 업로드할 수 있으며, 플레이어 비밀번호, 동적 URL 링크 도용 방지의 두 가지 보안 기능으로 사용자가 열심히 제작한 교육용 콘텐츠를 철저히 보호합니다. 플레이어 인터페이스를 전면 개방하여 사용자 정의 시간 제한 재생, 드래그 금지 등 각종 교육 기능을 편리하게 이용할 수 있습니다.

<img width="700px" src="https://main.qcloudimg.com/raw/703985c9b2c1c71720ae97fc7c2c10c6.png" >

### 방송계 뉴 미디어

Tencent Cloud CSS는 뉴미디어 방송 업계에 최적화된 라이브 방송 서비스를 제공하고, 뉴스, 미디어, 예능 등의 전문 라이브 방송 서비스 플랫폼에 빠르게 액세스하여, 전통 미디어를 보다 쉽게 인터넷 미디어로 전환할 수 있게 합니다.

<img width="700px" src="https://main.qcloudimg.com/raw/b0e4015f9691a503414fc5b6ccdf5023.png" >

### 전자상거래 라이브 방송

Tencent Cloud CSS는 전자상거래 플랫폼의 발전을 도와, 업체가 상품 정보를 더욱 복합적으로 전달할 수 있게 하고, 사용자의 효과적인 의사 결정을 도와 마케팅 비용 절감과 거래량을 증가를 달성했습니다.

## LCB

### 보안 모니터링

복수의 카메라로 다양한 각도의 시나리오에 실시간 라이브 방송 푸시 스트리밍을 진행하고, 동시에 비디오 실시간 조회를 지원하여 식품 제조 안전을 모니터링하는 오픈형 주방, 실시간 교통정보 모니터링, 캠퍼스 보안 등 환경에 효과적으로 적용합니다.

<img width="700px" src="https://main.qcloudimg.com/raw/619ee324e482d5a99dbbb31a9fb6037d.png" >

###  관광지

윈난의 다수 관광지에 카메라를 설치하여 24시간 촬영하고, Tencent Cloud 제품의 높은 동시 푸시 스트리밍 기능 및 비디오 배포 기능을 이용하여 사용자가 언제 어디서나 실시간으로 윈난의 여러 관광지를 조회할 수 있습니다.

<img width="700px" src="https://main.qcloudimg.com/raw/de3dc77fb8002c450ff81e724571b9a2.png" >

### Automobile Sales Servicshop 4S 매장

4S는 매장 내 카메라로 Youtu Box(엣지 터미널)을 통해 매장 내 인파 밀집 구역 및 고객을 인식하고, 클라우드 단에서 매장 내 고객의 밀집 구역을 분석하여 매장을 효율적으로 이용할 수 있도록 합니다. Tencent Cloud API 인터페이스를 통해 기업의 매장별 고객 이벤트 정보를 연동하여 잠재 고객이 매장에 들어올 때 신속히 인식하고, 다른 매장의 데이터를 동기화하여 매장 내 판매 직원에게 푸시로 알려주게 합니다.

<img width="700px" src="https://main.qcloudimg.com/raw/9d328daeefc066a90cb8215d0dea428d.png" >

## LEB

### 스포츠 경기

LEB는 스포츠 경기에 대해 초저 딜레이의 라이브 방송 기능을 제공하여, 라이브 방송을 통해 경기 결과를 빠르게 사용자에게 전달하여 시청자에게 경기 상황을 실시간으로 알려줍니다.

<img width="700px" src="https://main.qcloudimg.com/raw/6490b52a3a94705d32508079f95785f5.png" >

### 전자상거래 라이브 방송

전자상거래 라이브 방송에서 상품 할인 판매, 프로모션 등의 거래 피드백은 실시간성이 매우 중요하며, LEB의 초저 딜레이 기능으로 호스트와 시청자가 거래 피드백을 실시간으로 주고 받으며, 보면서 구매하는 향상된 경험을 제공합니다.

<img width="700px" src="https://main.qcloudimg.com/raw/e18c8b5162fe6a5d818d698f3ccd9702.png" >

### 온라인 수업

LEB의 초저 딜레이 기능으로 교사와 학생에게 라이브 방송을 통한 온라인 수업을 제공합니다. LEB는 교사와 학생 양쪽의 라이브 방송 화면에 실시간으로 상대방을 동기화하여 대면 수업처럼 자연스러운 온라인 수업을 진행합니다.

<img width="700px" src="https://main.qcloudimg.com/raw/c35da4634957c2732ff9962204e94282.png" >

### 온라인 퀴즈

전통적인 온라인 퀴즈는 딜레이가 존재하여 종종 시청자 단에서 프레임 보정을 해야 시청자와 진행자 양쪽을 동시에 표시할 수 있었습니다. LEB의 초저 딜레이 기능으로 이 문제를 완벽히 해결하고, 양쪽 모두가 실시간으로 퀴즈 화면을 볼 수 있게 하였으며, 구현 난이도를 낮추고 원활한 체험을 보장했습니다.

<img width="700px" src="https://main.qcloudimg.com/raw/a0201d06ada05c387338ab4b180186ff.png" >

### 쇼 인터랙션

LEB를 쇼 라이브 방송 시나리오에 적용하여, 시청자 선물 증정 등 화면의 실시간 특성이 중요한 라이브 방송 인터랙션 시나리오에서의 시청자 인터랙션 체험을 극대화했습니다.
