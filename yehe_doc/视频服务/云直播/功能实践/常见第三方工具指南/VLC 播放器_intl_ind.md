Pemutar media VLC adalah pemutar dan kerangka multimedia lintas platform sumber terbuka yang dapat memutar ulang sebagian besar file multimedia serta DVD, CD audio, VCD, dan berbagai protokol streaming secara gratis. Pemutar media VLC mendukung sistem operasi, seperti OS X, Windows, Linux, iOS, Android, dan Chrome OS serta semua format file umum untuk streaming langsung, seperti RTMP, FLV, dan M3U8. Anda dapat mengunjungi [situs web resmi VLC](http://www.videolan.org/vlc/?spm=a2c4g.11186623.2.13.72e92ad705krbE) untuk mengunduh versi terbaru.

## Memutar ulang Video dalam VLC
1. Untuk mendapatkan [alamat pemutaran ulang streaming video](https://intl.cloud.tencent.com/document/product/267/31559), pilih nama domain pemutaran ulang yang sudah Anda ajukan sesuai dengan kebutuhan di [Domain Management (Manajemen Domain)](https://console.cloud.tencent.com/live/domainmanage), lalu sambungkan alamat pemutaran ulang dalam format yang sudah ditentukan di konfigurasi pemutaran ulang.
![](https://main.qcloudimg.com/raw/f597c01a9a9a4d8b777f4a3222bfbd4b.png)
2. Buka VLC, lalu pilih **File** > **Open Network** (File > Jaringan Terbuka).
![](https://main.qcloudimg.com/raw/ff6842cf3ca5aff6055b4eb7fa598712.jpg)
3. Masukkan alamat pemutaran ulang langsung ke kotak dialog pop-up.
![](https://main.qcloudimg.com/raw/f5f277cff87c55eceb1a48c3a7d58a93.jpg)

4. Kemudian, klik **Open** (Buka) untuk mengonfirmasi pemutaran ulang. Jika pull dari alamat pemutaran ulang berjalan normal, jendela pemutaran ulang akan muncul. 
![](https://main.qcloudimg.com/raw/7b182dbf9d48f5afe588b38a834e050d.jpg)
