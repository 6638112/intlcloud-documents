[SDK LEB](https://intl.cloud.tencent.com/document/product/1071/41875) adalah CSS versi seluler dengan latensi ultra-rendah. SDK ini menawarkan **solusi yang dapat diintegrasikan dengan cepat** untuk pemutaran ulang di perangkat seluler dengan latensi milidetik. SDK LEB cocok untuk berbagai skenario dengan kebutuhan yang tinggi terhadap latensi, misalnya kegiatan pembelajaran online, streaming pertandingan olahraga, kuis online, dan siaran lelang langsung.

## Pengunduhan SDK dan Kode Sumber

SDK LEB telah diintegrasikan ke dalam SDK MLVB. Anda dapat menggunakan [MLVB International](https://intl.cloud.tencent.com/document/product/1071/38150) untuk mengaktifkan dengan cepat pemutaran ulang di perangkat seluler dengan latensi milidetik. Informasi selengkapnya dapat dilihat di [Download (Pengunduhan)](https://intl.cloud.tencent.com/document/product/1071/38150).
