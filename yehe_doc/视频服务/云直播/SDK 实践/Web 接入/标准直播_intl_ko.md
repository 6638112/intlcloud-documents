## Web 푸시 스트림

- **CSS 콘솔** > **보조 툴** > **[Web 푸시](https://console.cloud.tencent.com/live/tools/webpush)**로 이동할 수 있으며, 자세한 내용은 [Web 푸시 스트림](https://intl.cloud.tencent.com/document/product/267/31558#web)을 참고하십시오.
- TXLivePusher 푸시 스트림 SDK를 통해 스트리밍할 수 있으며, 자세한 사항은 [WebRTC 푸시 스트림](https://intl.cloud.tencent.com/document/product/267/41620)을 참고하십시오.

>? WebRTC 푸시 스트림의 오디오 인코딩 방식은 opus 코덱입니다. 재생에 LVB 프로토콜(RTMP, FLV, HLS)을 사용하는 경우 시스템이 자동으로 aac 코덱으로 전환하므로 오디오 트랜스 코딩 요금이 발생합니다. 자세한 내용은 [오디오 트랜스 코딩 과금 가격](https://intl.cloud.tencent.com/document/product/267/39604#a_trans)을 참고하십시오.
