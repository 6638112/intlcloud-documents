## Web 端推流

- 可通过**云直播控制台** > **辅助工具** > **[Web 推流](https://console.cloud.tencent.com/live/tools/webpush)**，详情请参见 [Web 端推流](https://intl.cloud.tencent.com/document/product/267/31558#web)。
- 可通过 TXLivePusher 推流 SDK 进行推流，详情请参见 [WebRTC 推流](https://intl.cloud.tencent.com/document/product/267/41620)。

>? WebRTC 推流的音频编码方式为 opus 编码，若使用标准直播协议（RTMP、FLV、HLS）播放，系统会自动转为 aac 编码，从而会产生音频转码费用，详情请参见 [音频转码费用说明](https://intl.cloud.tencent.com/document/product/267/39604#a_trans)。
