## Push WebRTC

[SDK TXLivePusher](https://intl.cloud.tencent.com/document/product/267/41620) digunakan untuk push streaming dalam LEB (streaming dengan latensi ultra-rendah). SDK ini dapat melakukan push pada audio dan video yang diambil browser dari kamera, layar, atau file media lokal ke server streaming langsung melalui WebRTC.

>? Push WebRTC menggunakan codec audio Opus. Jika Anda menggunakan protokol LVB (RTMP, HTTP-FLV, atau HLS) untuk pemutaran ulang, sistem secara otomatis akan mengonversi audio menjadi AAC sehingga Anda akan dikenai biaya transcoding. Informasi selengkapnya dapat dilihat di [Live Transcoding > Audio Transcoding](https://intl.cloud.tencent.com/document/product/267/39604) (Transcoding Langsung > Transcoding Audio).
