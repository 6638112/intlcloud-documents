## Push dari Web

- Anda dapat melakukan push streaming dari web di konsol CSS melalui **CSS Toolkit** > **[Web Push](https://console.cloud.tencent.com/live/tools/webpush)** (Toolkit CSS > Push Web). Informasi mendetail dapat dilihat di [Web push (Push web)](https://intl.cloud.tencent.com/document/product/267/31558#web).
- Anda juga dapat menggunakan SDK TXLivePusher untuk push streaming. Informasi mendetail dapat dilihat di [WebRTC Push (Push WebRTC)](https://intl.cloud.tencent.com/document/product/267/41620).

>? Push WebRTC menggunakan codec audio Opus. Jika Anda menggunakan protokol LVB (RTMP, HTTP-FLV, atau HLS) untuk pemutaran ulang, sistem secara otomatis akan mengonversi audio menjadi AAC sehingga Anda akan dikenai biaya transcoding. Informasi mendetail dapat dilihat di [Live Transcoding > Audio Transcoding](https://intl.cloud.tencent.com/document/product/267/39604#a_trans) (Transcoding Langsung > Transcoding Audio).
