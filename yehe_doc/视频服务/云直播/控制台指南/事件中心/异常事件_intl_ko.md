CSS 콘솔에서 라이브 스트리밍 중에 발생한 오류를 볼 수 있습니다.

## 전제 조건
[CSS 콘솔](https://console.cloud.tencent.com/live)에 로그인되어 있어야 합니다.

## 작업 단계
1. 왼쪽 사이드바에서 **이벤트 센터** > **[오류](https://console.cloud.tencent.com/live/event/error-event)**를 선택하여 오류 이벤트 페이지로 이동합니다.
2. 지난 7일 동안 스트림에서 발생한 오류를 쿼리하려면 스트림 ID를 입력하십시오. 데이터 쿼리 시간 범위는 3시간 미만입니다.
![](https://qcloudimg.tencent-cloud.cn/raw/d4a6f79951032f8cc34d1fdc9bd6d8a1.png)

## 이벤트 유형
다음은 라이브 스트리밍 중에 발생할 수 있는 오류 목록입니다. 

| 오류 유형                |
| ----------------------- |
| 비디오 타임스탬프가 뒤로 이동했습니다          |
| 오디오 타임스탬프가 뒤로 이동했습니다          |
| 비디오 타임스탬프가 눈에 띄게 증가했습니다      |
| 오디오 타임스탬프가 눈에 띄게 증가했습니다      |
| chunk size가 너무 큽니다          |
| 비디오 프레임 도달 간격이 너무 깁니다  |
| 오디오 프레임 도달 간격이 너무 깁니다  |
| 비디오 코덱이 변경되었습니다    |
| 오디오 코덱이 변경되었습니다    |
| 비디오 프레임 도달 전에 codec 헤더가 없습니다 |
| 오디오 프레임 도달 전에 codec 헤더가 없습니다 |
