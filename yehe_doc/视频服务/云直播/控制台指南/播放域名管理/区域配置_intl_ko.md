재생 도메인을 서로 다른 리전에서 배포, 가속, 재생하기 위해 도메인 가속 리전 전환이 필요한 경우 리전 설정에서 도메인 재생 리전을 설정할 수 있습니다.

## 전제 조건
- [CSS 콘솔](https://console.cloud.tencent.com/live)에 로그인되어 있어야 합니다.
- **재생 도메인**이 추가되어 있어야 합니다. 
>? 재생 도메인 추가 시 **중국대륙**과 같은 가속 리전을 선택하고, **확인**을 클릭합니다.

![](https://qcloudimg.tencent-cloud.cn/raw/1915d256b28e4e9a6b997ae464326b8a.png)

## 주의 사항
- **중국대륙 외부의 라이브 방송 요금은 중국대륙 요금과 다릅니다. 과금에 대한 자세한 내용은 [가격 리스트](https://intl.cloud.tencent.com/document/product/267/2819)를 참고하십시오.**
- 비가속 리전에서 해당 재생 주소를 사용하는 경우 정상적으로 사용할 수 없습니다.
- 가속 리전에 중국대륙이 포함되는 경우 재생 도메인에 대한 ICP비안을 완료해야 합니다.
- 가속 리전을 변경하면 대역폭 제한이 재설정됩니다. 다시 구성해야 합니다.

## 작업 단계
1. [**도메인 관리**](https://console.cloud.tencent.com/live/domainmanage)로 이동합니다. **재생 도메인 이름**을 클릭하거나 오른쪽에서 **관리**를 클릭합니다.
2. **고급 구성** 탭을 선택하고 **리전 구성**을 찾습니다.
3. **편집**을 클릭합니다. 팝업 창에서 가속 리전을 중국대륙, 글로벌 또는 중국대륙 외부로 변경할 수 있습니다.
4. **저장**을 클릭합니다.
    ![](https://qcloudimg.tencent-cloud.cn/raw/7d6d384588bedb5f51fe54dc5b13dd9e.png)

| 가속 리전            | ICP비안 필요 여부 | 설명                                                       |
| ------------------- | ------------ | ---------------------------------------------------------- |
| 중국대륙    | Yes | 중국 본토 외부의 요청을 처리할 수 없습니다.                 |
| 글로벌            | Yes | 가속은 전 세계적으로 지원되지만 중국 본토 내부와 외부의 가격이 다릅니다.               |
| 중국 본토 외부 | No | 중국 본토 내에서 요청을 처리할 수 없습니다. 중국 본토 내부와 외부의 가격이 다릅니다. |
