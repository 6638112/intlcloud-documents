## Catatan
CSS menyediakan nama domain pengujian `xxxx.livepush.myqcloud.com` secara default. Anda dapat menggunakannya untuk pengujian push, tetapi kami tidak menyarankan Anda untuk menggunakannya sebagai nama domain push untuk bisnis real Anda. Nama domain ini tidak dapat dihapus.

## Prasyarat
1. Anda telah mengaktifkan layanan [CSS](https://cloud.tencent.com/product/css).
2. Anda telah menambahkan [nama domain](https://intl.cloud.tencent.com/document/product/267/35970).



## Melihat Nama Domain
Di halaman **[Domain Management](https://console.cloud.tencent.com/live/domainmanage)** (Manajemen Domain), Anda dapat melihat status konfigurasi CNAME, jenis, status, waktu penambahan, dan waktu kedaluwarsa untuk nama domain yang telah ditambahkan. Untuk melihat detailnya, klik nama domain atau **Manage** (Kelola) di sebelah kanan.
![](https://qcloudimg.tencent-cloud.cn/raw/1185effbf2807268ab782557bfed00c5.png)

## Mengonfigurasi Nama Domain
Jika perlu mengonfigurasi nama domain push, Anda dapat mempelajarinya di [Push Configuration (Konfigurasi Push)](https://intl.cloud.tencent.com/document/product/267/31059).
Jika perlu mengonfigurasi nama domain pemutaran ulang, Anda dapat mempelajarinya di [Playback Configuration (Konfigurasi Pemutaran Ulang)](https://intl.cloud.tencent.com/document/product/267/31058).


## Menonaktifkan Nama Domain
Untuk menonaktifkan nama domain, klik **Disable** (Nonaktifkan) di sebelah kanan, lalu klik **Confirm** (Konfirmasi) di jendela pop-up. Nama domain berhasil dinonaktifkan jika statusnya telah berubah dari **Enabled** (Diaktifkan) menjadi **Disabled** (Dinonaktifkan).
>? Setelah dinonaktifkan, nama domain tidak dapat diakses. Nama domain dapat diakses lagi setelah diaktifkan. Langkah-langkah mengaktifkan dan menonaktifkan untuk nama domain pemutaran ulang dan push sama.

![](https://qcloudimg.tencent-cloud.cn/raw/dedf2826ba84a215a22120c5bb8518c9.png)

## Mengaktifkan Nama Domain
Untuk mengaktifkan nama domain yang dinonaktifkan, klik **Enable** (Aktifkan) di sebelah kanan. Nama domain berhasil diaktifkan jika statusnya telah berubah dari **Disabled** (Dinonaktifkan) menjadi **Enabled** (Diaktifkan).
![](https://qcloudimg.tencent-cloud.cn/raw/2132cd3c69e68f844731ed272872ab14.png)


## Menghapus Nama Domain
Untuk menghapus nama domain, klik **Delete** (Hapus) di sebelah kanan, lalu klik **Confirm** (Konfirmasi) di jendela pop-up.
![](https://qcloudimg.tencent-cloud.cn/raw/5086887185d04fcdab0ee74209393eb6.png)
