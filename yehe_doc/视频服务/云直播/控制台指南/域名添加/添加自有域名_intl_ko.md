CSS를 사용하려면 푸시 도메인 1개와 재생 도메인 1개, 즉 **2 개** 이상의 도메인이 있어야 합니다. 푸시와 재생 모두에 동일한 도메인을 사용할 수 없습니다.


## 전제 조건
1. [Tencent Cloud CSS 서비스](https://intl.cloud.tencent.com/product/css)가 활성화된 상태여야 합니다.

## 외부 도메인 추가
1. [CSS 콘솔](https://console.cloud.tencent.com/live)에 로그인하여 **도메인 관리**를 선택합니다.
2. **도메인 추가**를 클릭하여 도메인 추가 페이지로 이동해 다음과 같이 설정합니다.
    1. **푸시 도메인**을 추가하려면: 유형으로 **푸시 도메인**을 선택하고 도메인 이름을 입력한 후 **다음**을 클릭합니다.
    2. **재생 도메인**을 추가할 경우: 유형으로 **재생 도메인**을 선택하고 가속 리전(기본값: **중국대륙**)을 선택하고 도메인 이름을 입력하고 **다음**을 클릭합니다.

![](https://qcloudimg.tencent-cloud.cn/raw/b6687c4c83f8f2fcfd5732f017cc7b13.png)
>! 
>- 도메인 이름은 최대 45자까지 가능하며 알파벳 대문자를 포함할 수 없습니다. **45자** 이하의 알파벳 소문자로 된 도메인 주소를 입력하십시오.
>- 기본적으로 각 Tencent Cloud 계정에 최대 100개의 도메인을 추가할 수 있습니다. 100개 이상의 도메인을 추가하려면 [티켓을 제출](https://console.cloud.tencent.com/workorder/category)하여 신청하십시오.
>- 자신의 도메인 추가가 완료되면 추가된 도메인의 가속 리전을 변경할 수 있습니다. 도메인 관리 페이지에서 도메인 이름을 클릭하거나 오른쪽의 **관리**를 클릭합니다. **고급 구성** 탭을 선택하고 **리전 구성** 영역에서 **편집**을 클릭하고 팝업 창에서 가속 영역을 다시 선택하고 **확인**을 클릭합니다.



## 도메인 소유권 인증
도메인을 소유자에 의해서만 추가할 수 있도록 하려면 CSS 콘솔에 추가하기 전에 도메인 소유권을 인증해야 합니다. 예를 들어, `a.test.com`을 추가하려면 `test.com`의 소유권을 인증해야 합니다. `b.test.com`과 같이 상위 도메인이 동일한 도메인을 추가할 때 다시 인증할 필요가 없습니다. DNS 레코드를 추가하거나 HTML 파일을 업로드하여 도메인을 인증할 수 있습니다. 이전에 추가한 도메인이 인증되지 않은 경우 동일한 상위 도메인으로 도메인을 추가할 때 여전히 인증이 필요합니다.

[](id:DNS)
### DNS 레코드
DNS 공급자에서 DNS 레코드를 추가하여 도메인 소유권을 확인할 수 있습니다. Tencent Cloud의 DNS 서비스를 사용하는 경우 아래 단계에 따라 DNS 레코드를 추가하십시오.
1. [DNSPOD 콘솔](https://www.dnspod.com/login?s_url=https%3A%2F%2Fwww.dnspod.com%2F%3Flang%3Den)에 로그인합니다.
2. 추가하려는 도메인의 상위 도메인을 클릭합니다.
3. 레코드 관리 페이지에서 **레코드 추가**를 클릭합니다.
4. 신규 열에 도메인 레코드를 입력합니다. 구체적인 입력 내용은 다음과 같습니다. 
<table>
<thead><tr><th>매개변수</th><th>설명</th></tr></thead>
<tbody><tr>
<td>호스트</td>
<td>입력: cssauth</td>
</tr><tr>
<td>레코드 유형</td>
<td>TXT</td>
</tr><tr>
<td>레코드 값</td>
<td>CSS는 각 도메인에 고유한 레코드 값을 할당합니다. 도메인을 추가할 때 CSS 콘솔에서 볼 수 있습니다.</td>
</tr></tbody></table>

5. **확인**을 클릭합니다. TXT 레코드는 약 5분 후에 적용됩니다.
6. CSS 콘솔로 돌아가서 **인증**을 클릭합니다. 인증이 성공하면 다음 단계로 진행할 수 있습니다.
![](https://qcloudimg.tencent-cloud.cn/raw/2ba35210b6afd68653d32aef998b009c.png)

### HTML 파일
HTML 파일을 업로드하여 도메인 소유권을 인증할 수도 있습니다.
1. CSS 콘솔에서 도메인 소유권을 인증하라는 메시지가 표시되면 HTML 파일을 선택합니다.
2. **다운로드**를 클릭하여 파일을 다운로드합니다.
3. 두 번째 수준 도메인의 루트 디렉터리에 파일을 업로드합니다.
4. `http://second-level domain/second-level domain_cssauth.html`을 통해 파일에 액세스할 수 있는지 확인하십시오.
5. **인증**을 클릭합니다. 인증이 성공하면 다음 단계로 진행할 수 있습니다.
![](https://qcloudimg.tencent-cloud.cn/raw/e2f4412681655ec81b55965ed4216a9d.png)

>! 기본 설정을 완료한 후 CNAME 구성 단계를 진행할 수 있습니다. CNAME 구성에 대한 자세한 내용은 [도메인 CNAME 설정](https://intl.cloud.tencent.com/document/product/267/31057)을 참고하십시오.
