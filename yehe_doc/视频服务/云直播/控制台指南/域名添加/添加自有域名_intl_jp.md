CSSサービスを使用するには、少なくとも**2つ**のドメイン名が必要です。1つはプッシュドメイン名、もう1つは再生ドメイン名で、同じドメイン名をプッシュと再生の両方に使用することはできません。



## 前提条件
1.  [Tencent CSSサービス](https://intl.cloud.tencent.com/product/css)がアクティブ化されていること。
2.すでにドメイン名をお持ちの方。（中国本土のリソースを使用するには、ドメイン名はICP登録しなければならない）						
	- ドメイン名のICP登録が完了していない場合は、Tencent CloudのWebサイトからドメイン名のICP登録を完了することができます。
>!
>- 中国国家工業情報化部の規定により、ドメイン名のICP登録は必ず行わなければなりません。ICP登録には数営業日かかるため、事前にICP登録を行うことをお勧めします。詳細については、[ドメイン名の登録と設定に関するよくあるご質問](https://intl.cloud.tencent.com/document/product/267/32478)をご参照ください。
>- ドメイン名のICP登録の完了時間は、ドメイン名サービスプロバイダが定めた時間になります。中国国家工業情報化部からICP登録完了の通知を受け取っても、それから1時間から24時間お待ちください。[中国国家工業情報化部ICP登録照会ウェブサイト](https://beian.miit.gov.cn/#/Integrated/index)でICP登録を行ったドメイン名をクエリーして見つかった時点から、Tencent Cloud CSSにICP登録済みドメイン名を追加することができます。
>- 新規にICP登録を行ったドメイン名がTencent Cloud CVMに同期されるまで約1日かかります。そのため、このタイプのドメイン名を追加したときに、ドメイン名がICP未登録と表示される場合があります。


## 操作手順
<span id="step1"></span>
### 手順1：独自ドメイン名の追加
1. [CSSコンソール](https://console.cloud.tencent.com/live)にログインし、【ドメイン名管理】を選択します。
2. 【ドメイン名の追加】をクリックしてドメイン名の追加ページに進み、次のとおり設定を行います。
	1. **プッシュドメイン名**を追加する必要がある場合：ドメイン名を入力し、ドメイン名のタイプのうち【プッシュドメイン名】を選択して、【OK】をクリックします。
	2. **再生ドメイン名**を追加する必要がある場合：ドメイン名を入力し、ドメイン名のタイプのうち【再生ドメイン名】を選択し、アクセラレーションリージョンを選択します。デフォルトは【中国大陸】です。【OK】をクリックします。

![](https://main.qcloudimg.com/raw/a602d5953a0c0fcfd66ba8801c0266b3.png)
>! 
>- ドメイン名の桁数は29桁に制限されており、現時点では大文字のドメイン名はサポートされていません。**29桁**を超えない小文字のドメイン名アドレスを入力してください。
>- 独自ドメイン名の追加が成功したら、ドメイン名の管理リストを使用して、変更するドメイン名をクリックするか、右側の【管理】をクリックしてドメイン名の詳細ページに入ります。【高度な設定】を選択して【リージョン設定】タブをクリックして【編集】を選択し、地域設定変更ページに進み、アクセラレーションリージョンを再選択して【保存】をクリックします。

<span id="step2"></span>
### 手順2：ドメイン名CNAMEの完了
ドメイン名の追加が成功した後、システムは一つのCNAMEドメイン名（`.liveplay.myqcloud.com`を拡張子とする）を自動的にアサインします。CNAMEドメイン名は直接アクセスができませんので、ドメイン名のサービスプロバイダでCNAME 設定を完了する必要があります。設定が有効になったら、クラウドのCSSサービスを楽しむことができるようになります。具体的な操作は [CNAME設定](https://intl.cloud.tencent.com/document/product/267/31057)をご参照ください。

>? 追加が成功したドメイン名を管理する必要がある場合は、 [Domain Management](https://intl.cloud.tencent.com/document/product/267/31056)をご参照ください。
