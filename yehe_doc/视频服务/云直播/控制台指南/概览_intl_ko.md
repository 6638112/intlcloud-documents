CSS 콘솔은 라이브 방송 도메인 관리, 라이브 방송 스트림 관리, 트랜스 코딩, 녹화 등 기본 기능의 빠른 설정뿐만 아니라, Web 푸시 스트리밍, 가속 설정, 리소스 모니터링, 로그 관리 서비스 등 다양한 작업을 제공합니다. 

##  전제 조건

- [Tencent Cloud CSS](https://intl.cloud.tencent.com/)가 활성화된 상태여야 합니다. 
- [CSS 콘솔](https://console.cloud.tencent.com/live/livestat)에 로그인되어 있어야 합니다.

## 개요

오른쪽 탐색 메뉴에서 [개요]를 클릭하면 전일 트래픽 요금, 라이브 방송 실시간 데이터, 현재 푸시 스트림 채널 수, 동시 접속 수, 최근 몇 일 간의 과금 대역폭, 과금 트래픽 및 푸시 스트림 채널 수 추세 등 관련 데이터를 확인할 수 있으며, 필요에 따라 과금 방식 또는 시간 세분성을 변경할 있습니다. 가이드가 필요한 경우 우측 상단의 [초보자 액세스 가이드]를 클릭하면 CSS 입문을 위한 단계별 가이드를 볼 수 있습니다.

### 사용 추이

주로 당일, 전일, 최근 7일, 최근 30일의 라이브 방송 사용량 추이를 보여주며, ‘대역폭 추이’, ‘트래픽 추이’ 및 ‘푸시 스트림 채널 수’ 데이터를 표시합니다.

#### 시간 세분성 변경

CSS 콘솔은 과금 대역폭 추이, 과금 트래픽 추이, 푸시 스트림 채널 수 확인 시 시간 세분성 변경을 지원합니다. [시간 세분성]을 클릭하고 오른쪽의 시간 세분성 유형을 클릭하면 변경됩니다.
