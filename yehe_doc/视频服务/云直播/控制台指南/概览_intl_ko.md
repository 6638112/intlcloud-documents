CSS 콘솔은 라이브 방송 도메인 관리, 라이브 방송 스트리밍 관리, 트랜스 코딩, 녹화 등 기본 기능의 빠른 설정뿐만 아니라, Web 푸시 스트리밍, 가속 설정, 리소스 모니터링, 로그 관리 서비스 등 다양한 작업을 제공합니다. 

## 전제 조건

- Tencent Cloud CSS 서비스를 활성화해야 합니다. 
- [CSS 콘솔](https://console.cloud.tencent.com/live/livestat)에 로그인되어 있어야 합니다.

## 개요

오른쪽 탐색 메뉴에서 [개요]를 클릭하면 전날 트래픽 요금, 라이브 방송 실시간 데이터, 동시 접속 수, 최근 몇 일 간의 과금 대역폭 및 과금 트래픽 추세 등 관련 데이터를 확인할 수 있으며, 필요에 따라 과금 방식 또는 시간 단위를 변경할 있습니다. 가이드가 필요한 경우 우측 상단의 [초보자 액세스 가이드]를 클릭하면 CSS 입문을 위한 단계별 가이드를 볼 수 있습니다.
![](https://main.qcloudimg.com/raw/e898cd3b3df8bc651d5c3a0d21cef738.png)

### 과금 현황

<table>
<thead><tr><th width="20%">통계 항목</th><th width="80%">설명</th></tr></thead>
<tbody><tr>
<td>전일 트래픽 소모</td>
<td>전일 사용자의 라이브 방송 재생으로 발생한 다운스트림 트래픽/대역폭 사용 요금을 표시합니다. 과금 방식은 <strong> 일 결산 트래픽 과금</strong> 또는 <strong>일 결산 대역폭 피크 과금</strong> 중에 선택할 수 있습니다. </td>
</tr>
</tbody></table>

**과금 방식 변경**
현재 과금 방식이 일 결산 과금인 경우, [전일 트래픽 소모량] 통계 항목에서 [전환]을 클릭하면 과금 방식 변경이 안내되고, [확인]을 클릭하면 과금 방식이 변경됩니다. 과금 방식 변경에 대한 더 자세한 내용은 [과금 변경](https://intl.cloud.tencent.com/document/product/267/30411)을 참조하십시오.

>? 과금 요금에 대한 자세한 내용은 [가격 리스트](https://intl.cloud.tencent.com/document/product/267/2819)를 참조하십시오.

### 금일 데이터

주로 당일 영상 라이브 방송의 다운스트림 대역폭 피크, 다운스트림 트래픽 사용량을 표시하고 당일 동시 접속 상황에 대한 통계를 냅니다.

<table>
<thead><tr><th width="20%">통계 항목</th><th width="80%">설명</th></tr></thead>
<tbody><tr>
<td>실시간 다운스트리밍 대역폭</td>
<td>현재 모든 재생 도메인의 재생 가속 사용으로 발생된 다운스트림 대역폭 피크값입니다. </td>
</tr>
<tr>
<td>금일 다운스트리밍 트래픽</td>
<td>당일 모든 재생 도메인의 재생 가속 사용으로 발생된 다운스트림 트래픽의 총합입니다. </td>
</tr>
<tr>
<td>동시 접속 수</td>

</tr>
</tbody></table>


### 사용 추이

주로 당일, 전일, 최근 7일, 최근 30일의 라이브 방송 트래픽 사용량 추이를 보여주며, ‘대역폭 추이’와 ‘트래픽 추이’ 데이터도 표시합니다.

<table>
<thead><tr><th width="20%">통계 항목</th><th width="80%">설명</th></tr></thead>
<tbody><tr>
<td>대역폭 추이</td>
<td>조회한 기간 내에서 모든 재생 도메인의 재생 가속 사용으로 발생된 다운스트림 대역폭 피크값의 통계를 나타냅니다. </td>
</tr>
<tr>
<td>트래픽 추이</td>
<td>조회한 기간 내에서 모든 재생 도메인의 가속 서비스 사용으로 소비된 다운스트림 트래픽의 총합에 대한 통계를 나타냅니다. </td>
</tr>
</tbody></table>


#### 시간 단위 변경

CSS 콘솔은 과금 대역폭/ 과금 트래픽 추이 확인 시 시간 단위 변경을 지원합니다. [시간 단위]를 클릭하고 오른쪽의 단위 시간 유형을 클릭하면 변경됩니다.
