### 과금 사용량

Tencent Cloud 콘솔에서는 LVB 및 LEB를 위해 과금 사용량 조회 기능을 제공합니다. 푸시 스트림 트래픽/대역폭, 재생 트래픽/대역폭, 라이브 방송 녹화, 라이브 방송 트랜스 코딩, 라이브 방송 화면 캡처, 라이브 방송 마이크 연결, 라이브 방송 릴레이 등의 서비스 사용 현황을 확인하려면 CSS 콘솔의 **데이터 통계** > [과금 사용량](https://console.cloud.tencent.com/live/analysis/bill)에서 1개월 이내의 CSS 서비스 사용 관련 데이터를 확인할 수 있습니다.

### 월간 소비량
**라이브 방송 화면 캡처** 탭을 선택하면 라이브 방송 화면 캡처의 월간 소비 데이터를 볼 수 있습니다. 최근 6개월 동안의 화면 캡처 수 세부 정보 **다운로드**를 지원합니다. ‘월간 소비량’ 태그를 확인하고 해당 월의 우측 **세부 사항 내보내기**를 클릭하면 상세 라이브 방송 화면 캡처 정보를 내보내기할 수 있습니다.
![](https://qcloudimg.tencent-cloud.cn/raw/d2d7aef04f809a9a29bb0ea9cd97f53b.png)

### 최근 30일 소비량
