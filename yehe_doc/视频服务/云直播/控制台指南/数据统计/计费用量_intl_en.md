### Billing Usage Statistics

Tencent Cloud Console provides a billable usage query feature for Cloud Streaming Services (CSS). You can log in to the CSS console and go to **Statistics** > **[Billing Usage Statistics](https://console.cloud.tencent.com/live/analysis/bill)** to query the traffic/bandwidth, and usage statistics of live recording, live transcoding, live screencapture and push relay for the last 30 days.

### Details
You can view traffic/bandwidth usage from the last 30 days. You can also check the traffic/bandwidth of a specific time point.

![](https://main.qcloudimg.com/raw/0cc583f5d7c71f7c59a3c29a9703c2b4.jpg)


### Monthly Usage Details
Click **Live Recording** tab to view monthly usage details. You can **download** the screencapturing and recording usage details from the last 6 months. You can go to “Monthly Usage” and click **Export Details** to export recording channel details.
![](https://main.qcloudimg.com/raw/330438b2235f320ffa695eb791fb8ed4.png)

### Usage from the Last 30 Days

Click **Live Transcoding** tab to view usage details from the last 30 days. You can view start time, end time ,duration, codec, bitrate and transcoding type.
![](https://main.qcloudimg.com/raw/ce7bff4c9e8747d12eb7c9b33176647f.jpg)
