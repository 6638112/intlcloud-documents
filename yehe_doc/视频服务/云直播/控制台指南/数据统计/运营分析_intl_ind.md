Anda dapat melakukan kueri dan melihat bandwidth, lalu lintas, jumlah permintaan, dan jumlah koneksi bersamaan di halaman [Operation Analysis (Analisis Operasi)](https://console.cloud.tencent.com/live/analysis/total) di konsol CSS. Anda juga dapat melakukan kueri data dengan perincian nama domain (pemutaran ulang) berdasarkan wilayah atau ISP.
- Anda dapat melihat data dari satu atau beberapa provinsi saat melakukan kueri data untuk Tiongkok Daratan. Namun, saat ini Anda tidak dapat memilih banyak provinsi saat melakukan kueri data dari lebih dari satu hari.
- Saat ini, data untuk di luar Tiongkok Daratan digabungkan, tidak spesifik berdasarkan negara/wilayah/ISP.
>!Data untuk di luar Tiongkok Daratan adalah data lalu lintas yang dihasilkan oleh CDN Tencent Cloud luar negeri untuk streaming langsung. Alamat IP klien tidak digunakan sebagai standar statistik.


![](https://qcloudimg.tencent-cloud.cn/raw/6c16ed7196afe8896b9f8d52046536c5.png)

**Bandwidth**: data yang dihasilkan oleh pemutaran ulang downstream streaming langsung.
**Traffic** (Lalu lintas): lalu lintas kumulatif dalam satu menit berjalan selama streaming langsung, yang juga merupakan data yang dihasilkan oleh pemutaran ulang downstream streaming langsung.
**Number of requests** (Jumlah permintaan): jumlah kumulatif koneksi antara pemutar dan platform CSS dalam satu menit berjalan selama streaming langsung.
**Number of concurrent connections** (Jumlah koneksi bersamaan): jumlah koneksi bersamaan antara pemutar dan platform CSS saat ini selama streaming langsung.
