Login ke konsol CSS, pilih [Stream Data Query (Kueri Data Streaming)](https://console.cloud.tencent.com/live/analysis/stream) di bilah sisi kiri, lalu masukkan nama streaming untuk melihat [data pemutaran ulang]( #play), [data push](#push), [rekaman streaming langsung](#live), dan [peristiwa panggilan balik](#reback).
![](https://qcloudimg.tencent-cloud.cn/raw/f71a4c071eb9869222ec6523fbf1fc6e.png)

[](id:play)
### Data pemutaran ulang 
Di tab **Playback Data** (Data Pemutaran Ulang), Anda dapat melihat data seputar bandwidth, lalu lintas, dan koneksi bersamaan dari 20 hari terakhir. Periode kueri maksimum adalah 24 jam.
![](https://qcloudimg.tencent-cloud.cn/raw/d22104aec88718b575007cc4fe503200.png)

[](id:push)
### Data Push
Di tab **Push Data** (Data Push), Anda dapat melihat frame rate upstream dan data bitrate streaming tertentu dari tiga hari terakhir. Periode kueri maksimum adalah tiga jam.
Setelah Anda memasukkan nama streaming, pilih **Push Data** (Data Push) untuk melihat data push.
![](https://qcloudimg.tencent-cloud.cn/raw/5fff5d3ddb4048970bc5407924beb945.png)

 [](id:live)
### Rekaman streaming langsung
Di tab **Live Streaming Records** (Rekaman Streaming Langsung), Anda dapat melihat informasi streaming langsung, termasuk nama streaming serta nama domain push, nama aplikasi streaming langsung, waktu mulai, waktu berakhir, durasi push, dan ID klien push streaming tersebut dari 60 hari terakhir. Anda juga dapat mengeklik **Details** (Detail) untuk melihat data push. - Periode kueri maksimum adalah 30 hari.
![](https://qcloudimg.tencent-cloud.cn/raw/6c8cc59b1143d0dc19b6282d9623d71f.png)

[](id:reback)
### Peristiwa panggilan balik
Di tab **Callback Event** (Peristiwa Panggilan Balik), Anda dapat melihat nama streaming, domain push, waktu picu callback, waktu respons, dan hasil dari push, gangguan, atau callback perekaman dalam lima hari terakhir. Anda juga dapat mengeklik **View** (Lihat) untuk melihat konten suatu panggilan balik. Periode kueri maksimum adalah 24 jam.
![](https://qcloudimg.tencent-cloud.cn/raw/30af9d529e32594589478f6374d66873.png)
