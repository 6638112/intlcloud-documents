새 버전의 CSS 콘솔에는 재생 인증, 라이브 방송 스트리밍 관리 및 사용자 정의 템플릿 설정 등의 기능이 추가되었습니다. 새 버전의 콘솔을 라이브 방송 API3.0과 함께 사용하면 더욱 포괄적이고 편리한 설정 및 관리가 가능합니다. 구 버전의 CSS 콘솔에는 더 이상 새로운 기능을 추가하지 않고 점검만 진행할 예정입니다. 새 버전의 콘솔로 전환하면 CSS의 업데이트된 기능과 서비스를 계속 사용할 수 있습니다.

새 버전 콘솔로 전환한 뒤 이전 버전의 녹화 및 콜백 설정은 전역에서 도메인 차원으로 전환되며, 향후 추가되는 푸시 도메인은 각각 다른 녹화 및 콜백 템플릿에 바인딩될 수 있습니다. 또한 캡처 및 음란물 감지 설정은 전역으로 유지되며, 설정 취소를 원하는 경우 [티켓 제출](https://console.cloud.tencent.com/workorder/category)을 통해 취소가 가능합니다. 도메인 차원의 캡처 및 음란물 감지 설정을 추가하는 경우 도메인 설정이 우선 적용됩니다.

현재 「라이브 방송 코드 액세스」 방식을 사용중인 경우, CSS 콘솔의 [[라이브 방송 콘솔 액세스]](https://console.cloud.tencent.com/live/livecodemanage) 페이지의 [새 버전 콘솔로 전환]을 통해 전환할 수 있습니다.
현재 「채널 호스팅」 방식을 사용중인 경우, [티켓 제출](https://console.cloud.tencent.com/workorder/category)을 해주시면 저희 기술 인력이 새 버전 콘솔로의 전환을 도와드립니다. 콘솔의 사용에 관한 문서는 [콘솔 가이드(새 버전)](https://intl.cloud.tencent.com/document/product/267/31054)를 참조하십시오.
