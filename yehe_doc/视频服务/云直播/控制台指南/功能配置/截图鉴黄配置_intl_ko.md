본 문서에서는 [LVB 콘솔](https://console.cloud.tencent.com/live)을 통해 화면 캡처 템플릿을 생성하는 방법을 소개합니다. 생성 완료 후 해당 푸시 스트리밍 도메인에서 [음란물 감지 화면 캡처 설정](https://intl.cloud.tencent.com/document/product/267/31063)을 연결해야 하며, 연결하고 약 5~10분 후 적용됩니다. API를 통해서도 라이브 방송 채널에 화면 캡처 템플릿을 생성할 수 있습니다.


## 화면 캡처 템플릿 생성
LVB 콘솔에 로그인하여 왼쪽 메뉴바에서 [기능 템플릿]>[음란물 감지 화면 캡처 설정](https://console.cloud.tencent.com/live/config/jtjh)을 선택한 후, [+] 버튼을 클릭해 설정 정보를 작성하고 [저장]을 클릭합니다.
화면 캡처 간격은 일반적으로 10초로 기본 설정되어 있으며, 설정 가능 범위는 5초~300초입니다. 화면 캡처 간격 조정 시에는 반드시 5배수로 설정해야 합니다.

![](https://main.qcloudimg.com/raw/30b549740abdf77f002fba3069737e52.png)

>
>1. 화면 캡처 기능은 단독으로 활성화하여 사용할 수 있습니다. 그러나 음란물 감지 기능은 화면 캡처 기능을 활성화해야만 활성화할 수 있으며 단독으로 사용할 수 없습니다.
>2. 화면 캡처 및 음란물 감지는 유료 기능입니다. 활성화 후 화면 캡처 기능은 1000장당 0.0176USD의 요금이 발생하며, 음란물 감지 기능은 1000장당 0.2294USD의 요금이 발생합니다.
>3. 화면 캡처 및 음란물 감지 이미지는 사용자의 COS에 저장되며 COS 저장 비용이 발생합니다. 자세한 내용은 [COS 제품 가격](https://intl.cloud.tencent.com/document/product/436/6239)을 참조하십시오.
>4. 화면 캡처 기능을 활성화하려면 먼저 COS bucket에서 LVB 서비스의 데이터 쓰기 권한을 설정해야 합니다.


## 도메인 연결
화면 캡처 및 음란물 감지 템플릿 생성 후 [Domain Management](https://console.cloud.tencent.com/live/domainmanage)에서 해당 푸시 스트리밍 도메인 또는 오른쪽에 있는 [관리]를 클릭해 도메인 상세 페이지로 이동하고 [템플릿 설정]을 선택하여 해당 도메인에 화면 캡처 및 음란물 감지 템플릿을 지정해 줍니다.
![](https://main.qcloudimg.com/raw/680a7ca00df99cb79de9dd937554aa5c.png)
>
>- 음란물 감지 화면 캡처 설정 바인딩 해제가 필요한 경우, [템플릿 설정]에서 [편집]을 클릭하여 해당 템플릿 선택을 해제한 후 [저장]을 클릭하면 해당 템플릿과 도메인 연결이 해제됩니다.
- 콘솔의 음란물 감지 화면 캡처 템플릿 관리에서는 현재 도메인 차원에서 연결 인터페이스 생성 규칙을 취소할 수 없습니다. 음란물 감지 화면 캡처 관련 인터페이스를 통해 지정한 스트림과 연결하고 있는 경우, [화면 캡처 템플릿 삭제](https://intl.cloud.tencent.com/document/product/267/30832)를 참조하여 연결을 해제해야 합니다.
