타임 시프트는 CSS의 녹화 기능에 의해 구현됩니다. 이를 통해 사용자는 라이브 스트림의 이전 부분을 되감고 재생할 수 있습니다. 이는 일반적으로 라이브 스트리밍 스포츠 이벤트의 하이라이트를 재생하는 데 사용됩니다.

## 주의 사항
- 타임 시프트 템플릿을 생성한 후 푸시 도메인에 바인딩해야 합니다. 구성은 바인딩 후 5 - 10분 후에 적용됩니다.
- 타임 시프트는 타임 시프트가 활성화된 라이브 스트림의 트래픽을 기준으로 요금이 청구됩니다. 이 기능을 사용할 경우 [트래픽/대역폭 요금](https://intl.cloud.tencent.com/document/product/267/2819)도 발생합니다.
- 트랜스코딩된 라이브 스트림을 타임 시프트하려면 스트림에 대한 트랜스코딩 작업을 미리 구성해야 합니다. 이로 인해 트랜스코딩 비용이 발생합니다. 사용된 트랜스코딩 템플릿이 삭제되지 않았는지 확인하십시오.

## 전제 조건
-  CSS를 활성화하고 [푸시 도메인 이름](https://intl.cloud.tencent.com/document/product/267/35970)을 추가했습니다.

## 도메인 이름 바인딩

1. CSS 콘솔에 로그인하고 왼쪽 사이드바에서 **기능 구성** > [**타임 시프트**](https://console.cloud.tencent.com/live/config/time-shift/template)를 선택합니다.
   - **기존 템플릿에 도메인 바인딩:** 왼쪽 상단에서 **도메인 바인딩**을 클릭합니다.
   - **타임 시프트 템플릿 생성 후 도메인 바인딩:** [타임 시프트 템플릿 생성](#C_record) 후 팝업 창에서 **도메인 바인딩**을 클릭합니다.

2. 팝업창에서 **타임 시프트 템플릿**과 **푸시 도메인**을 선택한 후 **확인**을 클릭합니다.

>? **추가**를 클릭하여 여러 푸시 도메인을 템플릿에 바인딩할 수 있습니다.

## 도메인 이름 바인딩 해제

1. CSS 콘솔에 로그인하고 왼쪽 사이드바에서 **기능 구성** > [**타임 시프트**](https://console.cloud.tencent.com/live/config/time-shift/template)를 선택합니다.
2. 도메인 이름이 바인딩된 타임 시프트 템플릿을 선택하고 대상 도메인 이름을 찾은 다음 **바인딩 해제**를 클릭합니다.
3. 팝업창에서 **확인**을 클릭합니다.

>?  타임 시프트 템플릿의 바인딩을 해제해도 진행 중인 라이브 스트림에는 영향을 미치지 않습니다.

## 템플릿 수정

1. **기능 구성** > [**타임 시프트**](https://console.cloud.tencent.com/live/config/time-shift/template)로 이동합니다.
2. 대상 타임 시프트 템플릿을 선택하고 오른쪽 **편집**을 클릭하여 설정을 수정한 후 **저장**을 클릭합니다.

[](id:delete)
## 템플릿 삭제

1. CSS 콘솔에 로그인하고 왼쪽 사이드바에서 **기능 구성** > [**타임 시프트**](https://console.cloud.tencent.com/live/config/time-shift/template)를 선택합니다.
2. 대상 타임 시프트 템플릿을 선택하고 우측 상단의 **삭제**를 클릭합니다.
3. 현재의 타임 시프트 템플릿을 삭제할지 확인한 후, **확인**을 클릭하면 삭제됩니다.

>! 
>- 도메인 이름이 템플릿에 바인딩되어 있는 경우 [바인딩 해제](#unite)해야 템플릿을 삭제할 수 있습니다.
>- 콘솔에서 타임 시프트 템플릿은 도메인 레벨에서 관리됩니다. API에 의해 스트림에 바인딩된 타임 시프트 규칙을 바인딩 해제할 수 없습니다.

## 관련 작업

도메인 관리 페이지에서 도메인과 타임 시프트 템플릿을 **바인딩 해제** 및 **바인딩**할 수도 있습니다. 자세한 내용은 타임 시프트 구성을 참고하십시오.
