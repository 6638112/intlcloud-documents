스탠바이 스트림은 라이브 스트림이 중단될 때 자동으로 활성화되는 비디오 또는 이미지를 표시하여 시청 환경을 개선하는 데 도움이 됩니다. 기존 스트림이 복구되면 CSS가 다시 전환됩니다. 본문은 스탠바이 스트림 템플릿을 생성, 바인딩, 바인딩 해제, 수정 및 삭제하는 방법을 안내합니다.

## 주의 사항
- 템플릿을 생성한 후 푸시 도메인에 바인딩해야 합니다. 구성은 바인딩 후 5 - 10분 후에 적용됩니다.
- 템플릿 바인딩, 바인딩 해제 또는 수정은 새로운 라이브 스트림에만 영향을 미치고 진행 중인 스트림에는 영향을 주지 않습니다. 진행 중인 라이브 스트림에 변경 사항을 적용하려면 스트림을 중지하고 다시 푸시해야 합니다.
- 최대 **50**개의 스탠바이 스트림 템플릿을 생성할 수 있습니다.

## 전제 조건

Tencent CSS 서비스를 활성화하고 [푸시 도메인](https://intl.cloud.tencent.com/document/product/267/35970)이 추가된 상태여야 합니다.

## 스탠바이 스트림 템플릿 생성
1. CSS 콘솔에 로그인합니다. 왼쪽 사이드바에서 **기능 구성** > **스탠바이 스트림**을 선택합니다.
2. **템플릿 생성**을 클릭합니다.
3. 템플릿 이름을 입력합니다. 최대 30자까지 가능하며 중국어, 영어, 숫자, \_, -을 포함할 수 있습니다.
4. 템플릿은 1024자 이내의 중국어, 영어, 숫자, \_, -로 입력합니다.
5. 스트림 유형(이미지 또는 비디오)을 선택합니다.
   - 이미지를 선택한 경우: 5M 이하의 JPG 또는 PNG 이미지를 업로드합니다.
   - 비디오를 선택한 경우: 동영상(FLV 또는 MP4) 또는 오디오(AAC) 파일의 VOD URL을 입력합니다.
6. 대기 시간을 구성합니다(0 - 6초).
7. 최대 재생 시간을 구성합니다. 이 매개변수 값에는 상한선이 없습니다.
8. **저장**을 클릭합니다.

## 바인딩 도메인 이름
1. CSS 콘솔에 로그인하고 왼쪽 사이드바에서 **기능 구성** > [**스탠바이 스트림**](https://console.cloud.tencent.com/live/config/pad)을 선택합니다.
2. 다음 두 가지 방법 중 하나로 도메인을 템플릿에 바인딩할 수 있습니다.
  - **기존 템플릿에 도메인 바인딩**: 왼쪽 상단의 **도메인 이름 바인딩**을 클릭합니다.
  -  **템플릿 생성 후 도메인 바인딩**: 스탠바이 스트림 템플릿 생성 후 팝업 창에서 **도메인 바인딩**을 클릭합니다.

3. 팝업창에서 **스탠바이 스트림 템플릿**과 **푸시 도메인**을 선택한 후 **확인**을 클릭합니다.

>? **추가**를 클릭하여 여러 푸시 도메인을 템플릿에 바인딩할 수 있습니다.

## 도메인 이름 바인딩 해제
1. CSS 콘솔에 로그인하고 왼쪽 사이드바에서 **기능 구성** > [**스탠바이 스트림**](https://console.cloud.tencent.com/live/config/pad)을 선택합니다.
2. 대상 템플릿을 선택하고 **바인딩 해제**를 클릭합니다.
3. 팝업 창에서 **확인**을 클릭합니다.

[](id:change)

## 템플릿 수정

1. **기능 구성** > [**스탠바이 스트림**](https://console.cloud.tencent.com/live/config/pad)으로 이동합니다.
2. 대상 템플릿을 선택하고 오른쪽 **편집**을 클릭하여 템플릿 정보를 수정합니다.
3. 수정 후 **저장**을 클릭합니다.


[](id:delete)
## 템플릿 삭제
템플릿이 도메인에 바인딩된 경우 템플릿을 삭제하기 전에 먼저 바인딩을 해제해야 합니다. 도메인 바인딩 해제 방법은 [도메인 바인딩 해제](#untie)를 참고하십시오.
1. **기능 구성** > [**스탠바이 스트림**](https://console.cloud.tencent.com/live/config/pad)으로 이동합니다.
2. 삭제할 템플릿을 선택하고 우측 상단의 **삭제**를 클릭합니다.
3. 현재의 스탠바이 템플릿을 삭제할지 확인한 후 **확인**을 클릭하면 삭제됩니다.
