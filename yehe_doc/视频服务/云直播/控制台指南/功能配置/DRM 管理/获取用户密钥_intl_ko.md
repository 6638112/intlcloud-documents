DRM 암호화 기능에 대한 라이선스는 서드 파티 라이선스 서비스 SDMC에서 제공합니다. 이 기능을 사용하려면 SDMC에서 얻은 UID, SecretID 및 SecretKey를 제공해야 합니다. 본문은 정보를 얻는 방법을 보여줍니다.
 

## 작업 단계

1. [SDMC DRM 서비스](https://www.xmediacloud.com/contact-us/)에 등록합니다.
2. SDMC에서 신청서를 검토하고 DRM 콘솔 주소와 초기 비밀번호를 이메일로 보내드립니다.
3. [SDMC DRM 콘솔](https://www.xmediacloud.com/contact-us/)에 로그인하고 DRM 설정 페이지로 이동하여 UID, SecretID, SecretKey를 얻습니다.
4. CSS 콘솔의 [DRM 관리](https://console.cloud.tencent.com/live/config/drm) 페이지에서 획득한 정보를 입력합니다.
