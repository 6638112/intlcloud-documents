어댑티브 비트 레이트 기능을 사용하면 라이브 스트림의 재생 비트 레이트가 네트워크 상태에 따라 매끄럽게 변경될 수 있습니다. 이렇게 하면 변화하는 네트워크 조건에서 원활한 재생 경험을 보장합니다.

## 주의 사항
1. 템플릿 생성 후 재생 도메인과 바인딩할 수 있습니다. 템플릿은 5 - 10분 후에 적용됩니다.
2. 재생 도메인은 **여러 어댑티브 비트 레이트 템플릿**과 바인딩될 수 있으며 어댑티브 비트 레이트 템플릿은 **여러 재생 도메인**에 바인딩될 수 있습니다.
3. 각 어댑티브 비트 레이트 템플릿에 최대 15개의 스트림을 생성할 수 있습니다.
4. 어댑티브 비트 레이트 기능이 작동하려면 플레이어가 어댑티브 비트 레이트를 지원해야 합니다.
5. 어댑티브 비트 레이트 템플릿의 스트림에 대한 GOP는 동일해야 합니다.
6. 어댑티브 비트 레이트 템플릿의 스트림에 대한 코덱은 동일해야 합니다.



## 어댑티브 비트레이트 템플릿 생성
1. CSS 콘솔에 로그인합니다. 왼쪽 사이드바에서 기능 구성, 어댑티브 비트레이트를 선택합니다.

2. **어댑티브 비트 레이트 템플릿 생성**을 클릭하고 다음 설정을 완료합니다.

  - 기본 정보: 템플릿 이름 및 설명. 자세한 내용은 [기본 정보](#C_trans_normal)를 참고하십시오.
  - 스트림 정보: 자세한 내용은 [스트림 정보](#C_trans_high)를 참고하십시오.

3. 스트림 추가를 클릭하여 템플릿에 새 스트림을 추가합니다. 각 템플릿에 최대 15개의 스트림을 추가할 수 있습니다.

4. **저장**을 클릭합니다.

## 바인딩 도메인 이름
1. CSS 콘솔에 로그인합니다. 왼쪽 사이드바에서 **기능 구성** > **어댑티브 비트 레이트**를 선택합니다.
2. 다음 방법 중 하나로 도메인 이름 바인딩 페이지로 이동합니다.
  - **기존 트랜스코딩 템플릿에 도메인 이름 바인딩:** 왼쪽 상단에서 **도메인 이름 바인딩**을 클릭합니다.
    ![](https://qcloudimg.tencent-cloud.cn/raw/7b99a75c0b09c75c4f8a226dc23e9080.png)
  - **템플릿 생성 후 도메인 바인딩**: [어댑티브 비트 레이트 템플릿 생성](#create) 후 팝업 창에서 **도메인 이름 바인딩**을 클릭합니다.
    ![](https://qcloudimg.tencent-cloud.cn/raw/9e6816d793fdbeea9adfe939d28cd95c.png)
3. 도메인 이름 바인딩 창에서 **어댑티브 비트 레이트 템플릿**과 **재생 도메인**을 선택한 후 **확인**을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/1db512981db33ae4c7c8454005d29e99.png)
>?**추가**를 클릭하여 여러 재생 도메인 이름을 템플릿에 바인딩할 수 있습니다.

## 도메인 이름 바인딩 해제
1. CSS 콘솔에 로그인합니다. 왼쪽 사이드바에서 **기능 구성** > **어댑티브 비트 레이트**를 선택합니다.
2. 템플릿을 선택하고 타깃 도메인 이름을 찾은 다음 **바인딩 해제**를 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/cc40b63592a7ffb550709359f79691d0.png)
3. 팝업 창에서 **확인**을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/7ce4c285ca40ab72dd62d75e855ea78e.png)



[](id:modify)
## 템플릿 수정
1. CSS 콘솔에 로그인합니다. 왼쪽 사이드바에서 **기능 구성** >  **어댑티브 비트 레이트**를 선택합니다.
2. 대상 어댑티브 비트 레이트 템플릿을 선택하고 오른쪽의 **편집**을 클릭하여 수정합니다.
3. 수정 후 **저장**을 클릭합니다.

![](https://qcloudimg.tencent-cloud.cn/raw/05ca4a09ac9b73bcaaecdd504c8b4e7e.png)

## 템플릿 삭제
>!   템플릿이 도메인 이름으로 바인딩된 경우 템플릿을 삭제하기 전에 [바인딩 해제](#unite)해야 합니다. 

1. CSS 콘솔에 로그인합니다. 왼쪽 사이드바에서 **기능 구성** >   **어댑티브 비트 레이트**를 선택합니다.
2. 재생 도메인 이름과 연결되지 않은 템플릿을 선택하고 **삭제**를 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/ed74cfb664fd83e67d567a933e848f2f.png)
3. 팝업 창에서 **확인**을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/f1b8df196cb97c8787bfd4598dffce5a.png)
