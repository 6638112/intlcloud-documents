Untuk membantu Anda memulai penggunaan konsol CSS dengan cepat, dokumen ini akan memperkenalkan beberapa layanan CSS yang sering digunakan dan mengategorikannya menurut kebutuhan berbagai pengguna. Konsol CSS terdiri dari empat modul: Layanan Dasar, Layanan Khusus Skenario, Pusat Data, dan Toolkit CSS.

## Pusat Data
Modul ini menyediakan layanan analisis data profesional. Anda dapat melakukan kueri penggunaan lalu lintas/bandwidth, transcoding, pemberian watermark, relai push, pengambilan tangkapan layar, dan deteksi pornografi dalam periode tertentu. Modul ini juga dibekali fitur analisis log yang dapat Anda gunakan untuk memantau sumber daya dan melacak data yang berguna.
