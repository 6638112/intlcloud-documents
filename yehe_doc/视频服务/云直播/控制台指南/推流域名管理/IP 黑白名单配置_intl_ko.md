본문에서는 요청을 필터링하고 스트리밍 콘텐츠에 대한 액세스를 제어하기 위해 IP 얼로우/블록리스트를 설정하는 방법을 보여줍니다.

## 설정 원리
- IP **얼로우리스트**: 목록에 있는 IP 주소만 스트리밍 콘텐츠에 액세스할 수 있습니다.
- IP **블록리스트**: 목록에 있는 IP 주소는 스트리밍 콘텐츠에 액세스할 수 없습니다.

## 참고 사항
- IP 블록리스트/얼로우리스트 구성이 완료되면 약 5분 후에 적용됩니다.
- IP 블록리스트/얼로우리스트 구성 완료 후, 진행 중인 라이브 스트림을 중지하고 다시 푸시해야만 적용됩니다.

## 전제 조건
- CSS 서비스가 활성화되어 있고 [CSS 콘솔](https://console.cloud.tencent.com/live/livestat)에 로그인되어 있어야 합니다.
- [푸시 도메인](https://intl.cloud.tencent.com/document/product/267/35970)이 추가되어 있어야 합니다.

## IP 얼로우/블록리스트 설정 수정
1. [도메인 관리](https://console.cloud.tencent.com/live/domainmanage)를 선택하고 IP 얼로우/블록리스트 설정을 수정할 타깃 **푸시 도메인**을 클릭하거나 우측의 **관리**를 클릭하여 도메인 관리 페이지로 이동합니다.
2. **고급 설정** > **IP 얼로우/블록리스트**에서 **편집**을 클릭합니다.
3. 설정을 수정하고 **저장**을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/61ded2bf69e3e38163b1b85716ff9477.png)

2. **고급 설정> IP 얼로우/블록리스트**에서, ![img](https://main.qcloudimg.com/raw/e72f89a0deb6858428dc3e93ce7e7088.png)을(를) 클릭해 IP 얼로우/블록리스트 비활성화를 선택합니다.
