본문은 스탠바이 스트림 템플릿을 푸시 도메인에 바인딩하여 해당 도메인에 대한 스탠바이 스트림 기능을 활성화하는 방법과 템플릿 바인딩을 해제하여 기능을 비활성화하는 방법을 보여줍니다. 스탠바이 스트림 기능은 기본적으로 비활성화되어 있습니다.

## 주의 사항
- 템플릿은 도메인에 바인딩된 후 약 5 - 10분 후에 적용됩니다.
- 템플릿이 푸시 도메인에 성공적으로 바인딩되면 구성된 스탠바이 스트림이 해당 도메인의 모든 푸시 주소에 적용됩니다.

## 전제 조건
- [CSS 콘솔](https://console.cloud.tencent.com/live)에 로그인되어 있어야 하며 [푸시 도메인](https://intl.cloud.tencent.com/document/product/267/35970)이 추가되어 있어야 합니다.
- [스탠바이 스트림 템플릿 생성](https://intl.cloud.tencent.com/document/product/267/31073)이 완료되어 있어야 합니다.


## 스탠바이 스트림 템플릿 바인딩
1.  [**도메인 관리**](https://console.cloud.tencent.com/live/domainmanage)로 이동하여 **푸시 도메인**을 클릭하거나 **관리**를 클릭하여 도메인 세부 정보 페이지로 이동합니다.
2.  **템플릿 구성** 탭을 선택하고, **스탠바이 스트림 구성** 영역 우측 상단의 **편집**을 클릭합니다.
3. 스탠바이 스트림 템플릿을 선택하고 **확인**을 클릭합니다.

>? 작업 열에서 **미리보기**를 클릭하여 스탠바이 스트림을 미리 볼 수 있습니다.

## 스탠바이 스트림 템플릿 바인딩 해제
1. [**도메인 관리**](https://console.cloud.tencent.com/live/domainmanage)로 이동하여 대상 **푸시 도메인**을 클릭하거나 **관리**를 클릭하여 도메인 세부 정보 페이지로 이동합니다.
2. **템플릿 구성** 탭을 선택하고, **스탠바이 스트림 구성** 영역 우측 상단의 **편집**을 클릭합니다.
3. 템플릿을 선택 취소하고 **확인**을 클릭합니다.
