Leveraging Tencent's powerful technologies and utilizing core capabilities of Tencent Video, Live Video Broadcasting (LVB) is a professional, high-quality, and stable live streaming service. It powers live stream broadcasting and distribution so that you can provide your audience with ultra-low latency connections and sustain massive volumes of concurrent access requests.

Before you use LVB APIs, make sure that you fully understand the LVB features and agree to the LVB [billing rules](https://intl.cloud.tencent.com/document/product/267/2818).
