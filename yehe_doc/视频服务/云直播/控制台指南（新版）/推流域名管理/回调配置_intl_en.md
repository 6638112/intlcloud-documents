## Operation Scenarios

The callback feature is disabled by default for LVB push. If you want to set or modify it, you can do so in the callback configuration. Then, associate a specified domain name in Domain Management.

## Prerequisites

- You have logged in to the [LVB Console](https://console.cloud.tencent.com/live).
- You have added a **push domain name**.

## Directions
1. 	Go to **[Domain Management](https://console.cloud.tencent.com/live/domainmanage)**, click the **push domain** to be configured or **Manage** to enter the Domain Management page.
2. 	Select **Template Configuration** to view the **Callback Configuration** tab.
3. 	Click **Edit** to select a callback configuration which specifies the corresponding callback template for the playback address under the domain name.
4.	Click **Save**.
