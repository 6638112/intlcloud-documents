## Data Center
This module provides users with professional data analysis services. You can query usage of traffic/bandwidth, transcoding, watermark, push to third-party platforms, screencapturing & porn detection within a specific time granularity. And the log analysis feature makes it easy for you to monitor resources and keep track of useful data.
