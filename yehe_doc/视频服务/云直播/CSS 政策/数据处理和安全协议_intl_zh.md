## 1\. 背景

本模块仅在您使用 Cloud Streaming Services（简称“**功能**”）时适用。本模块包含在位于 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据处理和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. 处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **配置数据：**<li>域名配置（配置身份验证、带宽上限配置、https 配置（包括 ssl 证书）、区域配置、配置模板、原服务器配置）；<br><li>录制模板（模板名称、录制文件格式、录制单个文件时长、保留期限、恢复超时（仅适用于 hls 格式）、录制至 VOD 子应用）；<br><li>水印模板（模板名称、水印图片、水印位置）；<br><li>转码模板（模板名称、比特率、实时屏幕高度、GOP、编码方法）；<br><li>截图模板（模板名称、截图频率、截图存储的 cos 位置、截图文件名称、智能鉴黄）。 | 我们仅出于按照您的特定配置向您和您的最终用户提供本功能的目的处理这些数据。<br/>请注意，此数据将存储在云数据库 MySQL 功能中。 |
| **直播录制视频**（如果选择录制直播视频） | 我们仅出于为您提供本功能的目的处理此数据。<br/>请注意，这些数据存储在我们的云点播 (VOD) 功能中。 |
| **直播截图**（如果选择录制直播截图），包括在检测到色情内容之后生成直播截图（如果选择启用“鉴黄配置”功能） | 我们仅出于为您提供本功能的目的处理此数据。<br/>请注意，这些数据将存储在我们的云对象存储 (COS) 功能中。<br/>请注意，如果启用此功能的“鉴黄配置”功能，则存储在 COS 中的此类数据将传输到我们的“图片内容安全 (IMS)”功能，以便根据需要为您提供此功能（包括审核识别色情信息并返回相关结果）。IMS 将仅在完成色情检测过程并为您返回相关结果的必要期限内保留直播截图。 |

| **个人信息**                                | **保留政策**                                         |
| ------------------------------------------------------- | ------------------------------------------------------------ |
| 配置数据直播录制直播视频截图 | 我们会在您选择的数据保留期限内保留此数据。此外，若您删除帐户或终止使用本功能，我们将在 60 天后删除此数据。 |

您声明、保证并承诺，需要处理与本功能有关的最终用户个人数据时，您将根据适用法律获得并维持最终用户提供的所有必要同意，以便我们遵守适用法律的要求。
