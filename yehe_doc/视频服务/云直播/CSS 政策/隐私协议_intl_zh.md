## 1\. 简介

本模块仅在您使用 Cloud Streaming Services（简称“**功能**”）时适用。本模块已纳入[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. 可用性

本功能可供全球范围内的用户使用。

## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **使用日志数据：** 直播帧率、比特率、流量、带宽、时间戳、配置功能的使用记录（例如录制、转码、水印和截图）、APPID、UIN | 我们将这些信息用于以下目的：<li>确保功能按要求运行， <br><li>确保发现和补救程序错误，和 <br><li>计费。请注意，此数据将存储在云数据库 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言必不可少。 |
| **计费数据：** 带宽计费、转码通道数量、录制数量、截图数量、APPID、UIN | 我们将此类信息用于计费目的。<br/>请注意，此数据将存储在云数据库 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言必不可少。 |
| **访问日志数据**（最终用户）**：** 请求时间、客户端 IP、访问域名、URL、访问直播期间的带宽和数据量、国家/地区 ID、省、运营商、HTTP 状态码、HTTP 来源信息、播放时长、用户-代理信息、范围参数、HTTP 方法、流 ID、缓存 HIT/MISS、节点 IP、服务器区域、服务器国家/地区、连接文件描述、APPID、UIN | 我们将这些信息用于以下目的：<li>确保功能按要求运行，<br><li>确保发现和补救程序错误。<br/>请注意，此数据存储在我们的日志服务 (CLS) 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言必不可少。 |

## 5\. 我们如何 存储和共享个人信息

见隐私政策中的规定。

## 6\. 我们如何 共享个人信息

## 7\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息** | **保留政策** |
| ------------------------ | -------------------- |
| 使用日志数据           | 存储 60 天。  |
| 计费数据             | 存储 2 年。  |
| 访问日志数据          | 存储 30 天。  |
