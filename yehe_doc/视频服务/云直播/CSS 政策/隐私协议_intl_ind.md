## 1\. PENGANTAR

Modul ini berlaku jika Anda menggunakan fitur Cloud Streaming Services (“**Fitur**”). Modul ini tergabung dalam kebijakan privasi yang terdapat di [Kebijakan Privasi](https://intl.cloud.tencent.com/document/product/301/17345). Istilah-istilah yang digunakan tetapi tidak didefinisikan dalam Modul ini memiliki arti yang ditentukan dalam Kebijakan Privasi. Jika terdapat perbedaan antara Kebijakan Privasi dan Modul ini, maka Modul inilah yang akan berlaku apabila terjadi inkonsistensi.

## 2\. PENGONTROLAN

Pengontrol informasi pribadi yang dijelaskan dalam Modul ini adalah sebagaimana yang ditentukan dalam Kebijakan Privasi.

## 3\. KETERSEDIAAN

Fitur ini tersedia untuk pengguna secara global.

## 4\. CARA KAMI MENGGUNAKAN INFORMASI PRIBADI

Kami akan menggunakan informasi dengan cara berikut dan sesuai dengan dasar hukum berikut:

| **Informasi Pribadi**                                     | **Penggunaan**                                                      | **Dasar Hukum**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Data Log Penggunaan**: kecepatan bingkai langsung, kecepatan bit, lalu lintas, bandwidth, stempel waktu, catatan penggunaan fungsi konfigurasi (seperti perekaman, transcoding, watermarking, dan tangkapan layar), APPID, UIN  | Kami menggunakan informasi ini:<li>untuk memastikan fungsi Fitur sesuai kebutuhan, <br><li>untuk memastikan bahwa bug terdeteksi dan diperbaiki, serta <br><li>untuk tujuan penagihan. Harap perhatikan bahwa data ini disimpan dalam fitur TencentDB for MySQL kami. | Kami memproses informasi ini karena kami perlu melaksanakan kontrak kami dengan Anda untuk menyediakan Fitur. |
| **Data Penagihan:** penagihan bandwidth, jumlah saluran transcoding, jumlah rekaman, jumlah tangkapan layar, APPID, UIN  | Kami menggunakan informasi ini untuk tujuan penagihan.<br/>Harap perhatikan bahwa data ini disimpan dalam fitur TencentDB for MySQL kami. | Kami memproses informasi ini karena kami perlu melaksanakan kontrak kami dengan Anda untuk menyediakan Fitur. |
| **Data Log Akses** (dari pengguna akhir)**:** waktu permintaan, IP klien, nama domain yang diakses, URL, bandwidth dan volume data saat mengakses streaming langsung, ID negara, provinsi, operator, kode status HTTP, informasi perujuk HTTP, durasi pemutaran, informasi Agen Pengguna, Parameter rentang, Metode HTTP, ID streaming, cache HIT/MISS, IP node, wilayah server, negara server, deskripsi file koneksi, APPID, UIN  | Kami menggunakan informasi ini:<li>untuk memastikan fungsi Fitur sesuai kebutuhan, dan<br><li>untuk memastikan bahwa bug dan error terdeteksi dan diperbaiki.<br/>Harap diperhatikan bahwa data ini disimpan dalam fitur Cloud Log Service (CLS) kami. | Kami memproses informasi ini karena kami perlu melaksanakan kontrak kami dengan Anda untuk menyediakan Fitur. |
