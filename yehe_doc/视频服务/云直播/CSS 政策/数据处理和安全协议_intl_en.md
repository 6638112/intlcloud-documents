## 2. PROCESSING

We will process the following data in connection with the Feature:

| **Personal Information**                                     | **Use**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Configuration Data:**<li>Domain name configuration (configure authentication, bandwidth cap configuration, https configuration (including ssl certificate), region configuration, configure template, origin server configuration);<br><li>Recording template (template name, recording file format, recording single file duration, retention period, resuming timeout (only for hls), Record to VOD subapplication);<br><li>Watermark template (template name, watermark picture, watermark location);<br><li>Transcoding template (template name, bit rate, live screen height, GOP, encoding method);<br><li>Screenshot template (template name, screenshot frequency, cos location of screenshot storage, screenshot file name, Smart Porn Detection). | We only process this data for the purposes of providing the Feature to you and your end users, in accordance to your specific configuration.<br/>Please note that this data is stored in our TencentDB for MySQL feature. |
| **Live Recording Videos** (if you opt to record videos of live streams) | We only process this data for the purposes of providing the Feature to you.<br/>Please note that this data is stored in our Video on Demand (VOD) feature. |
| **Live Screenshots** (if you opt to record live screenshots), including the resulting live screenshots following porn detection (if you opt to enable the Porn Detection Configuration function) | We only process this data for the purposes of providing the Feature to you.<br/>Please note that this data is stored in our Cloud Object Storage (COS) feature .<br/>Please note that if you enable the Porn Detection Configuration function of this Feature, such data stored in COS will be transmitted to our Image Moderation System (IMS) feature, in order to provide you with this Feature as required (including to review and identify pornographic information and return results regarding the same). IMS will retain the live screenshots only for so long as necessary to complete the porn detection process and return results regarding the same to you. |
