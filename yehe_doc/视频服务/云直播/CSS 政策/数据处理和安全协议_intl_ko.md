## 1\. 배경

Cloud Streaming Services 기능("**기능**”)을 사용하는 경우 이 모듈이 적용됩니다. 이 모듈은 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 처리 및 보안 계약에 포함됩니다. 이 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에서 정의된 것과 같은 의미를 갖습니다. DPSA와 이 모듈 간에 상충이 있는 경우, 이 모듈은 상충되지 않는 범위 내에서 적용됩니다.

## 2\. 처리

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **이용 목적**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **구성 데이터:**<li>도메인 이름 구성(구성 인증, 대역폭 한도 구성, https 구성(SSL 인증서 포함), 지역 구성, 구성 템플릿, 원본 서버 구성)<br><li>녹화 템플릿(템플릿 이름, 녹화 파일 형식, 녹화 단일 파일 지속 시간, 보유 기간, 재개 타임아웃(HLS만 해당), VOD로 녹화 하위 애플리케이션)<br><li>워터마크 템플릿(템플릿 이름, 워터마크 그림, 워터마크 위치)<br><li>트랜스코딩 템플릿(템플릿 이름, 비트 전송률, 라이브 화면 높이, GOP, 인코딩 방식)<br><li>스크린샷 템플릿(템플릿 이름, 스크린샷 빈도, 스크린샷 스토리지의 COS 위치, 스크린샷 파일 이름, 스마트 포르노 감지) | 당사는 귀하의 개별 구성에 따라 귀하 및 귀하의 최종 사용자에게 기능을 제공하는 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 TencentDB for MySQL 기능에 저장된다는 점에 유의하십시오. |
| **라이브 녹화 비디오**(라이브 스트리밍 비디오를 녹화하도록 선택하는 경우) | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 당사의 주문형 비디오(VOD) 기능에 저장됩니다. |
| 포르노 감지 이후 결과에 표시되는 라이브 스크린샷을 포함하여(포르노 감지 구성 기능을 사용하도록 선택하는 경우) **라이브 스크린샷**(라이브 스크린샷을 기록하도록 선택하는 경우) | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 당사의 클라우드 객체 스토리지(COS) 기능에 저장됩니다.<br/>이 기능의 포르노 감지 구성 기능을 사용하도록 설정하는 경우, COS에 저장된 해당 데이터는 요구된 바와 같이 귀하에게 이 기능을 제공하기 위한 목적으로(포르노그래픽 정보를 검토 및 식별하고 이와 관련된 결과를 반환하기 위한 목적 포함) 당사의 이미지 조정 시스템(IMS) 기능에 전송됩니다. IMS는 포르노 감지 프로세스를 완료하고 이와 관련된 결과를 귀하에게 반환하는 데 필요한 기간 동안만 라이브 스크린샷을 보관합니다. |

## 5\. 데이터 보존

당사는 기능과 관련하여 처리된 개인 데이터를 다음과 같이 저장합니다.

| **개인 정보**                                | **보존 정책**                                         |
| ------------------------------------------------------- | ------------------------------------------------------------ |
| 구성 데이터 라이브 녹화 라이브 스크린샷 | 당사는 귀하가 선택한 데이터 보존 시간 제한 동안 데이터를 보존합니다. 귀하가 계정을 삭제하거나 기능의 사용을 종료하는 경우 당사는 60일 후에 이러한 데이터를 삭제합니다. |

## 6\. 특수 조건

개인 데이터 처리에 동의할 수 있는 최소 연령을 만족하는 개인이나 부모의 동의를 얻은 최소 연령 미만의 개인인 최종 사용자만 이 기능을 사용하도록 해야 합니다. 이는 최종 사용자의 관할권에 따라 다를 수 있습니다.

귀하는 관련 법률에 따라, 그리고 당사가 관련 법률을 준수할 수 있도록, 기능에 대해 개인 데이터의 처리와 관련하여 최종 사용자로부터 필요한 모든 동의를 획득하고 유지해야 한다는 것을 진술하고 보증하며 약속합니다.
