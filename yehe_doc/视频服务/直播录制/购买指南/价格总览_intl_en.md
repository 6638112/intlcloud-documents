The billable items of LVB recording include **LVB platform fees**, **VOD platform fees**, and **LVB recording fees** as shown below:
<img src="https://main.qcloudimg.com/raw/d6d26e320fa315cdd01eb3e9be69e5a8.png" width="750px"/>


| Billable Item | Billing Description | Billing Mode |
| ----------------- | ----------------- | ----------------- |
| [LVB recording fees](https://intl.cloud.tencent.com/document/product/1082/39177) | <li/>The fees incurred by using the LVB recording service, which are charged according to the monthly peak number of concurrent recording channels. <li/>The bill is displayed in the LVB billing information. | Monthly peak number of recording channels * valid day coefficient * unit price |
| [LVB fees](https://intl.cloud.tencent.com/zh/document/product/267/2818) | <li/>The fees incurred by using the LVB service, which are charged according to the LVB resource usage. <li/>The bill is displayed in the LVB billing information. | The LVB billing mode you selected |
| [VOD fees](https://intl.cloud.tencent.com/document/product/266/2838) | <li/>The fees incurred by using the VOD service, which are charged according to the VOD resource usage. <li/>The bill is displayed in the VOD billing information. | The VOD billing mode you selected |

>? Live recording will definitely incur VOD [storage fees](https://intl.cloud.tencent.com/document/product/266/14666).
