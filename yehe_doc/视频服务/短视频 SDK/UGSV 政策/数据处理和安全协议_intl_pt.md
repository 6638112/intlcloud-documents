## 1\. CONSIDERAÇÕES INICIAIS

Este Módulo se aplica se você usar o User Generated Short Video SDK (“**Recurso**”). Esse Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em  (“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)”). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.

## 2\. PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados da licença do cliente (fornecidos por você):** dados para permitir que você desbloqueie o acesso ao SDK, ou seja, nome do pacote, ID do pacote, ID do aplicativo, URL da licença, chave de licença | Usamos essas informações com a finalidade de fornecer o Recurso para você, incluindo a resolução de problemas.<br/>Observe que esses dados estão armazenados em nossos recursos Cloud Log Service (CLS) e TencentDB for MySQL (MySQL), e têm uma cópia de segurança no nosso recurso MySQL. |

Observe que não coletamos, processamos, armazenamos ou, de outra forma, temos acesso aos dados faciais do usuário final ou a conteúdos/ vídeos curtos gerados usando esse Recurso (“**Conteúdo UGC**”). O Conteúdo UGC é processado por você e/ou seu produto, serviço ou plataforma (conforme aplicável) que integra o Recurso.
