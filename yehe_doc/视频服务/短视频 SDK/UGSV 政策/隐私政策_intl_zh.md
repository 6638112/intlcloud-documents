## 1\. 简介

本模块仅在您使用短视频 SDK（User Generated Short Video SDK，简称“**功能**”）时适用。本模块已纳入隐私政策（简称“[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)”）中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，本模块适用。

## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. 可用性

本功能可供全球用户使用。

## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **最终用户网络和设备信息：** 连接/WiFi 状态、系统属性、设备型号、操作系统、设备屏幕状态（水平或竖直）、用户 IP | 我们使用此类信息为您提供本功能，确保本功能按要求运行，并进行故障排除。<br/>请注意，此类数据存储在我们的日志服务 (CLS) 和云数据库 MySQL (MySQL) 功能中，并备份在我们的 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合约向您提供本服务而言是必要的。 |
| **运行数据：** <li>您的控制台配置信息（您的腾讯云控制台配置信息）；<li>您的后端日志文件；<li>您的仪表板信息（例如与您的网络波动和音频/视频质量问题相关的信息）；<li>关于 SDK 使用情况的日志数据（例如 SDK是否正常运行，以及发生滞后的情况）； <li>客户 SDK 版本号；<li>最终用户操作系统类型 | 我们使用此类信息进行故障排除和运维分析。<br/>请注意，此类数据存储在我们的 CLS 和 MySQL 功能中，并备份在我们的 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合约向您提供本服务而言是必要的。 |

## 5\. 我们如何存储和共享个人信息

见隐私政策中的规定，包括 Aceville Pte Ltd。 

## 6\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息**                | **保留政策**                                         |
| --------------------------------------- | ------------------------------------------------------------ |
| 最终用户网络和设备信息 | 存储 14 天。在特殊情况下（例如您的帐户受到攻击或黑客攻击），我们将尽快删除此类数据。 |
| 运行数据                          | 存储 14 天。在特殊情况下（例如您的帐户受到攻击或黑客攻击），我们将尽快删除此类数据。 |
