### 功能说明



移动直播目前提供简单易上手的**基础版 SDK**，可通过移动直播基础版 License 使用相应的直播功能，包括直播推流、直播播放和基础美颜（美白、磨皮等）等。我们之后将推出更全能的**企业版 SDK**，带给您极致的直播体验。


### SDK 与授权 License 

您可以通过申请测试的移动直播基础版 License，或购买正式的移动直播基础版 License 对移动直播基础版 SDK 进行使用。
