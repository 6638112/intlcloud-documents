

## 1.	はじめに

本モジュールは、お客様が Real-Time Cloud Rendering 機能（以下「**本機能**」）を使用する場合に適用されます。本モジュールは、[**プライバシーポリシー**](https://intl.cloud.tencent.com/document/product/301/17345)に掲載されているプライバシーポリシーに組み込まれます。本モジュールにおける用語のうち本書内に定義のないものは、プライバシーポリシー内で定義された意味を有するものとします。プライバシーポリシーと本モジュールとの間に矛盾がある場合には、その不一致の範囲で本モジュールを優先するものとします。

## 2.	管理者

本モジュールに記載されている個人情報の管理者は、プライバシーポリシーに明記されている通りです。

## 3.	可用性

本機能は全世界のユーザーが利用可能ですが、最適なパフォーマンスを得るためには、選択したサービス地域と同じ国/地域にいるユーザーを主な対象としています。

## 6.	データ保持

当社は、以下の条件に従って個人情報を保持します。

| **個人 情報**      | **保持ポリシー**                                         |
| ------------------------------ | ------------------------------------------------------------ |
| アプリケーションデータ               | お客様が本機能の サブスクリプションを終了するか、またはお客様のアカウントを終了/削除する（いずれか 早い方）まで保存されます。 |
| 操作および保守データ | お客様が本機能の サブスクリプションを終了するか、またはお客様のアカウントを終了/削除する（いずれか 早い方）まで保存されます。 |
| ログ情報                | お客様が本機能のサブスクリプションを終了する日、またはお客様の アカウントを終了/削除 した日（いずれか早い方）の翌日から 60 日間保存されます。 |
| 操作データ                 | お客様が本機能の サブスクリプションを終了するか、またはお客様のアカウントを終了/削除する（いずれか 早い方）まで保存されます。 |
| 課金データ                   | お客様が本機能のサブスクリプションを終了した日、またはお客様の アカウントを終了/削除 した日（いずれか早い方）の翌日から12か月間保存されます。 |
