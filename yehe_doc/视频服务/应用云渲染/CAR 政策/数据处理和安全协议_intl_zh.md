## 1.	背景

本模块仅在您使用 Real-Time Cloud Rendering 功能（简称“**功能**”）时适用。本模块已纳入位于 [**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347) 的数据处理和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，本模块适用。

## 2.	处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **上传的 3D 应用程序渲染数据**                      | 我们仅为了向您提供本功能而处理此类数据。 |
| **配置数据**：应用程序名称、起始路径、进程列表、应用程序启动参数、窗口捕获模式 | 我们仅为了向您提供本功能而处理此类数据。 |
| **并发资源数据**：应用程序 ID、应用程序名称、应用程序状态、应用程序类型、应用程序配置信息、项目 ID、创建时间、并发 ID、并发类型、并发区域、并发状态。   | 我们仅为了向您提供本功能而处理此类数据。 |
