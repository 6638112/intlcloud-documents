## 1.	CONSIDERAÇÕES

Esse Módulo se aplica se você usar o recurso Real-Time Cloud Rendering (“**Recurso**”). Esse Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em [**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.

## 2.	PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de renderização de aplicativos 3D carregados**                      | Apenas processamos  esses dados com a finalidade de fornecer o Recurso a você. |
| **Dados de configuração**: nome do aplicativo, caminho inicial, lista de processos,  parâmetros de inicialização do aplicativo e modo de captura de janelas | Apenas processamos  esses dados com a finalidade de fornecer o Recurso a você. |
| **Dados de recursos simultâneos**: ID da aplicação, nome da aplicação,  status da aplicação, tipo de aplicação, informações de configuração da aplicação,  ID do projeto, CreateTime, ID da requisição simultânea, tipo de requisição simultânea, região da requisição simultânea e status da requisição simultânea. | Apenas processamos  esses dados com a finalidade de fornecer o Recurso a você. |
