## 1.	배경

Real-Time Cloud Rendering 기능("**기능**")을 사용하는 경우 이 모듈이 적용됩니다. 이 모듈은 [**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347)에 있는 데이터 처리 및 보안 계약에 포함됩니다. 본 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에서 정의된 것과 같은 의미를 갖습니다. DPSA와 본 모듈 간에 상충이 있는 경우, 해당 상충 범위까지 본 모듈이 적용됩니다.

## 2.	처리

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **사용**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **업로드된 3D 응용 프로그램 렌더링 데이터**                      | 당사는 귀하에게 기능을 제공하기 위한 목적으로만  이 데이터를 처리합니다. |
| **구성 데이터**: 응용 프로그램 이름, 시작 경로, 프로세스 목록,  응용 프로그램 시작 매개 변수, 창 캡처 모드 | 당사는 귀하에게 기능을 제공하기 위한 목적으로만  이 데이터를 처리합니다. |
| **동시성 리소스 데이터**: ApplicationId, ApplicationName,  ApplicationStatus, ApplicationType, 응용 프로그램 구성 정보,  ProjectId, CreateTime, ConcurrentId, ConcurrentType, ConcurrentRegion, ConcurrentStatus. | 당사는 귀하에게 기능을 제공하기 위한 목적으로만  이 데이터를 처리합니다. |
