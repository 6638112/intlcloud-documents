## 기본 개념

[](id:b1)
### MPS란?
Media Processing Service(MPS)는 대용량 멀티미디어 데이터에 특화된 클라우드 트랜스코딩과 오디오 프로세싱 서비스를 제공합니다. 필요에 따라 클라우드 스토리지의 비디오 파일을 OTT, PC 혹은 모바일에서 재생 가능한 형식으로 트랜스코딩할 수 있으며, 각 플랫폼에서 비디오 파일을 다양한 비트레이트와 해상도로 전환할 수 있습니다. 워터마크, 비디오 화면 캡처, 스마트 커버, 스마트 편집 등 MPS 서비스를 제공합니다.

[](id:b2)
### 비디오 비트레이트이란 무엇입니까?
비디오 비트레이트는 데이터가 전송되는 동안 단위 시간당 전송되는 데이터 비트의 수이며 일반적으로 우리가 사용하는 단위는 kbps입니다.
기본 계산법: 비트레이트(kbps) = 파일 크기(KB) × 8 ÷ 시간(초)입니다.

[](id:b3)
### MPS TESHD란 무엇인가요?
TESHD는 이미지 품질 보장 및 향상의 전제 하에, 다양한 유형의 비디오를 최적화하여, 대역폭을 줄이고 고객 리소스를 절약하여 사용자에게 낮은 대역폭으로 고화질 시청 경험을 제공합니다.

[](id:b4)
### MPS는 하드 디코딩입니까? 아니면 소프트 디코딩입니까?
MPS는 클라우드 트랜스코딩, 워터마크 중첩, 비디오 화면 캡처, 스마트 커버, 스마트 편집 등과 같은 MPS 서비스를 제공하며, 디코딩 기능이 없습니다.

[](id:b5)
### MPS 서비스의 주요 프로세스는 무엇입니까?
MPS를 예로 들면 워크플로에는 주로 워크플로 구성, 트랜스코딩 트리거, 트랜스코딩 작업 처리 및 이벤트 알림 보내기 등이 포함됩니다. [워크플로](https://intl.cloud.tencent.com/document/product/1041/33475)는 다음과 같습니다.

## 기능 지원
[](id:f1)
### 테스트 계정이 제공됩니까?
MPS는 현재 테스트 계정이 제공되지 않습니다.

[](id:f2)
### 어떤 화면 캡처 유형이 지원됩니까?
MPS 화면 캡처 유형에는 시점 화면 캡처(단일 화면 캡처), 샘플링 화면 캡처(다중 화면 캡처), 스프라이트 이미지 화면 캡처가 포함됩니다. 자세한 내용은 [화면 캡처 템플릿](https://intl.cloud.tencent.com/document/product/1041/33486)을 참고하십시오.

[](id:f3)
### 정규식이 지원됩니까?
지원되지 않습니다.

[](id:f4)
### 멀티태스킹 트랜스코딩이 지원됩니까?
멀티태스킹 트랜스코딩을 지원합니다.

[](id:f5)

### 전체 속도에 영향을 줍니까?
아니오.

[](id:f6)
### 비디오 스플라이싱이 지원됩니까?
지원합니다. 관련 인터페이스를 호출하여 처리해야 합니다.(콘솔 미지원) 호출 방법 관련 자세한 내용은 [비디오 편집](https://intl.cloud.tencent.com/document/product/1041/37460)을 참고하십시오.

[](id:f7)
### 비디오 압축이 지원됩니까?
지원되지 않습니다.

[](id:f8)
### 비디오 각도 회전이 지원됩니까?
지원되지 않습니다.
MPS는 클라우드 스토리지의 오디오 및 비디오 파일을 OTT, PC 또는 모바일에서 재생하기에 적합한 형식으로 트랜스코딩하고 워터마크 및 화면 캡처 추가 등의 작업을 수행합니다.

[](id:f9)
### 이미지를 동영상으로 변환하는 기능이 지원됩니까?
지원되지 않습니다.

[](id:f10)
### AI 스마트 분석 태그 수 확장이 지원됩니까?
태그는 Top5까지 표시됩니다. 현재 조정이 지원되지 않습니다. 더 많은 태그를 표시하려면 ‘프레임 태그’ 사용을 권장합니다.

[](id:f11)
### 비디오의 음성을 텍스트로 변환하는 기능이 지원됩니까?
MPS 비디오 콘텐츠 ​​인식이 비디오에 대한 콘텐츠 인식을 수행한 후. 실행 결과를 비디오 음성에서 변환된 텍스트로 제공합니다.

[](id:f12)

### 스마트 커버를 다이내믹 이미지로 설정할 수 있습니까?
지원되지 않습니다.
스마트 커버 추출은 비디오 콘텐츠 분석을 통해, 비디오에서 하나 또는 여러 개의 화면 캡처를 추천 커버로 선택하는 것으로, 다이내믹 커버는 스마트 생성될 수 없습니다. 애니메이션 이미지 커버는 템플릿 설정만 지원됩니다. 구체적인 작업은 [VOD 애니메이션 이미지 생성](https://intl.cloud.tencent.com/document/product/266/33941)을 참고하십시오.

[](id:f13)
### 워크플로가 동시에 여러 디렉터리를 수신할 수 있습니까?
MPS는 다중 출력 디렉터리를 설정할 수 없지만, 다중 출력 bucket을 설정하거나 여러 워크플로를 설정하여 다중 출력 디렉터리 기능을 구현할 수 있습니다.
