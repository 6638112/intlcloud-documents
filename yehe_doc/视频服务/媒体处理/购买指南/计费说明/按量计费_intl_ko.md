MPS는 일 결산(일 단위로 결제)과 월 결산(월 단위로 결제)의 두가지 모드로 요금을 부과합니다. 요금 방식은 기본적으로 일 결산으로 적용하며, 변경이 필요한 경우 영업팀에 문의하시기 바랍니다.

### 과금 안내

+ 과금 규칙: 비디오 트랜스코딩 요청에 따른 인코딩 방식, 해상도, 비디오 트랜스코딩 시간을 기반으로 하는 일 결산 과금 방식입니다.
+ 과금 공식: 비디오 트랜스코딩 요금 = 출력 비디오 길이(분) x 코덱 및 해상도에 따른 트랜스코딩 단가(USD/분)
+ 트랜스코딩 작업을 한 번 수행할 때마다, 트랜스코딩 요금이 한 번 부과되며, 트랜스코딩 시간이 1분 미만인 경우 1분으로 계산됩니다.
 
### 과금 예시

예를 들어, 1월 1일에 미디어 처리를 위해 일반 트랜스코딩 서비스를 사용하고 뭄바이 리전을 사용하여 60분 길이의 H.264 인코딩을 통해 2560 × 1440 해상도의 비디오를 전송했습니다. 1600 × 980 해상도, 100분 길이의 비디오를 전송하는 경우 1월 2일에 지불해야 하는 트랜스코딩 요금은 다음과 같습니다.
트랜스코딩 요금 = 0.0355(USD/분) x 60(분) + 0.023(USD/분) x 100(분) = 4.43USD
