## 종량제-일 결산[](id:p1)
[Daily Billing](https://intl.cloud.tencent.com/document/product/1041/33478#day)을 사용하면 실제 사용량에 따라 VOD 서비스가 과금됩니다. 잔액 부족으로 인해 계정이 연체되는 것을 방지하려면 아래 단계에 따라 콘솔에서 즉시 계정을 충전하십시오.
1. Tencent Cloud 콘솔에 로그인하고 오른쪽 상단 모서리에 있는 [**과금 센터**](https://console.cloud.tencent.com/account/recharge)를 클릭한 다음 지금 충전을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/8a90294c5bbfd0a90a09fe6862ab31a4.png)

2. 결제 방법을 선택하고 충전 금액을 입력한 후 **결제하기**를 클릭합니다.

## 종량제-월 결산
[Monthly Billing](https://intl.cloud.tencent.com/document/product/1041/33478#month)에 대한 자세한 내용은 Tencent Cloud 영업 담당자에게 문의하십시오. 가격은 계약을 기준으로 합니다.

## 선불-리소스 팩[](id:p2)
리소스 팩의 경우 콘솔에서 셀프 서비스 구매 및 갱신이 지원되며 구체적인 작업 단계는 다음과 같습니다.
1. [미디어 프로세싱 콘솔](https://console.cloud.tencent.com/mps)에 로그인한 후, 콘솔 개요 페이지 우측의 [**리소스 팩 구매**](https://buy.cloud.tencent.com/mps)를 클릭하여 리소스 팩 구매 페이지로 이동합니다.
2. 리소스 팩 구매 페이지에서 필요에 따라 리소스 팩을 구매합니다.

>!
>- 리소스 팩은 중첩 구매를 지원합니다. 즉 유효 기간 동안 리소스 사용량은 중첩되고 여러 개의 리소스 팩은 만료 시간에 따라 순차적으로 차감됩니다.
>- 각 리소스 팩의 유효 기간은 독립적으로 계산되며 여러 리소스 팩의 유효 기간은 중첩되지 않습니다.
>- 미디어 프로세싱에서 제공하는 리소스 팩은 유료 서비스에 해당하는 미디어 프로세싱 제품의 사용에 대해서만 차감될 수 있습니다.
>- 리소스 팩 소진 시 자동으로 [Pay-As-You-Go](https://intl.cloud.tencent.com/document/product/1041/33478) 과금 방식으로 전환되어 차감됩니다.
