## 작업 시나리오
COS에 파일을 업로드하거나 콘솔에서 수동으로 시작한 미디어 처리 작업 또는 API 호출에 의해 미디어 처리 작업이 자동으로 트리거된 후 [작업 관리](https://console.cloud.tencent.com/mps/tasks) 페이지에서 작업을 보고 관리할 수 있습니다.

### 작업 쿼리[](id:tquery)
1. 왼쪽 사이드바에서 [작업 관리](https://console.cloud.tencent.com/mps/tasks)를 선택합니다. 목록에는 현재 계정에서 시작한 작업이 표시됩니다.
2. 오른쪽 상단의 검색 상자에 **작업 ID**를 입력하여 작업을 검색하거나 **상태** 헤더를 클릭하여 보고 싶은 작업 유형을 선택합니다.


### 작업 생성[](id:cquery)
1. [작업 관리](https://console.cloud.tencent.com/mps/tasks) 페이지에서 [작업 생성](https://console.cloud.tencent.com/mps/tasks/create)을 클릭합니다.
2. 처리할 파일과 출력 경로를 선택하고 트랜스 코딩 매개변수를 지정합니다.

### 작업 운영[](id:operate)
작업의 세부 사항 펼치기, 작업 재시작, 작업 종료, 작업의 소스 비디오 재생이 지원됩니다.
- 세부 사항 펼치기: **세부 사항 펼치기**를 클릭하면 모든 서브 작업의 정보를 볼 수 있습니다.
- 재시작: ‘완료’된 작업의 경우 **재시작**을 클릭하여 서브 작업을 다시 실행할 수 있습니다.
- 종료: **종료**를 클릭하여 ‘대기’ 상태인 작업을 취소할 수 있습니다.
- 소스 비디오 재생: **소스 비디오 재생**을 클릭하여 작업의 소스 비디오를 재생할 수 있습니다.

### 서브 작업 리스트[](id:slist)
**세부 사항 표시**를 클릭하여 작업의 서브 작업 정보를 볼 수 있습니다:

| 헤더     | 설명 |
| -------- | -------- |
| 서브 작업 번호 | 작업의 자동 증가 시퀀스 |
| 서브 작업 상태 | 대기 중, 진행 중, 성공 또는 실패할 수 있는 서브 작업의 상태 |
| 서브 작업 유형 | 오디오/비디오 트랜스 코딩, 스크린샷, 콘텐츠 검색, 심사 또는 오디오/비디오 향상 등 서브 작업 유형 |
| 시작 시간   | 서브 작업이 시작된 시간 |
| 종료 시간   | 서브 작업이 종료된 시간 |
| 출력 | 서브 작업의 출력 경로(콘텐츠 검색 또는 심사 작업에 대한 출력 경로가 없음) |
| 운영 | 아래의 [서브 작업 작업](#squery) 참고 |

### 서브 작업 운영[](id:squery)
서브 작업의 **세부 사항**를 볼 수 있을 뿐만 아니라 해당 오디오/비디오/이미지를 **재생/조회** 및 **다운로드**할 수 있습니다:

| 운영  | 설명 |
| -------- | -------- |
| 세부 사항      | 서브 작업의 기본 정보, 템플릿, 입력 및 출력 정보를 볼 수 있습니다.                   |
| 재생/조회 | 오디오/비디오 트랜스 코딩 서브 작업의 경우 재생을 클릭하여 오디오/비디오를 재생할 수 있습니다. 스크린샷 서브 작업의 경우 조회를 클릭하여 이미지를 조회할 수 있습니다.        |
| 다운로드      | **다운로드**를 클릭하여 서브 작업의 출력 파일을 다운로드할 수 있습니다. 오디오/트랜스 코딩, 스크린샷 및 오디오/비디오 향상 작업만 이 작업이 지원됩니다. <br>**참고: 현재 여러 스크린샷을 찍은 경우 첫 번째 스크린샷만 다운로드할 수 있습니다. 향후 패키지 다운로드 지원을 추가할 예정입니다.** |
