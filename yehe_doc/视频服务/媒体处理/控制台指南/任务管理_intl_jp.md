## 操作シナリオ
COSへのアップロード、トランスコードの手動での開始、 API呼び出しのいずれかの方法でメディア処理を開始すると、開始したタスクの情報はすべて[タスク管理](https://console.cloud.tencent.com/mps/tasks)で確認および管理することができるようになります。

### タスクの照会[](id:tquery)
1. [タスク管理](https://console.cloud.tencent.com/mps/tasks)をクリックし、タスク管理ページに進みます。このリストにはそのアカウント下で開始したメインタスクの記録が表示されます。
2. リスト右上隅の**タスクID検索**ボックスまたはリストの**タスクステータスのフィルタリング**項目で、リストから必要なメインタスク情報をフィルタリングします。


### タスクの作成[](id:cquery)
1. [タスク管理](https://console.cloud.tencent.com/mps/tasks)ページに進み、[タスクの作成](https://console.cloud.tencent.com/mps/tasks/create)をクリックしてタスク作成ページに進みます。
2. 処理したいビデオファイル、出力パス、トランスコードテンプレートなどの情報を選択し、タスクを開始します。

### タスクの操作[](id:operate)
タスク管理でサポートされるタスク操作には、詳細表示、タスク再起動、タスク終了、ソースビデオの再生があります。
- 詳細表示：**詳細を見る**をクリックすると、そのメインタスク下のすべてのサブタスクの情報が表示されます。
- 再起動：ステータスが「完了」になっているタスクは再起動できます。**再起動**をクリックすると、そのメインタスク下のすべてのサブタスクが再起動して実行されます。
- 終了：ステータスが「キューイング中」になっているタスクは終了できます。**終了**をクリックすると、タスクキューがキャンセルされ、それ以上実行されなくなります。
- ソースビデオの再生：**ソースビデオの再生**をクリックすると、そのタスクの入力ビデオファイルを取得して再生します。

### サブタスクリスト[](id:slist)
タスクリストで**詳細を見る**をクリックすると、そのメインタスク下のすべてのサブタスクの情報が表示されます。サブタスクの説明は次のとおりです。

| 列名     | 説明 |
| -------- | -------- |
| サブタスク番号 | メインタスク下のサブタスクに自動採番を行い、サブタスクを区別します |
| サブタスクステータス | サブタスクのステータス情報です。キューイング中、実行中、成功、失敗があります |
| サブタスクタイプ | サブタスクのタイプ情報です。オーディオビデオトランスコーディング、スクリーンキャプチャ、コンテンツ理解、審査、オーディオビデオエンハンスメントなどがあります |
| 開始時間   | このサブタスクの開始時間です |
| 終了時間   | このサブタスクの終了時間です |
| 出力位置   | このサブタスクの出力位置です（サブタスクのタイプがコンテンツ理解、審査の場合は出力位置はありません） |
| 操作 | 下記の[サブタスクの操作](#squery)の説明をご参照ください |

### サブタスクの操作[](id:squery)
サブタスクリストでは、**詳細**、**再生/確認**および**ダウンロード**操作をサポートしています。

| 操作名  | 説明 |
| -------- | -------- |
| 詳細      | 「詳細」をクリックすると、サブタスクの基本情報、テンプレート、インプット、 アウトプット情報が表示されます                  |
| 再生/確認 | オーディオビデオトランスコードサブタスクの場合、「再生」をクリックするとオーディオビデオを再生できます。スクリーンショットサブタスクの場合は、「確認」をクリックするとイメージを表示できます。        |
| ダウンロード      | **ダウンロード**をクリックすると、このサブタスクの出力ファイルのうち、サブタスクのタイプがオーディオビデオトランスコーディング、スクリーンキャプチャ、オーディオビデオエンハンスメントタイプのタスクのみダウンロード操作を行うことができます<br>**注意：現在はスクリーンキャプチャが複数枚ある場合は、1枚目の画像のみダウンロードできます。今後、パッケージ化によるダウンロードもサポートする予定です** |
