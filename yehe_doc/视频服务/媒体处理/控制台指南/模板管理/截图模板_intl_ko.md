MPS는 스키마에 직접 추가할 수 있는 사전 설정 스크린샷 템플릿을 제공합니다. 현재 시점 스크린샷, 샘플링된 스크린샷 및 이미지 스프라이트 스크린샷의 세 가지 유형의 스크린샷이 지원됩니다. **템플릿 생성**을 클릭하여 자신만의 스크린샷 템플릿을 사용자 정의할 수도 있습니다.

## 시점 스크린샷[](id:time)

|  설정 항목| 설명 |
| --- | ----|
| 템플릿 이름 | 중국어, 영어, 숫자, 언더바(_), 붙임표(-), 마침표(.)만 지원하며, 최대 64자까지 입력할 수 있습니다.|
| 이미지 형식 | JPG |
| 이미지 사이즈 | 이미지 폭과 높이 128px - 4096px로 제한|

## 이미지 스프라이트 스크린샷[](id:sprite)
스크린샷 유형에서 스프라이트 이미지 스크린샷를 선택합니다.

|  설정 항목| 설명 |
| --- | ----|
| 템플릿 이름| 중국어, 영어, 숫자, 언더바(_), 붙임표(-), 마침표(.)만 지원하며, 최대 64자까지 입력할 수 있습니다.|
| 이미지 형식 | JPG |
| 이미지 사이즈| 이미지 폭과 높이 128px - 4096px로 제한|
| 샘플링 간격| 샘플링 간격은 백분율(%) 또는 시간(s)에 따르며, 백분율 선택 시 간격은 100을 초과할 수 없습니다.|
| 작은 이미지 행 수| 자연수로 입력하며, 작은 이미지 행 수에 작은 이미지 열 수를 곱한 값이 100을 초과해서는 안됩니다.|

## 애니메이션 이미지 스크린샷[](id:move)
시스템에서 애니메이션 이미지 사전 설정 템플릿을 제공하며 워크플로 관리에서 직접 사용할 수 있습니다. 이외에도 비즈니스 요구사항에 맞게 애니메이션 이미지 템플릿을 사용자 정의하여 생성할 수 있습니다. 애니메이션 이미지 템플릿으로 전환해 **애니메이션 이미지 템플릿 생성**을 클릭하여 템플릿 사용자 정의 설정으로 이동합니다.
애니메이션 이미지의 시점은 워크플로 관리에서 설정해야 합니다. 템플릿은 이미지 유형, 프레임 레이트, 이미지 품질, 이미지 사이즈만 설정할 수 있으며, 자세한 설정 방법은 [워크플로 관리](https://intl.cloud.tencent.com/document/product/1041/33485)를 참고하십시오.

|  설정 항목| 설명 |
| --- | ----|
| 템플릿 이름  | 중국어, 영어, 숫자, 언더바(_)만 지원하며, 최대 64자까지 입력할 수 있습니다.  |
| 이미지 유형  | WEBP, GIF 지원  |
| 프레임 레이트  | 1fps | 30fps  |
| 이미지 품질  | 0 - 100 (이미지 품질은 애니메이션 이미지의 화질을 의미하며 화질이 높을수록 애니메이션 이미지 저장 용량이 커짐)  |
| 이미지 사이즈  | 이미지 폭과 높이는 0 및 128px - 4096px로 제한  |

생성한 애니메이션 이미지 템플릿은 템플릿 리스트에 표시되며, 해당 리스트에서 애니메이션 이미지 템플릿 이름, 이미지 유형, 프레임 레이트, 이미지 품질, 이미지 사이즈, 템플릿 유형 등의 정보를 조회할 수 있습니다. 또한 사용자 정의 템플릿을 조회, 편집, 삭제할 수 있습니다. 시스템 사전 설정 템플릿은 조회만 가능하며 편집 및 삭제는 지원하지 않습니다.

### 사전 설정 템플릿

| 템플릿 ID | 이미지 형식(Format) | 해상도(Resolution) | 프레임 레이트(FPS) |
| ------- | ------------------ | -------------------- | ----------- |
| 20000   | GIF                | 원본과 동일                 | 2           |
| 20001   | WebP               | 320 * 비율에 따라 축소/확대                 | 2           |
