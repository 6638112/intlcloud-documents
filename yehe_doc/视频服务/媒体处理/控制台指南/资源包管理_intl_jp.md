## ユースケース
このページでは、このアカウントで購入したMPS通常トランスコード、高速高画質トランスコーディングリソースパックの確認、ならびにリソースパック利用明細の照会をサポートします。
## リソースパックの照会
[MPSコンソール](https://console.cloud.tencent.com/mps)にログインし、[リソースパック管理](https://console.cloud.tencent.com/mps/resources?tab=package)をクリックし、「リソースパックリスト」ページに進むと、下図に示すように、そのページで購入したリソースパックの情報を確認することができます。

## 利用明細の照会
[MPSコンソール](https://console.cloud.tencent.com/mps)にログインし、[リソースパック管理](https://console.cloud.tencent.com/mps/resources?tab=package)をクリックして「リソースパックリスト」ページに進み、そのページで上方のタブを**利用明細**に切り替えると、下図のように、リソースパックの利用明細情報を確認できます。
![](https://qcloudimg.tencent-cloud.cn/raw/583c667367a63573868e1d18a44e0e23.png)
このページでは**控除日**と**リソースパックID**に基づいて、使用と控除の詳細情報の検索を行うことができます。

| 列名 | 説明     |
| ----------------- | ----------------- |
| 控除日 | この控除記録が発生した日です。 |
| リソースパックID | この控除記録が有効になったリソースパックのIDです。 |
| リソースパックタイプ | このリソースへの控除が有効になったリソースパックのタイプです。リストのリソースパックタイプから控除記録をフィルタリングできます。 |
| トランスコードタイプ | この控除記録に対応するトランスコードタイプです。毎日同じタイプのトランスコードが発生する場合は1つの控除記録に統合されます。 |
| 控除前のリソースパック残量 | この控除が発生する前の、このリソースパックの使用可能なリソース残量です。 |
| 控除後のリソースパック残量 | この控除が発生した後の、このリソースパックの使用可能なリソース残量です。 |
