사용자 정의 도메인이 VOD에 연결되면 시스템은 CNAME 도메인(접미사: .cdn.dnsv1.com)을 자동으로 할당합니다. VOD 콘솔 [도메인 관리](https://console.cloud.tencent.com/vod/distribute-play/domain)에서 조회할 수 있습니다. 자동으로 할당된 CNAME 도메인은 직접 액세스할 수 없습니다. 도메인 이름 서비스 제공 업체에서 CNAME 설정을 완료해야 합니다. 설정이 적용되면 정상적으로 액세스할 수 있습니다.

## Tencent Cloud 설정 방법
DNS 서비스 제공 업체가 Tencent Cloud인 경우 다음 단계에 따라 CNAME 레코드를 추가 할 수 있습니다.
1. [도메인 관리](https://console.cloud.tencent.com/domain) 콘솔에 로그인하고 ‘내 도메인’ 리스트에서 CNAME 레코드를 추가할 도메인 찾은 다음 작업 범주에서 [리졸브]를 클릭합니다.
2. 지정된 도메인의 **레코드 관리** 페이지로 리디렉션된 후 [레코드 추가]를 클릭합니다.
3. 팝업 창에서 **레코드 유형**을 CNAME으로 설정하고,**호스트 레코드**에서 도메인 접두사(예: www)를 입력한 후, **레코드 값**에 CNAME 도메인을 입력하고 [확인]을 클릭하여 CNAME 레코드를 추가합니다.

	1. 호스트 레코드에 서브 도메인을 입력합니다. 예를 들어 `www.123.com`의 리졸브를 추가해야 할 경우 호스트 레코드에 'www'만 입력하면 됩니다. `123.com`의 리졸브만 추가하려는 경우 호스트 레코드를 공백으로 남겨두면 시스템이 입력창에 ‘@’를 자동으로 입력합니다. @의 CNAME은 MX 레코드의 정상적인 리졸브에 영향을 주므로 신중하게 추가하십시오.
	2. 레코드 유형은 CNAME입니다.
	3. 회선 유형(필수 항목으로, 설정하지 않을 경우 일부 사용자가 리졸브할 수 없게 됩니다. 위 이미지에서 기본적으로 China Unicom 사용자를 제외한 모든 사용자가 1.com을 사용합니다.)
	4. 레코드 값은 CNAME이 가리키는 도메인 이름입니다. 도메인 이름만 입력할 수 있습니다. 레코드가 생성된 후 도메인 이름 뒤에 ‘.’가 자동으로 추가됩니다.
	5. MX 우선 순위는 작성할 필요가 없습니다.
	6. TTL을 입력할 필요는 없습니다. 추가할 때, 시스템에서 자동으로 생성되며 기본값은 600초입니다(TTL은 캐시 시간으로 값이 작을수록 레코드 변경이 더 빨리 적용됨).


## www.net.cn 설정 방법
DNS 서비스 제공 업체가 www.net.cn인 경우 아래 단계에 따라 CNAME 레코드를 추가할 수 있습니다.
1. www.net.cn에 로그인하십시오.
2. 회원 사이트의 왼쪽 메뉴에서 [제품 관리]> [나의 클라우드 리졸브]를 클릭하여 World Wide Web Cloud 리졸브 리스트 페이지로 이동합니다.
3. 리졸브하려는 도메인을 클릭하고 리졸브 레코드 페이지로 이동하십시오.
4. 분석 레코드 페이지에 들어간 후 분석 추가 단추를 클릭하여 설정을 시작하십시오.
5. CNAME 리졸브 레코드를 설정하려면 레코드 유형을 CNAME으로 선택하십시오. 호스트 레코드는 도메인 이름의 접두사이며 임의로 입력할 수 있습니다(예 :`www`). 레코드 값은 현재 도메인이 아닌 다른 도메인으로 채워집니다. 리졸브 회선은 TTL 기본값이면 됩니다.
6. 모두 입력 후 [저장]을 클릭하여 리졸브 설정을 완료하십시오.

## Sina 설정 방법
DNS 공급자가 새 네트워크인 경우 아래 단계에 따라 CNAME 레코드를 추가할 수 있습니다.
**별명 설정(CNAME 레코드)**
즉, 별명 레코드입니다. 이러한 종류의 로깅을 통해 여러 이름을 동일한 컴퓨터에 매핑할 수 있습니다. 일반적으로 WWW와 MAIL 서비스를 모두 제공하는 컴퓨터에서 사용됩니다. 예를 들어`host.mydomain.com` (A 레코드)이라는 컴퓨터가 있습니다. 또 동시에 사용자가 서비스에 액세스할 수 있도록 WWW 및 MAIL 서비스를 제공합니다. 이 컴퓨터에 대해 WWW와 MAIL의 두 개의 별명(CNAME)을 설정할 수 있습니다.

## CNAME 인증 유효성 확인
DNS 서비스 제공 업체에 따라 CNAME 적용 시간이 조금씩 다르며, 일반적으로 30분 내에 적용됩니다. 다음 방법을 통해 CNAME이 적용되었는지 확인할 수 있습니다.
- 방법1: [시작]→[실행]→cmd 입력 후 Enter 키를 누르고 PING 명령어를 입력하여 CNAME의 적용 여부를 쿼리하여 반환된 리졸브 결과가 도메인의 CNAME 값과 일치하면 CNAME 구성이 적용됩니다.
- 방법2: [시작]→[실행]→cmd 입력 후 Enter 키를 누르고 nslookup 명령어를 입력하여 CNAME의 적용 여부를 쿼리하여 반환된 리졸브 결과가 도메인의 CNAME 값과 일치하면 CNAME 구성이 적용됩니다.
