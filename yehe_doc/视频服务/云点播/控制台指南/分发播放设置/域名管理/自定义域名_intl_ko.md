## 작업 시나리오
VOD를 활성화하면 시스템에서 기본 도메인 `xxx.vod2.myqcloud.com`을 할당하며, VOD의 모든 리소스에서 기본적으로 이 도메인을 사용하게 됩니다. [VOD 콘솔](https://console.cloud.tencent.com/vod/overview)에 로그인하여 도메인을 사용자 정의 추가하거나 리졸브할 수 있습니다.
## 전제 조건
- VOD 서비스가 신청되어 있어야 합니다. 자세한 내용은 [구매 가이드](https://intl.cloud.tencent.com/document/product/266/39506)를 참고하십시오.
- 중국 진출 비즈니스의 경우, 추가할 도메인의 ICP비안이 완료되었는지 확인하시기 바랍니다.

## 도메인 추가
1. [VOD 콘솔](https://console.cloud.tencent.com/vod/overview)에 로그인하고 왼쪽 메뉴의 [배포 및 재생 설정]>[도메인 관리]를 선택합니다.
2. [도메인 추가]를 클릭하고 팝업창에 도메인을 입력한 후 [확인]을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/6ce1d60d21bf80b237a5a46a4b1aafed.png)
>?
>- 도메인이 추가되기까지 몇 분의 시간이 소요되며, 도메인이 추가되면 도메인 리스트에서 해당 도메인의 상태, CNAME 및 도메인 유형 등의 정보를 조회할 수 있습니다.
>- Tencent Cloud 한 계정당 최대 20개의 도메인을 추가할 수 있습니다.

## 도메인 리졸브
새 도메인을 추가한 뒤, 해당 도메인의 지정 DNS 서비스 제공 업체에서 CNAME을 구성해야만 도메인을 통해 사용자의 비디오 정보에 액세스할 수 있습니다. 구체적인 작업 순서는 다음과 같습니다.
>?다음은 Tencent Cloud DNS가 DNS 서비스 제공 업체인 경우를 예로 들어 CNAME 레코드를 추가하는 과정을 설명한 것이며, DNS 서비스 제공 업체마다 작업 방식이 조금씩 다를 수 있습니다. 자세한 작업 내용은 [CNAME 설정](https://intl.cloud.tencent.com/document/product/266/42076)을 참고하십시오.

1. [도메인 관리](https://console.cloud.tencent.com/domain) 콘솔에 로그인하고 [내 도메인] 리스트에서 CNAME 레코드를 추가할 도메인 찾은 다음 작업 리스트에서 [리졸브]를 클릭하십시오.
2. 해당 도메인의 [레코드 관리] 페이지에서 [레코드 추가]를 클릭합니다.
3. 팝업창에서 [레코드 유형]을 CNAME으로 설정하고 [호스트 레코드]에 도메인 접두사(예: www)를, [레코드값]에 CNAME 도메인을 입력한 뒤 [확인]을 클릭합니다.

>! 사용자 정의 도메인만 추가하고 비디오는 추가하지 않았는데 소량의 트래픽이 발생한 경우, 이는 도메인에 액세스하는 과정에서 소량의 패킷이 반환되었기 때문으로 정상적인 현상입니다.
