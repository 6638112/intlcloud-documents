
VOD 콘솔을 통해 기본 정보 편집, 비디오 배포, Player+ 미리보기 및 Web 플레이어 코드 생성 등 비디오 관리가 가능합니다. 본 문서에서는 비디오 관리 방법에 대해 소개합니다.

## 더 많은 페이지
[VOD 콘솔](https://console.cloud.tencent.com/vod/media)에 로그인하고 [더보기]를 클릭하면 스토리지 유형을 수정할 수 있으며, 수정 가능한 스토리지 유형은 표준 스토리지와 표준IA 스토리지입니다.

## 화면 캡처 정보

1. VOD 콘솔에 로그인하여 [미디어 자원 관리]>[[비디오 관리]](https://console.cloud.tencent.com/vod/media)를 선택하면 ‘업로드 완료’ 페이지로 이동합니다.
2. 타깃 비디오 파일이 있는 행에서 [관리]를 클릭하면 ‘기본 정보’ 페이지로 이동합니다.
3. [캡처 정보]를 클릭하여 ‘캡처 정보’ 페이지로 이동합니다. 해당 페이지에서는 원본 비디오 및 처리 후 비디오의 캡처 파일 리스트(gif, 스프라이트 이미지, 샘플링 및 시간대별 캡처 등)를 확인할 수 있습니다.

>?콘솔에 표시되는 캡처 정보는 최대 100개로 모든 캡처 정보를 확인하려면 정보 표시줄 오른쪽 상단의 다운로드를 클릭하십시오.

## 비디오 배포
1. VOD 콘솔에 로그인하여 [미디어 자원 관리]>[[비디오 관리]](https://console.cloud.tencent.com/vod/media)를 선택하면 ‘업로드 완료’ 페이지로 이동합니다.
2. 타깃 비디오 파일이 있는 행에서 [관리]를 클릭하면 ‘기본 정보’ 페이지로 이동합니다.

   
## Player+ 미리보기
>!Player+는 ABS 트랜스 코딩된 비디오에 대한 미리보기만 지원합니다. Player+ 요건에 부합하는 비디오를 빠르게 생성하려면 [Player+로 재생하기 - 액세스 가이드](https://intl.cloud.tencent.com/document/product/266/38098)를 참조하십시오.

ABS 트랜스 코딩된 비디오는 Player+로 미리보기가 가능하며, VOD는 **Web 플레이어**와 **단말 플레이어**의 미리보기를 지원합니다.

### 매개변수 정보
**재생 설정**: Player+ 설정 내의 모든 플레이어 설정이 포함되어 있으며, 그중 default와 basicDrmPreset로 기본 설정되어 있습니다. **default 재생 설정 외의 모든 재생 설정은 기본 배포 도메인의 [Key 링크 도용 방지 활성화](https://console.cloud.tencent.com/vod/distribute-play/domain)를 실행해야 사용할 수 있습니다**.
![](https://main.qcloudimg.com/raw/460c56006dcf8cb1ee5e060cc5195ee8.png)


### 설정 항목
**설정 항목**: 선택한 플레이어의 설정 정보를 보여줍니다.
- 해당 fileID에 재생에 사용되는 ABS 트랜스 코딩 이름과 썸네일 미리보기의 스프라이트 이미지가 있는 경우, Web 플레이어와 단말 플레이어로 미리보기가 가능합니다.
- 해당 fileID가 재생에 사용되는 ABS 트랜스 코딩 이름 또는 썸네일 미리보기의 스프라이트 이미지를 내보내지 않는 경우, 다음과 같은 알림이 나타납니다.
![](https://main.qcloudimg.com/raw/ab0d7dbfdcd921284ef1853ba772ce62.png)

### 재생 제어
Key 링크 도용 방지의 도메인 재생 설정 활성화: 매개변수를 설정하여 **재생 링크 만료 시간**, **미리보기 시간** 및 **최대 재생 가능 IP 수**를 제어할 수 있습니다.
- 재생 링크 만료 시간: 비디오 링크가 해당 시점에서 만료되도록 설정합니다.
- 미리보기 시간: 비디오 미리보기 시간을 지정합니다. 시간은 최소 30초를 초과하여야 하며, 입력하지 않을 경우 무제한 시청이 가능합니다.
- 최대 재생 가능 IP 수: 재생이 허용되는 서로 다른 IP의 최대 수입니다.

## Web 플레이어 코드 생성
1. VOD 콘솔에 로그인하여 [미디어 자원 관리]>[[비디오 관리]](https://console.cloud.tencent.com/vod/media)를 선택하면 ‘업로드 완료’ 페이지로 이동합니다.
2. 타깃 비디오 파일이 있는 행에서 [관리]를 클릭하면 ‘기본 정보’ 페이지로 이동합니다.
3. [Web 플레이어 코드 생성]을 클릭하여 ‘Web 플레이어 코드 생성’ 페이지로 들어갑니다. 해당 페이지에서 플레이어 정보 및 Web 플레이어 코드를 관리할 수 있습니다.
	- ‘매개변수 설정’ 섹터에서 [변경]을 클릭하면 ‘플레이어 변경’ 대화창이 팝업됩니다. 플레이어 드롭다운 리스트에서 설정 완료한 플레이어(플레이어 관련 설정은 [Web 플레이어 관리](https://intl.cloud.tencent.com/document/product/266/14056) 참조)를 선택하고 [확인]을 클릭합니다.
	- ‘Web 플레이어 코드’ 섹터의 ‘비디오 크기’ 드롭다운 목록에서 적합한 비디오 크기를 선택하고 필요에 따라 자동 재생을 활성화 또는 비활성화합니다. 코드 유형은 ‘HTML’ 또는 ‘IFRAME’으로 설정합니다.
4. [코드 복사]를 클릭하면 생성된 Web 플레이어 코드가 복사됩니다.
