This document describes how to customize the fields to display in the video list and export videos in the VOD console.

## Customizing List Fields


1. Log in to the VOD console and select **Media Assets** > **[Video Management](https://console.cloud.tencent.com/vod/media)**. The **Uploaded** page is displayed.
2. Click ![](https://main.qcloudimg.com/raw/f4d3608e1d8319051883e90226a86f50.png) in the upper right of the list, select desired fields, and click **Confirm**. You can select up to 14 fields. The **Video Name/ID** and **Operation** fields are selected by default.
![](https://qcloudimg.tencent-cloud.cn/raw/473896085055dc9a382e363cec4db9dc.png)



## Exporting Video Files

1. Log in to the VOD console and select **Media Assets** > **[Video Management](https://console.cloud.tencent.com/vod/media)**. The **Uploaded** page is displayed.
2. Click ![](https://main.qcloudimg.com/raw/f4d3608e1d8319051883e90226a86f50.png) in the upper right of the list, select **Export Format** and **Export Data**, and click **Confirm**. If **Export Format** and **Export Data** are not selected, all media file info will be exported by default.
![](https://qcloudimg.tencent-cloud.cn/raw/c390702b9e17fb783cae7e8047873e22.png)
3. You can export files in CSV and JSON Lines formats. You can export the information of specified files or all files.
 1. CSV format files are files of Character Separated Values. Such files store tabular data in plain text, as shown below:
