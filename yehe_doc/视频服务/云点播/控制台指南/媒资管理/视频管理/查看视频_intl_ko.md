## 작업 시나리오
VOD 콘솔을 통해 비디오를 조회할 수 있습니다. 본 문서에서는 비디오 조회 방법에 대해 소개합니다.
## 작업 단계
1. VOD 콘솔에 로그인하여 [미디어 자산 관리]>[[비디오 관리]](https://console.cloud.tencent.com/vod/media)를 선택하면 ‘업로드 완료’ 페이지로 이동합니다.
2. 비디오 리스트에서 타깃 비디오 파일이 있는 행의 기록을 선택하면 비디오 ID 하단의 빈 공간에 ‘빠른 조회’ 버튼이 나타나고, [빠른 조회]를 클릭하면 오른쪽에 해당 비디오 파일의 비디오 정보 창이 나타납니다.
	- 이 창에는 비디오의 ID, 이름, 카테고리, 태그, 크기, 길이, 만료 시간, 파일 설명(일부 속성 편집 가능)이 표시됩니다.
	- 해당 비디오의 URL 주소를 복사하여 비디오 처리 및 비디오 재생 등에 사용할 수 있습니다.
