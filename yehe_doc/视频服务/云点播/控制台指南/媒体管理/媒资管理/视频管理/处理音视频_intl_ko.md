## 작업 시나리오
이 문서는 트랜스 코딩, 워터마크, 비디오 심사 등 VOD 콘솔에서 오디오/비디오를 처리하는 방법을 설명합니다.

## 작업 단계
1. [VOD 콘솔](https://console.cloud.tencent.com/vod)에 로그인하여 왼쪽 사이드바에서 **애플리케이션 관리**를 선택합니다.
2. 오디오/비디오 파일을 처리할 애플리케이션 이름을 클릭하여 애플리케이션 관리 페이지로 들어갑니다.
3. 기본적으로 **미디어 자산 관리**>**오디오/비디오 관리**, ‘업로드됨’ 페이지로 이동합니다.
4. 하나 이상의 타깃 오디오/비디오 파일 선택 후 목록 상단의 **비디오 처리**를 클릭하고, 아래 표에 따라 ‘비디오 처리’ 팝업 창에서 비디오 처리 매개변수를 설정한 후 **확인**을 클릭하여 비디오 처리 작업 실행을 시작합니다.
