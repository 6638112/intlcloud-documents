## 작업 시나리오
본문은 VOD 콘솔에서 오디오/비디오를 삭제하는 방법을 설명합니다.
> !삭제된 오디오/비디오는 **복구 불가능** 방식으로 Tencent Cloud에서 완전히 제거되고 CDN 노드에서 액세스할 수 없으므로 주의하시기 바랍니다.

## 작업 단계
1. [VOD 콘솔](https://console.cloud.tencent.com/vod)에 로그인하여 왼쪽 사이드바에서 **애플리케이션 관리**를 선택합니다.
2. 오디오/비디오 파일을 처리할 애플리케이션 이름을 클릭하여 애플리케이션 관리 페이지로 들어갑니다.
3. 기본적으로 **미디어 자산 관리**>**오디오/비디오 관리**, ‘업로드됨’ 페이지로 이동합니다.
4. 대상 오디오/비디오 파일이 있는 행에서 **삭제**를 클릭하면 ‘오디오/비디오 삭제’ 대화 상자가 나타납니다. **확인**을 클릭하여 오디오/비디오를 삭제하거나 대상 오디오/비디오 파일을 선택하고 목록 위의 **일괄 삭제**를 클릭한 다음 ‘오디오/비디오 삭제’ 대화 상자에서 **확인**을 클릭합니다. 일괄 삭제를 완료하기 위해 선택한 모든 오디오/비디오를 삭제할 수 있습니다.
