## 작업 시나리오
VOD 콘솔을 통해 오디오/비디오를 조회할 수 있습니다. 본 문서에서는 오디오/비디오 조회 방법에 대해 소개합니다.
## 작업 단계
1. [VOD 콘솔](https://console.cloud.tencent.com/vod)에 로그인하여 왼쪽 사이드바에서 **애플리케이션 관리**를 선택합니다.
2. 오디오/비디오 파일을 처리할 애플리케이션 이름을 클릭하여 애플리케이션 관리 페이지로 들어갑니다.
3. 기본적으로 **미디어 자산 관리**>**멀티미디어 관리**, ‘업로드됨’ 페이지로 이동합니다.
	- 이 창에는 오디오/비디오의 ID, 이름, 카테고리, 태그, 크기, 길이, 만료 시간, 파일 설명(일부 속성 편집 가능)이 표시됩니다.
	- 해당 오디오/비디오의 URL 주소를 복사하여 오디오/비디오 처리 및 오디오/비디오 재생 등에 사용할 수 있습니다.
