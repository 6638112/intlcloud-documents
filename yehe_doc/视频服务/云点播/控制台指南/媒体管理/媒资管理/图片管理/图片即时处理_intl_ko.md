## 작업 시나리오
[VOD 콘솔](https://console.cloud.tencent.com/vod/pics)을 통해 실시간 이미지 처리 작업을 수행할 수 있습니다. 본문은 실시간 이미지 처리 템플릿을 통해 실시간 실시간 이미지 처리를 수행하는 방법을 소개합니다. URL 매핑 방식을 통해 실시간 이미지 처리 효과를 빠르게 얻을 수 있습니다.
>?원본 이미지는 32MB를 초과하지 않아야 하며 너비와 높이는 30000픽셀을 초과해서는 안 되며, 해상도는 총 2억 5천만 픽셀을 초과할 수 없습니다. 출력 이미지의 크기는 9,999픽셀을 초과할 수 없습니다.

## 작업 단계
### 1단계: 이미지 업로드
1. [VOD 콘솔](https://console.cloud.tencent.com/vod)에 로그인하여 왼쪽 사이드바에서 **애플리케이션 관리**를 선택합니다.
2. 이미지 파일을 처리할 애플리케이션 이름을 클릭하여 애플리케이션 관리 페이지로 들어갑니다.
3. 기본적으로 **미디어 자산 관리**>**오디오/비디오 관리**, ‘업로드됨’ 페이지로 이동합니다.
4. **미디어 자산 관리**>**이미지 관리**를 선택하여 기본적으로 ‘업로드됨’ 페이지로 이동합니다. **이미지 업로드**를 클릭합니다.

### 2단계: 템플릿 생성
1. 왼쪽 사이드바에서 **미디어 처리**>**템플릿 설정**을 선택하고 **이미지 처리 템플릿**을 클릭합니다.
2. 페이지에 나열된 사전 설정 템플릿 외에도 자신만의 템플릿을 만들 수 있습니다. 아래 스크린샷은 240 x 240 썸네일을 생성하는 템플릿을 만드는 방법을 보여줍니다.
![](https://qcloudimg.tencent-cloud.cn/raw/79eb769ca9593f0924a9247e2da07e80.png)
![](https://qcloudimg.tencent-cloud.cn/raw/828297905d8d08e7363a0bd683491df5.png)
3. 템플릿이 성공적으로 생성되면 템플릿 목록에 템플릿이 나타나고 템플릿 번호는 13321로 생성됩니다.
![](https://qcloudimg.tencent-cloud.cn/raw/35893ec11a0e762466bdce3083e1b865.png)

VOD의 API를 호출하여 영상 처리 템플릿을 생성할 수도 있습니다. 자세한 내용은 [VOD API](https://www.tencentcloud.com/document/product/266/52140)를 참고하십시오.

### 3단계: 실시간 이미지 처리

실시간 이미지 처리는 URL 매핑으로 수행되며 공식은 다음과 같습니다.
#### 처리된 이미지 URL = 원본 이미지 URL + 간격 식별자 + 이미지 템플릿 번호 + ‘.’ + 출력 이미지 형식
  - 이미지 원본 URL: 이미지가 [VOD](https://console.cloud.tencent.com/vod/pics)에 업로드된 후 생성되는 가속 URL입니다.
  - 간격 식별자: !
  - 출력 이미지 형식: JPG, JPEG, PNG.
