## 작업 시나리오
기본 배포 도메인 페이지에서 다음 작업을 수행할 수 있습니다.
1. 기본 배포 도메인을 보고 수정합니다.
콘솔에서 도메인을 추가하고 CNAME을 구성한 후 자신의 도메인을 통해 비디오에 액세스할 수 있습니다. 단, API나 콘솔에서 획득한 동영상 URL의 Host를 업데이트하려면 VOD 콘솔에서 기본 배포 도메인 정보를 수정해야 합니다.
2. 재생 키를 보고 수정합니다.
재생 키는 [Player 서명](https://intl.cloud.tencent.com/document/product/266/38099)을 생성하는 데 사용됩니다. 재생 장치는 서명의 유효 기간 내에서만 콘텐츠에 액세스할 수 있습니다.

## 작업 단계
1. [VOD 콘솔](https://console.cloud.tencent.com/vod)에 로그인하여 왼쪽 사이드바에서 **애플리케이션 관리**를 선택합니다.
2. 설정할 애플리케이션 이름을 클릭하여 애플리케이션 관리 페이지로 들어갑니다.
3. 기본적으로 **미디어 자산 관리**>**오디오/비디오 관리**, ‘업로드됨’ 페이지로 이동합니다.
4. 왼쪽 사이드바에서 **배포 및 재생**>**기본 배포 도메인**을 선택합니다.
5. 기본 배포 도메인 정보를 수정하려면 오른쪽의 **편집**을 클릭합니다.
 - 기본 배포 프로토콜: HTTP 또는 HTTPS.
 - 기본 배포 도메인: 기본적으로 시스템에서 할당한 도메인(`xxx.vod2.myqcloud.com`)을 사용합니다. 콘솔에서 [추가](https://intl.cloud.tencent.com/document/product/266/35572)하고 [CNAME 구성](https://intl.cloud.tencent.com/document/product/266/35572) 후 자신의 도메인을 사용할 수도 있습니다.
 - 재생 키: 가입 시 시스템에서 임의 재생 키를 할당합니다. 이 키를 수정할 수 있습니다. 프로덕션 환경에서 이미 키를 사용하고 있는 경우 이를 변경하면 키를 기반으로 생성된 서명이 무효화됩니다.
