1. [VOD 콘솔](https://console.cloud.tencent.com/vod)에 로그인하여 왼쪽 사이드바에서 **애플리케이션 관리**를 선택합니다.
2. 대상 애플리케이션을 선택하여 애플리케이션 관리 페이지로 이동합니다.
3. 왼쪽 사이드바에서 **비디오 처리**>**템플릿 설정**을 선택합니다. 이미지 처리 관련 템플릿은 **이미지 처리 템플릿**을 포함합니다.


## 이미지 처리 템플릿

필요에 따라 **이미지 처리 템플릿**을 선택하고 **이미지 처리 템플릿 생성**을 클릭하여 사용자 정의 템플릿을 생성합니다.
+ 템플릿 이름: 중국어, 영어, 숫자, 스페이스, 언더바(_), 하이픈(-), 마침표(.)만 지원되며, 최대 64자까지 입력할 수 있습니다.
+ 처리 방법: 크기 조정 또는 자르기.
+ 크기 조정
	+ 너비: 출력 너비(픽셀)를 지정합니다. 값 범위: 1-9999. 너비를 비례적으로 조정하려면 **선택하지 마십시오**.
	+ 높이: 출력 높이(픽셀)를 지정합니다. 값 범위: 1-9999. 높이를 비례적으로 조정하려면 **선택하지 마십시오**.
+ 자르기:
	+ 원으로 자르기
		+ 반지름: 출력 원의 반지름(0보다 큰 값)을 지정합니다.
	+ 직사각형으로 자르기
		+ 타깃 크기: 출력 사각형의 너비(px) x 높이(px)를 지정합니다. 값 범위: 1-9999.

사전 설정 이미지 처리 템플릿에 대한 자세한 내용은 [사전 설정 템플릿 - 이미지 처리](https://intl.cloud.tencent.com/document/product/266/47373)를 참고하십시오.
