### VOD 트래픽
- 과금 방식: 트래픽 과금
- 유효 일수: 월 결산에 적용된 유효 일수
- 과금 트래픽: 과금에 적용된 트래픽 사용량
트래픽 사용량은 매일 5분마다 기록되며 총 청구 대상 트래픽은 기록된 모든 시점의 트래픽 합계입니다.

<span id="p3"></span>
### VOD 종량제 스토리지
- 날짜: 월 결산에 적용된 유효 일수
- 사용량: 당일 최대 VOD 스토리지 사용량
>!월간 스토리지 요금 = 일별 최대 스토리지 용량 합계x 단가(VOD 콘솔에 당일 최대 스토리지 용량 표시됨)

### VOD 트랜스코딩
- 날짜: 역일
- Appid: 사용자 계정 ID
- 작업 ID: 실행된 작업의 ID
- 트랜스코딩 유형: 기본 트랜스코딩, TESHD 트랜스코딩, 오디오 트랜스코딩 및 비디오 편집
- 코덱: H.264, H.265
- 해상도: 트랜스코딩 템플릿에 지정된 해상도
- 트랜스코딩 시간: 당일 트랜스코딩 시간

### VOD 비디오 부적절한 콘텐츠 식별
- 날짜: 역일
- 사용량: 당일 부적절한 콘텐츠 식별을 진행한 비디오 길이
