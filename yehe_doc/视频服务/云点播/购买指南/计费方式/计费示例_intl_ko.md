## VOD 트래픽 사용량 추정
### 예시
동영상 웹사이트 플랫폼을 운영하는 사용자 A는 VOD 콘솔에서의 동영상 재생 가속 트래픽 사용량을 추정하려고 합니다. 관련 세부 사항은 다음과 같습니다.

- 시청자: 100명
- 길이: 1시간
- 비디오 비트레이트: 500Kbps

### 비용 추정
트래픽 = 비트레이트 x 지속 시간 x 시청자 수
>?사용자는 [템플릿 설정](https://intl.cloud.tencent.com/document/product/266/14059)에서 사전 설정된 템플릿의 비트레이트를 쿼리할 수 있으며 템플릿의 비트레이트를 정의할 수 있습니다.

상기 공식에 따르면 본 예시의 트래픽 소비는 ≈ 500/8KBps x 3600초 x 100명 = 22,500,000KB = 22.5GB입니다.


## VOD 트랜스코딩 종량제 비용 추정
### 예시
사용자 A는 사전 설정 템플릿을 사용하여 비디오를 트랜스코딩합니다. 관련 세부 사항은 다음과 같습니다.

- 입력 비디오 해상도: 1280 x 640
- 트랜스코딩 템플릿: 100030
- 비디오 길이: 100분

### 비용 추정
사전 설정 트랜스코딩 템플릿 100030의 매개변수: H.264 코덱, 비디오 높이 720픽셀 × 비례 너비, 비디오 출력 해상도 1440 × 720. [종량제(일 결산 후불)](https://intl.cloud.tencent.com/document/product/266/14666#.E6.99.AE.E9.80.9A.E8.BD.AC.E7.A0.81)의 과금 기준에 따르면 출력된 비디오의 높이가 720px이므로, 단가는 0.0061 USD/min입니다.
따라서, 본 예시의 비용은 = 0.0061 USD/분 x 100분 = 0.61USD입니다.

## VOD 스토리지 종량제 비용 추정
### 예시
사용자 A는 콘솔에서 중국 본토 이외 지역에 비디오를 저장한 다음 시청자에게 배포하려고 합니다. 사용자는 일별 스토리지 비용을 계산하고자 합니다.

- 비디오 크기: 50GB
- 스토리지 기간: 1년

### 비용 추정
- 종량제(일 결산 후불):
총 스토리지 요금 = 0.0009 USD/GB/일 x 365일 x 50GB = 16.425USD
