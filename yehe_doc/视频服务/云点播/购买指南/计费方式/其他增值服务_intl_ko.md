## AI 기반 영상 분석 기능

AI 기반 영상 분석 기능은 월 결산 후불 방식으로 과금됩니다. 과금 단위는 USD/분이며 이 서비스는 서비스 유형 및 해당 기간에 따라 과금됩니다.

- 결제 방법 : 월 결산 후불.
- 원본 비디오 길이가 1분 미만인 경우 1분으로 계산됩니다.

#### 과금 가격

AI 기반 비디오 분석 기능에는 다음과 같은 과금 항목이 있습니다.

- 스마트 하이라이트 비디오 분할: 스마트 비디오 분할은 비디오를 샷 또는 장면으로 분할합니다. 이 기능은 사용자 요구 사항에 따라 추가로 최적화할 수 있습니다. **하이라이트 생성 및 비디오 분할은 서로 다른 API를 사용하며 별도로 과금됩니다**.
- 스마트 비디오 태그 및 비디오 카테고리: 비디오에서 인물, 시나리오, 음성, 텍스트 및 기타 정보를 지능적으로 식별한 다음 비디오 태그, 카테고리, 요약을 자동으로 생성합니다. **비디오 태그 및 비디오 카테고리는 서로 다른 API를 사용하며 별도로 과금됩니다**.
- 비디오 썸네일 스마트 추천: 비디오의 하나 이상의 스크린샷을 썸네일로 추천합니다.

과금 관련 상세 내용은 다음과 같습니다.

| 과금 항목       | 과금 방식                 | 가격(USD/분) |
| ------------ | ------------------------ | ----------------- |
| 하이라이트 생성 | 하이라이트가 생성되는 소스 비디오의 길이 | 0.0572            |
| 비디오 세분화 | 분할된 소스 비디오의 길이 | 0.0572            |
| 비디오 라벨링 | 레이블이 지정된 소스 비디오의 길이 | 0.0572            |
| 비디오 분류 | 분류된 소스 비디오의 길이 | 0.0572            |
| 썸네일 생성 | 썸네일이 생성되는 소스 비디오의 길이 | 0.0572            |


#### 과금 예시

**사용자 A는 비디오 B를 처리하려고 합니다. 스마트 비디오 태그 및 스마트 비디오 카테고리 API를 동시에 호출하였으며, 비디오 B의 기존 재생 시간은 100분입니다.
서비스 요금 = 100 x 0.0572 + 100 x 0.0572 = 11.4USD**

## AI-기반 오디오/비디오 심사

AI-기반 오디오/비디오 심사의 과금 방식은 후불입니다. 과금 단위는 USD/분이며 서비스 유형 및 해당 비디오 길이에 따라 과금됩니다.
과금 규칙은 다음과 같습니다.

- 결제 방법: 후불.
- 원본 비디오 길이가 1분 미만인 경우 1분으로 계산됩니다.

#### 과금 가격

AI 기반 비디오 중재는 비디오의 이미지, 오디오 및 텍스트를 검토하여 부적절한 콘텐츠를 감지하고 필요에 따라 심사 결과를 생성합니다.

과금 관련 상세 내용은 다음과 같습니다.

| 과금 항목   | 과금 방식                 | 가격(USD/분) |
| -------- | ------------------------ | ----------------- |
| 비디오 조정 | 소스 비디오의 길이 조정됨 | 0.016             |

#### 과금 예시

 사용자가 이 기능을 사용하여 30분짜리 비디오를 심사한다고 가정합니다.
서비스 비용 = 30 x 0.016 = 0.48USD.



## DRM

VOD는 상용 DRM 솔루션을 사용하여 트랜스코딩 중에 비디오를 암호화합니다. DRM을 활성화하면 해당 비용이 트랜스코딩 요금에 포함됩니다. 또한 암호화된 비디오 재생 요청 횟수에 따라 요금이 청구됩니다.

#### 과금 가격

| 과금 항목                        | 과금 방식     | 단가          |
| :---------------------------- | :----------- | :------------ |
| 암호화된 동영상 재생 요청 횟수 | 후불 | 0.0012 USD/회 |



#### 과금 설명

- 과금 항목: DRM으로 암호화된 동영상을 재생하기 위해 수신된 요청 수에 따라 요금이 청구됩니다.
- 과금 규칙: DRM 암호화 서비스 사용 시, 동영상 암호화에 대한 트랜스코딩 비용 및 DRM 암호화 동영상 재생에 대한 License 요청 횟수도 청구됩니다.
- DRM 수수료 = DRM 트랜스코딩 요금 + DRM 동영상 재생 요금(재생 요청 건수로 계산)



#### 과금 예시

1월 1일에 상용 DRM 솔루션을 사용하여 비디오 A(길이: 10분, 해상도: 2560 x 1440)를 암호화하고 비디오 B(길이: 10분, 해상도: 1280 x 640)를 생성하였으며, 암호화된 동영상 재생 요청이 50건이라고 가정합니다.
1월 2일에는 다음 DRM 수수료가 부과됩니다.
0.0061(USD/분) x 10(분) + 50(요청) * 0.0012(USD/요청) = 0.121 USD

>?  DRM 트랜스코딩은 일반 트랜스코딩과 동일한 방식으로 청구됩니다. 비디오를 암호화할 때마다 DRM 트랜스코딩 요금이 부과됩니다.
