## 라이브 스트림 클리핑

라이브 스트림이 종료되기 전에 클립을 실시간 편집하는 데 발생하는 요금입니다.
가격 세부 정보는 다음과 같습니다.

| 과금 항목       | 가격(USD/분) | 과금 대상                   |
| :----------- | :---------------- | -------------------------- |
| 실시간 클리핑 | 0.00098            | 출력 파일의 지속 시간 |

#### 과금 설명

- **과금 항목**: 실시간 클리핑.
- **과금 규정**: 라이브 스트리밍이 종료되기 전에 클립을 생성하는 경우 요금이 발생합니다.
- **요금 계산**: 라이브 스트림 클리핑 요금 = 파일 길이(분) x 단가(USD)

#### 과금 예시

1월 1일에 **실시간 클리핑** 기능을 사용하여 100분 길이의 클립을 생성했다고 가정할 때, 1월 2일에 지불해야 하는 라이브 스트림 클리핑 요금을 다음과 같습니다.

1월 1일 라이브 스트림 클리핑 요금 = 0.00098(USD/분) x 100(분) = 0.098(USD)

라이브 스트림 클리핑에 대한 자세한 내용은 [라이브 스트림 클리핑](https://www.tencentcloud.com/document/product/266/49280)을 참고하십시오.



## 클라이언트 업로드 가속

클라이언트 업로드 가속 기능(글로벌 네트워크 가속 또는 QUIC 전송) 사용에 대해 발생하는 요금은 가속된 트래픽량을 기준으로 합니다.
가격 세부 정보는 다음과 같습니다.

| 유형     | 단가         |
| :----------- | :----------- |
| 글로벌 네트워크 전송 | 0.072 USD/GB |
| QUIC 전송    | 0.086 USD/GB |

#### 과금 설명

- **과금 항목**: 글로벌 네트워크 가속 트래픽 및 QUIC 전송 트래픽
- **과금 규칙**: 클라이언트 업로드 가속 요금은 클라이언트에서 파일을 업로드할 때 소비되는 트래픽량에 따라 부과됩니다.
- **요금 계산**: 클라이언트 업로드 가속 요금 = 글로벌 네트워크 가속 트래픽(GB) x 단가(USD/GB) + QUIC 전송 트래픽(GB) x 단가(USD/GB).

#### 과금 예시

글로벌 네트워크 가속 기능을 사용하여 550 GB의 데이터를 업로드하고 QUIC 전송 기능을 사용하여 100 GB의 데이터를 업로드했다고 가정합니다. 업로드 가속 요금은= 550(GB) x 0.072(USD/GB) + 100(GB) x 0.086(USD/GB) = 48.2(USD)입니다.

클라이언트 업로드 가속에 대한 자세한 내용은 [클라이언트 업로드 가속](https://www.tencentcloud.com/document/product/266/49083)을 참고하십시오.
