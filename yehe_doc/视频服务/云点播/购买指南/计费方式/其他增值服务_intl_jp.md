ショート動画SDKライセンスの説明は次のとおりです：

- ショート動画のベース版ライセンスを無料で申請し、28日間トライアルが可能です。

##オーディオ・ビデオAI-ビデオ分析

オーディオ・ビデオAI-ビデオ分析は、月末締めの後払いです。課金単位はドル/分です。ユーザーが利用したサービスとその利用時間に基づいて課金されます。利用説明は次のとおりです：

- 課金方法：月末締めの後払い。
- オリジナルのビデオ長さが1分未満の場合、1分として計算されます。

#### 課金価格

オーディオ・ビデオAI–ビデオ分析には、次の課金項目が含まれています：

- スマートビデオコレクション・スプリット：内容に従ってスマートに分割します。シーン分割機能とシングルショット分割機能の2種類を提供し、お客様のニーズに合わせて最適化できます。**そのうち、コレクションとスプリットは2つの異なるインターフェースであり、インターフェースの呼出しに基づいて個別に課金されます**。
- スマートビデオタグ分類：ビデオコンテンツを分析し、ビデオ中の人物、シーン、音声、テキストなどの情報を識別して、ビデオのタグ、分類、サマリなどを自動的に生成します。**そのうち、タグ分類は2つの異なるインターフェースであり、インターフェースの呼出しに基づいて個別に課金されます**。
- スマートビデオカバー：ビデオの中から1枚または複数枚のスクリーンショットを推奨カバーとして選択します。

課金の詳細は次のとおりです：

| 課金項目       | 課金方法                 | 価格（ドル/分） |
| ------------ | ------------------------ | ----------------- |
| スマートビデオコレクション | 分析対象となるビデオの元の長さによる課金 | 0.0572            |
| スマートビデオスプリット | 分析対象となるビデオの元の長さによる課金 | 0.0572            |
| スマートビデオタグ | 分析対象となるビデオの元の長さによる課金 | 0.0572            |
| スマートビデオ分類 | 分析対象となるビデオの元の長さによる課金 | 0.0572            |
| スマートビデオカバー | 分析対象となるビデオの元の長さによる課金 | 0.0572            |


#### 課金例

**ビデオBに対してユーザーAがビデオ分析を行い、呼び出したインターフェースにスマートタグインターフェースとスマート分類インターフェース両方があり、ビデオBの元の長さが100分の場合、
このサービスの利用料金=100×0.0572+100 x 0.0572 =11.4ドル**

## オーディオ・ビデオAI-ビデオ審査

オーディオ・ビデオAI-ビデオ審査は後払いです。課金単位はドル/分です。ユーザーが利用したサービスとビデオの審査時間に基づいて課金されます。
利用説明は次のとおりです：

- 課金方法：後払い。
- オリジナルのビデオ長さが1分未満の場合、1分として計算されます。

#### 課金価格

オーディオ・ビデオAI-ビデオ審査は、ビデオコンテンツ中の画像、音声、テキスト、また、ビデオ中の不適切な内容を審査し、お客様の要求に従って審査結果を出力します。

課金の詳細は次のとおりです：

| 課金項目       | 課金方法                 | 価格（ドル/分） |
| -------- | ------------------------ | ----------------- |
| ビデオ審査 | 審査対象となるビデオの元の長さによる課金 | 0.016            |

#### 課金例

 ビデオAに対してユーザーがビデオコンテンツを審査し、ビデオAの長さが30分の場合、
このサービスの利用料金=30 x 0.016 =0.48ドル。



## DRM暗号化サービス

オン・デマンドのDRM暗号化サービスは、エンタープライズ級のDRM暗号化ソリューションを使用し、ビデオを暗号化します。DRM暗号化サービスはオン・デマンドとトランスコーディングをベースにして実装されています。DRM暗号化サービスが有効になっている場合、DRM暗号化トランスコーディングの料金は、個別に課金されず、トランスコーディング料金に加算されます。Video on Demand DRM暗号化サービスは、端末のDRM暗号化ビデオライセンスのリクエスト回数に基づいて課金されます。

#### 課金価格

| 課金項目                        | 課金方法     | 価格          |
| :---------------------------- | :----------- | :------------ |
| DRM暗号化ビデオライセンスのリクエスト回数 | 後払いのみ利用可 | 0.0012ドル/回 |



#### 課金説明

- 課金項目：DRM暗号化ビデオを再生する時に発生したライセンスリクエスト回数
- 課金ルール：ユーザーがDRM暗号化サービスを利用する時に発生した暗号化トランスコーティング料金と、DRM暗号化ビデオを再生する時に発生したライセンスリクエスト回数。
- 課金計算式：DRM暗号化サービス利用料金 = DRM暗号化トランスコーディング料金 + 暗号化ビデオの再生によって発生したライセンスリクエスト回数



#### 課金例

1月1日にVideo on Demand DRMサービスを利用し、オリジナルビデオA（長さが10分、解像度が2560×1440）に対してエンタープライズ級DRM暗号化を行い、トランスコーディング結果として暗号化ビデオB（長さが10分、解像度が1280 x 640）を出力し、このDRM暗号化ビデオがライセンスを50回リクエストした場合。
1月2日に支払うエンタープライズ級DRMの利用料金は以下となります：
1月1日のビデオ編集料金 = 0.0061（ドル/分）× 10（分）+ 50（回）* 0.0012 (ドル/回) = 0.121ドル

>? DRM暗号化トランスコーディングの利用料金はオン・デマンド標準トランスコーディングの利用料金と同じです。暗号化トランスコーディングサービスは、処理を行うたびに1回の利用料金を請求します。そうでない場合、料金を請求しません。
