## 5일 무조건부 환불

보다 편리한 VOD 사용을 위해 구입한 패키지를 사용하지 않은 경우 구매 후 5일 이내 무조건부 환불을 지원합니다.

## 환불 정책[](id:p1)

- VOD 신규 사용자 체험 패키지는 환불이 지원되지 않습니다.
- 단일 계정의 경우 미사용 VOD 패키지는 신규 구매일로부터 5일 이내(5일 포함) 무조건 환불이 가능하며, 사용한 경우에는 환불되지 않습니다.
- UGSV SDK License가 활성화되면 환불할 수 없습니다(패키지와 License가 바인딩된 후에는 패키지를 환불할 수 없으며 사용한 것으로 간주됨).
- 이상 또는 악의적인 환불이 의심되는 경우 Tencent Cloud는 귀하의 환불 신청을 거부할 권리가 있습니다.
- 5일 무조건부 환불 요건을 충족하는 주문의 경우 환불 금액은 현금 계좌 금액, 계좌 이체 금액, 증정 계좌 금액을 포함하여 구매 시 사용한 전체 소비 금액입니다.

>!
>- 단일 계정은 무조건부 환불이 1회만 지원됩니다.
>- 할인 및 크레딧은 반환되지 않습니다.
>- 환불 금액은 모두 Tencent Cloud 계정으로 환불됩니다.

## 환불 단계

1. [VOD 콘솔](https://console.cloud.tencent.com/vod/overview)에 로그인하고, **서비스 개요** 페이지의 **리소스 패키지** 모듈 아래의 **사용 가능한 리소스 패키지**의 해당 수량을 클릭합니다.
2. **리소스 패키지** 페이지에서 환불할 리소스 패키지을 선택하고 선택한 리소스 패키지 오른쪽의 **반환**을 클릭합니다.

>!  리소스 패키지 반환 전, 해당 리소스 패키지가 [환불 정책](#p1)에 부합하는지 확인하시고 부합되는 경우 **반환**을 클릭합니다. 정책에 부합되지 않으면 반환되지 않습니다.
