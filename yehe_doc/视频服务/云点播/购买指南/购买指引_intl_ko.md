
## Tencent Cloud 가입
VOD를 사용하려면 우선 [Tencent Cloud 가입](https://intl.cloud.tencent.com/document/product/378/17985)을 완료해야 합니다.


## VOD 서비스 신청
1. [VOD 소개 페이지](https://intl.cloud.tencent.com/product/vod)로 이동하여 [시작하기]를 클릭하면 ‘**개발자 자격 인증**’ 페이지로 이동합니다.
2. [확인]을 클릭하여 자격 확인 페이지로 이동하여 필수 정보를 입력합니다. [실명 인증](https://intl.cloud.tencent.com/document/product/378/3629) 통과 후, Tencent Cloud 콘솔에 액세스할 수 있습니다. 미인증 사용자는 중국 본토의 VOD 인스턴스를 구매할 수 없습니다.

## VOD 서비스 구입
VOD는 현재 **일 결산(후불) 과금 방식**을 지원합니다.

- **일 결산 과금 방식**은 서비스 사용 후 요금을 지불하는 것을 의미합니다. 이 방식은 사전에 Tencent Cloud 계정을 [충전](https://console.cloud.tencent.com/expense/recharge)해야 합니다. 시스템은 매일 전날의 실제 사용량을 계산하여 청구서를 발송하고, 사용량에 따라 계정 잔액에서 요금을 차감합니다. 자세한 내용은 [일 결산(후불) 과금 방식](https://intl.cloud.tencent.com/document/product/266/14666)을 참고하십시오.

## VOD 서비스 이용
VOD 서비스를 구매한 후 Tencent Cloud 콘솔에서 [제품]>[비디오 서비스]>[[VOD]](https://console.cloud.tencent.com/vod)를 선택하여 사용하십시오.
