ここでは、Tencent Cloud VODコンソールにおける、以下のFairPlay証明書情報の提出についてご説明します。

- FPS証明書ファイル(.cer)
- 秘密鍵ファイル(.pem)
- 秘密鍵パスワード
- ASK(Application Secret Key)

以上のFairPlay証明書情報をまだ申請していない場合は、[FairPlay証明書情報の申請方法](https://intl.cloud.tencent.com/document/product/266/46643)をご参照ください。

## 操作手順
1. Tencent Cloud VODコンソールにログインします。

2. 左側ナビゲーションバーの`Video Cloudの設定`をクリックして展開し、`DRM暗号化の設定`をクリックし、右側の`編集`をクリックします。
   ![image-20220425210931543](https://qcloudimg.tencent-cloud.cn/raw/041b560536deebd1bd5bc986d95ed289.png)

3. 証明書ファイル(`fairplay.cer`)、秘密鍵ファイル(`privatekey.pem`)、秘密鍵パスワード、ASKなどを含むFPS証明書情報を設定し、`保存`をクリックします。

   ![image-20220425211140740](https://qcloudimg.tencent-cloud.cn/raw/effefe51d8ca82e46d292112eab9a3b4.png)

4. 保存すると、`FairPlay`の証明書情報を確認することができます。

   ![image-20220426191830269](https://qcloudimg.tencent-cloud.cn/raw/8d64c3dbb65ac6f54a81f08314ca1473.png)

## まとめ

この時点で、Tencent Cloud VODコンソールでの`FairPlay`証明書情報の設定が完了しました。
