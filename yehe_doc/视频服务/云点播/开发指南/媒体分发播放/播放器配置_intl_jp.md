Player+を使用してオンデマンドビデオを再生する場合、画面コンテンツやサムネイルを指定する必要があります。

ビデオ**画面コンテンツ**とは、以下の3つのうちのいずれかです。

* [アダプティブビットレートストリーミングへのトランスコード](https://intl.cloud.tencent.com/document/product/266/33942)の出力ビデオ
* [トランスコード](https://intl.cloud.tencent.com/document/product/266/33938)の出力ビデオ
* [アップロード](https://intl.cloud.tencent.com/document/product/266/9760)のオリジナルビデオ

**サムネイル**プレビューは、[スプライトイメージ](https://intl.cloud.tencent.com/document/product/266/33940)の出力画像です。

アダプティブビットレートストリーミングには複数のサブストリームが含まれるので、解像度の切り替え時に異なる解像度名に対応します。プレーヤーに表示される各サブストリームの解像度名を指定する場合は、サブストリームの解像度名の命名規則を指定する必要があります。

## プレーヤー設定項目

VODは、**プレーヤー設定**を提供します。設定において、アダプティブビットレートストリーミングに変換された出力ビデオ、トランスコードされた出力ビデオ、アップロードされたオリジナルビデオのうち、どれを再生するか指定できます。
アダプティブビットレートストリーミングに変換された出力ビデオを再生する場合は、**ABSテンプレートID**を指定する必要があります。変換された出力ビデオを再生する場合は、**トランスコードテンプレートID**を指定する必要があります。サムネイルプレビューを使用する場合は、**スプライトイメージテンプレートID**を指定する必要があります。
プレーヤーの設定により、個々のサブストリームの**解像度命名ルール**をカスタマイズすることもできます（例：LD、SDおよびHDなど）。

| 設定項目 | 設定方法 | 機能 |
| -- | -- | -- |
| 展示用の画面コンテンツ | フィールドのタイプで次のいずれかを選択します。<li>アップロードしたオリジナルビデオを再生する。</li><li>トランスコードされた出力を再生する：トランスコードテンプレートID</li><li>アダプティブビットレートストリーミング出力を再生する：ABSテンプレートID</li> | プレーヤーが再生する画面コンテンツを制御します |
| サムネイルプレビュー用のスプライトイメージ | スプライトイメージテンプレートID | プログレスバーに表示されるサムネイルを制御します |
| プレーヤー表示用のサブストリームの解像度名 | サブストリーム短辺の長さに対する命名 | 解像度切り替え時の各解像度名を制御します |

## カスタマイズ設定
プリセットされたプレーヤーの設定のほかにも、[コンソール](https://intl.cloud.tencent.com/document/product/266/38261)または[API](https://intl.cloud.tencent.com/document/product/266/37567)を介して設定をカスタマイズすることができます。
カスタマイズされたプレーヤーを使用して再生を設定します。具体例については、[アクセスガイド](https://intl.cloud.tencent.com/document/product/266/38098)をご参照ください。

## 使用設定
プリセット設定されたdefault以外の設定を使用してビデオを再生する場合は、[プレーヤー署名](https://intl.cloud.tencent.com/document/product/266/38099)を使用し、署名パラメータの`pcfg`パラメータで使用する設定名を指定する必要があります。
プレーヤー署名を使用してビデオを再生します。具体例については、[アクセスガイド](https://intl.cloud.tencent.com/document/product/266/38098)をご参照ください。
