규정을 준수하지 않는 콘텐츠는 비즈니스를 법적 위험에 노출시키고 브랜드를 손상시킬 수 있습니다. VOD는 콘텐츠의 규정 준수를 보장하는 [오디오/비디오 콘텐츠 조정](https://intl.cloud.tencent.com/document/product/266/33944) 기능을 제공합니다.

## 오디오/비디오 콘텐츠 조정
VOD의 [오디오/비디오 콘텐츠 조정](https://intl.cloud.tencent.com/document/product/266/33944) 기능은 비디오 콘텐츠를 분석하고 콘텐츠 통과 또는 차단 여부를 제안합니다. 개발자는 결과에 따라 오디오/비디오 콘텐츠를 처리할 수 있습니다.

다음 두 가지 방식으로 오디오/비디오 콘텐츠 조정 작업을 시작할 수 있습니다.
1. VOD [콘솔](https://intl.cloud.tencent.com/document/product/266/33897)을 통해 작업합니다.
2. 서버 측 API [오디오/비디오 콘텐츠 조정](https://intl.cloud.tencent.com/document/product/266/33944)을 통해 호출합니다.

### 오디오/비디오 콘텐츠 조정을 위한 서버 API 호출
예를 들어 비디오 App이 VOD 오디오/ 비디오 조정 서비스에 액세스하는 경우 비디오 조정을 시작하기 위해 서버측 API를 호출하는 프로세스는 다음과 같습니다.
![](https://main.qcloudimg.com/raw/99544692ba8d62874a0f718e805f2d57.png)

1. App 백엔드는 콘텐츠 공급자를 인증하고 인증이 통과된 후 [클라이언트 업로드 서명](https://intl.cloud.tencent.com/document/product/266/33922)을 배포합니다.
2. 콘텐츠 제공자는 공유할 콘텐츠를 VOD에 업로드합니다.
3. VOD는 업로드된 비디오의 FileId 및 재생 URL과 같은 [관련 정보](https://intl.cloud.tencent.com/document/product/266/33950)를 App 백엔드에 알립니다.
4. VOD는 업로드 시 지정된 `procedure` 매개변수에 따라 조정 작업(`ReviewAudioVideoTask`)을 실행합니다.
5. VOD는 [조정 완료](https://www.tencentcloud.com/document/product/266/50677) 알림을 통해 App 백엔드로 결과를 보냅니다.
 - 조정 결과가 ‘block’인 경우 콘텐츠가 규정을 준수하지 않을 가능성이 높습니다. 콘텐츠를 차단하는 것이 좋습니다.
 - 조정 결과가 ‘pass’인 경우 콘텐츠가 규정을 준수하지 않을 가능성이 낮습니다. 이 경우 콘텐츠를 통과하도록 허용하는 것이 좋습니다.
 - 조정 결과가 ‘review’인 경우 콘텐츠가 규정을 준수하지 않을 가능성이 높습니다. 이 경우 수동 재확인이 권장됩니다.
6. App 백엔드는 ‘통과 권장’인 비디오와 ‘재확인 권장’을 받고 수동 재확인을 통과한 비디오를 게시합니다.
7. 콘텐츠 소비자는 App 백엔드에서 게시된 비디오의 재생 URL을 요청합니다.
8. 콘텐츠 소비자는 URL을 통해 비디오를 재생합니다(VOD는 가속 서비스를 제공합니다).

4 - 6단계는 7단계에서 얻은 비디오가 규정을 준수하는지 확인합니다.

>! 상기 과정에서 비디오는 ‘게시되기 전에 조정’됩니다(검토 또는 검토를 통과한 동영상만 게시됨). 조정이 수행되기 전에 비디오가 먼저 게시되는 ‘사후 조정’ 모드를 사용할 수도 있습니다(비준수 콘텐츠가 감지되면 제거됨).

## 이미지 조정
VOD의 [이미지 조정](https://www.tencentcloud.com/document/product/266/47138) 기능은 이미지 콘텐츠를 분석하고 콘텐츠 통과 또는 차단 여부를 제안합니다. 오디오/비디오 콘텐츠 조정과 달리 이미지 조정은 동기화 프로세스이며 결과는 작업이 시작된 직후에 얻을 수 있습니다. 사용자측은 제안에 따라 이미지 게시 여부를 결정할 수 있습니다.
- 조정 결과가 ‘block’인 경우 콘텐츠가 규정을 준수하지 않을 가능성이 높습니다. 이미지를 차단하는 것이 좋습니다.
- 조정 결과가 ‘pass’인 경우 콘텐츠가 규정을 준수하지 않을 가능성이 낮습니다. 이 경우 콘텐츠를 통과하도록 허용하는 것이 좋습니다.
- 조정 결과가 ‘review’인 경우 콘텐츠가 규정을 준수하지 않을 가능성이 높습니다. 이 경우 수동 재확인이 권장됩니다.

이미지 조정을 시작하는 방법:
1. 서버 API [이미지 조정](https://www.tencentcloud.com/document/product/266/47138)을 통해 호출합니다.
