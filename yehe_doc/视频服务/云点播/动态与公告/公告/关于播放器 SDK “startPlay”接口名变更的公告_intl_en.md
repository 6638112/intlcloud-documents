The `startPlay` API will be renamed in v10.7 of the Player SDK for mobile devices (to be released on September 19, 2022).

The `startPlay` API is used to start playback. Its name will be changed as follows:

- The `startPlay` API of `TXVodPlayer` will be renamed `startVodPlay`. For details, see [VOD API Documentation (iOS)](https://www.tencentcloud.com/document/product/266/47844), [VOD API Documentation (Android)](https://www.tencentcloud.com/document/product/266/47851), and [VOD API Documentation (Flutter)](https://www.tencentcloud.com/document/product/266/42099).
- The `startPlay` API of `TXLivePlayer` will be renamed `startLivePlay`. For details, see [Live Streaming API Documentation (iOS)](https://www.tencentcloud.com/document/product/266/47841#.E6.92.AD.E6.94.BE.E5.9F.BA.E7.A1.80.E6.8E.A5.E5.8F.A3) and [Live Streaming API Documentation (Android)](https://www.tencentcloud.com/document/product/266/47850#.E6.92.AD.E6.94.BE.E5.9F.BA.E7.A1.80.E6.8E.A5.E5.8F.A3).

>?
>- Versions earlier than v10.7 are not affected by the above changes.
>- Please pay attention to the changes if your SDK version is 10.7 or later.
