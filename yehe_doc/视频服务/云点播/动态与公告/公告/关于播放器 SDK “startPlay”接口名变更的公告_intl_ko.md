'startPlay' API는 모바일 장치용 플레이어 SDK v10.7에서 이름이 변경됩니다(2022년 9월 19일 출시 예정).

'startPlay' API는 재생을 시작하는 데 사용됩니다. 이름이 다음과 같이 변경됩니다.

- `TXVodPlayer`의 `startPlay` API의 이름이 `startVodPlay`로 변경됩니다. 자세한 내용은 [VOD API 문서(iOS)](https://www.tencentcloud.com/document/product/266/47844), [VOD API 문서(Android)](https://www.tencentcloud.com/document/product/266/47851), [VOD API 문서(Flutter)](https://www.tencentcloud.com/document/product/266/42099)를 참고하십시오.
- `TXLivePlayer`의 `startPlay` API의 이름이 `startLivePlay`*로 변경됩니다. *자세한 내용은 [실시간 스트리밍 API 문서(iOS)](https://www.tencentcloud.com/document/product/266/47841#.E6.92.AD.E6.94.BE.E5.9F.BA.E7.A1.80.E6.8E.A5.E5.8F.A3), [실시간 스트리밍 API 문서(Android)](https://www.tencentcloud.com/document/product/266/47850#.E6.92.AD.E6.94.BE.E5.9F.BA.E7.A1.80.E6.8E.A5.E5.8F.A3)를 참고하십시오.

>?
>- v10.7 이전 버전은 상기 변경 사항의 영향을 받지 않습니다.
>- SDK 버전이 10.7 이상인 경우 변경 사항에 주의하시기 바랍니다.
