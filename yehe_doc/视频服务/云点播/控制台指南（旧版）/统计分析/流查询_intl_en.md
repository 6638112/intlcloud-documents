Log in to the LVB Console, select **Statistical Analysis** > **[Query Stream](https://console.cloud.tencent.com/live/analysis/stream)**, and enter a stream ID to query push and playback information of the stream as shown below:
![](https://main.qcloudimg.com/raw/954239f7911eec091684bc9b718deaed.png)

### Playback Data Query 
The playback data page displays the data of bandwidth, traffic, and concurrent connections.
![](https://main.qcloudimg.com/raw/9cab8ba274c16e819e4131067c5de5d8.png)

### Push Data Query
You can query the push frame rate and bitrate of a stream in the last 5 days, and the query period cannot be over 3 hours.
After entering the stream ID, select **View push data** to query the push information of the stream as shown below:
![](https://main.qcloudimg.com/raw/fb21f8ab85513059e69f0186dc19c2ae.png)

You can also query stream interruption data as shown below:
![](https://main.qcloudimg.com/raw/b2c3e79ee99b2ed41a93adac8c915132.png)
