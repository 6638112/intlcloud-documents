Tencent Cloud View Cube Player Demoは、完全な製品レベルのインタラクティブインターフェースと業務ソースコードを提供します。開発者はDemoをもとにプレーヤーの各種機能を体験できます。これにより、業務上のニーズへの素早い対応とその実現、開発時間とコストの削減を可能にします。このドキュメントはSuper PlayerのWeb端末とモバイル端末での体験Demoを提供します。現在のプロジェクトニーズによって相応するデバッグを実行できます。

[](id:web)
## Web端末
Web端末プレーヤーは、PC端末とモバイル端末のブラウザでビデオを再生することをサポートします。豊富で柔軟なインターフェースにより、ユーザーは独自のWebアプリケーションとの統合をすばやく実行できます。

### Demoダウンロード
[Web端末Super Player - DemoのVOD再生](https://tcplayer.vcube.tencent.com/intl/index.html)

[](id:mobile)
## モバイル端末
Tencent Cloud Toolkit Appは、Tencent Cloudオーディオビデオチームが開発した、複数の製品と機能を統合する最高のエクスペリエンスソリューションです。Tencent Cloud Toolkitをダウンロードするには、**Tencent Cloud Toolkit**>**Player**>**Super Player**の順に選択してください。必要に応じて、対応する機能を選択して体験することができます。
