ここでは、Tencent Cloud View Cubeプレーヤー（以下、プレーヤーと略します）のVOD再生シナリオに関わる基本的な概念について説明し、お客様がプレーヤーのビデオオンデマンド機能を速やかに理解し、使用できるようにしています。

[](id:basicConception)
## 基本概念
プレーヤーは、VODプラットフォームでプレーヤーとサードパーティプレーヤープラグインをサポートするという2つの方法により、VODサービスにアクセスします。

### プレーヤー
プレーヤーは、ビデオの暗号化、サムネイルプレビュー、解像度切り替えなどの総合的なビデオ再生機能を備えた独立した完全なビデオプレーヤーです。このプレーヤーはパーフェクトなUIと体験Demoを備えているだけでなく、Tencent Cloud VODと深く連携し、VODファイルIDを使ってVODのリソースを再生できます。またプレーヤーは、VODフルリンクのビデオ再生品質のデータサービスも提供しています。プレーヤーへ速やかにアクセスしたい場合は、[プレーヤーチュートリアル](https://intl.cloud.tencent.com/document/product/266/38098)をご参照ください。


### サードパーティプレーヤープラグイン
サードパーティプレーヤープラグインは、サードパーティのプレーヤーとTencent Cloud VODリソースを接続するために使用するプレーヤープラグインで、ビデオ再生とビデオ暗号化機能を備えています。サードパーティプレーヤープラグインをユーザーのサードパーティプレーヤーに統合することにより、VODファイルIDを使ってVODのリソースを再生できます。サードパーティプレーヤープラグインへ速やかにアクセスしたい場合は、各端末の統合ドキュメントをご参照ください。

## プレーヤーの選び方
VODでは、ユーザーがプレーヤーサービスにアクセスする際は、アクセス難易度を下げ、またご自身のビジネスシナリオにマッチさせるため、最も適したタイプでアクセスすることをお勧めします。
![](https://qcloudimg.tencent-cloud.cn/raw/f793f98980f8f0adb38ee4292ed10d67.png)

- **プレーヤー**：統合されたプレーヤーはまだないが、VODビデオ再生機能を速やかにセットアップする必要があるユーザー向けです。Playerは充実した機能と便利なアクセス機能を備えています。VODでは、プレーヤーへのアクセス方法について、詳細なチュートリアルをユーザーに提供しています。詳細については、[プレーヤーチュートリアル](https://intl.cloud.tencent.com/document/product/266/38098)をご参照ください。
- **サードパーティプレーヤープラグイン**：自社開発/サードパーティのプレーヤーを使用して、VODリソースを再生する必要があるユーザー向けです。VODではサードパーティプレーヤープラグインを提供し、お客様がVODプラットフォームのリソースにスムーズにアクセスし、使用できるようにしています。

各タイプのプレーヤーがサポートするプラットフォーム端末のタイプは以下のとおりです。
