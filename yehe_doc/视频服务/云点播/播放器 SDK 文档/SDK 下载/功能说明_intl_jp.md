## Super Player
Super Playerは各端末向けの充実した機能を提供しています。各プラットフォームがサポートする機能リストは次のとおりです。

| 機能ポイント                 | 機能説明                                                     | iOS & Android | Web  |
| ----------- | -------------------------------------------------- | ------------- | --- |
| 複数の形式               | RTMP、FLV、HLS、MP4、WebRTCなど豊富なオーディオビデオ形式をサポート         | &#10003             | &#10003   |
| URL再生               | ネットワークビデオのURL方式による再生をサポート                                  | &#10003             | &#10003   |
| DASHプロトコル              | 標準プロトコルのDASH形式のビデオをサポート                                 | ×             | &#10003   |
| FileID再生            | VODのFileID方式による再生をサポート                             | &#10003             | &#10003   |
| 最初の画面のインスタントブロードキャスティングプリロード        | ビデオコンテンツのプリロードをサポートし、最初の画面がインスタントブロードキャスティング                       | &#10003             | &#10003   |
| 高速seek              | 指定位置への正確かつ高速なseek再生をサポート                           | &#10003             | &#10003   |
| H.265ハードウェアデコード | H.265ハードウェアデコードと再生をサポート                                      | &#10003             | &#10003   |
| ソフトウェアとハードウェアデコード自動切り替え         | 端末がハードウェアデコードをサポートしていない場合、自動的にソフトウェアデコードに切り替えます                         | &#10003             | &#10003   |
| アダプティブビットレートストリーミング             | HLSアダプティブビットレートストリーミングを再生する場合、手動による指定またはネットワーク帯域幅に応じた解像度ストリームの自動選択をサポートします | &#10003             | &#10003   |
| 解像度の切り替え             | ユーザーによる複数の解像度ストリームのスムーズかつラグのない切り替えをサポート                         | &#10003             | &#10003   |

| プレビュー機能        | プレビュー機能が有効となっているビデオの再生をサポート                                      | &#10003             | &#10003   |
| プログレスバー操作             | プログレスバーをドラッグして進捗状況を切り替えます                                           | &#10003             | &#10003   |
| プログレスバーマークとサムネイルプレビュー | プログレスバーへのマーク情報の追加とサムネイル（スプライトイメージ）プレビューをサポート       | &#10003             | &#10003   |

| スクリーン塗りつぶしの適応 | 画面サイズに合わせてビデオ画面の各種塗りつぶしモードの選択をサポート                 | &#10003             | &#10003   |
| ミニウィンドウ再生        | ミニウィンドウ再生への切り替えをサポート                                          | &#10003             | &#10003   |
| ビデオイメージ        | 水平、垂直等方向のイメージをサポート                                      | &#10003             | &#10003   |
| ビデオ回転               | 角度に応じたビデオ画面の回転をサポートするとともに、ビデオファイルの内部rotateパラメータに基づいたビデオの自動回転をサポート | &#10003             | ×   |

| ロールイメージ               | 一時停止した際の広告表示用のロールイメージをサポート                         | &#10003             | &#10003   |
| ビデオスクリーンキャプチャ               | 再生画面の任意のフレームのキャプチャをサポート                                   | &#10003             | ×   |

| マルチインスタンス                 | 1つのインターフェースに同時再生する複数のプレーヤーの追加サポート             | &#10003             | &#10003   |
| ダウンロードしながらの同時再生               | ビデオを再生しながらそれ以降のコンテンツの同時キャッシュ・ダウンロードをサポート                         | &#10003             | &#10003   |
| Refererリンク不正アクセス防止         | 再生リクエストに含まれるRefererフィールドを介してリクエストのソースを識別し、ブラックリストまたはホワイトリストによるソースリクエストの制御をサポート | &#10003             | &#10003   |
| Keyリンク不正アクセス防止             | 再生リンクへの制御パラメータの追加、リンクの有効時間、プレビュー時間、再生が許可されるIP数などの制御をサポート | &#10003             | &#10003   |

| HTTPS対応    | HTTPSのビデオリソース再生をサポート                                   | &#10003             | &#10003   |
