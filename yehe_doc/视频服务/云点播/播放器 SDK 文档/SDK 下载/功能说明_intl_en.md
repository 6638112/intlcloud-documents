## Superplayer Adapter
Superplayer Adapter provides a wide range of capabilities for different platforms.

| Feature      | Description                                         | iOS and Android | Web |
| -------- | -------------------------------------------- | ----------- | --- |
| QUIC protocol   | Support for QUIC                                     | ×           | &#10003   |
| Playback by file ID   | Play videos by their VOD file ID                               | &#10003             | &#10003   |
| Video quality change       | Smoothly switch between multiple video quality options                                  | &#10003             | &#10003   |
| Custom names for video quality options       | Give custom names to video quality options                                   | &#10003             | &#10003   |
| Progress bar marking | Mark a location on the progress bar                        | &#10003             | &#10003   |
| Thumbnail preview | Display thumbnail sprites on the progress bar | &#10003 | &#10003 |
| Custom video thumbnails        | Add custom thumbnails to videos                                        | &#10003             | &#10003   |
