Tencent Cloud RT-Cube·Player Demo는 완전한 제품 수준의 인터랙티브 UI 및 비즈니스 소스 코드를 제공합니다. Demo를 기반으로 플레이어의 다양한 기능을 시험해 볼 수 있으므로 비즈니스 요구 사항을 신속하게 일치 및 구현하고 개발 시간과 비용을 절약할 수 있습니다. 이 문서는 Web 및 모바일용 Superplayer Demo를 제공합니다. 프로젝트 니즈에 따라 디버깅할 수 있습니다.

[](id:web)
## Web
Web용 플레이어는 PC 및 모바일 브라우저에서 비디오 재생을 지원합니다. 풍부하고 유연한 API는 귀하의 Web 애플리케이션과 신속하게 통합될 수 있습니다.

## 모바일
Tencent Cloud Toolkit App은 Tencent Cloud 오디오/비디오 팀에서 개발한 여러 제품과 기능을 통합한 최고의 평가판 솔루션입니다. **Tencent Cloud Toolkit** > **Player** > **Superplayer**에서 앱을 다운로드하고 평가판에 필요한 기능을 선택할 수 있습니다.
