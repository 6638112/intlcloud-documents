## 소개
미디어 재생 차단을 사용하면 지정된 미디어 콘텐츠의 재생을 차단하거나 차단 해제할 수 있습니다. 규정 위반이 의심되는 미디어 콘텐츠가 발견되면 해당 콘텐츠의 재생을 즉시 금지할 수 있습니다. 차단이 적용된 후 미디어 콘텐츠(소스 미디어 파일, 출력 트랜스코딩된 파일 및 스크린샷과 같은 다양한 미디어 리소스 포함)는 전체 네트워크를 통해 액세스할 수 없으며 VOD 콘솔에서만 미리 볼 수 있습니다. 이는 비준수 미디어 콘텐츠가 더 이상 확산되는 것을 방지하고 플랫폼 보안 리스크와 브랜드 이미지 손상을 줄이는 데 도움이 됩니다. 콘텐츠를 검토하고 비준수 위험을 해결한 후 미디어 콘텐츠의 차단을 해제하여 전체 네트워크에서 다시 액세스할 수 있도록 할 수 있습니다.

## 사용 방법
콘솔에서 또는 API를 호출하여 미디어 재생을 차단하거나 차단을 해제할 수 있습니다. 자세한 내용은 다음을 참고하십시오.
- 콘솔 가이드 [오디오/비디오 관리](https://intl.cloud.tencent.com/document/product/266/33896).
- 서버 API [ForbidMediaDistribution](https://intl.cloud.tencent.com/document/product/266/34180).
