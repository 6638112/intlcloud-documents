## Overview
VOD provides a Player SDK for multiple platforms, which has the following strengths:
<table>
    <tr>
        <th style="width:150px">
            Strength               
        </th>
				<th>
           Description
        </th>
    </tr>
 <tr>
        <td>
            Compatibility with various platforms
        </td>
				<td>
            <li>The Player SDK is compatible with iOS, Android, web (Flash/HTML5), and Flutter.</li>
        </td>
 </tr>
 <tr>
        <td>
            Rich features
        </td>
				<td>
				<li>The Player SDK provides various features such as instant broadcasting of the first frame, pre-roll, mid-roll, and post-roll images, buffering while playback, adjustable-speed playback, video timestamping, on-screen comments, addon subtitles, player logo customization, and player password configuration.</li>
				<li>The Player SDK supports video security solutions such as hotlink protection, URL authentication, HLS encryption, private protocol encryption, and offline download. It also features video security capabilities such as dynamic watermarking to help protect the security of your media assets in different scenarios.</li>
				<li>The Player SDK offers APIs for getting the playback status and setting events.</li>
				<li>The Player SDK supports quick release through iframe and progressive loading of video files.</li>
        </td>
 </tr>
 <tr>
        <td>
            High playback quality
        </td>
				<td>
				<li>The Player SDK utilizes CDN acceleration for the audio/video content to play back media more smoothly.</li>
				<li>The Player SDK supports the QUIC protocol, which delivers a better quality under poor network conditions.</li>
        </td>
 </tr>
  <tr>
        <td>
            Ease of use
        </td>
				<td>
            <li>The playback APIs are simple and can play back media files by media asset ID.</li>
						<li>The Player SDK provides third-party player plugins to play back VOD videos.</li>
        </td>
 </tr>
 <tr>
        <td>
            Detailed playback quality data
        </td>
				<td>
            <li>The Player SDK supports monitoring the video playback quality over the entire linkage through multidimensional metrics such as playback performance, user behaviors, and file characteristics, so as to facilitate efficient business operations.</li>
        </td>
 </tr>
</table>

The Player SDK and CDN jointly implement high-quality, rich-featured, and easy-to-use media playback across various platforms.

 </tr>
 <tr>

            Long video playback
        </td>
				<td>
            The Player SDK integrates VOD features such as adaptive bitrate streaming, seamless definition switch, thumbnail generation, screencapturing, and adjustable-speed playback. It can be used for playback of long videos such as TV series on video platforms as well as portal development.
        </td>
 </tr>
 <tr>
        <td>

        </td>
				<td>
