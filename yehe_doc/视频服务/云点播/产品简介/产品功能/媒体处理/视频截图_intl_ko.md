## 소개

비디오 화면 캡처 기능은 지정된 시간에 비디오의 스크린샷을 찍어 이미지를 생성하는 기능입니다. VOD는 다음과 같은 화면 캡처 방법을 지원합니다.

| 기능  | 설명  |
| -- | -- |
| 시점 스크린샷 | VOD는 지정된 시점의 동영상 스크린샷을 캡처할 수 있습니다. |
| 샘플 스크린샷 | VOD는 지정된 시간 간격으로 여러 장의 동영상 스크린샷을 캡처할 수 있습니다. |
| 썸네일 스크린샷 | VOD는 지정된 시점에 동영상의 스크린샷을 캡처하여 동영상 썸네일로 사용할 수 있습니다. |
| 이미지 스프라이트 | VOD는 지정된 시간 간격으로 비디오의 스크린샷(하위 이미지) 세트를 캡처하고 함께 연결하여 진행률 표시줄에 썸네일을 표시하기 위한 큰 이미지(즉, 이미지 스프라이트)와 VTT 파일을 생성할 수 있습니다. |

## 적용 시나리오

화면 캡처 기능은 다음 애플리케이션 시나리오 니즈를 충족할 수 있습니다.

| 사용 사례 | 설명 |
| -- | -- |
| 썸네일 생성 | 동영상의 지정된 시점에 캡처한 스크린샷을 동영상 썸네일로 사용할 수 있습니다. |
| 하이라이트 컬렉션 | 동영상의 일부 하이라이트를 캡처하여 더 많은 시청자의 관심을 끌 수 있습니다. |
| 수동 조정 | 업로드된 동영상을 조정하기 위해 비디오에서 샘플 스크린샷을 찍을 수 있으며 조정자는 스크린샷을 기반으로 동영상이 규정을 준수하는지 빠르게 판단할 수 있습니다. |
| 비디오 시놉시스 | 이미지 스프라이트는 여러 서브 이미지로 구성된 큰 이미지로, 시청자에게 비디오 콘텐츠의 빠른 개요를 제공하기 위해 비디오의 여러 이미지를 보여줍니다. |
| 진행률 표시줄에서 미리보기 | VTT 파일과 함께 이미지 스프라이트를 사용하여 플레이어 진행률 표시줄의 특정 시점에서 비디오 콘텐츠를 미리 볼 수 있습니다. |
