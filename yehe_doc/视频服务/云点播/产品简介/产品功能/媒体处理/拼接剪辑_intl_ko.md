## 소개

스플라이싱 및 클리핑 기능을 사용하면 오디오/비디오 파일을 밀리초 단위로 정확하게 클리핑하고 여러 개의 서로 다른 오디오/비디오 파일을 새 오디오/비디오 파일로 스플라이싱할 수 있습니다.

| 기능  | 설명  |
| -- | -- |
| 오디오/비디오 클리핑 | VOD의 파일을 클리핑하여 새 오디오/비디오 파일을 생성합니다. |
| 오디오/비디오 스플라이싱 | VOD의 여러 파일을 스플라이싱하여 새 오디오/비디오 파일을 생성합니다. |
| 오디오/비디오 클리핑 및 스플라이싱 | VOD에서 여러 파일을 클리핑한 다음 클립을 스플라이싱하여 새 오디오/비디오 파일을 생성합니다. |
| 온 디맨드 오디오/비디오로 라이브 스트림 | VOD에 녹화된 라이브 스트림을 처리하여 새로운 오디오/비디오 파일을 생성합니다. |
| 라이브 스트림 클리핑 | VOD에 녹화된 라이브 스트림을 클리핑하여 새로운 오디오/비디오 파일을 생성합니다. |
| 라이브 스트림 스플라이싱 | VOD에 녹화된 여러 라이브 스트림을 스플라이싱하여 새 오디오/비디오 파일을 생성합니다. |
| 라이브 스트림 클리핑 및 스플라이싱 | VOD에 녹화된 여러 라이브 스트림을 클리핑한 다음 클립을 스플라이싱하여 새 오디오/비디오 파일을 생성합니다. |

## 적용 시나리오

| 사용 사례 | 설명 |
| -- | -- |
| 오프닝 및 엔딩 세그먼트 스플라이싱 | 오디오/비디오 스플라이싱을 사용하여 광고, 홍보 자료 또는 비디오 소개와 같은 시작 및 끝 부분을 비디오에 추가할 수 있습니다. |
| 하이라이트 클리핑 | 오디오/비디오 클리핑을 사용하여 미디어 파일에서 하이라이트 클립을 만들 수 있습니다. |
| 중단된 라이브 스트림 스플라이싱 | 라이브 스트리밍이 중단되면 라이브 스트리밍을 VOD로 녹화할 때 여러 개의 동영상 파일이 생성됩니다. 이 경우 라이브 스트림 스플라이싱을 사용하여 완전한 VOD 비디오를 생성할 수 있습니다. |
