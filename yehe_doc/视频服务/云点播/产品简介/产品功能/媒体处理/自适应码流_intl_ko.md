## 소개

어댑티브 비트 레이트 스트리밍(Adaptive Bitrate Streaming, ABR)에는 다양한 비트 레이트로 오디오/비디오 파일을 만드는 작업이 포함됩니다. 어댑티브 비트 레이트 스트리밍을 사용한 후 플레이어는 시청자의 현재 대역폭을 기반으로 가장 적절한 비트 레이트를 동적으로 선택하고 재생할 수 있습니다. VOD는 비디오를 HLS 및 DASH와 같은 주요 어댑티브 비트 레이트 스트리밍 형식으로 변환할 수 있습니다.

| 기능 | 설명 |
| -- | -- |
| 지원되는 형식 | HLS 및 DASH. |
| 초저지연 재생 시작 | 어댑티브 비트레이트 스트리밍에는 여러 해상도가 포함되어 있습니다. 플레이어는 일반적으로 재생을 더 빨리 시작할 수 있도록 저해상도에서 재생을 시작합니다. |
| 스마트 해상도 전환 | 플레이어는 현재 대역폭을 기반으로 재생에 가장 적합한 해상도를 동적으로 선택합니다. |
| 제로 랙 전환 | 어댑티브 비트레이트 스트리밍에서 각 해상도의 프레임이 정렬되므로 해상도가 전환될 때 랙이 없습니다. |
| UHD 이미지 및 오디오 | <li>최대 8K UHD 해상도 지원</li><li>스테레오 지원</li><li>HDR 지원</li>|
| 고급 인코딩 기술 | H.265, H.266, AV1과 같은 고급 인코딩 포맷과 최고 속도 코덱 트랜스코딩을 사용하여 비디오 비트 레이트를 줄일 수 있습니다. |
| 암호화 및 DRM 지원 | 미디어 암호화, Widevine 및 FairPlay와 같은 DRM 저작권 보호 체계는 모두 어댑티브 비트레이트 스트리밍에 의존합니다. |


>! **어댑티브 비트레이트 스트리밍**과 **트랜스코딩**의 차이점:
>- 어댑티브 비트레이트 스트리밍 URL에는 여러 해상도의 출력이 포함되지만 트랜스코딩 URL에는 하나의 해상도만 출력이 포함됩니다.
>- 어댑티브 비트스트림을 재생할 때 플레이어는 현재 네트워크 상태에 따라 실시간으로 최적의 해상도로 전환되지만 트랜스코딩으로 출력되는 비디오는 재생 중 지능형 해상도 전환을 지원하지 않습니다.

## 적용 시나리오

| 사용 사례 | 설명 |
| -- | -- |
| 온라인 교육 | 과목 교육 및 교육 과정과 같은 온라인 교육 시나리오에서 비디오는 일반적으로 DRM을 기반으로 암호화되거나 저작권이 보호되어야 합니다. |
| 비디오 웹사이트 | 비디오 웹 사이트에는 풍부한 비디오 리소스가 있으며 어댑티브 비트레이트 스트리밍은 보다 부드러운 시청 경험을 제공할 수 있습니다. |
| 온라인 TV | 온라인 TV 플랫폼은 리소스를 보호해야 할 수 있습니다. 또한 시청자는 비디오를 시청하는 동안 더 높은 해상도로 전환하고 싶어할 수 있습니다. |
