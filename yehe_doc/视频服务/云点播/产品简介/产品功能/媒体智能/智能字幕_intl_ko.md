## 소개
스마트 자막을 사용하면 미디어 파일의 오디오에 대한 자막 파일을 자동으로 생성할 수 있습니다. 이 기능은 온라인 코스 녹음, 오디오/비디오 회의록 및 음성 품질 검사에 유용합니다. VOD의 스마트 자막 기능은 고급 음성 인식 알고리즘과 방대한 양의 훈련 데이터를 기반으로 업계 최고의 음성 인식 기능을 활용하여 주변 소음 및 배경 소음과 같은 다양한 간섭 속에서도 높은 인식 정확도를 제공합니다.

## 사용 방법
다음과 같이 [비디오 콘텐츠 인식](https://intl.cloud.tencent.com/document/product/266/33946)에 설명된 대로 **전체 음성-텍스트 인식** 기능을 통해 스마트 자막을 구현할 수 있습니다.
1. [CreateAIRecognitionTemplate](https://intl.cloud.tencent.com/document/product/266/37568), 전체 음성-텍스트 인식(`AsrFullTextConfigure`)을 활성화하고 자막 생성을 지정합니다.
```
{
	"AsrFullTextConfigure": {
		"Switch": "ON",
                "SubtitleFormats": ["vtt", "srt"]
	}
}
```
응답에서 템플릿 ID를 가져옵니다.
2. 1단계에서 얻은 템플릿 ID로 [비디오 콘텐츠 인식](https://intl.cloud.tencent.com/document/product/266/33946#.E4.BB.BB.E5.8A.A1.E5.8F.91.E8.B5.B7)에 따라 스마트 자막 작업을 시작합니다.
3. [비디오 콘텐츠 인식](https://intl.cloud.tencent.com/document/product/266/33946#.E7.BB.93.E6.9E.9C.E8.8E.B7.E5.8F.96)에 따라 작업 결과를 가져옵니다.
