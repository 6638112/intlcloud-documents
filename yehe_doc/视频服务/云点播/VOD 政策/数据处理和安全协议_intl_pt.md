## 1\. CONSIDERAÇÕES INICIAIS

Este Módulo se aplica se você usar o recurso de Vídeo Sob Demanda ( Video on Demand) (“**Recurso**”). Este Módulo está incorporado no Contrato de Segurança e Processamento de Dados localizado no [(“DPSA”)](https://intl.cloud.tencent.com/document/product/301/17347). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, este Módulo se aplicará conforme a inconsistência.

## 2\. **PROCESSAMENTO**

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações Pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de configuração:** ID do arquivo, definição, nome, FPS, largura, altura, espaço de cor, bitrate, tempo de criação, tempo de atualização, nome da mídia, tamanho da mídia, tipo de mídia, região de armazenamento, caminho de armazenamento, domínio | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você e a seus usuários finais, conforme sua configuração específica.<br/>Observe que esses dados estão armazenados com backup em nosso recurso TencentDB for MySQL. |
| **Arquivos de mídia:** vídeos, imagens, arquivos de áudio carregados por você | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você, inclusive para efeitos de armazenamento.<br/>Observe que esses dados são carregados por você e armazenados em nosso recurso Cloud Object Storage (Armazenamento de objetos na nuvem, COS) e são armazenados em cache usando nosso recurso Content Delivery Network (Rede de entrega de conteúdo, CDN). |

## 3\. REGIÃO DO SERVIÇO

Conforme especificado no DPSA.

## 4\. SUBPROCESSADORES

Como especificado no DPSA, incluindo Aceville Pte. Ltd.

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações Pessoais**      | **Política de Retenção**                                         |
| ----------------------------- | ------------------------------------------------------------ |
| Configuração de arquivos DataMedia | Retemos os dados de acordo com o seu limite de tempo de retenção de dados escolhido. Caso contrário, quando você excluir sua conta ou encerrar o seu uso do Recurso, nós vamos excluir esses dados após 60 dias. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

Você deve garantir que este Recurso seja usado apenas por usuários finais que têm, pelo menos, a idade mínima para que um indivíduo possa consentir com o processamento de seus dados pessoais ou com o consentimento dos pais obtido para pessoas com idade inferior à mínima exigida. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.

Você declara, garante e assume que deve obter e manter todos os consentimentos necessários dos Usuários Finais em relação ao processamento dos dados pessoais deles em relação ao Recurso, de acordo com as leis aplicáveis e de modo a nos permitir cumprir essas leis.
