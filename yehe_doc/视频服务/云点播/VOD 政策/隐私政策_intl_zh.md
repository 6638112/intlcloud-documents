## 1\. 简介

本模块仅在您使用云点播功能（下称“**功能**”）时适用。本模块已纳入[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)的隐私政策中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. 可用性

本功能可供全球范围内的用户使用。

## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **日志数据：** APPID、时间戳、错误消息               | 我们出于定位问题和故障排除的目的而使用此信息。<br/>请注意，此数据存储在我们的日志服务 (CLS) 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |
| **使用数据：** APPID、时间、CDN 流量、主机、请求、标准存储、低频存储、归档存储、深度归档存储、转码分钟、分辨率 | 我们将此信息用于计费目的，以及在控制台上显示您的使用情况。<br/>请注意，这些数据存储和备份在我们的腾讯云数据库 MySQL 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |
| **监控数据：** APPID、请求类型、错误代码、错误消息、请求时间、平台、设备、操作系统类型、网络类型、文件大小、文件类型、UUID、包名称、应用名称 | 我们使用此信息的目的是为您提供本功能以及进行性能分析。<br/>请注意，这些数据存储在我们的 CLS 功能中。 | 我们会处理此类信息，因为这对于我们履行与您签订的合同向您提供本功能而言是必要的。 |

## 5\. 我们如何分享和存储个人信息

见隐私政策中的规定。

## 6\. 我们如何分享个人信息

见隐私政策中的规定，包括 Aceville Pte Ltd。

## 7\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息** | **保留政策** |
| ------------------------ | -------------------- |
| 日志数据                 | 存储 7 天。   |
| 使用数据               | 存储 1 年。   |
| 监控数据          | 存储 7 天。   |
