| **Personal Information**                                     | **Use**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Configuration Data:** file ID, definition, name, FPS, width, height, colorspace, bitrate, create time, update time, media name, media size, media type, storage region, storage path, domain | We only process this data for the purposes of providing the Feature to you and your end users, in accordance to your specific configuration.<br/>Please note that this data is stored and backed up in our TencentDB for MySQL feature. |
