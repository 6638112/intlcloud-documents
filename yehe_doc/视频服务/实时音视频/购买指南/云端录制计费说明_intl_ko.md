## 사용량 통계 방식

TRTC는 동일한 Tencent Cloud 계정의 모든 애플리케이션이 클라우드 녹화를 사용하여 전송한 결과의 **녹화 시간**을 기준으로 클라우드 녹화 서비스 사용량을 계산합니다. 녹화 시간은 클라우드 녹화 결과에 따라 **비디오 시간**과 **음성 시간**으로 나뉩니다.

>! 시간은 초 단위로 계산되며, 누적된 일일 초 단위는 분 단위로 전환하여 과금합니다. 1분 이하의 경우 1분으로 계산합니다.


### 비디오 시간
비디오 시간은 녹화 결과 중 비디오 화면이 포함된 시간입니다. TRTC는 녹화된 비디오의 해상도에 따라 비디오 레벨을 분류한 후, 분류된 레벨의 비디오 시간에 따라 과금합니다. 비디오 해상도에 따른 레벨 분류는 다음과 같습니다.

|비디오 레벨|해상도|
|-------|--------|
|SD|640×480 이하(포함)|
|HD|640×480~1280×720(포함)|
|FHD|1280×720 초과|


- 같은 시간에 녹화한 하나의 녹화 파일에 비디오와 오디오가 모두 있는 경우, 비디오 시간만 계산하며 음성 시간을 중복 계산하지 않습니다.
- 녹화 후 실제 전송된 해상도는 입력 해상도에 따라 달라질 수 있습니다. TRTC는 레벨에 따라 서비스 사용량을 계산하고 5분에 한 번씩 업데이트합니다.


### 음성 시간
음성 시간은 녹화 결과 중 순수하게 오디오만 기록된 시간입니다.

## 과금 방식
요금 지불 방식입니다. TRTC 클라우드 녹화 서비스는 **일 결산 후불** 방식만 지원하며, 일일 과금 방식에 따라 매일 오전 10시에 전날 발생한 요금을 차감합니다.

>!클라우드 녹화는 **CSS**와 **VOD**에 종속되며, LVB 또는 VOD로 인해 Tencent Cloud 계정의 요금 연체가 발생할 경우 클라우드 녹화에 실패합니다.

## 과금 예시
>!
>- 본 문서는 정가 기준으로 계산한 과금 예시입니다. Tencent Cloud 비즈니스 매니저와 계약을 체결한 경우 계약에서 약정한 금액을 기준으로 합니다.
>- 모든 TRTC 방에서 모든 사용자의 멀티미디어 스트림은 기본적으로 개별 파일로 녹화됩니다. 한 방에서 여러 사용자의 비디오 화면을 1개의 파일로 녹화하고 싶다면 [클라우드 혼합 스트림 트랜스 코딩](https://intl.cloud.tencent.com/document/product/647/34618)을 통해 여러 개의 비디오 화면을 하나의 파일로 통합할 수 있습니다. 클라우드 혼합 스트림 트랜스 코딩 기능을 사용하면 릴레이 트랜스 코딩 요금이 추가로 발생합니다. 자세한 내용은 [클라우드 혼합 스트림 트랜스 코딩 과금 설명](https://intl.cloud.tencent.com/document/product/647/38929)을 참조하십시오.

사용자 A, B, C가 함께 TRTC 방에서 10분 연속 통화하는 경우, 통화 과정 전체를 녹화해야 합니다. A, B, C의 업스트림 푸시 스트림 유형과 해상도 정보는 다음과 같습니다.

|사용자|푸시 스트림 유형|해상도|상응하는 과금 유형|
|---|---|---|---|
|A|오디오만|없음|음성|
|B|비디오+오디오|640×360|SD|
|C|비디오만|1280×720|HD |

### 혼합 스트림 녹화가 아닌 경우
혼합 스트림이 아닌 경우 A, B, C가 각각 1개의 파일에 녹화되므로 3명에게 총 3개의 파일이 생성되며, 해당 통화로 발생한 녹화 요금은 다음과 같습니다.
>A의 녹화 요금+B의 녹화 요금+C의 녹화 요금=음성 시간 단가×A의 음성 시간+ SD 비디오 시간 단가×B의 SD 비디오 시간+HD 비디오 시간 단가×C의 HD 비디오 시간=0.499USD/1,000분×10분/1,000+0.99USD/1,000분×10분/1000+1.99USD/1,000분×10분/1,000=0.03479USD

### 혼합 스트림 후 재녹화하는 경우

혼합 스트림인 경우 A, B, C 혼합 스트림 후 1개의 파일로 녹화합니다. 혼합 후 전송된 비디오 해상도가 1280×720을 유지했다면 해당 통화로 발생한 녹화 요금은 다음과 같습니다.
>HD 비디오 시간 단가×HD 비디오 시간=1.99USD/1,000분×10분/1,000=0.0199USD

클라우드 혼합 스트림 트랜스 코딩 기능을 사용하면 릴레이 트랜스 코딩 요금이 추가로 발생합니다. 자세한 내용은 [클라우드 혼합 스트림 트랜스 코딩 과금 설명](https://intl.cloud.tencent.com/document/product/647/38929)을 참조하십시오.
