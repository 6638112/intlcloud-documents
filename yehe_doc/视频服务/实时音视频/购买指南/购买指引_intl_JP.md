
Tencent Real-Time Communication （TRTC）は前払いパッケージと後払いをサポートしています。デフォルトでは前払いパッケージを採用し、後払いは購入したパッケージを消費したり、期限切れになったりした後に自動的に開始され、直接開始することはできません。ここでは主に前払いパッケージの購入方法をご紹介します。

## 前提条件
TRTC基本サービスパッケージを購入する前に、まず[Tencent Cloudアカウントの登録](https://intl.cloud.tencent.com/document/product/378/17985)を行い、[実名認証](https://intl.cloud.tencent.com/document/product/378/3629)を完了する必要があります。実名認証を実行していない場合は、中国国内のTRTCインスタンスを購入することができません。

## 操作手順
1. [TRTCパッケージ購入画面](https://buy.cloud.tencent.com/trtc)にログインします。
2. 【通話タイプ】、【パッケージタイプ】、および【パッケージ内容】を選択します。
3. 【私はTencent Cloud TRTC[『サービスレベルプロトコル』](https://intl.cloud.tencent.com/document/product/647/35661) を読んで同意しました】にチェックを入れ、【今すぐ購入】をクリックします。
4. 情報を確認し、支払いを完了します。
