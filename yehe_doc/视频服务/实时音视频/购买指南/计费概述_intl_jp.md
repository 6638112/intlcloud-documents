# 課金概要

- **使用量に応じた支払い額が明確な請求書**

  [ここをクリック](https://intl.cloud.tencent.com/contact-us)し、ビジネスに関するお問い合わせから、詳細情報をお調べください。

- **毎月、10000分間[無料時間](https://intl.cloud.tencent.com/document/product/647/42735)が提供されます**。

- **[前払いパッケージ](https://intl.cloud.tencent.com/document/product/647/42736)から予約購入すると割引が適用されます**。

  TRTC前払いパッケージの購入前に、まず[Tencent Cloudアカウントの登録](https://intl.cloud.tencent.com/document/product/378/17985)を行った上で、[お問い合わせ](https://intl.cloud.tencent.com/contact-us)にご連絡いただき、ホワイトリストの審査を受けていただく必要がありますので、ご注意ください。

## 毎月の無料時間10000分

TRTCをアクティブ化したすべてのTencent Cloudアカウントには、毎月10000分の無料時間パッケージが提供されます。ここからオーディオビデオ通話、インタラクティブライブストリーミング、クラウドレコーディング、ミクスストリーミングトランスコードなど、すべての項目を差し引くことができます。詳細については、[無料時間の説明](https://intl.cloud.tencent.com/document/product/647/39784)をご参照ください。

## 基本サービス

基本サービスは、具体的なユースケースによって、ボイス・インタラクティブストリーミング、ビデオ・インタラクティブストリーミング、音声通話、ビデオ通話の4つに分けられ、4つの基本サービスは単独で使用することも、重複して使用することも可能です。各基本サービスに対応する課金説明は以下のとおりです。

| サービス     |  サービス説明                                                     | 課金説明                                                     |
| :------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| 音声通話 | 2人または多人数での音声チャット。48kHz、ダブルサウンドチャンネルをサポート。また1つのルームで最大300人のオンライン同時接続と、最大50人のマイク通話同時接続をサポートしています。 | [TRTC課金](https://intl.cloud.tencent.com/document/product/647/42734)|
| ビデオ通話 | 2人または多人数でのビデオ通話では、720P、1080PのHD画質をサポートします。1つのルームで最大300人のオンライン同時接続と、最大50人のカメラ同時起動をサポートしています。 | [TRTC課金](https://intl.cloud.tencent.com/document/product/647/42734) |
| インタラクティブライブストリーミング | キャスターと視聴者間のオーディオビデオマイク接続やキャスターのルーム間（ライブストリーミングルーム間）PKもサポートしています。スムーズなマイクのオン・オフをサポートしており、切替時に待つ必要がなく、キャスターの遅延は300ms未満です。1つのルームでのマイク接続人数を無制限にでき、最大50人のマイク同時接続をサポートします。低遅延ライブストリーミングモードでは、視聴者10万人の同時再生をサポートし、再生の遅延を1000msまで低減させます。CDN Relayed Live Streamingモードでは、視聴者数は無制限です。[TRTC課金](https://intl.cloud.tencent.com/document/product/647/42734) | [TRTC課金](https://intl.cloud.tencent.com/document/product/647/39785) |
