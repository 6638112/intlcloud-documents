## 基本サービス料金

基本サービスとは、音声通話、ビデオ通話、双方向対話型マイク接続、低遅延ライブストリーミング等の基本となるTRTC通信サービスを指し、基本サービスを利用する際に発生する通話料金を基本サービス料金といいます。 




## 月締め後払い

TRTCの基本サービスの利用により発生した通話料金はいずれも月締め後払いの方式を採用し、毎月1-5日に前月（暦月 ）の請求書が発行され、費用が引き落とされます。

具体的な課金基準は下表の通りとなります。

| 通話タイプ              | 単価             |
| :-------------------- | :--------------- |
| 音声通話              | 0.99米ドル/1000分  |
| 標準（SD）ビデオ通話    | 1.99米ドル/1000分  |
| 高精細度（HD）ビデオ通話    | 3.99米ドル/1000分  |
| 超高精細度（HD+）ビデオ通話 | 14.99米ドル/1000分 |

**課金説明：**

1. 課金項目：音声通話、標準（SD）ビデオ通話、高精細度（HD）ビデオ通話、超高精細度（HD+）ビデオ通話。

2. 課金方式：後払い

3. 課金周期：月ごとに課金し、每月1～5日に前月度の請求書を発行します。

4. 課金ルール：コーディング方式と解像度にもとづき、1日（暦日）のコーディング方式と解像度ごとのトランスコーディング時間に、対応する単価を掛けて統計します。

5. 課金の例：

   仮に、2020年1月1日にTRTCアプリケーションを作成し、1ヶ月に使用した総音声通話時間が10000分、総標準（SD）ビデオ通話時間が10000分、総高精細度（HD）ビデオ通話時間が10000分、総超高精細度（HD+）ビデオ通話時間が10000分とします。

   音声通話料金 = (10000/1000)1000分 * 0.99米ドル/1000分 = 9.9米ドル

   標準（SD）ビデオ通話料金 =  (10000/1000)1000分 * 1.99米ドル/1000分 = 19.9米ドル

   高精細度（HD）ビデオ通話料金 =  (10000/1000)1000分 * 3.99米ドル/1000分 = 39.9米ドル

   超高精細度（HD+）ビデオ通話料金 = (10000/1000)1000分*14.99米ドル/1000分 = 149.9米ドル

   総費用 = 音声通話料金 + 標準（SD）ビデオ通話料金 + 高精細度（HD）ビデオ通話料金 + 超高精細度（HD+）ビデオ通話料金 = 9.9米ドル + 19.9米ドル + 39.9米ドル + 149.9米ドル  = 219.6米ドル。

   > 説明：
   >
   > - TRTCの各課金項目で課金対象となる通話時間が1分未満の場合は1分で計算します（例：音声通話時間が1000.5分の場合、課金時間は1001分となります）。
