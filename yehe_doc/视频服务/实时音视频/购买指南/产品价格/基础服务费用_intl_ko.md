## 기본 서비스 요금

기본 서비스란 음성 통화, 비디오 통화, 인터랙션 마이크 연결, 저 딜레이 라이브 방송 등 기본적인 TRTC 통신 서비스이며, 이러한 기본 서비스를 이용하여 발생하는 통화 비용이 기본 서비스 요금입니다.(하기 이미지 참조)

## 월 결산 후불 관련 요금

TRTC 기본 서비스를 이용하여 발생하는 통화 비용은 모두 월 결산 후불 방식으로, 매월 1~5일에 지난 달 청구서가 발행되고 비용이 차감됩니다.

구체적인 과금 기준은 다음과 같습니다.

| 통화 유형              | 단가             |
| :-------------------- | :--------------- |
| 음성 통화              | 0.99 USD/1,000분  |
| SD 비디오 통화    | 1.99 USD/1,000분  |
| HD 비디오 통화    | 3.99 USD/1,000분  |
| HD+ 비디오 통화    | 14.99 USD/1,000분  |

**결제 설명: **

1. 과금 항목: 음성 통화, SD 비디오 통화, HD 비디오 통화, HD+ 비디오 통화

2. 과금 방식: 후불

3. 과금 주기: 월간 결산, 매월 1~5일 청구서 발행

4. 과금 규정: 코딩 방식과 해상도에 따라 과금, 1일간 각 코딩 방식과 해상도의 트랜스 코딩 시간 × 해당 단가

5. 과금 예시:

   2020년 1월 1일 TRTC 애플리케이션을 생성했고, 1월 음성 통화 시간이 총 10,000분, SD 비디오 통화 시간이 총 10,000분, HD 비디오 통화 시간이 총 10,000분, HD+ 비디오 통화 시간이 총 10,000분인 경우,

   음성 통화 비용 = (10,000/1,000)1,000분 * 0.99 USD/1,000분 = 9.9 USD

   SD 비디오 통화 비용 =  (10,000/1,000)1,000분 * 1.99 USD/1,000분 = 19.9 USD

   HD 비디오 통화 비용 =  (10,000/1,000)1,000분 * 3.99 USD/1,000분 = 39.9 USD

   HD+ 비디오 통화 비용 = (10,000/1,000)1,000분 * 14.99 USD/1,000분 = 149.9 USD

   총 요금 = 음성 통화 비용 + SD 비디오 통화 비용 + HD 비디오 통화 비용 + HD+ 비디오 통화 비용 = 9.9 USD + 19.9 USD + 39.9 USD + 149.9 USD  = 219.6 USD

   > 설명:
   >
   > - TRTC 각 과금 항목의 과금 통화 시간이 1분 이하인 경우 1분으로 계산합니다. 예: 음성 통화 시간이 1,000.5분인 경우 1,001분으로 계산
