## 使用量の統計方法

Tencent Real-Time Communication（TRTC）は、同じTencent Cloudアカウント下の全アプリケーションがMCUミクスストリーミングトランスコードを経た後に出力された**トランスコード時間**に基づき、バイパストランスコードサービスの使用量を統計します。トランスコード時間はコーデックとトランスコードの結果によって異なり、[ビデオ時間](#m_video) と [音声時間](#m_voice)に分かれます。

>!  時間の統計精度は秒とし、日毎に秒数を分数に変換してから課金します。1分未満は1分で計算します。

<span id="m_video"></span>
### ビデオ時間
ビデオ時間は、トランスコード結果に含まれるビデオ画面の時間を指します。TRTCは、トランスコード後に出力されるビデオの解像度に応じてビデオのグレードを区分し、その後グレードごとのビデオ時間に対してそれぞれ課金します。ビデオのグレードと解像度の対応関係は下表のとおりとなります。

| ビデオのグレード   | 出力解像度                   |
| ---------- | ---------------------------- |
| 標準SD    | 640 × 480（480を含む）以下        |
| 高精細度HD    | 640 × 480 - 1280 × 720（720を含む）|
| フル高精細度FHD | 1280 × 720より上               |

- トランスコード後に出力される同一ストリームの同一時間内にビデオもオーディオも存在する場合は、ビデオ時間のみで統計し、音声時間が重複して計算されることはありません。
- トランスコード後に出力される同一ストリームの解像度は変化する可能性があります。TRTCではサービス使用量を分割して統計し、通常の状況下では60秒に1回更新します。解像度に変化が生じた場合はすぐに報告され、更新されます。

<span id="m_voice"></span>
### 音声時間

音声時間とは、トランスコード結果の中にオーディオのみが存在する時間を指します。

## サービス料金
TRTC Cloud MixTranscodingサービスの公表価格は下表のとおりです。

| コーデック | 課金項目                   | 単価（米ドル/分）|
| -------- | ------------------------ | ------------------- |
| オーディオトランスコード | バイパストランスコード-音声            | 0.000799米ドル/分   |
| H.264    | バイパストランスコード-H264-標準SD    | 0.00228571米ドル/分 |
| H.264    | バイパストランスコード-H264-高精細度HD    | 0.00464286米ドル/分 |
| H.264    | バイパストランスコード-H264-フル高精細度FHD | 0.00899米ドル/分    |

<span id="Billing_method"></span>

## 課金方式
支払い方式。TRTC Cloud MixTranscodingサービスは**日次決済後払い**方式のみをサポートしています。日ごとに課金し、毎日午前10時に前日に発生した費用を引き落とします。

<span id="Billing_examples"></span>
## 課金の例
>!本文の課金の例は公表価格を採用して計算しています。お客様とTencent Cloudのビジネスマネージャーが契約を交わしている場合は、契約で取り決めた価格を基準とします。

1月1日にTRTCのMCUクラスターを使用してCloud MixTranscodingを行い、H.264コーデックエンコード方式を使用して、解像度1920 × 1080、640 × 360のビデオを各100分間出力し、同時にオーディオトランスコードを使用して100分間の音声を出力したとします。

**分析：**
- 解像度1920 × 1080で出力したビデオのグレードはフル高精細度(FHD)であり、単価は0.00899米ドル/分です。
- 解像度640 × 360で出力したビデオのグレードは高精細度(HD)であり、単価は0.00464286米ドル/分です。
- 音声出力の単価は0.000799米ドル/分です。

この場合、1月2日に支払うべき、1月1日に発生したバイパストランスコード料金は、次のようになります。0.00899米ドル/分 × 100分 + 0.00464286米ドル/分 × 100分 +0.000799米ドル/分 × 100分 = 2.16米ドル。
