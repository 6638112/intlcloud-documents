TUIkitは、Tencent Cloudオーディオビデオチームが5000を超えるお客様へのサービス経験を蓄積し、業界内で主流のオーディオビデオシーンを踏まえてブラッシュアップしたオープンソースソリューションです。ビデオ通話コンポーネント、ライブストリーミングコンポーネント、ビデオルームコンポーネントなどの複数のクライアントオーディオビデオコンポーネントが含まれ、通話、カスタマーサービス、ライブストリーミング、ボイスチャット、教育などのシーンをスピーディーに構築したい開発者に役立つソリューションとなっています。

>?TUIKitシリーズコンポーネントはTencent Cloudの[TRTC](https://intl.cloud.tencent.com/document/product/647/35078)と[IM](https://intl.cloud.tencent.com/document/product/1047/35448)という2つの基本的なPaaSサービスを同時に使用し、TRTCをアクティブにした後、IMサービスを同期してアクティブ化することができます。 IMサービスの課金ルールの詳細については、[Instant Messagingの料金説明](https://intl.cloud.tencent.com/document/product/1047/34350)をご参照ください。TRTCをアクティブ化すると、関連するIM SDKの体験版がデフォルトでアクティブ化されます。これは100 DAUのみをサポートします。

## TUIKit製品ラインナップ
![](https://qcloudimg.tencent-cloud.cn/raw/22b2ed779af3b76550ae5eefc8704e4c.png)

上の図のように、TUIKitはTUICompenontとTUIWidgetの2種類に分かれており、オプションの基本バックエンドサービスも同時にサポートしています。
```java
├── TUIComponent
│   ├── TUICalling     // 通話コンポーネント（WeChat通話に類似）。ビデオ通話、カスタマーサービス、金融審査などのオーディオビデオシーン向けです。
│   ├── TUIRoom        // 多人数ビデオルームコンポーネント。教育、ミーティング、面接試験などのオーディオビデオシーン向けです。ミュート、画面オフなどの機能を備えています。
│   ├── TUILiveVoice   // ボイスチャットライブストリーミングコンポーネント。フレンドとのボイスチャット、キューボイスチャット、音楽ルームなどのオーディオシーン向けです。 
│   ├── TUILiveVideo   // ビデオインタラクティブストリーミングコンポーネントです。マイク接続、PK、オーディオエフェクトなどの機能を備えています。
│   ├── TUIChatSolon   // ボイスサロンコンポーネントです。円卓会議、コミュニティフォーラムなどのオーディオビデオシーン向けです。
│   ├── TUIPusher      // 完全なUIを備えたプッシュコンポーネントです。複数のプロトコルプッシュ、PKなどの機能をサポートし、オーディオエフェクト、弾幕などのウィジェットのマウントをサポートします。
│   ├── TUIPlayer      // 完全なUIを備えたプルコンポーネントです。複数のプロトコルプル、マイク接続などの機能をサポートし、弾幕、ギフトなどのウィジェットのマウントをサポートします。
│   ├── TUIChorus       // 新しいコンポーネントです。合唱の新しい楽しみ方向けのオーディオビデオコンポーネントであり、Tencent Cloud正規版楽曲データベースと組み合わせるなど、楽しみ方をさらに広げることができます。
│   ├── TUIKaraoke      // 新しいコンポーネントです。オンラインKTVなどの新しい楽しみ方向けのオーディオビデオコンポーネントであり、Tencent Cloud正規版楽曲データベースと組み合わせるなど、楽しみ方をさらに広げることができます。
├── TUWidget
│   ├── TUIBeauty      // 美顔ウィジェット
│   ├── TUIAudioEffect // オーディオエフェクトウィジェット
│   ├── TUIBarrage     // 弾幕ウィジェット
│   ├── TUIGift        // ギフトウィジェット
├── aPass （オプション）
│   ├── Room Service   // ルームリスト、ルーム人気度、キャスターリストなどの機能をご提供します。
```
