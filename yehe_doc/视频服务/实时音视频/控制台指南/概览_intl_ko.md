Tencent Real-Time Communication(TRTC)은 사용량 통계, 모니터링 대시보드, 개발 보조, 패키지 관리, 애플리케이션 관리 등의 기능 설정과 관리 작업을 지원합니다.

## 개요
왼쪽 메뉴 [개요](https://console.cloud.tencent.com/trtc)를 클릭하여 당월 누적 시간, 최근 30일 대역폭 추세, 패키지 사용 상황 등의 관련 데이터를 조회할 수 있으며 단축 툴 버튼으로 TRTC Demo 빠른 시작, 기존 생성 애플리케이션 관리, 통화 품질 실시간 모니터링, 관련 문서 조회 등의 작업을 할 수 있습니다.
![](https://main.qcloudimg.com/raw/a934844335e5d6edddc7d55a128469d9.png)


## 사용량 통계
### 당월 누적 시간
**음성 시간**, **표준 화질 비디오 시간**, **고화질 비디오 시간**, **초고화질 비디오 시간** 등의 당월 누적 TRTC 사용 시간 데이터를 보여줍니다.
![](https://main.qcloudimg.com/raw/7f5cf4df97ada1923636c4b70e686cd4.png)
사용량 데이터에 대한 자세한 내용은 우측 상단의 [자세한 내용]을 클릭하고 [사용량 통계](https://console.cloud.tencent.com/trtc/statistics)로 이동하여 조회할 수 있습니다. 자세한 내용은 [사용량 통계](https://intl.cloud.tencent.com/zh/document/product/647/39066)를 참조하십시오.


### 최근 30일 시간 추세
최근 30일의 TRTC 사용량 추세(분 단위까지 정확함)를 보여주며 마우스를 이동하여 최근 30일 내의 하루 사용량 데이터를 조회할 수 있습니다.
![](https://main.qcloudimg.com/raw/a52e20a1bf042e71dd2003eeac53a768.png)


## 패키지 개요
본 계정 패키지의 최근 사용 근황을 보여줍니다. 패키지 ID, 패키지 유형, 패키지 이름, 패키지 기한, 남은 시간, 생성 시간, 종료 시간, 상태 등 데이터 패키지를 보여줍니다.
![](https://main.qcloudimg.com/raw/514a61b55aae8b0445b9ecc65d9065fa.png)
- 패키지 상태가 “사용 중”이라면 패키지 우측의 [알람 설정]을 통해 잔량 알람 기능을 활성화할 수 있습니다.
<img src="https://main.qcloudimg.com/raw/e1161d443c5bf5de5691ccc01faafe36.png" width="450px"/>
- 새로운 패키지 구매가 필요한 경우 [패키지 구매]를 클릭하여 TRTC 범용 패키지 구매 페이지에서 구매를 진행할 수 있습니다.
>? 패키지를 구매하기 전 과금 정보를 확인하십시오. 자세한 내용은 [과금 개요](https://intl.cloud.tencent.com/document/product/647/34610)를 참고하십시오.
- 기존 패키지 관리가 필요한 경우 [패키지 관리](https://console.cloud.tencent.com/trtc/package)를 클릭하여 관련 작업 페이지로 이동할 수 있습니다. 자세한 내용은 [패키지 관리](https://intl.cloud.tencent.com/zh/document/product/647/39075)를 참고하십시오.
