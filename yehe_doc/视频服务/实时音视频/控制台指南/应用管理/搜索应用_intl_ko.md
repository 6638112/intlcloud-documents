TRTC 콘솔에서는 사용자가 필요한 애플리케이션을 빠르게 검색할 수 있도록 애플리케이션 검색 기능을 제공하고 있으며, 애플리케이션 이름 또는 SDKAppID, 태그에 따라 애플리케이션을 필터할 수 있습니다.

## 주의 사항
 - 검색창은 동일한 리소스 속성에 다수의 검색 정보 입력을 지원하여 엔터키를 누르면 검색 정보를 분할합니다.
 - 검색창은 동일하지 않은 리소스 속성에 대해서는 여러 검색 정보 입력을 지원하지 습니다. 고객이 동일하지 않은 리소스 속성에 여러 개의 검색 정보를 입력하면 시스템은 기본적으로 가장 마지막에 입력한 리소스 속성 정보를 인덱스합니다.


## 작업 순서
1. [Application Management](https://console.cloud.tencent.com/trtc/app)에서 애플리케이션 리스트 상단의 검색창을 클릭합니다.
2. 리소스 유형을 선택하고 실제 상황에 따라 검색 정보를 입력한 후 검색 버튼![](https://main.qcloudimg.com/raw/8f708a40d39225bf23332c4a6eebbc06.png)을 클릭하면 검색이 진행됩니다.
> ? 
>- 직접 검색 정보를 입력하는 경우 기본적으로 “애플리케이션 이름”에 해당하는 애플리케이션을 검색합니다.
> - 검색 정보 1개를 수정해야 하는 경우 해당 검색 정보를 클릭한 뒤 내용을 수정하고 엔터키를 누르면 수정이 완료됩니다.
> - 수정 정보를 삭제해야 하는 경우 검색 정보 오른쪽에 있는 삭제 버튼![](https://main.qcloudimg.com/raw/e01d054360c7920c55edb5c4995a7c1a.png)을 클릭하면 삭제됩니다.

- 클라우드에서 혼합 스트리밍 트랜스 코딩 시 사용자 정의 배경 이미지 설정이 필요한 경우 리소스 관리에서 해당하는 이미지 리소스를 추가할 수 있으며, 자세한 방법은 [리소스 관리](https://intl.cloud.tencent.com/zh/document/product/647/39081)를 참조하십시오.
- 애플리케이션 빠른 실행 관련 Demo 소스 코드에 대한 자세한 내용은 [퀵 스타트](https://intl.cloud.tencent.com/zh/document/product/647/39082)를 참조하십시오.
