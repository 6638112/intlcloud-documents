애플리케이션 생성 후 [퀵 스타트]의 가이드를 통해 TRTC에서 제공하는 Demo를 빠르게 통합할 수 있습니다.

## 작업 순서
1. TRTC 콘솔의 [애플리케이션 관리](https://console.cloud.tencent.com/trtc/app)에서 관리할 애플리케이션을 선택합니다.
2. 애플리케이션 작업 열의 [애플리케이션 정보]를 클릭하여 애플리케이션 상세 페이지에 들어간 후 [퀵 스타트] 탭을 선택합니다.
3. 실제 비즈니스 시나리오에 따라 SDK 및 관련 Demo 소스 코드 다운로드를 클릭합니다.
4. [Copy Secret Key]를 클릭하여 애플리케이션 UserSig 키를 복사합니다.
5. 다운로드한 소스 패키지의 압축을 해제하고 지정된 파일을 찾아 연 다음 SDKAppID와 키를 지정 위치에 붙여넣습니다.


## 관련 문서
- 신규 애플리케이션 생성에 대한 자세한 방법은 [애플리케이션 생성](https://intl.cloud.tencent.com/document/product/647/39077)을 참조하십시오.
- 애플리케이션 리스트에서 관련 애플리케이션을 검색하는 자세한 방법은 [애플리케이션 검색](https://intl.cloud.tencent.com/document/product/647/39078)을 참조하십시오.
- 애플리케이션 기본 정보 조회에 대한 자세한 방법은 [애플리케이션 정보](https://intl.cloud.tencent.com/document/product/647/39079)를 참조하십시오.
- 애플리케이션 기능 설정 또는 설정 정보 확인에 대한 자세한 방법은 [기능 설정](https://intl.cloud.tencent.com/document/product/647/39080)을 참조하십시오.
- 클라우드 혼합 스트림 트랜스 코딩 시 사용자 정의 배경 이미지 설정이 필요한 경우 리소스 관리에서 해당하는 이미지 리소스를 추가할 수 있습니다.
