클라우드에서 혼합 스트림 트랜스 코딩 시 사용자 정의 배경 이미지 설정이 필요한 경우, 리소스 관리에서 상응하는 이미지 리소스를 추가할 수 있습니다.

<span id="add"></span>
## 이미지 리소스 추가
1. TRTC 콘솔의 [애플리케이션 관리](https://console.cloud.tencent.com/trtc/app)에서 관리할 애플리케이션을 선택합니다.
2. 애플리케이션 작업 열의 [애플리케이션 정보]를 클릭하여 애플리케이션 상세 페이지에 들어간 후 [리소스 관리] 탭을 선택합니다.
3. [이미지 추가]를 클릭하여 이미지 리소스를 추가하고 다음과 같이 설정합니다.
<table id="inform">
<tr><th width="17%">설정 항목</th><th  width="31%">설정 방법</th><th>설명</th></tr>
<tr>
<td>이미지 이름</td>
<td>이미지 이름 입력</td>
<td>중국어, 영어, 숫자, _, -만 지원하며 30자까지 입력 가능</td>
</tr><tr>
<td>로컬 이미지 업로드</td>
<td>[이미지 선택] 클릭 후 로컬 이미지를 선택하여 업로드</td>
<td>최적의 그래픽 효과를 위해 png 포맷의 이미지를 업로드해야 하며, 이미지 크기는 2M 미만</td>
</tr><tr>
<td>이미지 크기 및 좌표 설정</td>
<td><ul style="margin:0"><li>리소스 이미지를 드래그하여 위치 조정</li><li>리소스 이미지 확대</li><li>리소스 이미지 축소</li></td>
<td>드래그하여 이미지 위치를 조정하면 '표시 위치 X축 방향'과 '표시 위치 Y축 방향' 매개변수도 그에 따라 변경</td>

<td>리소스 이미지 X축 방향 위치 조정</td>
<td>1~100까지의 숫자만 입력할 수 있으며, 입력 후 미리보기 창의 내용이 해당 값으로 변경</td>
</tr><tr>
<td>표시 위치 Y축 방향</td>

<td>1~100까지의 숫자만 입력할 수 있으며, 입력 후 미리보기 창의 내용이 해당 값으로 변경</td>
</tr></table>
4. [저장]을 클릭하면 이미지 리소스가 생성됩니다.

<span id="preview"></span>
## 이미지 효과 미리보기
1. [이미지 리소스 추가](#add) 완료 후, 리스트에서 확인할 이미지 리소스를 선택하고 오른쪽의 [미리보기]를 클릭합니다.
2. 미리보기 팝업창에서 이미지가 삽입된 재생 페이지를 미리 볼 수 있습니다.



<span id="change"></span>
## 이미지 리소스 수정
1. [애플리케이션 관리](https://console.cloud.tencent.com/trtc/app)에서 관리할 애플리케이션을 선택합니다.
2. 애플리케이션 작업 열의 [애플리케이션 보기]를 클릭하여 애플리케이션 상세 페이지에 들어간 후 [리소스 관리] 탭을 선택합니다.
3. 왼쪽 리스트에서 수정할 이미지 리소스를 선택한 후, 리스트 열 또는 '이미지 정보' 창의 [편집]을 클릭합니다.
3. 이미지 수정 페이지에서 [이미지 정보](#inform)를 수정하고 [저장]을 클릭합니다.


<span id="delete"></span>
## 이미지 리소스 삭제
1. [애플리케이션 관리](https://console.cloud.tencent.com/trtc/app)에서 관리할 애플리케이션을 선택합니다.
2. 애플리케이션 작업 열의 [애플리케이션 보기]를 클릭하여 애플리케이션 상세 페이지에 들어간 후 [리소스 관리] 탭을 선택합니다.
3. 왼쪽 리스트에서 삭제할 이미지 리소스를 선택한 후, 리스트 열 또는 '이미지 정보' 창의 [삭제]를 클릭합니다.
4. 이미지 삭제 팝업창에서 해당 이미지 리소스의 삭제 여부를 확인합니다.
5. [확인]을 클릭하면 삭제가 완료됩니다.
