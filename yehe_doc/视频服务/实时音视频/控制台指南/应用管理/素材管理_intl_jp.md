Cloud MixTranscoding時にカスタマイズしたした背景画像を設定したい場合、素材管理の中で対応する画像素材を追加できます。

[](id:add)
## 画像素材の追加
1. TRTCコンソールに入り、【[アプリケーション管理](https://console.cloud.tencent.com/trtc/app)】の中で管理が必要なアプリケーションを選択します。
2. アプリケーション操作バーの【アプリケーションの確認】をクリックしてアプリケーション詳細画面に入り、【素材管理】タブをクリックします。
3. 【+】をクリックして画像素材を新規追加し、次のように設定します。
![](https://main.qcloudimg.com/raw/9534d8f779fef707a97de1384af71a41.png)
<table id="inform">
<tr><th width="17%">設定項目</th><th  width="31%">設定方法</th><th>説明</th></tr>
<tr>
<td>画像名称</td>
<td>画像名称の入力</td>
<td>中国語、英語、数字、「_」、「-」のみサポートしています。30文字以内</td>
</tr><tr>
<td>ローカル画像アップロード</td>
<td>【选择画像の選択】をクリックし、选择ローカル画像を選択してアップロードします</td>
<td>視覚効果を最適化するため、请上传 png 形式の画像をアップロードしてください。画像の大きさは2M未満</td>
</tr><tr>
<td>画像の大きさと座標の設定</td>
<td><ul style="margin:0"><li>拖动素材画像をドラッグして位置の調整</li><li>素材画像の大きさの拡大</li><li>素材画像の大きさの縮小</li>を行います。td>
<td>ドラッグして画像の位置を調整した後、“表示位置X軸方向”と“表示位置Y軸方向”の2つのパラメータもそれに応じて変化します</td>
</tr><tr>
<td>表示位置 X 軸方向</td>
<td>素材画像 X 軸方向の位置の調整</td>
<td>1 - 100 の数字のみ記入できます。記入が完了するとプレビューボックスも対応して変化します</td>
</tr><tr>
<td>表示位置 Y 軸方向</td>
<td>素材画像 X 軸方向の位置の調整</td>
<td>1 - 100の数字のみ記入できます。記入が完了するとプレビューボックスも対応して変化します</td>
</tr></table>
4. 【保存】をクリックすれば、画像素材の作成が完了します。

## 画像効果のプレビュー
1. [画像素材の追加](#add)の完了後、リストの中から確認したい画像素材を選択し、下側の【プレビュー】をクリックすることができます。
![](https://main.qcloudimg.com/raw/32782b288a5aae052e146248283a1bbe.png)
2. プレビューのポップアップ画面で、画像をブロードキャスト画面に埋め込んだ後のプレビュー効果を確認できます。
![](https://main.qcloudimg.com/raw/108ef481cf4d876e56effc83d8e552fd.png)



[](id:change)
## 画像素材の修正
1. 【[アプリケーション管理](https://console.cloud.tencent.com/trtc/app)】の中から管理が必要なアプリケーションを選択します。
2. アプリケーション操作バーの【アプリケーションの確認】をクリックして、アプリケーション詳細画面に入り、【素材管理】タブを選択します。
2. 左側のリストの中から修正が必要な画像素材を選択し、【編集】をクリックします。
![](https://main.qcloudimg.com/raw/02571d080432999261b3c643efc7aeb0.png)
3. 画像修修正画面に入り、[画像情報](#inform) に対して修正を行い、【保存】をクリックすれば完了です。


[](id:delete)
## 画像素材の削除
1. 【[アプリケーション管理](https://console.cloud.tencent.com/trtc/app)】の中から管理が必要なアプリケーションを選択します。
2. アプリケーション操作バーの【アプリケーションの確認】をクリックして、アプリケーション詳細画面に入り、【素材管理】タブを選択します。
3. 左側のリストから削除が必要な画像素材を選択し、削除ボタン!をクリックします[](https://main.qcloudimg.com/raw/12a12e329a1828de1c9a104b0accd539.png)。
4. 画像削除のポップアップウィンドウの中で、当該画像素材が削除するかどうかを確認します。
![](https://main.qcloudimg.com/raw/d4d2100c34a8720de9ad5fc85afbde2b.png)
5. 【確定】をクリックすれば、削除が完了します。
