Cloud MixTranscoding時にカスタマイズした背景画像を設定したい場合、素材管理で対応する画像素材を追加できます。

<span id="add"></span> 
## 画像素材の追加
1. TRTCコンソールに入り、【[アプリケーション管理](https://console.cloud.tencent.com/trtc/app)】で管理したいアプリケーションを選択します。
2. アプリケーション操作バーの【アプリケーション情報】をクリックして、アプリケーション詳細画面に入り、【素材管理】のタブを選択します。
3. 【+】をクリックして画像素材を新規追加し、次のように設定します。
<table id="inform">
<tr><th width="17%">設定項目</th><th  width="31%">設定方法</th><th>説明</th></tr>
<tr>
<td>画像名</td>
<td>画像名の入力</td>
<td>中国語、英語、数字、_（アンダーバー）、-（ハイフン）のみサポートしています。30文字を超えないこと。</td>
</tr><tr>
<td>ローカル画像のアップロード</td>
<td>【画像の選択】をクリックし、ローカル画像を選択してアップロードします。</td>
<td>最高の視覚効果を得るには、画像をpng形式でアップロードしてください。画像のサイズは2M未満です。</td>
</tr><tr>
<td>画像のサイズと座標の設定</td>
<td><ul style="margin:0"><li>ドラッグして素材画像の位置を調整</li><li>素材画像サイズを拡大</li><li>素材画像サイズを縮小</li></td>
<td>ドラッグして画像の位置を調整すると、「表示位置X軸方向」と「表示位置Y軸方向」の2つのパラメータもこれに応じて変化します</td>
</tr><tr>
<td>表示位置 X 軸方向</td>
<td>素材画像 X 軸方向の位置の調整</td>
<td>1～100の数字のみ入力できます。入力が完了するとプレビューボックスも対応して変化します</td>
</tr><tr>
<td>表示位置 Y 軸方向</td>
<td>素材画像 X 軸方向の位置の調整</td>
<td>1～100の数字のみ入力できます。入力が完了するとプレビューボックスも対応して変化します</td>
</tr></table>
4. 【保存】をクリックすると、画像素材が完成します。

<span id="preview"></span>

## 画像効果のプレビュー
1. [画像素材の追加](#add)の完了後、リストの中から確認したい画像素材を選択し、下側の【プレビュー】をクリックします。
2. プレビューのポップアップボックスで、再生画面に埋め込まれた画像のプレビュー効果を確認できます。



<span id="change"></span>

## 画像素材の変更
1. 【[アプリケーション管理](https://console.cloud.tencent.com/trtc/app)】で管理したいアプリケーションを選択します。
2. アプリケーション操作バーの【アプリケーションの確認】をクリックして、アプリケーション詳細画面に入り、【素材管理】のタブを選択します。
3. 左側のリストの中から変更したい画像素材を選択し、【編集】をクリックします。
3. 画像変更画面に入って、[画像情報](#inform) を変更し、【保存】をクリックします。

<span id="delete"></span>

## 画像素材の削除
1. 【[アプリケーション管理](https://console.cloud.tencent.com/trtc/app)】で管理したいアプリケーションを選択します。
2. アプリケーション操作バーの【アプリケーションの確認】をクリックして、アプリケーション詳細画面に入り、【素材管理】のタブを選択します。
3. 左側のリストから削除したい画像素材を選択し、削除ボタン![](https://main.qcloudimg.com/raw/12a12e329a1828de1c9a104b0accd539.png)をクリックします。
4. 画像削除のポップアップボックスの中で、当該画像素材が削除されるかどうか確認します。
5. 【OK】をクリックして、削除します。
