TRTC는 다양한 개발 환경별 Demo를 제공하여 귀하의 TRTC Demo를 빠르게 제작할 수 있습니다.

## 전제 조건
TRTC 서비스를 활성화한 상태에서 [TRTC 콘솔](https://console.cloud.tencent.com/trtc)로 이동합니다.

## 작업 순서
<span id="step1"></span>

### 1단계: 애플리케이션 생성

1. TRTC 콘솔에 로그인하여 [Development Assistance]>[Demo 빠른 실행](https://console.cloud.tencent.com/trtc/quickstart)을 선택합니다.
2. [시작하기]를 클릭하고 애플리케이션 이름(예: `TestTRTC`)을 입력한 후 [Create Application]을 클릭합니다.
애플리케이션 이름은 숫자, 중문, 영어와 언더바만 포함할 수 있으며 16자를 초과할 수 없습니다.

<span id="step2"></span>

### 2단계: 소스 코드 다운로드

1. 커서를 해당하는 카드로 이동한 후 실제 비즈니스 수요에 따라 SDK 및 관련 Demo 소스 코드를 다운로드합니다.
2. 다운로드 후 [다운로드 완료, 다음 단계]를 클릭합니다.

<span id="step3"></span>

### 3단계: 설정 수정

1. 설정 수정 페이지로 이동하여 다운로드한 소스 패키지에 따라 해당하는 개발 환경을 선택합니다.
2. 해당 파일을 찾아 연 후, 생성된 SDKAppID와 키를 지정 위치에 붙여넣습니다.
3. 붙여넣은 후 [붙여넣기 완료, 다음 단계]를 클릭하면 Demo [컴파일 실행](#step4)이 시작됩니다.

>!키는 중요 정보입니다. 유출되지 않도록 주의하십시오.

<span id="step4"></span>

### 4단계: 컴파일 실행

통합할 개발 환경에 따라 해당하는 컴파일 방법을 선택합니다. 자세한 내용은 다음을 참조하십시오.
- [Demo 제작(iOS&Mac)](https://intl.cloud.tencent.com/document/product/647/35086)
- [Demo 제작(Android)](https://intl.cloud.tencent.com/document/product/647/35084)
- [Demo 제작(Windows)](https://intl.cloud.tencent.com/document/product/647/35085)
- [Demo 제작(데스크톱 브라우저)](https://intl.cloud.tencent.com/document/product/647/35607)
- [Demo 제작(Electron)](https://intl.cloud.tencent.com/document/product/647/35089)

컴파일 완료 후 [가이드 닫기, 콘솔 관리 애플리케이션으로 열기]를 클릭합니다.
