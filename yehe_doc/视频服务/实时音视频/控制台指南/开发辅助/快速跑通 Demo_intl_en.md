TRTC offers demos for different platforms to help you get started quickly.

## Prerequisites
You have activated TRTC and opened the [TRTC console](https://console.cloud.tencent.com/trtc).

## Directions
### Step 1. Create an application<span id="step1"></span>
1. Log in to the TRTC console and select **Development Assistance** > **[Demo Quick Run](https://console.cloud.tencent.com/trtc/quickstart)**.
2. Enter an application name such as `TestTRTC` and click **Create**.
![](https://main.qcloudimg.com/raw/df3897d3c0fc42e952b170dda9688c07.png)
>? An application name can contain up to 16 characters. Only digits, letters, Chinese characters, and underscores are allowed.


### Step 2. Download the source code<span id="step2"></span>
1. Download the SDK and demo source code that suit your needs.
2. After the download, click **Next**.
![](https://main.qcloudimg.com/raw/886a78a68fc48dc88ab0b5d94ce55584.png)

### Step 3. Modify configuration<span id="step3"></span>
1. In the **Modify Configuration** step, select the development platform in line with the source package downloaded.
2. Find and open the specified file, and paste the SDKAppID and key generated to the specified locations in the file.
3. Click **Next** and proceed to [Compile and run](#step4) the demo.

![](https://main.qcloudimg.com/raw/37aec16300737efe6528e1e4236a1249.png)
>! Keep the key confidential and do not disclose it to others.

### Step 4. Compile and run<span id="step4"></span>
Use the method that suits your development platform to compile the project. For details, see below:
- [Demo Quick Start (iOS & macOS)](https://intl.cloud.tencent.com/document/product/647/35086)
- [Demo Quick Start (Android)](https://intl.cloud.tencent.com/document/product/647/35084)
- [Demo Quick Start (Windows)](https://intl.cloud.tencent.com/document/product/647/35085)
- [Demo Quick Start (Desktop Browser)](https://intl.cloud.tencent.com/document/product/647/35607)
- [Demo Quick Start (Electron)](https://intl.cloud.tencent.com/document/product/647/35089)

After the compilation, click **Return to Overview Page**.
