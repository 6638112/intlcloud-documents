Tencent Cloud TRTC는 개발자를 위해 온라인 통화 품질 상세 정보 기록 모니터링 대시보드를 제공하며, 이를 통해 통화 상세 정보 및 정보를 확인해 단말 사용자의 통화 상태를 확인할 수 있습니다.

## 통화 기록 리스트

 TRTC 콘솔에 로그인하여 [모니터링 대시보드](https://console.cloud.tencent.com/trtc/monitor)에서 현재 계정의 모든 방 통화 기록을 확인할 수 있습니다. 모니터링 대시보드의 통화 리스트에는 기본적으로 가장 마지막에 생성한 애플리케이션의 당일 방 통화 기록 정보가 표시되며, 통화 시간에 따라 역순으로 배열되어 있습니다.

통화 리스트에 표시되는 데이터 정보는 다음과 같습니다.

| 데이터 항목             | 설명                                                         |
| ------------------ | ------------------------------------------------------------ |
| 방 ID            | 애플리케이션 통화에 사용하는 roomid                                    |
| 시작시간 및 종료시간 | 통화 방의 시작 시간과 종료 시간                                         |
| 방 지속 시간       | 첫 번째 사용자 입장 시간부터 마지막 사용자 퇴장 시간까지의 시간으로, 통화가 아직 종료되지 않은 경우 첫 번째 사용자 입장한 시간부터 조회 시간까지의 지속시간이 표시됩니다. |
| 참여 사용자 수         | 방에 입장한 누적 사용자 수                                     |
| 작업               | [통화 상세 정보 보기](https://intl.cloud.tencent.com/zh/document/product/647/39070)를 클릭하여 통화 상세 정보 페이지에 들어가 상세 그래프를 확인할 수 있습니다.  |

## 통화 정보 검색

모니터링 대시보드에서는 다양한 검색 방법을 지원하며, 방 리스트를 데이터 필터링할 수 있습니다.

- 특정 애플리케이션의 통화 방 정보를 조회하고 싶은 경우, [애플리케이션 선택]을 클릭하여 조회할 애플리케이션을 선택한 후 [검색]을 클릭합니다.
- 특정 시간대의 통화 방 정보를 조회할 수 있으며 오늘, 어제, 최근 6시간, 최근 14일 또는 사용자 정의 시간대를 설정하고 [검색]을 클릭하여 데이터를 조회할 수 있습니다. 
- 특정 통화 방 정보를 조회하고 싶은 경우 해당 시간대를 선택하고 방 ID(roomid) 또는 사용자 ID(userid)를 입력한 후 [검색]을 클릭하여 조회할 수 있습니다.

## 추가 작업

- 특정 방 입장 후 자세한 통화 데이터를 조회하고 싶은 경우 [통화 상세 정보](https://intl.cloud.tencent.com/zh/document/product/647/39070)를 참조하십시오.
