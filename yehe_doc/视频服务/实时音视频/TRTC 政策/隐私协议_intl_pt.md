## 1\. INTRODUÇÃO

Esse módulo se aplica se você usa o recurso Tencent Real-Time Communication  (“**Recurso**”). Este Módulo está incorporado à política de privacidade localizada em (“[Política de Privacidade](https://intl.cloud.tencent.com/document/product/301/17345)”). Os termos usados, mas não definidos nesse Módulo, devem ter o significado dado a eles na Política de Privacidade. No caso de qualquer conflito entre a Política de Privacidade e esse Módulo, esse Módulo será aplicado conforme a inconsistência.

## 2\. CONTROLADORIA

O controlador das informações pessoais descritas nesse Módulo é o especificado na Política de Privacidade.

## 3\. DISPONIBILIDADE

Esse recurso está disponível para os usuários em todo o mundo.

## 4\. COMO USAMOS AS INFORMAÇÕES PESSOAIS

Usaremos as informações das seguintes formas e de acordo com as seguintes bases jurídicas:

| **Informações Pessoais**                                     | **Uso**                                                      | **Base Jurídica**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de configuração do console:** ID do aplicativo TRTC, nome do aplicativo, se a gravação está aberta, se a função de bypass está ativada ou não, método de cobrança selecionado | Usamos essas informações para determinar o uso do Recurso, para fins de faturamento. <br/>Observe que esses dados são armazenados em nosso recurso Elasticsearch Service (ES). | Processamos essas informações porque são necessárias para a execução de nosso contrato com você para o fornecimento do Recurso. |
| **Dados de registro de backend:** ID do usuário referente a qualquer participante do evento de transmissão ao vivo, número da sala, IP do cliente relativo a qualquer participante do evento de transmissão ao vivo, versão SDK do cliente, tipo de SO relacionado a qualquer participante do evento de transmissão ao vivo | Usamos essas informações para garantir que o Recurso funcione conforme necessário, inclusive para resolução de problemas.<br/> Observe que esses dados estão armazenados em nosso recurso ES. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você para o fornecimento do Recurso. |
| **Informações do painel:** informações sobre a qualidade de áudio e vídeo durante as chamadas: ID da aplicação do usuário final, dados relativos às funções controladas pelo usuário final (habilitar vídeo, desativar vídeo, ativar áudio, desativar áudio), entrar na sala, deixar sala, ID da sala, função mudo, uso de CPU, uso de memória, latência da rede, perda de pacotes, resolução, bitrate, taxa de quadros,  volume | Usamos essas informações para garantir que o Recurso funcione conforme necessário, inclusive para resolução de problemas. <br/>Observe que esses dados estão armazenados em nosso recurso ES. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você para o fornecimento do Recurso. |
| **Dados de registro SDK (do usuário final):** ID do usuário, número da sala, número da versão SDK do cliente, tipo de SO da sala TRTC | Usamos essas informações para garantir que o Recurso funcione conforme necessário, inclusive para resolução de problemas. <br/>Observe que esses dados estão armazenados em nosso recurso ES. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você para o fornecimento do Recurso. |
| **UIN**                                                      | Usamos essas informações para identificar o uso do Recurso.<br/>Observe que esses dados estão armazenados em nosso recurso ES. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você para o fornecimento do Recurso. |
| **ID da aplicação do SDK** (criado com seu UIN para diferentes aplicativos) | Usamos essas informações para identificar o uso dos seus  aplicativos como parte do Recurso.<br/>Observe que esses dados estão armazenados em nosso recurso ES. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você para o fornecimento do Recurso. |
| **Dados para resolução de problemas:** ID da aplicação do usuário final, dados relativos às funções controladas pelo usuário final (habilitar vídeo, desativar vídeo, ativar áudio, desativar áudio), entrar na sala, deixar sala, ID da sala, função mudo,  uso de CPU, uso de memória, latência da rede, perda de pacotes, resolução, bitrate, taxa de quadros, volume | Usamos essas informações para detectar e localizar problemas vivenciados pelos usuários finais, inclusive para resolução de problemas. <br/>Observe que esses dados estão armazenados em nosso recurso ES. | Processamos essas informações porque são necessárias para a execução de nosso contrato com você para o fornecimento do Recurso. |

## 5\. COMO ARMAZENAMOS E COMPARTILHAMOS INFORMAÇÕES PESSOAIS

Conforme especificado na Política de Privacidade.

## 6\. RETENÇÃO DE DADOS

Reteremos informações pessoais de acordo com o seguinte:

| **Informações Pessoais**                                     | **Política de Retenção**                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| Dados de configuração                                   | Retemos esses dados enquanto você usar o Recurso. Quando a sua utilização do Recurso for encerrada ou quando sua conta for excluída, excluiremos esses dados dentro de 7 dias. |
| Registros de backend  | Armazenados por 14 dias.                                          |
| Informações do painel | Armazenados por 14 dias.                                          |
| Dados do SDK | Armazenados por 14 dias.                                          |
| UIN | Armazenados por 14 dias.                                          |
| ID da aplicação do SDK | Armazenados por 14 dias.                                          |
| Dados de resolução de problemas | Armazenados por 14 dias.                                          |
