## 1\. 背景

本モジュールは、お客様がTencent Real-Time Communication機能（以下「**本機能**」）を使用する場合に適用されます。本モジュールは、[Data Processing and Security Agreement](https://intl.cloud.tencent.com/document/product/301/17347)（データ処理およびセキュリティに関する契約）（以下「DPSA」）に組み込まれます。本モジュールにおける用語のうち本書内に定義のないものは、DPSA内で定義された意味を有するものとします。DPSAと本モジュールとの間に矛盾がある場合には、その不一致の範囲で本モジュールを優先するものとします。

## 2\. 処理

当社は、本機能に関連して以下のデータを処理します。

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **エンドユーザーのオーディオおよびビデオコンテンツ**                     | 左記情報は、お客様に本機能を提供するという目的（すなわち、オーディオビジュアルコンテンツの提供）でのみ処理されます。<br/> （この目的において左記情報を保存しません） |
| **顧客の一時キー情報：** SDK APPID、ユーザー名、秘密キー | 左記情報は、お客様による本機能の利用状況の特定に使用されます。 |
| **アプリケーションに関連する顧客ログデータ：** SDK APPID、アプリケーション名、タグ、サービス状況、作成時刻、操作 | 左記情報は、お客様が本機能を使用して作成したアプリケーションに関連する記録を維持するために使用されます。 |

## 5\. データ保持

当社は、本機能に関連して処理された個人データを以下のとおり保管します。

| **個人情報**                                     | **保持ポリシー**                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **顧客の一時キー情報：** SDK APPID、ユーザー名、秘密キー | 当社は、お客様が本機能を使用している限り、当該データを保持します。お客様が本機能の使用を終了するか、お客様のアカウントが削除されると、当社はこのデータを7日以内に削除します。 |
| **アプリケーションに関連する顧客ログデータ：** SDK APPID、アプリケーション名、タグ、サービス状況、作成時刻、操作 | 当社は、お客様が本機能を使用している限り、当該データを保持します。お客様が本機能の使用を終了するか、お客様のアカウントが削除されると、当社はこのデータを7日以内に削除します。 |

お客様は、DPSAに基づき、当該個人データの削除を依頼することができます。

## 6\. 特記条項

お客様は、本機能を、個人データの処理に同意できる最低年齢に達しているエンドユーザー、または最低年齢に達していない個人の場合、親の同意が得られているエンドユーザーのみが使用することを保証する必要があります。その年齢は、エンドユーザーの管轄法域によって異なることがあります。

お客様は、本機能に関するエンドユーザーの個人データの処理に関して、適用法に従い、かつ当社が適用法を遵守できるように、エンドユーザーから必要な全ての同意を取得し、維持することを表明し、保証し、引き受けるものとします。
