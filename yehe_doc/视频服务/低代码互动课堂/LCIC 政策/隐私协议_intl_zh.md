## 1\. 简介

本模块仅在您使用 Tencent Low-Code Interactive Classroom（简称“**功能**”）时适用。本模块已纳入隐私政策（简称“[隐私政策](https://intl.cloud.tencent.com/document/product/301/17345)”）中。本模块中使用但未作定义的术语应具有在隐私政策中规定的含义。如果隐私政策与本模块之间发生任何冲突，对于此等不一致，本模块适用。

## 2\. 控制方

本模块中所述个人信息的控制方见隐私政策中的规定。

## 3\. 可用性

本功能可供全球用户使用。

## 4\. 我们如何使用个人信息

我们将按照以下法律依据和方式使用信息：

| **个人信息**                                     | **用途**                                                      | **法律依据**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **日志数据，包括：** <br/>1\. SDK 日志，即相关房间号和客户端 SDK 版本号以及房间操作系统类型。<br/>2\. 与您选择购买的有关应用程序相关的 SDK APP ID。 | 我们出于向您提供本功能的目的，包括用于调试和识别的目的使用这些信息。<br/>请注意，这些数据暂时存储并备份在我们的日志服务 (CLS) 功能中。 | 我们处理这些信息是为了履行我们与您签订的合同，向您提供本功能。 |
| **配置数据，** 包括购买的 APP ID、APP 名称。 | 我们出于向您提供本功能的目的，包括用于调试和登录验证的目的使用这些信息。 <br/>请注意，这些数据将存储和备份在我们的云数据库 MySQL (MySQL) 功能中。 | 我们处理这些信息是为了履行我们与您签订的合同，向您提供本功能。 |

## 5\. 我们如何存储和共享个人信息

见隐私政策中的规定。 

## 6\. 数据保留

我们将根据下列规定保留个人信息：

| **个人信息**   | **保留政策**                                         |
| -------------------------- | ------------------------------------------------------------ |
| 日志数据配置数据 | 我们将这些数据保留 180 天，之后将自动删除它们。 |
