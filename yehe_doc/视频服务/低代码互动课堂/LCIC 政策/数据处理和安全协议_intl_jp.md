## 1\. 背景

本モジュールは、お客様がTencent Low-Code Interactive Classroom（「**本機能**」）を使用する場合に適用されます。本モジュールは、DPSAに掲載されているData Processing and Security Agreement（データ処理およびセキュリティに関する契約）（以下「[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)」）に組み込まれます。本モジュールにおける用語のうち本書内に定義のないものは、DPSA内で定義された意味を有するものとします。DPSAと本モジュールとの間に矛盾がある場合には、その不一致の範囲で本モジュールを優先するものとします。

## 2\. 処理

当社は、本機能に関連して以下のデータを処理します。

| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **以下のログデータ：**<br/>1\. バックエンドログ：ユーザーID、ユーザーのIP、本機能のライブストリーミング機能に参加しているユーザーに関連付けられたOSタイプ、 ルーム番号、ユーザーのSDKバージョン。<br/>2\. SDK ログ：ユーザー ID。 | 当社は、お客様に本機能を提供する目的でのみ、このデータを処理します。<br/>このデータは、当社のCloud Log Service （CLS）機能に一時的に保存およびバックアップされます。 |
| **以下の品質監視データ：**<br/>ユーザーのアプリID、ビデオおよびオーディオ機能情報（ユーザーの選択により有効化/無効化）、入退室に関連する情報、ルームID、ミュート機能の使用、CPU使用率、メモリ使用率、ネットワーク遅延、パケットロス、解像度、ビットレート、フレームレート、ボリューム。 | 当社は、お客様に本機能を提供する目的、ならびに検証、デバッグ、およびサービスの最適化の目的でのみ、このデータを処理します。<br/>このデータは当社のTencentDB for MySQL（MySQL）機能に保存およびバックアップされますので、その旨ご留意ください。 |
| **ビジネスデータ：**<br/>受講中にお客様と当社のユーザーが記録したオーディオファイルおよびビデオファイルを含みます。 | このデータは、お客様に本機能を提供し、クラウド録画（クラスルームセッションの記録を希望する場合）および本サービスが提供するストレージ機能を有効にする目的でのみ処理されます。<br/>このデータは、当社のCloud Object Storage（COS）機能に保存およびバックアップされますので、ご注意ください。 |

## 5\. データ保持

当社は、本機能に関連して処理された個人データを以下のとおり保管します。

| **個人情報**         | **保持ポリシー**                                         |
| -------------------------------- | ------------------------------------------------------------ |
| ログデータ品質監視データ | 当社は当該データを180日間保持します。その期間が経過すると自動的に削除されます。 |
| ビジネスデータ                    | 当社は、お客様がクラウドアカウント機能を使用してデータを削除するように指示するまで、または本機能の使用が終了するまで、当該データを保持します。当社は、お客様の有効な要求の直後、またはお客様が本機能の使用を終了してから7日以内にお客様のデータを削除します。 |

お客様は、DPSAに従って、当該個人データの削除を要請することができます。

## 6\. 特別な条件

お客様は、本機能を、個人が自己の個人データの処理に同意できる最低年齢に達しているエンドユーザーのみが使用できるようにしなければなりません。これは、エンドユーザーが所在する司法管轄区によって異なる場合があります。

お客様は、本機能に関するエンドユーザーの個人データの処理に関して、適用法令に従い、かつ当社が適用法令を遵守できるように、エンドユーザーから必要な全ての同意を取得し、維持することを表明し、保証し、約束するものとします。

本機能は、機密データの処理を目的としたものではありません。お客様は、本機能がお客様またはお客様のエンドユーザーによる機密データの転送またはその他の処理に使用されないことを保証しなければなりません。
