## 1\. CONSIDERAÇÕES INICIAIS

Este Módulo se aplica se você usar o Tencent Low-Code Interactive Classroom (“**Recurso**”). Este Módulo está incorporado ao Contrato de Segurança e Processamento de Dados localizado em (“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)”). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.

## 2\. PROCESSAMENTO

Processaremos os seguintes dados em conexão com o Recurso:

| **Informações pessoais**                                     | **Uso**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Dados de registro,** incluindo:<br/>1\. Registros de back-end: identificação do usuário, IP dos usuários, tipo de sistema operacional associado a todos os usuários que participam da funcionalidade de transmissão ao vivo do Recurso, números de sala, versões do SDK dos usuários; e<br/>2\. Registros do SDK: identificação de usuário. | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você.<br/>Observe que esses dados estão armazenados temporariamente e tem uma cópia de segurança em nosso recurso Cloud Log Service (CLS). |
| **Dados de monitoramento de qualidade,** incluindo:<br/>identificação do aplicativo dos usuários, informações de funcionalidade de vídeo e áudio (habilitadas ou desabilitadas, conforme selecionado pelos usuários), informações relacionadas aos usuários que entram em uma sala e saem de uma sala, identificação da sala, uso da função mudo, uso da CPU, uso da memória, latência da rede, perda de pacotes, resolução, taxa de bits, taxa de quadros e o volume. | Nós apenas processamos esses dados para fins de fornecimento do Recurso a você e para fins de verificação, depuração e otimização de serviços.<br/>Observe que esses dados estão armazenados com e tem uma cópia de segurança no nosso recurso TencentDB for MySQL (MySQL). |
| **Dados empresariais,** incluindo os arquivos de áudio e vídeo gravados por você e nossos usuários durante as aulas. | Nós apenas processamos esses dados para fins de fornecer o Recurso a você, para habilitar a gravação em nuvem (caso você deseje gravar a sessão em sala de aula) e as funções de armazenamento que o Serviço oferece.<br/>Observe que esses dados são armazenados e tem uma cópia de segurança em nosso recurso Cloud Object Storage (COS) |

## 5\. RETENÇÃO DE DADOS

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

| **Informações pessoais**         | **Política de retenção**                                         |
| -------------------------------- | ------------------------------------------------------------ |
| Dados de monitoramento da qualidade dos dados de registro | Conservamos tais dados por 180 dias, após os quais serão automaticamente apagados. |
| Dados comerciais                    | Retemos esses dados até que você nos diga para excluí-los usando a funcionalidade da sua conta de nuvem ou até que o uso do Recurso cesse. Eliminaremos os seus dados imediatamente após o seu pedido válido ou no prazo de sete dias após a cessação da sua utilização do Recurso. |

Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA.

## 6\. CONDIÇÕES ESPECIAIS

Você deve garantir que esse Recurso seja usado somente por usuários finais com a idade mínima exigida para que possam consentir com o processamento de seus dados pessoais. Pode haver diferenças dependendo da jurisdição em que o usuário final está localizado.

Você declara, garante e assume que deve obter e manter todos os consentimentos necessários dos usuários finais em relação ao processamento dos dados pessoais desses usuários concernentes ao Recurso, de acordo com as leis aplicáveis e de modo a nos permitir cumprir essas leis.

Este Recurso não se destina ao processamento de dados confidenciais. Você deve garantir que este Recurso não seja usado para transferir ou processar de outra forma quaisquer dados confidenciais por você ou seus usuários finais.
