## 1\. 배경

이 모듈은 Tencent Low-Code Interactive Classroom("**기능**")을 사용하는 경우 적용됩니다. 이 모듈은 ([“DPSA”](https://intl.cloud.tencent.com/document/product/301/17347))에 있는 데이터 처리 및 보안 계약에 포함되어 있습니다. 본 모듈에서 사용되었지만 정의되지 않은 용어는 DPSA에서 정의된 것과 같은 의미를 갖습니다. DPSA와 본 모듈 간에 상충이 있는 경우, 해당 상충 범위까지 본 모듈이 적용됩니다.

## 2\. 처리

당사는 기능과 관련하여 다음과 같은 데이터를 처리합니다.

| **개인 정보**                                     | **사용**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 다음을 포함하는 **로그 데이터**:<br/>1\. 백엔드 로그: 사용자 ID, 사용자의 IP, 기능의 라이브 스트리밍 기능에 참여하는 사용자와 관련된 OS 유형, 룸 번호, 사용자의 SDK 버전, 그리고<br/>2\. SDK 로그: 사용자 ID. | 당사는 귀하에게 기능을 제공하기 위한 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 당사의 Cloud Log Service(CLS) 기능에 일시적으로 저장 및 백업된다는 점에 유의하십시오. |
| 다음을 포함하는 **품질 모니터링 데이터**:<br/>사용자의 앱 ID, 비디오 및 오디오 기능 정보(사용자가 선택한 활성화 또는 비활성화 여부), 사용자의 룸 입장, 룸 퇴장 관련 정보, 룸 ID, 음소거 기능 사용, CPU 사용량, 메모리 사용량, 네트워크 대기 시간, 패킷 손실, 해상도, 비트 전송률, 프레임 속도 및 볼륨. | 당사는 귀하에게 기능을 제공하고 검증, 디버깅 및 서비스 최적화를 하기 위한 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 TencentDB for MySQL(MySQL) 기능에 저장되고 백업됩니다. |
| 수업 중에 귀하와 당사 사용자가 녹음한 오디오 및 비디오 파일을 포함하는 **비즈니스 데이터**. | 당사는 귀하에게 기능을 제공하고 서비스에서 제공하는 클라우드 녹화(교실 세션을 녹화하려는 경우) 및 저장 기능을 활성화하기 위한 목적으로만 이 데이터를 처리합니다.<br/>이 데이터는 당사의 클라우드 객체 스토리지(COS) 기능에 저장 및 백업됩니다. |

## 5\. 데이터 보존

당사는 기능과 관련하여 처리된 개인 데이터를 다음과 같이 저장합니다.

| **개인 정보**         | **보존 방침**                                         |
| -------------------------------- | ------------------------------------------------------------ |
| 로그 데이터 품질 모니터링 데이터 | 당사는 그러한 데이터를 180일 동안 보유하며 그 후에는 자동으로 삭제됩니다. |
| 비즈니스 데이터                    | 당사는 귀하가 클라우드 계정 기능을 사용하여 데이터를 삭제하도록 당사에 지시할 때까지 또는 귀하의 기능 사용이 중단될 때까지 이러한 데이터를 보유합니다. 당사는 귀하의 유효한 요청 직후 또는 귀하의 기능 사용이 중단된 후 7일 이내에 귀하의 데이터를 삭제합니다. |

귀하는 DPSA에 따라 해당 개인 데이터의 삭제를 요청할 수 있습니다.

## 6\. 특수 조건

본 기능을 사용하려는 경우, 개인 데이터 처리에 동의할 수 있는 최소 연령 이상인 최종 사용자여야 합니다. 이는 최종 사용자가 거주하는 관할권에 따라 다를 수 있습니다.

귀하는 관련 법률에 따라, 그리고 당사가 관련 법률을 준수할 수 있도록, 기능과 관련한 개인 데이터의 처리와 관련하여 최종 사용자로부터 필요한 모든 동의를 획득하고 유지해야 한다는 것을 진술하고 보증하며 약속합니다.

본 기능은 민감한 정보를 처리하기 위한 것이 아닙니다. 귀하는 귀하 또는 귀하의 최종 사용자가 본 기능을 통해 민감한 정보를 전송하거나 처리하지 않도록 해야 합니다.
