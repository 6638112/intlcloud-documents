We will process the following data in connection with the Feature:

| **Personal Information**                                     | **Use**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Log Data,** including:<br/>1. Backend logs: User ID, users’ IP, OS type associated with any users participating in the Feature’s live streaming functionality,  room numbers, users’ SDK versions; and<br/>2. SDK logs: user ID. | We only process this data for the purposes of providing the Feature to you.<br/>Please note that this data is temporarily stored and backed up in our Cloud Log Service (CLS) feature. |
