## 1\. 背景

本模块仅在您使用 Tencent Low-Code Interactive Classroom（简称“**功能**”）时适用。本模块已纳入数据处理和安全协议（简称“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)”）中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，本模块适用。

## 2\. 处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **日志数据，** 包括：<br/>1\. 后端日志：用户 ID、用户 IP、与参与本功能直播功能的任何用户关联的操作系统类型、房间号、用户的 SDK 版本；以及<br/>2\. SDK 日志：用户 ID。 | 我们仅出于向您提供本功能的目的处理这些数据。<br/>请注意，这些数据暂时存储并备份在我们的日志服务 (CLS) 功能中。 |
| **质量监控数据，** 包括：<br/>用户 APP ID、视频和音频功能信息（由用户选择是否启用或禁用）以及与用户进入房间、离开房间、房间 ID、静音功能的使用、CPU 使用、内存使用、网络延迟、数据包丢失、分辨率、比特率、帧速率和音量有关的信息。 | 我们仅出于向您提供本功能以及验证、调试和服务优化的目的处理这些数据。<br/>请注意，这些数据将存储和备份在我们的云数据库 MySQL (MySQL) 功能中。 |
| **业务数据，** 包括您和我们的用户在课堂上录制的音频和视频文件。 | 我们仅出于向您提供本功能，以启用云录制（如果您希望录制课堂会话）和服务提供的存储功能的目的处理这些数据。<br/>请注意，这些数据存储并备份在我们的对象存储 (COS) 功能中 |

## 5\. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述：

| **个人信息**         | **保留政策**                                         |
| -------------------------------- | ------------------------------------------------------------ |
| 日志数据质量监控数据 | 我们将这些数据保留 180 天，之后将自动删除它们。 |
| 业务数据                    | 我们会保留此类数据，直到您要求我们使用您的云帐户功能将其删除，或者直到您停止使用本功能。我们将在您提出有效请求后立即删除您的数据，或在您停止使用本功能后七天内删除您的数据。 |

您可以根据 DPSA 要求删除此类个人数据。

## 6\. 特殊条款

您必须确保使用本功能的最终用户至少达到个人可同意处理自己个人数据的最低年龄。根据最终用户所在的司法管辖区，这一要求可能会有所不同。

您声明、保证并承诺，将根据适用法律并为使我们能够遵守适用法律，就处理与本功能有关的最终用户个人资料一事，取得及保留最终用户的所有必要同意。

本功能不适用于处理敏感数据。您必须确保您或您的最终用户不会将本功能用于传输或以其他方式处理任何敏感数据。
