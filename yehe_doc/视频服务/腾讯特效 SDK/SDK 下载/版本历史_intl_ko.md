## Android
### Version 2.5.0.207 @2022.10.25
- Avatar 기능이 추가되었습니다.
- 오디오 효과 음소거 및 음소거 해제에 대한 지원이 추가되었습니다.
- updateProperties에 대한 콜백 API를 추가했습니다.
- AI 모델 기반 리소스를 동적으로 추가하는 데 사용되는 setBundleToLightEngine API를 추가했습니다.
- 코드 난독화에 대한 지원이 추가되었습니다.
- SDK에서 LogUtils 클래스를 삭제했습니다.


### Version 2.4.2.324 @2022.09.13
경우에 따라 블러셔, 윤곽, 립스틱 효과가 작동하지 않는 문제를 수정했습니다.


### Version 2.4.2.322 @2022.07.19
뷰티필터의 일부 효과 오류를 수정했습니다.

### Version  2.4.2.317 @2022.07.13
`beacon.aar`를 삭제했습니다. v2.4.2.317 이전 버전에서 업데이트하면 새 버전에 'beacon.aar'가 포함되지 않습니다.

###  Version 2.4.2 @ 2022.07.07
- 일부 시나리오에서 불안정한 인식 및 추적 성능을 수정하기 위해 얼굴 인식 알고리즘을 최적화했습니다.
- 이제 updateProperty API의 매개변수가 인증되었습니다. 매개변수 요구 사항에 대한 자세한 내용은 [updateProperty](https://intl.cloud.tencent.com/document/product/1143/45399)를 참고하십시오.
- isDeviceSupport에서 반환하는 값이 때때로 부정확한 문제를 수정했습니다.
- SDK의 libs 디렉터리에 `beacon.aar`를 추가했습니다. 프로젝트에 복사해야 합니다. 프로젝트에 이미 beacon이 있는 경우 충돌이 발생할 수 있습니다. 이를 방지하려면 aar 파일을 가져오지 마십시오.
- Unity에 대한 지원이 추가되었습니다. 자세한 내용은 SDK 패키지의 Unity 폴더를 참고하십시오.
