## Android端末
### Version 2.5.0.207 @2022.10.25
- Avatar機能を追加しました。
- 動的エフェクトの音声オンおよびミュート機能を追加しました。
- updatePropertiesメソッドにコールバックインターフェースを追加しました。
- AIモデルリソースの動的な追加に用いるメソッドsetBundleToLightEngineを追加しました。
- SDK内のコードに対し難読化処理を実施しました。
- SDK内のLogUtilsクラスを削除しました。


### Version 2.4.2.324 @2022.09.13
一部のシーンでチーク、立体、リップの属性設定が無効になる問題を修正しました。


### Version 2.4.2.322 @2022.07.19
美顔の一部に異常が発生する問題を修正しました。

### Version 2.4.2.317 @2022.07.13
`beacon.aar`を削除しました。2.4.2.317以下からバージョンアップした場合、旧バージョンの`beacon.aar`を維持する必要はありません。

###  Version 2.4.2 @ 2022.07.07
- 顔認識アルゴリズムを最適化し、一部のシーンで顔認識と追跡が不安定になる問題を解決しました。
- updatePropertyインターフェースのパラメータを検証しました。パラメータのルールについては[updateProperty](https://intl.cloud.tencent.com/document/product/1143/45399)をご参照ください。
- SDKのisDeviceSupportインターフェースの戻り値が正確ではない可能性がある問題を修正しました。
- SDKパッケージのlibsディレクトリに`beacon.aar`を追加しました。プロジェクトにcopyする必要があります。プロジェクトにすでにbeaconを統合済みの場合は競合が発生する可能性がありますが、このaarをインポートしないことで対処できます。
- Unityプラットフォームの美顔エフェクトをサポートしました。詳細についてはSDKパッケージのUnityフォルダをご参照ください。
