Tencent Effect SDK는 **기본 에디션(A 시리즈)**, **고급 에디션(S 시리즈)**, **아바타 에디션(V 시리즈)**의 세 가지 범주로 분류되는 14개 에디션으로 제공됩니다. 각각 기능이 다르며 귀하의 필요에 가장 잘 맞는 것을 선택할 수 있습니다.

- [기본 A 시리즈 에디션](#setA): 기본적인 뷰티필터 기능을 제공하며, 얼굴 보정에 대한 높은 요구 사항이 없는 고객에게 적합합니다.
- [고급 S 시리즈 에디션](#setS): 고급 뷰티필터 기능(스티커, 메이크업 효과 포함)을 제공하며, 얼굴 보정에 대한 요구 사항이 높은 고객에게 적합합니다.
- [아바타 V 시리즈 에디션](#setV): 가상 아바타 Animoji, 커스텀 아바타, 표정 미러링 기능을 제공합니다. 아바타 기반 소셜라이징 및 라이브 스트리밍과 같은 애플리케이션을 위해 설계되었습니다.

  또한 [X 시리즈](#pluginX)는 능력 수준에서 Tencent Effect 서비스를 제공하여 더 큰 유연성과 확장성을 제공하므로 필요한 특정 기능을 통합할 수 있습니다. 자세한 내용은 [기능 설명](https://intl.cloud.tencent.com/document/product/1143/45376)을 참고하십시오.

Tencent Effect SDK의 다른 버전을 사용하려면 다른 License(에디션)를 [구매](https://buy.cloud.tencent.com/vcube?type=magic)해야 합니다.

## 과금 안내

- 각 License는 하나의 Bundle ID와 하나의 Package Name을 바인딩할 수 있습니다. 공식 License에 바인딩된 Bundle ID 또는 Package Name은 변경할 수 없습니다. 하나 이상의 애플리케이션에서 SDK를 사용하려면 Tencent Effect 에디션을 여러 개 구매해야 합니다.
- License의 유효기간은 1년입니다. 만료되면 새 License를 구입해야 합니다. 그렇지 않으면 해당 기능을 사용할 수 없게 됩니다.
- **섀도우 에디션 설명**: 65,000 USD 이상의 Tencent Effect 에디션을 구매하면 2개의 섀도우 에디션을 무료로 사용할 수 있습니다. 섀도우 에디션은 영업팀에 문의하십시오. 공식 애플리케이션과 섀도우 애플리케이션 모두에 Bundle ID/Package Name을 제공해야 합니다. 2개 이상의 섀도우 애플리케이션을 갖고 싶다면 추가 섀도우 에디션을 구매하십시오(각각 5,000 USD).
- **구매한 소재의 유효 기간은 1년입니다**. 만료된 소재를 계속 사용하려면 다시 구매해야 합니다.
