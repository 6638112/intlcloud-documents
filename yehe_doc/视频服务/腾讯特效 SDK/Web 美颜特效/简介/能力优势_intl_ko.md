## 고성능
SDK는 Web 페이지에 AR 효과를 고효율적으로 적용하는 기능을 제공합니다. 1080P 비디오의 경우 초당 최대 60프레임의 렌더링 레이트를 달성할 수 있습니다. 또한 Web Worker와 WASM을 사용하여 메인 스레드의 렌더링 성능을 보장합니다.

## 강력한 기능
Tencent Effect Web SDK는 업계 내 모바일 애플리케이션과 유사한 효과를 제공하는 몇 안 되는 Web 기반 얼굴 AR 솔루션 중 하나입니다. AI 기반 뷰티 필터, 메이크업, 스티커 등의 효과 렌더링이 가능합니다. **3D 효과, Animoji 및 가상 아바타도 v0.3.0에 도입되었습니다.**

## 온라인 운영
당사의 SDK 및 콘솔을 사용하여 온라인에서 효과 소재를 사용자 정의하고 관리할 수 있으며, 여러 Web 프로젝트에서 사용할 수 있습니다.

## 편리한 사용
2분 이내에 Demo 프로젝트를 실행하고 최소한의 코딩으로 기본 기능을 통합할 수 있습니다. SDK는 TRTC 및 WebRTC와 함께 사용할 수도 있습니다.
