Tencent Cloud View Cube·Web美顔エフェクトはオーディオビデオ端末SDK（Tencent Cloud View Cube）の重要な構成部分であり、美顔エフェクトなどのAR効果をWebで実現できるオールラウンドなソリューションを提供しています。
この製品のSDKおよび素材制作管理ツールを使用すると、AI美顔、フィルタリング、メイクアップ、ステッカー、Animoji表情、バーチャルキャラクターなどのAR効果を、PC Web、モバイルWebページに手軽に追加することができます。
