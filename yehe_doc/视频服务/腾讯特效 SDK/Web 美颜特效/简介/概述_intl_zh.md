腾讯云视立方·Web 美颜特效是音视频终端 SDK（腾讯云视立方）的重要组成部分，提供在 Web中实现美颜特效等 AR 效果的整体解决方案。
您可以使用本产品提供的 SDK 以及素材生产管理工具，轻松将 AI 美颜、滤镜、美妆、趣味贴纸、Animoji 表情、虚拟形象等 AR 效果添加到您的PC Web、移动 Web 页面中。
