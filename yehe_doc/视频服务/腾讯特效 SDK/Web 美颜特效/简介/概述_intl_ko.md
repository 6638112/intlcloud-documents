Tencent Effect는 Tencent Cloud의 Media SDK(RT-Cube) 중 하나입니다. Tencent Effect Web SDK는 포괄적인 Web 기반 AR 효과 솔루션을 제공합니다.
SDK와 소재 생성 및 관리 툴을 사용하여 AI 기반 뷰티필터, 필터, 메이크업, 스티커, Animoji , 가상 아바타 등의 AR 효과를 PC Web이나 모바일 Web 페이지에서 손쉽게 구현할 수 있습니다.

## 무료 신청
현재 Tencent Effect Web SDK는 프로모션 중입니다. 평가판을 [**무료 신청**](https://www.tencentcloud.com/document/product/1143/50806)할 수 있습니다. 신청서 검토는 영업일 기준 1일이 소요됩니다.

당사 웹사이트의 **클라이언트 SDK 가격**은 Tencent Effect Web SDK에 적용되지 않습니다. Web SDK에 대한 과금 방식 및 시기는 아직 미정입니다. **무료 공식 License는 기본적으로 6개월 동안 유효합니다.** Web SDK 정식 과금이 시작되어도 License는 영향을 받지 않습니다. 만약 그 전에 License가 만료되면 자동으로 갱신되며 갱신 세부 정보를 알려드립니다.
