Tencent Effect는 Tencent Cloud의 Media SDK(RT-Cube) 중 하나입니다. Tencent Effect Web SDK는 포괄적인 Web 기반 AR 효과 솔루션을 제공합니다.
SDK와 소재 생성 및 관리 툴을 사용하여 AI 기반 뷰티필터, 필터, 메이크업, 스티커, Animoji , 가상 아바타 등의 AR 효과를 PC Web이나 모바일 Web 페이지에서 손쉽게 구현할 수 있습니다.
