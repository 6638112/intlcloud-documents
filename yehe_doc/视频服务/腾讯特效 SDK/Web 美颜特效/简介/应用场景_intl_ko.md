## 실시간 비디오
비디오 스트리밍, 인터랙티브 라이브 스트리밍 또는 영상 통화 Web 페이지에 SDK를 통합하여 보다 재미있고 인상적인 사용자 경험을 제공합니다.

## 메이크업 체험
SDK의 3D 효과 기능을 기반으로 Web 페이지에 가상 메이크업 체험(예: 컬러 렌즈, 립스틱 등)을 구현하여 오프라인 쇼핑 경험을 재현하고 구매 전환율을 높입니다.

## 마케팅
브랜드 고유의 소재로 AR 홍보 영상을 제작하고 온라인에 공유하여 브랜드 영향력을 높일 수 있습니다.

## 쇼트 비디오
SDK를 사용하여 Web 페이지에서 쇼트 비디오에 뷰티 필터 및 AR 효과를 구현할 수 있습니다.
