SDK는 v0.3.0부터 3D 효과를 지원합니다. 이 기능을 사용하기 전에 SDK 버전을 확인하십시오.

표준 헤드 모델을 기반으로 Blender 또는 3DSMax와 같은 소프트웨어를 사용하여 보다 사실적인 3D 모델을 생성하고 콘솔에 업로드할 수 있습니다.

## 제한

보다 적용 가능하고 실감나는 효과 위해 [표준 헤드 모델](https://webar-static.tencent-cloud.com/assets/model/3D-temp.glb)을 기반으로 3D 모델을 만드는 것이 좋습니다.

| 유형 | 제한 |
|---------|---------|
| 형식 | GLB 파일 또는 GLTF 폴더 |
| 크기 | 로딩 시간을 고려하여, 모델의 크기를 5M 이내로 유지 |
| 폴리곤 | 출력 파일의 크기를 고려하여, 모델의 폴리곤 수를 10만 이내로 유지 |
| 텍스처 맵 | PNG 형식의 정사각형 텍스처를 사용하고 각 맵의 크기를 1024\*1024 이내로 유지. 일반적으로 사용되는 디멘션: 256, 512, 1024 <br>품질을 손상시키지 않으면서 크기를 가능한 한 작게 유지 |
| 재질 | 표면에 Principled BSDF 사용, 그렇지 않으면 렌더링 오류 발생 |

## 제작 단계

### 1단계: 소재 생성

1. 표준 헤드 모델을 다운로드하여 3D 소프트웨어로 가져옵니다(이 문서에서는 Blender가 예로 사용됨). 모델을 기반으로 소재를 생성하고 GLB 또는 GLTF 형식으로 내보냅니다.

>!
>- 생성된 소재가 사용자의 관점에 맞도록 위치 및 크기와 같은 모델의 변형 옵션을 수정하지 마십시오.
>- 출력 파일의 크기를 작게 유지하고 로딩 시간을 짧게 유지하려면 모델을 GLB 형식으로 생성하는 것이 좋습니다.

### 2단계: 모델을 콘솔로 가져오기

1. 콘솔의 소재 사용자 지정 페이지로 이동합니다. 내장된 2D 효과 모델이 기본적으로 표시됩니다. 상단의 **3D 효과**를 클릭하면 위에서 언급한 표준 헤드 모델과 동일한 3D 모델로 전환됩니다.
2. 오른쪽의 **모델 선택**을 클릭하고 모델 형식을 선택합니다(이 예시에서는 GLB를 선택함).

>!
>- 한 번에 하나의 3D 효과 시나리오만 편집할 수 있습니다. 여러 3D 모델을 가져오려면 왼쪽의 **+**를 클릭합니다.
> -  편집 중에 왼쪽 목록에서 모델을 클릭하여 다른 모델로 전환할 수 있습니다.


### 3단계: 매개변수 수정
일반적으로 3D 모델은 표준 헤드 모델을 기반으로 이미 구축되었기 때문에 콘솔로 가져온 후 사용할 준비가 되어 있어야 합니다. 모델의 매개변수를 수정해야 하는 경우 상단의 **속성 편집 모드**를 선택하고 편집 영역의 요소를 드래그하여 설정을 변경하거나 오른쪽에서 직접 매개변수를 수정합니다.


>!모델이 표준 헤드 모델을 기반으로 생성되지 않은 경우 추적 지점을 조정하여 모델을 더 잘 추적할 수 있습니다. 예를 들어 콧대를 안경 모델의 추적 지점으로 설정하고 인중을 콧수염 모델의 추적 지점으로 설정할 수 있습니다.

### 4단계: 효과 미리보기
내장 모델에서 소재를 미리 볼 수 있습니다. 카메라로 전환하여 카메라로 캡처한 비디오에서 효과를 볼 수도 있습니다.

### 5단계: 소재 내보내기
오른쪽 상단에서 **내보내기**를 클릭합니다. 팝업 창에서 썸네일을 선택하고 효과 이름 등의 정보를 입력한 후 **게시**를 클릭합니다. 게시된 소재는 **소재 관리** 페이지에서 볼 수 있습니다.
