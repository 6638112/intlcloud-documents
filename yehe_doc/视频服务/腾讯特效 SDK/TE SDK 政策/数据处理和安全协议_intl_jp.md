| **個人情報**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **エンドユーザーの写真やビデオ**                       | 当社は、関連する美化効果を適用することにより、本機能をお客様に提供する目的で本情報を使用します。<br/>この機能は、美化効果が適用された後、写真やビデオを保存したり、さらに処理したりすることはありません。 |
