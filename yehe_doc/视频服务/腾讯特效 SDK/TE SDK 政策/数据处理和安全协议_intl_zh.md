## 1\. 背景

本模块仅在您使用 Tencent Effect SDK（简称“**功能**”）时适用。本模块包含在位于 [DPSA](https://intl.cloud.tencent.com/document/product/301/17347) 的数据处理和安全协议中。本模块中使用但未作定义的术语应具有在 DPSA 中规定的含义。如果 DPSA 与本模块之间发生任何冲突，对于此等不一致，则本模块适用。

## 2\. 处理

我们将针对本功能处理以下数据：

| **个人信息**                                     | **用途**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **您的最终用户的照片或视频**                       | 我们将出于为您提供本功能的目的使用此信息，而且应用相关的美化功能。<br/>应用美化效果后，该功能将不会存储或进一步处理照片或视频。 |
| **您的最终用户的设备和技术数据：**  Wi Fi 状态、 系统属性（系统特性）、设备型号、操作系统、传感器信息（设备的水平和垂直屏幕状态） | 我们将出于为您提供本功能的目的使用此信息。<br/>应用美化效果后，该功能将不会存储或进一步处理照片或视频。 |

我们的工作人员不会存储或访问上述数据。

## 5\. 数据保留

我们将存储因本功能而处理的个人数据，具体如下所述：

| **个人信息**                        | **保留政策**     |
| ----------------------------------------------- | ------------------------ |

| 您的最终用户的设备和技术数据 |      未存储此数据。                     |
