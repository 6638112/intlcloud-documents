## 申请测试版License
我们为基础版SDK提供测试体验（免费测试有效期为14天，可续期1次，共28天），只需如下3步开启您的体验。

1. 登陆云直播控制台，在左侧菜单栏中选择【直播SDK】>[【License】](https://console.cloud.tencent.com/live/license)。

   ![](https://main.qcloudimg.com/raw/99435a29a00bde5ad866480738d45df4.png)

2. 单击【Get  License】，填写【Package Name】为 Android 的包名，【Bundle Id】为 iOS 的 Bundle ID。（若无 Package Name 或 Bundle Id，可填写“-”。）

   ![](https://main.qcloudimg.com/raw/5f721dd4406fbc7fb8019591c771b467.png)

3. 单击【确定】即可成创建测试版License。单击左侧小三角icon便可查看License信息，请记录其中的Key 和LicenseUrl，便于在SDK初始化时使用。

   ![](https://main.qcloudimg.com/raw/b4d1f52146cd25c595a45a3c56c4dff5.png)

4. 测试版License初次申请有效期默认为14天，期满后您可续期1次，单击测试版License右侧的【edit】进入修改界面，单击【ok】重新保存即可续期14天。

   ![](https://main.qcloudimg.com/raw/b9f30d4ca8a3349b2003d15008a85b83.png)

## 购买正式版License

1. 如果您需要购买正式的基础版SDK  License，请直接点击【Purchase Official License】栏里的【Contact Sales】。

   ![](https://main.qcloudimg.com/raw/4fc8840a064c7ebb1ccbef999574fa47.png)

2. 填写提交表格后，我们会第一时间让商务与您取得联系，并帮助您购买获得正式版License。

   ![](https://main.qcloudimg.com/raw/107540d0d8d7e29177400a83689bd36d.png)

   单击【确定】前需要再次确认 Bundle  ID 和 Package Name，如与提交到应用商店的不一致请提前进行修改，一旦提交成功将无法再修改 License 信息。     正式版 License 不支持信息修改。

3. 购买生效的正式版License记录会出现在你的直播SDK控制台中，您可以在这里对它们进行管理。单击左侧小三角icon便可查看License信息，请记录其中的Key 和LicenseUrl，便于在SDK初始化时使用。

   ![](https://main.qcloudimg.com/raw/d2dd4ee349a52b10c4609d655fd54bc8.png)

   如果您有什么困惑想要咨询，请不要犹豫地通过【[Contact  Sales](https://intl.cloud.tencent.com/contact-sales)】与我们取得联系。
