Tencent Cloud offers a straightforward and easy-to-understand API example project regarding frequently asked questions among developers. You can use it to quickly learn how to use different APIs.

## Download Address

| Platform |                         GitHub Address                          |
| :------: | :----------------------------------------------------------: |
| Flutter  | [GitHub](https://github.com/LiteAVSDK/Live_Flutter.git) |

## Contents
The demo project covers the following features:
  - [Publishing from Camera](https://github.com/LiteAVSDK/Live_Flutter/tree/main/Live-API-Example/lib/page/push/live_camera_push.dart)
  - [Publishing from Screen](https://github.com/LiteAVSDK/Live_Flutter/tree/main/Live-API-Example/lib/page/push/live_screen_push.dart)
  - [Playback](https://github.com/LiteAVSDK/Live_Flutter/tree/main/Live-API-Example/lib/page/play/live_play.dart)
  - [Same-Room Communication](https://github.com/LiteAVSDK/Live_Flutter/tree/main/Live-API-Example/lib/page/link_mic)
  - [Cross-Room Communication](https://github.com/LiteAVSDK/Live_Flutter/tree/main/Live-API-Example/lib/page/pk)
