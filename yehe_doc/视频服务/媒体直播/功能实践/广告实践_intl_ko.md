StreamLive를 사용하여 라이브 스트림에 광고를 삽입할 수 있습니다. 다음과 같은 광고 삽입 솔루션을 제공합니다.

1. 입력 스위치
channel에 대한 입력으로 비디오 파일 또는 라이브 스트리밍 URL을 바인딩합니다. Plan 페이지에서 지정된 시간 또는 구성 직후에 비디오 또는 라이브 스트림을 삽입하는 Input Switch 이벤트를 생성합니다. 아래 이미지는 지정된 시간에 라이브 스트림에 비디오 파일을 삽입하는 방법을 보여줍니다.
![](https://qcloudimg.tencent-cloud.cn/raw/0e5789754fc7fd86e362f55adbe30e8b.png)
StreamLive는 지정된 utc 시간에 동영상 파일로 전환하여 광고를 재생합니다. 광고가 끝나면 StreamLive는 원래 라이브 스트림으로 다시 전환됩니다. 전체 프로세스는 사람의 개입 없이 자동으로 실행됩니다.

2. SCTE-35
입력이 MPEG-2 TS 스트림인 경우 SCTE-35 payload를 입력에 포함할 수도 있습니다. StreamLive는 페이로드를 인식하고 출력에 표시할 수 있는 정보로 변환합니다. 플레이어가 정보를 인식하고 광고로 전환하는 데 표준 방법이 사용됩니다. 해당 Output에 대해 SCTE-35 통과만 활성화하면 됩니다. 자세한 내용은 SCTE-35 문서를 참고하십시오.
