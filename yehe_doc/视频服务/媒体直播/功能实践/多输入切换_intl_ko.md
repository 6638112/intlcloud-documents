입력 스위치를 사용하면 여러 입력을 사용하여 라이브 스트리밍 경험을 풍부하게 할 수 있습니다. Channel에 두 개의 PUSH 입력과 다른 유형의 여러 입력을 추가할 수 있습니다. 입력 스위치를 구성하려면 다음 단계를 따르십시오.

1. **Input Setting** 페이지에서 사용하려는 Input을 추가합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/ef775d0351a5c1bb135f75a5db724758.png)

2. 채널을 생성하거나 편집한 후 **Channel Management** 페이지에서 Channel을 찾고 이름을 클릭하여 세부 정보 페이지로 들어갑니다. **Plan** 탭을 선택하고 **Create Event**를 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/d893a5ea7f36b4b0d7b066b26f6c8919.png)
![](https://qcloudimg.tencent-cloud.cn/raw/81469ba91b25d56feb978202b089aa1b.png)

**Input Attachment**의 드롭다운 목록에서 방금 채널에 추가한 Input을 선택하고 **Start Type**을 지정합니다. 자세한 내용은 [Plan Management](https://www.tencentcloud.com/document/product/1048/49402)를 참고하십시오.
