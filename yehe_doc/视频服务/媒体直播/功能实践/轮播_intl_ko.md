비디오 루핑은 입력 스위치의 일반적인 시나리오입니다. 입력 스위치를 구성하는 방법은 이전 문서를 참고하십시오. 이 기능을 사용하여 라이브 스트림이 시작되기 전에 루프에서 티저를 재생하거나 라이브 스트리밍 세션이 끝난 후 반복적으로 라이브 스트리밍 세션을 재생할 수 있습니다.

비디오 루핑 기능은 기존 비디오를 재생하므로 Input 유형은 **MP4_PULL** 또는 **HLS_PULL**이어야 합니다. Channel이 현재 이러한 Input과 연결되어 있지 않으면 입력 스위치 이벤트를 구성하기 전에 먼저 하나를 추가하십시오. 아래 그림은 PULL Input에 대한 일반적인 입력 스위치 이벤트의 구성을 보여줍니다. 장애 조치를 위해 두 개의 입력을 지정할 수 있습니다. **Address** 중 공개 URL 또는 공개 읽기 액세스 권한이 있는 Tencent Cloud COS 주소를 입력하십시오.
