# 频道质量监控
打开「Channel Management」界面点击频道名称即可进入频道详情页面。频道详情页面会显示该频道的所有输入、输出、Alerts、Health 等信息。
当某个频道的任一管道中发生问题或潜在问题时，MediaLive 都会为该频道生成警报。 
![](https://main.qcloudimg.com/raw/35f6b92e513e1063e53de64c686c2796.jpg)
Heath 向用户提供频道输入和输出信息及对应的曲线图，以确定当前频道是否正常工作。
