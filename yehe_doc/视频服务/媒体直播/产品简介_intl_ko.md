# 개요

Tencent Cloud StreamLive는 Tencent Cloud가 새롭게 출시한 고품질의 스트림 처리 플랫폼입니다. 방송 수준의 실시간 온라인 스트림 미디어 처리 서비스 제공이 가능합니다. 사용자는 해당 서비스를 통해 고품질의 비디오 스트림을 생성하여 여러 유형의 디바이스에 배포할 수 있습니다.

StreamLive는 Tencent Cloud의 독자적인 고성능 비디오 인코딩 및 압축 알고리즘을 채택하여, 더욱 개선된 시청 경험을 보장하면서도 전송 트래픽은 절약해 줍니다. 또한 7 * 24시간의 높은 가용성을 보장합니다.

# 특징

**멀티프로토콜 멀티 스트림 입력**

StreamLive는 RTP, RTMP, UDP, HLS, HTTP-MP4, RTP-FEC 입력과 같은 멀티 스트림 미디어 입력 프로토콜을 지원하고, PULL과 PUSH 2가지 방식을 제공합니다. 또한 RTP MPEG-TS 입력 방식에 멀티 오디오 트랙, 비디오 스트림 입력을 지원하며, 모든 오디오 트랙(언어 종류)에 대해 독립적인 트랜스 코딩 설정이 가능합니다.

**고품질 멀티 트랜스 코딩**

멀티 해상도, 멀티 트랜스 코딩, 멀티 프레임 레이트의 트랜스 코딩 기능을 지원하며 SD, HD, UHD, 2K, 4K 등 해상도를 고객이 선택할 수 있게 해줍니다. Tencent Cloud의 초고속 고화질(Top Speed Codec Transcoding) 고성능 트랜스 코딩 서비스를 제공하며, 해당 기능을 활성화하면 AI 알고리즘 기능을 통해 실시간으로 비즈니스 시나리오에 최적화된 동적 인코딩 매개변수를 컴퓨팅하여 낮은 비트 레이트의 고품질 트랜스 코딩 서비스를 구현합니다.

**디지털 저작권 관리****(DRM)**

더 전문적인 디지털 저작권 관리 솔루션(DRM)을 제공하여 다방면에서 사용자의 비디오 스트림 안전을 지켜주며, Fairplay 및 Widevine 암호화 솔루션을 지원하여 대부분의 디지털 저작권 보호 수요를 만족시켜 줍니다.

**멀티프로토콜 멀티 방식 출력**

여러 양식의 출력 캡슐화 유형을 지원하며, 적응형 비트 레이트 HLS, DASH, COS로의 출력 및 보관, StreamPackage로의 출력 등 다양한 양식의 출력을 지원합니다.

**독립 채널의 탄력적인 구성****,** **다양한 출력 양식의 조합**

StreamLive는 채널 관리(Channel Management) 모드를 기반으로 서비스됩니다. 동일한 채널에 서로 다른 Input과 Outputs을 구성할 수 있고, 동일한 Output을 이미 생성된 오디오 트랜스 코딩 및 비디오 트랜스 코딩 템플릿에 조합하여 연결할 수 있으며, 구성 출력을 유연하게 조합하고 적응형 비트 레이트 배포 기능을 구현합니다.

**스트리밍 품질 모니터링**

채널의 실행 상태를 기반으로 상세한 Health 체크를 제공하고, 각종 Alerts 정보를 표시하여 사용자가 실시간으로 스트리밍 품질을 확인할 수 있게 해줍니다.

**고성능**

SSIM과 VMAF 두 가지 표준을 기반으로 하는 Tencent Cloud 초고속 고화질 인코딩은 Aws Elemental StreamLive-qvbr의 동종 제품 보다 훨씬 뛰어난 성능을 자랑합니다.

**저렴한 비용**

초고속 고화질 최적화 전송 비트 레이트는 CDN 대역폭 비용 절감에 대한 클라이언트의 요구에 부합합니다. 비트 레이트가 10% 정도 떨어져도 최상의 성능이 유지됩니다.

**아키텍처 설계**

StreamLive의 모든 채널은 input 구성부터 기본/보조 채널 입력 설정이 가능한 이중 채널 구성을 지원합니다. 그중 Pipeline A는 필수 입력 항목이며 Pipeline B는 선택 입력 항목입니다.

Pipeline A와 Pipeline B는 각자 독립적으로 실행됩니다. 즉, input A로부터 입력된 stream은 각 output group의 destination A로 출력되고, input B로부터 입력된 stream은 각 output group의 destination B로 출력됩니다.
