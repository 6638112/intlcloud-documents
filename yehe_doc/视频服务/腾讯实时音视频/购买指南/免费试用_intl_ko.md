## 무료 베타 패키지
2019년 10월 11일부터 [TRTC 콘솔](https://console.cloud.tencent.com/trtc)에서 처음으로 애플리케이션을 생성한 Tencent Cloud 계정에 10,000분 무료 베타 패키지를 증정합니다. 무료 패키지는 [영상 통화](https://intl.cloud.tencent.com/document/product/647/39788), [음성 통화](https://intl.cloud.tencent.com/document/product/647/39787), [비디오 ILVB](https://intl.cloud.tencent.com/document/product/647/39786), [음성 ILVB](https://intl.cloud.tencent.com/document/product/647/39785) 서비스 용량을 차감하는 방식으로 사용할 수 있습니다.

베타 패키지 설명:
- 베타 패키지의 유효 기간은 증정 당일~다음 해 당월 말일까지입니다.
 예를 들면, 2020년 05월 01일 증정 받은 베타 패키지의 유효 기간은 2020년 05월 01일~2021년 05월 31일입니다.
- 음성, 표준 화질, 고화질, 초고화질 사용 시간에 따라 **1:1:1:1**로 차감됩니다(유형에 상관없이 1분을 사용하면 베타 패키지에서 1분 차감). 유형별 실제 사용 시간에 따라 베타 패키지에서도 실시간으로 잔여 사용 시간이 차감됩니다.
- 초과 요금이 부과되지 않도록 **베타 패키지를 모두 사용하거나 기간이 만료되면 자동으로 서비스가 중지됩니다**. 필요한 경우 [패키지 구매](https://buy.cloud.tencent.com/trtc)에서 서비스를 다시 활성화하십시오. 구매한 패키지는 결제 완료 후 약 5분 뒤 적용되며, 새 패키지가 적용되면 자동으로 서비스가 다시 활성화됩니다.
- 베타 패키지 유효 기간이 만료되면 잔여 시간이 자동으로 삭제되며 복구할 수 없습니다.
