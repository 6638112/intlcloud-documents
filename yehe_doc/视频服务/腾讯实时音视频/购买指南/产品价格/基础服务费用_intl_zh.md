## 月结后付费相关费用

使用实时音视频基础服务产生的通话费用都采用月结后付费，每月1-5日出上一个自然月的账单并扣除费用。

具体的计费标准如下表所示：

| 通话类型              | 单价             |
| :-------------------- | :--------------- |
| 语音通话              | 0.99美元/千分钟  |
| 标清（SD）视频通话    | 1.99美元/千分钟  |
| 高清（HD）视频通话    | 3.99美元/千分钟  |
| 超高清（HD+）视频通话 | 14.99美元/千分钟 |

**计费说明：**

1. 计费项：语音通话、标清（SD）视频通话、高清（HD）视频通话、超高清（HD+）视频通话。

2. 计费方式：后付费计费。

3. 计费周期：按月计费，每月1 - 5日出上月账单。

4. 计费规则：按编码方式和分辨率，统计一个自然日内的不同编码方式和分辨率的转码时长乘以对应单价。

5. 计费示例：

   假设您于2020年1月1日创建实时音视频应用，在1月份使用总语音通话时长为10000分钟、总标清（SD）视频通话时长为10000分钟、总高清（HD）视频通话时长为10000分钟、总超高清（HD+）视频通话时长为10000分钟。

   语音通话费用 = (10000/1000)千分钟 * 0.99美元/千分钟 = 9.9美元

   标清（SD）视频通话费用 =  (10000/1000)千分钟 * 1.99美元/千分钟 = 19.9美元

   高清（HD）视频通话费用 =  (10000/1000)千分钟 * 3.99美元/千分钟 = 39.9美元

   超高清（HD+）视频通话费用 = (10000/1000)千分钟*14.99美元/千分钟 = 149.9美元

   总费用 = 语音通话费用 + 标清（SD）视频通话费用 + 高清（HD）视频通话费用 + 超高清（HD+）视频通话费用 = 9.9美元 + 19.9美元 + 39.9美元 + 149.9美元  = 219.6美元

   > 说明：
   >
   > - 实时音视频各计费项计费通话时长不足1分钟按1分钟计算，如：语音通话时长1000.5分钟则计费时长为1001分钟。
