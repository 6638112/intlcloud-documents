Tencent Effect SDK には全部で10のパッケージがあり、10のパッケージは、**Aシリーズベーシックパッケージ**と**Sシリーズハイグレードパッケージ**という2つのシリーズに分かれています。シリーズとパッケージの違いによって対応する機能が異なり、ニーズに合わせてパッケージを選ぶことができます。より詳細なパッケージの機能については、[機能説明](https://intl.cloud.tencent.com/document/product/1143/45376)をご参照ください。

- [Aシリーズベーシックパッケージ](#setA)：汎用型のスタイリング機能を提供します。美顔調整へのニーズが比較的低いお客様に適しています。
- [Sシリーズハイグレードパッケージ](#setS)：ハイグレードなスタイリング機能（エフェクトステッカーやメイクアップを含む）を提供します。美顔調整へのニーズが比較的高いお客様に適しています。

[Tencent Effect SDK購入ページ](https://buy.intl.cloud.tencent.com/vcube)に進み、該当のSDKパッケージを購入してLicenseを取得すると、対応する機能をご利用いただけます。

## 課金説明

- 1つのLicenseに1つのBundle IDと1つのPackage Nameをバインドできます。正式版Licenseにバインドしたパッケージ名の情報は変更できません。複数のパッケージ名でアクセスしたい場合は、対応する数のTencent Effect Licenseを購入してバインドする必要があります。
- Licenseの有効期間は1年間です。期限切れとなった場合、新たにパッケージを購入して更新し、Licenseの新しい有効期間を取得する必要があります。更新しない場合、対応する機能は使用できなくなります。
- **シャドウパッケージの説明**：USD 65,000以上のパッケージを購入すると、2つのシャドウパッケージを利用できます。シャドウパッケージを使用したい場合は、担当者にご連絡いただければ手動で発行します。権限発行の際に、シャドウパッケージ名と正式版パッケージ名の両方を一度にご提供ください。業務上、さらに多くのシャドウパッケージが必要な場合は、1パッケージの権限につきUSD 5000が必要です。
- **素材の有効期間は1年間です**。翌年も継続購入した場合、無料素材は引き続き使用できます。
- **購入した素材は、素材購入時点のLicenseでしか使用できません**。どのような形式であっても素材の互換は許可されず、贈呈、転売、賃貸（が含まれますが、これらに限定されません）などの方法での取引も許可されません。Tencent Cloudはあらゆる**法的な追及を行う権利**を保持します。



## 課金の例

ユーザーのAさんはAシリーズベーシックパッケージA1 - 02を購入し、2022年1月1日に審査によって承認され、権限の発行を受け、License Aを取得しました。SシリーズハイグレードパッケージS1 - 01を購入し、2022年6月1日に審査によって承認され、権限の発行を受け、License Bを取得しました。パッケージ名は2022年6月1日に先にLicense Aをバインド解除した後、License Bをバインドしました。この場合は次のようになります。

-  License Aの有効期限は2023年1月2日 00:00:00です。これにはAシリーズベーシックパッケージのベーシック美顔、画面調整、ベーシックスタイリング、フィルターおよびステッカー機能が含まれます。
-  License Bの有効期限は2023年6月2日 00:00:00です。これにはSシリーズハイグレードパッケージのベーシック美顔、画面調整、ハイグレードスタイリング、フィルター、ステッカー、ハイグレードステッカー、メイクアップ機能が含まれます。

すなわち、このパッケージ名のアプリケーションは2022年1月1日 - 2022年6月1日の**バインド解除前**まではAシリーズベーシックパッケージA1 - 02の機能（ベーシック美顔、画面調整、ベーシックスタイリング、フィルターおよびステッカー機能）を使用でき、2022年6月1日の**バインド変更後** から2023年6月2日 00:00:00までの間は、 SシリーズハイグレードパッケージS1 - 01の機能（ベーシック美顔、画面調整、ハイグレードスタイリング、フィルター、ステッカー、ハイグレードステッカー、メイクアップ機能）を使用できます。

バインドを解除したAシリーズベーシックパッケージA1 - 02のLicenseは、まだバインドしていない他のLicenseのアプリケーションに引き続きバインドすることができます。有効期限は2023年1月2日 00:00:00です。
