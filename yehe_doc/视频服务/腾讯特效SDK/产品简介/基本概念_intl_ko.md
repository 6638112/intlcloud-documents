### 뷰티 필터

스마트 뷰티 필터 기능을 달성하기 위해 다양한 스타일을 지원하고 다양한 미용 옵션을 제공하며 약 30가지의 얼굴 세부 조정을 지원합니다. 피부 보정, 얇은 얼굴, 큰눈, 투명, 미백, 작은 얼굴, 좁은 얼굴, 가는 광대, 얇은 코, 입 모양, 이마, 립스틱, 블러셔, 컨투어링, 치아 미백, 다크 써클, 팔자 주름, 홍조, 포화 ,콘트라스트, 주름 제거, 눈꼬리, 짧은 얼굴, 또렷한 눈, 콧볼, 코 위치, 안면하수 제거, 입술 두께, 눈 간격, 턱.

### 필터

프리미엄 고성능 필터 내장, 실시간 화면 프리톤, 이미지 융합 기술 접목, 자연스러운 핏, 화이트, 내추럴, 첫사랑, 클리어, 고백, 설렘, 맑은 등 총 37가지의 필터를 선택할 수 있다.

### 애니메이션 효과

정확한 얼굴 특징 위치 지정, 안정적인 얼굴 추적, 제스처 인식, 동적 트리거, 2D/3D 동적 효과 스티커, 1명/그룹 스티커 등을 포함한 최대 58개의 동적 효과 재료를 지원합니다.

### 메이크업

실시간 피부 보정 브라이트닝을 지원하며, 얼굴 이목구비를 정교하게 세팅하고, 다양한 트렌드에 맞춰 원터치 메이크업으로 자연스러운 메이크업을 완성합니다. 인기 메이크업은 벚꽃, 활력, 복숭아, 퀸카 등이 있습니다.

### 키잉

정확한 키잉을 지원하며, 배경을 섬세하고 역동적으로 바꿀 수 있습니다. 9가지 배경을 선택할 수 있습니다.

### Package Name
Package Name은 애플리케이션의 고유 식별자로서 Android에서 고유한 개념입니다.
Android 프로젝트의  `Mainfest.xml`  파일에서 패키지 이름을 얻을 수 있습니다.
```
<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
package="com.huawei.player"
android:versionCode="20181111"
android:versionName="1.0">
```
