###  과금 안내

StreamLink 콘솔은 스트림 차원을 기반으로 관리하며 각 스트림은 각 스트림 전송 링크에 해당합니다. StreamLink 콘솔을 기반으로 비디오 스트리밍 미디어를 빠르고 안정적으로 전송할 수 있으며, 동시에 비디오 스트리밍에 대한 전체 품질 모니터링도 수행할 수 있습니다. StreamLink 과금 항목에는 다음 세 가지 항목이 포함됩니다.

| 과금 항목 | 과금 설명 | 과금 방식 |
| -----------------| ---| --------- |
| 전용 회선 전송료 |전용 회선 전송 사용 요금. 각 스트림의 당일 최대 대역폭을 합산하여 과금 | 후불-일 결산 |
| 머신 실행 요금 |머신 전송 사용 요금. 당일 작업 건수와 시간(단위: 시간)을 기준으로 과금 | 후불-일 결산 |
| 푸시 트래픽 비용 |최대 8Mbps의 대역폭. 당일 푸시 리전 및 트래픽을 기준으로 과금 | 후불-일 결산 |

### 전용 회선 전송 요금

#### 과금 가격

사용자는 StreamLink를 통해 콘솔에서 새 전송 링크를 생성할 수 있으며, 당일 실제 사용한 최대 대역폭을 기준으로 과금됩니다.

| 과금 항목   | 과금 방식          | 가격          |
| -----------------| ---| --------- |
| 전송 대역폭 |일일 전용 회선 전송 최대 대역폭을 기준으로 과금 | 7.6923USD/Mbps/일 |

#### 과금 예시

2022년 01월 01일에 StreamLink 서비스를 사용하고 두 개의 스트림 A와 B를 생성하였다고 가정합니다. 스트림 A는 정오 12:00에 최대 대역폭 10Mbps에 도달하였고, 스트림 B는 밤 22:00에 최대 대역폭 10Mbps에 도달한 경우, 2022년 01월 02일에 지불해야 하는 라이브 스트리밍 트래픽 청구 내역은 다음과 같습니다.
일일 전용 회선 전송 비용 = 7.6923(USD/Mbps/일)× 20(Mbps)= 153.846USD.



### 머신 실행 요금

#### 과금 가격

StreamLink 서비스의 머신 실행 요금의 과금 방식은 다음과 같습니다.

| 과금 항목     | 과금 방식                               | 가격             |
| ---------- | -------------------------------------- | ---------------- |
| 머신 실행 요금 | 머신 실행 시간 요금. 각 스트림의 시간당 요금 계산 | 0.25USD/스트림/시간 |


>!
> - 스트림 수는 input과 output 수량을 계산합니다. 1개의 input 스트림과 3개의 output에 따라 계산되며, 4 방향 스트림 컴퓨팅 머신 실행 시간에 따라 실행 시간을 계산합니다.
> - 1개의 프라이머리와 세컨더리 스트림을 포함하는 작업으로, 2 방향 스트림 컴퓨팅 머신 실행 시간을 계산합니다.
> - 시간 단위로 요금을 계산하며, 사용 시간 1시간 미만은 1시간으로 계산합니다. 정산 주기는 일 결산입니다.

#### 과금 예시

2022년 01월 01일에 StreamLink 서비스를 사용하고 하나의 input과 두 개의 output 작업을 생성했으며, 당일 사용 시간이 6시간이라고 가정합니다. 2022년 01월 02일에 지불해야 하는 라이브 스트리밍 트래픽 청구 내역은 다음과 같습니다.

일일 머신 실행 요금=3(스트림 수)× 6(시간) × 0.25(USD/시간)= 4.5USD.
