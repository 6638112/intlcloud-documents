On the [Flow Management](https://console.tencentcloud.com/mdc/flow) page of the StreamLink console, click a flow to go to its details page.
![](https://qcloudimg.tencent-cloud.cn/raw/70fa8ba987f834659b631e163cf67b11.png)

## Editing the Input
On the flow details page, select the **Input Source** tab and click **Edit** in the top right corner.
![](https://qcloudimg.tencent-cloud.cn/raw/34f122d1d8b90552e00dd1415002061a.png)

- If a flow is idle, you can change all its input settings except **Max Bandwidth**.
- If a flow is active, you can only change the CIDR IP allowlist.

## Editing an Output
On the flow details page, select the **Output** tab, find the output you want to edit, and click **Edit**.
![](https://qcloudimg.tencent-cloud.cn/raw/ecf04e2621fc971f27babd6a983d89f1.png)

- If a flow is idle, you can change all its output settings except the output region and output protocol.
- If a flow is active, you can only change the CIDR IP allowlist.
