StreamLink 콘솔의 [Flow Management](https://console.tencentcloud.com/mdc/flow) 페이지에서 Flow를 클릭하여 Flow의 세부 정보 페이지로 이동합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/70fa8ba987f834659b631e163cf67b11.png)

## Input 편집
Flow 세부 정보 페이지에서 **Input Source** 탭을 선택하고 **Input Source** 페이지 오른쪽 상단에서 **Edit**을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/34f122d1d8b90552e00dd1415002061a.png)

- Flow가 Idle 상태인 경우 Max Bandwidth를 제외한 모든 입력 설정을 변경할 수 있습니다.
- Flow가 Running 상태인 경우 CIDR IP Allowlist만 변경할 수 있습니다.

## Output 편집
Flow 세부 정보 페이지에서 **Output** 탭을 선택하고 편집할 Output을 찾은 다음 **Edit**을 클릭합니다.
![](https://qcloudimg.tencent-cloud.cn/raw/ecf04e2621fc971f27babd6a983d89f1.png)

- Flow가 Idle 상태인 경우 Output Region 및 Output Protocol을 제외한 모든 출력 설정을 변경할 수 있습니다.
- Flow가 Running 상태인 경우 CIDR IP Allowlist만 변경할 수 있습니다.
