###  課金説明

StreamLinkコンソールはストリームディメンションに基づいて管理を行い、ストリームごとにストリームの伝送リンクが1個ずつ対応します。StreamLinkコンソールに基づいて迅速かつ安定してビデオストリームメディアを伝送でき、さらに伝送中のビデオストリームに対してオールラウンドな品質監視を行うことができます。StreamLinkの課金項目は以下の3つを含みます。

| 課金項目 | 課金の説明 | 支払い方法 |
| -----------------| ---| --------- |
| 専用線伝送料金 |専用線で伝送する際の課金です。発生した料金は各ストリームの当日のピーク帯域幅の累計に応じて課金されます | 後払い-日次決済 |
| マシン運用料金 |マシンで伝送する際の課金です。発生した料金は当日のタスク数と時間に応じて時間単位で課金されます | 後払い-日次決済 |
| 転送トラフィック料金 |帯域幅は最高8Mbpsです。発生した料金は当日プッシュしたエリアとトラフィックサイズに応じて課金されます | 後払い-日次決済 |

### 専用線伝送料金

#### 課金価格

StreamLinkはユーザーがコンソールで伝送リンクを作成することを許可し、当日実際に使用した帯域幅ピーク値に応じて精算されます。

| 課金項目   | 課金方式           | 価格          |
| -----------------| ---| --------- |
| 伝送帯域幅 |1日あたりの専用線のピーク帯域幅に応じて課金されます。 | 7.6923米ドル/Mbps/日 |

#### 課金例

お客様は2022年1月1日にStreamLinkサービスを使用し、A、Bの2つのストリームを作成しました。Aストリームは昼の12時にピーク帯域幅10Mbpsになり、Bストリームは夜の22時にピーク帯域幅10Mbpsになりました。このとき、2022年1月2日に支払うライブストリーミングトラフィック請求書は以下のとおりとなります。
1日あたりの専用線伝送料金 = 7.6923（米ドル/Mbps/日）× 20（Mbps）= 153.846米ドル。



### マシン運用料金

#### 課金価格

StreamLinkサービス使用におけるマシン運用料金の課金方式は以下のとおりです。

| 課金項目     | 課金方式                               | 価格             |
| ---------- | -------------------------------------- | ---------------- |
| マシン運用料金 | マシン運用時間料金はストリームごと、時間ごとに料金がかかります。 | 0.25美元/ストリーム/時間 |


>!
> - ストリーム数はinputおよびoutput数に応じて計算されます。inputストリームが1個、outputストリームが3個の場合、4個のストリームに応じてマシン運用時間が計算されます。
> - マスター、スレーブストリームを含む1つのタスクは、2個のストリームに応じてマシン運用時間が計算されます。
> - 課金は時間単位で計算されます。1時間に満たない使用時間は1時間として計算され、精算サイクルは日次決済で計算されます。

#### 課金例

お客様は2022年1月1日にStreamLinkサービスを使用し、inputが1個、outputが2個あるタスクを作成しました。当日の使用時間は6時間です。このとき、2022年1月2日に支払うライブストリーミングトラフィック請求書は以下のとおりとなります。

1日単位のマシン運用料金=3（ストリーム数）× 6（時間） × 0.25（米ドル/時間）= 4.5米ドル。
