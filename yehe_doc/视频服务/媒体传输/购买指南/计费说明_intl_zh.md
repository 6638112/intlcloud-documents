### 计费说明

StreamLink控制台基于流维度进行管理，在每个流中基于所配置的输入地域和输出地域可包含多条传输链路。基于StreamLink控制台可快速稳定地传输视频流媒体，同时还可对传输过程中的视频流进行全方位的质量监控。StreamLink计费项包含以下三项：

| 计费项 | 计费说明 | 付费方式 |
| -----------------| ---| --------- |
| 传输费用 |使用传输链路时计费。产生的费用按照每条链路的当日峰值带宽进行累加计费。 | 后付费-日结 |
| 机器运行费用 |使用机器资源进行传输时计费。由于对流媒体的传输需要消耗机器资源，根据任务时长将在输入端和输出端产生机器运行费用。 | 后付费-日结 |
| 转推流量费用 |将视频流转推至第三方时计费。根据对外输出流量，以及输出节点所在地域的单价进行计费。 | 后付费-日结 |

### 传输费用

#### 计费价格

StreamLink允许用户在控制台新建传输链路。在一个流当中的一个输入地域和一个输出地域之间定义为一条链路，对当日实际使用的各链路，将分别独立统计其带宽峰值，然后进行累加计费。此外，对于容灾配置的主路链路和备路链路，不会当作两条链路来分别独立统计带宽峰值，而是会合在一起计算带宽峰值。对于不同的流，每个流都将进行独立的费用计算。

| 计费项 | 计费方式 | 价格 |
| -----------------| ---| --------- |
| 传输费用 |按照每条链路的当日带宽峰值进行累加计费 | 7.6923美元/Mbps/天 |

#### 计费示例

您在2023年01月01日使用StreamLink服务，并且创建有A、B两个流。在A流中配置了输入和输出，创建了一条链路1。在B流中配置了输入和输出，创建了一条链路2。A流-链路1在中午12点达到峰值带宽10Mbps，B流-链路2在晚间22点达到峰值带宽10Mbps，则2023年01月02日您需支付的直播流量账单如下：
日传输费用 = 7.6923（美元/Mbps/天）× 20（Mbps）= 153.846美元。


### 机器运行费用

#### 计费价格

由于对流媒体的传输需要消耗机器资源，根据任务时长将在输入端和输出端产生机器运行费用。StreamLink服务中使用的机器运行费用计费方式如下：

| 计费项     | 计费方式                               | 价格             |
| ---------- | -------------------------------------- | ---------------- |
| 机器运行费用 | 按照各输入节点和输出节点分别统计机器运行运行时长，并进行累加计费 | 0.25美元/小时 |

>!
>- 按照input和output数量计算，例如：一个input和三个output，按照4个任务计算机器运行时间。
>- 如果包含主、备流，则主备流需要分别统计机器运行时间。
>- 计费按照小时为单位计算，不足一小时的使用时间按一小时计算，结算周期按日结进行计算。

#### 计费示例

您在2023年01月01日使用StreamLink服务，并创建有一个input，两个output的任务，当日使用时间为6个小时，则2023年01月02日您需支付的直播流量账单如下：

日机器运行费用=3（任务数量）× 6（小时） × 0.25（美元/小时）= 4.5美元。
