### 计费说明

StreamLink控制台基于流维度进行管理，每一个流对应一个流的传输链路。基于StreamLink控制台可快速稳定地传输视频流媒体，同时还可对传输过程中的视频流进行全方位的质量监控。StreamLink计费项包含以下三项

| 计费项 | 计费说明 | 付费方式 |
| -----------------| ---| --------- |
| 专线传输费用 |使用专线传输时计费；产生的费用按照每路流当日峰值带宽的累加进行计费 | 后付费-日结 |
| 机器运行费用 |使用机器进行传输时计费；产生的费用按照当日任务数量和时长以小时为单位进行计费 | 后付费-日结 |
| 转推流量费用 |带宽最高8Mbps；产生的费用按照当日推送的地区和流量大小进行计费 | 后付费-日结 |

### 专线传输费用

#### 计费价格

StreamLink允许用户在控制台新建传输链路，并根据当日实际使用的带宽峰值进行结算。

| 计费项 | 计费方式 | 价格 |
| -----------------| ---| --------- |
| 传输带宽 |按照日专线出口峰值带宽进行计费 | 7.6923美元/Mbps/天 |

#### 计费示例

您在2022年01月01日使用StreamLink服务，并且创建有A、B两条流，A流在中午12点达到峰值带宽10Mbps，B流在晚间22点达到峰值带宽10Mbps，则2022年01月02日您需支付的直播流量账单如下：
日专线传输费用 = 7.6923（美元/Mbps/天）× 20（Mbps）= 153.846美元。



### 机器运行费用

#### 计费价格

StreamLink服务使用中的机器运行费用计费方式如下：

| 计费项     | 计费方式                               | 价格             |
| ---------- | -------------------------------------- | ---------------- |
| 机器运行费用 | 机器运行时长费用，按照每路流每小时收费 | 0.25美元/流/小时 |


>!
> - 流数量按照input和output数量计算，一个input流和三个output，按照4路流计算机器运行时间。
> - 一个包含主、备流的任务，按照2路流计算机器运行时间
> - 计费按照小时为单位计算，不足一小时的使用时间按一小时计算，结算周期按日结进行计算。

#### 计费示例

您在2022年01月01日使用StreamLink服务，并创建有一个input，两个output的任务，当日使用时间为6个小时，则2022年01月02日您需支付的直播流量账单如下：

日机器运行费用=3（流数量）× 6（小时） × 0.25（美元/小时）= 4.5美元。
