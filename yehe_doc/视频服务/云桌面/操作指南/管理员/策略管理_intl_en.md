## Creating a custom policy
1. Click **Create policy** on the **Policy Management** page in the CVD console.
![](https://main.qcloudimg.com/raw/f4077fea6858b859c89c86f0b4ab160a.png)
2. Configure the following items as prompted:
![](https://main.qcloudimg.com/raw/aacf9c5549fa102b778cf62d98bead39.png)

| Configuration Item  | Description      | 
| ----------------- | --------------- | 
| Policy | Enter the policy name to differentiate between different policies. | 
| Description | Enter the optional policy description to further differentiate between different policies. |
| File redirection | Set whether to allow the end user to operate local files from within the CVD instance. |
| Clipboard redirection | Set the two-way or one-way copy permission between the CVD instance and local device. |
| Device redirection | Set whether to allow the CVD instance to use the USB storage devices and COM port peripheral devices connected to the local device. |
| Watermark | Specify the watermark content and display for the policy. |
| Users | Specify the users for whom the custom policy takes effect. Then, the policy will take effect for all the CVD instances of the specified users. |

3. After confirming that everything is correct, click **Save**. The policy will take effect at the next login.

## Deleting a policy
1. On the **Policy Management** page in the [CVD console](https://console.cloud.tencent.com/cvd), select the target policy and click **Delete policy** in the **Operation** column on the right or at the top.
![](https://main.qcloudimg.com/raw/6d20cfc720b0bd59215c63d63745e5bb.png)
2. Confirm the deletion.


## Modifying a policy
On the **Policy Management** page in the [CVD console](https://console.cloud.tencent.com/cvd), select the target policy and click **Modify** in the **Operation** column on the right.
![](https://main.qcloudimg.com/raw/5c39b0b496956cc7fe1befd9cd26a13f.png)
