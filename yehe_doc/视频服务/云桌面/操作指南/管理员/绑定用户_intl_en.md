## Overview
You can assign CVD resources to end users.
 
## Directions
### Binding one user
1. Log in to the [CVD console](https://console.cloud.tencent.com/cvd).
2. On the **Desktop List** page, select the target CVD instance and click **Bind user** in the **Operation** column on the right or at the top.
![](https://main.qcloudimg.com/raw/1d5b01f08a0113a64551c7396406c502.png)
3. In the **Bind user** pop-up window, select the user based on the department structure and click **Confirm**.

>!When binding one user, you need to select one CVD instance.

### Batch binding users
1. Log in to the [CVD console](https://console.cloud.tencent.com/cvd).
2. On the **Desktop List** page, select the target CVD instances and click **Bind user** at the top.
3. In the **Bind user** pop-up window, select users based on the department structure and click **Confirm**.
>!When binding multiple users, you need to select the same number of users as the selected CVD instances.
