## Overview
This document shows you how to assign desktops to end users.
 
## Directions
### Assigning one desktop
1. Log in to the [CVD console](https://console.cloud.tencent.com/cvd).
2. On the **Desktop List** page, select the target CVD instance and click **Bind user** in the **Operation** column on the right or at the top.
![](https://main.qcloudimg.com/raw/1d5b01f08a0113a64551c7396406c502.png)
3. In the **Bind user** pop-up window, select the group and then select the user to bind. Click **Confirm**.

>! If you assign one desktop at a time, you need to select one user to bind each time.

### Assigning multiple desktops
1. Log in to the [CVD console](https://console.cloud.tencent.com/cvd).
2. On the **Desktop List** page, select the target CVD instances and click **Bind user** at the top.
3. In the **Bind user** pop-up window, select the group and then select the users to bind. Click **Confirm**.
>! If you assign multiple desktops at a time, make sure the number of users you select is the same as the number of desktops to be assigned.
