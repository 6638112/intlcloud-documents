The CVD overview page displays your CVD usage in different networks.
![](https://qcloudimg.tencent-cloud.cn/raw/5161bc8910c439cdb02dbe1a65ae199e.png)

### Filtering desktops
You can filter your CVD desktops by region and network.
- Filter by region: Select a region ("All regions" or "Hong Kong" currently) to view desktops in that region.
- Filter by network: Select a [VPC](https://www.tencentcloud.com/document/product/215) in the selected region to view VPCs in that network.

### Viewing CVD usage
After **filtering** your CVD instances by region and network, you can view the following usage data:
- Total desktops: The total number of desktops.
- Running: The number of CVD desktops that are running.
- In use: The number of CVD desktops being used by end users. CVD instances in this status cannot be reset. Before you shut down, restart, terminate, or return such instances, be sure to notify your end users first.
- Shut down: The number of CVD desktops that are shut down.
- Expire in 7 days: The number of CVD desktops that will expire in seven days.

### Table fields
The table on the overview page shows the information of VPCs that meet your filter conditions:
- Network ID and name
- Subnet ID and name
- IP address
- Desktop count
- NAT network information
- NAT network bandwidth information
