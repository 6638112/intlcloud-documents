## Feature Overview
Enabled by [Tencent Cloud CAM](https://console.tencentcloud.com/cam), the user management feature allows you to set up an account system for CVD end users. It offers quick authentication management and secure login configuration services.
![](https://qcloudimg.tencent-cloud.cn/raw/80bf33f1ab284f469f13dc2cbec45ac7.png)

## Creating an End User

1. Log in to the [CAM console](https://console.tencentcloud.com/cam).
2. Create a sub-user. For detailed directions, see [Creating Sub-User](https://intl.cloud.tencent.com/document/product/598/13674).
3. Associate the sub-user with the QcloudVPCFullAccess or QcloudVPCReadOnlyAccess policy.
4. For the sub-user to access your desktops, you also need to associate them with the cloudCVDFullAccess or QcloudCVDReadOnlyAccess policy.
