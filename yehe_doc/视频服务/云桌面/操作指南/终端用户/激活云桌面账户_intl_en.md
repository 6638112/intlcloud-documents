1. After the CVD admin adds you as an end user, you will receive an activation email or SMS message.
![](https://main.qcloudimg.com/raw/e8e63f8eb1c14c884ff769f4d16b82e6.png)
2. Click **Activate now** in the email or SMS message to go to the activation page, where you need to set your password and security question.
>! In order to ensure the security of your CVD account, the password must meet the rules; otherwise, it cannot be set successfully.
3. After setting the password, you will be redirected to the CVD login page. We recommend you use Chrome for access.
![](https://main.qcloudimg.com/raw/2013e80bff0f63567a8732798b9e6000.png)
4. Enter the user ID and password to access CVD resources.
![](https://main.qcloudimg.com/raw/99dd9d5b4c3039afb6cf455bd300c110.png)
