| **Data** **Konfigurasi** (endpoint, input dan saluran) | Kami hanya memproses data ini untuk tujuan menyediakan Fitur dan sesuai konfigurasi spesifik kepada Anda dan pengguna akhir Anda. |

## 5.**PENYIMPANAN DATA**

Kami akan menyimpan data pribadi yang diproses sehubungan dengan Fitur sebagai berikut:

**Data Konfigurasi:** sampai Anda menghapus data tersebut secara manual, jika tidak, data akan dihapus secara otomatis ketika Anda menghapus akun Anda atau mengakhiri layanan.

**Data Streaming Langsung:** sementara di-cache dalam server selama streaming langsung.

Anda dapat meminta penghapusan data pribadi tersebut sesuai dengan DPSA (secara default, data tersebut disimpan sebagaimana ditetapkan di atas).
