## **1. HISTÓRICO**

Este módulo se aplica se você usa o StreamPackage (“**Recurso**”). Este módulo está incorporado ao Contrato de privacidade e segurança de dados localizado no (“[**DPSA**](https://intl.cloud.tencent.com/document/product/301/17347)”). Os termos utilizados, mas não definidos neste Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e este Módulo, ele deve ser aplicado conforme a inconsistência. 

## **2. PROCESSAMENTO**

Processaremos os seguintes dados em conexão com o Recurso:


| Informações pessoais                                         | Uso                                                          |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Configuração** **Dados** (ponto de extremidade, entrada e canal) | Apenas processamos esses dados com a finalidade de fornecer o Recurso de acordo com sua configuração específica a você e seus usuários finais. |
| **Dados de transmissão** **ao vivo** (dados de vídeo e áudio) | Apenas processamos esses dados com a finalidade de fornecer o Recurso a você e seus usuários finais. Observe que se você optar por criptografar os dados de transmissão ao vivo, este Recurso compartilhará dados de transmissão ao vivo com nosso recurso DRM (Digital Rights Management, gerenciamento de direitos digitais) para criptografia. Se você optar por distribuir transmissões ao vivo em nosso conteúdo Recurso cdn (Delivery Network, rede de entrega), este recurso compartilhará os dados de transmissão ao vivo com nosso recurso CDN. |


## **3. REGIÃO DOS SERVIÇOS**

Conforme especificado no DPSA.


## **4. SUBPROCESSADORES**

Conforme especificado no DPSA, incluindo Aceville Pte Ltd.

## **5. RETENÇÃO DE DADOS**

Armazenaremos dados pessoais processados em conexão com o Recurso da seguinte forma:

- **Dados de configuração:** até que você exclua manualmente esses dados, caso contrário, eles serão excluídos automaticamente quando você excluir sua conta ou encerrar o serviço.
- **Dados de transmissão ao vivo:** temporariamente armazenados em cache em servidores durante o livestream. Você pode solicitar a exclusão desses dados pessoais de acordo com o DPSA (por padrão, esses dados são retidos conforme estabelecido acima).


## **6. CONDIÇÕES ESPECIAIS**

Você deve garantir que este recurso seja usado somente por usuários finais com a idade exigida para que possam consentir com o processamento de seus dados pessoais. Aqui pode haver diferença dependendo da jurisdição em que o usuário final está localizado.
