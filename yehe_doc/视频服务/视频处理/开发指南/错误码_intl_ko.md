## 에러 코드 리스트
### 비디오 처리 유형 에러 코드

| 에러 코드 | 의미 |
| -- | -- |
| InvalidInput | 유효하지 않은 매개변수. 입력한 매개변수를 확인하십시오. |
| InvalidInput.InvalidTimeOffset | 유효하지 않은 매개변수: 지정한 시간이 유효하지 않습니다. |
| InvalidInput.DefinitionNotExist | 유효하지 않은 매개변수: 지정한 템플릿 ID가 존재하지 않습니다. |
| SourceFileError | 원본 파일 오류(비디오 데이터 손실 등). 원본 파일에 문제가 없는지 확인하십시오. |
| SourceFileError.NoVideoMedia | 원본 파일 오류: 비디오 트랙 화면이 없습니다. |
| SourceFileError.NoVideoResolutio | 원본 파일 오류: 원본 파일의 해상도를 가져올 수 없습니다. |
| InternalError | 내부 서비스 오류. 다시 시도하세요. |
