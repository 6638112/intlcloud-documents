- [User Guide for Desktop Version](https://intl.cloud.tencent.com/document/product/436/32565)
- [User Guide for Mobile Version](https://intl.cloud.tencent.com/document/product/436/32566)

## COSBrowser Desktop Version

COSBrowser Desktop Versionfocuses on resource management and allows you to upload and download data in batches.

> !COSBrowser Desktop Version uses the system-configured proxy to connect to the internet. Please make sure that your proxy is set up properly or disable the proxy configuration if it fails to connect to the internet.
> - For queries on Windows, go to "Internet Options".
> - For queries on macOS, go to "Network Preferences".
> - For queries on Linux, go to System Settings > Network > Network Proxy.

#### Basic Features

COSBrowser Desktop Version has the following features:

| Feature | Description |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| [Uploading file/folder](https://cloud.tencent.com/document/product/436/38103#upload) | This feature allows you to upload files/folders to a bucket separately, in batches, or incrementally |
| [Downloading file/folder](https://cloud.tencent.com/document/product/436/38103#download) | This feature allows you to download files/folders to the local file system separately, in batches, or incrementally |
| [Deleting file/folder](https://cloud.tencent.com/document/product/436/38103#delete) | This feature allows you to delete files/folders from a bucket separately or in batches |
| [Copying and pasting file](https://cloud.tencent.com/document/product/436/38103#copy) | This feature allows you to copy files/folders separately or in batches from one directory to another directory |
| [Renaming file](https://cloud.tencent.com/document/product/436/38103#rename) | This feature allows you to rename files in a bucket |
| [Creating folder](https://cloud.tencent.com/document/product/436/38103#newfolder) | This feature allows you to create folders in a bucket |
| [Viewing file details](https://cloud.tencent.com/document/product/436/38103#view) | This feature allows you to view the basic information of files in a bucket |
| [Generating file link](https://cloud.tencent.com/document/product/436/38103#generatelinks) | This feature allows you to generate a file access link with a validity period by requesting a temporary signature |
| [Previewing file](https://cloud.tencent.com/document/product/436/38103#preview) | This feature allows you to preview media files (images, videos, and audios) in a bucket |
| [Searching for file](https://cloud.tencent.com/document/product/436/38103#searchfile) | This feature allows you to search for files in a bucket through prefix search |
| [Searching for bucket](https://cloud.tencent.com/document/product/436/38103#searchbuckete) | This feature allows you to search for created buckets |
| [Viewing multiple file versions](https://cloud.tencent.com/document/product/436/38103#viewfiles) | This feature allows you to view the historical versions of files in a bucket where versioning is enabled |
| [Viewing bucket details](https://cloud.tencent.com/document/product/436/38103#viewbucket) | This feature allows you to view the basic information of buckets |
| [Setting up proxy](https://cloud.tencent.com/document/product/436/38103#sets) | This feature allows you to set up a proxy to access COS |
| [Setting the number of concurrent transfers](https://cloud.tencent.com/document/product/436/38103#sets) | This feature allows you to set the number of concurrent transfers for file upload or download |
| [Setting the number of parts to be transferred](https://cloud.tencent.com/document/product/436/38103#sets) | This feature allows you to set the number of parts for multipart upload or download |
| [Setting the number of retries upon transfer failure](https://cloud.tencent.com/document/product/436/38103#sets) | This feature allows you to set the number of retries upon upload or download failure |
| [Setting upload check](https://cloud.tencent.com/document/product/436/38103#sets) | This feature allows you to double-check files uploaded to a bucket |
| [Setting MD5 checksum calculation during upload](https://cloud.tencent.com/document/product/436/38103#sets) | This feature allows you to calculate MD5 checksums of files uploaded to a bucket and add them to custom headers |
| [Viewing local log](https://cloud.tencent.com/document/product/436/38103#sets) | This feature allows you to save the record of operations on COSBrowser in the form of local log |

## COSBrowser Mobile Version

COSBrowser Mobile Version focuses on viewing and monitoring of resources and allows you to monitor data of COS such as storage and traffic anytime, anywhere.

#### Basic Features

COSBrowser Mobile Version has the following features:

| Operation Name | Description |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| [Login with WeChat](https://cloud.tencent.com/document/product/436/38105#dulu) | This feature allows you to log in with WeChat |
| [Data overview](https://cloud.tencent.com/document/product/436/38105#dateview) | This feature allows you to view recent data usage |
| [File batch operations](https://cloud.tencent.com/document/product/436/38105#filebatch) | This feature allows you to upload, download, delete, copy, or move files in a bucket in batches |
| [Sharing and uploading](https://cloud.tencent.com/document/product/436/38105#shareupload) | This feature allows you to share and upload files from third-party apps to a bucket |
| [Renaming file](https://cloud.tencent.com/document/product/436/38105#rename) | This feature allows you to rename files in a bucket |
| [Creating folder](https://cloud.tencent.com/document/product/436/38105#newfolder) | This feature allows you to create folders in a bucket |
| [Viewing file details](https://cloud.tencent.com/document/product/436/38105#view) | This feature allows you to view the basic information of files in a bucket |
| [Generating file link](https://cloud.tencent.com/document/product/436/38105#generatelinks) | This feature allows you to generate a file access link with a validity period by requesting a temporary signature |
| [Searching for file](https://cloud.tencent.com/document/product/436/38105#searchfile) | This feature allows you to search for files in a bucket through prefix search |
| [Searching for bucket](https://cloud.tencent.com/document/product/436/38105#searchbuckete) | This feature allows you to search for created buckets |
| [Viewing bucket details](https://cloud.tencent.com/document/product/436/38105#viewbucket) | This feature allows you to view the basic information of buckets |
| [Creating bucket](https://cloud.tencent.com/document/product/436/38105#createbucket) | This feature allows you to create buckets |
| [Adding access path](https://cloud.tencent.com/document/product/436/38105#addaccess) | This feature allows a sub-account that has no access to the bucket list to enter a bucket for resource management by adding an access path |
